# 星运自坐（十二长生 / 日主自坐）

## 定义
- **星运**：天干在十二地支上的"**生命阶段**"状态 → 又称**十二长生**。
- **自坐**：**日主（日干）"坐"在日支上** → 指日干在日支地支所处的生命阶段。
- **星运自坐** = 日主对自己所坐地支的**生命阶段判断**，直接描述**一个人"本人状态"的底色**。
- 是八字分析中**最核心的身强弱判断工具**之一。

## 十二长生：天干的生命周期
把天干比作一个生命体，它在十二地支上有 12 个阶段：

| 阶段 | 意象 | 能量状态 |
|---|---|---|
| **长生** | 婴儿出生、新生萌芽 | 初生、有希望 |
| **沐浴** | 婴孩洗澡、青春迷茫 | 桃花、冲动、不定 |
| **冠带** | 少年加冠、即将独立 | 上升期、青涩有力 |
| **临官**（禄） | 青年就业、成家立业 | 稳固、成熟 |
| **帝旺**（刃） | 壮年鼎盛、顶点 | 极旺、刚烈 |
| **衰** | 老年体衰、退意 | 回落、转守 |
| **病** | 疾病缠身 | 虚弱、需养 |
| **死** | 死亡 | 最低点 |
| **墓** | 入墓归藏 | 收敛、保存 |
| **绝** | 断绝无踪 | 空无、新生前夜 |
| **胎** | 受孕成胎 | 潜伏、孕育 |
| **养** | 母腹养育 | 静养、准备 |

- 这 12 个阶段**循环往复**：绝 → 胎 → 养 → 长生 → ……
- **不是字面意义的生死**，是**能量周期的象征**。

## 核心：日支 = 命主的"内在发条"
日支是八字中**距离日干最近**的地支：
- 代表**身体、内心、夫妻宫、潜意识、本能**。
- 日主坐在日支上 → 就像**一个人脚踩的地面**。
- 这个地面**是什么状态**（长生 / 帝旺 / 墓 / 绝 ...）→ 决定这个人**底层能量的稳定性**。

## 三类自坐的意义

### 第一类：能量强势自坐（身强底色）
- **自坐长生**：有根、能持续生发、不易倒。
- **自坐临官（禄）**：身自立、稳固有力。
- **自坐帝旺**：身极旺，刚烈、锋芒、带羊刃气。
- **自坐冠带**：上升中的力量，青涩但向上。

对应：**身强、有根、独立、能扛**。

### 第二类：能量中间自坐（过渡态）
- **自坐沐浴**：桃花、不稳、情绪起伏。
- **自坐衰**：力量下降，内心偏静。
- **自坐养**：静中有养，潜力未显。

对应：**介于强弱之间，需结合大运判断**。

### 第三类：能量弱势自坐（身弱底色）
- **自坐病**：虚弱多思、身体易弱。
- **自坐死**：能量最低、需外力支援。
- **自坐墓**：内收藏锋、外显不足。
- **自坐绝**：无根无援、需依附他人。
- **自坐胎**：潜伏未显、晚发。

对应：**身弱、依附、内敛、多思**。

## 常见误区

### 1. "自坐死 / 自坐绝 = 早亡"
- **错**。这是字面意义的误读。
- 十二长生是**能量周期**，不是生死判断。
- 自坐绝的人只是"**当前能量最弱**"，需要其他字 / 大运来生扶。
- 许多大富大贵者都自坐死 / 绝 / 墓（靠外应成就）。

### 2. "自坐帝旺一定好"
- 自坐帝旺 = **身极旺** → 好处是**刚烈有力**。
- 坏处是：
  - **难妥协、易冲动**。
  - **需要有出口**（食伤泄秀 / 官杀驾制），否则内耗。
- 不配合 → 同样可能一事无成。

### 3. 忽视日支的其他意义
- 日支不仅是"自坐"位，还是：
  - **夫妻宫**。
  - **身体 / 内心**。
  - **潜意识 / 理想型**。
- 分析日支要综合多角度。

### 4. 把十二长生当全部
- 十二长生只是**天干在地支的状态参考**。
- 不能单独用来判命。
- 必须**结合日主强弱、十神、格局、大运**综合考量。

## 十二长生 + 十神的叠加效果
日主坐下的十神与十二长生叠加 → 双重信息：

| 组合示例 | 含义 |
|---|---|
| 自坐正官 + 临官 | 事业根基稳、体制内发力 |
| 自坐七杀 + 帝旺 | 压力大 + 身刚 → 有成就但内耗重 |
| 自坐正财 + 墓 | 财归库 → 守财 / 积累型 |
| 自坐偏财 + 绝 | 财无根 → 财来财去、难聚 |
| 自坐比肩 + 长生 | 自立而有根 → 独立创业型 |
| 自坐食神 + 帝旺 | 才华外显 → 创作爆发型 |
| 自坐伤官 + 沐浴 | 锋芒 + 桃花 → 才华 + 情感波澜 |

## 自坐与六亲的提示
- **男命自坐偏弱**（如病、死、绝、墓）→ 妻缘或身体需关注。
- **女命自坐官杀旺位**（如帝旺）→ 夫星气势强，夫妻关系可能"强对强"。
- **无论男女自坐比劫位** → 独立性强，亲密关系中主导意识强。

## 核心判断
- 自坐是**最快速把握日主底色**的工具：
  - 一眼看到"**这个人站在什么土地上**"。
- 但**不能单独定论**：
  - 身强 / 身弱还需综合月令、天干、其他地支。
  - 吉凶要看格局、用神、大运。
- 识别意义：
  - **理解自己的底层能量状态**。
  - **为选择配偶、事业、大运方向提供参考**。

## 一句话总结
- 星运自坐 = **你脚下的土地是什么**。
- 土地肥沃（长生 / 禄 / 帝旺）→ 本人自带底气。
- 土地贫瘠（死 / 绝 / 墓）→ 本人需借外力支援。
- 但肥沃不必然贵、贫瘠不必然贫 —— 关键在于**你在这片土地上种什么、怎么种**。
