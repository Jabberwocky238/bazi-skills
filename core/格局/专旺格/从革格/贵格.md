# 从革清秀（从革格贵格）

> **定位**：从革格的**贵格升级态**——从革（金专旺）+ 水泄秀气 → 成就"**锋锐兼清秀**""**武贵中见文采**"的顶级金气贵格。
> 本文是 `从革格/成立条件.md` 中"**条件 6（助力条件）**"的完整专论。

## 定义
- **从革清秀**：从革格（金专旺）+ **水泄秀气**（金生水）→ 成就"**武贵清秀**""**刚锐中显通透**"的复合贵格。
- "**清秀**"二字：
  - **清** = 水之清澈（金生水则水更清）；
  - **秀** = 金气被水泄而显秀——锋锐之金得水洗涤，由"**刚而孤**"升为"**刚而明**"。
- 古籍《穷通宝鉴》："**庚金逢壬水，为金白水清**"——从革之金得水而显秀，即从革清秀。

## 与普通从革格的区别

| 对比项 | 纯从革格 | 从革清秀（本条）|
|---|---|---|
| 金气 | 满盘金气（三合或三会金局）| 满盘金气 + **水点缀** |
| 秀气 | 缺——偏冷刚孤寂 | 有——金生水成秀 |
| 气质 | 锋锐决断但偏冷孤 | **刚锐 + 清秀 + 通透** |
| 贵气 | 贵格（纯专旺）| **贵格之贵**（专旺 + 秀气双全）|
| 代表行业 | 军警 / 司法 / 锋锐职业 | **高端司法 + 分析 + 策略**（武中带文） / **精英思辨型**（刚锐判断 + 清晰思考）|
| 气象 | 剑锋出鞘 | **剑如秋水**（锋锐之刃倒映清水之光）|

## 从革清秀的气机

- **金生水 → 水泄金**：
  - 金之刚锐被水**化成流动的智慧**。
  - 孤冷之金得水**滋润而不失其锐**。
  - 武贵得文采 → 从"**纯粹的断**"升为"**断中有思**"。
- 古籍："**金白水清者，贵而清秀**"——金为刚锐的决断，水为清澈的智慧，二者合为"**思辨型精英**"。

## 与"金白水清"（两气成象）的关系

**金白水清**在命理中有两种身份：

### 身份一：两气成象·金水对（金水两气双清）
- 结构：金 + 水两气力量相当。
- 位置：`../../../两气成象/金水对/金白水清/`
- 气象：两气双清，金水平衡。

### 身份二：从革清秀（从革格 + 水泄秀）
- 结构：金专旺（满盘）+ 水点缀。
- 位置：本文。
- 气象：金一气独旺 + 水点缀为秀。

**两者同名而结构不同**——详见后文对比表。

## 成立条件

### 条件 1：先满足从革格所有成立条件
- 日主为庚或辛金。
- 地支会成金局（巳酉丑 三合 或 申酉戌 三会）。
- 天干多透金。
- 原局无丙丁火透干熔金。
- 原局无甲乙木过重对抗。
- 详见 [`成立条件.md`](./成立条件.md) 前五条。

### 条件 2：有水（壬癸 或 亥子）点缀泄秀

- **古籍出处**：
  - 《穷通宝鉴·庚金章》："**庚金逢壬水，为金白水清**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 水（壬癸天干 或 亥子地支）在命局中**至少一位**——透干最佳，藏支亦可。
  - 水**不宜过多**——一至两位为"点缀"，三位以上则过度泄金，降格为"金水两气成象"（金水对）。
- **不满足则破**：
  - 无水 → 只成纯从革格，不成清秀。
  - 水过多 → 金被水泄过度 → 格变（非从革清秀，转入两气成象·金水对讨论）。

### 条件 3：水须得金生而不被土克

- **古籍出处**：
  - 《穷通宝鉴》："金白水清之水，须金生而无土克。"
- **客观判据**：
  - 水的位置靠近金——即金与水紧贴相生（天干庚辛旁有壬癸；或地支申 / 酉旁有亥 / 子）。
  - 原局无戊己土透干克水（戊己本应生金而不克水，但若土过重反堵水之泄路）。
- **不满足则破**：水被土克、水远离金 → 清秀之气不足。

### 条件 4：水不过旺（维持"从革为主、水为秀"的基调）

- **古籍出处**：
  - 《滴天髓·从象》："一气成象，**微泄秀气方妙**"——关键在"微"。
- **客观判据**：
  - 水的数量 ≤ 金的一半（如金四位、水一至两位）。
  - 月令仍为金（巳 / 酉 / 丑 / 申 / 戌），若月令变水则格变。
- **不满足则破**：水过旺 → 本末倒置 → 不再是从革清秀，降为金水两气对。

---

**四条同时满足**方为真从革清秀。核心在**"主次有序"**——从革（金）为主、水为辅。水是"秀气的出口"，不是"取代金"的主角。

## 喜忌总则（相对纯从革格的变化）

| 类别 | 纯从革格 | 从革清秀 |
|---|---|---|
| **金运** | 大喜 | 喜（维持本气）|
| **土运** | 喜（生金）| 喜（但忌过多克水断秀）|
| **水运** | 喜（适度泄秀）| **大喜**（从革清秀兑现）|
| **木运** | 大忌（克金）| 大忌 |
| **火运** | 大忌（熔金）| 大忌 |

## 现实表现

### 气质（两种层次）
- **纯从革**：锋锐 + 决断 + 孤冷（武贵型，偏冷刚）。
- **从革清秀**：锋锐 + 决断 + **清透智慧**——刚锐之中带文采，**武贵 + 文才双全**。

### 典型人生节奏
- **早年**：锋芒显现，被称"**聪明且果断**"。
- **中年**：从纯锋锐走向"**锋锐 + 洞察**"——决断能力与思辨深度并行。
- **晚年**：成为**"武中带文"的领军人物**——既能决断又能著书立说。

### 人物画像
- 高级法官、大律师、顶级检察官——决断 + 思辨两全。
- 军事家 + 战略家——既能带兵又能谋划。
- 金融分析师、战略咨询师——金锐之断 + 水清之思。
- 精密科研工作者——刀锋般的精确 + 水般清澈的思维。
- 典型："**能写出兵法的将军**""**能下判决的法官**"。

## 从革清秀 vs 金白水清（两气成象）—— 易混淆对比

| 对比项 | 从革清秀（本条，专旺格贵）| 金白水清（两气成象）|
|---|---|---|
| **所在分类** | 专旺格（金一气独旺）| 两气成象（金水对）|
| **金气** | 四柱满盘金 | 金为两气之一（非满盘）|
| **水气** | 点缀（一至两位）| 两气之一（与金力量相当）|
| **气象** | 锋锐之金得水洗涤 | 金水两气双清 |
| **贵气来源** | 金专旺 + 水吐秀 | 金水相生双清 |
| **详见** | 本文 | `../../../两气成象/金水对/金白水清/` |

  - "**生生不息，流通有情**。"——清秀的本质即生生不息。

## 一句话总结
- 从革清秀 = **剑锋倒映秋水**——锋锐之金得清水洗涤，成就"**武贵带文采**"。
- 纯从革是"**锋**"——锋利果决；从革清秀是"**锋而秀**"——锋利之中见清透。
- 关键在"**主次有序**"——金是主、水是秀；金专旺生秀则贵，水过反客则降。
- 与润下清华（水主 + 金木秀）为一对——水主则润下、金主则从革，两者同理反向。
