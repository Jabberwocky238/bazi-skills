# 阴阳五行

## 定义
- **阴阳** = 中国古代用以描述事物**对立统一**关系的两类基本属性，是一切变化的动力之源。
- **五行** = 木、火、土、金、水——古人用以描述世界万物生成、运动、转化的**五种基本形态**。
- 阴阳是"对立的两面"，五行是"循环的五态"，两者合称**阴阳五行**，为所有易学科目（八字、六爻、奇门、风水、梅花）的总根。

## 一、阴阳

### 1. 阴阳与矛盾
- 阴阳即古代的**两点论**（对立统一），与形而上学的"一点论"相对。
- 对立统一律揭示：**一切事物都因内部矛盾而发展**，矛盾即阴阳。
- 偏激、非黑即白的人不明阴阳之理——**把话说满、把事做绝**，正是违背阴阳辩证。
- 事物的转变需要推动力；单一因素只能单向发展——"没有白天就没有黑夜"。

### 2. 阴阳的特性

| 阳 | 阴 |
| --- | --- |
| 雄性、刚健、向上、生发 | 雌性、柔弱、向下、收敛 |
| 展示、外相、伸展 | 隐蔽、内向、收缩 |
| 明朗、积极、好动 | 暗沉、消极、喜静 |
| 发散（如太阳、火） | 收缩（如月亮、水） |

### 3. 阴阳的相对性
- 阴阳**不是绝对**的标签，而是**相对**的：
  - 男性为阳，但成年男性中，强壮为阳、瘦弱为阴。
  - 有钱男性为阳；同为有钱男性中，健康为阳、不健康为阴。
  - 国家与国家之间，强国为阳、弱国为阴。
- 关键在于**理解与应用**，不是死记硬背。

### 4. 阴阳的日常体感
- **宠物**：咬人的为阳、温顺的为阴。
- **昼夜**：白天为阳、黑夜为阴。
- **体感**：热、出汗为阳；冷、起鸡皮疙瘩为阴。
- **形态**：方为阳、圆为阴；胖为阳、瘦为阴。
- **情绪**：积极阳光为阳、愁眉苦脸为阴。

### 5. 阴阳的转化
- 阴阳转化在**特定条件下**发生——"**物极必反**"。
- 例：
  - 黑夜过去即光明；严冬过去即春天。
  - 胆小懦弱的人，在极大压力下会转为敢作敢为。
- 中国哲学重**中庸**——不走极端、讲究平衡。

## 二、五行

### 1. 五行的内涵
- **木**：代表**生长**——向上、伸展、生发。
- **火**：代表**爆发**——上升、发光发热。
- **土**：代表**融合**——承载、储藏、过渡（每个季节的最后一个月皆属土）。
- **金**：代表**敛聚**——收束、肃杀、结构化。
- **水**：代表**浸润**——向下、流动、滋养。

五行是**原始的系统论**，描述事物的运动形式与转化关系；阴阳是对立的两面，五行是循环。

### 2. 五行相生
**木生火、火生土、土生金、金生水、水生木**——循环相生。

| 相生 | 现实意象 |
| --- | --- |
| 水生木 | 水滋润树木成长 |
| 木生火 | 树木燃烧生火 |
| 火生土 | 火燃烧成灰烬变土 |
| 土生金 | 土中蕴藏矿石、金属 |
| 金生水 | 高温融化金属化为液态；也可理解为西方属金、大江大河源自西方高山（金生水） |

### 3. 五行相克
**木克土、土克水、水克火、火克金、金克木**——循环相克。

| 相克 | 现实意象 |
| --- | --- |
| 木克土 | 树木破土而出 |
| 土克水 | 堤坝改变水流 |
| 水克火 | 水能灭火 |
| 火克金 | 火可熔金 |
| 金克木 | 金属刀斧可砍伐树木 |

### 4. 五行与方位、季节、色、脏、德、味、情
| 五行 | 方位 | 季节 | 颜色 | 五脏 | 五德 | 五味 | 情质 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| **木** | 东 | 春 | 青绿 | 肝 | 仁 | 酸 | 正直、仁义 |
| **火** | 南 | 夏 | 红 | 心 | 礼 | 苦 | 热情、重礼节 |
| **土** | 中央 | 四季之末月 | 黄 | 脾 | 信 | 甘 | 稳重、讲信誉 |
| **金** | 西 | 秋 | 白/金 | 肺 | 义 | 辛 | 刚烈、重义气 |
| **水** | 北 | 冬 | 黑/蓝 | 肾 | 智 | 咸 | 智慧、善变 |

- 记忆口诀：东方多水木盛、南方炎热火旺、西方沙漠植被少属金、北方寒冷属水、中央为土（万物归宿）。
- 每季末的土 = 一个过渡中介，把前一季节收拢毁灭，进入下一季节。

## 三、五行生克的辩证性

### 1. 相生不是绝对——过生反害

| 辩证规律 | 含义 |
| --- | --- |
| 木多火塞 | 木材过多则火被掩盖或熄灭 |
| 火多土焦 | 火势过猛则土被烧焦 |
| 土多金埋 | 土过厚则金被埋 |
| 金多水浊 | 金过多则水泛 |
| 水多木漂 | 水过剩则木漂浮或腐烂 |

### 2. 相克不是绝对——反侮成克

| 辩证规律 | 含义 |
| --- | --- |
| 木坚金缺 | 木过硬则金刀折断 |
| 土重木折 | 土过厚则木无法破土 |
| 水大土荡 | 水过强则土被冲毁 |
| 火烈水干 | 火势极大则水被蒸发 |
| 金多火熄 | 金过多则火无法熔之 |

**重点：相生相克都是辩证的、而非绝对的。** 八字中只有一个木，却有三四个水同时生它，反而会出现"水多木漂"的不良局面，从相生变成相克。

## 四、阴阳五行与八字平衡

### 1. 得月令 ≠ 一定身强
- 不能只看月令就判断身强身弱。
- 要综合八个字的**阴阳、五行、燥湿、寒暖**是否平衡：
  - **燥**（干燥）、**湿**（潮湿）、**寒**（寒冷）、**暖**（温暖）——八字中过燥、过湿、过寒、过暖都是问题。
- 运势好常因**大运流年把阴阳五行燥湿寒暖调至平衡**。

### 2. 例：戊土生丑月
- 粗看：戊土得月令丑土助扶 → 身强 → 喜财官（水、木）。
- 综合看：臣丑土一片湿寒 → 水虽有根却不能用 → 需**木火济湿**调候。
- 大运走木火 → 升职加薪。

这说明：**判断喜用神必须结合整体平衡，而非只看月令强弱**。

## 五、五行与性格

- **木行人**：重仁义、正直、有主见。
- **火行人**：重礼节、热情、直率。
- **土行人**：重信誉、稳重、厚道。
- **金行人**：重义气、刚烈、决断。
- **水行人**：主智慧、灵活、多变。

五行偏旺或为用，则其所代表的性格特质更突出。

## 六、核心原则：中庸与平衡

- 易学最终都归于**致中和**。
- 八字平衡者：
  - 命局中没有过于旺盛或过于衰弱的元素。
  - 身体健康、寿命较长。
- 八字严重失衡者：
  - 过旺或过衰的元素带来难以承受的痛苦。
  - 易在那个五行对应的领域频出问题。
- 天时、地利、人和的调节，本质都是为了**五行平衡、阴阳和合**。

## 七、一句话总结

- **阴阳是变化的动力、五行是变化的轨迹**。
- 五行生克有常、又有辩证——**过生反害、过克反侮**。
- 八字吉凶的根本，不在某一字是喜是忌，而在**整体平衡与流转顺畅**。
- 学八字即是学习在动态变化的阴阳五行里找到平衡点。
