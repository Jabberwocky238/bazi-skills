# 穿（六害）

## 定位
- 地支之间一种**非常激烈的矛盾关系**。
- 代表"你中有我、我中有你，互相牵制、互相损耗，谁也占不到便宜"。
- 在地支的刑冲合害形态中，**是最锐利、最不好相处的一种**。
- 与"冲"的区别：
  - **冲**：对顶、掀桌子、明面上撕破脸。
  - **穿**：按理较劲、互相穿心、互相拆台，表面勉强维持，里面已一塌糊涂。

## 六穿（六害）
- 子未穿
- 丑午穿
- 寅巳穿
- 卯辰穿
- 酉戌穿
- 申亥穿

### 分类
- **申穿**（以申亥为代表）：多是阴四川、申亥穿，穿里还带一点生扶，气机没那么绝。
- **克穿**（子未、丑午、卯辰、酉戌）：以相克为主，伤害程度更重。

## 六穿的具体含义

### 子未穿
- 子属水、未属土，土克水，未中藏火藏木，里外互相搅。
- 身体：肾、膀胱、泌尿生殖反复不适，睡眠易被情绪牵着走。
- 关系：子多主子女、未多主家族责任 → 常见为孩子操心、为家里掏空自己。

### 丑午穿
- 丑为湿土，午为火，火被湿土压住，气上不来。
- 性格：怒而不发，心里有火嘴上不说。
- 健康：心脏、血压、情绪系统易有反应。
- 生活：想做点什么总被现实拖住，有冲劲却被琐事拖回原地。

### 寅巳穿
- 寅为阳木，巳为阳火，木生火看似相生，但地支位置不对就会变成"过火"。
- 性格：急性子、火爆、思路快、情绪来得快去得快。
- 事业：频繁跳槽、频繁折腾，爱折腾也能折腾出成绩，但对身体、感情消耗大。

### 卯辰穿
- 卯为纯木，辰为湿土中带水带木，两边搅在一起 → "地基不稳"。
- 表现：内心没有安全感，想法多但落地能力弱，项目常做到一半就变化。
- 生活：装修、房产投资、搬家迁移往往不止一次；情绪紧张焦虑、想冲又怕出事。

### 酉戌穿
- 酉为金，戌为燥土中带火。
- 健康：心经被燥土害，或金火相克。
- 性格：立场倔强、不服输，说话直接犀利、易树敌。
- 现实：合作关系破裂、合同纠纷、口舌是非多；一个小误会可拖成多年心结。

### 申亥穿
- 申为金，亥为水，金生水、水又耗金，互相牵扯 → "越帮越乱"。
- 健康：呼吸系统、肺、腰、肾这条线时好时坏。
- 情绪：敏感多疑，一会理性一会想得玄。
- 倾向：对玄学、心灵、宗教天然好奇，也容易在这些圈子里遇人遇事。

## 六亲维度上的穿

- **父母宫**：不是简单的不孝顺，而是"相看两厌"，各说各的，谁也说服不了谁。常见长期分离或同住一屋各有各的世界。
- **婚姻宫**：感情里有暗伤，与配偶之间总有一道过不去的坎，说开很难、不说又压着，易卷入三角关系或暧昧不清。
- **子女宫**：为孩子劳心劳力；孩子身体、情绪、学业路径让父母提心吊胆，不绝对凶但肯定不省心。
- **事业宫 / 用神之上**：职场暗斗，小人背后捅刀，项目半途生变，签好的合作临时反悔；外表正常，水底下浪很大。

## 核心理解：互相损耗
- 穿的核心就是四个字：**互相损耗**。
- 不是简单谁输谁赢，而是两个地方互相穿透、两败俱伤。
- 受伤轻重看：谁的力量大、谁是用神、谁是忌神、谁落在关键位置。

## 关键判断
- 命盘有穿，**不等于一生就一定大祸临头**。
- 更多的是提醒：某个领域（感情、家庭、健康、合作）天生带着暗线。
  - 不适合存侥幸心理。
  - 有问题要早看清、早处理。

### 两种情形
- **穿在用神上** → 顺畅度打折；但反过来可能逼着你早早看清现实、学会止损、学会看人。
- **穿在忌神上** → 有时反而是好事——把原本容易出祸的那股劲消耗掉，人生少走一些极端。

## 核心判断
看穿：
- 不要只记住"某某穿主大凶"。
- 也不要被吓住。
- 真正有价值的判断是：看懂这股互相损耗的力量最终是把你拖下水，还是逼着你学会转身。
- 命盘给的是倾向，穿提醒的是"在这些位置上做人做事千万不要糊涂"。
