# 十神概述

## 什么是十神

十神是以**日元**（日柱天干、命主自身）为太极点，依据其他七字（三柱天干 + 四柱地支藏干）与日元之间的**五行生克**与**阴阳异同**，衍生出来的**十种关系标签**。

- 本质：十神不是神仙鬼怪，而是"**两种五行之间的一种搭配组合**"。
- 作用：用十个坐标点把命盘这张五行地图标注出来，覆盖**运势、事业、学业、健康、性格、六亲**等全部命理维度。
- 范围：本命盘主星（天干）、副星（地支藏干）、大运流年天干，都要与日元挂出十神。

## 十神的五组十种命名

从日元"我"出发，与其他干支的五行关系只有五路；每路再按阴阳切开，合成十种。

| 五行关系 | 十神大类 | 正（阴阳不同） | 偏（阴阳相同） |
| --- | --- | --- | --- |
| 生我 | 印星 | **正印** | **偏印**（又名枭神）|
| 克我 | 官杀 | **正官** | **七杀**（又名偏官）|
| 我生 | 食伤 | **伤官** | **食神** |
| 我克 | 财星 | **正财** | **偏财** |
| 同我 | 比劫 | **劫财**（异阴阳） | **比肩**（同阴阳） |

**关键记忆点**：

- 印、官、财三组，"**正**"都是异阴阳、"**偏**"都是同阴阳——因为异性相生 / 相克属于"和顺端正"的作用方式。
- **食伤一组反过来**："**食神**"是我生同阴阳、"**伤官**"是我生异阴阳——同性相生更温和稳定（食神），异性相生更锋芒外泄（伤官）。
- **比劫一组也反过来**："**比肩**"是同五行同阴阳、"**劫财**"是同五行异阴阳——完全相同才叫"比肩并立"，阴阳相异反而劫夺力更强。

> 一条口诀：**印官财——正异偏同；食伤——食同伤异；比劫——比同劫异**。

## 十神相生图（五行 + 阴阳双重）

五行相生的顺序：木 → 火 → 土 → 金 → 水 → 木。对应到十神，就形成一条闭环：

**印星 → 比劫 → 食伤 → 财星 → 官杀 → 印星**

| 相生 | 形象解读 |
| --- | --- |
| 印生比劫 | 学识修养越广，越受人敬重，身边朋友兄弟（比劫）自然多 |
| 比劫生食伤 | 朋友多 → 聚会吃喝享乐机会多 → 口福与欲望流出 |
| 食伤生财 | 聚会中信息、资源、商机流动 → 变现机会增加 |
| 财生官杀 | 有财者纳税捐赠、谋取位置 → 获得官职与话语权 |
| 官杀生印 | 当官后需匹配格局眼界修养 → 学识、修养自然增长 |

## 十神相克图

五行相克的顺序：木克土、土克水、水克火、火克金、金克木。对应到十神，也是一条闭环：

**印星 → 食伤 → 官杀 → 比劫 → 财星 → 印星**

| 相克 | 形象解读 |
| --- | --- |
| 印克食伤 | 热爱学习克住不切实际的空想与贪玩 → 减少口舌是非 |
| 食伤克官杀 | 太好空想、贪享乐 → 没有纪律 → 当不了官 |
| 官杀克比劫 | 当官后言行谨慎 → 不再乱交朋友 → 狐朋狗友远离 |
| 比劫克财 | 兄弟朋友多 → 容易被惦记 → 钱财被分夺 |
| 财克印 | 满脑子想赚钱 → 静不下心读书 → 压制自我提升 |

> 所有生克关系都是**意象层面的形象类比**，用来记忆和理解，并不严谨。具体吉凶仍要看整个命局的组合、强弱、喜忌。

## 阴阳双重作用的含义

两个十神之间的生克，不是简单 1 对 1：

- **同阴阳相生**：温和、纯粹、稳定（如食神生财、比肩生食神）。
- **异阴阳相生**：贴切、有情、流畅（如正印生日主、正财被劫财夺）。
- **同阴阳相克**：无情之克、锋利、破坏力大（如七杀克日主、偏印夺食）。
- **异阴阳相克**：有情之克、规范、约束力强（如正官克日主、正财被比肩分）。

**同一条相生相克链，阴阳不同，力度与味道差别很大**——这是十神区别于纯五行论的核心。

## 十神的记忆法

### 方法一：五路口诀
- **生我**→印，**克我**→官杀，**我生**→食伤，**我克**→财，**同我**→比劫。
- 阴阳同异再分正偏。

### 方法二：日元五行代入法
- 定日元五行（如甲木），按五行生克图逐一推出：水生我 = 印，金克我 = 官杀，火我生 = 食伤，土我克 = 财，木同我 = 比劫。
- 再按阴阳对位分正偏。

### 方法三：相生链口诀
- **印 → 比 → 食 → 财 → 官 → 印**（按相生闭环背诵一遍即可记住十神上下游）。

### 方法四：六亲反推（男命）
- 母 = 正印、父 = 偏财、妻 = 正财、子 = 七杀、女 = 正官、兄弟 = 比劫——用最熟悉的家庭角色辅助记忆十神。

## 传统吉神 vs 凶神的划分

古籍传统（《渊海子平》《三命通会》）把十神分为两类：

### 四吉神
- **正官**：端正有规、名利双全
- **正印**：庇护有情、学识清贵
- **正财**：勤俭持家、财源稳定
- **食神**：温和食禄、才艺享受

### 四凶神（或七凶）
- **七杀**（偏官）：冲克日主，不制则凶
- **偏印**（枭神）：夺食破印，孤寡刑克
- **伤官**：克官损夫、叛逆外泄
- **劫财**：分财夺妻、冲动破败

> 偏财在传统上**不完全算凶**——居中，喜用为财，忌用为风流破财。

## 为什么"吉神不一定吉、凶神不一定凶"

这是现代八字（也是本体系）对古籍吉凶二分法的重要修正：

1. **喜忌决定吉凶，而非名目**——为本命喜用神时吉，为忌仇神时凶。
2. **组合决定作用**——七杀有食神制、有正印化，就是大贵之象（杀印相生、食神制杀）；正官无印无财护，反而被伤官所破。
3. **强弱决定功过**——吉神太旺也成忌（正印过旺变母慈灭子、正财过旺克印）；凶神得制反成用（伤官配印、劫财帮身）。
4. **位置决定力度**——年月提纲、日支配偶宫、时柱子女宫，各宫位上的十神作用差别巨大。

**一句话**：**十神没有绝对的吉凶，只有喜忌、组合与强弱**。

## 十神在命盘中的整体作用机制

1. **主星**（天干十神）：直接、外显、作用力强，代表看得见的事。
2. **副星**（地支藏干十神）：内藏、基础、作用力持久，代表心性底色与潜伏的资源 / 隐患。
3. **大运十神**：十年一换，改变整个命局的十神平衡，是命运节奏的主开关。
4. **流年十神**：一年一换，触发具体事件，与命局、大运三方互动定吉凶。

十神的作用永远是**相互牵连**的：一个十神的吉凶从不孤立存在，必须看其**生者、克者、所生、所克**四个方向的配合。

## 常见误区

- **误区一**：一见七杀就凶、一见正官就吉——不论喜忌组合。
- **误区二**：数十神数量定强弱——忽略根气、司令、透藏。
- **误区三**：只背象意清单，不理解作用方向——学成"十神字典"，不会看命。
- **误区四**：把"正 / 偏"当好坏——正偏是气质差别，不是价值判断。
- **误区五**：忽视阴阳只论五行——失掉十神相对于纯五行论的全部精度。

## 古籍出处

- **《子平真诠·论十神》**（沈孝瞻）：奠定以日元为中心、按生克阴阳定十神的格局体系。
- **《渊海子平》**（徐大升）：最早系统记载十神吉凶划分与六亲对应。
- **《三命通会》**（万民英）：汇总前人十神象意、六亲、格局法则，是十神体系最完整的古典汇编。
- **《滴天髓》**（刘伯温传）：从气势、旺衰、通关的角度重新诠释十神，反对机械套用吉凶。

## 一句话总结

十神是日元与世界的**十种关系坐标**——**谁滋养我、谁压制我、谁被我控制、我如何输出、谁与我同行**。学十神不是背意象，而是理解每一种关系对日主的**作用方向、力度与吉凶切换条件**，再回到整个命局里看它是疏通还是阻塞、是助力还是拖累。
