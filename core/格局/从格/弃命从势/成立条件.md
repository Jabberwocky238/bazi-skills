# 弃命从势

> **命名说明**：
> - "**弃命从势**"为**现代合成名**，与 `弃命从财`、`弃命从煞` 保持命名一致。
> - 古籍原名为"**从势**"或"**从象·从势**"（《滴天髓》）。
> - 《子平真诠》只立"**弃命保财**""**弃命从煞**"两种弃命格——古籍本身**不立"弃命从势"**。
> - 本格名为便于分类与检索而用"弃命"前缀；概念依据为《滴天髓·从象》。

## 成立条件（严格依《滴天髓·从象》任铁樵注）

### 条件 1：日主无根（天干无比印同现，地支本气 / 中气 / 余气皆无比劫 / 印星）

- **古籍出处**：
  - 《滴天髓·从象》（任铁樵注）："**日主孤立无根**，四柱财、官、食伤势均力敌。"
- **客观判据**：
  - ① 日主天干无比劫 / 印星透出；
  - ② 日主在四地支的本气、中气、余气中皆无任何比劫 / 印星藏干。
- **不满足则破**：有一丝根 → 非从势，按身弱多病局处理。

### 条件 2：命局中财、官杀、食伤三者之中有两种以上强势并立

- **古籍出处**（本条最核心）：
  - 《滴天髓·从象》（任铁樵注）："**又有从势者——日主孤立无根，四柱财、官、食伤，势均力敌，不分强弱**，又无劫印生扶日主。"
- **客观判据**（三类组合，任一成立）：
  - ① 财 + 官杀 同时强旺（均有透干 + 通根）；
  - ② 财 + 食伤 同时强旺；
  - ③ 食伤 + 官杀 同时强旺；
  - 或三者全强。
- **不满足则破**：仅单一五行成势 → 转入单从格（弃命从财 / 弃命从煞 / 从儿）；无势可从 → 身弱破局。

### 条件 3：三势之间能相生通关

- **古籍出处**（本条核心）：
  - 《滴天髓·从象》（任铁樵注）："**惟视其财、官、食伤三者中何者最旺而从之**。"
- **客观判据**：
  - 食伤 → 财 → 官杀 的顺流链条至少部分贯通：食伤生财、财生官杀两链至少一链清晰可见。
- **不满足则破**：多势互相冲击、不相生通关 → 不成从势，属"杂乱下局"——即任氏所谓"不能从一神而去"的反面。

### 条件 4：原局无印星

- **古籍出处**：
  - 《滴天髓·从象》："**若有一毫印绶，即不为从**。"
- **客观判据**：
  - 天干无印星透出；地支本气、中气无印藏干。
- **不满足则破**：印存则日主有源，不从。

### 条件 5：原局无比劫

- **古籍出处**：
  - 《滴天髓·从象》："**比劫一见，即不为从**。"
- **客观判据**：
  - 天干无比劫透出；地支本气、中气无比劫藏干。
- **不满足则破**：比劫存则日主有帮扶，不从。

---

**五条同时满足**方为真弃命从势格。本格为从格中最难识别者——任铁樵特别注明"惟视何者最旺而从之"，意即多势并存时仍须辨主势；非机械"什么都顺"。满足此五条者极稀有，故此格向来被视为最高难判之外格。

## 喜忌总则

- **喜**：
  - **财运**：势的枢纽加强
  - **官杀运**：身份力量加持
  - **食伤运**：表达与变现渠道打开
- **忌**：
  - **印运**：破从 → 人生剧变
  - **比劫运**：抗势 → 全线失守
