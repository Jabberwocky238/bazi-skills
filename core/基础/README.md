## 阅读建议

1. 新手建议按顺序阅读：命盘构成 → 阴阳五行 → 身强弱 → 喜用神 → 干支作用 → 大运流年 → 格局 → 实战断事。
2. 喜用神体系中，优先掌握"生帮克泄耗"与"扶抑"，再辅以"调候 / 通关 / 救应"的特殊场景。
3. 实战断事建立在前述全部理论之上，不建议作为入门读物。
