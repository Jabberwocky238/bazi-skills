# 八字 / 传统命理解读 Skill

## 职责边界

- **只做解读，不做排盘**。
- 不根据阳历 / 阴历生日换算干支，不处理真太阳时、节气换月、早晚子时等排盘技术问题。
- 若用户只给出生日期时间、未给干支四柱，**先明确告知需由排盘工具生成四柱**，再使用本 Skill 解读。

## 输入要求

**八字类解读**：
1. 已排好的四柱八字（年柱、月柱、日柱、时柱的天干地支）
2. 性别（判断大运顺逆与部分十神取用；女命另见 `core/基础/实战断事/女命看法.md`）
3. 起运岁数与大运干支序列（若涉及大运 / 流年分析）

**非命盘类解读**（易经、哲学议题）：
- 用户具体困境或问题描述即可。

## 适用场景

- 八字命局解读：日主强弱、喜忌、用神、调候、通关、流通（见 `core/基础/`）
- 十神结构：比劫 / 食伤 / 财星 / 官杀 / 印星 及相互对照
- 格局判断：正格（月令取用，含建禄 / 阳刃）、从格（从财 / 从杀 / 从官 / 从儿 / 从势）、专旺格（曲直 / 炎上 / 稼穑 / 从革 / 润下）、两气成象（木火通明 / 水火既济 / 壬骑龙背等 22 对）、特殊格局（魁罡 / 化气 / 天元一气 / 帝王命造 / 母慈灭子等）合计约 60 格
- 神煞判读（含贵人煞、凶煞、感情煞）
- 宫位分析（年月日时四柱、夫妻宫）
- 六亲关系（父、母、子女、配偶、情人）
- 婚姻专题（女命官杀、男女命无财、择偶、父母干涉等）
- 大运 / 流年 / 流月引动、当下时运（2026 丙午年、九紫离火运等）
- 易经卦辞解读（乾坤两卦）
- 紫微斗数星曜（当前已整理星曜有限）
- 七政四余基础与对比八字
- 命理哲学：阴阳、天道、对立统一、因果、福报显化等底层框架

## 解读流程

### A. 八字命局类
1. **接收四柱** — 确认已提供年月日时干支与性别；缺失则先要求补全。
2. **基础判断** — 月令司权与日主强弱（`core/基础/身强弱/`）、喜忌与用神（`core/基础/喜用神/`），必要时参 `core/基础/命盘构成/` 厘清空亡、时辰、纳音等前提。
3. **十神标注** — 以日主为基准（`core/十神/`，含"纲要"与"vs 对照"文件）
4. **天干地支作用** — 统一在 `core/天干地支/` 下按作用关系索引子目录：`天干/` / `地支/` / `合/` / `冲/` / `刑/` / `克/` / `会/` / `墓库/`
5. **日主自坐与十二长生** — `core/星运自坐/`，判断日主气势与性格基调
6. **格局判定** — 优先检索 `core/格局/`（正格 / 从格 / 专旺格 / 两气成象 / 顶级特殊格与两神母子象格）。每个格局为一子目录，**先读 `成立条件.md` 判定是否入格、再读 `内容.md` 做深度分析**。
7. **神煞标注** — 仅采用 `core/神煞/` 已整理的神煞，贵人与凶煞兼看，避免堆砌
8. **宫位与六亲** — `core/宫位/` + `core/六亲/`
9. **婚姻专题**（若涉及）— `core/婚姻/`，结合 `core/宫位/夫妻宫.md`
10. **大运 / 流年引动**（若提供）— `core/基础/大运流年/`，含当下 2026 丙午年、九紫离火运、壬辰月等时运文件
11. **综合解读** — 贴合用户咨询方向，引用知识库原文口径

## 重要原则

- **以知识库为准**：解读论理必须与 `core/` 中 md 内容一致；不引入未整理的流派观点。
- **多维度叠加**：复杂问题常需跨目录综合（如"婚姻困局" = 婚姻 + 夫妻宫 + 大运流年 + 格局 + 六亲）。
- **格局优先于堆砌**：八字解读若能定格，远优于罗列十神神煞。
- **不贴标签**：如 `core/神煞/桃花.md` 强调"桃花 ≠ 异性缘"；`core/十神/财星vs印星.md` 等对照文件是破单标签解读的关键资源。
- **不做排盘技术判断**：早晚子时、节气换月、真太阳时等均交上游排盘工具。
- **遇知识库未覆盖内容**：明确告知"该部分知识库暂未整理"，避免臆断与幻觉。
- **中立表达**：不作绝对化断言，不宣扬迷信，健康 / 重大决策类话题明确提示仅供参考（见 `DISCLAIMER.md`）。
