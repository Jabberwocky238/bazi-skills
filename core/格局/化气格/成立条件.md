# 化气

## 成立条件（古籍 + 客观判据）

### 条件 1：日干参与合化（年月相合不论）

- **古籍出处**：
  - 《子平真诠·论化气》："化之真者，日元亲合用神。"
- **客观判据**：
  - 必须是**日干与月干或日干与时干**形成五合；**年月相合不构成化气格**。
  - 日元不参与，则化气之象不在太极点之上，格不成立。
- **不满足则破**：日元不参与合化 → 按普通五合异性相克论，不论化气。

### 条件 2：化神在月令当令或局中极旺

- **古籍出处**：
  - 《子平真诠·论化气》："**化之真者，月令得其气**。"
  - 《滴天髓·化象》："化神要旺，旺则贵；化神无气，虚化也。"
- **客观判据**（以合化所成之五行计）：
  - **最佳**：月令本气即为化神——甲己化土月令辰戌丑未、乙庚化金月令申酉、丙辛化水月令亥子、丁壬化木月令寅卯、戊癸化火月令巳午。
  - **次之**：月令非化神，但原局地支会局 / 三合 / 方局令化神极旺（如丁壬化木，地支见寅卯辰三会或亥卯未三合）。
  - **底线**：月令五行**绝对不能克制化神**（甲己化土忌月令寅卯；丁壬化木忌月令申酉；余类推）。
- **不满足则破**：化神失令又无地支会局助旺 → 虚化无力，化气不成。

### 条件 3：化神无外来克制（化神不受伤）

- **古籍出处**：
  - 《渊海子平·化气格》："化之既成，不可再见冲克之神，破则不化。"
  - 《滴天髓·化象》："化神被伤，反为不吉。"
- **客观判据**：
  - 原局天干、地支皆不得见明显克化神之五行。
  - 若出现克神，则需同时有他神将其化掉 / 合掉 / 反克掉（如丁壬化木局见土，若地支寅卯辰会木则土已被化神所克，不作破论）。
- **不满足则破**：克神紧贴化神且无解 → 化象夭折，落入"化气不成反为凶"的凶险区。

### 条件 4：无争合妒合

- **古籍出处**：
  - 《三命通会·论化气》："合而见争，化象不成。"
- **客观判据**：
  - 参与化气的两干必须"一对一"。
  - 典型破败结构：甲己化土局出现**二甲一己**或**二己一甲**——日主被争合 / 妒合 → 心思分散 → 化气失败。
  - 乙庚、丙辛、丁壬、戊癸同理。
- **不满足则破**：日主争合 → 不专心 → 化气不成立，退为普通格局。

### 条件 5：化干（日主之原本五行）无根

- **古籍出处**：
  - 《子平真诠·论化气》："化神要旺，化干要虚。"
- **客观判据**：
  - 日主之本气必须在地支**毫无根气**——不通本气、不通余气、不通库气。
  - 哪怕一丝微根，也将使日主心有所系而不肯让度，化气即败。
  - **例外**：**乙庚合化金**对此条件较宽——因乙木之气本弱，易被庚金所化；其余四化几乎都要求日主深弱无根。
- **不满足则破**：日干有根 → 不肯从化 → 化气失败，按普通格局论。

---

**五条须同时满足**方为真化气。五条之中，**日元参与**（条件 1）、**化神旺**（条件 2）、**无争合**（条件 4）、**化干无根**（条件 5）是化气格区别于其他格局的铁律；条件 3 为保底。
