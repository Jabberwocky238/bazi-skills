# 三奇贵人

## 定义
- **三奇**：**三颗奇异之星**连成一气的特殊组合。
- **三奇贵人**：命中**年、月、日 或 月、日、时 三柱天干**按固定顺序依次出现三奇之字 → 为极贵之象。
- 属**最高级的贵人神煞**之一，古籍赞为"**国之重器**"。

## 三奇的三种组合
《三命通会·论三奇贵人》原文："**天上三奇甲戊庚，地下三奇乙丙丁，人中三奇壬癸辛**。"

| 三奇类型 | 组合 | 象义 |
|---|---|---|
| **天上三奇** | 甲、戊、庚 | 贵居庙堂、功名显赫 |
| **地下三奇** | 乙、丙、丁 | 聪明才智、文采出众 |
| **人中三奇** | 壬、癸、辛 | 智勇双全、谋略超群 |

## 成立条件

### 1. 须在年月日三柱或月日时三柱天干顺序出现
- **年、月、日**三天干依次为三奇 → 典型三奇；或
- **月、日、时**三天干依次为三奇 → 亦可论三奇。
- **顺序可以"顺"或"逆"**（如甲戊庚或庚戊甲），只要三字齐现且相邻。

### 2. 三字须不被冲克
- 三奇所在天干若被合化为异神 → 奇气散。
- 三奇所在地支若被冲破 → 根基动摇。

### 3. 最好搭配其他贵神
- **三奇 + 天乙贵人**：加倍贵气。
- **三奇 + 财官印齐备**：极贵之命。

### 4. 日主须不过弱
- 三奇气大，日主太弱不堪承载——沈氏"**贵气须身能受**"。

## 核心作用：极贵的底牌
- 三奇贵人是神煞中**最高级的吉神之一**：
  - **天赋异禀**：智慧、能力、气场全面高于常人。
  - **贵气先天**：不必后天努力太多，也会被人看重。
  - **机会自来**：关键时刻会有贵人、平台、机遇推到面前。
  - **三奇合德** = 古人眼中**能成就大事**之象。

## 三种三奇的区别

### 1. 天上三奇（甲戊庚）
- **气象**：刚、正、威。
- **典型路径**：仕途、体制、功名——**走正路成大器**。
- **性格**：端方、威严、有领袖气。
- **现代对应**：政界、军警、大型机构的掌舵者。

### 2. 地下三奇（乙丙丁）
- **气象**：柔、明、灵。
- **典型路径**：文化、艺术、学术——**以才华立身**。
- **性格**：聪明、秀丽、有文采。
- **现代对应**：学者、作家、艺术家、文化大家。

### 3. 人中三奇（壬癸辛）
- **气象**：智、勇、谋。
- **典型路径**：智慧型职业——**以谋略制胜**。
- **性格**：深沉、机敏、有胆识。
- **现代对应**：战略家、企业家、顾问、外交官。

## 判断维度

### 1. 位置
- **年月日三奇**：从少年起就贵气外显 → 早年发达。
- **月日时三奇**：中后段发力 → 中年后大放异彩。
- **三奇跨越四柱（如年 + 日 + 时）**：中间隔开，格品降，不算真三奇。

### 2. 顺逆
- **顺排**（如甲 → 戊 → 庚）：顺贵之象。
- **逆排**（如庚 → 戊 → 甲）：逆贵之象。
- 古籍认为顺逆各有其贵，关键在"**三字齐全、位置相连**"。

### 3. 三奇是否真贵
- 三奇字**都有根通地**：真贵。
- 三奇字中有一字**虚透无根**：贵气打折。
- 三奇字**被冲合破坏**：只剩形式，不成真贵。

## 常见误区

### 1. "八字里有甲、戊、庚 = 天上三奇"
- 必须**在相邻三柱天干上按顺序出现**，非任意散见。
- 例：年甲、日戊、时庚——中间隔月柱 → **不构成真三奇**。

### 2. 把三奇等同"一定大贵"
- 三奇是**底牌极好**，但仍须：
  - 日主能承载。
  - 大运流年配合。
  - 格局清而不杂。
- 光有三奇 + 身弱 + 大运不配 → 空有贵气而不显。

### 3. 忽视身弱承载的问题
- 三奇气势大，**身弱者反被压**。
- 典型："少年才气出众但中年折戟"——往往是身弱三奇的表现。

## 现实表现

### 性格
- **天资聪颖**：领悟力 / 学习力超群。
- **气场与众不同**：从小被人说"**这孩子不一般**"。
- **对平庸不耐受**：身上有股"非做大事不可"的劲。
- **关键时刻能出众**：人才济济的场合也能被看出来。

### 事业路径
- 因三奇类型而异：
  - 天上三奇 → 体制 / 管理层。
  - 地下三奇 → 文化 / 学术。
  - 人中三奇 → 智慧 / 谋略型。
- 共性：**能站到行业前列**。

### 人生节奏
- **少年得志**（年月日三奇）或**中年发力**（月日时三奇）。
- **成就的天花板**远高于一般命造。
- **但也更依赖大运配合** → 大运不配时闲置十年二十年都可能。

## 核心判断
- 三奇是"**天赋的顶点标记**"——极少人命中带。
- 识别关键：
  - 相邻三柱天干按序出现。
  - 三字皆有根、不被破坏。
- 识别意义：**这类人不应放在平庸岗位**——天赋浪费是最大损失。

## 一句话总结
- 三奇贵人 = **命中自带的高起点**。
- 三种三奇各有风格：**天上为贵、地下为才、人中为谋**。
- 但不同贵人：带三奇的人**被命运要求走到不平凡的位置** —— 躲不掉。
