# 自坐墓

## 定义
- **墓**：十二长生的第九阶段，象征**入墓归藏、收敛沉寂**。
- **自坐墓**：**日主天干坐在墓库位地支上** → 日主能量**归藏入库**、不外显。
- 各天干墓位：
  - **甲乙木** 墓在 **未**。
  - **丙丁火、戊己土** 墓在 **戌**。
  - **庚辛金** 墓在 **丑**。
  - **壬癸水** 墓在 **辰**。

## 典型日柱
- **丙戌 / 戊戌**（火土墓）。
- **壬辰 / 癸辰**（水墓）。
- **乙未**（木墓）。
- **辛丑**（金墓）。
- 另：**甲辰**（甲日坐辰 = 水库，木在水中偏藏）、**戊辰**等也有墓意。

## 核心：内收藏锋、深不可测
- 墓 = **把气藏起来** → 不是死，是**收**。
- 自坐墓的人：
  - **能量向内聚拢** → 不外显。
  - **表面安静 / 低调** → 实则内有积累。
  - **关键时刻一开库** → 出手惊人。
- 比喻：**宝藏埋在土里**，平时看不见，用时惊艳。

## 与相关结构对比

| 结构 | 关键 | 特征 |
|---|---|---|
| **自坐墓**（本条） | 日主坐墓库 | 深藏不露、后发制人 |
| **自坐死** | 日主坐死位 | 能量最弱，需外援 |
| **自坐绝** | 日主坐绝位 | 无根，靠缘分 |
| **自坐长生** | 日主坐长生 | 生机初起 |
| **自坐帝旺** | 日主坐刃位 | 气势到顶 |

**自坐墓 与 自坐死 / 绝的区别**：墓是"**有气但藏**"，死绝是"**几乎无气**"。

## 判断维度

### 1. 墓库是否被"冲开"
- **辰戌冲 / 丑未冲**：日墓被冲 → 库开 → **能量被触发**：
  - 喜用被冲开 → 大好事年。
  - 忌神被冲开 → 大坏事年。
- **六合 / 三合**：视合化是否有利。
- **无冲不合**：墓安静 → 能量长期归藏。

### 2. 大运流年冲墓
- 自坐墓的人**一生最重要的节点常在"冲墓年"**：
  - 壬辰日 → **遇戌年 / 戌运** → 辰戌冲。
  - 丙戌日 → **遇辰年 / 辰运** → 戌辰冲。
  - 辛丑日 → **遇未年 / 未运** → 丑未冲。
  - 乙未日 → **遇丑年 / 丑运** → 未丑冲。
- 冲开时：
  - **身体 / 事业 / 感情** 容易出现重大转折。
  - 善用 = 贵；用不当 = 伤。

### 3. 身的整体强弱
- **身强 + 自坐墓**：藏锋不露的强者 → 关键一出手。
- **身弱 + 自坐墓**：本就弱 + 再入库 → 能量极低 → 极需生扶。

### 4. 墓中藏干
- 辰戌丑未**都是杂气土**，藏干多样：
  - 辰藏：戊、乙、癸（水库）。
  - 戌藏：戊、辛、丁（火库）。
  - 丑藏：己、癸、辛（金库）。
  - 未藏：己、乙、丁（木库）。
- 日主坐墓的实际意义取决于**藏干与日主的关系**：
  - 藏干有印生身 → 墓中有贵。
  - 藏干皆忌 → 墓变牢。

## 常见误区

### 1. "自坐墓 = 命短 / 入棺材"
- **错**。墓是"藏"，不是"死"。
- 许多自坐墓的人**健康长寿、成就深远**。
- "入墓"是能量形态，不是生死判断。

### 2. 把墓当"低迷"
- 墓确实让能量"收"，但这也是**积累期**。
- 自坐墓的人**前期看似低迷，后期积累深厚**。
- 是"**厚积薄发**"的典型底色。

### 3. 以为冲墓一定好
- "**财入墓库需冲开**" —— 指财在墓时要冲开才能用。
- 但日主本身入墓 → 冲开未必全好：
  - 喜用被冲开 → 好事。
  - 忌神被冲开 → 祸事。
- 必须看具体结构。

### 4. 忽视墓的心理影响
- 自坐墓的人**内心常有"藏"的倾向**：
  - 不爱表达。
  - 心里有话说不出。
  - 容易积累未表达的情绪。
- 需要**主动寻找表达 / 倾诉的出口**。

## 现实表现

### 性格
- **深沉、内敛、有分量**。
- **话不多但句句有料**。
- **喜欢独处 / 沉思**。
- **表面低调，内有积累**。
- **关键时刻能担大事**。
- **不善主动示好**：但**认定的人会默默守护**。

### 事业
- **适合"厚积薄发"型岗位**：
  - 学术、研究、深度咨询。
  - 档案、记录、收藏、考古。
  - 金融（尤其是存储、保险、信托）。
  - 中医、心理、精神工作。
  - 文史、博物馆、图书馆。
- **不适合"即时反馈"**的岗位（如短期销售 / 冲量工作）。

### 成长节奏
- **前期积累慢 / 不出众**。
- **中期偶有爆发**（遇冲墓大运）。
- **后期价值显现**：**越老越值钱**。

### 感情
- **不易轻易表达情感**。
- **深情但内藏**。
- **对"被理解"有极深的渴望**。
- **伴侣需要耐心走进他 / 她的内心**。
- **相处久才能看到真正的一面**。

### 身体
- 自坐墓对应**身体的"储藏系统"**：
  - 消化（墓库为土 → 脾胃）。
  - 存储（肾气、精气）。
- 整体身体耐用，但**冲墓的年份易生大病 / 大事**。
- **长寿比例高**（与自坐死绝相反）。

### 心理
- 容易积累**不表达的情绪** → 成年后可能表现为：
  - 慢性抑郁倾向。
  - 突然的情绪爆发。
  - 对关系的长期默默耿耿。
- 需要**主动建立表达通道**（写作、艺术、倾诉、心理咨询）。

## 自坐墓 + 十神的叠加

| 日柱 | 十神 + 墓组合 | 含义 |
|---|---|---|
| **丙戌** | 丙坐食神库（戌中戊、辛、丁）+ 墓 | 才华藏而不发、**后发型创作者** |
| **戊戌** | 戊坐比劫 + 墓 | 身库收藏 → **深藏不露的实业家** |
| **壬辰** | 壬坐杀（辰中戊）+ 墓 + 魁罡 | **深藏的将帅**（壬骑龙背） |
| **癸辰** | 癸坐官（辰中戊）+ 墓 | 智慧藏锋 → **后起的智者** |
| **乙未** | 乙坐财库（未中己）+ 墓 | 财归库 → **守财型、慢富型** |
| **辛丑** | 辛坐印库（丑中己、癸、辛）+ 墓 | 学识藏深 → **沉静的学者型** |

## 大运流年影响

| 大运 | 效果 |
|---|---|
| **冲墓的大运**（辰冲戌 / 丑冲未） | **关键节点** → 视喜忌定吉凶 |
| 比劫 / 印运 | **喜** → 助身 → 墓中有贵 |
| 食伤运 | 视结构 → 墓气泄秀 → 才华外显 |
| 财官运 | 看入墓情况 → 财官入墓需冲开 |
| 与墓支同类之运 | 加强墓势 → 更深、更静 |

## 自坐墓的正确用法

### 1. 接受"藏"是本性
- 不强求自己外显、活跃、社交。
- 允许自己"**慢慢来、深深做**"。

### 2. 建立表达通道
- 写作、艺术、私人日记。
- 找一两个可以说心里话的人。
- 避免长期压抑。

### 3. 选对"冲墓"的时机
- 知道哪些年份会"冲开你"：
  - 壬辰日 → 戌年。
  - 丙戌日 → 辰年。
  - 辛丑日 → 未年。
  - 乙未日 → 丑年。
- 在这些年里**大事可为**（前提是原局喜用对应冲开）。

### 4. 懂得"墓"的修行价值
- 自坐墓的人**天然适合"深度修行 / 内观 / 禅修"**。
- 很多成就深远的高人都自坐墓。

## 核心判断
- 自坐墓不是"命苦"，是"**深**"的底色。
- 关键识别：
  - 墓中藏什么（喜用还是忌神）。
  - 大运何时冲开。
- 成立时：**厚积薄发的大器晚成**。
- 不成立时：**一生压抑、积郁成疾**。

## 一句话总结
- 自坐墓 = **你的气藏在土里了**。
- 不是死，是**藏**。
- 深藏的东西最终都会有冲开的一天 —— **愿意等、敢于深、懂得藏，是这类命造最大的智慧**。

## 古典标准：墓的阶段特征与类象

### 墓的本质

**墓就跟人死后埋进坟墓一样，也可理解为是万物成熟或生产完成后归入仓库**——也**象征着旧事物死亡后的遗迹被送进博物馆，或者被收到仓库里，所以也叫"库"**。

### 墓的记忆口诀（以日主为基准）

- **甲日见未、乙日见戌、丙日见戌、丁日见丑**
- **戊日见戌、己日见丑、庚日见丑、辛日见辰**
- **壬日见辰、癸日见未**

### 墓在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 出生在经济比较落后的家庭，小时候比较穷 |
| **月柱** | 人比较节俭、会存钱，但收入不多；**中年以后会有转机** |
| **日柱** | 总是过得很辛苦、一生变化比较大；生活吝啬节俭；**中年以后可以大展宏图，有很强的赚钱能力** |
| **时柱** | 孩子可能比较少，或者让人操心烦恼；晚年能有成果，可以享受辛苦的成果 |

### 自坐墓的典型日柱

**丙戌日、丁丑日、戊戌日、己丑日、壬辰日、癸未日**——日柱为墓的六组组合。

### 自坐墓的命主特征

1. 不喜欢打扮、外表性格内向；做事有条理、为人有操守；但**心胸有点狭窄、听不进别人的意见**，会疏远亲友，夫妻之间平淡相处。
2. 大多**出生在贫穷家庭**，先天六亲缘分淡薄；**小时候就离开家吃苦**；壮年之前家里经常搬家，必须承担照顾家人的责任。
3. 是**心里多忧愁烦恼的命格**，但**有很好的事业和财运**——最适合做**实业或在企业工作**，有相当强的赚钱能力。一般**在中年期间得到发展**，但在没走好运之前盛衰起伏很大。
4. **婚姻障碍比较多**；婚姻看起来不错其实经常有争吵；**女命**做事很有条理、人也很简朴，就是**对老公疑心比较重**。

### 古典标准铁律

> 十二长生在八字命理中**只起到辅助作用**，不能作为主要依据；以上类象**准确率并不是很高，只能作为参考**——"墓"即"库"，是**能量归藏/收纳**的阶段。

## 现代解读

以下内容为**现代综合性延伸**：
- "厚积薄发、大器晚成"等标签属现代通俗化表述。
- "墓库冲开"在经典十二长生章节中并未直接展开论述——相关冲库开库原理应参见**地支墓库专篇（节54-59）**，此处的"冲墓年份 = 事件年"为现代实务归纳。
- "适合修行、内观、禅修"等精神性描述属现代延伸解读。
