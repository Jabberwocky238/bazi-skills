# 阳刃格（羊刃格）

## 定义

- **阳刃**（亦称"羊刃"）：阳干日主在十二长生中"**帝旺**"之地对应的地支。
  - 甲木刃在卯、丙戊刃在午、庚金刃在酉、壬水刃在子。
  - 阴干古籍多不单立阳刃格（虽有"阴刃"之说——乙辰、丁己未、辛戌、癸丑，但作用显著弱于阳刃，实操中不作为基础正格）。
- **阳刃格**：**月令为日主之刃位**的基础正格。
- 《子平真诠》十正格之一——与建禄格对照：
  - 建禄：月令为禄（临官之地，身自旺）。
  - 阳刃：月令为刃（帝旺之地，身过旺）。
- 《子平真诠·论阳刃》原文："**甲禄在寅，刃在卯；丙戊禄在巳，刃在午；庚禄在申，刃在酉；壬禄在亥，刃在子**。"

## 核心：刚烈过旺、极凶之力

- 阳刃 = 阳干之气"**旺极转厉**"之象。
- **禄**是站稳，**刃**是过盛——过盛则刚烈，刚烈则需制。
- 古籍论："**阳刃如刀，须有鞘**"——必得官杀相制方为贵格。
- **专克正财**——是正财的七杀；居于月令、其破坏性极大。
- **无论日元强弱、必须要治之**：
  - **身强 + 羊刃**：高大强壮者握大刀——无约束则完全显劫财之凶、甚至引发是非或意外伤灾。
  - **身弱 + 羊刃**：瘦弱孩童持刃——自身难以驾驭、反而因无力掌控而招致麻烦。

## 与其他相关格局的对比

| 对比项 | 阳刃格 | 建禄格 |
|---|---|---|
| 月令性质 | 帝旺（过旺）| 临官（正旺）|
| 日主状态 | 身极旺、刚烈 | 身旺、稳重 |
| 取用必要 | **必需官杀制** | 官财食伤均可取用 |
| 气质 | 刚猛、易冲动 | 稳重、不动声色 |

| 格局 / 组合 | 核心定位 |
|---|---|
| **建禄格** | 月令为禄——身自旺、需找出口 |
| **阳刃格** | 月令为刃——身过旺、必须官杀制 |
| **羊刃驾杀** | 刃 + 七杀 + 身强 → 武贵、将帅之命（见 [`../羊刃驾杀.md`](../羊刃驾杀.md)） |
| **羊刃劫财** | 羊刃的近亲——更强调分财夺财（见 [`../羊刃劫财.md`](../羊刃劫财.md)） |
| **神煞羊刃** | 从五行能量"帝旺极点"角度看（见 [`../../神煞/羊刃.md`](../../神煞/羊刃.md)） |

## 成立条件（严格依《子平真诠·论阳刃》）

### 条件 1：日主必为阳干

- **古籍出处**：《子平真诠·论阳刃》："**阳刃者，阳干之刃也**。"
- **客观判据**：日主天干必须为甲、丙、戊、庚、壬五阳干之一。
- **不满足则破**：阴干日主不入本格。

### 条件 2：月令为日主之刃位

- **古籍出处**：《子平真诠·论阳刃》："**月令逢刃，谓之阳刃格**。"
- **客观判据**：月令地支为条件 1 对应的刃位（甲卯、丙戊午、庚酉、壬子）。
- **不满足则破**：月令非刃 → 不入本格。

### 条件 3：必有官杀制刃（阳刃格铁律）

- **古籍出处**：
  - 《子平真诠·论阳刃》："**阳刃用官则喜露，用杀则不忌——刃以杀制为贵**。"
  - 《子平真诠·论阳刃》："**阳刃无官杀，凶顽之辈**。"
- **客观判据**：
  - ① 正官透干通根；或
  - ② 七杀透干通根；
  - 有其一制刃即可。
- **不满足则破**：阳刃无官杀 → "**凶顽之辈**"——身极旺而无制、气无出口、易走极端。

### 条件 4：制刃之官杀有根有力

- **古籍出处**：《子平真诠·论阳刃》："**刃杀相济，声威赫奕**——刃强杀强则贵。"
- **客观判据**：制刃之官或杀须在地支本气或中气有根、力量与阳刃大致对等或略强。
- **不满足则破**：官杀虚透无根 → 制不住刃 → 阳刃格变"阳刃独旺"的破局。

### 条件 5：原局无伤官克官（若用官制刃时）

- **古籍出处**：《子平真诠·论阳刃》："**阳刃用官，忌伤官**——伤官克官、制刃之物反被克。"
- **客观判据**：若取正官制刃，原局无伤官紧贴克官；若有伤官须有印制之。
- **不满足则破**：伤官克官 → 制刃链断 → 转入"阳刃无制"或"伤官见官"讨论。

### 条件 6（格局升级）：身杀两停 → 升为"阳刃驾杀"

- **古籍出处**：《子平真诠·论阳刃》："**羊刃无杀，不足为贵；七杀无刃，不足为威**。"
- **客观判据**：若用杀制刃且刃杀力量势均 → 转入"**阳刃驾杀**"贵格讨论。

---

**前五条必须同时满足**方为真阳刃格；条件 6 为升级路径。本质是"**身极旺必须用官杀制之**"——无制之阳刃判为"凶顽"。

## 判断维度

### 1. 制刃之物的选择
- **用正官制**：气质偏文——管理、体制、规范型领导。
- **用七杀制**：气质偏武——军警、强硬型企业家。
- **刃杀两停**：**阳刃驾杀**——最高境界。

### 2. 官杀有无透出及根器深浅
- **官杀显露且根基深厚** → 富贵程度高。
- **官杀隐藏不显、或虽显露但根基浅薄** → 富贵程度低。

### 3. 阳刃的其他位置
- **月刃**：标准阳刃格。
- **日支阳刃（坐刃）**：自坐刚烈——脾气大、婚姻多波折。
- **时支阳刃**：晚年或子女刚烈。

## 喜忌总则

### 阳刃格六大取用规则

#### 规则一：官杀制刃——喜官杀有根而透出有力
- 官杀须**有根器且透出有力**方能发挥制化羊刃之作用。
- 官杀显露根深 → 富贵程度高。
- **注意**：羊刃格本命配置高且运势配合 → 贵格；但**喜用神不一定就是官杀**——具体喜用须看全局身强身弱判定。

#### 规则二：官杀跟刃偏弱——应以财印助之
- 若官杀偏弱 → 借助**财星生助**（财能生官杀）。
- **印星**的运用须结合具体：官杀过强时印可化官杀之威、护卫日主。
- 目标是达到**官杀强旺有力制刃**的格局——不管是官杀本就通根有力、财旺生杀有力、或两者并存皆可。

#### 规则三：官杀制刃而见食伤者——必须有印护卫
- 若同时出现官杀与食伤且势均力敌 → 亦形成"**伤官见官**"之局。
- 此时须**印星压制食伤、调和矛盾**、起护卫作用。
- **偏印**在特定格局中也能发挥重要作用——不可因其"偏"而轻视。

#### 规则四：官杀太重——也可取食伤之化
- 官杀过重时可用食伤制化，但**效果稍逊**。
- **日元身弱时**：阳日干命局中采用食神制杀**负面影响较大**；**阴日干**负面影响会小很多。

#### 规则五：官杀同透混杂者——喜去掉其一、取清则贵
- 官杀同透 → **取清**（去其一使格局纯粹）。
- 纯粹则力量集中、更易成事。
- 羊刃格命局**最喜保留七杀**。

#### 规则六：羊刃驾杀者多贵
- **羊刃驾杀**：命局同时出现羊刃与七杀这两股强大力量、且日主不能太弱（最好身强）——能以羊刃刚猛驾驭七杀之凶险。
- **象意**：
  - 七杀 = **来犯的强大敌军**、重大困难、小人。
  - 羊刃 = **麾下最精锐的王牌部队**、自身的勇气魄力。
  - **驾** = 指挥王牌部队迎战并击败强敌。
- 七杀不能虚浮无力——必须是实实在在的强敌——对手越强、战胜后成就越高。

### 总原则
- 喜神：能制刃之官杀、生官杀之财（若官杀弱）、护官杀之印（若伤官克官）。
- 忌神：伤官（克官）、比劫 / 印（助刃身更旺）、食伤（制官杀破格，除非官杀过重）。

## 现实表现

### 性格
- **意志坚强、竞争意识特别强、敢于直面压力**。
- **气势强、决策果决、执行力超群**。
- **刚烈、果敢、有主见、不怕事、遇强更强**。
- **身体健、精力旺、行动力强、有领袖气**。
- **脾气直、不善掩饰**。
- **易自负固执**——特别是无官杀制化时。

### 典型职业路径（需配官杀）
- **军警、武职**：刃 + 杀的典型路径、刀气与决断感强。
- **司法、执法、监察**：体制内的刀剑结构。
- **竞技体育、高压决策岗**。
- **企业家、强势管理、强势谈判**：身旺能担事业。
- **外科医生、急救**：七杀 = 病痛、羊刃 = 刀 → 用刀解决病痛。
- 事业方面非常适合**在竞争激烈、需要决断力和爆发力的领域**——巨大压力下脱颖而出、建功立业。

### 无制阳刃的典型
- 脾气暴躁、不懂约束。
- 易与人冲突、官非是非多。
- 冲动决策、冒险失控。
- 现代版："**愤青型**"或"**易怒型**"人格。

## 破格与救格

### 破格情形
1. **羊刃无官杀制** → 凶顽之辈——气无出口、易走极端。
2. **官杀虚浮无根** → 制不住刃、羊刃格变"羊刃独旺"的破局。
3. **用官制刃时见伤官克官且无印救** → 制刃链断。
4. **官杀混杂且无清法** → 格局不纯。

### 救格
- 伤官见官 → 得印化伤护官。
- 官杀太过 → 得食伤制化（阴日干优先）。
- 官杀弱 → 得财生、得比肩扶杀星（间接）。
- 官杀混杂 → 得合去其一取清。

## 大运流年影响

- **官杀运**：制刃加强 → 升职、权柄到手。
- **食伤运**：制住官杀 → 破格（除非有印救）；若原局官杀过重则反为救。
- **财运**：身旺能任财 → 财运亨通；亦助官杀制刃（财生官杀）。
- **比劫 / 印运**：身更旺 → 需要更强制刃、反而易失衡。
- **刃运（本命刃字再临）**：刃气加倍 → 需格外警觉冲动事件。
- **对抗性流年**（如身旺阳刃格走羊刃年再配冲激）：易现官非、冲突、伤病。

## 常见误区

### 1. "带阳刃 = 一定凶"
- 阳刃只有"**无制 + 身过旺**"时才真凶。
- 有官杀制 + 身能任 → 阳刃反而**贵气过人**（将帅 / 强人）。

### 2. 把阳刃等同"羊刃煞"
- 阳刃格（月令刃位 → 正格）与羊刃神煞有重叠但不完全相同。
- 格局层面强调"月令关系"、神煞层面强调"刃气标记"——实操中两者常并论。

### 3. 忽视阳刃格的"必须要制"
- 阳刃格与其他正格最大区别：**必需官杀制刃**、否则纯破格。
- 不像其他格局有多条出路，阳刃格的出路**只有一条**：有官杀 + 官杀能扛住刃。

### 4. 阳刃格的喜用神必然是官杀？
- 不对。官杀是"**制刃成格**"的必要条件，但**命主喜用神须看全局身强身弱判定**。
- 例如本命配高但身偏弱者 → 喜用神仍为印比；这并不违背"官杀制刃成格"的条件。

## 一句话总结

- 阳刃 = **身极旺的双刃剑**——有官杀这把鞘则是**利器 / 将军 / 英雄**；**无鞘则伤己伤人**。
- 阳刃格必须给自己找"**对手**"，否则对手就是自己。
