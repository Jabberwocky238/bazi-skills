# 寒木向阳

## 成立条件
1. **日主为木**：甲或乙。
2. **月令为冬季**：亥、子、丑 为主（寅月带寒可算）。
3. **命局有火**：
   - **丙火透干**最妙（太阳直射）。
   - **丁火透**次之（烛火，小范围暖）。
   - **地支巳午**为根。
4. **火不过烈**：
   - 火过旺 → 寒水被蒸干 → 木反无根。
   - 理想：**火能暖、不能焚**。
5. **水不过多**：
   - 水少适度润木 → 好事。
   - 水过多 → 再暖也调不过来 → 变"水多木漂"。
6. **有土更稳**：
   - 戊己土（尤其戊土）承载木根 → 根基更稳。
