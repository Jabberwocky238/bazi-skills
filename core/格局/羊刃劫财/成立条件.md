# 羊刃劫财判断维度

### 1. 羊刃与劫财的位置
- **都在月、日支**：直接贴身 → 作用最强。
- **分散在年、时柱**：作用打折，但仍有影响。
- **劫财透干 + 羊刃在月支**：最典型的羊刃劫财，格扰动大。

### 2. 是否有化泄之物
回到身旺命造的三条出路：

| 出路 | 结构 | 去向 |
|---|---|---|
| **化泄（食伤泄秀）** | 身旺 → 食伤 | 多余的能量变成才华、作品 |
| **克制（羊刃驾杀）** | 身旺 → 官杀 | 多余的能量变成权力、纪律 |
| **夺财（群比夺财）** | 身旺 + 财 + 无食伤通关 + 无官护财 | 大忌 → 辛苦为他人作嫁衣 |

羊刃劫财格**能否成立有用**，就看**日主过剩的能量有没有被正确引导**。

### 3. 身旺的程度
- 身旺到"刃"已经是极致。
- 再叠加劫财 → 若没有**食伤 / 官杀 / 财（三者之一有合理配合）** → 则变成"**野马无缰**"。

### 4. 财星位置
- **财近日主 + 无官护财 + 无食伤通关**：**群比夺财** → 最大忌。
- **财远 + 有食伤生财 / 有官杀克劫**：劫财之扰被化解 → 财能守住。
