# 自坐病

## 定义
- **病**：十二长生的第七阶段，象征**老年疾病缠身** → 能量**虚弱、需养**的状态。
- **自坐病**：**日主天干坐在病位地支上** → 日主能量虚、容易疲惫、需要外援。
- 各天干病位：
  - **甲木** 病在 **巳**。
  - **乙木** 病在 **子**。
  - **丙戊** 病在 **申**。
  - **丁己** 病在 **卯**。
  - **庚金** 病在 **亥**。
  - **辛金** 病在 **午**。
  - **壬水** 病在 **寅**。
  - **癸水** 病在 **酉**。

## 典型日柱
- **甲巳（极少）、乙子、丙申、戊申、丁卯、己卯、庚亥、辛午、壬寅、癸酉**。

## 核心：虚弱、多思、易感病
- 病 = **能量被疾病侵蚀** → 不能全力运转。
- 自坐病的人：
  - **身体 / 精神都容易疲倦**。
  - **思虑多、易焦虑**。
  - **对外界敏感**：小事在心里反复放大。
  - 带**"病弱型的细腻**"气质：柔软但不堪重负。

## 与相关弱位对比

| 阶段 | 能量 | 气质 |
|---|---|---|
| **衰** | 刚回落 | 老成守势 |
| **病**（本条） | 虚弱 | 多思内耗、易焦虑 |
| **死** | 最低点 | 归零、需借力 |
| **墓** | 归藏 | 深藏不露 |
| **绝** | 断灭 | 无根无援 |

**病与死的区别**：病是**还活着但虚**，死是**归零**。

## 判断维度

### 1. 病位的五行与十神
- 病位的五行**与日主相反方向**：
  - 甲病在巳（巳火泄甲木）→ 自身气被泄。
  - 丙病在申（申金克丙火 + 水源地）→ 自身气被克 + 生财。
  - 庚病在亥（亥水泄庚金）→ 泄气。
  - 壬病在寅（寅木泄壬水）→ 泄气。
- 病位基本都是"**泄 / 克**"日主的位置。

### 2. 原局有无救
- **有印生身**：病中得救 → 仍能稳住。
- **有比劫帮身**：病中有同伴 → 扛得住。
- **无印无比**：纯弱 → 很依赖大运。

### 3. 大运流年
- **印 / 比劫运**：救病 → 明显好转。
- **食伤 / 财官运**：加耗 → 更虚。
- **冲病位的大运**：视冲开的是喜是忌。

## 常见误区

### 1. "自坐病 = 一定体弱"
- 倾向体弱，但**不等于必然**。
- 许多自坐病的人通过**养生 + 生活管理**保持健康。
- 结构性风险，不是不可避免的命运。

### 2. 以为"病" = 活不长
- 病是能量阶段，不是"死"。
- 很多自坐病的人**长寿** → 因为**早早开始养生**。
- 反倒是身强的人容易"**挥霍身体**"，寿命未必长。

### 3. 把"多思"当缺点
- 自坐病的人思虑多是本性。
- 用对地方 → 成为**哲学家、心理学家、研究者**。
- 用错地方 → 焦虑症 / 抑郁倾向。

### 4. 强行模仿身强者
- 自坐病的人**最忌模仿身强者的节奏**：
  - 通宵赶工、拼命竞争、高压工作。
- **必须走"慢而深"**的路。

## 现实表现

### 性格
- **细腻、敏感、有同理心**。
- **思虑多于行动**。
- **易焦虑、易失眠**。
- **对情绪变化敏感**。
- **不爱热闹 / 偏内向**。
- **对他人的痛苦能感同身受**。

### 事业
- **适合"**深思 + 细腻**"的岗位**：
  - 心理、医疗、咨询、疗愈。
  - 学术研究、深度写作。
  - 艺术、文学、哲学。
- **不适合"**高压 + 快节奏**"的岗位**：
  - 销售前线、创业初期、竞技型岗位。
- **走"专业深度"路线**而非"规模速度"路线。

### 成长节奏
- **前期缓慢 / 多反思**：青年期看似不如人。
- **中年找到方向**：一旦走对就稳步发展。
- **长期耐力**：**越老越值钱**。

### 感情
- **不易主动**：喜欢等对方先表达。
- **深情内敛**。
- **对"被理解"的渴望极强**。
- **伴侣需要有足够的温度与耐心**。
- **关系中容易过度解读对方信号** → 需要自我觉察。

### 身体（重点）
- **先天底子偏虚**：
  - 易疲劳、易感冒。
  - 免疫系统敏感。
  - 消化 / 神经 / 内分泌**其中一个系统偏弱**。
- **情绪直接影响身体**：心病连环躯体症状。
- **长寿的前提是**"**终身养生**" → 不是随意选项。

### 心理
- **多思多虑 → 易焦虑抑郁倾向**。
- **存在感较弱**：常感觉"自己不被看见"。
- **需要建立强大的内在意义系统**。

## 自坐病 + 十神的叠加

| 日柱 | 组合 | 含义 |
|---|---|---|
| **乙子** | 乙坐印（子中癸）+ 病 | 有印生身 → **病中有救** |
| **丙申** | 丙坐财（申中庚、壬）+ 病 | 财在病位 → 求财辛苦 |
| **戊申** | 戊坐食神（申中庚）+ 病 | 才华虚耗型 |
| **丁卯** | 丁坐印（卯中乙）+ 病 | 学识型但偏消耗 |
| **己卯** | 己坐官（卯中乙）+ 病 | 事业艰辛型 |
| **庚亥** | 庚坐食神（亥中壬）+ 病 | 表达但内耗 |
| **辛午** | 辛坐官（午中丁）+ 病 | 官杀攻身 → 压力型 |
| **壬寅** | 壬坐食神（寅中甲）+ 病 | 泄秀型 |
| **癸酉** | 癸坐印（酉中辛）+ 病 | 有印救助 → 学者型 |

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 印运 | **大喜** → 救病补身 |
| 比劫运 | **喜** → 帮身 → 得同伴 |
| 禄 / 帝旺运 | **喜** → 气势回升 |
| 食伤运 | **忌** → 加耗 |
| 财官运 | **视结构** → 无救则忌 |
| 冲病位的大运 | 关键节点 → 看冲开方向 |

## 自坐病命造的成就路径

### 1. 接受"**弱**"是底色
- 不强求自己"**像别人一样扛**"。
- 用对方向 = 优势。

### 2. 借力而行
- 天然需要**导师 / 伴侣 / 机构 / 平台**的支持。
- **建立关系网**比独自硬拼重要。

### 3. 深度立身
- 放弃速度战。
- 用**深度 + 敏锐 + 专业**立足。

### 4. 身体是底座
- **养生不是可有可无** → 是基本功。
- 规律作息、适度运动、慢节奏生活。
- 远离高压 / 高噪 / 高消耗的环境。

### 5. 心理建设
- 学会**与焦虑共处**，而不是消灭它。
- 建立**内在意义感** → 减少外部评价的依赖。
- 必要时寻求专业帮助（心理咨询）。

## 核心判断
- 自坐病不是"命差"，是"**敏感而虚**"的特殊底色。
- 许多**心理学家、哲学家、艺术家、疗愈师**都自坐病 → **用敏感转化为深度**。
- 关键识别：
  - 有无印 / 比劫救应。
  - 大运何时补身。
  - 生活方式是否匹配。

## 一句话总结
- 自坐病 = **你天然感受得更深，也消耗得更快**。
- 别和身强者拼"**扛**"，你赢在"**懂**"。
- 给自己慢一点、深一点、稳一点的节奏 —— 你会活得久、看得透、做得深。

## 古典标准：病的阶段特征与类象

### 病的本质

**病就像人得了病一样——身体虚弱、多灾多病、精神不振、运气也不好**——意味着**旧事物已经千疮百孔、积重难返**了。

### 病的记忆口诀（以日主为基准）

- **甲日见巳、乙日见子、丙日见申、丁日见卯**
- **戊日见申、己日见卯、庚日见亥、辛日见午**
- **壬日见寅、癸日见酉**

### 病在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 父母身体不佳；自己在运势低落的时候出生，先天可能就体弱多病 |
| **月柱** | 内心比较悲观消极、自寻烦恼、不思进取；比较缺乏实践能力；中年之前要注意，可能会经常换工作 |
| **日柱** | 配偶身体不好，自己也要注意身体保养；婚姻很难美满 |
| **时柱** | 晚年工作或失业波折多，可能经济困难难得清闲；子女身体也不好，容易生病 |

### 自坐病的典型日柱

**丁卯日、戊申日、己卯日、壬寅日、癸酉日**——日柱为病的五组组合。

### 自坐病的命主特征

1. 特点是**外表看起来无忧无虑，内心却总是忧心忡忡**；**同情心强，为人尽力能得人缘**；但和家人缘分淡薄。
2. **举止缓慢，选择太多，做事过于敏感，过分考虑个人得失**，因此**错失很多良机**。
3. 优点是对于交代的事情大多能**尽心尽责**；这种命**多才多艺**，适合从事**设计、企划、记忆、发展**等工作；**生性意志薄弱，容易幻想，缺乏果断和实践能力**。

### 古典标准铁律

> 十二长生在八字命理中**只起到辅助作用**，不能作为主要依据；以上类象**准确率并不是很高，只能作为参考**——关键要配合日主旺衰、十神、格局、大运综合判断。

## 现代解读

以下内容为**现代综合性延伸**：
- "心理学家、哲学家、疗愈师"等现代职业标签属现代岗位归纳。
- "终身养生是必修课"、"建立内在意义感"等心理学化表述属现代延伸解读。
- 经典病的核心在于**能量虚弱+需要扶助**这一能量学认知。
