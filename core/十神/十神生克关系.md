# 十神生克关系

## 一、本质：五行生克 + 阴阳分化

十神之间的生克，本质是**五行生克**的投影，再叠加**阴阳异同**的力度调节：

- 五行决定"**是生还是克、方向如何**"。
- 阴阳决定"**是有情还是无情、力度如何**"。

| 阴阳搭配 | 作用性质 |
| --- | --- |
| 异阴阳相生 | 贴切有情、流畅顺畅 |
| 同阴阳相生 | 纯粹稳定、温和持久 |
| 异阴阳相克 | 有情之克、规范约束 |
| 同阴阳相克 | 无情之克、破坏锋利 |

## 二、完整相生链

**印星 → 比劫 → 食伤 → 财星 → 官杀 → 印星**（闭环）

| 链节 | 五行对应 | 作用解读 |
| --- | --- | --- |
| 印生比劫 | 生日主五行 → 同日主五行 | 学识修养增益自身与同类群体 |
| 比劫生食伤 | 日主五行 → 日主所生 | 自身力量 → 输出才华与欲望 |
| 食伤生财 | 日主所生 → 日主所克 | 才华 → 变现为财 |
| 财生官杀 | 日主所克 → 克日主 | 财聚招权、纳税升官 |
| 官杀生印 | 克日主 → 生日主 | 权力反哺修养与格局 |

### 细化：阴阳决定哪一个"正偏"相生最切
- **正印生比肩**：稳定庇护聚拢同气同党（阴阳同路，偏正互换需细看）。
- **食神生偏财**：同阴阳连生，温和广进。
- **伤官生正财**：异阴阳连生，才华直接换成稳定收入。
- **偏财生七杀**：同阴阳连生，财助势起。
- **正财生正官**：异阴阳连生，正业立身、名利双全。

## 三、完整相克链

**印星 → 食伤 → 官杀 → 比劫 → 财星 → 印星**（闭环）

| 链节 | 五行对应 | 作用解读 |
| --- | --- | --- |
| 印克食伤 | 生日主五行 → 日主所生 | 学识理智压制空想外泄 |
| 食伤克官杀 | 日主所生 → 克日主 | 才华叛逆破坏规矩权威 |
| 官杀克比劫 | 克日主 → 同日主 | 规矩压制兄弟党羽 |
| 比劫克财 | 同日主 → 日主所克 | 同类分夺我的钱财 |
| 财克印 | 日主所克 → 生日主 | 物欲遮蔽静心修养 |

### 细化：同异阴阳克的差别
- **七杀克比肩**：同阴阳无情之克——权力打压最烈。
- **正官克劫财**：异阴阳有情之克——规则约束冲动。
- **伤官克正官**：异阴阳克，但被称"**伤官见官，祸患百端**"——才华犯上。
- **食神克七杀**：同阴阳克，但被誉为"**食神制杀**"——才华化解凶煞。
- **劫财克正财**：异阴阳克——最典型的分夺妻财。
- **比肩克偏财**：同阴阳克——分夺流动财。

## 七、十神在大运流年中的作用

### 大运十神（十年一换）
- 改变整个命局的**十神配比**——原来弱的得到补、原来旺的被再加强。
- 决定**这十年最主要的人生主题**：走印运重学术、走财运重事业、走官运重身份、走食伤运重表达、走比劫运重人际。
- 大运十神为喜用：十年顺遂；为忌仇：十年蹉跎。

### 流年十神（一年一换）
- 触发具体事件，是"**事件的扳机**"。
- 必须和**命局原局 + 当前大运**三方合参——原局有、大运引、流年发。
- 流年冲克原命喜用神：该年有灾；流年合助原命喜用神：该年得利。

### 三合参口诀
1. **看原命喜忌**——确定哪些十神是用神、哪些是忌神。
2. **看大运助忌还是助用**——定十年大局。
3. **看流年触发哪个十神、冲合原局何处**——定具体事应。

## 八、一句话总结

十神生克是**五行生克 + 阴阳分化**的精细投影：五行定方向、阴阳定力度，相生一条闭环、相克一条闭环，任何一个十神的作用都不能孤立看——**必看其生者、克者、所生、所克四个方向的配合**，才能判断它在具体命局中是助力还是拖累。
