# 从杀格

## 定义
- **从**：顺从，放弃自身立场。
- **杀**：七杀（正官偶尔也算，但主论七杀）。
- **从杀格**：日主**极弱**、**七杀极旺**、**无印化无食制** → 日主无力抗杀 → **反过来顺从七杀**的气势。
- 属**外格**，与从财格、从官格、从儿格并列。
- 成格时：贵而猛，典型是**古代将帅、企业强人、危机中崛起的领导者**。

## 核心：弱到极致反而臣服于压力
- 正格思路：身强抗杀、有印化杀、有食制杀。
- 外格思路：身**无力抗拒、无印无食、无比劫帮身**时，不如"**认命 + 顺从**"。
- 从杀 = **日主把自己交给压力** → 压力不再毁人，反而**推人向上**。
- 比喻：**被洪水冲时，不再挣扎，顺流而下反而活下来**，并到达更远的地方。

## 成立条件（严格依《滴天髓》《子平真诠》）

### 条件 1：日主无根（天干无比印同现，地支无比劫 / 印星之本气、中气、余气根）

- **古籍出处**（本条最核心）：
  - 《滴天髓·从象》（任铁樵注）："**日主孤立无气，天地人元绝无一毫生扶之意**。"
  - 《子平真诠·论杂格》："日主无根，不得不从其旺神。"
- **客观判据**：
  - ① 日主天干周围无一比劫或印星透出；
  - ② 日主在年 / 月 / 日 / 时四地支的本气、中气、余气中**皆无任何比劫或印星藏干**。
- **不满足则破**：日主有任何一丝根 → 沈氏、任氏均严诫"**有一丝根则不从**" → 只是"身弱杀重"的普通破局，非真从杀。

### 条件 2：七杀（或官杀）成气势满局

- **古籍出处**：
  - 《滴天髓·从象》（任铁樵注）："**财官强甚**，乃日主顺从财官之势也。"
- **客观判据**（三者之一）：
  - ① 月令地支本气为七杀 + 七杀透干且有根；
  - ② 地支三合或三会成七杀局（甲日巳酉丑会金、乙日申子辰不会杀故另判，等等——须按日干与所克我之五行核对）；
  - ③ 七杀多透且遍根（年月日时多位七杀气气势统治全局）。
- **不满足则破**：七杀仅一透无根或一藏支 → 气势不足 → 转入身弱杀重讨论。

### 条件 3：原局无印星化杀

- **古籍出处**（本条铁律）：
  - 《滴天髓·从象》："**若有一毫印绶，即不为从**。"
- **客观判据**：
  - 原局天干无印星透出；地支本气、中气无印。（若仅地支余气藏一位印，气微勉强可论从，但降为"假从"。）
- **不满足则破**：印有力 → 日主借印化杀 → 当论"杀印相生"，非从杀。

### 条件 4：原局无食伤制杀

- **古籍出处**：
  - 《滴天髓·从象》（任铁樵注）："**日主既顺从他神，则他神即为我主，不可再言食伤生财、官杀化印之类**。"
- **客观判据**：
  - 原局天干无食神 / 伤官透出；地支本气、中气无食伤。
- **不满足则破**：食伤有力 → 日主借食制杀 → 当论"食神制杀"，非从杀。

### 条件 5：原局无比劫帮身

- **古籍出处**：
  - 《滴天髓·从象》："**比劫一见，即不为从**。"
- **客观判据**：
  - 原局天干无比劫；地支本气、中气无比劫。
- **不满足则破**：比劫透干或有本气根 → 日主有抗杀之助 → 假从，归入"身弱杀重"破格讨论。

### 条件 6（助力条件）：有财星透干生杀

- **古籍出处**：
  - 《滴天髓·从象》："**从神又有吉和凶**——从杀以财为喜，财旺则从势益真。"
- **客观判据**：
  - 财星透干且通根，位置在七杀之前或同柱（财→杀顺流生）。
- **非必要**：不满足不破格，但格品降——无财则杀独立成势，未经财助。

---

**前五条必须同时满足**方为真从杀格；条件 6 为助力加分。真从杀格之所以稀有，正在"无根 + 无印 + 无食伤 + 无比劫 + 杀旺"五项全无退路——《滴天髓》"若有一毫印绶，即不为从"为本格识别的绝对底线。

## 从杀格 vs 身弱杀重

| 对比项 | 从杀格 | 身弱杀重（破格） |
|---|---|---|
| 日主 | 极弱无根 | 微弱尚有根 |
| 杀的气势 | 满局成势 | 重但非一统 |
| 面对杀 | 心甘情愿顺从 | 被迫对抗 |
| 结果 | 贵格，危机成就 | 下局，灾祸不断 |
| 补救 | 顺势而为 | 补身（印 / 比劫） |

- 同样"身弱 + 杀旺"，日主**还有没有一丝根**是关键。
- 有根 → 挣扎中被压碎。
- 完全无根 → 顺流反而借势起飞。

## 判断维度

### 1. 真从 vs 假从
- **真从**：一丝印比都没有，日主彻底融入杀气。
- **假从**：原局有微弱印比 → 日主想从又抗 → 陷入"身弱杀重"的惨局。

### 2. 杀的纯度
- **纯七杀成势**：格局最纯，最典型。
- **官杀混杂**：气不清，从得不彻底，格次一级。
- **七杀 + 财配合**：财生杀，格局更稳。

### 3. 财星角色
- **有财生杀**：顺势，贵格加持。
- **无财**：杀独立成势，看杀能否自足。
- **财过旺破印（若原局有印）**：财破印 → 顺便打破退路 → 反而有利于真从（但要小心走偏）。

### 4. 食伤角色
- **无食伤**：最干净，从杀最纯。
- **有食伤制杀**：直接破格，从不成。
- **大运走食伤**：从杀格遇食伤运 = **大凶**。

## 常见误区

### 1. "身弱杀重 = 从杀格"
- 最大误判。
- 真正的从杀格**极为罕见**。
- 大部分"身弱杀重"是**普通破格** → 该补身、该求印，不是该顺杀。
- 误判代价：把该"避祸"的人当成该"冲锋"的人 → 一辈子被压力碾压。

### 2. 以为从杀 = 权贵一生
- 是贵格，但**极度依赖大运配合**。
- 大运走印、比劫 → 打破从格 → **瞬间反噬**。
- 从杀格人生往往**暴起暴落**，不像官印相生那种温和的上升。

### 3. 忽视心理代价
- 从杀的人**内心长期处于高压状态**。
- 外表看起来强悍、有魄力，内里：
  - **焦虑、警觉、难以放松**。
  - **对"失控"极度恐惧**。
  - **情绪波动大**。
- 成就很高，**心理消耗也很重**。

### 4. 把"从"当作"软弱"
- 从不是怯懦，是**承认力量悬殊并借势**。
- 真正从杀格的人**胆识过人、决断果断** —— 因为知道退无可退，反而一往无前。

## 现实表现

### 性格
- **胆大、果断、不怕事**。
- **关键时刻能上**：越危险越冷静，越压力越集中。
- **执行力极强**：不犹豫、不内耗。
- **眼神里有"狠劲"**：外人一眼能感到。
- **不走寻常路**：传统的稳定路径对他们来说反而不适应。

### 事业路径
- **军警、特种行业**：高风险、高强度岗位。
- **企业家、创业者**：敢于赌未来、敢于承担风险。
- **危机管理者**：乱世中显英雄。
- **竞技、谈判、高压决策岗**。
- **改革派 / 破局者**：在原有秩序里横空出世。

### 成就特征
- **非传统路径**：不靠学历 / 资历一步步升，**靠关键时刻的关键一跃**。
- **大起大落**：命运曲线非常陡峭。
- **耀眼但不稳定**：闪耀时光芒万丈，跌落时也极剧烈。

## 破格：假从反噬

### 破格表现
- 原局有印一点 / 比劫一丝 → 日主既从不干净又抗不住：
  - 一辈子被压力追着跑。
  - 事业多波折、官非、牢狱、意外频发。
  - 典型画像："**扛不住又不肯认命**"。

### 为什么假从比正格更惨
- 真从 = 顺流借势。
- 正格 = 有路可走。
- 假从 = **无路可退 + 不知道自己无路**。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 官杀运 | **大喜** → 杀气加持，贵格兑现 |
| 财运 | **喜** → 财生杀，顺势加强 |
| 食伤运 | **大忌** → 食伤制杀，破格 → 重大反转 / 下台 |
| **印运** | **大忌** → 印化杀，破从 → 失去势头 |
| **比劫运** | **大忌** → 比劫帮身，从格瓦解 |

## 与相关格局的关系

| 格局 | 日主状态 | 顺从对象 | 气质 |
|---|---|---|---|
| **从杀格** | 极弱 | 七杀 | 猛烈、贵而凶 |
| **从官格** | 极弱 | 正官 | 顺势、仕途路径 |
| **从财格** | 极弱 | 财 | 重物质 |
| **从儿格** | 极弱 | 食伤 | 才华横溢 |
| **杀印相生** | 中等 | 印化杀 | 温和抗压 |
| **食神制杀** | 身强 | 食神制杀 | 主动解决 |

## 核心判断
- 从杀格的**真正稀缺**：不是"身弱杀重"就能成，而是"**彻底无退路**"才能成。
- 识别从杀格必须极度谨慎 —— 宁可错判为普通破格而补身，也不要误判为从格而顺杀。
- 大运配合时：**一战封神**；大运不配合时：**瞬间倾塌**。

## 一句话总结
- 弱到极致无路可退 → 顺势而行反而借到最强的力。
- 从杀格的贵不是"不受压"，是**让压力变成自己的推进器**。
- 但这条路的代价是：**一生都在高压里走，不能退也不能歇**。
