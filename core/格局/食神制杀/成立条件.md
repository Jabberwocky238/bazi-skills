# 食神制杀

## 成立条件（严格依《子平真诠》《滴天髓》）

### 条件 1：月令为七杀，或七杀透干且通根

- **古籍出处**：
  - 《子平真诠·论七杀》："月令七杀，以食制之为上格。"
  - 《子平真诠·论用神》："用神专求月令，以日干配月令地支。"
- **客观判据**：
  - ① 月令地支本气为七杀（如甲日生申月、乙日生酉月），或
  - ② 七杀透于年 / 月 / 时干，且在地支有一位以上本气或中气通根。
- **不满足则破**：七杀仅为地支藏干的余气（如甲日见戌，戌中辛金为余气），不足以立格；只能论为"杂气七杀"，非本格。

### 条件 2：食神通根月令或日时支，且透于天干

- **古籍出处**：
  - 《子平真诠·论食神》："食神生旺，胜似财官。"
  - 《子平真诠·论食神》："食神必得生旺之地，方能吐秀制杀。"
- **客观判据**：
  - ① 食神透于月干或时干（不取年干——年干距日主远，作用力衰），且
  - ② 地支有一位食神本气，或两位食神余气通根。
- **不满足则破**：食神虚透而无根（沈氏谓之"浮露"），或有根而不透（沈氏谓之"暗藏"），均不能构成"制"的动作。

### 条件 3：食神与七杀力量两停（食 ≥ 杀，且差距不超过一倍）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论七杀》："食杀两停，贵格也。"
  - 《子平真诠·论七杀》："**食神轻而七杀重，反为所伤；食神重而七杀轻，反为无用**。"
- **客观判据**（将沈氏三分法量化）：
  - ① **食杀两停**（食 ≈ 杀，或食略过于杀）→ 成格、贵格。
  - ② **食轻杀重**（七杀透干且三会 / 三合成局，食神仅一位透出无助）→ 破格，"反为所伤"。
  - ③ **食重杀轻**（食神月令得令且透 + 有根，七杀仅藏支一位）→ 破格，"反为无用"（杀被制尽，无事可成）。
- **不满足则破**：沈氏明文三种破格情形任一成立即不成格。

### 条件 4：食神与七杀的柱位相邻（紧贴或隔一柱可制）

- **古籍出处**：
  - 《子平真诠·论用神》："用神须紧贴日主，方为真用。"
  - 《滴天髓·通神论》："贴身者力重，遥隔者力轻。"
- **客观判据**：
  - ① 食神与七杀分别居于**日主相邻的两柱**（月柱 / 时柱，或日支 / 月支），或
  - ② 食神与七杀位于**同一柱的干与支之间**（食神透干、七杀藏支，或反之）。
- **不满足则破**：食神在年柱、七杀在时柱，中间被印星 / 财星阻隔（沈氏谓"用神被隔"），则克不相及。

### 条件 5：原局无偏印（枭神）紧贴克食神

- **古籍出处**（本条用其典名）：
  - 《子平真诠·论食神》："**食神最畏枭神**。"
  - 《子平真诠·论偏印》："偏印夺食，福去祸至，轻则贫困，重则夭折。"
- **客观判据**：
  - ① 天干无偏印与食神**紧贴相克**（如食神在月干、偏印在年干或日干），且
  - ② 地支无偏印本气位于食神本气的相邻柱。
- **救格条件**（亦出古籍）：若偏印存在但有**财星紧贴克偏印**（《子平真诠》："财能破印，救食神"），则本格仍成立。
- **不满足则破**：枭神贴食、又无财救 → 本格立破，转入"枭神夺食"讨论。

### 条件 6：日主不至极弱（至少在月令有余气，或他柱有比劫印星相助）

- **古籍出处**：
  - 《子平真诠·论七杀》："**身强食旺，名利双收；身弱食衰，无以制之**。"
  - 《滴天髓·体用》："身旺者任克任泄，身弱者不任克泄。"
- **客观判据**：
  - ① 日主于月令本气为比劫 / 印星 / 禄 / 刃（身强），或
  - ② 日主于月令虽非比印，但年、日、时三柱至少有两位比劫或印星通根（身中）。
- **不满足则破**：日主无根 + 印比皆无 → 沈氏"身弱食衰，无以制之"——名为"制杀"，实则日主自身已不堪。此时极弱若满足其他条件，可能转入"从杀格"讨论（反面）。

---

**六条同时满足**方为真食神制杀格。任一条不满足即破格或降格，须按沈氏 / 刘氏原文对应的破格结论处置。
