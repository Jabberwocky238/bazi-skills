# 天元一气格

## 定义
- **天元**：八字中的天干（年 / 月 / 日 / 时四柱之天干）。
- **一气**：同一种五行之气、同一个字、同一股气势。
- **天元一气格**：**四柱天干完全相同**的八字（如甲甲甲甲、庚庚庚庚、癸癸癸癸）。
- **最极端形态**：**天地同流** —— 四柱天干相同且四柱地支亦全相同（如癸亥、癸亥、癸亥、癸亥），全局八字只剩两个字。
- 极为**罕见**的特殊格局，五行偏枯到极致；命主命运往往极端——**要么大贵、要么大凶**，甚难中庸。

## 核心原理
- 普通八字讲究**五行流通、刚柔相济、日主与十神之平衡**。
- 天元一气则**反其道而行**：四干同气，强行打破平衡，形成**一股压倒性的气势**。
- 既然五行已偏到极端，**再谈"平衡"已无意义** → 唯一的出路是**顺势而行**：
  - 气势要水 → 顺着走水与水所生之物。
  - 气势要火 → 顺着走火与火所生之物。
  - 气势要金 → 顺着走金与金所生之物。
- **偏枯之格、取用反简**：不必纠结中和，只认"**气势方向**"。
- **五行越偏、寿元越难长**：这是天元一气格普遍的一体两面——气势越纯粹，生理上越难中和。
## 取用原则（核心方法）

**所有天元一气格的取用口诀**："**顺其气势、不逆其锋**"。具体分三种：

### 1. 相生类天元一气 → 取"**日主 + 支中之气**两类为用，再加一个顺生的**五行**
- 例：壬寅 → 日主水 + 支中木，两者相生 → 取**水木**两类为用，再加一个木所生的**火**顺生。
- 例：丁未 → 日主火 + 支中土 → 取**火土**为用，再加一个土所生的**金**顺生。
- **喜**：日主之行 + 支中主气 + 顺生之行。
- **忌**：逆气势之行（克日主或克支中主气的五行）。

### 2. 相克类天元一气 → 取"**日主 + 支中之气**两类为用，再加一个**通关**的五行
- 例：乙酉 → 日主木 + 支中金，金克木 → 取**金木**为用，再加**水**通关（金生水、水生木）。
- 例：甲戌 → 日主木 + 支中土，木克土 → 取**木土**为用，再加**火**通关（木生火、火生土）。
- **喜**：日主之行 + 支中主气 + 通关之行。
- **忌**：加剧相克的五行、破坏通关之物。
- **注**：相克类在现实中常兼论"身杀两停"之意——因日主与支气恰成克用。

### 3. 全同类天元一气 → 取"**日主同类 + 生日主的印星**"为用
- 例：癸亥 → 日主水 + 支亥水，全同 → 取**水**（比劫）与**金**（印星）为用。
- **喜**：同类 + 印星。
- **忌**：财星（克印断根）、官杀（克日主夺势）、食伤（此类极偏之局不能再耗气势，与普通专旺格可用食伤不同）。
- **特别注意**：此类气势**比专旺格还要极端**——专旺格允许印星，天元一气全同类不单允许印星，**连食伤都不宜用**，一切都要**锁在比劫和印星这条线内**。

## 与相关格局的对比

| 格局 | 天干结构 | 地支结构 | 取用关键 | 性质 |
|---|---|---|---|---|
| **天元一气格** | 四干同一字 | 有配合（生 / 克 / 同三类） | 顺气势、不论中和 | 极偏枯、极罕见、极端命 |
| **专旺格** | 不要求同字 | 会局或方局成一方之势 | 顺势同党为用 | 偏枯、较罕、贵格 |
| **从强 / 从旺格** | 比印满局 | 比印满局 | 从日主之强 | 偏枯、依附日主之势 |
| **化气格** | 日主与月干或时干合化 | 化神得令得助 | 从化神之气 | 偏枯、依合化而成 |
| **两干不杂格** | 仅两个天干交替 | 地支相配 | 取两干之用 | 较偏、贵气结构 |

### 天元一气 vs 专旺格
- 专旺格：四柱五行**以某一行为主**但**允许印星存在**，结构相对宽松。
- 天元一气（全同类）：四柱天干**必须完全一字**，连印星都不允许插入天干 → **更纯粹、更极端**。
- 专旺格可以用**食伤泄秀**；天元一气全同类**不可用食伤**——因一用即泄，破坏纯气之势。

### 天元一气 vs 从强格
- 从强格强调"日主得比印围抱、顺其强势"，仍可能带其他五行点缀。
- 天元一气在天干层面**无点缀空间**——四干必须同字，结构上比从强更刚性。

## 判断维度

### 1. 四干同字是第一硬指标
- 缺一即非本格。
- 不能由"三干同 + 一干合化"凑数——合化不改变字面。

### 2. 地支配合的三类属性
- **相生类**：命运方向偏贵、偏柔、偏成就。
- **相克类**：命运方向偏猛、偏特殊、偏高成就 / 高风险并存。
- **全同类**（极端）：命运方向**要么极贵、要么极凶**，无中间地带。

### 3. 地支刑冲的程度
- 完全无刑冲 → 气势圆满 → 格局上乘。
- 轻微刑冲但被化解 → 格局中上。
- 严重冲克 → 气势被破 → 格局由贵转凶。

### 4. 大运的方向
- 顺气势之运 → 格局发挥到极致 → 可能大贵。
- 逆气势之运 → 气势被冲 → 可能大凶。
- **天元一气格的大运吉凶对比差极大**：顺运时的人生高度与逆运时的人生低谷之间差距远超常命。

### 5. 五行偏枯对身体的影响
- 越极端的天元一气（尤其全同类）→ 身体五行越难中和。
- 古书提示：**此类命造少见长寿之人**，是格局本身的生理代价。
- 从健康角度，**五行中和**的普通八字反而更利于长寿——这是本格无法规避的一面。

## 现实表现

### 性格
- **极端纯粹**：一股劲、一种脾气、一个方向。
- **意志力惊人**：认定一件事就死磕到底。
- **不易被外力动摇**：心性坚定近乎固执。
- **情绪两极化**：高时极高、低时极低，少有平缓。
- **常带一种"非人间"气质**：要么气场凛冽、要么神色飘忽。

### 事业路径
- **单一领域的极致深耕**：一辈子只干一件事并做到顶尖。
- **非主流赛道**：因偏枯之气与常人不合，反而在冷门 / 极端 / 特殊行业出众。
- **宗教 / 玄学 / 艺术 / 武道 / 科研**：这类"钻一门到底"的领域。
- **帝王 / 枭雄 / 开创者**：历史上亦有天元一气成大事者，但代价极重。
- 共性：**越纯越猛、越偏越能出头**，但**退路极少**。

### 成就特征
- **要么大成、要么大败**：中间档极少。
- **大运顺时可能登顶一方**；**大运逆时可能万劫不复**。
- **人生落差极大**：起伏远超常人。

### 人际与感情
- **知己极少**：气质偏枯，一般人难靠近。
- **婚姻多波折**：对象需极强心理承受力，否则难以长期共处。
- **六亲缘薄**：五行偏枯导致十神偏枯，亲属关系多有缺憾。
- **六亲特别易损**（尤其全同类）：克父母、克子女、刑克兄弟等情形古书多有记载。

## 常见误区

### 1. "四干同字就是贵格"
- **不一定**。需看地支是否配合、有无严重刑冲。
- 地支背离或冲破 → 非但不贵，反而极凶。

### 2. 套用普通八字的"中和 / 平衡"思路
- 普通八字求中和，天元一气格**必偏不求中**。
- 一味追求补弱行 → 反而逆气势 → 格局被破坏。
- 必须**放弃中和思维**、改用**顺势思维**。

### 3. "用神看不懂就不取用"
- 有些人看到天元一气格就傻眼，以为无法取用——错。
- **所有八字都有用神**：天元一气格的用神规则清晰（见"取用原则"），只是与普通八字不同。
- 关键是**找对方法**，而非拒绝取用。

### 4. 全同类可用食伤（误）
- 普通专旺格可以用食伤泄秀。
- 但天元一气**全同类**（如癸亥癸亥癸亥癸亥）**不能再用食伤**：
  - 目的是让日元更旺、不消耗任何气势。
  - 用食伤反而稀释了"纯气"。

### 5. 墓库要小心（甲戌等克类）
- 有人看到甲戌、辛卯等克类天元一气，就纠结戌是火库、卯是木库等墓库逻辑。
- **特殊格局面前，普通墓库理论不适用** → 不要再论墓库。
- **只看气势流向**，不纠结地支暗藏的细节。

### 6. 误以为一定长寿
- 五行偏枯**生理上难中和**，古书多记天元一气格命主**虽贵难寿**。
- 格局与健康是两回事——格局上的"贵"不能等同于寿命上的"长"。

## 破格与救格

### 破格
- **地支严重冲克**：打破一气贯通 → 格局瓦解。
- **大运逆气势**：大运之气与本局气势相克 → 阶段性破格。
- **流年五行正好冲破地支之根**：气势被断 → 流年极凶。
- **原局出现异类天干（误判为天元一气而实则不是）**：根本不成格。

### 救格
- **大运回到顺气势之行**：重新激活气势 → 破格转贵。
- **轻微刑冲有合化解之**：如地支相冲但被另一字合去 → 冲破被化解。
- **顺生 / 通关之行补位**：对相生类补顺生行、对相克类补通关行、对全同类加印星。
- **不救的救法**：有时最大的救是"**不动**"——天元一气格怕乱，不怕静。

## 大运流年影响

| 类型 | 喜 | 忌 |
|---|---|---|
| **相生类**（壬寅 / 己巳 / 丁未 等） | 日主之行 + 支中主气 + 顺生之行 | 克日主 / 克支中主气之行 |
| **相克类**（乙酉 / 甲戌 / 辛卯 / 丙申 等） | 相克两方 + 通关之行 | 加剧克战 / 破坏通关之物 |
| **全同类**（癸亥 / 戊午 等） | 同类 + 印星 | 财（克印）、官杀（克身）、食伤（泄气） |

### 大运的"顺与逆"决定命运起伏幅度
- 天元一气格**对大运的敏感度远超常命**：
  - 顺运：可能一飞冲天、成就非凡。
  - 逆运：可能急转直下、凶险非常。
- 大运顺行或逆行的差别巨大——男命阳年 / 女命阴年顺行，反之则逆。

### 走完好运易立即接衰运
- 天元一气格**起伏剧烈**：好运巅峰后常紧接着衰运低谷。
- 这是气势极端纯粹、大运一转即反的必然后果。
- 命主需有极强心理准备——"得时莫狂、失时莫丧"。

## 一句话总结
- 天元一气 = **四干同字、一气贯通**的极端格局。
- 普通八字讲中和，此格**不求中和只求顺**。
- 贵与凶的分水岭不在格局本身，而在**大运是否顺其气势**。
- 命运极端、起伏极大、寿元难长—— 这是**"气势之纯"**必付的代价。
- **所有八字都有用神**：看不懂天元一气，不是命无用神，而是思路还停留在中和思维里。
