# 女生择偶

## 核心命题
择偶不是"找一个好男人"这么简单，而是**先看清：他的命局里那股力量最强，在婚姻里会把这股力量用在哪**。
- 同一股能量，在朋友圈可能是优点。
- 进入婚姻却可能是消耗源。

**要区分的是"外部型人格"与"内部型人格"的区别**——有些男人是为外场而活，有些男人才为家而活。

## 最典型的一类要警惕：劫财过重的男人
> 详见 `婚姻/劫财男.md`。

**核心警示**：
- 不是"坏"，而是**注定让你累**。
- 什么都要和你"分"，没有界限感，家庭不稳。
- 表面仗义，骨子里自我。

## 择偶时应该看的几条线

### 1. 日主与夫妻宫
- **日主稳、夫妻宫安静** → 婚后能定下来。
- 日主过弱、夫妻宫受冲刑 → 婚姻易动荡，自己也难稳定。

### 2. 财星的质量（男命）
不是看他有没有钱，而是看他**对待资源的方式**：
- **财清、财根稳** → 会过日子，能为家经营。
- **财多而杂、财被劫夺** → 要么欲望多选择多，要么留不住。
- **无财 + 比劫重** → 对"经营一个家"这件事不擅长（详见 `婚姻/男命无财.md`）。

### 3. 官星与担当
- 官星清、有根 → 有责任感、扛得起事。
- 官星杂乱、伤官见官 → 容易"嘴上承担，行动逃避"。

### 4. 比劫与边界感
- 比劫重 → 圈子热闹但**家里冷清**。
- 比劫无制 → 容易在关系里较劲、争强。

### 5. 伤官与情绪稳定度
- 伤官带印 → 有才华又不失分寸。
- 伤官无制 → 嘴快、情绪化、面子重、难以被管理。

## 择偶三个现实角度

### 角度一：他现在靠什么获得"存在感"
- 靠朋友圈热闹 → 把家当"休息站"。
- 靠事业成就 → 稳定但可能忽略情感。
- 靠家庭情感 → 才是真正"为家庭活"的那类。

### 角度二：他在压力下会做什么
- 往外跑（应酬、喝酒、消失）→ 情绪处理能力低。
- 往内缩（沉默、冷淡、僵硬）→ 沟通能力低。
- 面对、谈、解决 → 成熟度够。

### 角度三：他怎么对待自己的原生家庭
- 过度依附 → 进了婚姻会把你当"替代母亲"。
- 完全割裂 → 情感模式不健康，婚姻里也容易冷漠。
- 有距离、有责任 → **成熟信号**。

## 最重要的提醒

### 别把"表面好"当全部
- 人缘好、嘴甜、仗义 → **这些在婚姻里未必兑现**。
- 真正婚姻里的"好"是：**分得清家里家外、扛得住平淡、能承担冲突后果**。

### 命局遇上了怎么办
- 如果命里硬是遇上了一个"消耗型"伴侣，别幻想他为你改变。
- 调整的是**你自己**：
  - 自己的经济筹码。
  - 自己的情绪系统。
  - 自己的边界感。
- **命局决定底盘，你的选择决定走向**。

## 一句话总结
女生择偶不怕他穷、不怕他一般，就怕他**内在结构注定无法承担一段稳定关系**——
- 先看结构，再看感觉。
- 先看他日常怎么做，再看他嘴上怎么说。
- 先看他对家的态度，再看他对外的表现。
