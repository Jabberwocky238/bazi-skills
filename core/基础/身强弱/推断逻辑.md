# 身强弱推断逻辑

> 本文是一份**流程手册**。拿到任何一张八字，按本文步骤逐项判断、逐项计分、逐项定级，即可得出日元身强弱的结论。

## 一、前置概念

在进入算法流程前，先锁定 5 个最小工作概念：

- **日元**：日柱天干，即命主本人。一切旺衰讨论围绕日元展开。
- **月令**：月支。代表出生大环境，为整局能量最大的单字，占全局权重约三成。
- **地支藏干**：每个地支内藏 1～3 个天干之气，分**本气 / 中气 / 余气**三级。
- **十神生克**：相对日元而言——
  - **同党**（助身）：比肩、劫财（帮身）；正印、偏印（生身）。
  - **异党**（耗身）：食神、伤官（泄身）；正财、偏财（耗身）；正官、七杀（克身）。
- **根**：天干在地支藏干中能找到相同五行，即为"通根有根"。根是日元是否真正"站得住"的生命线。

## 二、四项判据

身强弱判断的四大支柱——**得令、得地、得生、得势**。每一条各管一层。

### 1. 得令（月令层）

#### 什么算得令

- 月令五行**生日元（印）**或**同日元（比劫）**——得令。
- 月令五行**克/泄/耗日元（官杀/食伤/财）**——失令。

#### 人元司令修正

月令并非整月都由本气当值。交节后按**余气 → 中气 → 本气**依序轮值：

| 月令 | 余气（上月余绪） | 中气 | 本气（主气） |
|---|---|---|---|
| 寅 | 戊（约7天） | 丙（约7天） | 甲（约16天） |
| 卯 | 甲（约10天） | — | 乙（约20天） |
| 辰 | 乙（约9天） | 癸（约3天） | 戊（约18天） |
| 巳 | 戊（约5天） | 庚（约9天） | 丙（约16天） |
| 午 | 丙（约10天） | 己（约9天） | 丁（约11天） |
| 未 | 丁（约9天） | 乙（约3天） | 己（约18天） |
| 申 | 己戊（约10天） | 壬（约3天） | 庚（约17天） |
| 酉 | 庚（约10天） | — | 辛（约20天） |
| 戌 | 辛（约9天） | 丁（约3天） | 戊（约18天） |
| 亥 | 戊（约7天） | 甲（约5天） | 壬（约18天） |
| 子 | 壬（约10天） | — | 癸（约20天） |
| 丑 | 癸（约9天） | 辛（约3天） | 己（约18天） |

> 交节后第 N 天落入哪段，由该段司令。**司令之神**决定本月对日元是实实在在的生扶还是克耗。

**重要提醒**：人元司令是判断**天干旺衰的权威依据**，但**不是定格局的依据**，格局仍以月令本气为先。

#### 得令 / 失令判据表

| 日干 | 得令（旺相）月份 | 死绝最弱月份 | 其余 |
|---|---|---|---|
| 甲乙（木） | 寅、卯、亥、子 | 申、酉 | 休囚 |
| 丙丁（火） | 巳、午、寅、卯 | 亥、子 | 休囚 |
| 戊己（土） | 巳、午、未、戌 | 寅、卯 | 辰湿土、丑冻土帮扶差 |
| 庚辛（金） | 申、酉、辰、丑 | 巳、午、未 | 戌燥土淬金，生金不力 |
| 壬癸（水） | 亥、子、申、酉 | 未、戌 | 丑：壬水休囚，癸水有根；辰：水库一般不克水 |

### 2. 得地（地支根层）

**得地 = 日元在地支藏干中有本气强根**。核心在"根的级别"。

#### 三级根力

| 根级 | 条件 | 力量 |
|---|---|---|
| **本气根（强根）** | 日元五行 = 地支本气五行 | 最强 |
| **中气根（中强根）** | 日元五行 = 地支中气五行 | 中等 |
| **余气根（弱根）** | 日元五行 = 地支余气五行 | 最弱，可忽略 |

另按位置细分：

- **正根**：天干坐下地支通根（天覆地载），力量最大。
- **旁根**：非坐下的地支通根，力量次之。

#### 各地支对日主的根层级对照表

| 日主 | 本气根（强根） | 中气根 | 余气根（弱根） |
|---|---|---|---|
| 甲乙（木） | 寅、卯 | 亥（甲）、未（乙）、辰（乙） | — |
| 丙丁（火） | 巳、午 | 寅（丙）、戌（丁）、未（丁） | — |
| 戊己（土） | 辰、戌、丑、未 | 巳（戊）、寅（戊）、申（戊） | 午（己，藏中气） |
| 庚辛（金） | 申、酉 | 巳（庚）、丑（辛） | 戌（辛） |
| 壬癸（水） | 亥、子 | 申（壬）、辰（癸） | 丑（癸） |

#### 得地的计分标准

| 情形 | 权重 |
|---|---|
| 本气正根（坐下通根） | +3 |
| 本气旁根 | +2 |
| 中气正根 | +1.5 |
| 中气旁根 | +1 |
| 余气根 | +0.5 |
| 无根（虚浮） | 0 |

**铁律**：判断天干强弱时，**首先看是否通根**，而不是看天干上有无印比。无本气根则"虚不受补"，任凭天干印比成群也只是虚强。

### 3. 得势（全局层）

**得势 = 全局同党总数 > 异党总数**。只数"人头"，不论位置。

#### 判据

- 数出八字天干 + 地支藏干中所有的**印、比**（同党）个数。
- 数出八字天干 + 地支藏干中所有的**食伤、财、官杀**（异党）个数。
- 同党 > 异党 → **得势**；反之 → **失势**。

### 4. 得生（天干层）

**得生 = 天干上有字对日元起生扶作用**；细分为：

- **得生**：天干有印星（生我）。
- **得助**（又称"得柱"）：天干有比劫（帮我）。

得生的前提是**日元自身要有根**——无根者不受生。印多反成"水多木漂""金多水泛"之害。

## 三、加权计分法

以下是一套**可机械执行**的打分算法。任何八字都可按下表打分、汇总、定级。

### 计分规则

| 项目 | 分值 |
|---|---|
| **得令**（月令为日元印/比劫） | **+3** |
| **失令**（月令为日元食/伤/财/官/杀） | **−3** |
| 人元司令当值为同党（印/比） | 额外 +1（叠加在月令分上） |
| 人元司令当值为异党 | 额外 −1 |
| 本气正根（日柱坐下通根） | **+3** |
| 本气旁根（月、年、时支通根） | **+2** |
| 中气正根 | **+1.5** |
| 中气旁根 | **+1** |
| 余气根（任何位置） | **+0.5** |
| 天干见印（每位） | **+1** |
| 天干见比劫（每位） | **+1** |
| 天干见食伤（每位） | **−1** |
| 天干见财星（每位） | **−1** |
| 天干见官杀（每位） | **−1** |
| 月令为印比但日元月令无中气及以上根 | **−1**（修正：虽得令而根基不稳） |

### 定级区间

按最终分数 **S** 落入下表区间定级：

| 分数 S | 等级 | 状态描述 |
|---|---|---|
| S ≥ 10 | **身极旺（旺极）** | 气势溢出，多转专旺/一行格 |
| 7 ≤ S < 10 | **身旺（太旺/偏旺）** | 根深令得，能量过剩，需克泄 |
| 4 ≤ S < 7 | **身中强** | 同党胜异党，格局偏强 |
| 1 ≤ S < 4 | **身中（中和偏强）** | 接近平衡，偏向同党 |
| −1 ≤ S < 1 | **身中（中和偏弱）** | 接近平衡，偏向异党 |
| −4 ≤ S < −1 | **身略弱** | 日元稍有根气，但异党压过 |
| −7 ≤ S < −4 | **身弱** | 根浅令失，异党强势 |
| −10 ≤ S < −7 | **身极弱（太弱/弱极）** | 接近从势 |
| S < −10 | **近从弱 / 从格候选** | 需进一步考察是否真从 |

> 注：此评分法**仅作参考基线**。八字中一旦出现合、冲、刑、害、三合局、三会局、六合局等复杂作用关系，需在定级后做动态修正。

## 四、六大判断步骤（Algo Flow）

```
┌── Step 1：锁定日主五行
│      输入：日柱天干 → 输出：日主五行、阴阳
│
├── Step 2：判月令（得令？）
│      2.1 查四时旺衰表 → 得令 / 失令基础判断
│      2.2 算交节后天数 → 查人元司令分野 → 司令修正
│      2.3 落盘月令分数：+3 / −3（并酌加 ±1 司令修正）
│
├── Step 3：查四地支对日主的根层级（得地？）
│      遍历年支、月支、日支、时支：
│        - 命中本气 → 记本气根（正/旁）
│        - 命中中气 → 记中气根（正/旁）
│        - 命中余气 → 记余气根
│      累加根分
│
├── Step 4：数天干（得生 + 得势？）
│      4.1 遍历年干、月干、时干（日干本身不计）：
│          印/比 → +1；食伤/财/官杀 → −1
│      4.2 汇总全局同党 vs 异党头数 → 得势 / 失势
│
├── Step 5：汇总加权总分 S → 定级
│      S = 月令分 + 根分 + 天干分 + 修正
│      对照第三节定级区间表 → 得出身强弱级别
│
└── Step 6：大运流年动态调整
       - 大运带来本气强根 → 原本虚强 / 身弱者可转为真旺
       - 大运克冲原命根（冲月令、刑日支等）→ 原本身强者可转弱
       - 岁运合化改变局中五行比重 → 当下实际强弱再校准
```

### Step-by-Step 示例执行说明

拿到一张八字，严格按顺序跑一遍：

1. **第一步**：看日干——譬如"庚"，定主五行为金、阳金。
2. **第二步**：看月支——譬如"申"。查表：庚金生申月为**得令**，+3。再算交节后天数，若处于壬水中气当值段，修正 −1（因中气为水，泄金）。
3. **第三步**：查四地支——譬如"酉、申、寅、巳"。酉为金本气旁根 +2；申为金本气正根（日坐下）+3；寅无金根 0；巳藏庚（中气）为金中气旁根 +1。根分合计 +6。
4. **第四步**：数天干——譬如"庚、丙、壬"（除日干外）。庚为比肩 +1，丙为七杀 −1，壬为食神 −1。天干分 −1。
5. **第五步**：S = 3 − 1 + 6 − 1 = **+7**。落入 `7 ≤ S < 10` 区间 → **身旺**。
6. **第六步**：若行巳午火运，火克金，原旺可转中和偏强；若行亥子水运，泄金过甚，可转中强偏弱。

## 五、特殊情形

### 身极旺 → 转专旺格候选

- 条件：得令 + 多本气强根 + 印比占多数 + 局中无有力异党或异党全部被合化 / 冲破。
- 取用反转：不再抑身，改为**顺其旺势**——用印比生助或食伤泄秀，忌官杀来逆。

### 身极弱 → 转从格候选

- 条件：日元全局无本气根、无中气中强根（有也被冲破）、无天干有力印比贴身生扶，异党占绝对多数。
- 取用反转：不再扶身，改为**顺其弱势**——**从财、从官杀、从食伤**或**从势**。

### 真从 vs 假从的边界

| 类型 | 条件 | 特征 |
|---|---|---|
| **真从** | 日元彻底无根、异党纯粹无杂、无印比来救 | 顺之则吉 |
| **假从** | 日元有余气弱根、或天干挂印比无根、或异党中夹杂同党 | 顺之半吉，带反复 |

**关键判据**：命局中**有无化神为日元印比的六合、三合、三会局**——有则通常打破从格，回归身弱论；无则可成从。

### 身中 + 用神清 → 正格最常见

- 中和的八字是**最标准的用神取用模型**。身强取克泄耗，身弱取印比扶。
- 中和偏强 / 偏弱的判断有时非常难，可结合评分法 + 合冲关系综合考量。

## 六、常见误区

1. **只看月令、忽视地支根**——月令固然权重最大，但无根的"得令"只是表面风光，日元虚不受补。
2. **不分本气 / 中气 / 余气**——将三级根等同视之，会把中气、余气的弱根当强根用，结论全盘偏差。
3. **混用"身强"与"身旺"**——身强靠"数"（同党压倒异党），身旺靠"根"（本气强根 + 得令）。**身旺必身强，身强未必身旺**。
4. **不做大运流年动态调整**——原命局是静态照片，真实强弱随岁运起伏。只看原局不看动态，必错过关键转折。
5. **把"无根 + 天干印比成堆"误判为身强**——无根则不受生，天干再多印比也只是漂浮。
6. **土日主忽视寒热燥湿**——辰湿土、丑冻土对戊己帮扶差；未火土、戌燥土才是真帮身。金日主同理（湿土生金，燥土不生）。
7. **把余气弱根当得地**——真正的"得地"至少是**一个中气的中强根**；只有余气弱根，应视为不得地。

## 七、实战口诀 / 关键记忆点

- **"月令三成，四支三成，干透四成"**——月令权重最大但不独大，地支根与天干势平分秋色。
- **"无根不受生，虚强一场空"**——日元无本气根，天干印比越多越漂。
- **"本气为强、中气为中、余气可忽"**——三级根力差距巨大，不可混同。
- **"身旺必身强，身强未必身旺"**——"旺"要根，"强"要数。
- **"得令三分贵，大运再添威"**——原局旺衰定底色，岁运起伏定当下。

## 八、一句话总结

**先查月令定大势，再点地支查根气，后数天干论同异，末归岁运定当下**——**四维定分，分数定级**，即可判尽天下八字身强弱。
