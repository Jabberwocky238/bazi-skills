# 日照江河

## 成立条件
1. **日主为丙火**（必须）：辛日、丁日都不是此格。
2. **丙火有根 / 透干清秀**：
   - 丙火坐地支火根（寅、午、巳）。
   - 丙火力量不能过弱。
3. **命局水旺且流通**：
   - 壬癸透干。
   - 地支亥子丑 / 申子辰合成水局。
   - 水气**有进有出、不堵不滞**。
4. **无土拦水**：
   - 土重 → 塞水 → 水不流通 → 日照的画面破。
   - 适度土承载无妨。
5. **丙火不被水灭**：
   - 水虽多，但丙火要能立于水上（有根有透）。
   - 若丙火被水彻底压死 → 转为"水多火弱"的破局。
