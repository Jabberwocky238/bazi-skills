# 职业判断

## 定位
- 本篇是**职业方向**维度的综合索引——把十神、身强身弱、大运、内外环境几条线串起来，给出完整的判断流程。
- 职业是八字最常被问到的议题之一，但也是最容易被**"盲目跟风、无脑追热门"**坑到的维度。
- 相关细节分散在：

## 一、基本思路：顺本心、不凑热门

### 多元路径、分层选择
- 一个人的八字通常具有**三条主要路径**，对应年柱、月柱、时柱。
- 除三条主路径外，还会派生出**无限多的小路径**。
- 离开这些路径去选职业 → 事倍功半；按路径而行 → 事半功倍。

### 四大基本原则
1. **就近原则**：天干地支的相互作用以"贴近"为优先。
2. **混沌流通**：流通不是贪吃蛇式的顺生，而是在混沌中就近发生。
3. **不确定性通路**：多条备选路径同时存在。
4. **分离用神**：事业、身体、婚姻、学业的喜用不一定重合。

### 不要无脑追热门
- 带硬性结构的八字可以无脑追国运行业（房地产、计算机、新能源）。
- 但若本身没有该行业对应的路径 → 硬上热门只会做得很难受、收入也不匹配。
- 每个人的八字都有独特的优势——识别这条路才是命理师的任务。

## 二、内环境 vs 外环境

| 维度 | 宫位 | 作用 |
|---|---|---|
| **内环境（心）** | 日柱地支 + 时柱 + 月柱地支隐干 | 心理特征、思考方式、偏好 |
| **外环境（社会）** | 年柱 + 月柱天干 + 时柱天干外层 | 社会表征、行业适配度 |

### 选择原则
- **优先按内环境选择**——心里不舒服的职业，走不远。
- 内外一致 → 内心舒适 + 外在顺畅。
- 内外矛盾 → 优先服从内环境 → 至少心理上过得去。

## 三、判断流程

### 第一步：看日主与月令的组合
- 日主在月令中的状态（临官、冠带、长生、墓库……）。
- 这一层决定了日主的**基础能量状态**。

### 第二步：看内宫位驱动
- 时干 → 月干的驱动决定做事动机。
- 内驱动是什么十神，决定你"为什么想做事"。

### 第三步：看外环境财官
- 年柱 + 月柱天干的财官结构决定外在发展路径。
- 外在财官是否顺畅，决定社会能不能接住你的能量。

### 第四步：看地支中气
- 各地支是否呈现主导特性、是否配合天干流通。
- 地支稳的人更容易"长期做一件事做成"。

### 第五步：看身强身弱
- 身强 → 能扛能独立，适合创业、管理、高压。
- 身弱 → 需要借力，适合体制、团队、合伙。
- 详见 [`身强.md`](./身强弱/身强.md) 与 [`身弱.md`](./身强弱/身弱.md) 的职业补充章节。

### 第六步：看十神主导
- 识别十神在原局里的主要排布——印旺、食伤旺、财旺、官杀旺、比劫旺。
- 对应的职业方向详见 [`十神/十神与职业.md`](../十神/十神与职业.md)。

### 第七步：看大运节奏
- 大运是否助推此路径？
- 顺运则出发，逆运则蓄力。
- 详见 [`大运流年/大运.md`](../大运流年/大运.md) 的职业节奏部分。

### 第八步：决定稳定还是创业
- 参考 [`稳定vs创业.md`](./稳定vs创业.md)。

## 四、典型解读示例

### 日主在月令冠带 + 内部食神驱动
- 性格偏旺、喜看正错误、孤气少许。
- 善于求知、心思全面。
- 推荐路径：
  - **学术 / 研究 / 基础学科教育**（上策，需考研读博支撑）。
  - **中高层技术主管**（中策，需培养领导能力）。
  - **中小学教师 / 监察类公职**（小路径，但稳）。

### 食神内驱 + 外部财官相生 + 贵人多
- 事业稳、官运无阻。
- 适合：
  - **崇尚型从商**（分蛋糕而非做蛋糕）。
  - **合伙创业**（由自己主导）。
  - **体制内中层管理**。

### 内部驱动软 + 外部财官难得
- 本人易被公司 / 机构剥削。
- 但可得官运。
- 需培养**领导风格** → 从"理性的软"转变为"成熟的硬"。

## 五、实务判断清单

按顺序检查：

1. **内环境是什么**：你的心是怎么想的？
2. **外环境是什么**：你在社会上的表现是怎样的？
3. **主做工是什么**：食神生财、伤官配印、官印相生、财官双显？
4. **能否落地**：这个做工能否**自然落在**你想做的行业里？
5. **大运配合**：十年运是否助推此路径？
6. **身强身弱**：独立承担 vs 借势发展？
7. **稳定创业**：制度内生长 vs 自己搭台？

## 六、路径失败的四种典型误区

### 误区 1：无脑跟风热门
- 八字偏文、选计算机 → 大学挣扎、考研再换语言学。
- 损失的是**黄金的四年**。

### 误区 2：强行从事高门槛路径
- 无印、无食伤，却硬要做学术研究。
- 三年博士读不下来、最终放弃。

### 误区 3：忽视内环境、只看外环境
- 外环境看似风光（金融、销售），但内环境偏学术。
- 做几年就抑郁、辞职。

### 误区 4：逆运期强行转行
- 大运走忌神时非要做大决策。
- 新行业入不去、旧行业回不去 → 双重损失。

## 七、一句话总结

- **职业选择不是凑热门，而是顺本心**。
- 内环境决定你能不能长期做下去。
- 外环境决定你能走多远。
- 主路径决定你的事业天花板。
- 小路径给你备胎与灵活性。
- **选对路径 = 人生的下限不会差，上限取决于大运与努力**。
