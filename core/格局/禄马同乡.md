# 禄马同乡

## 定义
- **禄**：十干**临官之地** —— 甲禄在寅、乙禄在卯、丙禄在巳、丁禄在午、戊禄在巳、己禄在午、庚禄在申、辛禄在酉、壬禄在亥、癸禄在子。
- **马**：**驿马星** —— 命局中根据年支或日支确定的一颗神煞，代表**流动、变化、远行**。
- **同乡**：落在同一个地支（或紧邻）。
- **禄马同乡**：日主的**禄位 + 驿马**同时出现在**同一地支 / 同柱** → **稳定根基 + 主动流动**两股力量**集中于一处**。
- 属**特殊贵格**之一，传统命理中被认为是**"动中取贵"**的结构。

## 驿马与禄的对应规则
### 驿马星（按年支或日支推）
| 三合局 | 驿马 |
|---|---|
| 寅午戌 | 申 |
| 巳酉丑 | 亥 |
| 申子辰 | 寅 |
| 亥卯未 | 巳 |

### 常见的禄马同乡组合
- **甲木日 + 寅中藏禄 + 寅为驿马**（如日柱甲寅 + 生于申子辰年）。
- **庚金日 + 申中藏禄 + 申为驿马**（如日柱庚申 + 生于寅午戌年）。
- **壬水日 + 亥中藏禄 + 亥为驿马**（如日柱壬亥 + 生于巳酉丑年）。
- 其他日主对应类推。

## 核心：稳中有动，动中有根
- 禄 = **稳定、根基、自立**。
- 马 = **变化、流动、远行**。
- 两者**本质相对**：
  - 单有禄 → 稳 → 但易守成不进取。
  - 单有马 → 动 → 但易漂泊无根。
- 禄马同乡 = **两者合一** → 既有根基又能远行：
  - **走得远、也能回得来**。
  - **变化大、但不动摇核心**。
- 传统命理中典型的"**动中求贵**"结构。

## 与相关结构对比

| 结构 | 关键 | 特征 |
|---|---|---|
| **禄马同乡**（本条） | 禄与驿马同地 | 动中有根 |
| **建禄格** | 月令为禄 | 身强自立 |
| **驿马独立** | 驿马无禄配 | 漂泊奔波 |
| **伏吟 / 自刑** | 重字自牵 | 原地打转 |
| **天罗地网** | 辰戌巳亥困锁 | 进退两难 |

禄马同乡是"**建禄**的升级版** —— 在自立的基础上加上流动的优势"。

## 成立条件（严格依《渊海子平》《三命通会》）

### 条件 1：日主的禄位落于命局某一地支

- **古籍出处**：
  - 《渊海子平·论禄》："**干禄于支——甲禄寅、乙禄卯、丙戊禄巳、丁己禄午、庚禄申、辛禄酉、壬禄亥、癸禄子**。"
- **客观判据**（十干禄位对照表）：
  - 日干在年 / 月 / 日 / 时四地支任一位上，遇到本干之禄位地支。
- **不满足则破**：日主无禄于命局 → 不具"禄"之半 → 不入本格。

### 条件 2：驿马星出现于命局地支

- **古籍出处**：
  - 《三命通会·论驿马》："**寅午戌马在申，申子辰马在寅，巳酉丑马在亥，亥卯未马在巳**。"
- **客观判据**（三合局对照）：
  - 按年支（传统主法）或日支（现代兼用）查驿马：
    - 年 / 日支属寅午戌 → 驿马为申；
    - 年 / 日支属申子辰 → 驿马为寅；
    - 年 / 日支属巳酉丑 → 驿马为亥；
    - 年 / 日支属亥卯未 → 驿马为巳。
  - 命局年 / 月 / 日 / 时任一地支上须见该驿马字。
- **不满足则破**：命局无驿马 → 不具"马"之半 → 不入本格。

### 条件 3：禄与驿马落于同一地支（"同乡"的本义）

- **古籍出处**（本条最核心、为本格定义）：
  - 《三命通会·论驿马》："**驿马与禄并居一宫，为之禄马同乡**。"
  - 《渊海子平·禄马赋》："**禄马同乡，身居富贵之地**。"
- **客观判据**：
  - 条件 1 所得的禄支与条件 2 所得的驿马字**为同一地支**。
  - 典型结构例：
    - 日主甲木 + 年支申 → 驿马寅 + 甲禄寅 → 寅为禄马同乡；
    - 日主庚金 + 年支寅 → 驿马申 + 庚禄申 → 申为禄马同乡；
    - 日主壬水 + 年支巳 → 驿马亥 + 壬禄亥 → 亥为禄马同乡。
- **不满足则破**：禄与马分居不同地支 → 非"同乡"，只作独立的"带禄""带马"论，不入本格。

### 条件 4：禄马同乡之地支不被冲、不被合化

- **古籍出处**：
  - 《渊海子平·禄马赋》："**禄马逢冲，功名无托**。"
- **客观判据**：
  - ① 禄马所在地支与其他地支无六冲（如禄马在寅，全局无申来冲；禄马在亥，全局无巳来冲）；
  - ② 该地支不因三合 / 三会 / 六合而合化为他气。
- **不满足则破**：冲者禄马散；合化者根基转移——《渊海子平》"功名无托"即指此败局。

### 条件 5：日主不至极弱（能承载禄马之"动"）

- **古籍出处**：
  - 《三命通会·论驿马》："**马忌身衰**——身弱而马旺，奔波劳苦。"
- **客观判据**：
  - 日主于月令至少有比印余气，或年 / 月 / 时有一位比劫 / 印星通根。
- **不满足则破**：身极弱而禄马同乡 → 动而无力，《三命通会》"马忌身衰"——奔波而无成。

---

**五条同时满足**方为真禄马同乡。本格核心在一"同"字——禄（古籍定义甲禄寅等）与马（古籍定义寅午戌马申等）必须落入同一地支，方得"稳中有动、动中有根"的气机结构。

## 判断维度

### 1. 禄马位置
- **日支禄马同乡**：自身直接承担，发力最强。
- **月支禄马同乡**：中年事业期爆发。
- **时支禄马同乡**：晚年发力或子女 / 后代受益。

### 2. 是否被冲 / 合
- **冲**：禄马散 → 格破。
- **合**：禄马被合走 → 根基转移 → 视合化之神是否有利。
- **最稳状态**：禄马紧贴 + 无冲无合。

### 3. 大运流年
- **驿马被引动的大运 / 流年**：禄马兑现 → **远行 / 跳槽 / 迁居 / 出国 / 升职**。
- **冲破禄马的大运**：格瓦解 → 人生动荡。

### 4. 命主是否接受"动"
- 禄马同乡带来的贵是"**动中得**"。
- 若命主执着守静 → 禄马的福气兑现不了。
- 反之顺着命局的流动能量 → 人生节节高。

## 常见误区

### 1. "有禄 + 有马 = 禄马同乡"
- 必须**同地支 / 紧贴**才叫"同乡"。
- 禄在月、马在时 → 不紧邻 → 不成格。

### 2. 把禄马同乡等同"一定出国 / 远行"
- 驿马的"动"有多种形式：
  - **身体的动**：出国、远行、搬迁。
  - **职业的动**：跳槽、换行业、晋升。
  - **信息的动**：进入高速流动的领域（互联网、媒体、贸易）。
  - **心智的动**：学习、视野开阔、思维升级。
- 不一定是物理意义的远行。

### 3. 以为动就是漂泊
- 禄马同乡的"动"是"**动中有根**"：
  - 不是无目的的漂泊。
  - 是**带着方向的流动**。

### 4. 忽视冲克对禄马的破坏
- 禄马最怕"**冲**"。
- 大运流年出现冲禄马的字 → **那一年易有重大变动**（出事 / 意外 / 突变）。

## 现实表现

### 性格
- **独立而灵活**：能扎根也能离开。
- **行动力强**：想做就去做、不拖延。
- **视野开阔**：不被一地一景局限。
- **适应力强**：新环境新人群都能快速上手。
- **骨子里有"闯"的基因**。

### 事业路径
- **跨区域 / 跨城市工作**：跨国公司、外派、区域经理。
- **商贸 / 物流 / 运输**：字面意义的驿马。
- **旅游 / 酒店 / 航空**：行业天然流动。
- **互联网 / 媒体**：信息流动型。
- **外交 / 翻译 / 国际合作**：跨文化流动。
- **创业者**：不惧变动、敢于扩张。
- 共性：**需要"流动 + 适应 + 扩展"**的岗位。

### 典型画像
- 年轻时离开家乡谋发展。
- 中年在多地拥有资源 / 事业 / 房产。
- 晚年仍然活跃、不愿完全静下来。
- **走得越远 / 见得越多，越有成就**。

### 家庭
- **异乡成家**：多半在出生地之外成家立业。
- **家庭支持远方发展**：父母或伴侣理解并支持流动。
- **子女视野受益**：因父母的流动，子女往往视野开阔、更早独立。

### 身体
- 禄马同乡 + 适度流动 → 身体活力好。
- 但若**流动过度、休息不足** → 慢性疲劳、颈椎 / 腰椎问题（长期出差 / 坐车 / 飞行的代价）。

## 化解与兑现

### 兑现禄马同乡
- **主动顺应流动的命格**：
  - 不要强求原地 / 强求稳定。
  - 允许人生有多段不同地点的经历。
- **在每一段流动里都扎下小根**：
  - 新城市买房 / 认识本地朋友 / 理解新文化。
  - 这是"**禄**"的那一面 —— 动中扎根。

### 避免禄马散
- **保护禄马**：观察大运流年是否有冲，**冲的年份减少重大变动**。
- **合化的年份**：看合化是否对自己有利；有利则顺势、不利则稳守。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 驿马 / 禄马引动的年份 | **喜** → 远行 / 跳槽 / 晋升 / 大动作 |
| 财官生旺禄马的大运 | **喜** → 动中得财 / 得位 |
| 比劫印禄的大运 | **喜** → 根基加强 |
| **冲破禄马的大运** | **大忌** → 动中破 → 意外 / 损失 |
| 合走驿马的大运 | 视合化 → 合好则稳；合忌则绊 |

## 与相关格局的关系

| 格局 | 关键 | 特点 |
|---|---|---|
| **禄马同乡**（本条） | 禄与马同地 | 动中有根 |
| **建禄格** | 禄在月令 | 自立稳重 |
| **羊刃格** | 刃在月支 | 身极旺 |
| **驿马独旺** | 驿马多现 | 漂泊奔走 |
| **魁罡格** | 特殊日柱 | 刚烈将帅 |

禄马同乡强调的是"**流动中的稳定**" —— 不是建禄格的纯静、也不是驿马独旺的纯动，而是两者结合。

## 核心判断
- 禄马同乡不是"一定远行"，是**"流动中有根"的人生结构**。
- 识别后 → **允许自己顺着"动"的能量走**：
  - 不强求守在原地。
  - 也不无目的漂泊。
- 成格时：**动而贵、动而稳、动而有成**。

## 一句话总结
- 有根不怕走、会走不怕远。
- 禄马同乡 = **走出去是福气、走回来还有家**。
- 适合这样结构的人：**不要害怕离开 —— 流动本身就是你的贵人**。
