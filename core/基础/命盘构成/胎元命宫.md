# 胎元、命宫、身宫

## 三元溯源

胎元、命宫、身宫的概念源自盲派命理的**三元**——八字除四柱之外还有**三元**，盲派命理中三元被视为"**真我**"，重要性不可忽视。

三元的原始定义是**紫微元、太微元、天使元**，但在命理学中分别代表：**胎元、命宫、身宫**。

通过排盘软件，专业排盘抬头的"胎、命、身"按钮可以在原来大运、流年、流月的位置显示胎元、命宫、身宫这三柱。

身宫的重要性相较于胎元和命宫没那么大，此处不作为重点。

---

## 胎元

### 基本概念

**胎元**即人受精怀胎的月份——此时人处于母胎之中，刚刚形成胚胎雏形，因此称为胎元。四柱以人出生时间推测命运，而**胎元作为受精怀胎时间，自然也会对命运产生影响**。

### 重要性

胎元被认为与**健康、寿命、祖宗福德**有关。命理典籍《预兆定真经》中胎元被置于与**本命年柱同等重要**的地位，特别是它与年柱的关系可以揭示许多事情——在一些命书古籍中，**胎元常常被当作四柱之外的第五柱**。

### 推算方法

胎元通过**月柱**推算：
- **出生月的月干后数一位** = 胎元天干。
- **出生月的月支后数三位** = 胎元地支。

### 口诀

> **胎元起法记心中：生月干支要记清；月干后一为胎元干，月支后三支配成。**

### 示例

命主出生于**乙亥**月：
- 月干乙木 → 后数一位 → **丙**。
- 月支亥水 → 后数三位：亥 → 子 → 丑 → **寅**。
- 胎元 = **丙寅**。

### 胎元的应用

#### 一、胎元空亡 → 身体不佳、花钱较大
判断空亡需**以日柱空亡为主**——若不落日柱空亡再考虑年柱空亡，其他柱不再考虑。

#### 二、胎元冲年 → 父母不全
年代表父母，胎元冲年 → 父母中一人可能不在或早逝。

#### 三、胎元与年柱同位 → 父母不正
"不正"可理解为生活作风有点**不正派**（花心、爱赌、或有点邪气）。

#### 四、胎元为阴支冲 → 母亲可能被抱养
胎元为阳支冲 → 父亲可能被抱养。

#### 五、胎元遇刑冲穿破

| 情形 | 类象 |
| --- | --- |
| 出生艰难、体质不佳 | 可能难产或未足月提前出生，导致体质不好 |
| 胎元受冲 → 奔波劳碌 | 胎元被本命局地支冲克 |
| 胎元遇刑冲穿绝、受伤较重 | 人生会有很多无法言说的缺憾 |

## 命宫

### 基本概念

**命宫**指人出生时在**东方升起的星座**——主宰一个人的**天赋才能和天生性格**。命宫在人体中对应**大脑**，掌管思想和行为。

《三命通会》记载："神无庙无所归，人无事无所欺，命无功无所主，故有命宫之说。不然流年星辰为凶为吉，以何凭据？"

**盲派命理认为**：命宫就如同人所居住的房子，**是评定吉凶的基点**。

### 推算方法（以月支数 + 时支数为基础）

#### 第一步：十二月建排序

以十二月建为地支排序：
| 月支 | 数字 |
| --- | --- |
| 寅 | 1 |
| 卯 | 2 |
| 辰 | 3 |
| 巳 | 4 |
| 午 | 5 |
| 未 | 6 |
| 申 | 7 |
| 酉 | 8 |
| 戌 | 9 |
| 亥 | 10 |
| 子 | 11 |
| 丑 | 12 |

#### 第二步：中气修正

若**出生日期已过本月中期（中气）** → 按**下一个月**算（序号 + 1）。

例：出生在寅月，但日期在**雨水之后** → 按**卯月**来算。

#### 第三步：计算公式

- 以**月支数 + 时支数**为 A。
- 若 A ≤ 14 → 命宫序号 = **14 − A**。
- 若 A > 14 → 命宫序号 = **26 − A**。
- 所得序号对应的地支 = **命宫地支**。

#### 第四步：命宫天干

命宫的天干按**年上起月法**确定（见《排盘.md》）。

### 示例

某命主亥月子时出生：
- 月支亥 = 10，时支子 = 11 → 相加 = 21 > 14 → 26 − 21 = 5 → 对应**辰**。
- 但需**检查中气**：立冬（亥月起始）11 月 8 日 → 小雪（亥月中气）11 月 22 日 → 大雪（亥月结束）12 月 7 日。若出生日期 **12 月 3 日** → 已过亥月中气且接近结束 → 应**按子月算**（月支数 +1 = 11）。
- 修正后：月支数 11 + 时支数 11 = 22 → 26 − 22 = 4 → 对应**卯** → 命宫为卯宫。

## 命宫、身宫的应用（综合）

### 与空亡、刑冲、合贵的组合

| 组合 | 象义 |
| --- | --- |
| **命宫、身宫空亡** | 低调处世，如隐世居于山林，然努力常徒劳无益 |
| **命宫、身宫遇刑冲穿绝** | 劳碌奔波，纵然命吉易惘然；体质欠佳，易遭手术车祸之苦 |
| **命宫、身宫遇合、得贵** | 贵人相助，如虎添翼 |

### 示例

某极弱的从财格：
- 大运配合得当 → 财运尚可。
- 命宫在未宫 → 一生辛劳，要离开祖籍才能安定。
- 胎元落空亡 → 身体不好、开销较大。
- 命宫落空亡 → 为人较低调。
- 身宫亥水与年柱巳火相冲 → 经常劳碌奔波。

## 核心总结

| 三元 | 概念 | 推算起点 | 主断方向 |
| --- | --- | --- | --- |
| **胎元** | 受精怀胎月份 | 月柱（月干后1、月支后3） | 健康、寿命、祖宗福德、父母关系 |
| **命宫** | 出生时东方升起星座 | 月支 + 时支（需中气修正） | 天赋才能、天生性格、吉凶基点 |
| **身宫** | 辅助宫位 | （与命宫相对） | 辅助参考 |

三元虽只起**辅助作用**，但在实际应用中留意它们，有时能在某些方面发挥意想不到的效果。
