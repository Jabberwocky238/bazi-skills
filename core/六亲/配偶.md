# 配偶

> 参见：`六亲/六亲总论.md`、`婚姻/十神择偶.md`、`婚姻/合盘.md`、`六亲/情人外遇.md`

## 一、核心对应（男女有别）

### 男命

- **妻星**：**正财**（主 = 正妻）/ 偏财（偏妻 / 情人）
- **妻宫**：**日支**（夫妻宫）
- **双线判断**：妻星 + 妻宫 **二者合一**方为清晰。

### 女命

- **夫星**：**正官**（主 = 正夫）/ 七杀（偏夫 / 情人）
- **夫宫**：**日支**（夫妻宫）
- **双线判断**：夫星 + 夫宫 **二者合一**方为清晰。

## 二、星宫配合的四种格局

| 结构 | 男命 | 女命 | 象意 |
| --- | --- | --- | --- |
| 星入宫 | 正财在日支 | 正官在日支 | 配偶位置清晰、婚姻稳 |
| 星不入宫 + 星为喜 | 正财透在他柱 | 正官透在他柱 | 配偶贤但非在身边 / 晚婚遇 |
| 星入宫 + 星为忌 | 日支正财为忌 | 日支正官为忌 | 配偶干扰、关系紧 |
| 星不见 / 星被克尽 | 无财 / 财被比劫劫尽 | 无官 / 官被食伤克尽 | 配偶缘薄、晚婚或孤 |

## 三、男命：妻的判断

### 妻星（正财）

| 命局结构 | 象意 |
| --- | --- |
| 正财为喜 + 月支临天乙贵人 | 娶贤妻、婚姻幸福 |
| 正财林文昌贵人 | 娶才女 / 名门女 |
| 日支正财 + 无刑冲 | 妻贤淑、善理财、占有欲强 |
| 正财入墓无冲 | 妻守财 / 内敛，需冲库方发 |
| 比劫透干争财 | 妻病 / 妻多灾 / 夺妻风险 |
| 偏财为主 + 正财不见 | 偏妻 / 情人成正位 |
| 时干正财 + 晚婚 | 娶少妻、或结婚较晚 |

### 妻宫（日支）

- **日支比肩 / 劫财** → 夫妻如朋友 / 脾气犟 / 婚前同居或奉子成婚。
- **日支伤官** → 克夫 / 克妻星、外貌协会、心性不稳。
- **日支正印 / 偏印** → 夫妻相敬如宾但平淡，妻母性强。
- **日支食神** → 妻善良懂生活、会享受。
- **日支七杀** → 妻急性子、配偶强势、易争执。

## 四、女命：夫的判断

### 夫星（正官 / 七杀）

| 命局结构 | 象意 |
| --- | --- |
| 日干与月干正官相合 + 独官无杂 + 为喜 | 嫁贵夫、夫妻和睦 |
| 日支正官 + 无形客 | 嫁能夫、秀外慧中 |
| 正官被伤官克（伤官见官） | 夫运不佳 / 生离死别 |
| 正官被征合 / 官杀混杂 | 感情波折、婚姻反复 |
| 日支七杀 | 配偶急躁强势、与命主势均力敌易冲突 |
| 月支七杀 + 食神制 | 食神制杀贵格，夫寿富贵 |
| 七杀为主 + 正官不见 | 偏夫 / 情人型婚姻结构 |

### 夫宫（日支）

- **日支正官** → 夫贤、配偶端庄有文化。
- **日支伤官** → 夫星受制，婚缘不佳、离异率高、丈夫身体易有疾。
- **日支比肩 / 劫财** → 夫妻如朋友 / 有同性恋倾向 / 婚前同居。
- **日支食神 / 伤官** → 心直口快，夫星受泄或受克。
- **日支正印 / 偏印** → 得夫呵护或依赖过度。

## 五、配偶宫的冲 / 合 / 刑

### 冲

- **日支被月支冲 / 被时支冲 / 被流年冲** → 婚姻宫动摇：吵架、分居、离异、或搬家、甚至另一半生病。
- **子午、卯酉、寅申、巳亥** 四正四隅之冲，力量最大。
- **年日双冲** → 配偶与原生家庭对立。

### 合

- **日支与他支合** → 配偶易被合走（外遇、被夺），亦可为夫妻亲密、共谋一事。
- **日支合大运或流年之地支** → 桃花、迁居、异性介入的应期。
- **日支自合（如丁亥、戊子、壬午等暗合日）** → 内心多情 / 容易暗恋、桃花自来。

### 刑

- **日支受刑** → 配偶身体 / 情绪困扰，婚姻关系纠缠不清。
- **寅巳申三刑、丑未戌三刑** 入日支 → 配偶易有官非、病灾。

### 害

- **日支逢六害** → 夫妻有暗中不满、口舌、隔阂。

## 八、古籍补充（作为真实标准）

- **女命月干正官与日干相合（如甲日逢己土）+ 为喜用 + 独官无杂 → 多嫁贵夫，夫妻和睦**。
- **日支正官 → 男命配偶贤惠顾家、女命嫁贵夫难得贤妻**。
- **女命月支正官 + 不逢刑冲 → 婚姻稳定、难得贤妻、女嫁能夫**。
- **日支七杀 → 配偶脾气差、家庭欠和谐、双方都不愿妥协**。
- **日支伤官 → 女命婚缘不佳、离异率高、丈夫身体有疾**。
- **日支正财为喜 + 无形冲 → 配偶贤淑、善理财、财运在中年达高峰**。
- **丙子、丁亥等日坐正官带桃花 → 亦有婚前同居或奉子成婚**。
- **己为日生者（己未日）日支未土为正官 + 众藏乙木七杀 → 夫位官杀混杂，间接消耗代表父亲的财星**。
- **孤鸾日（乙巳、丁巳、辛亥、戊申等）→ 主婚姻孤淡 / 晚婚**——八字基础反复示警。

## 九、关于婚姻的根本理解

八字所呈现的从来不是"一个配偶的样子"，而是**"命主与配偶之间力量如何流动"**：

- **星宫一致 + 喜用明确 + 宫位无冲** → 婚姻坦途。
- **星宫分离 / 喜忌颠倒 / 宫位受冲** → 婚姻多波折。
- 但**命理显示趋势，不决定结局**——婚姻质量仍取决于双方经营与自身修为。

## 十、一句话总结

**男命以财为妻、日支为宫；女命以官为夫、日支为宫——星宫合位则婚姻安稳，冲合刑破则感情动荡；配偶是你命气向外延伸的镜像，读懂自己，才读得懂另一半。**
