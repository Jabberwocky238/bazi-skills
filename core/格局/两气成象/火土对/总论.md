# 火土对（总论）

## 性质：相生两气对（火→土）
- **五行关系**：火生土（印→身 或 身→食伤的顺流）。
- **属"印生两神成象格"之一**——《滴天髓》五种相生组合中的"火土相生格"。
- 相生两气双清，若气纯无杂，属贵格。

## 两气成象通则（适用于本对）

### 成立条件（严格依《滴天髓》两气双清论）
1. **命局"两气双清"**——只见火土两种五行。
2. **火生土**——火为母、土为子，顺流无克。
3. **原局无财官**（水为财/官在特定日主、金为食伤）——一见财官则破局。
4. **日主属火或土**，且不过弱。

### 具体形式（火土相生格）
- **丙丁火日主 + 戊己土食伤**：身→食伤顺流（火生土成气）。
- **戊己土日主 + 丙丁火印绶**：印→身顺流（火土夹带）。

### 气质
- **温厚 + 包容** —— 厚德、稳重、有慈爱、有权威、重承诺。
- 典型特征：**稳如泰山** —— 不张扬但让人有安全感。

### 典型职业
- 管理、建筑、地产、实业、政治、慈善、宗教、教育管理、大型机构。

### 禀赋与缺失
- **禀赋**：温厚、包容、稳定、有权威、有耐心。
- **缺失**：
  - 缺金 → 缺决断、缺变现力（财气不现）。
  - 缺水 → 易僵化、易燥、易刻板。

### 贵格 vs 病象
- **火土夹带**（贵格）：火生土流畅、温厚厚德之象。
- **火炎土燥**（病象）：火过旺烤土 → 土焦裂 → 气机断、性格急躁。

### 喜忌
- **喜**：火（加强本气）、土（加强本气）、**金**（通关泄秀，火→土→金成生生不息）。
- **忌**：
  - **水**（克火破局）——大忌，水透灭火则格败。
  - **木**（克土破局）——大忌。
- **调候之需**：本对偏燥（火土两旺），**微水润土不忌**（润而不克灭火）；但水过则灭火格毁。

### 古籍背书
- 《滴天髓·源流》："**两气双清，其象最妙**。"
- 《滴天髓》："**火土相生，敦厚之象**。"
- 《穷通宝鉴》："戊己得丙丁，**厚德载物**——火土同功。"

## 一句话总结
- 火土对 = **火生土之相生双清**——温厚、稳重、慈爱有权威。
- 贵在"**火生土而润**" → 火土夹带；忌"**火过燥烤土**" → 火炎土燥。
- 关键看**火土比例 + 微水润**：火适度生土 + 微水调候 = 贵。
