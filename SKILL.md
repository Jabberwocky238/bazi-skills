# 八字 / 传统命理解读 Skill

## 职责边界

- **只做解读，不做排盘**。
- 不根据阳历 / 阴历生日换算干支，不处理真太阳时、节气换月、早晚子时等排盘技术问题。
- 若用户只给出生日期时间、未给干支四柱，**先明确告知需由排盘工具生成四柱**，再使用本 Skill 解读。

## 输入要求

**八字类解读**：
1. 已排好的四柱八字（年柱、月柱、日柱、时柱的天干地支）
2. 性别（判断大运顺逆与部分十神取用；女命另见 `core/基础/女命看法.md`）
3. 起运岁数与大运干支序列（若涉及大运 / 流年分析）

**非命盘类解读**（易经、哲学议题）：
- 用户具体困境或问题描述即可。

## 适用场景

- 八字命局解读：日主强弱、喜忌、用神、调候、通关、流通（见 `core/基础/`）
- 十神结构：比劫 / 食伤 / 财星 / 官杀 / 印星 及相互对照
- 格局判断：正格（建禄、月令取用）、从格（从财 / 从杀 / 从势 / 专旺 / 稼穑 / 从革）、特殊格局（魁罡、壬骑龙背、木火通明、水火既济等 30+ 格局）
- 神煞判读（含贵人煞、凶煞、感情煞）
- 宫位分析（年月日时四柱、夫妻宫）
- 六亲关系（父、母、子女、配偶、情人）
- 婚姻专题（女命官杀、男女命无财、择偶、父母干涉等）
- 大运 / 流年 / 流月引动、当下时运（2026 丙午年、九紫离火运等）
- 易经卦辞解读（乾坤两卦）
- 紫微斗数星曜（当前已整理星曜有限）
- 七政四余基础与对比八字
- 命理哲学：阴阳、天道、对立统一、因果、福报显化等底层框架

## 解读流程

### A. 八字命局类
1. **接收四柱** — 确认已提供年月日时干支与性别；缺失则先要求补全。
2. **基础判断** — 月令司权、日主强弱、喜忌与用神（`core/基础/`）
3. **十神标注** — 以日主为基准（`core/十神/`，含 vs 对照文件）
4. **天干地支作用** — 天干五合虚透（`core/天干/`）、地支刑冲合穿（`core/地支/`）
5. **格局判定** — 优先检索 `core/格局/` 中是否成格；含正格 / 从格 / 特殊格局 / 五行象格四大类
6. **神煞标注** — 仅采用 `core/神煞/` 已整理的神煞，贵人与凶煞兼看，避免堆砌
7. **宫位与六亲** — `core/宫位/` + `core/六亲/`
8. **婚姻专题**（若涉及）— `core/婚姻/`，结合 `core/宫位/夫妻宫.md`
9. **大运 / 流年引动**（若提供）— `core/大运流年/`，含当下 2026 丙午年、九紫离火运等时运文件
10. **综合解读** — 贴合用户咨询方向，引用知识库原文口径

### B. 易经类
- 按卦 / 爻索引 `core/易经/`；结合 `core/哲学/` 提供世界观框架

### C. 紫微斗数类
- 目前仅整理少量星曜与专题，超出范围明确告知"紫微斗数知识库仅整理了少量主题"

### D. 七政四余类
- 介绍性问题引 `core/七政四余/入门.md` 与 `七政四余.md`
- 对比八字时引 `八字与七政对比.md`
- 深度解读目前范围有限，遇细节超出明确告知

### E. 哲学 / 议题类
- 不必先有命盘，直接索引 `core/哲学/`

## 重要原则

- **以知识库为准**：解读论理必须与 `core/` 中 md 内容一致；不引入未整理的流派观点。
- **多维度叠加**：复杂问题常需跨目录综合（如"婚姻困局" = 婚姻 + 夫妻宫 + 大运流年 + 格局 + 六亲）。
- **格局优先于堆砌**：八字解读若能定格，远优于罗列十神神煞。
- **不贴标签**：如 `core/神煞/桃花.md` 强调"桃花 ≠ 异性缘"；`core/十神/财星vs印星.md` 等对照文件是破单标签解读的关键资源。
- **不做排盘技术判断**：早晚子时、节气换月、真太阳时等均交上游排盘工具。
- **遇知识库未覆盖内容**：明确告知"该部分知识库暂未整理"，避免臆断与幻觉。
- **中立表达**：不作绝对化断言，不宣扬迷信，健康 / 重大决策类话题明确提示仅供参考（见 `DISCLAIMER.md`）。
