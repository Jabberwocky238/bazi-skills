# 水火对（总论）

## 性质：相克两气对
- **五行关系**：水克火（官杀 / 财关系）。
- **非"印生两神成象格"**——该格专论相生两气，水火属相克，不入双清贵格。
- 属"两气成象"的**相战 / 对立**一支，但有特殊贵格"水火既济"。

## 两气成象通则（适用于本对）

### 成象要件
- 命局只见水、火两种五行。
- 水、火力量大致相等。
- 月令为水或火之一。

### 与相生对的根本区别
- **相生对**：两气顺生、气通为清。
- **水火对（本条）相克**：水克火、气机对立——是最激烈的相克对（冰火两极）。

### 特殊情况：水火既济（易经贵格）
- 《易经·既济卦》"**水在火上，既济**"——水火虽克，若有**木通关**（水→木→火），则化战为济。
- 条件：
  - 水火势均力敌。
  - **必有木通关**（木在水火之间）——这是本格的核心。
  - 无金强克木。
- 水火既济 = 相克两气得木通关后的**平衡贵格**——《易经》评为"亨"。

### 特殊情况：日照江河（丙火专用贵格）
- 丙火日主 + 水成江河 + 丙火有根 → **日照江河**之象。
- 条件：
  - 日主必为丙火。
  - 水成江河（申子辰三合 / 多透壬）。
  - 丙火根不被水彻底灭。

### 病象：水火相战
- 水火势均力敌 + 无木通关 → 纯相战 → 冲突不断、格败。
- 现实：内耗严重、两极分化、矛盾尖锐。

### 取用要点
- **喜木通关**（水→木→火）→ 化战为济。
- **喜土调和**（土克水护火，但土克水本身也是克）→ 须辨情势。
- **忌金强**（金克木断通关，或金过泄水→水弱火胜）。
- 关键看**水火平衡 + 通关之木**。

### 古籍背书
- 《易经·既济卦》："**水在火上，既济**——亨，小利贞。"
- 《滴天髓·通神论》："**水火相见，贵乎相济；相济者两停，不相济者两伤**。"
- 《穷通宝鉴·丙火》："**丙火见壬水，水辅阳光，为江辉之象**。"
- 《滴天髓》："**两气相战，得通关之神，则两益**。"

## 本对下属格局
- **水火既济**：水火得木通关的平衡贵格。详见 `水火既济/`。
- **日照江河**：丙火 + 水成江河的贵格。详见 `日照江河/`。
- **水火相战**：水火无通关的冲突病象。详见 `水火相战/`。

## 一句话总结
- 水火对 = **相克两气、最激烈的两极**。
- 水火既济（木通关）→ 贵格。
- 水火相战（无通关）→ 病格。
- 关键在**木**——水→木→火则济、无木则战。
