# 月德合

## 定义
- **月德合**：与**月德贵人**构成**天干五合关系**的天干——月德之气的"合煞神"。
- 本质：**承接月德之气、辅助月德贵人**的辅助型神煞。
- 与月德贵人**配对出现**，但独立成神煞——非月德贵人的别名或延伸。

## 月德贵人 vs 月德合

| 对比项 | 月德贵人 | 月德合 |
|---|---|---|
| 起法 | 按三合局查固定天干 | 与月德贵人天干五合之字 |
| 性质 | **现世人事加持的主神** | **合月德之气的辅神** |
| 作用 | 日常事务润物无声的化解 | 承接月德之气、间接受福 |

**关系**：月令三合局定月德 → 与月德相合之天干 = 月德合。两者**同气相接**——月德合到位时，月德贵人之气被"合住、留住"。

## 天德合 vs 月德合

| 对比项 | 天德合 | 月德合 |
|---|---|---|
| 主神 | 天德贵人（天时加持）| 月德贵人（人事加持）|
| 起法 | 月支查天德 → 查合神 | 三合局查月德 → 查合神 |
| 性质 | 合天时福气之锁 | 合人事福气之锁 |
| 作用 | 关键时刻的天时缓冲锁 | 日常事务的人事缓冲锁 |

**双合并现**：**天德合 + 月德合**同现 → 古籍谓"**双合护命**"，命中福气全面锁定。

## 核心作用：承接月德之气
- 月德贵人是"**现世人事的润物之福**"——日常层面的化解。
- 月德合是"**把月德之气留住**"的辅助：
  - 合月德贵人之天干 → 贵人之气不散、不被冲破。
  - 即便命中无月德贵人，只要有月德合出现 → 仍受月德之气的间接加持。
- 古籍论："**月德合入命，人事圆融，小险化无**。"

## 判断维度

### 1. 月德合单独出现 vs 与月德贵人并现
- **单独月德合**：承接月德之气 → 日常福气间接兑现。
- **月德合 + 月德贵人并现**：**双重人事福护**——做人做事皆有缓冲。

### 2. 位置
- **年柱月德合**：家族和睦、早年人事顺。
- **月柱月德合**：成长期合月德之气——典型位置。
- **日柱月德合**：本人为人和顺、贴身福气。
- **时柱月德合**：晚年 / 子女人事圆融。

### 3. 配合其他贵神
- **月德合 + 天德合**：双合护命——福气完整锁定。
- **月德合 + 月德 + 天德 + 天德合**："**四德全备**"——传统命理中极贵的标记。
- **月德合 + 天乙贵人**：人事缓冲 + 命运修正双重。

### 4. 是否被破坏
- 月德合天干被合化为异神 → 合力转移。
- 被冲克 → 合力减弱。

## 常见误区

### 1. "月德合 = 月德贵人"
- 最大误判。两者起法不同、性质有别。
- 月德贵人是"**主神**"（直接福气）；月德合是"**合神**"（锁定福气）。

### 2. 把月德合视作"次级月德"
- 月德合的力量**单论时不如月德贵人直接**。
- 但"合"的本质是"**留住 + 承接**" → 与月德贵人并现时产生叠加效应。

### 3. 忽视"合"的方向性
- 月德合是"**天干相合**"——必须**天干位置上真的相合**才起作用。
- 月德合字在地支藏而不透 → 合力大减。
- 月德合字紧贴月德贵人 → 合力最强。

### 4. 混淆天德合与月德合
- 天德合按月令每个月单列的天德之字查其合神。
- 月德合按三合局的月德之字（丙/壬/甲/庚四种）查其合神（辛/丁/己/乙四种）。
- 两者起法、作用都有区别。

## 现实表现

### 性格
- **和顺、圆融、有福相**。
- **善于与人合作**——"合"的气场让人际关系自然顺畅。
- **不激进、不尖锐**：处事有温度。
- **做事有"**巧劲儿**"**——不硬冲、能找到缓冲之法。

### 典型情境
- **日常事务多顺心**：签合同、出行、交易、合作多有缓冲。
- **人际关系和睦**：家庭、同事、邻里关系好。
- **遇小麻烦能自然化解**：该出的问题出一点就打住。
- **与他人合作时"合拍"**：合伙、联姻、团队都配合得好。

### 与月德贵人并现的典型
- 一生人际关系顺畅。
- **日常小福气不断**：该得的奖金、该成的合作、该顺的小事都顺。
- **身边常有"**合得来**"的人**：朋友不多但都舒服。

## 核心判断
- 月德合是"**月德之气的锁**"——把人事福气锁住。
- 与月德贵人并现 → 人事圆融牢不可破。
- 识别意义：这类命造**一生不惊天动地，但日常福气稳稳**——"**温吞但不漏**"是其典型命运画风。

## 一句话总结
- 月德贵人 = 润物无声的人事之福。
- 月德合 = **把这份福气"**合住**"的那股合气**。
- 合字一到，福气就不跑 —— **这是月德合的本质**。
