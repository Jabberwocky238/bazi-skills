# 得势

## 定义
- **得势**：在八字整个格局中，对日主形成有利**生扶合力**的全局态势，简言之即"人多势众"。
- 得势主要由两部分组成：**得生**（他干支对日元的生扶）与**得助**（天干同类五行的帮扶）。
- 得势不针对单一柱位，而是对**全局五行势力**的综合评估。

## 核心原理
- 一个人强不强，不仅看自己出生的环境（令），不仅看自己有没有扎根（地），还要看**身边有多少人在帮忙**。
- "好汉三个帮"——得势就是这层盟友关系在命盘中的体现。
- 得生与得助的前提都是一个：**受生方/受帮方本身必须有根**。无根者不受生，这是得势成立的底线。
- 生方与助方自身也需有力，若力量不足，反而会因强烈泄耗自身而帮不动。

## 判断条件

### 得生
- **含义**：八字中的其他天干地支对日元有生扶作用。
- 从日元角度看得生；从月干、时干等其他天干角度看亦可同理推演。

#### 得生成立的两个条件
1. **受生方有根**：日元（或被生之干）必须在地支通根；无根则"不受生"。
2. **生方本身有力**：生方必须透干有根，力量足够；否则会被强势受生方泄耗殆尽。

#### 三种典型情境
| 情境 | 生方 | 受生方 | 判断 |
|---|---|---|---|
| 双方俱旺 | 透干且有根 | 透干且有根 | 得生成立，生扶有效 |
| 生方虚浮 | 仅天干孤露、地支无根 | 地支一片受生方 | 生方力竭被焚，不能成生 |
| 受生方无根 | 虽有中气根支撑 | 完全无根 | 受生方拒补，反受其害 |

- 无根者勉强受生，可能引发**水多木漂、金多水泛、土重金埋**等反效应。

### 得助
- **含义**：八字中与日元五行相同的**天干**之间的帮扶。
- 得助**仅针对天干**，与地支无关——这是与"得地/通根"的严格分界。
- 例：癸水日元，年干壬水、时干癸水同时出现 → 日元癸水天干得助。

#### 得助与得地的区别
| 概念 | 作用面 | 对象 |
|---|---|---|
| 得地（通根）| 地支藏干 | 天干与地支的联系 |
| 得助 | 天干之间 | 同类五行天干的横向帮扶 |

### 得势综合判据
- **真得势**：得生 + 得助俱全，生方与受生方俱有根。
- **假得势**：表面生扶多，但受生方无根或生方虚浮，实际无效。
- **失势**：全局印比稀少，财官食伤林立，日元孤立无援。

### 干支对日元作用力由大到小
| 顺序 | 位置 | 理由 |
|---|---|---|
| 1 | 月令 | 大环境，占全局至高权重 |
| 2 | 日支 | 紧贴日元，夫妻一体，生克之力最强 |
| 3 | 月干 | 月令之上，属大环境一部分 |
| 4 | 时干 | 紧邻日干 |
| 5 | 时支 | 距离较近 |
| 6 | 年干 | 较远 |
| 7 | 年支 | 最远，影响力最小 |

- 判断得势时须按此力度加权，不能简单数人头。

## 与得令、得地的关系
- **互相独立**：得令只看月支；得地只看地支通根；得势看全局五行势力与生扶结构。
- **互相补充**：
  - 失令 + 得地 + 得势 → 日元仍可中和偏旺。
  - 得令 + 失地 + 失势 → 看似得令，实际虚而不强。
- **加权权重**：三者之中，得令权重最大，得地次之，得势再次；**但得势是前两者能否真正发力的关键催化**。
- **有根为本、有势为用**：得地是"自立"，得势是"借力"。自立者方能借力，借力者方得大成。

## 误区
- **数天干比劫论得势**：忽视地支根系，把一片虚浮比劫当成得势，实则外强中干。
- **忽视生方自身力量**：只看到有印星透出，未查印星是否通根，结果印虚不能生身。
- **把得助等同于得地**：把天干比劫的帮扶当成根，是学八字的常见混淆。
- **日元无根而求生扶**：无根者拒补，强补反受其害——许多身弱命造的问题根源在此。
- **孤立评估势力**：只数阴阳五行个数，不按距离加权，把年支上的比劫视同日支上的正根。
- **把异党数量少误当得势**：异党少不等于得势，必须印比本身**既有根又有力**才算真得势。

## 现实应用
- **取喜用神**：得势强者，喜官杀食伤财来克泄；失势者，喜印比生扶。
- **合作与贵人**：得势之命团队缘好、朋友助力多；失势之命往往凡事需亲力亲为。
- **看印星实效**：判断母亲、长辈、学业资源是否真正能利己，关键看日元是否有根受印。
- **财富承载力**：有根有势者方能"担财""担官"；无根无势者，财官再多也守不住。
- **岁运转机**：失势之命在印比大运得以暂补势力，为人生阶段性上升窗口。
- **事业选择**：得势者适合竞争激烈、人脉密集型行业；失势者宜选稳、选专、走技术路线。

## 古籍背书
- 《子平真诠·论用神因成得败》论及"日主强弱，不独在月令，亦在全局之配合"——正是得势思想的源头。
- 《滴天髓·衰旺》：**"能知衰旺之真机，其于三命之奥，思过半矣。"** 并细论"真神、假神、助气、党气"，即得势之真假。
- 《三命通会·论日干生旺死绝》：**"干强不如根深，根深不如势众。"** 直接点出得势的最高层级。
- 《渊海子平》论生扶帮助："**身旺喜泄、身弱喜扶，扶之有道在于根势相应。**" 即得生得助须根势呼应方为真。

## 一句话总结
- **得势看全局，生助齐聚且各自有根，方为真势；无根之势只是虚张声势。**
