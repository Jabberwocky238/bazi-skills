# 己土

## 定位
- 阴土。土在五行中居中，承载万物、参与转换——这是土的共性。
- 戊土偏聚成一块（讲承托整体），己土偏散布一层（讲包容与渗透）。
- "己"字本身有收敛归纳之意，抓住的是土气内向、细密、能藏纳万物的状态。

## 十神对应（以己为日主）
- 火生土为印：阳火为正印，阴火为偏印。
- 土生金为食伤。
- 木克土为官杀：阳木为正官，阴木为七杀。
- 土克水为财。
- 同类为比劫。

## 己土的特点
### 1. 依赖环境
- 不像戊土厚土成山，天然成型；更像土壤本身，需要水润火暖才能发挥作用。
- 过燥则裂，过湿则泥泞。
- 关键判断点落在"质地"，而非单纯旺衰。

### 2. 得令与失令
- 得令于辰戌丑未月，尤其丑未中藏己，有根易稳定。
- 身在寅卯之月，木旺土松；在午火生扶之月易燥。
- 关键看"能不能收纳其他五行"。

### 3. 与水的关系
- 己土比戊土更易被水改变形态。
- 适度之水润土育物。
- 水多无火，土变泥泞难以治水。
- 局中有火：火暖土燥，水气得以调节，土既能承载又能治水。
- 不能见"己土见财旺"便断能"长财"——缺条件。

### 4. 擅长调和
- 土本居中，己土细密，更易与其他五行交织。
- 木克土时，若火在中间，火生土而泄木，冲突缓和。
- 水多土弱时，若火土相继而来，土气重新成型。
- 很多命局里己土不是主导力量，却在维持整体稳定。

## 透干与藏支
- 透干：外在对事物的操持与收拾。
- 藏支：内部的支撑。
- 透而无根：土浮于表，看似能担实则易变。
- 藏而有根：虽不显眼却稳固。

## 常见误解
1. 直接等同于"性格温和、细心"——这只是象的延伸，不是结构判断。
   - 比劫重而官杀弱的己土日主可能执拗而不柔和。
2. "土多而不通"——离开流通谈厚薄容易看错。

## 核心判断
己土像命局里的**收纳之地**。能不能收得住取决于：
- 有没有火去生扶
- 有没有水但不过度侵蚀
- 根器是否存在
