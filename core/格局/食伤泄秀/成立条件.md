# 食伤泄秀

## 成立条件（严格依《子平真诠》《滴天髓》）

### 条件 1：日主身强（本格用法的前提）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论食神》："**食神吐秀，身旺者贵**。"
  - 《滴天髓·顺逆》："**旺者宜泄不宜克，衰者宜帮不宜损**。"
- **客观判据**：
  - ① 日主于月令本气为比劫或印星（身旺），或
  - ② 日主于月令为禄 / 刃（建禄 / 阳刃），或
  - ③ 他柱至少三位比劫印星通根（身偏旺至身旺）。
- **不满足则破**：身弱用食伤 → 沈氏"旺者宜泄，衰者宜帮"——身弱泄之为损，非"秀"；此时食伤当转为需被印制或比劫救。

### 条件 2：食神或伤官透干且通根，或月令本气为食伤

- **古籍出处**：
  - 《子平真诠·论食神》："食神必得生旺之地，方能吐秀。"
- **客观判据**：
  - ① 食神或伤官透于月 / 日支 / 时干，且地支有本气或中气通根至少一位；或
  - ② 月令地支本气为食神或伤官。
- **不满足则破**：食伤虚透 → 泄得不出，身气仍壅塞。

### 条件 3：食神与伤官不同时并强（不混杂）

- **古籍出处**：
  - 《子平真诠·论食神》："食神清而不杂为吐秀之象。"
  - 《渊海子平·食伤诀》："食伤混杂，不清不秀。"
- **客观判据**：
  - 原局以食神为主（食神透干通根、伤官仅藏支或不透）→ 食神吐秀（温厚型）；
  - 或以伤官为主（伤官透干通根、食神仅藏支或不透）→ 伤官吐秀（锋锐型）。
  - 若食神与伤官均透干且均通根且力量相当 → 属"食伤混杂"。
- **不满足则破**：食伤混杂 → 降格，转入"食伤混杂"讨论。

### 条件 4：原局无偏印（枭神）紧贴克食伤

- **古籍出处**：
  - 《滴天髓·食神》："**食神吐秀，最忌枭神**。"
  - 《子平真诠·论食神》："**食神最畏枭神**……枭印夺食，福去祸至。"
- **客观判据**：
  - ① 天干无偏印与食神 / 伤官紧贴相克（月 / 日、日 / 时邻柱）；
  - ② 若有偏印，须有财星透干紧贴克之（财克枭救食神）。
- **不满足则破**：枭印夺食且无救 → 转入"枭神夺食"破格讨论。

### 条件 5（进阶条件）：最好有财星通根作为食伤之泄的最终落点

- **古籍出处**：
  - 《子平真诠·论食神》："食神生财，美格也。"
  - 《滴天髓·食神》："**食神有力，胜于财官**——若更兼生财，则富贵双全。"
- **客观判据**：
  - 财星透干且通根，位于食伤之后柱（形成食→财顺流）。
- **非必要**：本条不满足不破格，但格品降一等——食伤泄秀而无财接，沈氏谓"秀而不实"（才华未落于财禄）。

---

**前四条必须同时满足**方为真食伤泄秀；条件 5 为进阶条款，关乎格品高低而非成败。本格与"食神制杀"之别：食神制杀用食之克（针对七杀）；食伤泄秀用食之泄（针对身旺）——两用法所取身强之要一致，但去处不同。
