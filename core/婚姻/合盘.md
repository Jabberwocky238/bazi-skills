# 合盘：看两个人有没有缘分

## 一、合盘的真相：远比想象复杂

- **合盘不是拿两个八字对齐一眼就能下结论**。
- 一次完整的合盘至少要综合**六大维度**：
  1. 合**喜用**
  2. 合**四大基本特征**（心理、性格、社会行为、身体健康）
  3. 合**宫位**（地支相互作用）
  4. 合**桃花**（在哪条桃花线上相遇、能否落实）
  5. 合**主体**（一方八字拆入另一方各宫位）
  6. 合**格局**（门当户对、家庭背景、社会层次）

### 为什么合盘复杂
- 每个人的八字各方面都有**不同的喜用**（身体、事业、考学、财运都不一样）。
- 每个维度都是一次独立的对比。
- 合盘最终结论 = **六大维度交叉验证**后的综合判断。

## 二、合性格：从内宫位入手

### 关键宫位
- **内宫位** = 日支 + 时干 + 时支（核心）+ 月干 + 月支（次之）。
- **外宫位** = 年柱 + 月柱天干。
- 性格判断主要看内宫位，尤其是**日支与时支**。

### 对比方法
- 双方的内宫位是否**和谐、互补**？
- 双方在外环境上的表现（事业、社交、表达）是否**互相促进**？
- 双方的驱动（时柱天干 → 月柱天干）是否**同向或互补**？

## 三、合宫位：四个地支关系

### 核心对比点
两人对应地支之间只有四种关系：
| 关系 | 特点 | 意义 |
|---|---|---|
| **合** | 地支相合 | 关系紧密、同向而行 |
| **同（铜）** | 地支五行相同、力量同向 | 同心同力 |
| **刑冲** | 地支相刑、相冲 | 欢喜冤家（在合盘整体好的前提下） |
| **暗、静** | 既不合也不冲也不同 | **最差** —— 没有实质缘分、无真实联系 |

### 外环境相合（年柱地支）
- 双方年支相合 → **事业上结识**；
- 由"主导方"的五行定性（例如化水 = 水性主导方领导）。
- 可演化出多种取向：
  - 一方事业带另一方。
  - 一方工作环境 = 另一方的发展场域。
  - 一方的财 = 另一方的印（可以提供事业支持）。

### 近环境相合（月柱地支）
- 双方月支相合 → **生活环境主导权**之争。
- 若合化的五行只透在一方天干 → 双方潜移默化中公认此方在家中说话算数。
- 若另一方天干也透出合化五行 → **权威之争公开化** → 家庭冲突。

### 夫妻宫（日支）
- 最关键：**忌既不合、又不同、也不冲，只是暗藏作用**。
  - 暗静型 = "阴暗爬行一样的作用" → 连朋友都不如。
- **同** → 同心同力干同一件事。
- **相冲**（在合盘整体好的前提下）→ 欢喜冤家，鸡毛蒜皮的冲突反而巩固感情。
- **合** → 关系密切、灵活默契。

### 内交友宫（时支）
- 表示双方内部关系、心理亲密度。
- 如果夫妻宫不好但时支**同、合** → 至少在心理层面有共鸣。

## 四、外环境好、内环境差：缘分在外不在内

### 典型画面
- 两人在**事业、工作**上的关系非常紧密。
- 但地支夫妻宫、内交友宫的关系**既不合、也不冲、也不同**，几乎是暗静。
- 结论：**真实缘分只在事业层面**，婚姻或挚友层面没有实质联系。

### 判断原则
- 外环境好 → 可以做同事、合作伙伴、事业搭档。
- 内环境好 → 才能做夫妻、挚友、亲密关系。
- 如果只有外环境好、内环境一片空白 → 勉强在一起也只是"事业拍档"。

## 五、合化的多级引动

### 基本规则
- 地支合化并非只看一次，还要看是否能**多级接力**。
- 例如：
  - 子辰合水 → 需搭配壬水方向。
  - 寅午合火 → 需搭配丙火方向。
  - 若水火两股合化力相冲 → **合化到此为止**。

### 理想的多级接力
- 外环境合 → 引动近环境 → 再引动夫妻宫 → 再引动内交友宫。
- 这种层层叠加的合化 = "**事业促成婚姻、婚姻巩固内在**"。
- 是合盘最理想的推进路径。

## 六、合格局：门当户对

### 不是封建残余，而是命理的客观判断
- **格局高低** = 八字整体做工的清纯度、贵气是否成立。
- 若两人格局相差悬殊 → 易因**个人素质 / 收入 / 事业成就**产生家庭冲突。
- 典型：一方挣钱多但修养一般；另一方挣钱少但文运好、官运好 → 会为家庭话语权产生巨大纷争。

### 格局判断参考
- **流通是否清纯**：做工混杂 vs 单向顺流。
- **有无相刑相冲**：地支相形越少，格局越稳。
- **天干做工是否成立**：伤官见官、食神制杀能否落到位。

## 七、合喜用：双方能否互补

### 分离用神的前提
- 每个人在不同领域（身体、事业、婚姻、财运）喜用不同。
- 合盘只讨论**事业喜用与婚姻喜用的重叠**。

### 双箭头最好、单箭头次之
- **双箭头互旺**：甲方能提供的气 = 乙方喜神；乙方能提供的气 = 甲方喜神。
  - 真实缘分深，事业互相成就。
- **单箭头**：只有一方旺另一方 → 感情失衡，容易产生付出不对等。

### 现实操作
1. 分别写出男方、女方的**事业喜用**。
2. 对照双方**能提供的五行**（通过原局结构判断）。
3. 看重合度 → 得出"事业上互旺 / 单向旺 / 不旺"的结论。

## 八、合盘的判断流程

推荐顺序：

1. **先看性格与心理** → 能否合得来、是否基本匹配。
2. **再看宫位关系** → 夫妻宫、内交友宫、外环境、近环境四处。
3. **看合化引动** → 能否从外环境顺利接到内宫位。
4. **看格局匹配** → 是否门当户对、层次差距有多大。
5. **看喜用互补** → 能否互旺事业、互旺身心健康。
6. **看桃花引动** → 是否正在桃花应期、桃花在哪条路径上引动。

## 九、一句话总结

- 合盘的核心问题从来不是"**合不合得来**"，而是"**缘分在哪一层**"。
- 有的缘分只在外在事业 → 合作可以，婚姻不行。
- 有的缘分只在内心默契 → 婚姻可以，但事业拍档不顺。
- **全维度都合** = 正缘 + 事业搭档 + 终身伴侣。
- **全维度都不合** = 哪怕表面条件再好，强行在一起也只是消耗。
