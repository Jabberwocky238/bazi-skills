# 从旺格（含从强格）

## 成立条件（严格依《滴天髓·从旺 / 从强》）

### 条件 1：日主极旺——比劫 + 印星共同主导全局

- **古籍出处**（本条最核心）：
  - 《滴天髓·从旺》："**从旺者，四柱皆比劫，无官杀制，有印生之，其势满盘皆是旺神**。"
  - 《滴天髓·从强》："**从强者，四柱印绶重重，比劫叠叠，日主又当令**。"
- **客观判据**：
  - ① 月令地支本气为比劫或印星；
  - ② 全局比劫 + 印星位数占大半（透干与通根合计至少四至五位）；
  - ③ 月令、日支、时支至少两位为比印本气。
- **不满足则破**：比印不成压倒之势 → 只是普通身旺，非从旺；当按正格论用神。

### 条件 2：原局无官杀克身

- **古籍出处**（本条铁律）：
  - 《滴天髓·从旺》："**无官杀制**——官杀一见，旺气被压，即不为从旺。"
- **客观判据**：
  - 原局天干无正官 / 七杀透出；地支本气、中气无官杀藏干（若地支余气藏一位官杀，气微勉强可论假从旺，格品降）。
- **不满足则破**：官杀存 → 日主有制 → 旺势被克 → 当按身旺用官杀论，非从旺。

### 条件 3：原局无财破印（对从强格尤其致命）

- **古籍出处**：
  - 《滴天髓·从强》（任铁樵注）："**从强最忌财星**——财破印则从强之势被夺。"
- **客观判据**：
  - 原局天干无财星与印星紧贴相克（若财在远柱而印在近柱，不紧贴则影响较轻）。
- **不满足则破**：财贴印 → 印被克 → 从强根基瓦解 → 转入"以财破印"或身旺用财讨论。

### 条件 4：食伤可有可无——微泄秀气可喜，重泄伤旺势

- **古籍出处**：
  - 《滴天髓·从旺》："**从旺喜食伤吐秀**，忌官杀制伏。"
  - 《滴天髓》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 食伤一位透干或地支一位根 → 微泄秀气，可喜；
  - 食伤多透多根 → 重泄。
- **不满足则破**：食伤过重泄旺 → 旺势被削 → 按身旺食伤泄秀的正格论，非从旺。

### 条件 5（细分条件）：区分从旺 / 从强

- **古籍出处**：
  - 《滴天髓·从旺》："**四柱皆比劫**，有印生之——从旺。"
  - 《滴天髓·从强》："**四柱印绶重重**，比劫叠叠——从强。"
- **客观判据**（二选其一）：
  - ① 比劫位数多于印星位数 → **从旺格**（以比劫为主）；
  - ② 印星位数多于比劫位数 → **从强格**（以印星为主）。
- **意义**：分主次决定喜忌之重——从旺尤喜比劫运，从强尤喜印星运。

---

**前四条必须同时满足**方为真从旺 / 从强格；条件 5 为亚型区分。
- **从旺**：比劫主导 → "靠自己打天下"的气质，事业多自立门户。
- **从强**：印星主导 → "靠资源起步"的气质，事业多借家学 / 贵人 / 平台。
