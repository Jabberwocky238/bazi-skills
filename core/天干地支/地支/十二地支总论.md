# 十二地支总论

## 定义
- **十二地支** = 子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥，共十二个符号。
- 与十天干配合组成**六十甲子**，用于纪年、月、日、时。
- 地支代表**地元**，属静、属内、属藏；古人云"**天主动、地主静**"。
- 地支同时是**生肖、方位、季节、时辰**的载体，是八字分析中**最复杂、最关键**的一层。

## 一、天干与地支的根本差异

### 1. 动静之别
> 天干主天，主动，显露于外，代表动则有为，彼之主；
> 地支主地，静藏其内，代表静以待用。

- 天干**外露**——能引发并作用于某些事件，象征"有事发生"。
- 地支**内藏**——喜静不喜动；如同大地里的庄稼需要慢慢生长，若反复扰动则生机尽毁。

### 2. 动起来的差异
- **天干相冲**：如两人公开吵架——矛盾显而易见、吉凶直接。
- **地支相冲**：非简单可断——必须结合命主喜忌：
  - **忌神被冲** → 好事。
  - **喜神被冲** → 坏事。
  - 例：原局婚姻宫被冲，大运流年又冲婚姻宫，**反而可能结婚**。
- 一句话：**天干动起来还好；地支一动起来，吉凶难以预测**。

### 3. 根苗之别
> 天干、地支相互配合：地支为天干之根基，天干为地支之苗芽。
> 地支值年则代表众煞之首；值月则主宰生杀之权。

- 天干若无地支之根，则**无法显现或发挥作用**。
- 地支**值年** = 当年太岁，决定一年的命运趋势。
- 地支**值月** = **月令**——八字中能量最强的地支，主宰格局与身强身弱。

**故地支的重要性远超天干，内容也比天干复杂许多。**

## 四、十二地支与季节、月份

春天始于**立春**——约公历 2 月，对应农历**正月**。春季的第一个月是**寅月**（并非排在首位的子月）——此点极易混淆，务必牢记。

| 季节 | 主气 | 三个地支 | 说明 |
| --- | --- | --- | --- |
| 春 | 木 | 寅、卯、辰 | 寅卯属木，辰为季末土 |
| 夏 | 火 | 巳、午、未 | 巳午属火，未为季末土 |
| 秋 | 金 | 申、酉、戌 | 申酉属金，戌为季末土 |
| 冬 | 水 | 亥、子、丑 | 亥子属水，丑为季末土 |

**四季末皆为土**——土为过渡、收束与承载。此规律将贯穿后续"地支三会"的学习。

## 六、十二地支方位图

- **易学方位遵循"上南下北、左东右西"**——与中国古代坐北朝南的思想一致，与现代地图上下相反。
- 四季所主之方位：
  - **春属木** → 寅卯辰在东方。
  - **夏属火** → 巳午未在南方。
  - **秋属金** → 申酉戌在西方。
  - **冬属水** → 亥子丑在北方。

### 三组特殊定位

#### 1. 四正位（桃花）——子午卯酉
- 位于正北、正南、正东、正西。
- 位置最"**正、纯**"，气纯而力专。

#### 2. 四驿马（偏锋）——寅申巳亥
- 位于东北偏东、西南偏西、东南偏南、西北偏北。
- 像古代驿站、以马传递——具有**积极、前进、动态**的性格特质。

#### 3. 四季土（墓库）——辰戌丑未
- 位于东南偏东、西北偏西、东北偏北、西南偏南。
- 又称"**四墓库**"：辰水库、未木库、戌火库、丑金库。

> 三组定位将在后续三合、三会、墓库课程中反复用到，必须先立概念。

## 八、掌诀（掐指诀）

- 左手掌常用于掐指推算；右手用来记录。
- 从**无名指的第三指节根部**开始，按顺时针方向：
  - **子** → 无名指根 → **丑** → 中指根 → **寅** → 食指根 → **卯** → 食指第二节 → **辰** → 食指尖 → **巳** → 中指尖 → **午** → 无名指尖 → **未** → 小指尖 → **申** → 小指第二节 → **酉** → 小指根 → **戌** → 无名指第二节 → **亥** → 中指第二节。
- 多练几遍即能熟记。

## 九、地支的关系速览（后续学习入口）

地支间的作用远比天干复杂，涉及：

- **六合**：子丑、寅亥、卯戌、辰酉、巳申、午未。
- **六冲**：子午、丑未、寅申、卯酉、辰戌、巳亥。
- **三合**：申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金。
- **三会**：亥子丑会水、寅卯辰会木、巳午未会火、申酉戌会金。
- **相刑**：子卯相刑、寅巳申三刑、丑戌未三刑、辰辰/午午/酉酉/亥亥自刑。
- **六害**：子未、丑午、寅巳、卯辰、申亥、酉戌。
- **相破**：子酉、卯午、辰丑、戌未、寅亥、巳申。
- **暗合**：隐而不显的地支牵动。
- **拱合**：两支之间合拱出隐藏的地支。

这些作用决定了八字的刑冲合化、婚姻事业、吉凶祸福。

## 十一、一句话总结

- 十二地支 = **地元**——静而藏，主宰格局与应期。
- **天动地静**——天干一动多在事上可见；地支一动吉凶难料，必须结合喜忌。
- 先记清**阴阳、五行、方位、月份、时辰**五项基础，再深入学合冲刑害破。
- 地支是八字最核心、最复杂的一层——**看懂地支，就看懂了命运的根基**。
