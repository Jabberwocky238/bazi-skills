# 空亡（空无 / 旬空 / 天中煞）

## 定义
- **空亡**：十天干配十二地支、每一旬（十组）必余出两支不配干 → 这两支称为"**空亡**"。
- 又名：**旬空、天中煞、空无**。
- 本质：**所处地支之五行之气"落空"、虚无、不能实实在在发挥作用**。
- 古籍论："**空亡入命，主虚无、不实、落空**。"

## 起法（按日柱旬查地支）
《三命通会·论空亡》原文："**甲子旬中戌亥空，甲戌旬中申酉空，甲申旬中午未空，甲午旬中辰巳空，甲辰旬中寅卯空，甲寅旬中子丑空**。"

### 六旬空亡对照表

| 旬（日柱所在的十干支）| 空亡（两支）|
|---|---|
| 甲子旬（甲子 → 癸酉）| **戌、亥** |
| 甲戌旬（甲戌 → 癸未）| **申、酉** |
| 甲申旬（甲申 → 癸巳）| **午、未** |
| 甲午旬（甲午 → 癸卯）| **辰、巳** |
| 甲辰旬（甲辰 → 癸丑）| **寅、卯** |
| 甲寅旬（甲寅 → 癸亥）| **子、丑** |

### 查法步骤
1. 确定日柱（如甲寅日）。
2. 查日柱所在旬（甲寅日属甲寅旬）。
3. 该旬对应的两支即空亡位（甲寅旬空子、丑）。
4. 命中年、月、时支若见该空亡地支 → 该柱带空亡。

### 举例
- 日柱为**甲寅**（甲寅旬）→ 空亡在**子、丑**。
- 若年支为子 / 月支为丑 / 时支为子丑 → 相应柱带空亡。

## 核心作用：虚无落空
- 空亡主"**气虚无实、事难成形**"：
  - 所空之字的五行**不起实际作用**。
  - 该柱所代表的人 / 事 / 物**缘浅、落空、无着落**。
- 古籍论："**空亡者，有如未成、有如已散**。"

## 空亡在四柱的不同含义

| 空亡位置 | 所代表之事 | 现实表现 |
|---|---|---|
| **年柱空亡** | 祖辈、早年环境、家族根基 | 祖业无承、父母缘薄、少年漂泊 |
| **月柱空亡** | 父母、事业、青年期 | 与父母聚少离多、事业起伏、中年前不稳 |
| **日支空亡** | 配偶、婚姻 | 婚姻缘薄、夫妻关系淡、晚婚或反复 |
| **时柱空亡** | 子女、晚年 | 子女缘薄、晚年孤独或远行 |

## 空亡与十神结合

| 空亡之十神 | 落空的含义 |
|---|---|
| **财星空亡** | 财运落空、破财、求财辛苦 |
| **官星空亡** | 事业不稳、升职无望、女命夫缘薄 |
| **印星空亡** | 学业中断、贵人无力、母缘薄 |

## 空亡的"解"与"不解"（最关键的判断）

### 空逢冲则实（破空）
- **冲破空亡**：命中空亡之支被冲 → 反而"空而不空、填实而有用"。
- 例：空亡在子、命中有午 → 子午冲 → 子的空性被冲开，反而能实际发挥五行作用。
- 古籍论："**空而逢冲，不为空也**。"

### 空逢合则填（合空）
- **合住空亡**：空亡被三合、六合合住 → 气被拉回、填实。
- 例：空亡在丑、命中有子丑合 → 丑被填实。

### 空逢克则不解
- 空亡被克（非冲、非合）→ 仍为空。

### 重空则解
- 若命中多处空亡（如年月日时有两处以上空亡）→ **"负负得正"** → 反而格局清奇、可能显特殊才能。
- 古籍论："**空亡多者，反主奇异**。"

### 真空 vs 半空
- **真空**：地支为空亡 + 十干纳音也弱 → 空得彻底。
- **半空**：地支为空亡 + 十干纳音有力 → 空得不彻底、落空力减。

## 常见误区

### 1. "带空亡 = 一定倒霉"
- 空亡的本质是"**不实、落空**"——不等于必然凶。
- 忌神空亡 → **反而是好事**（该坏的事气散了）。
- 喜神空亡 → **才真正是损失**（该好的事落空了）。
- **必须结合喜忌判断**，不能一见空亡就断凶。

### 2. 空亡逢冲反凶
- 古籍明言："**空而逢冲，不为空**"——冲是空的救，不是加凶。
- 很多命书把"冲"一律当凶 → 忽略了冲可解空。

### 3. 忽视"空亡的程度"
- 天干透出而地支空亡 → 气虚但有形。
- 地支空亡且天干也虚 → 彻底落空。
- 空亡地支被刑冲合害 → 空性被搅动，需具体分析。

### 4. 只查日柱空亡
- 传统以**日柱**为主查空亡（最常用），但也可以：
  - **年柱查空亡**（某些流派）。
  - **同时查日、年**（高阶判断）。
- 本文以日柱查为主。

## 空亡的特殊含义

### 正面含义（意外之福）
- 忌神空亡 → 少了一分凶。
- 空亡 + 华盖 → 精神世界深厚、宗教缘、玄学缘。
- 空亡 + 贵人 → 贵人突然出现、意外援助。
- 空亡逢冲带喜用 → 转化为新机会。

### 负面含义（落空之憾）
- 喜用神空亡 → 该得的没得、该成的没成。
- 六亲位空亡 → 与亲人缘浅、聚少离多。
- 空亡逢克且无救 → 落空成定局。

## 现实表现

### 带空亡的人典型特征
- **容易有"**人生空茫感**"**：觉得努力不到实处。
- **对宗教、玄学、精神性话题有天然兴趣**：空亡引向虚无。
- **关键时刻事情"**突然没了**"**：合同黄了、升迁泡汤了、关系散了。
- **或反向**：原以为没希望的事**突然成了**（空逢冲则实）。

### 六亲缘薄
- 父母位空亡 → 父母早亡或聚少离多。
- 夫妻宫空亡 → 婚姻缘浅、晚婚、分居。
- 子女宫空亡 → 子女缘薄、无子或子女远走。

### 事业、财运
- 财星空亡 → 求财过程中常"**差一点就成**"。
- 官星空亡 → 升职关键年被调走、政策变动。
- 印星空亡 → 考证关键时刻失利、学历路径转向。

## 化解与应对

### 结构层
- 喜神空亡且命中有冲 → 冲破空亡反成吉。
- 命中有合能填空 → 合住空亡填实。
- 大运流年填实空亡 → 空亡之事在那段时期兑现。

### 人事层
- **识别空亡后学会"**不执着**"**：该空的就让它空、不强求。
- **把精力放在实处**：不在空亡代表的领域过度投入。
- **利用空亡走精神 / 宗教 / 艺术路径**：空亡天然适合"**虚而不实**"的领域。

### 心态层
- 接受"**人生有些事本就要落空**"。
- 空亡命的人常有深度哲学思考——善用之可成智者。

## 与其他神煞的配合

- **空亡 + 华盖**：宗教、哲学、玄学深度——"出世之象"。
- **空亡 + 驿马**：漂泊成空——一生动荡无定所。
- **空亡 + 桃花**：感情落空——爱而不得。
- **空亡 + 天乙贵人**：贵人虚有——期望中的贵人未能到位。
- **空亡 + 天德 / 月德**：福气落空——应得的福未能兑现（但多重空亡反成"空即是实"的反转）。

## 核心判断
- 空亡不是凶煞也不是吉神——是"**虚无、不实、落空**"的气质标记。
- 识别关键：
  - 喜神空亡 = 损失。
  - 忌神空亡 = 化解。
  - 空逢冲 / 合 = 空性被破。
- 核心意义：**让命主了解"**有些事本就是要空的**"，不必强求**。

## 一句话总结
- 空亡 = **地支的虚空之气**。
- 忌神空了是福、喜神空了是憾 —— 空本身无善恶。
- 有些人一辈子都在等"该来的东西"，结果就是空 —— 不如把精力放到空亡不到的地方，**那里才是你真正的根**。
