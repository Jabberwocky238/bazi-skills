# 劫财见财

## 构成条件（严格依《子平真诠》《滴天髓》《渊海子平》）

> **注**：本条为"病象 / 破格"，非"成格"——故用"构成条件"。当以下条件**全部满足**时方构成真正的"劫财见财"夺财之象；有任一救应则不破。

### 条件 1：劫财（或比劫）透干且通根，或月令本气为比劫

- **古籍出处**：
  - 《滴天髓·顺逆》："**比劫太重，财必被夺**。"
- **客观判据**：
  - ① 劫财透于年 / 月 / 时干，地支本气或中气通根至少一位；或
  - ② 月令本气为比肩 / 劫财（如甲日见寅月、庚日见申月）。
- **不构成**：比劫虚透无根、力量轻微 → 劫而无力，不构成真见财之夺。

### 条件 2：财星透干或本气在地支，能被劫财直接克到

- **古籍出处**：
  - 《渊海子平·论财》："财逢比劫，须官星为救。"
- **客观判据**：
  - 财星透于天干、或在地支本气至少一位，**位置与比劫相邻或同柱**（才真能被克）。
- **不构成**：财完全藏于远支 + 比劫在近位 → 克不到，只是"远夺虚象"。

### 条件 3：财星力量**弱于**比劫（比劫强而财弱）

- **古籍出处**（本条最核心）：
  - 《滴天髓·财》："**财轻劫重，财必被分**。"
- **客观判据**：
  - 以透干数 + 通根位数综合比较：比劫总力（透 + 根）明显强于财星总力。
- **不构成**：财 ≥ 比劫 → 财能自守，不被夺 → 甚至反成"身弱用财"的另一结构。

### 条件 4：原局无官星护财

- **古籍出处**（本条为关键救应门槛）：
  - 《渊海子平·论财》："**财逢比劫，得官星制之则无忧**。"
  - 《子平真诠·论财》："财怕比劫之夺，**官能制劫护财**。"
- **客观判据**：
  - 天干无正官或七杀与比劫紧贴相克；地支本气无官杀制比劫之根。
- **不构成（救应成立）**：有官星透干紧贴克比劫 → 劫被制、财得护 → "**劫财见财 + 官护**"反为可用结构；《渊海子平》直言"**得官星制之则无忧**"。

### 条件 5：原局无食伤通关

- **古籍出处**：
  - 《滴天髓·食神》："**食神泄劫生财，劫不伤财**。"
- **客观判据**：
  - 天干无食神 / 伤官在比劫与财之间通关；若有食伤 → 比劫 → 食伤 → 财 → 本条不成立。
- **不构成（救应成立）**：食伤通关 → 劫财的能量经食伤转化后生财，反成"**食伤生财**"正格。

---

**五条同时满足**方构成真正的"**劫财见财破局**"。条件 4、5 任一成立即不破——古籍明文两条救应：
- **官星护财**（《渊海子平》）；
- **食伤通关**（《滴天髓》）。
一字之差：**无救为病，有救为用**。
