# 丁火

## 定位
- 十天干第四位，属阴火。
- 共性：火主炎上，由内向外发散。
- 阴阳之分决定发散方式：丙为阳火，外放炽烈、照面广；丁为阴火，内聚、细微、光点集中。
- "丁"字本身带有"小而明、点而不张"的意味，其价值在穿透力与持续性，而非声势。

## 十神对应（以丁为日主）
- 水克火为官杀：阴水（癸）为七杀，阳水（壬）为正官。
- 火克金为财。
- 木生火为印：阴木为偏印，阳木为正印。
- 火生土为食伤。
- 同类为比劫。

## 常见误解
1. 把丁火等同于"温柔体贴、心思细腻、多愁善感"。
   - "阴火之细"指气的状态，不是情绪一定敏感。
2. 把"丁火弱"等同于"人生必然坎坷"。
   - 旺衰只是气势状态，不是命运宣判。
   - 弱而得助可成事，旺而失衡也会走偏。

## 旺衰判断
- 得令得根，火气有缘，虽为阴火也能持续发光。
- 身处寒水之月，无木生扶、无土承载，丁火易被水压制，光点难以持续。
- **火最怕的不是被克，而是无所附着**——无木引发、无土承托，很难稳定。

## 通关与去处
- 水克火为官杀，是基本关系。
- 若命局有木，木生火而治水，形成"水—木—火"流转，克转为调节。
- 丁见壬：若木旺在中间，有压力但不至于熄灭；若无木，壬水之冲丁火难支。
- 火要有去处，生土为食伤是一个出口。
- 水多未必凶：丁火旺而燥时，水反成调候。判断必须建立在旺衰基础上。

## 位置
- 透干：显，易被外界看到，也易被外界干涉。
- 藏支：内，更多在内部消化。
- 透而无根：似灯芯无油，外亮内虚。
- 不透但地支得根：外表未必张扬，底子却稳。

## 核心判断
丁火是一种**精细而持续的能量源**，不靠声势而靠稳定的闪点。要结合官杀、财星、比劫等整体配置判断其表达方式。

## 古典标准：丁火论命思路

### 日主五行本性
- 丁火为阴火，号"人间烟火"、"灯烛之火"。
- 性柔中而内性昭融——外柔内明、温和而通达。
- 不似丙火轰烈，却不轻易熄灭；衰而不穷、旺而不烈。
- 带书生气、傲气，天生有"传道"之象，宜文化、教育、玄学、心理、医道之途。
- 《滴天髓》总纲：**"丁火柔中，内性昭融，抱乙而孝，合壬而忠，旺而不烈，衰而不穷，如有嫡母，可秋可冬。"**

### 日主逢不同月令的状态

| 月令 | 气势 | 核心矛盾 | 取用要点 |
| --- | --- | --- | --- |
| 寅卯月（春） | 木旺生火 | 木多火塞 | 庚金劈甲引丁；忌水局破火 |
| 辰月 | 湿土晦火 | 气转虚 | 需木透干引发 |
| 巳午未月（夏） | 火旺自焚 | 燥烈无制 | 急用癸水调候，成水火既济 |
| 申月 | 金旺身衰 | 财星夺气 | 木印化财生身，兼用丙火扶 |
| 酉月 | 病地衰极 | 火气无根 | 甲木为嫡母，次取丙火辅之 |
| 戌月 | 火库藏根 | 库而不发 | 木来引动、丙火辅光 |
| 亥子丑月（冬） | 水旺火衰 | 寒极灭火 | 必用甲木生身，绝处逢生 |

### 喜用神优先级

1. **调候优先**：
   - 夏月必用癸水，防自焚。
   - 冬月必用甲木，兼取丙火护持。
2. **扶抑次之**：
   - 身弱取甲木（嫡母）、丙火（同气）。
   - 身强取湿土泄秀或水杀约束。
3. **通关再次**：
   - 金旺耗身时，木印通关，金生水则不可，木反被水湿不生火。
4. **病药殿后**：
   - 丁见乙木——"抱乙而孝"——孝而自损，乙非真生；药在甲木透干。

### 与其他九个天干的关系

- **甲（阳木）**：正印（嫡母），干柴引丁，秋冬之靠山，丁火最爱。
- **乙（阴木）**：偏印（继母），湿木难生火，反损丁光，"抱乙而孝"。
- **丙（阳火）**：劫财，助火之光，亦争辉；丙夺丁光但冬月可借暖。
- **丁（同类）**：比肩。
- **戊（阳土）**：伤官，晦火之光，然燥土泄火有限。
- **己（阴土）**：食神，泄秀柔和。
- **庚（阳金）**：正财，劈甲引丁，为丁火创造用材。
- **辛（阴金）**：偏财，丁见辛则易伤辛，财弱易破。
- **壬（阳水）**：正官，丁壬相合——"合壬而忠"，合而不灭，忠肝义胆。
- **癸（阴水）**：七杀，雨露克灯烛，夏月为救命、冬月为凶。

### 不同宫位的象意

- **年柱丁火**：门风清雅、文脉深厚。
- **月柱丁火**：中年可成名师、文人、教化者。
- **日支通根（巳、午、未、戌）**：内心光明、配偶细腻。
- **时柱甲辰、甲午引丁**：子女显贵、晚境光明（"丁火身弱甲辰时"一般有贵气）。
- 丁火受伤之位对应心脏、血管、神经、眼目；丁癸冲、子午冲之流年防心脏与视力之患。

### 典型贵格组合

1. **如有嫡母格**：甲木透干或寅木得根——秋冬不畏金水。
2. **合壬而忠格**：丁壬相合而不化，又有戊土制旺壬——忠义显达。
3. **伤官配印格**：夏月丁火，癸水调候、甲木生身——水火既济兼才华内敛。
4. **杀印相生格**：秋月丁火，癸杀透而甲木化之——文化人兼权柄。
5. **食伤生财格**：丁得戊己泄秀、庚辛为财——技艺取财、细工精致。

### 典型破格组合

1. **抱乙伤身**：丁日旺中逢乙偏印独透，无甲木救——牺牲过甚、无力成事。
2. **夏火无癸**：巳午未月，无癸无壬——自焚夭折。
3. **冬火无甲**：亥子丑月，天干无甲、地支无寅——寒灭无救。
4. **木多火塞**：春月木过旺而无庚金疏——聪明却难发挥。
5. **丁癸相冲**：癸水透而无制——心血之疾、视力之患。

### 古籍出处

- 《滴天髓·天干论·丁火》："丁火柔中，内性昭融，抱乙而孝，合壬而忠，旺而不烈，衰而不穷，如有嫡母，可秋可冬。"
- 《穷通宝鉴·三夏丁火、三冬丁火》：以月令定甲庚癸之取用次第。
- 《三命通会·论丁火喜忌》：详论丁与壬合、丁与甲亲、丁不惧金水之理。

### 一句话总结

**丁火如灯烛，柔中昭融——最贵"得嫡母甲木为靠山"，夏需癸水调候、冬需甲木回春，合壬不灭是忠，抱乙自损是孝，旺而不烈、衰而不穷。**
