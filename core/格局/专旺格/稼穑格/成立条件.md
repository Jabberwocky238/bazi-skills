# 稼穑格

## 成立条件（严格依《渊海子平·稼穑格》《三命通会》）

### 条件 1：日主为戊或己土

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·稼穑格》："**稼穑格——戊己日生人**。"
- **客观判据**：日主天干必须为戊土或己土。
- **不满足则破**：非戊己日 → 不入稼穑格。

### 条件 2：月令为土（辰、戌、丑、未之一）

- **古籍出处**：
  - 《渊海子平·稼穑格》："**月令须是土月**。"
- **客观判据**：月令地支为辰、戌、丑、未四库之一。
- **不满足则破**：月令非土 → 无一气之根，不入稼穑。

### 条件 3：地支土气遍布（辰戌丑未全见，或四柱土多）

- **古籍出处**（本条核心）：
  - 《渊海子平·稼穑格》："**地支辰戌丑未全，或四柱土多**。"
- **客观判据**（二者之一）：
  - ① 地支四位见 辰 + 戌 + 丑 + 未 四库俱全（最典型）；或
  - ② 地支至少三位为土（含四库与戊己日支）。
- **不满足则破**：地支土气不成势 → 稼穑不聚象。

### 条件 4：原局无甲乙木透干克土

- **古籍出处**（本条核心）：
  - 《渊海子平·稼穑格》："**不见木克，方为稼穑格**。"
- **客观判据**：
  - ① 天干无甲木或乙木透出；
  - ② 地支本气无寅、卯成势（一位藏支中气无碍）。
- **不满足则破**：木透克土 → 《渊海子平》"见木克"即明破格 → 转入"木疏厚土"或身强见官杀格讨论。

### 条件 5：原局无重水冲土

- **古籍出处**：
  - 《三命通会·稼穑》："**稼穑忌水多冲**。"
- **客观判据**：
  - 天干壬癸不多透（一位有根尚可，二位以上有根则忌）；地支亥子本气不成势。
- **不满足则破**：水多冲土 → 稼穑根基被冲 → 破格转入财多身弱或身弱财多讨论。

### 条件 6（助力条件）：金点缀泄秀 → "稼穑毓秀"

- **古籍出处**：
  - 《三命通会》："**稼穑得金吐秀，名稼穑毓秀**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 金（庚辛或申酉）在命局中仅一位透干或藏支，为微泄秀气——此时稼穑格升级为"稼穑毓秀"复合贵格。
- **非必要**：不满足不破格；满足则格品升（稳重之上添秀气）。若金多过度泄土则降格（转入土薄金旺讨论）。

---

**前五条必须同时满足**方为真稼穑格；条件 6 关乎格品——纯稼穑成但偏闷，得金点缀则为毓秀。
