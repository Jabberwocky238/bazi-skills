# 亥水

## 定义
- 八字地支中的**阴水**。
- 不是一滴水，而是一片**深海、涵养天地**的阴水。
- 代表：**潜藏思绪、深层意识、生命的滋养力与包容力**。

## 核心：静的藏，不是冻的流
- **子水**：奔腾而外显，是"冻的流"。
- **亥水**：安静而内敛，是"静的藏"。
- 亥中所藏天干：**壬、甲**（内生木气，又带金水之气）。
- 金生水 → 水生木，是一个**生化连环的起点**。

## 判断维度

### 1. 性格气质
- 心思深远、洞察力强，不易被表面迷惑。
- 懂隐忍，善于在暗处蓄势等待机会。
- 但容易：
  - 自我封闭。
  - 情绪压抑。
  - 思虑过深。

### 2. 与火水木的配合
- **火旺而赤水**：深思转为焦虑。
- **木泄水气过度**：心智过度消耗 → 多思少行。
- **遇寅木**（寅亥合，或三合木局）：多有远行异地机会，情感经验丰富，常与宗教、心灵、玄学相关。
- **遇火旺的大运**：水被煎 → 烦躁易怒。
- **遇金旺、水旺的运**：思绪加深，容易陷入回忆。

### 3. 落在不同宫位的差异
| 位置 | 表现 |
| --- | --- |
| 年柱 | 多主早年家族庇荫，但父母关系疏远 |
| 月柱 | 思维深沉，情绪敏感 |
| 日柱 | 配偶心思难测，或婚姻中有隐情 |
| 时柱 | 晚年思虑重，有宗教信仰或精神寄托 |

## 现实表现
- 亥水旺的人：
  - **直觉强**，对情绪场感受敏锐。
  - 特别能感知他人的情绪与动机。
  - 适合做**研究、推理、创作、心理类**工作。
- 若日主弱、水气又重：
  - 容易**逃避现实、爱做梦、不喜约束**。

## 常见误区
- 把亥水单纯等同于"聪明敏感"。
- 忽略其"深水无底"的负面——思维与情感没有出口，反而成为负担。

## 优缺点
- **优点**：有智慧、有灵气、有包容心，能静下心思考世界的本质。
- **缺点**：情绪阴柔、不易放下，容易困在情感和旧事中走不出来。

## 调候要点
- **有土或火来调候** → 深水能生万物。
- **失于节制** → 深水无底，不见天日。

## 一句话总结
亥水之人的一生功课是：**让深水见光**。
- 让思维与情感找到出口。
- 不至于被深层的自我淹没。

## 补充：亥水的藏干结构与"无"之鱼气

### 基本定位
- 亥对应农历**十月**，冬季第一个月，节气**立冬至大雪前**。
- 狭义阴水，广义**阳水**（藏干均为阳干）。
- **藏干**：本气壬水、中气甲木、**鱼气戊土**（隐而不书）。

### 为何说亥水中仍有戊土
- 查月令司令：每年戌月向亥月过渡时，司令始终出现戊土。
- 亥中戊土未被列入常规藏干的原因有二：
  - 戊丙十二长生中，亥为**戊土绝胎**、火土之势近乎为零 —— 所需火土容量极低。
  - 甲木（朱元璋）换气只需"收人头"——不需过多戊土作引——故戊土虽存而甚微。
- 但亥水**还有第二部分**戊土——**十二地支之"鱼"**（驿马星之尾、十二支末位），承接每年火土/金水/木气旺衰不平衡所余的**煞气能量**。
- 故亥水兼具：**不稳壬水 + 强迫中气甲木 + 神秘鱼气"无"**，性质极为反常独特。

### 壬水的产生机制
- 戌月向亥月换气时，通过强行引入第三方甲木，过滤净化戌土中未转壬水的辛金、丁火鱼气、戊土火土性，产生壬水。
- 故亥中壬水**新建不稳，不可作天干壬癸水等同使用**。
- 亥中壬水只表达**水气旺但不稳定**的状态。

## 补充：亥水的静态五行性质

### 整体特征：不好利用但胜于未土
- 亥水结构上兼有：
  - 不稳定的本气（壬水）。
  - 强迫而纯正的中气（甲木）。
  - 混杂隐秘的鱼气（戊土之"无"）。
- 极易使原局变寒、变杂、变难用。
- 与未土相比，亥水尚可搭配金水，且亥未组合贵气天然；未土则只能绑定亥水。

### 亥水对五行的作用（静态）

#### 1. 寒木
- 亥水生甲木——甲木中气可等同于天干甲木，纯度高。
- 但亥水太寒（甚至比子水更寒，因子水奔流有能量）——养不活活木。
- 必须配**午火、未土、寅木**老三样净化调温；
  - 海草格（乙木或甲木地支全配水）为特殊活用之法。

#### 2. 伤火
- **丙火不喜亥水**（除非有午火羊刃救之）；丙辛化气以亥水为根尤忌——亥中杂气污染纯正之丙辛。
- **丁火不拒亥水**，但丁火带亥水者**忌见癸水**——七杀攻身极凶。
- 丁壬化气带亥水易成，但需寅卯正根。

#### 3. 空土
- 戊己土不喜无根——天干土的根器在辰戌丑未，亥不能作正根。
- 己亥若无木火调候又无根，必贫——"己亥难从"是定局。
- 天干戊土遇亥水需配午、未烧之，或直接走从格。

#### 4. 淤金
- 辛金喜水金局，但辛金洗纯净，亥水中戊土鱼气会污染之。
- 庚金偏火金局，亦不喜亥水之寒。
- 亥水本为死水，难以真正泄金；强泄则引动木气、鱼气中毒。

#### 5. 净水
- 壬水不爱亥水——亥是死水而壬喜流动。
- 癸水单出癸亥勉强可用，但癸水喜下润，最好配酉丑而非亥；若配火器需位置精巧。

## 补充：亥水的地支动态关系

### 1. 亥卯未三合木 / 亥未半合
- 亥未合化木**几乎完全**：
  - 未土焦而亥水寒——阴阳平衡完美抵消。
  - 合化暗藏**丁壬化木 + 甲己化土**两种天干化气的地支呈现——极贵气。
- 合化后未土可生辛金、亥水可作用于丁火——大幅增加原局用神数量。
- **忌夹合**：两亥夹未或两未夹亥必伤婚姻（阴阳失衡，夹爆一方）。

### 2. 亥子丑三会水
- 三会本质为**刑克**，戾气极重，力量远大于三合。
- 亥水鱼气在三会中被放大，易使原局陷入心态黑化。
- 拱会（缺子）不成立——金火土戾气无处消除。

### 3. 亥亥自刑
- 与辰辰自刑对照：
  - **辰辰自刑**：开出**高压、蓬勃、有力的癸水**——可就地作用他柱。
  - **亥亥自刑**：漏出**低压、静态、游离的"无"**——淤积于亥水之下，难以平均作用。
- 亥亥自刑把**甲木、戊土（鱼）**一并漏出：
  - 心境不稳、比劫夺位、财根流失。
  - 专旺水木径反更纯净（心态相对好转，但外事内事仍受伤）。

### 4. 午亥暗合
- **地支最完美的暗合**，无之一：
  - 午火对位能量容量 >> 亥水；午中丁火与己土稳定纯粹。
  - 丁壬化气解壬水不稳，甲己化气吸纳甲木之纯，剩余火土烧尽亥中戊土之无。
  - 自积为根，不需额外地支支撑。
- 取象：**耳目清明、锚定方向**——学业为"得文星"，事业为"奔波有向"。
- **忌两亥夹午**：夹爆午火反成寒局。

### 5. 寅亥刑合
- 结构：高位壬水克寅中丙火；戊土合并（寅中新生戊土可中和亥中病戊土）；木可净化戊土。
- 作用：合化出一定程度的木气，但**不稳定**——毕竟仍是驿马换气。
- 力气最弱的驿马冲，**通常不产生驿马星变**。
- **不稳定的合化**：不可视为真合化成木。

### 6. 申亥刑合
- 结构：申中庚金行克甲木；申中戊土（带病）与亥中戊土（带毒）合并更乱。
- 作用：剧烈的**消枭夺食**——最典型如壬水内印枭相随结构中，庚金完全克进甲木食神。
- 金水难纯——金生水过程被毒化。
- 驿马星变可能性中等。

### 7. 巳亥相冲
- 典型**换气相乱**之冲：
  - 本气丙壬对冲、中气庚甲对冲、鱼气戊戊合并加重水火交战。
  - 戊戊合并后一热一冷相剧烈，非合非化。
- 极易引发驿马星变——居住、工作、方向发生周期性转折。
- 威力不如辰戌冲、丑未冲（两者本身已不稳，再冲无多大差别；辰戌本稳再冲则凶）。

## 补充：亥水的利用方法

### 活用亥未组合
- 亥未为**定海神针**式的稳定作用，类似辰土的六边形战士。
- 原局见一组亥未组合可**正本清源**，成为中正调和之核。
- 忌同时出现第二组亥未——**亥未只喜出现一次**。
- 忌亥未被冲破（未被戌冲、亥被巳/寅/申冲）。

### 午亥组合的定山河
- 午亥合为文星之象——**主学业有成、事业定向**。
- 不保证必贵，但给原局一个稳定的方向锚点。

### 禁忌与注意
- 亥未为用时**最喜地支金、午火、辰土**辅助。
- 禁忌：再见亥水、被驿马冲散、再来未土使亥夹未。
- 癸水单出癸亥可用，但不宜原局癸亥再来火土挤压使癸水上干污浊。

### 四害冲的取象
- 巳亥冲不一定为驿马星变；当原局本就有巳亥冲时，主**精神、身体、父母**的冲动。
- 若伴随财制印与官印相生两种病叠加，冲出来的是一种**拧巴的水火交战**——气血不调、内心帝王气与亥水入墓同存。
