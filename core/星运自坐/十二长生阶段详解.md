# 十二长生阶段详解

本篇汇总**十二长生**各阶段的古典意象、记忆口诀、宫位类象与日柱特征——作为**古籍标准**用于辅助判断。

> 总原则：十二长生在八字命理中**只起辅助作用**，不能作为主要依据；类象准确率并非绝对，只能作为参考。

---

## 衰：盛极而衰

**衰**即盛极而衰——万物开始走下坡路，是一个**质变**的阶段。人在此时气息衰弱、精神疲惫、力不从心；新东西到了这个阶段也变成旧的，还要面对其他新事物的挑战。

### 记忆口诀（以日主为基准）
- **甲日见辰、乙日见丑、丙日见未、丁日见辰**
- **戊日见未、己日见辰、庚日见戌、辛日见未**
- **壬日见丑、癸日见戌**

### 衰在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 出生时环境不好，小时候生活艰难；身体虚弱、性格内向；小心被人拖累、可能负债；一生多凶兆，晚年辛苦 |
| **月柱** | 青年时期运势不好，做事容易受打击挫折；前半生顺利、后半生辛苦；注意帮人担保或借出去的钱收不回来 |
| **日柱** | 婚姻运不好、异性缘差；成家后难过上幸福生活；事业按部就班反而能有成就 |
| **时柱** | 晚年运势差，身体弱；子女身体不太好；对子女管教要特别注意，不然会因子女而不得安宁 |

### 自坐衰的典型日柱
**甲辰日、乙丑日、庚戌日、辛未日**

### 自坐衰的特征
1. 性格温顺，做事有点消极；**保守、不积极**，对事业光想不做，没什么闯劲；不爱出风头；对事物发展抱顺其自然态度，**一步一个脚印**，可能到中年事业或职位才有成就。
2. **技术研究方面可能有才能或成就**——适合做会计、金融等固定型工作，或学术、技艺、久留行业。
3. 先天**六亲缘分薄**，父母恩情少；婚缘不太好，往往交往或结婚后才发现一切不如原来预期。健康方面注意肠胃、脑神经痛，一辈子要小心意外事故。

### 核心提示
衰是**最糟心**的阶段——前期顺风顺水突然大转弯，各方面都出问题，走势从上升变成下降——很多事情没法一下子适应，**所以做啥都觉得困难重重**。

## 病：千疮百孔

**病**就像人得了病——身体虚弱、多灾多病、精神不振、运气不好。旧事物已经千疮百孔、积重难返。

### 记忆口诀
- **甲日见巳、乙日见子、丙日见申、丁日见卯**
- **戊日见申、己日见卯、庚日见亥、辛日见午**
- **壬日见寅、癸日见酉**

### 病在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 父母身体不佳；自己在运势低落时出生，先天可能体弱多病 |
| **月柱** | 内心悲观消极，自寻烦恼，不思进取；缺乏实践能力；中年之前要注意，可能经常换工作 |
| **日柱** | 配偶身体不好，自己也要注意身体保养；婚姻很难美满 |
| **时柱** | 晚年工作或失业波折多，可能经济困难难得清闲；子女身体也不好，容易生病 |

### 自坐病的典型日柱
**丁卯日、戊申日、己卯日、壬寅日、癸酉日**

### 自坐病的特征
1. 外表看起来无忧无虑，内心却总是忧心忡忡；**同情心强，为人尽力能得人缘**；但和家人缘分淡薄。
2. 举止缓慢，选择太多，做事过于敏感，过分考虑个人得失，因此**错失很多良机**。
3. 优点是对于交代的事情大多能**尽心尽责**；这种命多才多艺，适合从事**设计、企划、记忆、发展**等工作。生性意志薄弱，容易幻想，缺乏果断和实践能力。

## 死：能量最低

**死**就如人老之后身体死亡——万物都消亡了，力量和能量到了最低点。

### 记忆口诀
- **甲日见午、乙日见亥、丙日见酉、丁日见寅**
- **戊日见酉、己日见寅、庚日见子、辛日见巳**
- **壬日见卯、癸日见申**

### 死在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 小时候性格消极，没有积极向上的进取心；六亲缘分薄 |
| **月柱** | 青年时期运势不好；手足、男女朋友感情淡薄，比较无情；工作辛苦；中年之后注意健康 |
| **日柱** | 个性方面需多修养，以后容易起起伏伏；婚姻关系逐渐冷淡，夫妻缺乏共识，容易出现婚姻危机 |
| **时柱** | 子女缘分薄，得到子女的机会比较慢；个性急躁；晚年多有阻碍不顺 |

### 自坐死的典型日柱
**甲午日、乙亥日、庚子日、辛巳日**

### 自坐死的特征
1. 小时候身体多比较弱，但性格开朗、个性急躁；最大缺点可能是**因顽固、目光短浅导致优柔寡断**，缺乏决断力，经常自寻烦恼，对一件事情钻牛角尖。
2. 不管成功失败、有没有福气，都有**坚持到底不怕成败**的个性，所以会得到大家的好感。
3. 非常**勤奋努力**，大多有专门的知识和技术，一生能凭技艺获得平安顺利。
4. 男女感情表面上很好但实际比较虚，夫妻感情容易变化，甚至分居或离婚。
5. 女命行为缺乏表现力、比较畏缩柔弱；性格内向，容易想不开，多见**自我封闭、自我压抑**；属于"抓奸行"的人，要特别注意防止夫妻缘分容易变化。

## 墓：入墓归藏

**墓**如人死后埋进坟墓——也可理解为万物成熟或生产完成后归入仓库，象征旧事物死亡后的遗迹被送进博物馆或收入仓库。**也称"库"**。

### 记忆口诀
- **甲日见未、乙日见戌、丙日见戌、丁日见丑**
- **戊日见戌、己日见丑、庚日见丑、辛日见辰**
- **壬日见辰、癸日见未**

### 墓在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 出生在经济比较落后的家庭，小时候比较穷 |
| **月柱** | 人比较节俭会存钱，但收入不多；中年以后会有转机 |
| **日柱** | 总是过得很辛苦，一生变化比较大；生活吝啬节俭；中年以后可以大展宏图，有很强的赚钱能力 |
| **时柱** | 孩子可能比较少，或让人操心烦恼；晚年能有成果，可以享受辛苦的成果 |

### 自坐墓的典型日柱
**丙戌日、丁丑日、戊戌日、己丑日、壬辰日、癸未日**

### 自坐墓的特征
1. 不喜欢打扮、外表性格内向；做事有条理、为人有操守；**但心胸有点狭窄，听不进别人的意见**，会疏远亲友，夫妻之间平淡相处。
2. 大多出生在贫穷家庭，先天六亲缘分淡薄；小时候离开家吃苦；壮年之前家里经常搬家，必须承担照顾家人的责任。
3. 心里多忧愁烦恼的命格，但有**很好的事业和财运**，最适合做实业或在企业工作，有相当强的赚钱能力。一般在中年期间得到发展，但没走好运之前盛衰起伏很大。
4. **婚姻障碍比较多**，婚姻看起来不错其实经常有争吵；女命做事很有条理，人也简朴，但对老公**疑心比较重**。

## 绝：气息断绝

**绝**即形体灭绝化为泥土，也叫"受气"——就像人还没受孕时母亲肚子里空空如也——万物前气已绝、后气还未来；比喻**气息断绝、没有生机**。

### 记忆口诀
- **甲日见申、乙日见酉、丙日见亥、丁日见子**
- **戊日见亥、己日见子、庚日见寅、辛日见卯**
- **壬日见巳、癸日见午**

### 绝在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 得不到祖业，能在他乡白手起家；中年后财运好，晚年平安顺遂 |
| **月柱** | 不擅长交际、会避开大众而孤立；心情起伏不定；一生多变动——多因个人主观意识导致对现实不满，最终孤立无援，朋友不多 |
| **日柱** | 感情方面夫妻容易分离，总是喜新厌旧，花哨风流冲动急躁；事业上在他乡经营必定成功，但要注意不要轻易相信别人 |
| **时柱** | 容易对子女完全放任不管，最后得不到子女奉养；要关注子女性格发展，加强教育 |

### 自坐绝的典型日柱
**甲申日、乙酉日、庚寅日、辛卯日**

### 自坐绝的特征
1. 日干坐绝的人**喜欢群居生活，耐不住寂寞**；做事缺乏长远规划和思考；爱好广泛，凡事只有三分钟热度；**不太适合固定踏实的工作**，大多在艺术、自由职业、娱乐等领域有不错的发展。
2. 才华不错，思考和理解力也敏锐；行为处事比较**以自我为中心**，要多注意别人的举动；性格善良但个性急躁，做事缺乏坚持和执着，容易发怒、**有始无终**；个性胆怯轻率多嘴，容易被人误解。
3. 女命个性开朗，有时因语言行为过于开放容易受诱惑，经常因为喜欢而喜欢——婚姻缘分容易发生变化。感情上容易受骗导致情路坎坷，**最好晚婚或嫁给年纪大的男人**，可能比较吉利。

## 胎：万物萌芽

**胎**即受胎受孕——就像人在父母之气的结合下聚集成胎，指天地之气相交、后续之气来临并受胎，**处于万物萌芽的状态**。

### 记忆口诀
- **甲日见酉、乙日见申、丙日见子、丁日见亥**
- **戊日见子、己日见亥、庚日见卯、辛日见寅**
- **壬日见午、癸日见巳**

### 胎在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 出生时正值父母变动期；性格开朗，意志不坚定，容易受他人影响；身体状况较差，需要注意保养 |
| **月柱** | 青年时期工作和事业起伏不定，可能经常变换；**中年后吉祥顺遂，很有发展** |
| **日柱** | 能说会道的人，容易搬弄是非，祸从口出 |
| **时柱** | 女命怀孕几率高，容易得子；女子个性比较内向，要加强教育 |

### 自坐胎的典型日柱
**丙子日、丁亥日、戊子日、己亥日、壬午日、癸巳日**

### 自坐胎的特征
1. 日干坐胎总体**命运不错**，基本属于理想型的人——对事物充满好奇心，研究心理较强，安于现状，属于**大智若愚**的人。
2. 容易受环境变迁支配，做事缺乏一贯性——一生中若能培养做事的信念和持续力，将来成就事业的几率会更加顺利。**交际手腕不错，口才也好**。
3. 职业常见变化，比较适合自由职业；**事业在中年时开运**；一生要防止情色风波、防止再婚。
4. 幼年时多病体弱，青年后大多健康；一生中可能有一场大病徘徊在生死之间。**男命最好晚婚，女命要防止初婚不顺**。

## 养：胎腹孕育

**养**即人在母腹中孕育到初生——万物在自然中形成然后萌发，又要经历一个**生灭不息**的天道循环过程。

### 记忆口诀
- **甲日见戌、乙日见未、丙日见丑、丁日见戌**
- **戊日见丑、己日见戌、庚日见辰、辛日见丑**
- **壬日见未、癸日见辰**

### 养在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 一生容易受父母熏陶和培养，与双亲缘分深厚；但一生的观念和行为受母亲影响很大 |
| **月柱** | 可能会因异性问题而产生纠葛，会追求婚外情 |
| **日柱** | 婚姻看似美满和谐，但因多情而留下遗憾 |
| **时柱** | 晚年子女孝顺，也会奉养自己 |

### 自坐养的典型日柱
**甲戌日、乙未日、庚辰日、辛丑日**

### 自坐养的特征
1. 日干坐养**有进取心**，如果不断努力容易取得较大成功；一般**受母亲影响很深**；为人处世温和，即使在非常不和谐状态下也能识大体顾大局，并继承家业——但与兄弟不合。
2. 个性可以说是**八面玲珑**，为人诚实，做事风格踏实，但做事**比较缺乏积极和耐心**。
3. 男女都有感情问题，甚至总是不合。男命不适合创办事业，最好依靠别人，反而能获得相当的成就和发展。女命工作和婚姻要到晚年才能吉祥安稳；为人多有洁癖，做事比较拖泥带水，往往遇事不能随机应变。

## 现阶段学习重点

1. **长生** 阶段：生机勃勃、劲头十足、很有力量。
2. **临官、帝旺** 阶段：力量特别强，尤其帝旺可能过强 → 需注意脾气暴躁、固执己见。
3. **死、绝** 之地：力量非常弱，就算在五行规则上可以克别人，但**由于太弱可能也克不动**。

十二长生要联系**天干地支作用关系、盖头截脚、负载**综合考虑。
