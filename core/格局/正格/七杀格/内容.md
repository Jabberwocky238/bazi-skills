# 七杀格

## 定义
- **七杀**：与日主阴阳相同之克——阳日主见阳克我之五行为七杀；阴日主见阴克我之五行为七杀。
  - 例：甲日见庚（阳金）为七杀；乙日见辛（阴金）为七杀。
- **七杀格**：**月令取七杀为用神**的基础正格（亦称"偏官格"）。
- 正格十格之一——**刚克之用**，贵而险。
- 本质：七杀烈性强于正官，如猛虎——核心吉凶在"**制化得当**"。

## 核心：刚克之用
- 七杀 = 高压、挑战、外部强克、危难考验。
- 相对正官（柔克）：七杀是**刚克**——压迫性强、毫不留情。
- **无制之七杀** = 纯粹的克身 → 必凶。
- **有制或有化之七杀** = 可为用神 → 成贵格，且**比正官更出彩**。
- 古语："**大贵之格多带七杀**"——凶神得制，成就远超吉神。

## 七杀格 vs 正官格

| 对比 | 七杀格 | 正官格 |
|---|---|---|
| 贵气类型 | **偏武贵** | 偏文贵 |
| 克我方式 | 刚克（同阴阳之克） | 柔克（异阴阳之克） |
| 典型结构 | 杀印相生 / 食神制杀 / 羊刃驾杀 | 官印相生 |
| 岗位特征 | 掌实权、行事霸道果决、需他人敬畏 | 多文职统筹、性格温和端庄、重礼仪 |
| 凶险度 | 高（无制必凶） | 低（正官本身温和） |
| 贵气峰值 | 极高（一将制成则过人） | 平稳 |

## 七杀格的三种用法
核心总纲：**七杀为用，制之以食、化之以印、阳日以刃敌之**。

| 用法 | 动作 | 日主状态要求 | 对应格局 |
|---|---|---|---|
| **食神制杀** | 食神克七杀 | 身强、食神有力 | 见 `食神制杀.md` |
| **杀印相生** | 印化七杀 | 身弱、有印 | 见 `杀印相生.md` |
| **羊刃驾杀** | 刃敌七杀 | 阳日主 + 刃 + 身极强 | 见 `羊刃驾杀.md` |

三种用法各自独立成格，本文专论七杀格的**基础结构**与总纲。
## 七杀格取用四大要诀

### 核心总纲
七杀**烈性强于正官**——规矩要少一些；核心**必须制伏**；制化工具（食伤、印）**必须为喜用**——"**喜用就相当于工具趁手**"则格局显贵；用**忌神制杀则格局大降**。

### 要诀 1：身强首选食伤制杀
- 如"**拿趁手兵器打虎、借力显贵**"。
- **忌印星克食伤**——否则削弱制杀力、反降级为杀印相生。

### 要诀 2：身弱必用印星化杀
- 杀众身清虚 → **印是转化利器**——即"**杀印相生**"。
- **忌财破印**——财克印，不仅化不了杀、还会造成杀更旺。

### 要诀 3：食神制杀、杀印相生均忌见财
- **食神制杀见财** → 食神贪生财忘克杀 → 通关去了就不制杀了。
- **杀印相生见财** → **财坏印**、不仅化不了杀、反致杀更旺。

### 要诀 4：阴阳日干差异
- **阳日干太刚猛**：身强才用食伤制杀；阳日干身弱用食伤 → 一招灾祸。
- **阴日干身弱可食神制杀**：如乙木、丁火柔和可制辛金、酉金，不似阳干刚猛互克。

## 格局层级与制化

### 上等格局
- **食神制杀贵格**：身强、食干透杀——"**雷霆手段 + 菩萨心肠**"。
- **杀印相生**：**身强可食干掌权**、身弱靠印化杀。
- **羊刃驾杀**：阳日身刃两停抗杀 → 将帅之象。

### 下等格局
- **杀旺无制** → 贫贱、灾祸。
- **伤官见杀无制** → 易伤残、破相。
- **女命官杀混杂** → 烂桃花、是非。

### 制化原则
- **身强** → 用**食伤制杀**。
- **身弱** → 用**印星化杀**。
- **无印无比劫 + 全局杀旺** → 可从杀；否则必制化。
- **阳日主身极强** → 羊刃驾杀可成。

## 男女命差异

### 男命
- 七杀主**官位实权**——制化得当则领导能力强、乱世用杀更易崛起；无制则暴躁惹祸。

### 女命
- 七杀主**异性缘**——官杀纯净代表配偶事业能力，混杂则**仅为烂桃花**无实义；须化杀，否则入歧途。

## 官杀混杂处理
- **必须去一留一**——去官留杀或去杀留官。
- **本命局自带化解配置者贵气最纯**；大运流年化解 → 效果差、一生是非。
- 常见去留手段：
  - 合去其中一方（如伤官合官 → 留杀）。
  - 冲去其中一方。
  - 食伤制之、印化之。
- **关键提醒**：羊刃驾杀也是一种特殊的制化路径——详见 `羊刃格.md`、`羊刃驾杀.md`。

## 判断维度

### 1. 食制 vs 印化 vs 刃敌三路之别
- **食制**：主动解决型 → 事业心强、能成事。见 `食神制杀.md`。
- **印化**：被动接纳型 → 抗压内化、学识声望。见 `杀印相生.md`。
- **刃敌**：刚对刚 → 武职、将帅气质。见 `羊刃驾杀.md`。

### 2. 七杀的纯度
- **纯七杀成局**：杀旺成势而身有制化 → 典型贵格。
- **官杀混杂**：气不清 → 格次一级。

### 3. 财的角色
- **财生杀**：身强有制的前提下，财助杀反为贵。
- **财破印**（若为杀印格）：印被破 → 转化链断 → 忌。

### 4. 大运流年
- **食伤运**（食制七杀者）：制杀加强 → 贵格兑现。
- **印运**（杀印相生者）：化杀加强 → 学业 / 声望上升。
- **官运**（命中有正官）：引动官杀混杂 → 需警惕。
- **食伤运**（命中无食制）：可能触发伤官见官 → 警惕。

## 制杀结构的常见偏差

### 制杀太过
- 食伤力量远大于七杀 + 杀无根不透 → 七杀被压服过度。
- 现实：性格偏柔、权威难显、无震慑力、难以整合资源做大做强。
- 补救：大运补杀地（增强七杀）、流年补日主根基。

### 制杀不及
- 七杀旺而食伤或印微弱 → 制化无效。
- 现实：被压、多灾、事业起伏大。
- 补救：大运补制化之神。

### 食神制杀被财坏
- 原局食神制杀成局 + 财星贴食神 → 食神贪生财不制杀 → 杀重新攻身。
- 现实：格局降档、事业遇挫。

### 杀印相生被财坏
- 原局杀印相生成局 + 财星贴印 → 财克印 → 杀失化 → 反攻身。
- 现实：**财坏印**的典型破格。

## 常见误区

### 1. "七杀 = 凶煞"
- 最大的误解。七杀**本身无吉凶**：
  - 有制有化 → 贵格。
  - 无制无化 → 凶。
- 关键是"**治杀之物**"在不在。

### 2. 把七杀等同于正官
- 阳日阳克 = 七杀；阳日阴克 = 正官。
- 阴日阴克 = 七杀；阴日阳克 = 正官。
- 错判属性 → 用法全错。

### 3. 以为"七杀格必贵"
- 七杀格要成真贵，需**制化得当 + 身能任**。
- 身弱而杀无制 → 一生被压、多灾。
- 身弱而杀满局无制无印比 → 转入从杀格讨论（极罕见）。

### 4. 机械套用"食神制杀必贵"
- 食神制杀的关键是"**两停**"——食杀势均力敌。
- 食过强压杀过尽 → 制杀太过，权威不显。
- 食过弱压不住杀 → 制杀不及，反受杀伤。

## 现实表现

### 性格核心
- **机敏、坚韧、事业心极强、不容权威被挑战**。
- **抗压能力强**、越险越冷静。
- **决断力强、不犹豫**。
- 偏杀者说狠话有震慑力。
- 身上带一种"**不好惹**"的气场。

### 实用建议
- **杀旺无制** → 多修心性、选规则性岗位。
- **身弱杀旺** → 依附贵人。
- **女命官杀混杂** → 理清异性关系、避免是非。
- **官杀旺流年** → 易熬夜、焦虑，需注意身心调节。

### 职业路径
- **军警、武职**：军官、特警、情报。
- **高压决策岗**：企业家、投行、谈判专家。
- **外科医生、急救医师**：需要决断与刀兵之气。
- **执法、监察、司法**：法官、检察官、纪检。
- **改革派 / 破局者**：在旧秩序中杀出新路。

### 成就特征
- **非循规路径**：不靠学历资历慢慢爬。
- **关键一战立威**：一次重大决断奠定地位。
- **大起大落**：命运曲线远比正官格陡峭。

## 核心判断
- 七杀格的核心在"**治杀**"二字——食制、印化、刃敌三路必有其一。
- 制化得宜 + 身能任 → 贵而有成，甚至胜过正官格。
- 制化失当 + 身不任 → 一生被压，成就难显。
- **平衡与专一**：
  - **制化工具需趁手**——喜用力量配比均衡、杀略强于制化工具。
  - **格局需清纯**——官杀不混杂。
- 无论身强身弱——**找准制化路径、契合运势助力** → 就能将杀的烈性转化为贵气、实现事业与生活的顺遂。

## 一句话总结
- 七杀 = **高压 + 挑战 + 危难**。
- 有办法驾驭的七杀 = 让人成大事的燃料。
- 驾驭不了的七杀 = 把人烧干的火 —— 关键看**制化之物在不在、牢不牢**。
