# 自坐临官（自坐禄）

## 定义
- **临官**：十二长生的第四阶段，象征**青年人成年立业、获得官职与地位**。
- 又称"**禄位**" → 每个天干的禄 = 临官位。
- **自坐临官**：**日主天干坐在自己的禄位上** → 日主**自带底气、自立自强**。
- 各天干禄位：
  - **甲** 禄在 **寅**。
  - **乙** 禄在 **卯**。
  - **丙戊** 禄在 **巳**。
  - **丁己** 禄在 **午**。
  - **庚** 禄在 **申**。
  - **辛** 禄在 **酉**。
  - **壬** 禄在 **亥**。
  - **癸** 禄在 **子**。

## 典型日柱
- **甲寅、乙卯、丙巳、丁午、戊巳、己午、庚申、辛酉、壬亥、癸子**。
- 这些日柱**日主天干 + 日支本气五行相同** → **最典型的"自坐禄"**。

## 核心：自立自强、脚踏实地
- 禄 = **既成年又得位** → 人的"**壮年立业期**"。
- 自坐临官的人：
  - **脚下就是自己的地盘**。
  - **不依赖他人、自立自强**。
  - **端正、有分寸**（不像帝旺那样刚烈）。
- **比帝旺稳，比冠带熟**：介于上升与顶峰之间的**最舒服状态**。

## 与相关结构对比

| 结构 | 能量状态 | 气质 |
|---|---|---|
| **自坐临官（禄）**（本条） | 青年立业 | 自立、稳重、有底气 |
| **自坐帝旺** | 壮年顶峰 | 刚烈、带锋 |
| **自坐冠带** | 加冠即将立 | 上升、青涩 |
| **自坐长生** | 初生 | 柔软、有望 |
| **建禄格** | 月令为禄 | 扩大版的自坐禄 |

**自坐临官 + 月令为禄 = 建禄格**（见格局/建禄格.md）→ **底盘极稳**。

## 判断维度

### 1. 是否叠加建禄格
- 日支为禄 + 月令也为禄 = 建禄格 → 身强根深。
- 仅日支为禄 → 自坐禄 → 局部性强。

### 2. 天干是否再有比劫 / 印
- 再加比劫 / 印 → 身特别强 → 需要出口（食伤 / 官杀 / 财）。
- 没再加 → 身中偏强 → 格局端正。

### 3. 配合什么出路
- **食伤泄秀**：才华型。
- **财为用**：实业型。
- **官杀为用**：仕途 / 管理型。
- **无出口**：徒有其气，难以大成。

### 4. 大运流年
- 再走禄 / 比劫 / 印 → 身更旺 → 忌。
- 走财官食伤 → 有去向 → 喜。
- 禄位被冲 → 关键节点。

## 常见误区

### 1. "自坐禄 = 命一定好"
- 自坐禄**只保证"站得稳"**，不保证"飞得高"。
- 要**成就**，还需要：
  - 大运给机会。
  - 有合适的出口（财 / 官 / 食伤）。

### 2. 把禄等同"高位 / 做官"
- 禄原义是"**禄位**"（俸禄 / 职位），但在现代语境下更多是"**有底气立身**"。
- 不一定做官，可能是**独立专业人士、企业家、自由职业者**。
- 核心是"**不依赖他人而立**"。

### 3. 以为禄不能再补
- 禄是起点，不是终点。
- 大运再补食伤 / 财 / 官 → 才是完整的兑现。

### 4. 忽视禄被冲的风险
- 如甲寅日 → 申年冲寅 → 禄位动摇。
- 冲禄之年 = 关键节点 → 事业 / 人事 / 健康可能有重大变动。

## 现实表现

### 性格
- **自立、不求人**。
- **做事稳重、不浮躁**。
- **有分寸、讲原则**。
- **独立判断能力强**。
- **朋友少而真**：不混圈子但有坚定的同伴。
- **骨子里的"**自尊**"**：被轻视时反应很大。

### 事业
- **独立型岗位**：自由职业、合伙人、独立专家、独立创业。
- **稳定成长型**：一个领域深耕十几年。
- **不走捷径**：靠**积累 + 实力**立身。
- **典型画像**：资深医生、执业律师、独立顾问、技术专家、个体店主。

### 成长节奏
- **青年期即开始独立**：早早离家、早早立业。
- **中年期稳步上升**：不暴富但不穷。
- **晚年稳定**：禄位之稳延续到晚年。

### 感情
- **不轻易依赖人**：哪怕伴侣也保持独立性。
- **重视精神 / 人格上的平等**。
- **不喜欢黏人 / 控制型关系**。
- **婚姻往往稳定**：一旦选定就不随意动摇。
- **对伴侣的要求**：**独立 + 懂自己**。

### 身体
- 禄 = 气正旺稳 → 身体**底子好、耐力强**。
- **少大病**：但可能有**慢性、长期的小问题**（工作型疲劳）。
- **抗压能力强**。
- **长寿比例高**。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 食伤运 | **喜** → 泄秀成才 |
| 财运 | **喜** → 身能任财 |
| 官杀运（有制） | **喜** → 事业加持 |
| 印运 | 中性偏忌 → 身过旺易闷 |
| 比劫 / 禄运 | 视结构 → 加强根基但也可能群比夺财 |
| **冲禄位的大运** | **关键节点** → 人生转折 |

## 核心判断
- 自坐临官的底色 = **自立**。
- 这是十二长生里**最"平衡"**的一个阶段：
  - 不像帝旺那样刚烈过头。
  - 不像冠带那样青涩不定。
  - 不像长生那样还需长。
- 识别意义：
  - **给自己的人生选"独立"的路径**。
  - **不要硬把自己塞进依附型结构里**（那样会内耗）。

## 一句话总结
- 自坐临官 = **站在自己的禄位上**。
- 这种人**一辈子都不求人、但也不孤独** —— 因为他们**自己就是自己的底气**。
- 稳、立、独 —— 是自坐禄最美的三个字。
