# 干支作用（盖头 / 截脚 / 覆载）

本目录收录八字分析中**同柱内天干与地支直接作用关系**的三个核心条目。仅凭五行数量判断力量强弱是片面的——必须关注干支之间的真实作用，才能准确估算八字各五行的力量。

## 三文件索引

- [盖头.md](./盖头.md) —— 天干克地支（上克下），地支根基被压。
- [截脚.md](./截脚.md) —— 地支克天干（下克上），天干外显气被截。
- [覆载.md](./覆载.md) —— 天地同气或相生，整柱能量一体。

## 三者关系一览

| 关系 | 天干 vs 地支 | 能量效果 | 象意 |
|---|---|---|---|
| **盖头** | 天干克地支 | 地支被压制 | 外显压内在 |
| **截脚** | 地支克天干 | 天干无根基 | 根基反克上层 |
| **覆载** | 同气或相生 | 天地一体 | 能量最稳 |

- **盖头、截脚** = 同柱内的**对抗关系**——柱力打折。
- **覆载** = 同柱内的**合作关系**——柱力饱满。
- 三者共同描述：**一柱的真实出力能力**。

## 命理分析中的使用顺序

### 第一步：扫描全盘
- 遍历四柱，逐柱判断属于**盖头、截脚、覆载**哪一类（或三者皆不是的中性柱）。
- 先不考虑全局合化冲刑，只看同柱天干地支的直接关系。

### 第二步：辨清性质
- **盖头柱**：标记地支本气被削的程度（纯气直克 / 本气被克 / 中余气被克）。
- **截脚柱**：标记天干是否有他柱通根（全局无根 vs 他柱有根）。
- **覆载柱**：标记是同气还是相生，是强旺型还是流通型。

### 第三步：结合全局修正
判断须受以下关系修正：
- **合化**：合化成功可改变五行方向，盖头截脚可能消失，覆载也可能被拆。
- **三合三会**：地支成局，地支力量大增，可能反转盖头、截脚。
- **刑冲**：冲开地支可能卸掉盖头或瓦解截脚；冲散覆载亦须警惕。
- **通关**：通关五行出现，化克为生，盖头截脚的克力被化解。

### 第四步：落到喜忌
- 被盖头/截脚的是**喜神** → 命局发力点受损，不利。
- 被盖头/截脚的是**忌神** → 反成约束，未必坏。
- 覆载的是喜神 → 主气牢靠，富贵寿考有所依托。
- 覆载的是忌神 → 麻烦深扎难除。

### 第五步：对应宫位应事
- **年柱**：祖上、早年、外部环境。
- **月柱**：格局、气势、父母、事业平台。
- **日柱**：自身、配偶宫。
- **时柱**：子女、晚景、事情结果层。

## 基础规则回顾

### 天干与天干
- 相**生、克、合、冲**四种关系。

### 地支与地支
- 主要论**合、冲、刑、害**。
- 一般**不论相生、相克**——地支之间的相克作用很小。

### 天干与地支
- 主要通过**盖头、截脚、覆载**产生作用。
- 盖头、截脚**限于同柱**。
- 天透地藏的覆载**可以跨柱**（通根关系）。
- 其他情况下，天干与非同柱地支一般**无法直接作用**，须借其他干支桥接产生间接关系。

## 一句话总括

**盖头压根、截脚断气、覆载一体**——三者是判断一柱真实出力能力的基础，也是从"五行数量"走向"五行力量"的必经一步。
