# 伤官合杀

## 成立条件（严格依《子平真诠》《滴天髓》《渊海子平》）

### 条件 1：日主必为阴干（乙、丁、己、辛、癸之一）

- **古籍出处**：
  - 《渊海子平·论合》："甲己合、乙庚合、丙辛合、丁壬合、戊癸合。"
  - 《子平真诠·论伤官》："**合杀者，取其合以去杀**。"
- **客观判据（结构推演）**：
  - 伤官 = 我生之异性；七杀 = 克我之同性。
  - 逐日主枚举伤官与七杀所构成的天干对：
    - 甲日：伤官丁 vs 七杀庚 — 丁庚**不成五合**；
    - 乙日：伤官丙 vs 七杀辛 — **丙辛合** ✓；
    - 丙日：伤官己 vs 七杀壬 — 己壬**不成五合**；
    - 丁日：伤官戊 vs 七杀癸 — **戊癸合** ✓；
    - 戊日：伤官辛 vs 七杀甲 — 辛甲**不成五合**；
    - 己日：伤官庚 vs 七杀乙 — **乙庚合** ✓；
    - 庚日：伤官癸 vs 七杀丙 — 癸丙**不成五合**；
    - 辛日：伤官壬 vs 七杀丁 — **丁壬合** ✓；
    - 壬日：伤官乙 vs 七杀戊 — 乙戊**不成五合**；
    - 癸日：伤官甲 vs 七杀己 — **甲己合** ✓。
  - → **仅乙、丁、己、辛、癸五阴日主**在结构上可形成伤官与七杀之五合。
- **不满足则破**：甲、丙、戊、庚、壬五阳日主 → 伤官与七杀结构上无五合关系，本格根本不适用。

### 条件 2：伤官与七杀均透于天干

- **古籍出处**：
  - 《渊海子平·论合》："天干五合，合而化者从化论；不化者，羁绊而已。"（五合只在天干）
- **客观判据**：
  - 伤官与七杀均透于年 / 月 / 日 / 时天干——五合是天干现象，地支不论。
- **不满足则破**：其一仅藏于地支 → 无合可成。

### 条件 3：合的位置紧贴（同柱或相邻柱）

- **古籍出处**：
  - 《滴天髓·通神论》："**合之力，以紧贴为真**。"
  - 《子平真诠》："隔柱而合，合而不力。"
- **客观判据**：
  - 伤官与七杀两天干之间：月 / 日、日 / 时相邻，或月 / 时（间隔一柱，需条件 4 无争合才作真合）。
- **不满足则破**：年 / 时两端遥合 → 合力微弱，不作真合论。

### 条件 4：无其他天干争合（非"争合""妒合"）

- **古籍出处**：
  - 《渊海子平·论合》："**一物可合，二物不可合**——两干争一，谓之争合。"
- **客观判据**：
  - 原局无其他天干与伤官或七杀构成同一五合关系（如己日论伤官合杀，乙庚合；若另有一庚在年干争合，即为争合）。
- **不满足则破**：争合 → 合力分散，两边都合不实，气机混乱。

### 条件 5：合化或不化，其化神（若化）对日主非忌

- **古籍出处**（本条最核心）：
  - 《滴天髓·合化》："**合为解，合为化。解而不化，两神相绊而各不为害；化而成象，另起一局**。"
  - 《渊海子平·论合》："合而化者从化论；不化者，羁绊而已。"
- **客观判据**（两路分处）：
  - ① **不化路径**（月令五行不支持合化）：伤官与七杀"相绊而各不为害"——日主得清净，本格典型吉象。
  - ② **化成路径**（月令五行支持合化，且合化出的五行对日主为喜用神）：化神助日主，加贵。
- **不满足则破**：月令支持合化但化出的五行对日主为忌神 → 虽合了杀，却引入新忌 → 格降或破。

---

**五条同时满足**方为真伤官合杀。本格与"伤官见官"三救应之别：佩印是"压"，生财是"泄"，合杀是"合"——五合为真合、无争合、化神不忌，三项俱备方为沈氏所谓"取其合以去杀；杀去而伤官独清"。
