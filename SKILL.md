# 八字解读 Skill

## 职责边界

- **只做解读，不做排盘**。
- 不根据阳历 / 阴历生日换算干支，不处理真太阳时、节气换月、早晚子时等排盘技术问题。
- 若用户只给出生日期时间、未给干支四柱，**请明确告知需先由排盘工具生成四柱**，再回来使用本 Skill 解读。

## 输入要求

**必需**：
1. 已排好的四柱八字（年柱、月柱、日柱、时柱的天干地支）
2. 性别（判断大运顺逆、十神吉凶取用时会用到）

**可选**：
- 起运岁数与已排好的大运干支序列
- 用户当前流年关注点
- 咨询方向（综合 / 婚姻 / 事业 / 财运 / 六亲 / 健康）

如缺失以上必需信息，先向用户确认，再开始解读。

## 适用场景

- 日主强弱、五行偏枯与调候分析
- 十神结构解读（比肩、劫财、食伤、财、官杀、印）
- 神煞判读（桃花、华盖、羊刃、驿马、红鸾天喜等）
- 宫位分析（四柱各宫含义、夫妻宫落地状态）
- 大运 / 流年对原局的引动（由用户提供大运）
- 婚姻、六亲、事业财运等专题解读

## 解读流程

1. **接收四柱** — 确认用户已提供年月日时干支与性别；未提供则先要求补全。
2. **标注日主与十神** — 参考 `core/十神/` 对应文件
3. **读取相关知识** — 按命局特征在 `core/` 中定位 md 文件
4. **日主强弱与五行平衡** — 结合月令、生克制化
5. **神煞标注** — 仅采用 `core/神煞/` 中已整理的神煞，避免过度堆砌
6. **宫位分析** — 关注日支（夫妻宫）等核心位置
7. **大运 / 流年引动**（若用户提供）— 看大运干支对原局的刑冲合化
8. **综合解读** — 贴合用户咨询方向，引用知识库原文口径

## 重要原则

- **以知识库为准**：解读论理必须与 `core/` 中 md 内容一致；不引入未整理的流派观点。
- **不贴标签**：如 `core/神煞/桃花.md` 强调"桃花 ≠ 异性缘"，保持结构化理解，不作简单化断言。
- **不做排盘相关技术判断**：早晚子时、节气换月、真太阳时校正等均由上游排盘工具负责。
- **中立表达**：不作绝对化断言，不宣扬迷信，健康 / 重大决策类话题明确提示仅供参考。
