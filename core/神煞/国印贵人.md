# 国印贵人

## 定义
- **国印**：国家的印信、权柄之象。
- **国印贵人**：主**权柄、官职、盖章画押、正式身份认证**的贵人神煞。
- 与"印星"十神含义有交集但不同：
  - **印星**：生我之五行，主学识、资源、庇佑；
  - **国印贵人**：特定地支位，主**权力印信、官印加身**。

## 起法（按年干查地支）
《三命通会·论国印贵人》原文：国印贵人按**年干查地支**。

| 年干 | 国印贵人（地支）|
|---|---|
| 甲 | 戌 |
| 乙 | 亥 |
| 丙、戊 | 丑 |
| 丁、己 | 寅 |
| 庚 | 辰 |
| 辛 | 巳 |
| 壬 | 未 |
| 癸 | 申 |

**背后原理**：国印所在位对应**印绶长生之后**的官印合位——"印章"之象。
- 例：甲年→印为癸，癸禄于子，前推至戌位为"印印相叠"之处，象征权印。

## 核心作用：官印之位
- 国印贵人**专主"盖章画押、行使权力"的场景**：
  - 持有公章、担任领导、签发文件。
  - 获得正式资质证书（职业资格、行业牌照）。
  - 被授予头衔、职位、荣誉。
  - 从事与"印章"相关的行业（法律、行政、认证、审计）。
- 古籍论："**国印入命，主掌印之权**。"

## 判断维度

### 1. 位置
- **年支国印**：家族 / 祖辈有权印之业。
- **月支国印**：事业中能掌印、管权。
- **日支国印**：自身坐权柄——典型掌印命。
- **时支国印**：晚年或子女掌权柄。

### 2. 国印配十神
- **国印 + 正官**：体制内掌印——传统意义的"为官"。
- **国印 + 印星**：学历 + 权柄 → 学术权威 / 法律专家。
- **国印 + 财星**：经营权 + 财印 → 企业高管。
- **国印 + 食伤**：创造性权柄——艺术 / 媒体的"话语权"。

### 3. 与其他贵人配合
- **国印 + 天乙贵人**：贵人相助的权柄之路。
- **国印 + 文昌**：以文立身、以印行权——典型学者型官员。
- **国印 + 将星**：统帅之位的正式权柄——将相之象。

### 4. 国印是否被破坏
- 国印所在地支被冲 → 权柄动摇。
- 被合化 → 权力转移、被架空。

## 常见误区

### 1. "带国印 = 一定当官"
- 国印是"**掌印的气场**"，不保证当官。
- 现代语境：可能是**法务、审计、持证专业（律师、会计师、医师）、公司章管理、合规负责人**等"**印章相关岗位**"。
- **必须配合正官、财、印等十神与大运**方能转化为实际权位。

### 2. 把国印与印星混为一谈
- 国印 = **神煞**（按年干查地支），重点在"**印章**"的象。
- 印星 = **十神**（按日干与他干 / 支关系定），重点在"**生我之气**"。
- 两者可以并存，也可以单独存在。

### 3. 忽视现代化转译
- 古代"国印"专指官印，现代引申广泛：
  - 公司公章、法人章、财务章。
  - 各类职业资格证书（律师证、医师证、注会、教师证）。
  - 公众号 / 自媒体的话语权（"**影响力印章**"的现代版）。

## 现实表现

### 性格
- **重规则、讲程序**。
- **对"名分"敏感**：在不在位、有没有正式身份对他们很重要。
- **对公章 / 证书 / 文件天然尊重**。
- **责任感强**：掌了印就不容马虎。

### 职业路径
- **法律**：法官、律师、检察官、法务。
- **行政**：公务员、事业单位、大型机构行政岗。
- **专业持证**：会计师、医师、建筑师、教师等。
- **审计、合规、风控**：与"审核盖章"相关。
- **公司管理**：总经办、法人、董事。
- **文化**：主编、出版、官方媒体。

### 典型画像
- 早年读书就偏向"正规化"成长路径。
- 工作后常被任命为"要盖章负责的人"。
- 中年后持有多项资质证书或正式头衔。
- 晚年被尊为"**一方权威**"。

## 核心判断
- 国印贵人是"**权柄之路的底牌**"——让命主天然与"盖章、认证、正式身份"相关。
- 识别意义：这类人**适合需要资质、程序、权威感**的岗位，不适合完全自由的江湖赛道。

## 一句话总结
- 国印 = **手握印章的人**。
- 传统是做官、现代是做合规 / 做法务 / 做认证——**本质都是"以正式身份行权"**。
- 把国印的人放到无章可循的位置 → 憋屈；放到有规则有权柄的位置 → 得心应手。
