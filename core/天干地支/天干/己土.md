# 己土

## 定位
- 阴土。土在五行中居中，承载万物、参与转换——这是土的共性。
- 戊土偏聚成一块（讲承托整体），己土偏散布一层（讲包容与渗透）。
- "己"字本身有收敛归纳之意，抓住的是土气内向、细密、能藏纳万物的状态。

## 十神对应（以己为日主）
- 火生土为印：阳火为正印，阴火为偏印。
- 土生金为食伤。
- 木克土为官杀：阳木为正官，阴木为七杀。
- 土克水为财。
- 同类为比劫。

## 己土的特点
### 1. 依赖环境
- 不像戊土厚土成山，天然成型；更像土壤本身，需要水润火暖才能发挥作用。
- 过燥则裂，过湿则泥泞。
- 关键判断点落在"质地"，而非单纯旺衰。

### 2. 得令与失令
- 得令于辰戌丑未月，尤其丑未中藏己，有根易稳定。
- 身在寅卯之月，木旺土松；在午火生扶之月易燥。
- 关键看"能不能收纳其他五行"。

### 3. 与水的关系
- 己土比戊土更易被水改变形态。
- 适度之水润土育物。
- 水多无火，土变泥泞难以治水。
- 局中有火：火暖土燥，水气得以调节，土既能承载又能治水。
- 不能见"己土见财旺"便断能"长财"——缺条件。

### 4. 擅长调和
- 土本居中，己土细密，更易与其他五行交织。
- 木克土时，若火在中间，火生土而泄木，冲突缓和。
- 水多土弱时，若火土相继而来，土气重新成型。
- 很多命局里己土不是主导力量，却在维持整体稳定。

## 透干与藏支
- 透干：外在对事物的操持与收拾。
- 藏支：内部的支撑。
- 透而无根：土浮于表，看似能担实则易变。
- 藏而有根：虽不显眼却稳固。

## 常见误解
1. 直接等同于"性格温和、细心"——这只是象的延伸，不是结构判断。
   - 比劫重而官杀弱的己土日主可能执拗而不柔和。
2. "土多而不通"——离开流通谈厚薄容易看错。

## 核心判断
己土像命局里的**收纳之地**。能不能收得住取决于：
- 有没有火去生扶
- 有没有水但不过度侵蚀
- 根器是否存在

## 古典标准：己土论命思路

### 日主五行本性
- 己土为纯阴之土，号"田园之土"、"湿土"。
- 性卑湿、中正、蓄藏——低平则蓄水，湿润则生物。
- 能培木、能止水、能蓄藏，包容度极强。
- 《滴天髓》总纲：**"己土卑湿，中正蓄藏，不愁木盛，不畏水狂，火少火晦，金多金光，若要物旺，宜助宜帮。"**

### 日主逢不同月令的状态

| 月令 | 气势 | 核心矛盾 | 取用要点 |
| --- | --- | --- | --- |
| 寅卯月（春） | 木旺克土 | 气虚寒湿 | 丙火暖、癸水润、甲木疏 |
| 辰月 | 本气通根 | 湿土得令 | 配火则秀 |
| 巳月 | 火旺生土 | 渐燥 | 癸水为急 |
| 午未月 | 火炎土燥 | 焦裂 | 癸水润、丙火辅——癸丙并透则贵 |
| 申月 | 金泄气 | 土秀金多 | 火来生助，不宜再水 |
| 酉月 | 身弱秀灵 | 食神当令 | 印星扶助 |
| 戌月 | 燥土当令 | 秀气被锁 | 用癸水开库润之 |
| 亥子丑月（冬） | 水旺土冻 | 湿泥无生 | 急用丙火解冻，忌壬癸再透克火 |

### 喜用神优先级

1. **调候优先**：
   - 夏月先取癸水，无水则焦烈无用。
   - 冬月先取丙火，无火则废。
2. **扶抑次之**：
   - 身弱用丙丁印、戊土帮；身强用金泄、水耗。
3. **通关再次**：
   - 木克土时，火生土泄木，冲突转为印化。
4. **病药殿后**：
   - 己土若无官印，是"没有担当的人，不愿出头"——病在无丙火正印；药在丙火透干。

### 与其他九个天干的关系

- **甲（阳木）**：正官，甲己相合化土——不怕甲克，反以合化论。
- **乙（阴木）**：七杀，乙不合己而直克，易伤身，两乙夹己发小脾气、女命情乱。
- **丙（阳火）**：正印，暖土去湿、生万物，己土最爱。
- **丁（阴火）**：偏印，火少反晦——"火少火晦"，不如丙火直接。
- **戊（阳土）**：劫财，高山冲积帮身。
- **己（同类）**：比肩。
- **庚（阳金）**：伤官，土生金而己能生好金——粗秀。
- **辛（阴金）**：食神，"金多金光"，己土所生之辛最秀最贵。
- **壬（阳水）**：正财，壬为大江，己克不动——财旺反而浑浊。
- **癸（阴水）**：偏财，雨露滋润，夏月调候第一用神。

### 不同宫位的象意

- **年柱己土**：出身温润细密、家风含蓄。
- **月柱己土**：中年以包容性行业立身（农业、文化、教育、服务）。
- **日支通根（丑、未、辰、戌）**：配偶内敛精明；己丑、己未女命身材好、文静精明。
- **时柱己土**：子女多才、晚景安定。
- 身体对应脾、腹、皮肤、胰腺；土被木克或水反克之年防脾胃与皮肤病。

### 典型贵格组合

1. **官印相生格**：甲丙并透，甲木正官被丙火正印化——"有担当"之贵。
2. **癸丙并透格**：夏月己土，癸水润、丙火暖——阳光雨露俱全，层次极高。
3. **食神生财格（金多金光）**：己土透辛金、有壬癸为财——创造性取财、技艺显贵。
4. **蓄水成器格**：己土深厚而水财有限，能蓄能藏——财库稳固。
5. **寒谷回春格**：冬月己土，丙火透而甲木引——破湿冻而成贵。

### 典型破格组合

1. **夏无癸水**：巳午未月，四柱无水——焦土无生，一生碌碌。
2. **冬无丙火**：亥子丑月，无火调候——"一辈子就是普通农民，困滞不通"。
3. **两乙夹己**：乙木七杀双透，无制无化——小脾气频发，女命感情乱。
4. **无官无印**：己土孤立，无丙无甲——缺担当，太内秀而不出头。
5. **水旺土荡**：壬癸过旺无火制——湿泥烂田，身败无依。

### 古籍出处

- 《滴天髓·天干论·己土》："己土卑湿，中正蓄藏，不愁木盛，不畏水狂，火少火晦，金多金光，若要物旺，宜助宜帮。"
- 《穷通宝鉴·三春己土、三夏己土、三秋己土、三冬己土》：分月定癸丙甲之先后。
- 《三命通会·论己土喜忌》：详论甲己合化、金生则光之理。

### 一句话总结

**己土如田园之土，卑湿中正、能蓄能藏——贵在"癸丙并透"，夏润冬暖则万物生长；不怕甲木（合化）、不畏水狂（蓄藏）、能生辛金（金多金光），最忌无丙无官而失担当。**
