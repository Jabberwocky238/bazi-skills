# 天罗地网

## 本质
- 不是神秘凶煞，而是**阴阳之气交替、五行能量进入极端死角的"地支束缚场"**。
- 核心不在名头多吓人，而在这几张网是否框住了日主核心用神，以及是否叠加七杀、羊刃等极具破坏性的十神凶器。

## 构成
- **戌亥构成天罗**（天网）。
- **辰巳构成地网**。

### 底层逻辑
- 戌：火之墓库。
- 亥：水之长生。
- 戌→亥：阳气收敛到极致、阴气骤然开启的临界点。
- 辰：水之墓库。
- 巳：火之长生。
- 辰→巳：阴气消亡、阳气猛烈爆发的交界处。

命局中同时出现"戌和亥"或"辰和巳"时：日主所处时空环境卡在阴阳交替的"混沌区"。

## 作用机制
- 在这个区域里，五行之气**不流通、不清透**。
- 形成极具阻滞感和拉扯感的磁场。
- 心智与现实中最直观的表现：**受困、停滞**。

## 是利是害，取决于三种情形

### A. 叠加七杀或羊刃的凶局
- 日主深弱 + 地支构成罗网 + 强旺七杀或羊刃。
- **天罗地网彻底变成现实中的牢笼或刑具**。
- 现实映射：
  - 极端行为、违规操作、得罪强权 → 人身自由被强制剥夺。
  - 真实的牢狱之灾、长期司法诉讼。
  - 突发严重疾病、遭遇车祸、长期卧床。
- 因为代表危险的七杀或羊刃在网中无法被食伤化泄、无法被印星转化。

### B. 困住财星或官星的纠纷局
- 原局地支成网，网的核心落在财星（财富、配偶）或官星（职位、事业）。
- 现实应验：没有肉体禁锢，但陷入**极拖沓粘稠的世俗泥潭**：
  - 资金链断裂后被三角债死死套牢。
  - 投资房产或项目变成无法变现的烂尾工程。
  - 职场遭遇极压抑的排挤，想辞职却被合同或赔偿卡住。
  - 婚姻中无休止冷暴力与拉扯，想离离不掉。
- 本质：日主的"我克"和"克我"之物丢进阴阳不交的死角，越挣扎网收得越紧。

### C. 反向成为庇护所的特殊局
- 日主身旺能扛 + 网中包裹印星（思想、学术、庇护）+ 局中带华盖或太极贵人。
- 天罗地网的物理禁锢转化为**精神隔离**——屏蔽外界凡尘俗世与社交干扰。
- 现实映射：
  - 对神秘学、哲学、宗教或高精尖技术的极度狂热与专业能力。
  - 深居简出（实验室科学家、寺庙修行者）。
- 此时天罗地网不再是刑罚，而是**日主为完成精神或技术突破主动结成的保护茧**。

## 动态引发条件
- 原局只有戌无亥，或只有辰无巳 → "网破一半"，不起决定作用。
- 大运或流年**填实了缺失的那一个地支**（如原局有戌，走亥水大运） → 网瞬间收紧。
- 流年天干透出忌神 → 命主在这一年突然遭遇官非、破财或困局。
- 网的收拢需要岁运触发。

## 核心判断
- 天罗地网的应验本质 = 原局地支**阴阳气机的极端交界对日主行为空间的强行收缩**。
- 绝不能见"辰巳戌亥"就断"牢狱之灾"。
- 必须将其作为地支的客观环境，带入日主的旺衰与十神的生克路线中考量。
- 提供的是"一种致于困住"的结构，但**困住的是身体、财富还是思想**，全看：
  - 局中七杀是否肆虐。
  - 财星是否交战。
- 脱离五行制化与十神喜忌去单论神煞吉凶，是对八字整体结构逻辑的背离。
