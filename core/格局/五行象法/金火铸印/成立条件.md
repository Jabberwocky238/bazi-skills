# 金火铸印

## 成立条件
1. **金日主 + 火官杀**（最典型）：
   - 庚辛金为日主。
   - 命局有丙丁火（七杀 / 正官）作为锻造力。
2. **金有根**：
   - 地支见申酉。
   - 天干透庚辛有根。
3. **火有根但不过旺**：
   - 火透且有根（巳午），但不能**过度压倒金**。
   - 理想比例：**金三分、火两分**（金为主，火辅）。
4. **有水辅助最佳**：
   - 金生水 → 水再制火 → 炼金的同时有冷却 → **淬火**。
   - 没水的话，纯火炼金容易**火多金熔**。
5. **无重土泄火**：
   - 土泄火过多 → 火力不足 → 炼不透。
   - 反过来，**适度土**可以稳住火不至于暴烈，也好。
