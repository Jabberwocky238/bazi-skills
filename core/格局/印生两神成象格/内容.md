# 印生两神成象格

## 定义
- **印生两神成象格**：命局**只见两种相生的五行之气**（印 + 日主 / 日主 + 食伤）→ "**两气双清**"的格局。
- 出自《滴天髓》"**两气双清**"论——两股气势集中且相生，形成命局极为纯粹的贵格。
- 五种具体形式：
  - 水木相生格
  - 木火相生格
  - 火土相生格
  - 土金相生格
  - 金水相生格

## 核心：两气双清
- "**双清**" = **只有两种五行 + 相生** → 气势纯粹。
- "**印生两神**" = 其中一气为印（生我）、另一气为我（日主或食伤）→ 形成"**印 → 身**"或"**身 → 食伤**"的顺流。
- 古籍评价："**两气双清，其象最妙**"——因气纯、流通、无杂。

## 与相关格局的区别

| 格局 | 核心区别 |
|---|---|
| **印生两神成象格**（本条）| 两气相生（印→身 或 身→食伤）、气纯为贵 |
| **两气成象（总论）** | 两气并立的总名——含相生、相克两种 |
| **水火既济** | 两气相克（水 + 火）+ 木通关——与本格构造不同 |
| **专旺格**（曲直 / 炎上等）| **一气独尊** —— 本格是**两气**而非一气 |
| **从旺 / 从强格** | 比劫 + 印或纯比劫的身极旺——本格强调两气相生的纯度 |

## 五种具体形式

### 1. 水木相生格
- 组合：壬癸水 + 甲乙木（水生木）
- 气质：智慧 + 生长 —— **清秀聪明、有学问、善开拓**。
- 典型职业：学者、文化、教育、研究、出版。

### 2. 木火相生格
- 组合：甲乙木 + 丙丁火（木生火）
- 气质：生机 + 光明 —— **热情、有魅力、有影响力**。
- 典型职业：教师、演说家、宗教、思想家、表演、文化名人。

### 3. 火土相生格
- 组合：丙丁火 + 戊己土（火生土）
- 气质：光明 + 承载 —— **温厚、稳重、有权威**。
- 典型职业：管理、建筑、地产、实业、政治。

### 4. 土金相生格
- 组合：戊己土 + 庚辛金（土生金）
- 气质：厚德 + 决断 —— **沉稳、有威严、善治事**。
- 典型职业：司法、金融、精密制造、工程、管理。

### 5. 金水相生格
- 组合：庚辛金 + 壬癸水（金生水）
- 气质：清冷 + 智慧 —— **理性、有洞察、思辨深**。
- 典型职业：研究、分析、法律、金融、科技。

## 取用要点
- **顺势而为**：岁运应顺两气相生的方向，助力双清之势。
- **忌杂气破局**：财官若来，必破双清之格。
- **岁运走到比劫、食伤、印绶地**：格局兑现，事业顺利。
- **岁运走到财官地**：**大忌**——即便原局无财官，运至财官引动也会破格。

## 现实表现

### 性格
- **气质纯粹、专注**：不喜杂事。
- **精神境界高**：常有文化、学问、思想方面的追求。
- **对某一领域极其深入**：两气相生带来的"专精"气质。

### 事业
- **深耕型专业**：一辈子只做一两件事，做到极致。
- **以学问 / 技艺立身**：不走商业投机路径。
- **气象清雅**：适合需要"清气"的领域（学术、艺术、宗教）。

### 人生节奏
- 早期显才华，中年稳步上升，晚年极高成就。
- 一旦走财官运 → 人生出现大转折（常为跌落）。

## 常见误区

### 1. "两种五行 = 两神成象"
- 必须**两气相生 + 气纯 + 无财官**，否则只是"**多五行**"不是"**双清**"。

### 2. 把两气成象等同于专旺
- 专旺是**一气独尊**；两气成象是**两气相生**——结构不同。

### 3. 忽视财官的破坏性
- 许多命书把"财官"当好物——在两气成象格里，财官是**破格大忌**。

## 古籍原文
- 《滴天髓·源流》："**两气双清，其象最妙**——相生则贵，相克则战。"
- 《元理赋》：相关论述涉及"两神成象"的吉象。
