# 七杀格

## 定义
- **七杀**：与日主阴阳相同之克——阳日主见阳克我之五行为七杀；阴日主见阴克我之五行为七杀。
  - 例：甲日见庚（阳金）为七杀；乙日见辛（阴金）为七杀。
- **七杀格**：**月令取七杀为用神**的基础正格（亦称"偏官格"）。
- 《子平真诠》十正格之一——刚克之用，贵而险。

## 核心：刚克之用
- 七杀 = 高压、挑战、外部强克、危难考验。
- 相对正官（柔克）：七杀是**刚克**——压迫性强、毫不留情。
- **无制之七杀** = 纯粹的克身 → 必凶。
- **有制或有化之七杀** = 可为用神 → 成贵格，且比正官更出彩。

## 七杀格的三种用法
《子平真诠·论七杀》总纲："**七杀为用，制之以食，化之以印**。"

| 用法 | 动作 | 结构 |
|---|---|---|
| **食神制杀** | 食神克七杀 | 主动克制（见 `食神制杀.md`）|
| **杀印相生** | 印化七杀 | 被动转化（见 `杀印相生.md`）|
| **阳刃驾杀** | 刃敌七杀 | 刚对刚两停（见 `羊刃驾杀.md`）|

三种用法各自独立成格，本文专论七杀格的**基础结构**。

## 成立条件（严格依《子平真诠·论七杀》）

### 条件 1：月令地支本气为七杀，或七杀透干且月令有根

- **古籍出处**：
  - 《子平真诠·论七杀》："**七杀即偏官**——阳见阳克、阴见阴克。"
  - 《子平真诠·论七杀》："**七杀宜透不宜藏，透则其性显**。"
- **客观判据**：
  - ① 月令地支本气为七杀；或
  - ② 月令地支藏七杀且七杀透于天干。
- **不满足则破**：月令七杀不显 → 当以月令本气另论他格。

### 条件 2：七杀不混正官（杀纯为贵）

- **古籍出处**：
  - 《子平真诠·论七杀》："**七杀最忌官杀混杂**——官杀同现，气不清。"
- **客观判据**：
  - 天干无正官透出；若地支藏正官一位，气微可忽略或需合去。
- **不满足则破**：官杀混杂 → 转入"官杀混杂"病格讨论。

### 条件 3：七杀必须有制或有化（无制无化必凶）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论七杀》："**七杀为用，制之以食，化之以印，皆为贵格**。"
  - 《子平真诠·论七杀》："**七杀无制，如虎无笼——必伤人**。"
- **客观判据**（三者至少一成立）：
  - ① **食制**：食神透干且紧贴七杀相克；
  - ② **印化**：印星透干且位于七杀与日主之间；
  - ③ **刃敌**：阳日主有刃位且与七杀两停。
- **不满足则破**：七杀无制无化 → 纯克日主 → 非贵格，转入"身弱杀重"或"从杀格"讨论。

### 条件 4：日主有根（非极弱，否则须从）

- **古籍出处**：
  - 《子平真诠·论七杀》："**身强食旺，名利双收；身弱食衰，无以制之**。"
- **客观判据**：
  - 日主于月令有比印余气以上，或他柱至少有一位比劫 / 印星通根。
- **不满足则破**：日主极弱 → 若全无比印则考虑"从杀格"；若尚有微弱比印则成"身弱杀重"破局。

### 条件 5：制杀或化杀之物有根有力

- **古籍出处**：
  - 《子平真诠·论七杀》："**食神制杀，两停则贵**——食杀平衡方妙。"
- **客观判据**：
  - 食神或印星（视用法而定）透干且在地支有本气或中气根至少一位。
- **不满足则破**：制化之神虚透无根 → 制化无力 → 沦为身弱杀重。

---

**五条同时满足**方为真七杀格。七杀格的**贵气远高于正官格**（沈氏所谓"一将制之而成名"）—— 但前提是制化得当；一旦制化失衡，凶性亦远高于正官。

## 判断维度

### 1. 食制 vs 印化 vs 刃敌三路之别
- **食制**：主动解决型 → 事业心强、能成事。见 `食神制杀.md`。
- **印化**：被动接纳型 → 抗压内化、学识声望。见 `杀印相生.md`。
- **刃敌**：刚对刚 → 武职、将帅气质。见 `羊刃驾杀.md`。

### 2. 七杀的纯度
- **纯七杀成局**：杀旺成势而身有制化 → 典型贵格。
- **官杀混杂**：气不清 → 格次一级。

### 3. 财的角色
- **财生杀**：身强有制的前提下，财助杀反为贵。
- **财破印**（若为杀印格）：印被破 → 转化链断 → 忌。

### 4. 大运流年
- **食伤运**（食制七杀者）：制杀加强 → 贵格兑现。
- **印运**（杀印相生者）：化杀加强 → 学业 / 声望上升。
- **官运**（命中有正官）：引动官杀混杂 → 需警惕。
- **食伤运**（命中无食制）：可能触发伤官见官 → 警惕。

## 常见误区

### 1. "七杀 = 凶煞"
- 最大的误解。七杀**本身无吉凶**：
  - 有制有化 → 贵格。
  - 无制无化 → 凶。
- 关键是"**治杀之物**"在不在。

### 2. 把七杀等同于正官
- 阳日阳克 = 七杀；阳日阴克 = 正官。
- 阴日阴克 = 七杀；阴日阳克 = 正官。
- 错判属性 → 用法全错。

### 3. 以为"七杀格必贵"
- 七杀格要成真贵，需**制化得当 + 身能任**。
- 身弱而杀无制 → 一生被压、多灾。
- 身弱而杀满局无制无印比 → 转入从杀格讨论（极罕见）。

## 现实表现

### 性格
- **刚烈、果敢、有魄力**。
- **抗压能力强**、越险越冷静。
- **决断力强、不犹豫**。
- **不怕冲突**：该上就上。
- 身上带一种"**不好惹**"的气场。

### 职业路径
- **军警、武职**：军官、特警、情报。
- **高压决策岗**：企业家、投行、谈判专家。
- **外科医生、急救医师**：需要决断与刀兵之气。
- **执法、监察、司法**：法官、检察官、纪检。
- **改革派 / 破局者**：在旧秩序中杀出新路。

### 成就特征
- **非循规路径**：不靠学历资历慢慢爬。
- **关键一战立威**：一次重大决断奠定地位。
- **大起大落**：命运曲线远比正官格陡峭。

## 核心判断
- 七杀格的核心在"**治杀**"二字——食制、印化、刃敌三路必有其一。
- 制化得宜 + 身能任 → 贵而有成，甚至胜过正官格。
- 制化失当 + 身不任 → 一生被压，成就难显。

## 一句话总结
- 七杀 = **高压 + 挑战 + 危难**。
- 有办法驾驭的七杀 = 让人成大事的燃料。
- 驾驭不了的七杀 = 把人烧干的火 —— 关键看**制化之物在不在、牢不牢**。
