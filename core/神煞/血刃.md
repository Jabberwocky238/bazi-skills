# 血刃

## 定义
- **血**：血光。
- **刃**：刀刃、锐器。
- **血刃**：主**刀伤、血光、手术、流血事故**的神煞。
- 与"**白虎**"功能相近但起法不同：白虎按年支起，血刃按**年支 / 日支**起（流派有别）。

## 起法（按年支查地支——常用版本）
《三命通会·论血刃》原文（以年支论血刃）：

| 年支 | 血刃位置 |
|---|---|
| 子 | 丑 |
| 丑 | 未 |
| 寅 | 寅 |
| 卯 | 酉 |
| 辰 | 辰 |
| 巳 | 亥 |
| 午 | 午 |
| 未 | 丑 |
| 申 | 申 |
| 酉 | 卯 |
| 戌 | 戌 |
| 亥 | 巳 |

**另一版本（按日干查）**：
某些流派以日干论血刃——如辛日见戌、庚日见子等。**现代命理以年支查为主**。

## 血刃 vs 白虎

| 对比项 | 血刃 | 白虎 |
|---|---|---|
| 起法 | 按年支（配合流派）| 按年支 |
| 性质 | **偏锐器、刀伤** | 偏外伤、手术、意外 |
| 典型 | 刀剑、利器、手术开刀 | 摔伤、车祸、撞击 |
| 覆盖 | 较窄（锐器相关）| 较广（血光外伤）|

血刃可视为白虎在"**锐器层面**"的细分。

## 核心作用：锐器血光的预警
- 血刃主**与刀刃、锐器相关的血光**：
  - 手术（以刀刃切入的医疗行为）。
  - 刀伤（意外或人为的切割伤）。
  - 金属伤害：机械、工具、工业事故。
  - 女命流产、剖腹产等手术性处理。
- 古籍论："**血刃主刀兵之灾**。"

## 判断维度

### 1. 位置
- **年支血刃**：早年有锐器伤经历。
- **月支血刃**：中年易遇手术或刀伤。
- **日支血刃**：本人身体直接相关——典型手术命。
- **时支血刃**：晚年或子女的手术可能性。

### 2. 血刃配其他神煞
- **血刃 + 白虎**：血光事件重叠——**手术机率大**。
- **血刃 + 羊刃**："两刃并见"，刚猛 + 血光 → 主武职 / 外科 / 危险职业，或易斗殴。
- **血刃 + 七杀**：易遇凶险，包括被暴力伤害。
- **血刃 + 灾煞**：突发性血光。

### 3. 大运流年引动
- 大运 / 流年走血刃位 → 血光事件集中。
- 原局带血刃 + 流年引动 → **手术或刀伤年**的概率极高。

## 常见误区

### 1. "血刃 = 一定出大事"
- 血刃是"**锐器血光的倾向**"，现代大部分应验为**必要手术**：
  - 拔牙、阑尾切除、剖腹产、近视矫正等。
  - 这些都算"**血刃应验**"，但不算恶性事件。
- 真正恶性血刃（如被人伤）概率极低。

### 2. 把血刃视作"注定受伤"
- 许多带血刃的人从事**外科、军警、屠宰、机械**等行业——以**职业方式消耗血刃之气**，反而不在自己身上应验。
- 这是命理"**职业化解神煞**"的典型。

### 3. 起法版本混淆
- 血刃有年支版本和日干版本。
- 使用时**以一个版本为准**，避免两版本同查而夸大严重性。
- 常用：年支版本。

## 现实表现

### 典型情境
- 一生有一至两次较明显的手术经历。
- 从事外科、牙科、整形、针灸等"用刀"的医疗职业。
- 从事金属加工、屠宰、园艺修剪等涉及锐器的工作。
- 女命：剖腹产、宫外孕手术、流产等。

### 职业化解
- **医疗切开类**：外科、牙科、妇产科——最典型的"**以职业消血刃**"。
- **工业加工类**：金属工、屠宰、厨师——也能消化血刃之气。
- **武职 / 安保**：军警等岗位。

## 化解与应对

### 结构层
- 命局有印星、贵人 → 血刃力减。
- 大运走喜神 → 血刃被压制。

### 人事层
- **必要的手术提前预案**：定期体检避免小病拖成大病（大病才需大手术）。
- **从事锐器相关职业者**：遵守安全规程，减少操作意外。
- **日常生活**：注意用刀安全、厨房安全、儿童远离锐器。

### 时间层
- **流年走血刃**：避开非必要手术（推迟整形等）。
- 若遇必须手术：**选择正规大医院，做好术前检查**。

## 核心判断
- 血刃是"**与锐器相关血光的倾向**"——现代社会应验率较高但多为**可控手术**。
- 职业选择合理（医疗、工业）→ 血刃可转化为**专业能力的标记**。

## 一句话总结
- 血刃 = **与刀刃相遇的倾向**。
- 不一定是被伤，也可能是**你成为拿刀的那个人**（医生、厨师、工匠）。
- 关键不在"命里带刀"，在"**这把刀你用来做什么**"。
