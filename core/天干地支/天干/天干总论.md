# 天干总论

## 定义
- **十天干** = 甲、乙、丙、丁、戊、己、庚、辛、壬、癸，共十个符号。
- 天干代表**天元**——显露于外、主动、能被直接观察到的力量。
- 与十二地支、地支藏干一起构成**天地人三才**。
- 天干间的**生、克、冲、合**，是八字动态分析的起点。

## 二、天干的五类作用关系

### 1. 相生
按五行相生 → **甲乙生丙丁、丙丁生戊己、戊己生庚辛、庚辛生壬癸、壬癸生甲乙**。

### 2. 相克
按五行相克 → **甲乙克戊己、戊己克壬癸、壬癸克丙丁、丙丁克庚辛、庚辛克甲乙**。
- 分**有情之克**（异性相克、五合那五组）与**无情之克**（同性相克）。

> 详见：[天干相克](../克/天干相克.md)

### 3. 相冲（天干四冲）
方位对立 → **甲庚冲、乙辛冲、丙壬冲、丁癸冲**。
- **戊己居中央，不参与相冲**。
- 冲比克力量大、更突发剧烈。

> 详见：[天干相冲](../冲/天干相冲.md)

### 4. 五合
阴阳对立统一 → 五组和谐配对：

| 组合 | 化气 | 古称 |
| --- | --- | --- |
| 甲己 | 土 | 中正之合 |
| 乙庚 | 金 | 仁义之合 |
| 丙辛 | 水 | 威制之合 |
| 丁壬 | 木 | 仁寿之合 |
| 戊癸 | 火 | 无情之合（老少之合） |

- 合有：**合化成、合而不化、合绊、争合、妒合**等形态。

> 详见：[天干五合](../合/天干五合.md)

### 5. 争合、妒合
- **争合**：两个相同天干同争一合——主感情纠葛、风流。
- **妒合**：三干相邻，一合二另一克中间干——主嫉妒、破坏。

> 详见：[天干五合](../合/天干五合.md)

## 三、天干的层级关系

从宏观到微观的天干作用：

```
天干之气（天元）
    │
    ├── 相生（五行顺生）
    ├── 相克（五行逆克）
    │     ├── 有情之克（五合的五组）
    │     └── 无情之克（阳阳、阴阴相克）
    ├── 相冲（方位对立四冲）
    └── 合
          ├── 五合（本命局内或跨大运流年）
          │     ├── 合化成
          │     ├── 合而不化（合绊、贪合忘克）
          │     └── 反化 / 夫从妻化 / 妻从夫化
          ├── 争合（两干合一干）
          └── 妒合（旁干克合中干）
```

## 四、冲、克、合三者对照

| 对比项 | 相冲 | 相克 | 相合 |
| --- | --- | --- | --- |
| 本质 | 方位对立 | 五行属性对立 | 阴阳对立统一 |
| 力度 | 最大 | 中 | 有合有克 |
| 变化 | 突发剧烈 | 渐进缓慢 | 牵制转化 |
| 戊己 | 不参与 | 参与 | 参与（戊癸合） |
| 吉凶 | 看喜忌 | 看喜忌 | 看合化条件 |

## 五、地支的五类作用关系（对照学习）

地支的作用远比天干复杂，共涵盖：
- **六合**（子丑、寅亥、卯戌、辰酉、巳申、午未）。
- **六冲**（子午、丑未、寅申、卯酉、辰戌、巳亥）。
- **三合**（申子辰水、亥卯未木、寅午戌火、巳酉丑金）。
- **三会**（亥子丑水、寅卯辰木、巳午未火、申酉戌金）。
- **相刑**（子卯、寅巳申、丑戌未、自刑）。
- **六害**（子未、丑午、寅巳、卯辰、申亥、酉戌）。
- **相破**（子酉、卯午、辰丑、戌未、寅亥、巳申）。
- **暗合、拱合**等。

> 详见：[十二地支总论](../地支/十二地支总论.md)

## 六、天干在八字分析中的定位

### 1. 天干主外
- 天干代表**显露、公开、直接表现**的力量。
- 对应人生的**表象**、社会关系、公开事件。
- **天干为苗芽、地支为根基**——天干要有地支之根才能发挥作用。

### 2. 天干的断事方向
- **六亲关系**：天干间的生克冲合主要对应六亲关系的和睦与冲突。
- **身体健康**：天干主身体外层（头面、四肢、皮肤）的伤病。
- **事业与社会**：天干代表的事业、名誉、公开人际关系。
- **性格特征**：日干代表命主自己，日干性质决定性格底色。

### 3. 天干的喜忌判断
- 天干力量强弱依赖**地支通根**：
  - 有强根 → 天干有力、能发挥作用。
  - 无根 → 天干浮于外、虚而不实。
- 分析天干时必须**与地支配合**——单看天干易失真。

## 七、一句话总结

- 十天干 = **天元**——阴阳五行 × 具体意象 × 性格特质。
- 天干间的作用 = **相生、相克、相冲、相合（含争合妒合）**。
- **戊己居中央不冲，但参与克与合**。
- **冲 > 克；合中带克**。
- 天干**力量依赖地支通根**，须配合地支藏干与宫位综合判断。
- **先熟十干，再通地支**——这是传承千年的学习顺序。
