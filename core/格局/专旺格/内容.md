# 专旺格

## 定义
- **专**：专注、唯一。
- **旺**：极旺、一气独大。
- **专旺格**：命局**单一五行旺到极致**，其他五行**完全被这一气压制或消化** → 日主与这股独强的气**完全同频** → **顺势而不逆反**。
- 又称"**一行得气格**""**从旺格**"的一类 —— 属**外格**，与从财、从杀、从儿、从势并列。

## 一气为主：五类典型

| 格名 | 一气 | 条件 |
|---|---|---|
| **曲直格** | 木 | 木日主，全局木气一片 |
| **炎上格** | 火 | 火日主，全局火气一片 |
| **稼穑格** | 土 | 土日主，全局土气一片 |
| **从革格** | 金 | 金日主，全局金气一片 |
| **润下格** | 水 | 水日主，全局水气一片 |

这五个格合称为"**五行专旺格**"或"**一行得气格**"，是专旺格的标准类型。

## 核心：一色天下，逆则破格
- 正格思路：身强抗克、有制有化 → 平衡。
- 专旺格思路：身强到**一气独尊** → **不能再克、不能再化**，只能**顺气独行**。
- 比喻：
  - 正格 = 团队作战，各有角色。
  - 专旺 = 独角戏，一个人把所有戏份演完。
## 专旺 vs 从旺

| 对比项 | 专旺格 | 从旺格 |
|---|---|---|
| 定义 | 一行五行独大 | 比劫印星主导 |
| 侧重 | 某一五行（如木） | 自身气势 |
| 成格条件 | 更严格，要"一色天下" | 略宽松 |

实际使用中，**专旺格** 和 **从旺格**常混用 —— 本文可视为其合并理解。

## 判断维度

### 1. 气是否"纯"
- **全局一色**：格最纯 → 成就最高。
- **略有杂气**：格破成废 → 转为普通正格论。
- 判断标准：**有没有哪怕一丝"抵抗"**。

### 2. 印比的比例
- **比劫为主**：偏"独立自信"型，个人色彩强。
- **印星为主**：偏"学识资源"型，气场厚重。
- **印比并重**：典型的专旺成格。

### 3. 大运流年的关键
- 走**印比地**：一气加强 → 格更稳 → 大发。
- 走**食伤地**：轻度泄气 → 本格勉强不破，若食伤轻可喜；若食伤重则破。
- 走**财官地**：**大忌** → 直接打破一气 → 格瓦解 → 可能大起大落。

### 4. 日主位置与气场
- 日主透干 + 有强根 → 格完整。
- 日主坐下为自身五行 → 典型专旺的"坐下有根"。

## 常见误区

### 1. "身旺 = 专旺格"
- 身旺 ≠ 专旺。
- 正格身旺只是"强"，仍在正常范围。
- 专旺格是**极端情况**，要求一气独尊，几乎没有其他五行来牵制。

### 2. 把专旺格等同"必贵"
- 专旺是**贵格之一**，但：
  - **极度依赖大运顺气**。
  - 大运一背 → 比任何格都崩得快。
- 专旺格的人**人生曲线非常陡峭**：顺时狂飙，逆时骤降。

### 3. 忽视"无对手"的代价
- 专旺格的人**缺少外部制约**：
  - 内心里"**对自己太认同**"，常常**自负到听不进反对**。
  - **缺少挫折教育** → 走入顺境时过于张扬，走入逆境时难以调整。

### 4. 以为专旺"没弱点"
- 专旺的弱点是**不能被冲破**。
- 大运流年**任何对一气的冲击**都是致命的。

## 现实表现

### 性格
- **自信爆棚**：自己就是一面军队，不需要别人。
- **不妥协**：你说东我往东，我就往东；你说西我往西，我还是往东。
- **独立性极强**：不愿跟人合作，一个人也能把事做成。
- **气场强大**：外人一眼能感到"这个人不一般"。
- **我行我素**：对世俗规则缺乏敬意。

### 事业路径
- **单干型**：独立创业者、艺术家、自由职业。
- **意见领袖 / 精神领袖**型：自带磁场，有追随者。
- **极致专业型**：一辈子深耕一个领域，不旁顾。
- 共性：**不需要团队、不靠平台、自己就是品牌**。

### 人际
- **朋友稀少但极铁**。
- **婚姻独特**：要么找完全互补的配偶，要么不婚。
- **家族关系疏离**：容易独立生活、远离原生家庭。

## 破格

### 真假专旺
- **真专旺**：一气极纯 → 顺其气则成就非凡。
- **假专旺**：看似一气，实则夹杂微弱异气 → 最糟糕的格 → **既不是专旺，又不是正格** → 人生混乱。
- 识别要点：命局里有没有**哪怕一丝异气的字**。

### 大运破坏
- **财运**：打破一气 → 因财败事、意外变故。
- **官杀运**：一气被克 → 压力超载、突然失控。
- **冲破日主之根的大运 / 流年**：格瞬间瓦解。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 印比运 | **大喜** → 一气加强，大发 |
| 食伤运（适度） | 视结构 → 轻泄可喜；重泄则破 |
| 财运 | **大忌** → 一气破 → 重大变故 |
| 官杀运 | **大忌** → 一气被克 → 格局瓦解 |

## 与相关外格的关系

| 格局 | 顺从对象 | 状态 |
|---|---|---|
| **专旺格 / 从旺格** | 身与印比 | 身极强，顺自身气 |
| **从财格** | 财 | 身极弱 |
| **从杀格** | 七杀 | 身极弱 |
| **从儿格** | 食伤 | 身极弱 |
| **从势格** | 财官食伤综合 | 身极弱 |

专旺格是唯一**身极强**的外格，其他外格都是**身极弱**。两极对照：**强到一个人 = 外格；弱到没有人 = 外格**。

## 核心判断
- 专旺格的核心不是"强"，是**纯**。
- 一丝杂气 → 格破。
- 纯到独尊 → 借一色之气成就独一无二的人生。
- 识别要极度谨慎 —— 比一般格局更难判，错判代价极高。

## 一句话总结
- 专旺 = **一个人走完所有人的路**。
- 不合群、不妥协、不平衡，凭一股纯气走到底。
- 成格时惊艳，破格时惊险 —— 这是一种极致的人生方式。

## 古典标准：真假专旺格取用规则

### 真假之别

专旺格按"有无一党（财官食伤等逆势五行）"分为：
- **真专旺格**：命局中无任何可以破格的一党。
- **假专旺格**：有一党但力量极弱（虚浮无根、被合化夹克夹泄），对格局构不成实质破坏。

### 真专旺格

**成格条件**：
- **地支上**：要么会合成专旺五行的合局（如申酉戌三会金、巳酉丑三合金等），要么月令本气是日元的比劫、阳刃或禄神。
- **命局中不能有任何破局的一党**：天干地支都不能出现官杀、财星、食伤（地支藏干只看本气中气、余气不计）。
- **天干地支必须以比劫为主**：印星可以出现但力量不能超过比劫；若印星力量超过比劫则成从强格而非专旺格。

**性格特质**：
- 真专旺格的八字就像一块纯净而无比坚硬的单质物体，容不得一点杂质也受不得一点外力震荡——越纯净就越硬但韧性越差，就像钻石硬度极大却非常脆、受到外力容易断裂。
- 社会政治上也有这种特性，好比一个极端独裁的君主不容任何反对力量——越独裁其内力越脆弱，往往越接近摧毁。
- 这种人必定霸道、偏激、自负——因为全局力量纯粹、旺极无任何可以逆势之物，所以他不会是一个弱者，只知进不知退——喜欢别人顺从自己，不喜欢别人阻挠或说泄气话。
- 内心力量极强，不可能自卑，也不会承认失败——如果行运顺利他会成功非凡、永无止境的追求权力与物欲；如果行运不顺他会失败再失败、直至不能翻身，但他永远不会承认失败。

**核心追求**：
- 真专旺格以比劫为主——比劫代表争夺占有，所以这类人往往对权力和物欲有极强的追求。
- 精力充沛、干劲十足、自信心爆棚。

**行运喜忌**：
- 真专旺格只能顺其气势、不能逆克。
- **最喜印比运**：能增强自身气势。
- **次喜食伤运**：比劫可生食伤、印星力量小于比劫且由比劫通关不会克食伤——这叫"顺泄"，不算破格。
- **非常忌官杀运**：官杀来克比劫如同鸡蛋碰石头，激起比劫之怒——反应到现实就是引发官非、病灾甚至更大波折。
- **次忌财运**：比劫夺财导致破财、一败涂地。
- 真专旺就像一头猛虎只能顺着不宜激怒，尤其要注意月令是提纲——最怕大运流年双双冲克月令，一旦月令逢冲、格局立破，往往带来大变数、极可能凶亡；**羊刃逢冲尤其严重**。

### 假专旺格

**成格条件**：
- **地支条件与真专旺相同**：要么会合成专旺五行的合局，要么月令本气是日元的比劫。
- **命局中有一个一党但力量极弱**：或是天干见一党但虚浮无根、被合化受制；或是地支见一党但遭夹克夹泄、被合化。
- **天干地支仍以比劫为主**：印星可以出现但力量不能超过比劫；若印星力量超过比劫则成假从强格。
- 例：天干透一官杀但地支无根或被合化，就属假专旺。

**性格特质**：
- 假专旺格的人有胆有识、不畏险阻——见困难救上风险必冒；常把生命作为赌注，不肯过平淡生活——要么流芳百世，要么遗臭万年。
- 人生轨迹容易大成大败，或先大成后大败、或先大败后大成、又或者先大败但再无大成而离世。
- 可以看出比真专旺格要更激进。

**核心追求**：与真专旺类似，假专旺同样野心极大、一定要干大事、追求非凡成就、不甘平凡，而且比真专旺格更激进。

**行运喜忌**：
- 假专旺格只能顺其气势、不能逆客。
- 行运比真专旺复杂——同样忌财运（行财运必破财）；但食伤和官杀是否可用要看原局印星的透藏情况。
- 这里需要引入"**通道**"概念：天干是一个独立的通道，地支是另一个独立的通道——印星的透藏决定了哪个通道可以通关。
- **食伤可用的条件**：
  - 如果原局印星藏而不透（只在地支），那么**天干可以用食伤、地支不能用**。
  - 如果印星透而不藏（只在天干），那么**地支可以用食伤、天干不能用**。
  - 因为要在同一通道避开印星，否则印星会克食伤（如枭神夺食）。当然如果在一个通道中比劫的力量非常强大、即使有印的存在也被比劫泄尽力量，这时食伤也可以用。
- **官杀可用的条件**：
  - 如果原局印星天透地藏（天干地支都有），则**天干地支都可以行官杀**——形成"官杀生印、印生身"的流通通关有情。
  - 如果不是天透地藏，那么**只有印星出现的那个通道可以用官杀**（印星起通关作用）；没有印星的通道即使官杀，否则会犯旺激起比劫之怒，引发官非、刑克、凶祸甚至身亡（尤其以官杀来冲月令为甚）。

### 真专旺格案例（从革格）

- **八字**：戊申 庚申 辛酉 戊戌（男命）
- **分析**：辛金日元生于申月，申月是阳刃之地金气正旺；地支申申酉戌三会西方金局；全局除了日元只有戊土正印生身、没有一丝火木水等克泄耗之物——比劫极旺、印星为辅、气势纯粹、金气冲天。
- **格局**：真专旺格（从革格）——五行气势完全汇聚于金、没有任何破格的一党。
- **用神**：
  - **喜**：土（合金、土能生金、金能帮身）、食伤水（金生水顺势泄秀）。
  - **忌**：火（官杀来克金）、木（财星来了比劫会去夺）。
- **详细解读**：
  - 此八字比劫极旺如一块纯金、容不得半点杂质——其人心智坚定、主观意识空前强大，很少会自卑也不会妥协。
  - 行运喜印比非常吉，官杀一旦大运流年逢火（如午火、巳火）就会激起金之怒气，往往出现是非、病灾。
  - 若月令申金被冲（如大运或流年出现寅申、寅申相冲三会金局被破）——月令提纲动摇、格局立破、变数立至，往往是大灾的信号。
  - 这类人往往霸道、偏激、自负，只知进不知退——如果行运顺遂可成非凡大业（比如成为一方霸主或大企业家）；若行逆运则一败涂地甚至身败名裂；其性格就像历史上的某些枭雄，永不言败、宁折不弯。
  - 平时很难在普通人中见到他们——要么在巅峰、要么在深渊。

### 真专旺格案例（曲直格）

- **八字**：壬寅 癸卯 甲辰 甲子（女命）
- **分析**：甲木日元生于卯月帝旺之地、木气正盛；地支寅卯辰三会东方木局；天干壬癸水印星生深；全局木气极旺没有一丝金来破格——水木相生、气势磅礴。
- **格局**：真专旺格（曲直格）——木气极旺全局无金如同一片茂密的森林。
- **用神**：
  - **喜**：水、木（水生木、木帮木都是增强气势）；火也可以（木生火顺势而泄）。
  - **忌**：金、土——金是官杀来砍木的、土是财星来了比劫就会去抢（所以行金运容易犯旺出事、行土运容易破财）。
- **详细解读**：
  - 这个命主只能顺其刚猛之势——行印比运最佳能让他大展宏图；食伤运不忌（因为木能生火顺势而为）；行财运容易破财（因为木克土比劫争财）；行官杀运则容易招来麻烦事业受阻一败涂地。
  - **重点**：这个命主在丁酉大运丁酉年遭遇重大变故——月令卯木是阳刃，大运流年双双酉金来冲，月令提纲被冲、格局立破、凶险降临。
  - 为什么金冲卯木这么厉害？因为酉金本身就是正官代表约束和管制——双酉冲一卯好比两把斧头同时砍向一棵大树的树根，树根一断整片森林都会倾倒。古人讲"羊刃逢冲灾祸立至"——这就是典型的例子，可能是病灾、可能是车祸，形式不同但命运曲线往往相似。
  - 而且这个命局水木相生、印星透出、本身有很强的生命力，但一旦根基被毁、再旺的印星也救不了。

### 假专旺格案例（稼穑格）

- **八字**：己未 甲戌 己未 己巳（女命）
- **分析**：己土日元生于戌月得令土气正旺；地支两戌未、戌未戌相刑土气越刑越旺，时支巳火为羊刃生扶日主；全局印比成势、土势厚重；为天干甲木正官透出但甲木作戌土无根且与己土甲己合化土、被月令戌土引化成功——虽有克身之象却无克身之实，如纸刀不能伤人。
- **格局**：假专旺格（稼穑格）——甲木正官虚透合化、对格局无实质破坏、如同虚设。
- **用神**：
  - **喜**：火（生土）、土（帮身）。
  - **忌**：水（财星会引比劫夺财）。
  - 至于官杀与食伤是否可用，需视原局印星通关情况而定——具体在详细解读中阐述。
- **详细解读**：
  - 此命局为假专旺真土势从——原局看甲木虽透但被合化、格局尚纯。命主现实中为女校长、地位不俗，足见假专旺格能量之强——纵使运平也超常人。
  - **行运规则需分通道细论**：
    - **官杀之运用**：天干若再透甲乙木则甲己合化易破、官杀攻身带来压力；但若天干有丙丁火印星则可通关、使官杀生印印生身、此时可用官杀。地支若见寅卯木、因原局有巳火通关亦可少量出现，但不宜过多——如逢乙卯大运又与寅卯流年，则官杀过重仍会犯旺。
    - **食伤之运用**：天干不宜见食伤——因食伤庚辛金克制甲木导致甲己合化不成、反面引起甲木克身再遭反克；地支见申金则不忌——因原局土势极旺、印星巳火被旺土泄尽无力克食伤、反而形成"火生土、土生金"的顺泄之势。
  - 与真专旺相比，此局多一位虚浮之杀、虽暂被合化但行运中亦被激活——故人生亦大起大落；命主有胆有识、不甘平凡。
  - 此即假专旺之宿命——成也其势、败也其势。

### 专旺格核心要点总结

- **专旺格是比劫极旺的特殊格局**，分从革（金）、稼穑（土）、润下（水）、炎上（火）、曲直（木）五种——成格需地支会局或得月令，无足以破格的官杀或财星；以比劫为主、印星为辅。
- **真专旺无任何一党**：性格霸道、偏激、只知进不知退——行运非常怕官杀、次怕财运，喜印比运，食伤不忌，月令逢冲亦引发变数。
- **假专旺有一位虚浮一党、无根或受制**：性格有胆有识、野心大、亦大成大败；行运怕财运，食伤和官杀是否可用看原局印星透藏。
- **专旺格的人精力充沛、自信有骨气、亦当领导**——但六亲缘薄：与父母配偶子女亦有刑克，因命局只有比劫印星、其他十神衰微——虽婚姻宫喜用但往往晚婚或关系冷淡。
- **专旺格的人难在普通人中见到**：心高气傲、瞧不上平庸之辈——要么在干大事、要么已遭重大挫折，长处领导地位或风口浪尖。
- **最后再强调**：专旺格只能顺其气势、不宜逆客——行运顺则成功非凡、逆则一败涂地。
