# 伤官生财

## 成立条件（严格依《子平真诠》《滴天髓》《渊海子平》）

### 条件 1：伤官透干且通根，或月令本气为伤官

- **古籍出处**：
  - 《子平真诠·论伤官》："伤官旺而生财，身强为富贵之格。"
- **客观判据**：
  - ① 伤官透于月 / 日支 / 时干，且地支有本气或中气通根至少一位；或
  - ② 月令地支本气为伤官。
- **不满足则破**：伤官虚浮无根 → 无可生财之源头。

### 条件 2：财星透干且通根

- **古籍出处**（本条最核心）：
  - 《子平真诠·论伤官》："**伤官生财，财要有根；无根则不成格**。"
- **客观判据**：
  - 财星（正财 / 偏财皆可）透于天干，且地支有财本气或中气根至少一位。
- **不满足则破**：财虚透无根 → 沈氏"无根则不成格"——伤官之气无所归，形同自耗。

### 条件 3：伤官与财星位置相连（伤 → 财 无阻隔直生）

- **古籍出处**：
  - 《滴天髓·通神论》："用神要紧贴，格局要通气。"
  - 《滴天髓·伤官》："伤官生财，**财能化伤**。"
- **客观判据**：
  - ① 伤官与财星分居相邻两柱，或位于同柱之干与支（上下相生）；
  - ② 伤官与财星之间无印星阻隔（印克伤则链断）。
- **不满足则破**：伤官与财被印 / 官隔开 → 气不连贯 → 伤官生财链断。

### 条件 4：日主身强（此格专为身旺用财）

- **古籍出处**（本条核心）：
  - 《子平真诠·论伤官》："**伤官用财，身旺为宜**。"
  - 《子平真诠·论伤官》："身旺伤官旺，财更有根，大贵之格。"
- **客观判据**：
  - ① 日主于月令为比印本气，或
  - ② 他柱至少三位比劫 + 印星通根（身中偏强至身旺）。
- **不满足则破**：身弱 + 伤官生财 → 财多身弱，忙而无得；当转入"伤官佩印"用印救身。

### 条件 5：原局无正官紧贴（忌伤官见官）

- **古籍出处**：
  - 《渊海子平·伤官格》："**伤官用财取为奇，不宜官杀在傍随**。"
  - 《滴天髓·伤官》："伤官见官，祸端百出；伤官生财，财能化伤。"
- **客观判据**：
  - ① 原局天干无正官与伤官紧贴；
  - ② 若有七杀，须被合去 / 被食神制之 / 被财生而归顺（财生官成"伤官生财、财生官"的进阶三全贵局）。
- **不满足则破**：伤官贴官无救 → 破格为"伤官见官"，祸百端。

---

**五条同时满足**方为真伤官生财。本格进阶：若身足以再承官气，"伤官生财 → 财生官"可连成三全，沈氏称为"大贵之格"；身不足以承则须止于"伤→财"二级流通，避见官。
