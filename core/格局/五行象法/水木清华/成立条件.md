# 水木清华

## 成立条件
1. **水有力而不过旺**：
   - 壬癸透干有根。
   - 或地支亥子丑见一两位。
2. **木有根而透干**：
   - 甲乙透干。
   - 地支见寅卯 / 亥卯未合木局。
3. **水木比例合宜**：
   - 水过多 → 木漂（破象）。
   - 水太少 → 木枯（无气）。
   - 理想：**水足够润、木足够承** → 两者比例接近 1:1 到 1:2。
4. **有火点缀更佳**：
   - 水木清华 + 火点 → 转为"**水木清华 → 木火通明**"的完整流通 → 格局升级。
5. **无重金克木 / 无重土塞水**：
   - 金克木 → 木被砍 → 不清。
   - 土塞水 → 水失流 → 气不秀。
