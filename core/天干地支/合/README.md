## 各合类型力量等级

以下是合的力量由强到弱的总排序：

| 排名 | 合类型 | 力量级别 | 稳定性 |
| --- | --- | --- | --- |
| 1 | 地支三会 | **最大** | **最不稳定**（易被冲破） |
| 2 | 地支三合 | 很大 | **最稳定**（冲刑害破难破之） |
| 3 | 生地半三合（长生 + 帝旺） | 较大 | 较稳 |
| 4 | 地支六合 | 中 | 中等（须紧贴透干） |
| 5 | 墓地半三合（帝旺 + 墓库） | 较小 | 较稳但偏滞 |
| 6 | 地支暗合 | **最弱** | 隐性 |

**天干五合**作为天干层面的相合，力量级别与六合相当，但作用点在人际、情感、格局层面，不与地支合直接比较。

## 各合类型的现实意义

### 三会

- 代表**人生重大变化、季节性气势的全面转向**。
- 应事点：**岁运补齐缺字的那一年、那一月**，即重大转折应期。
- 适合判断：升迁、变故、健康剧变、家庭重大事件。

### 三合

- 代表**稳定合作、团队紧密、人生贵气流转**。
- 化解冲刑害破的**王牌**。
- 适合判断：合作创业成功、财官印格局成立、贵人相助。

### 半三合 + 拱合

- 代表**蓄势待发的潜伏力量**，应期由岁运激活。
- 批命时须特别关注：缺失之支何时到位。

### 六合

- 代表**情感结合、合作关系、化冲解绊**。
- 各宫位见六合，对应不同人际象意（祖辈、兄弟、婚姻、子女）。

### 暗合

- 代表**表面不容、内心紧密的隐秘关系**。
- 销售、谈判、情感暗线、私情暗恋——皆由此取象。

### 天干五合

- 代表**上下级、夫妻、君臣、金钱与利益**的合作关系。
- 不同合组有鲜明象意：中正、仁义、威制、淫匿、无情。

## 核心判断要点

1. **合化是否成功**，取决于：**齐聚 + 相贴（三会除外）+ 透干 + 无严重冲克**。
2. **合的吉凶**，取决于**合出或合绊的是喜用还是忌神**，不可字面论吉。
3. **遇冲刑害破**：
   - 六合 → 遇三合让位于三合。
   - 三合 → 不让位于三会，但三会不稳定易被冲破。
   - 化解冲刑害破，**以组成三合局为最优解**。
4. **批命实务**：半合、拱合、三会缺字 → **推算岁运填实的应期**，判断重大吉凶。

## 古训警示

- **新手怕冲，老手怕合**——各种合的作用关系具有一定难度，务必结合全局综合判断。
- **合不一定吉，冲不一定凶**；一切回到**日元喜忌**与**整体平衡**。
