# 炎上格

## 定义
- **炎上**：《尚书·洪范》"火曰炎上" —— 火的本性是**向上燃烧、光明外放**。
- **炎上格**：**火日主 + 全局火气一片** → 形成**火的专旺格局**。
- 属**五行专旺格**之一（与曲直格木、稼穑格土、从革格金、润下格水并列）。
- 典型人格：**热情、明亮、有感染力、精神层面的领袖**。

## 核心：光明外放、热烈不息
- 火 = **光明、热情、显化、礼节**。
- 炎上格 = 整个命局都是火 → 这个人的人生底色就是"**发光 + 发热**"。
- 火的"炎上"本性：
  - **炎**：燃烧、热烈。
  - **上**：向上升腾、不肯低沉。
- 合一后：**永不熄灭的生命力与影响力**。

## 成立条件（严格依《渊海子平·炎上格》）

### 条件 1：日主为丙或丁火

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·炎上格》："**炎上格——丙丁日生人**。"
- **客观判据**：日主天干必须为丙火或丁火。
- **不满足则破**：非丙丁日 → 不入炎上格。

### 条件 2：地支会成火局（三合寅午戌 或 三会巳午未）

- **古籍出处**（本条核心）：
  - 《渊海子平·炎上格》："**地支寅午戌三合火局，或巳午未南方火气**。"
- **客观判据**（二者之一）：
  - ① 地支见 寅 + 午 + 戌 三合火局；或
  - ② 地支见 巳 + 午 + 未 三会南方火。
  - 若仅半合（寅午、午戌）或二字会气，格品降但勉强成立。
- **不满足则破**：地支不成火局 → 一气不聚。

### 条件 3：天干多透火，最好有甲乙木生之

- **古籍出处**：
  - 《渊海子平·炎上格》："年月时干宜见丙丁……**有甲乙木相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位丙或丁透出；
  - ② 若另见甲或乙透干生火，格品升（薪柴充足）。
- **不满足则破**：天干火气不足 → 一气未遍布。

### 条件 4：原局无壬癸水透干灭火

- **古籍出处**（本条核心）：
  - 《渊海子平·炎上格》："**年月时干无水相克**，主威武贵显。"
- **客观判据**：
  - ① 天干无壬水或癸水透出；
  - ② 地支本气无亥水、子水成势。
- **不满足则破**：水透克火 → 《渊海子平》"水相克"即明破格 → 转入"水火相战"或普通克身格讨论。

### 条件 5：土金搭配适度（土微泄可喜，土重泄衰；金微耗可，金重则忌）

- **古籍出处**：
  - 《渊海子平·炎上格》："若得土稍泄其气，更为秀丽。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - ① 土透干或地支本气仅一位 → 微泄秀气，可喜；
  - ② 土多透多根 → 泄气过度 → 降格为"火土夹带"或"火炎土燥"讨论；
  - ③ 金微则火有用武之地，金重则耗火无谓。
- **不满足则破**：土过重或金过重 → 一气变。

---

**前四条必须同时满足**方为真炎上格；条件 5 关乎格品高低——有木生则薪柴不断，微土泄则秀气外显。

## 判断维度

### 1. 火的纯度
- **地支三合火 + 天干多火**：格最纯。
- **地支方局（巳午未）+ 天干多火**：次之。
- **散见火气但不成局**：炎上难成。

### 2. 木的参与
- **有木生火**：薪柴充足 → 格更稳。
- **无木**：纯火无源 → 烈但短暂。

### 3. 土的参与
- **适度土**：泄火秀气 → 火土夹带的美。
- **土过多**：泄火过度 → 火炎转衰 → 破格。

### 4. 大运流年
- **火木运**：顺气加强。
- **土运（适度）**：泄秀可喜。
- **水运**：灭火破格。
- **金运**：耗火 → 视量 → 过量则忌。

## 常见误区

### 1. "火日主 + 身强 = 炎上格"
- 身强 ≠ 炎上。
- 炎上要求"一气独尊 + 全局火气"。

### 2. 以为炎上 = 脾气大
- 炎上的"热"不等于暴躁。
- 真正成格的炎上人**是"温暖型领袖"**，不是"暴君型"。
- 脾气大是破格（火过旺无制）时的表现。

### 3. 忽视木的作用
- 纯火无木 → 烈但短暂。
- 有木炎上 → 薪火相传 → **持久的光明**。

### 4. 把炎上等同"外向"
- 炎上是**精神层面的明亮**，不仅仅是外向性格。
- 很多炎上格的人其实**内心温暖而言语不多**。
- 他们**发光的方式是作品 / 影响力 / 榜样**，不一定是"会说话"。

## 现实表现

### 性格
- **热情、开朗、有亲和力**。
- **正面积极**：不喜阴暗消极。
- **精神力强**：能带动氛围。
- **喜欢分享 / 表达**：像阳光一样照耀他人。
- **注重礼节**（火主礼）。
- **有理想主义情怀**。

### 事业路径
- **精神领袖 / 思想导师**：宗教、哲学、心灵成长、文化名人。
- **演艺 / 表演**：演员、歌手、主持人、演讲家。
- **教育 / 传播**：特别是能感染他人的老师 / 传道者。
- **品牌 / 营销**：需要"**点燃他人**"的岗位。
- **公共服务**：能温暖他人的岗位（医生、救援、公益领袖）。
- **艺术 / 创作**：画家、作家、视觉艺术家。
- 共性：**需要"照亮他人 + 带动氛围"**的岗位。

### 成长节奏
- **青年期即显光芒**：与曲直的慢生长不同，炎上往往**年少就显露**。
- **中年燃烧最烈**：事业高峰与个人影响力共振。
- **晚年如夕阳**：虽光辉但需警惕"燃尽"。

### 感情
- **热情主动、不掩饰情感**。
- **喜欢明朗的关系**：不喜暧昧含糊。
- **忠诚度高**但对伴侣期待也高。
- **容易陷入理想化**：对伴侣有"**照亮彼此**"的期待。

### 身体
- 火对应**心、血脉、眼目、小肠**。
- 炎上格：
  - 心气旺 → 精力充沛。
  - 但**炎上过度易心悸、高血压**。
  - 眼睛问题（干涩、视疲劳）。
  - 易失眠、上火。
  - 晚年需警惕心血管问题。

### 心理
- **长期"发光"的人**，自己可能会感到**空虚 / 倦怠**。
- 需要**周期性休养**，不能持续燃烧。

## 破格

### 破格情形
- **水重灭火**：阳光被乌云遮 → 格破。
- **金多耗火**：虽不直接克，但火用于克金 → 能量外耗。
- **土过多**：泄火过度 → 火之气衰 → 转为火土夹带但非炎上。

### 真假炎上
- **真炎上**：火气纯净 → 成明亮光辉。
- **假炎上**：夹杂水或重金 → 看似火旺实则不稳 → 情绪起伏大、易燃尽。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 火运 | **大喜** → 一气加强 |
| 木运 | **大喜** → 薪柴充足 |
| 土运（适度） | **喜** → 泄秀 |
| 土运（过度） | **忌** → 泄过度 → 火衰 |
| 金运 | **忌** → 耗火 |
| **水运** | **大忌** → 灭火 → 格破 |

## 与其他专旺格对比

| 格名 | 一气 | 核心品质 |
|---|---|---|
| **曲直格** | 木专旺 | 生长、仁爱 |
| **炎上格**（本条） | 火专旺 | 光明、热情、影响 |
| **稼穑格** | 土专旺 | 承载、稳重 |
| **从革格** | 金专旺 | 变革、裁断 |
| **润下格** | 水专旺 | 流动、深远 |

## 核心判断
- 炎上格的精髓在"**光明持续**"：
  - 有木 → 薪柴不断。
  - 无水 → 光不被灭。
- 成格时：**精神型领袖的典型命造**。
- 破格时：**燃尽、抑郁、起伏大**。

## 一句话总结
- 炎上 = **向上燃烧的生命力** —— 既温暖自己也温暖他人。
- 真正的炎上格不是"脾气大"，是"**把光变成礼物**"。
- 别用水灭他、别让他燃尽 —— 给他薪柴、给他舞台，他会温暖整个世界。
