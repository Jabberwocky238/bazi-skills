# 空亡

## 定义

天干与地支按一定顺序排列构成 60 个组合，因十二地支较十天干多出两字，**每十年会有两个地支轮空**，总计六个十年。每一旬对应的轮空地支即为**空亡**。

### 六旬与空亡对照

| 旬首 | 空亡地支 |
| --- | --- |
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

## 快速计算法

### 天干地支编号

**天干**（1–10）：甲=1、乙=2、丙=3、丁=4、戊=5、己=6、庚=7、辛=8、壬=9、癸=10。

**地支**（1–12）：子=1、丑=2、寅=3、卯=4、辰=5、巳=6、午=7、未=8、申=9、酉=10、戌=11、亥=12。

### 计算步骤

1. 以**地支数减去天干数**。若不够减，地支数先**加 12** 再减。
2. 所得之数**必为双数**。
3. 最后**加上前面的单数**即可得到该旬的一组空亡。

### 德数对应空亡速查表

| 地支数 − 天干数 | 空亡地支 |
| --- | --- |
| 2 | 子丑 |
| 4 | 寅卯 |
| 6 | 辰巳 |
| 8 | 午未 |
| 10 | 申酉 |
| 0（即戌亥空亡） | 戌亥 |

> 戌亥空亡的得数为 0，因其属于第一个十年（甲子旬），天干和地支前 10 个数具有相同编码，相减为 0。

### 计算示例

**庚申**：地支申=9，天干庚=7 → 9-7=2 → 双数编号 2 对应丑 → 加上前面的单数编号 1 对应子 → **空亡为子丑**。

**辛丑**：地支丑=2，天干辛=8 → 2-8 不够减 → (2+12)-8=6 → 双数编号 6 对应巳 → 加上前面的单数编号 5 对应辰 → **空亡为辰巳**。

**戊子**：戊=5、子=1 → (1+12)-5=8 → 编号 8 对应未 → 加上 7 对应午 → **空亡为午未**。

## 八字中如何判断哪一柱落空亡

### 判定依据
- 一般用**日柱和年柱**来判定空亡，其他柱通常不考虑。
- **以日柱为主**（日柱代表自己、最重要）。
- **日支是否空亡需以年柱来判断**（因为日柱判定的空亡，本柱地支不可能落空）。
- 年柱代表祖上、根源，作为辅助。

### 应用范围
空亡不仅针对本命局，也适用于**大运和流年**。

### 排盘软件
现代排盘软件可直接显示空亡（神煞区域会在落空亡的柱下标注）。**空亡属于神煞的一种**。

### 补充提示
落空亡时，**地支同柱的天干也会一起空亡**——整柱天干地支都落空亡。

## 空亡的应用

### 一、空亡要看喜忌

核心原则：
- **喜神空则不好**（好事打折）。
- **凶神空则好**（坏事打折）。

不管本命局还是大运流年，对命主有益的喜用之神落空亡 → 对命主不利（好事打折）；对命主有害的忌仇神落空亡 → 有益于命主（坏事打折）。

**特别注意：大运为喜却落空亡**
- 代表这波大运可能**留不下什么东西**。
- 比如大运为财运且为喜用 → 这一大运里会赚不少钱，但会因各种原因花掉或因生大病而败掉。
- 即使大运为喜用，也不是发达的大运。

**流年落空亡同理**——无论好事坏事都会打折。例如流年原本能赚 100 万，落空亡后可能只能赚 50 万甚至更少。

### 二、岁运能够填实空亡

若某一柱为空亡，而**流年或流月**出现与该柱相同的地支 → 可将本柱**填实**，即不再空亡。

**特别注意**：只有**流年和流月可以填实**，**大运不可**——因为大运是由月柱排出的，仍由本命局确定；流年才是外部力量的变化，流月则是流年的具体表现。

### 三、空亡遇冲合可解空

空亡可理解为**在特定条件下某个地支不发挥作用**。但在特定时间（大运或流年）遇到冲合 → 该空亡地支可被**激活**，重新参与命局作用。

### 四、空亡能与其他神煞、十神一起使用

神煞遇到空亡 → 好坏都减半：
- 天乙贵人遇空亡 → 好作用减半。
- 羊刃遇空亡 → 坏作用减半。
- 因此空亡的一个作用是 → **不管好坏都会变得比较平稳**。

## 空亡的十神取向

当某十神落空亡 → 对应方面会出现不好的状况。

### 事业与婚姻
- **官杀为喜用时落空亡** → 事业发展可能遇阻。
- **女性官杀落空亡**（无论喜忌）→ 婚姻可能不美满。

### 天生官星空亡的四组女命日柱
女命以下四日柱，婚姻多不顺：

| 日柱 | 空亡 | 官星 | 说明 |
| --- | --- | --- | --- |
| **戊申日、己酉日** | 寅卯 | 寅卯为戊己的官杀 | 官星天生落空亡 |
| **甲戌日、乙亥日** | 申酉 | 申酉为甲乙的官杀 | 官星天生落空亡 |

男命则需找**财星（正财、偏财）天生落空亡**的日柱，可举一反三。

### 印星空亡
- 印星若在命局中为喜用且强旺有力，因印星代表学习能力、六亲代表母亲 → **弱入空亡**：
  - 学历会大打折扣。
  - 母亲身体不好或与母亲缘分不佳。

## 空亡的宫位取向

### 年柱空亡（父母、祖上、少年）
- 难得父母祖上庇荫，少年艰辛，或小时缺乏父母照顾。
- 严重情况（父母星宫同位且被克严重）→ 从小父母可能不在；若父母星全部严重被克且在相应宫位 → 命主可能是**孤儿**。

### 月柱空亡（兄弟姐妹、社会、青年）
- 兄弟姐妹无靠或兄妹有损，难得兄弟姐妹之力。
- 青年时期运势不济，过得不顺。
- 社会关系或人际关系处理不好，或疏于交往应酬。

### 日柱空亡（配偶、中年）
- 配偶身体不好，难得配偶之力。
- 感情先浓后淡，容易出现转折。
- 中年运气不利。
- 日支是否空亡需**从年柱判断**（日柱空亡无论日支是什么十神，不需星宫同位）。

### 时柱空亡（子女、晚年）
- 不得子女之力或子女有损。
- 若再逢冲 → 即使有子女日后也不能在身边（子女都在外地工作、很少回家）。
- 晚年一般不幸福。

## 空亡互换（严重情形）

### 定义
日柱空亡与年柱空亡相互验证，若出现"日柱的地支落在年柱的空亡上、年柱的地支落在日柱的空亡上" → 即**空亡互换**。

**示例**：日柱空亡为申酉，年柱地支是申金 → 年柱落日柱空亡；年柱空亡为戌亥，日柱地支是戌土 → 日柱落年柱空亡。两柱**互换空亡**。

虽然实际中不用月柱、时柱的空亡判定其他柱是否落空亡，但考虑空亡互换时，月柱、时柱也可参与。

### 各柱互换空亡的象义

#### 1. 年柱–日柱互换空亡（最严重）
- 一生劳苦忙碌、多败少成。
- 孤苦无依、六亲缘浅。
- 有些人甚至有自闭症，对任何事都提不起兴趣。
- 因为年代表根基、日代表自身——**即使年柱日柱都是忌神，只要出现互换空亡情况，都是不好的**。

#### 2. 日柱–时柱互换空亡
- **对子女不利**——日柱代表自己和妻子，时柱代表孩子，两柱互换空亡可能**有损子息**。

#### 3. 月柱–日柱–时柱三柱俱空
- 如果四柱组合好 → 空亡不忌，反而可能**大富大贵**。
- 四柱配合好的较好格局，如**伤官配印、食神制杀**等。
- **三柱都是忌神时效果更好**。

#### 4. 年柱–月柱–日柱–时柱四柱俱空
- 如果四柱组合好 → 主**聪明富贵**。

### 核心规律

> **互换空亡时，空得越多反而可能越好**——比如三柱以上，特别是四柱，可能构成好的格局。

## 现实应用

空亡的应用要点：
1. **不单判**：空亡本身不是结论，必须结合喜忌。
2. **打折原则**：好事坏事都打折。
3. **填实与解空**：岁运可填实，冲合可解空。
4. **宫位代入**：看空亡落在哪一柱，对应人生哪个领域。
5. **互换特判**：空亡互换是特殊格局判定条件。

## 一句话总结

空亡 = **命局中暂时失效的格子**——喜神空是好事打折、忌神空是坏事打折；单柱空多是缺憾，多柱互换反可能成就大格局。
