# 从旺格（含从强格）

## 定义
- **从旺**：身**极旺**时，不再用克泄之道平衡，而**顺从旺势而行**。
- **从旺格**：命局**比劫 + 印星共同主导**，日主气势满盘独大，原局无官杀可克、无财可破印 → 顺从整体旺势为用。
- **从强格**（从旺之亚型）：**以印星为主**导的从旺——印绶重重 + 比劫叠叠。
- 属《滴天髓》"反从"类外格——与从财 / 从杀 / 从儿 / 从官 的"身弱顺从克我方"相反，从旺是"**身极强顺从自己**"。

## 从旺 vs 专旺格

| 对比项 | 从旺格 | 专旺格（曲直 / 炎上 等） |
|---|---|---|
| 核心 | 比劫 + 印星共同独大 | 日主五行一气独旺 |
| 侧重 | 两层生扶（印 + 比） | 单一五行成象 |
| 典型 | 印比不分主次，都多 | 月令主导一气 |
| 古籍 | 《滴天髓·从旺 / 从强》 | 《渊海子平·杂格》 |
| 命名 | 以"旺"为纲 | 以五行为名 |

**关系**：专旺格（曲直、炎上、稼穑、从革、润下）是从旺格**在单一五行独旺时的具体分化**；从旺格则是**比印并旺而不专归某一五行**的笼统名。两者在极端一致时重合。

## 从旺 vs 从强

| 对比项 | 从旺格 | 从强格 |
|---|---|---|
| 主导 | 比劫为主 | 印星为主 |
| 气质 | 自我独立 | 资源承载 |
| 典型人格 | 靠自己打天下 | 靠家学 / 贵人立身 |
| 喜神 | 比印食伤（顺）；最喜比劫 | 比印食伤（顺）；最喜印星 |

古籍中两者常并论（《滴天髓》分立两章，但任铁樵注常合说）。本文以从旺为主、兼论从强。

## 核心：旺到极致不再求平衡
- 正格思路：身旺用克泄——官杀制、财泄、食伤泄。
- 外格思路：身**旺到盘整皆是自己**时，克泄反而撼动根本 → **不如顺其旺势**。
- 比喻：大江奔流——靠修小坝去拦只会崩堤，**顺它入海方是正道**。
## 与专旺格的判定界限

- **真专旺**（曲直、炎上、稼穑、从革、润下）：日主五行**与月令本气一致 + 地支三合三会成象**。
- **真从旺**：比劫 + 印星**不分明具体归某一五行**——可能是比劫多于印，也可能印多于比。

实操中：
- 命局若月令三合或三会明确 → **按专旺格论**（具体名如曲直、炎上）。
- 命局若只是比印杂旺而不成一气 → **按从旺格论**（更笼统的称谓）。

## 判断维度

### 1. 真从旺 vs 假从旺
- **真从旺**：无官杀、无财破印、食伤微弱或无 → 纯从。
- **假从旺**：有微弱官杀或财 → 想从又不能尽从 → 身旺而用神不清的破局。

### 2. 大运流年
- **比劫、印星运**：顺势加强 → 格更稳，贵格兑现。
- **食伤运（微）**：秀气外显 → 才华发挥。
- **食伤运（过重）**：泄旺 → 格变。
- **官杀运**：**大忌** → 克身破旺 → 重大事件年。
- **财运（尤对从强）**：**大忌** → 破印破从 → 根基动摇。

### 3. 日常心态
- 从旺格的人**对外部制约极其抗拒**——一旦有人试图"管住"他们，反而激起更大的反弹。
- 识别后：尊重其"独行"与"独立"，不强求入群。

## 现实表现

### 性格
- **独立、自主、不喜被管**。
- **自信心极强**：内心笃定。
- **执行力强**：想做就做、不拖泥带水。
- **社交偏好**：重深交、轻泛交。
- **气场强大**：给人"不好惹"的感觉。

### 从旺型（比劫为主）
- 靠自己打天下的气质。
- 自由职业、独立创业、个人品牌、单打独斗型专家。

### 从强型（印星为主）
- 靠资源 / 家学 / 贵人立身。
- 学者、研究者、继承者、平台型人才。

## 与其他外格对比

| 格局 | 核心 | 日主状态 |
|---|---|---|
| **从旺格**（本条） | 比印主导 | 身极旺，顺自己 |
| **从强格** | 印星主导 | 身极旺，靠印生身 |
| **专旺格**（曲直炎上等）| 一气成象 | 身极旺，归某五行 |
| **从财格** | 顺财 | 身极弱 |
| **从杀格** | 顺杀 | 身极弱 |
| **从儿格** | 顺食伤 | 身极弱 |
| **从官格** | 顺正官 | 身极弱 |
| **从势格** | 顺财官食综合 | 身极弱 |

## 核心判断
- 从旺 / 从强是**身极强的反从**——与其他从格"弱到极致"相对应，"强到极致"也成外格。
- 成立要"**比印独大 + 无官杀 + 无财破印 + 食伤不重**"。
- 识别意义：这类命造不必"谦让"或"平衡"，**顺自己的旺势、做自己的主宰**才是正道。

## 一句话总结
- 旺到极致就不用再求平衡 —— 顺着自己的水、往自己的方向奔。
- 从旺 / 从强是外格中唯一"**强到成格**"的一类。
- 关键不是"变得更强"，是"**认清自己确实旺、该顺就顺**"。
