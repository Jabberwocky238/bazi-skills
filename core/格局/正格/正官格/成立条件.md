# 正官格

## 成立条件

### 条件 1：月令地支本气为正官，或正官透干且月令有根
- **判据**：
  - ① 月令地支本气为正官；或
  - ② 月令地支中气 / 余气为正官，且正官透于天干。
- **取用要点**：正官格的最优状态为**月令藏干正官且透于天干**。
- **不满足则破**：月令正官不显 → 当以月令本气另论他格。

### 条件 2：正官清纯，不混七杀
- **判据**：天干无七杀透出；若地支藏七杀一位，须有食神制之或合去（合杀留官）。
- **取用要点**：**官杀混杂则格局变乱、贵气搅碎**——必须通过相合、相克去其一方才能显贵。
- **不满足则破**：官杀混杂 → 转入"官杀混杂"病格讨论。

### 条件 3：原局无伤官紧贴克正官
- **判据**：原局天干无伤官与正官紧贴；若有伤官，须有印制（此时成"伤官佩印 + 官"复合结构）。
- **取用要点**：**伤官见官是正官格最凶的破败组合**——贵气彻底受损。唯一例外是"伤官伤尽"（伤官旺而官星不显被彻底制约）与"金水伤官喜见官"。
- **不满足则破**：伤官贴官无救 → 破格转入"伤官见官"讨论。

### 条件 4：日主身中偏强，能承载官气
- **判据**：日主于月令为比劫 / 印星本气，或他柱两位以上比劫 / 印星通根（身中偏强）。
- **取用要点**：**身强能扛官方成贵**——身弱扛不住重官且无印通官时，官星会从"护身贵官"变为"克身小鬼"，易陷入虚职陷阱。
- **不满足则破**：日主极弱 → 不任官 → 官反为克身之病，须印化之。

### 条件 5：官星有根，不虚透
- **判据**：正官透于天干且在地支有本气或中气根至少一位。
- **取用要点**：**身强而官清（官弱）须有财来生官**——否则官星无根基、贵气缺失，易走偏门。
- **不满足则破**：官虚透无根 → 无真官之气，格不成。

### 条件 6（辅佐条件）：有财生官或印护官
- **判据**：
  - 财星透干通根，位于官星之前（财→官顺流）；或
  - 印星透干通根，位于官星之后（官→印化身）。
- **取用要点**：
  - **身强 + 官清 + 财旺**（财官相生）→ 贵气最高、富贵双全、多掌实权——女性为"一品夫人"旺夫命。
  - **身弱 + 官重 + 印旺**（官印相生）→ 贵多富少、收入稳定但难聚大财——多为稳定公职。
  - **"有官有印"等于掌实权**，"有官无印"等于卧虚权。
- **非必要**：不满足不破格，但格品降——无财无印的正官格属"纯正官"、贵气略薄。
- **最佳结构**：财、官、印三者俱全、链条顺流 → 转入"**财官印全**"贵格讨论。

---

**前五条必须同时满足**方为真正官格；条件 6 关乎格品。正官格的最佳兑现路径是**身官平衡 + 财印搭配到位**——贵气源于品行端正与正统成就，非单纯名利可比。
