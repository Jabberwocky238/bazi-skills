## 天干妒合

## 定义

- **妒合**：在相邻的三个天干中，**两个天干相合，而第三个天干对这一合产生嫉妒**（克制合中一方、破坏合化），是**天干五合**之外的一种**破坏性三角结构**。
- 核心不在"争夺同一合伴"，而在**于嫉妒他人亲密而从中作梗**。
- 传统比喻："**小姑子嫉妒嫂子霸占哥哥的所有关爱**"——哥哥是合的阳干，嫂子是合的阴干，小姑子就是那个在旁边看着、忍不住插一脚的妒合之干。

## 与"争合"的本质区别

妒合常与争合混淆，二者形式相似但内核迥异：

| 对比点 | 争合 | 妒合 |
| --- | --- | --- |
| 结构 | 两个**相同天干**同时合一个异性天干 | 两个**相合天干**相邻，第三个**同性天干**克其中一方 |
| 相合性质 | **异性相合**（合本身存在），只是被争抢 | 合本是异性，**第三字与合中一方同性相克**，非来争合，而来**破合** |
| 动机象意 | "两人同追一人"——争夺 | "看不得你们好"——嫉妒破坏 |
| 常见心理 | 风流、多选、被动受宠 | 猜疑、妒忌、挑拨离间 |
| 组合举例 | 两庚争一乙（庚庚乙） | 甲己乙的组合：甲己相合，**乙木克己土**而成妒合 |
| 同性关系 | 同字自然相斥（比劫） | 克制关系（比劫克合中被合一方） |

**口诀**：争合是"争"，妒合是"妒"。争是两人抢一人，妒是旁人坏一对。

## 形成条件

妒合需**三字紧邻排列**，且顺序严格：

1. 第一位：**阳干**（合中主导方）。
2. 第二位：**阴干**（与阳干相合的一方）。
3. 第三位：**另一阴干**（与第二位同性，**克制第二位**，以日主身份嫉妒合局）。

- 阳在前、合之阴在中、克之阴在后——形成"**夫妻甜蜜在前，妹妹在后怒目**"的画面。
- **顺序不能换**：如"甲己乙"是妒合，"甲乙己"、"乙甲己"皆不是。

## 典型妒合组合

共**五种**，每一种都以**最后一个阴干为日主**：

| 妒合组合 | 合的关系 | 妒合原理 | 日主象意 |
| --- | --- | --- | --- |
| **甲己乙** | 甲己合化土 | 乙木克己土，破坏甲己之合 | 乙日主嫉妒甲己（兄嫂）之亲 |
| **乙庚辛** | 乙庚合化金 | 辛金克乙木，破坏乙庚之合 | 辛日主嫉妒乙庚之合 |
| **丙辛丁** | 丙辛合化水 | 丁火克辛金，破坏丙辛之合 | 丁日主嫉妒丙辛之合 |
| **丁壬癸** | 丁壬合化木 | 癸水克丁火，破坏丁壬之合 | 癸日主嫉妒丁壬之合 |
| **戊癸己** | 戊癸合化火 | 己土克癸水（同土排己癸不合而克），破坏戊癸之合 | 己日主嫉妒戊癸之合 |

- 全部是"**阳 + 合之阴 + 克合之阴的同性阴**"的排列。
- 这种同性排斥**暗藏比劫之争**（或克泄之意），心理上对应嫉妒心、破坏欲。

## 妒合的四种影响

### 一、人际中的同性竞争

- 妒合主**同性之间的较劲**：男对男、女对女。
- 职场里常见"**同级同事之间的暗中使绊**"、"同组里永远有人抢功劳"。
- 生活里常见闺蜜之间、兄弟之间**因嫉妒而失和**。

### 二、合作者之间的信任危机

- 当日主是妒合中的"妒者"，常对合作方之间的亲密关系**心存芥蒂**。
- 表现：
  - 两位合伙人走得近 → 怀疑被架空；
  - 下属两人交好 → 担心被结盟排挤。
- 猜忌心若不自察，容易把**本来稳定的合作生生搅散**。

### 三、情感中的"第三者"介入

- 妒合格局下，日主常**扮演"第三者"角色**——不一定是情感上的第三者，而是**对他人亲密关系的介入者**。
- 典型现实剧本：好友恋爱顺利时，自己**不自觉在对方面前贬低其男/女友**；表面关心，实则离间。
- 网络短视频常见的"闺蜜不停说男朋友坏话"情节，正是此格的写照。

### 四、兄弟姐妹之间的争夺

- 妒合对应**家庭内部同辈的资源争夺**：
  - 父母的偏爱；
  - 家产分配；
  - 长辈关爱。
- "小姑子嫉妒嫂子"的古老比喻至今成立——命格里带此结构者，**家庭关系中天然有嫉妒的种子**。

## 合化结果（通常妒合亦不化）

- **妒合的合本身依然成立**（阳干与阴干的合未被断），但——
- 第三字克合中的一方，使合化**条件受损**：
  - 被克方力量减弱 → 化神难以得气；
  - 合化五行虚浮 → 无法真正转化。
- 结果：妒合之局**九成不化**，呈**合而不化 + 妒克拉扯**的双重结构。
- **例外**：若地支化神极强、第三字又被冲合化解，则合可成，但日主（妒者）往往**事与愿违**，越妒越被孤立。

## 现实象意

- **性格画像**：
  - 猜疑心重、嫉妒心强。
  - 对别人的亲密关系敏感、对他人成功难以由衷祝福。
  - 擅于**挑拨离间、传话递话**，有时自己察觉不到。
- **人际模式**：
  - 朋友少、知己难；
  - 与其远离敌人，不如远离**妒合之命的"好友"**——因为伤你最深的往往不是对手。
- **事业象意**：
  - 同行之间的嫉妒、同事之间的暗斗。
  - 打小报告、传闲话、占功劳的行为多出此格。
- **情感象意**：
  - 自己恋爱时，容易被身边人破坏；
  - 若自己是妒者，则容易**因嫉妒而主动破坏他人感情**。

## 判断维度

| 维度 | 要点 |
| --- | --- |
| 三字排列 | 是否严格按"阳+合阴+克阴"顺序；顺序错即不成妒合 |
| 日主属性 | 末字为日主，判断其妒气来源与强弱 |
| 克制力量 | 第三字是否得根有气——有气则妒得动、破得成 |
| 地支化神 | 化神若旺，可减轻妒合破坏；化神若虚，妒合尽显 |
| 岁运引动 | 流年引出被克之字得根 → 妒合破局；引出比劫 → 妒意加剧 |

## 误区

- **误区一**：把"甲甲己"、"乙乙庚"这类同字相邻当作妒合。这些是**争合**，不是妒合。
- **误区二**：以为妒合只看末字。事实上三字**顺序严格**，末字只是日主定位。
- **误区三**：见妒合就断此人必是坏人。命格展现的是**倾向**，修养与环境可以改变显化程度。
- **误区四**：把妒合等同于普通克制。妒合的特殊之处在于**借由合关系产生的心理投射**——没有合的存在，就没有妒的来由。

## 古籍背书

- **《渊海子平·论合》**：论及**"合中有克，克则为妒"**——合关系存在，而克者从旁介入，合之义受损，是为妒。
- **《子平真诠·论用神》**：言**"用神相合而有克者，谓之妒合，其用为他干所嫉，福气难全"**。用神逢妒合，福分必受磨折。
- **《三命通会·天干争合》**（及邻近篇章）：列妒合五式，**"妒合之人，心怀不平，难容他美"**，并指出**妒合多应于兄弟、同事、同辈之间的嫉妒破坏**。

## 一句话总结

**妒合 = 旁有克者看不得合之亲，合不绝而难化，妒在心而害在人——同性之争、嫉妒之象、离间之行。**

## 附：争合 vs 妒合 的本质对比

| 对比维度 | 争合 | 妒合 |
| --- | --- | --- |
| 结构 | 两字同时合一字 | 三字相邻：合 + 克合者 |
| 阴阳 | 异性相合但有争夺 | 合本异性，妒者与合中一方**同性而克** |
| 典型例 | 两乙合一庚、两庚合一乙 | 甲己乙、乙庚辛、丙辛丁、丁壬癸、戊癸己 |
| 动作 | **争夺** | **嫉妒破坏** |
| 心理 | 风流、摇摆、被宠 | 猜疑、嫉妒、挑拨 |
| 人际象 | 三角恋、合作拉锯 | 闺蜜离间、同事暗斗、兄弟争宠 |
| 合化 | 多半不化（力涣） | 亦多不化（被克） |
| 破坏点 | 合的**专一**被瓦解 | 合的**稳定**被克制 |
| 解法 | 除去多出的同字 / 合化得地支强助 | 除去克者 / 化其妒气为正用 |

**一句话对照**：**争合是"谁和谁好"的悬而未决；妒合是"你们别太好"的暗中破坏。**
