# 寅巳申三刑（无恩之刑）

## 定义

- **寅巳申三刑**，又称**无恩之刑**。
- 三刑关系：**寅刑巳、巳刑申、申刑寅**。
- 能量与破坏力在地支相刑中**次于丑未戌三刑**。

## 核心原理

### 为何称"无恩之刑"

- 命带三刑俱全者，性格往往**冷酷、薄情、无义、背信弃义**。
- 甚至可能**恩将仇报**。
- 男性易遭**杀伤、背叛、血光、牢狱之灾**。
- 女性则多损孕、伤残。

### 驿马属性

- **寅、巳、申均为驿马**——代表不稳定与人际变动。
- 寅巳申三刑 → **多暗示奔波劳苦**；金木暗战，内心矛盾强烈、郁闷。
- 这种刑往往源自**外部人际关系或组织人事**引发的事件，如人事斗争、互相攻击、褒贬争夺。

### 藏干互克关系

- 寅藏甲丙戊，巳藏丙戊庚，申藏庚壬戊。
- 巳火旺 → 克申中庚金；庚金旺 → 克甲木；戊土也会克壬水。
- 与丑未戌三刑类似，**原局三支不齐，岁运可补齐**。

## 寅巳申三刑的三种成立情形

### 一、寅刑巳（寅巳半合火局 → 火旺）

- 古书云寅刑巳，本意是**寅木生巳火**。
- 寅午半合火局意味着火势旺盛；寅巳相刑亦可理解为寅巳半合火局。
- **成立条件**：寅、巳两支中至少有一支天干透火。
- 组合举例：**丙寅与巳**、或**丁巳与寅**。
- 火旺则可刑去申金。
- 象意：**动荡、破坏力、责难、受过、废弃、损害**——祸不单行。

### 二、巳刑申（三种情形）

#### （1）相生之意

- 巳火为申金之长生。
- 巳申相刑 = 土旺生金。
- **成立条件**：巳、申两支中至少一支天干透土（戊己土）；同时天干还需庚辛金透出。
- 组合举例：**戊申与巳**、或**己巳与申**。
- 金旺 → 可刑去寅木。

#### （2）相克之意

- 构成寅巳申三刑的基础上，全局**火不旺** → 巳刑申力量非常弱，起不到什么作用。

#### （3）相合之意（巳申相合化水）

- 巳申相合 = 水旺（六合之一）。
- **成立条件**：巳、申两支中至少一支天干透水。
- 组合举例：**壬申与巳**、或**癸巳与申**——巳申合化水。
- 水旺生木。
- 巳申相合后：巳不冲申、申也不冲巳 → 二者之间**只剩相刑**。
- 巳申相刑主灾重，有病灾、牢狱之灾，不过可减轻刑罚。

### 三、申刑寅（申寅相刑表现为申寅相冲）

- 如果出现第二种情况第三点（水旺），则申寅之间就不存在冲，只有刑。
- 目能量旺盛时：寅可刑去申。
- 金能量旺盛时：申可刑去寅。
- 申寅相刑者性格通常：**稳重、言语不多、欲望淡薄、无情感**，容易陷入失去道义、忘恩负义的境地。
- 如果申刑寅处于死绝状态 → 表面赞美、背后诋毁、忘恩弃义。

## 判断方法

- 判断寅巳申三刑的吉凶，首先需明确：
  - 在**哪种情况下火旺**？
  - 在**哪种情况下水旺**？
  - 在**哪种情况下金旺**？
- 总的来说，寅巳申三刑**并非循环刑无中止**：
  - 寅巳刑成火旺 → 火生申（化刑）。
  - 巳申刑成水旺 → 水克火、水生木（转化）。

## 宫位应事

- 与丑未戌三刑相同，可结合宫位和十神代表的六亲判断相刑吉凶。
- 刑到夫妻宫 → 夫妻感情出问题。
- 刑到申金（大肠、肺） + 申又是壬癸水之根（肾、泌尿）→ 对应脏腑同时出问题。
- 申金代表道路 → 注意**交通事故、血光之灾**。

## 案例

男性，癸水日元生于寅月，年支寅木、日支巳火。原局只有寅和巳，未齐全。
- **戊申大运** → 申补齐，构成寅巳申三刑。
- **壬寅流年**分析：
  - **寅刑巳**：全局火不旺且寅巳两干无火透出，无法满足寅巳半合火局的条件 → 巳刑申力量很弱，寅刑巳发生概率小。
  - **巳刑申**：
    - 相生意：大运天干透戊土满足一条件，但岁运命天干均无庚辛金 → 金不旺，无法刑寅。
    - 相克意：火不旺 → 作用微弱。
    - 相合意：**天干一片水、巳申合化水** → 水旺相当于巳申合化水，水旺生木；寅申不冲，只剩相刑。
  - 此时为**三寅一申相刑** → 申金被刑去、伤重。
    - 申金代表**大肠、肺部**；又是壬癸水之根 → **肾脏、泌尿系统**。
    - 壬寅年命主这些部位可能健康出问题。
  - 申也代表道路 → 小心**交通事故、血光之灾**。
  - 夫妻感情也会受较大影响。

## 现实象意

- **事业环境**：频繁变动、人事斗争、组织冲突。
- **健康**：肺部、大肠、肝胆、筋骨、头部；血光、手术。
- **交通**：申金主道路——驾车、出行务必小心。
- **人际**：恩情难续，被人背叛或自己背信。
- **牢狱**：命带三刑全者，男性需特别注意。

## 常见误区

- **机械套用**："寅巳申三刑必灾"——需按三种情形分别判断火、水、金哪个旺。
- **三刑不全不论**——岁运可以补齐，补齐之年才是应期。
- **忽视相合转化**：巳申相合化水之后，冲势可转化为纯刑。

## 一句话总结

- 寅巳申三刑 = 驿马金木火水的连锁暗战。
- 看这组刑，关键是**分清哪种五行旺**：火旺则刑申，水旺则化木，金旺则刑寅。
- 人事上的表现就是"恩情难续"——越往高处走、越需要守住一颗**感恩而不轻变**的心。
