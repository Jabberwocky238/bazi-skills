# 驿马

## 核心认知
驿马真正要看的从来不是"你动不动"，而是**你动的时候身上背着的是什么**。
- 单看"有驿马就要动"几乎是废话——这世界上每天无数人在动。
- **真正拉开命运差距的不是移动本身，而是移动背后的驱动力和被带走的东西**。

## 判断关键：驿马所连的十神

### 1. 驿马连财星、官星
- 不安感本质是**对更高位置的嗅觉**。
- 这个人坐在原地反而越来越难受。
- 焦虑是"雷达在响"，而不是内耗在作怪。
- 让这种人稳定，等于让猎犬拴在院子里看猎物从篱笆外跑过去。

### 2. 驿马连比劫
- 味道完全变了。
- 动是被**人际冲突挤压逼出来**的，不是被目标拉走。
- 说的永远是"这边太烂了，同事不行、老板有问题"，很少说"那边有什么值得我去"。
- 被负能量赶走，不是被目标拉走。
- 命理上这叫**流浪，不叫进阶**。

## 动的年份：内在气机必须有出口
- 驿马动的年份常见：坐立不安、心浮气躁、工作效率下降、关系开始变乱。
- 表面看是运气不好，实际是"必动的气"在体内找出口。
- 不让他往外走，他就往内炸 → 焦虑、自毁、翻脸。

## 正确的应对方式
真正懂的人会**主动把自己放进"流动"里**：
- 跑项目、跑客户。
- 出差、搬家、换环境。
- 不是为了迷信，而是给"必然的动"一个正确的泄洪口。

### 顺动 vs 憋动
- 顺着驿马走 → 机会。
- 硬憋着 → 事故。

## 两种人的分水岭
- 一个人在动之前脑子里只有"我受不了了" → 基本被**劫财驿马**牵着走。
- 脑子里有"我知道下一步要去哪" → 哪怕辛苦，也是在走向更高层级。

## 常见误区
命理新手最容易犯的错：把"动"当成美德，觉得变化就是好。
- 绝大多数人的"动"是在原地打转。
- **真正有价值的迁徙是从一个层级去到另一个层级**。

## 核心判断
看驿马要看"马背上背的是什么"：
- 背着财官 → 进阶的动。
- 背着比劫 → 流浪的动。
- 没人问"你动不动"，要问"为什么动"。
