# 假从格

## 定义

- **假从格**：从真从格变化而来的一类——日主根浅力薄不能自立，局中**虽有印星或比劫来帮身**，但这些印比**势单力薄、自顾不暇**，拿不住强大的一党旺势。
- 日主依靠不了这"一点虚浮的帮手"，最终**只能顺从一党的旺势**，取用方式**与真从格完全一致**。
- "假"字指的不是"伪装"，而是**看似身弱、实则从格**——形似破格、实际仍论从。
- 细分：**假从财、假从杀、假从儿、假从势**。

## 核心：看似有救、实则无救

- 天干虽有一位印比、但或被财克、或被官杀制、或被食伤泄尽、或被旺势反克 → 这位印比是个摆设。
- 若地支一旦有本气印比之根 → 就不是假从、归入身弱破格讨论。
- 虚浮印比平时靠旺神压制、不起作用；一旦大运流年得根激活 → 立即破从 → 剧烈反噬。

## 与其他相关格局的对比

### 假从 vs 真从

| 维度 | 假从格 | 真从格 |
|---|---|---|
| 天干印比 | 有一位虚浮无根者 | 无 |
| 地支印比根 | 绝无（共同底线） | 绝无 |
| 旺势 | 旺但略杂 | 纯而不杂 |
| 取用 | 顺旺势（与真从一致） | 顺旺势 |
| 运势起伏 | 动荡比真从更剧（虚浮印比是定时炸弹） | 顺运极贵、逆运暴跌 |

### 假从 vs 身弱破局（**最易误判**）

| 对比项 | 假从格 | 身弱有根 |
|---|---|---|
| 地支本气根 | **绝无** | 有 |
| 地支中气 / 余气根 | 严格论无、宽论可有微弱余气 | 可有 |
| 天干印比数量 | 最多一位、虚浮无根 | 可多位 |
| 补救 | 顺旺势、不可补身 | 必须补身（印比） |
| 结果 | 贵格 / 富格、运顺则飞 | 下局、忙来忙去无所得 |

**核心识别差 = 地支本气根**：一字之差、用法相反。

## 成立条件（严格依古籍）

### 条件 1：日主无根（地支本气无印比）

- **古籍出处**：
  - 《滴天髓·从象》：关于真从的铁律同样适用于假从——"**若有一毫印绶（的根），即不为从**"。
  - 《滴天髓》（任铁樵注）："**局中虽有印比，但被克泄殆尽，不能为日主之用者，亦当论从**。"
- **客观判据**：
  - 地支四位的**本气**皆非日主之比劫、印星；
  - 严格论：中气亦无；宽论可容微弱余气。
- **不满足则破**：地支有印比本气根 → 立即破从、归身弱破格。

### 条件 2：天干最多一位虚浮印比

- **古籍出处**：
  - 古法铁律："**天干只能有一位印比，看起来好像有帮身、其实一点用都没有。**"
- **客观判据**：
  - 天干仅容许**一位**印或比劫透出；
  - 这一位必须**虚浮无根**——即坐下无本气根、亦无可用中气根、或虽有余气根但被刑冲 / 被旺势压制。
- **不满足则破**：天干两位及以上印比 → 身弱破格、不可论假从。

### 条件 3：旺神成气势满局

- **古籍出处**：
  - 《子平真诠·论杂格》："**从者，必其势一气贯通。**"
- **客观判据**：
  - 命局中财、官杀、食伤之某一党压倒性独旺；
  - 月令为该党之本气、或地支月令参与三合三会成该党之局；
  - 天干地支中该党数量占大半。
- **不满足则破**：旺神势散、无一方独大 → 须按假从势格另论。

### 条件 4：虚浮印比确实被制 / 被泄 / 被克

- **客观判据**：天干那位印比满足以下任一条件——
  - 坐在财星上被财克（如庚金印坐寅木财）；
  - 被食伤泄尽（如壬水比上方大片甲乙木）；
  - 被旺势反克（如戊土印坐寅木中庚金绝地）；
  - 库根被刑冲破（如辰戌冲后、辰中癸无根）；
  - 被合化消弭（戊癸合化、丁壬合化等）。
- **不满足则破**：虚浮印比实有效用 → 日主真有救 → 归身弱论。

### 条件 5：一党内部不互相厮杀

- **客观判据**：
  - 假从财则无额外比劫夺财；
  - 假从杀则无食伤制杀（若有须弱且有财通关）；
  - 假从儿则无印克食伤、无官杀混入；
  - 假从势则财官食伤顺生不反克。
- **不满足则破**：一党内部有克制、旺势断裂、气不归一 → 假从不纯。

## 判断维度

### 1. 虚浮印比的常见类型

| 类型 | 判据 | 示例 |
|---|---|---|
| 坐财被克 | 干支相克 | 印坐财（戊土印坐寅木财对庚金日主） |
| 被食伤泄尽 | 天干自身被泄 | 印坐食伤或被食伤围攻 |
| 被旺势反克 | 旺神反扑 | 印坐绝地（对日主而言的十二长生绝地） |
| 刑冲破根 | 库根被冲破 | 辰戌冲后藏干无用 |
| 合化消弭 | 干合化成他象 | 戊癸合化火 |

### 2. 月令与旺势类型

| 类型 | 月令要求 | 旺势要求 |
|---|---|---|
| **假从财** | 月令财或三合三会财局 | 财 ≥ 食伤、财 > 官杀 |
| **假从杀** | 月令官杀或三合三会官杀局 | 官杀 ≥ 财、官杀 > 食伤 |
| **假从儿** | 月令食伤或三合三会食伤局 | 食伤 ≥ 比劫、食伤 > 财 |
| **假从势** | 月令可为食伤、财星或官杀（不可为印比）| 财官食伤同旺、无一方独大 |

### 3. 藏干用法铁律

1. **取格局时、藏干可忽略不计**——只看本气，不看中气、余气。
2. **藏干只在大运流年透出时**才起作用（里应外合）。
3. **月令藏干透出**亦要看月令本身有无被刑冲伤害。
4. **距离太远的藏干**作用微乎其微、可忽略。
5. **地支三合三会时**、藏干的力量被严重削弱。
6. **刑冲会破坏藏干**（辰戌冲、丑未冲、三辰自刑等 → 藏干都不能算根）。

初学者最易犯的错误就是死盯藏干——在月令本气和全局旺势面前、藏干那点余气须往后靠。

## 喜忌总则

### 总原则
- **与真从格完全一致**：喜神 = 顺从对象及其生助方；忌神 = 印星、比劫。
- 但假从需**特别小心天干那位虚浮印比所属五行的大运** → 一遇得根即引爆。

### 各假从格的取用

#### 假从财格
- **喜**：财星、食伤（生财）、官杀（泄财）。
- **忌**：印星（生日主 + 生该虚浮印比）、比劫（夺财 + 呼应虚浮比劫）。

#### 假从杀格
- **喜**：官杀、财（生杀）。
- **忌**：印（化杀并养虚浮印）、比劫（帮身抗杀）、食伤（制杀破格）。
- 木不可多见（针对"假从杀 + 木为印食者"）。

#### 假从儿格
- **喜**：食伤、比劫（适量生食伤、不可多）、财（泄食伤）。
- **忌**：印星（克食伤）、官杀（食伤克官 + 克身）。

#### 假从势格
- **喜**：一党中最旺者、能顺生者；若命局需通关、则通关之五行亦为喜。
- **忌**：印、比劫。

## 现实表现

### 与真从格的共性
- 性格特质基本一致：
  - 假从财 = 多情重义、异性缘佳、爱面子、企划能力强。
  - 假从杀 = 大佬气场、果决威严、领导才能、言出必行、嫉恶如仇。
  - 假从儿 = 聪明秀气、创造力强、才华横溢、深得下属拥戴。
  - 假从势 = 恭敬有礼、处事圆满、心地善良。
- 成格时福报仍在、大运顺时可有大成就。

### 假从独有的特点

- **对大运的敏感度更高**：一遇印比年剧烈波动。
- **人生起伏更大**：虚浮印比在原局就埋了伏笔、随时可能被引爆。
- **内心比真从更分裂**：真从是彻底认命、假从是"半顺半不甘"——深处仍有一丝挣扎。
- **六亲提示**：那一位虚浮印比所代表的亲人（印 = 母亲、比劫 = 兄弟），与命主缘分较浅或关系复杂。
- 典型场景："**前几十年风光得很，到某个印比大运突然一落千丈**"、"**事业顶峰时一年重大变故，从此轨迹改变**"。

## 破格与救格

### 破格情形
- 地支出现印比本气根 → 立即破假从、归身弱论。
- 天干印比超过一位 → 非假从。
- 虚浮印比被大运流年激活得根 → 剧烈反噬。
- 一党内部厮杀、气不归一 → 假从不成立。

### 救格
- 虚浮印比被刑冲合去、或被其他喜神克制 → 格局更清、更接近真从。
- 喜神大运接应及时 → 过了危险期、格局稳固。

## 大运流年影响

| 场景 | 表现 |
|---|---|
| 走顺势运（旺神及其生助方）| 大喜 → 贵格兑现 |
| 走印运 | 母亲有灾、学业停滞、身体不适 |
| 走比劫运 | 破财、兄弟姐妹疏远、合伙拆伙 |
| **流年天克地冲日柱或月柱** | 重大事件 → 婚变、官非、意外、健康重症 |
| **岁运并临（天干地支一气忌神）** | 生离死别之险 |

**古籍口吻**："**一旦走水木运如果碰到甲寅运这种干支皆为木（忌神），天干地支一气的忌神运力量极大——一旦遇到，轻则破大财，重则有性命之忧。**"

## 常见误区

### 1. "天干有印比 = 身弱 = 不能从"
- 错。关键要看**这一位印比是否虚浮无根**。
- 虚浮无根 → 假从成立、取用同真从。
- 有根 → 归入身弱破格。

### 2. "假从比真从差"
- 成格时、假从也是贵格。
- 差别在**稳定性**：假从的运势波动比真从更剧。
- 格局的贵气不打折、但人生的戏剧性更强。

### 3. 误把假从当真从
- 若原局地支确有印比根（哪怕中气 / 余气），就不是假从。
- 误判后按从格顺旺势走 → 该补身时不补 → 走到印比运时还以为"破从"、实际是"救命药来了反被认作毒药"。

### 4. 把"虚浮"理解错
- 虚浮 ≠ 天干有即虚浮。
- 虚浮 = 天干在地支**没有本气根、也没有可用的中气根**、或**有根但被冲破 / 被旺势压尽**。
- 判虚浮须一支一支确认、不能想当然。

### 5. "天干超过一位印比还能论假从吗？"
- **不能**。假从的前提是"天干最多一位虚浮印比"——两位及以上就不是假从、而是身弱破格。

### 6. 执着于藏干
- 取格局时只看本气、不看藏干。
- 藏干的作用主要体现在大运流年透出时以及断具体人事时。

## 古籍集锦

- 《滴天髓·从象》："**若有一毫印绶（的根），即不为从。**"
- 《滴天髓·从象》（任铁樵注）："**局中虽有印比，但被克泄殆尽，不能为日主之用者，亦当论从。**"
- 口诀："**取格局时藏干可忽略，只看本气；藏干的意义在于大运流年透出时刻里应外合。**"

## 一句话总结

- 假从格 = **真从格的孪生兄弟**——日主无根 + 旺神满局 + 天干一位虚浮印比。
- 看似破格、实则仍论从；取用与真从完全一致。
- 运势起伏比真从更剧——**那位虚浮印比是定时炸弹、印比大运一到就引爆**。
- 识别的终极判据：**地支有没有印比本气根**。有 → 身弱破格；无 → 假从仍论从。
