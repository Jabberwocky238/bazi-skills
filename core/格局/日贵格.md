# 日贵格

## 定义
- **日贵**：日主天干所坐之日支**正为该日干之天乙贵人**——贵人坐日柱，即"自作贵人"。
- **日贵格**：日柱为 **丁酉、丁亥、癸巳、癸卯** 四者之一，合称"**四日贵**"。
- 属**特殊格局**，以日柱干支定格，不由月令十神论。

## 四日贵日柱
| 日柱 | 天干 | 日支（贵人位） | 昼夜之分 |
|---|---|---|---|
| **丁亥** | 丁 | 亥（丁之天乙贵人） | **生于白天**：称"**日贵**" |
| **丁酉** | 丁 | 酉（丁之天乙贵人） | **生于晚上**：称"**夜贵**" |
| **癸卯** | 癸 | 卯（癸之天乙贵人） | **生于白天**：称"**日贵**" |
| **癸巳** | 癸 | 巳（癸之天乙贵人） | **生于晚上**：称"**夜贵**" |

**无论昼夜**，统称**日贵格**。

## 为何只有这四日？
**天乙贵人查法口诀**：
- 甲戊庚牛羊（丑未）
- 乙己鼠猴乡（子申）
- 丙丁猪鸡位（亥酉）
- 壬癸蛇兔藏（巳卯）
- 六辛逢虎马（寅午）

在十天干中，**只有丁和癸两个天干的天乙贵人恰可坐于日支**：
- 丁之贵人为**亥、酉** → 故丁亥、丁酉日。
- 癸之贵人为**巳、卯** → 故癸巳、癸卯日。

其余天干的贵人不落于日支本位，无法形成"自作贵人"之结构。

## 核心原理
- **天乙贵人**——神煞中最重要、最尊贵之贵人。传统认为能"**逢凶化吉、遇难呈祥**"。
- **自作贵人**——贵人坐日支，不假外求：
  - 一辈子自带贵人气场。
  - 本人即是"众人眼中的贵人"——**能吸引贵人主动前来**。
- 传统评断：
  - **谦和仁德**：不骄不躁，有长者风。
  - **仪表不凡**：灵秀清秀，气质脱俗。
  - **多有贵人相助**：关键时刻总有人出手。
  - **不破格者多为大富大贵**。
- **贵人位置的解读**：
  - 日柱贵人：代表**一辈子或晚年**贵人运强。
  - 时柱贵人：代表**晚年或子女**有贵。

## 成立条件（古籍出处 + 客观判据）

### 条件 1：日柱为四日贵之一

- **古籍出处**：
  - 《三命通会·论日贵》（万民英）："**日贵者，丁亥、丁酉、癸巳、癸卯四日是也**。贵人乘日干之位，故名日贵。"
  - 《渊海子平·论日贵》同。
- **客观判据**：
  - 日柱干支严格为 丁亥 / 丁酉 / 癸巳 / 癸卯 之一。
- **不满足则破**：非此四日，根本不入本格。

### 条件 2：最好有官星

- **古籍出处**：
  - 《三命通会·论日贵》："**喜官星，官星代表贵气有所依托**。"
- **客观判据**：
  - 原局天干或地支见官星或七杀（官杀皆可作贵气之承载）。
- **效应**：
  - 有官：贵气有依托 → 富贵可期。
  - 无官：仅具"贵人气场"，贵而不显。
- **不严格破格**：无官未必破，但格品下降。

### 条件 3：官星不被冲

- **古籍出处**：
  - 《三命通会·论日贵》："官星不忘需有财来升官，且官星忌冲。"
- **客观判据**：
  - 原局官杀之所在支不被冲。
- **不满足**：官星被冲 → 贵气折损。

### 条件 4：官星不孤 —— 有财来升官

- **古籍出处**：
  - 《三命通会·论日贵》："官星不忘需有财来升官。"
- **客观判据**：
  - 天干有财星通根，能生官 → 官贵可发挥。
- **效应**：财生官 → 官贵显化，贵气真正落地。

### 条件 5：贵人（日支）不被冲破合去

- **古籍出处**（本条最核心）：
  - 《三命通会·论日贵》："**日贵最喜待合，最忌冲破害空亡**。"
- **客观判据**：
  - 日支不被年 / 月 / 时支冲：
    - 丁亥日忌巳；丁酉日忌卯；癸巳日忌亥；癸卯日忌酉。
  - **喜带合**：
    - 丁酉日见辰（辰酉合）→ 贵气牢固。
    - 丁亥日见寅（寅亥合）→ 贵气牢固。
    - 癸卯日见戌（卯戌合）→ 贵气牢固。
    - 癸巳日见申（巳申合）→ 贵气牢固。
  - **或无冲**亦可。
- **不满足则破**：贵人被冲 → 贵气尽失；被合去（贵人与他支合化他物）→ 贵人不为己用。

### 条件 6：贵人 / 财官不在空亡位

- **古籍出处**：
  - 《三命通会·论日贵》："**贵人财官不能在空亡位**。"
- **客观判据**：
  - 日支贵人不落空亡；原局财官所在支不落空亡。
- **不满足则破**：空亡加身 → 贵气不实。

### 条件 7（加重）：不犯魁罡、不遇大运流年冲破合害

- **古籍出处**：
  - 《三命通会·论日贵》："若大运流年犯忌，再遇魁罡，主贫困早夭。"
- **客观判据**：
  - 原局不带魁罡相犯；
  - 大运流年不冲、不合、不害日支贵人。
- **例外**：若柱中构成**更大格局**（如杀印相生、食神制杀等），则让位于大格局论贵。

---

**前六条尽量满足**方为真日贵格。核心判据是**贵人日支不被冲破合去**；最能加分者为"**喜带合**"——贵气牢固。

## 与其他类似格局的对比

| 格局 | 结构 | 与日贵的对比 |
|---|---|---|
| **魁罡格** | 四魁罡日柱 | 魁罡**威猛高大**，日贵**灵秀秀气** |
| **日德格** | 五日德日柱 | 日德主**福厚德行**，日贵主**贵人贵气** |
| **天乙贵人（神煞）** | 查日干见地支贵人 | 日贵是**自作贵人**的格局化 |
| **官印相生** | 官→印→日主 | 官印偏向**学术地位**，日贵偏向**贵人提携** |
| **财官印全** | 财官印俱备 | 财官印须三者齐全，日贵仅需贵人坐日 |

**关键区分**：
- **天乙贵人神煞**：凡命带天乙均可论（有无均可）。
- **日贵格**：必须**贵人正坐日支**——结构性的"自作贵人"。
- 二者不是一回事——日贵格是神煞的**结构化、格局化**。

## 判断维度

### 1. 贵人是否"清"
- 无冲无害：贵气清纯。
- 有冲有合（非喜合）：贵气杂。

### 2. 贵人是否"合得好"
- 喜带合（四种配对之一）：贵气牢固。
- 合去为他支：贵人被夺，不为己用。

### 3. 官星配合
- 有官 + 财来升官：贵气显化。
- 无官或官破：贵而不显。

### 4. 日主强弱
- 身强：贵气有主承载。
- 身弱：虽有贵人，自身不济，贵气亦难发挥。

### 5. 其他格局是否更大
- 若同时构成杀印相生、从官格、财官印全等更大格局 → 让位于大格论贵。
- 日贵成为**附加贵气**。

## 现实表现

### 性格
- **谦和仁德**：待人温润，有长者风。
- **仪表不凡**：灵秀清秀，气质脱俗。
- **心思细腻**：对人对事感知敏锐。
- **不咄咄逼人**：即使身居高位亦谦卑。
- **自带磁场**：周围人乐于亲近、愿意出手相助。

### 事业
- **文化、教育、文艺**：灵秀之气的对口。
- **外交、公关、协调**：贵人磁场之运用。
- **参谋、幕僚、中介**：贵气辅佐型岗位。
- **金融、咨询**：财官配合得当时的方向。
- **不适合过于刚硬的武职**：与魁罡命造的武贵不同。

### 婚姻
- **男命日贵**：贵重**利于合作**——包括婚姻中的合作共赢。
- **女命日贵**：**贵重亦婚姻不美**——贵人气场强、本身要求高，加之"贵人不宜过多"的原则，婚姻中易有挑剔、缘分迟。
- 一般原则：**贵人不宜过多**——多则冗杂，反失其贵。

### 身体
- **灵秀之质**：体格非魁梧型，多清秀。
- **逢凶化吉**：大病大灾多能化解。
- **四柱纯阴之日贵**：需注意性格阴柔带来的情志病——敏感、疑心、钻牛角尖。

## 常见误区

### 1. 以为"命带天乙贵人 = 日贵格"
- 天乙贵人是神煞，到处可见。
- **日贵格**要求贵人**正坐日支**——必须是四日贵之一。

### 2. 以为日贵必富贵
- 日贵主贵气、贵人运，**不必然主大富**。
- 大富需财星配合、大贵需官星配合。

### 3. 忽视"贵人被合去"的破法
- 贵人不仅怕冲，**怕合去他方**——合去后贵人不为己用。
- 喜合与忌合的判断：合后是否化为他物、是否仍留日支。

### 4. 以为"阴日主必阴柔"
- **四柱纯阴**者（多见于日贵格）的确性格偏向保守内向。
- 但若大运行阳气之地，亦可激发阳刚之性。

### 5. 贵人多 = 吉
- 传统明言"**贵人不宜过多**"——多则杂、杂则不贵。
- 命中一二位贵人足矣，多反为累。

## 破格与救格

### 破格情形
1. **贵人被冲**：日支被冲，贵气尽失。
2. **贵人被合去**：贵人与他支合化他物，不为己用。
3. **贵人落空亡**：贵气不实。
4. **财官落空亡**：贵气无着。
5. **犯魁罡冲克**：大运流年加之 → 主贫困早夭。
6. **枭神夺食冲贵**：如癸卯日遇酉金月令之偏印夺食，贵人之卯被冲——贵人被冲破，沦为普通。

### 救格思路
- **贵人带合**：喜合救冲，贵气牢固。
- **比印助身**：日主不弱，贵气有载。
- **财来升官**：贵气真正落地。
- **避免大运流年冲合日支**。

### 破格后的表现
- 沦为普通命或更差，但**日贵性格仍在**（谦和仁德、仪表不凡）。
- 若四柱纯阴 + 贵人被破 → **性格阴郁、疑心重、钻牛角尖**——自己把贵人赶走。
- **关键警示**：贵人运的发挥**也看人品**——豁达开朗者得贵人多，偏执狭隘者即便命中贵人多亦难化吉。

## 大运流年影响

| 运 | 喜忌 | 效应 |
|---|---|---|
| 贵人再现之年 | **大喜** | 贵人之运叠加 |
| 带喜合之年 | **大喜** | 贵气牢固 |
| **冲日支之年** | **大忌** | 贵气尽失，关键事件年 |
| **合去贵人之年** | **忌** | 贵人不为己用 |
| 财旺运 | **喜**（若有官） | 财来升官，贵气显化 |
| 官运 | **喜**（若身不弱） | 贵气有依托 |
| 食伤夺食冲贵 | **大忌** | 贵人被夺，化吉之力失 |
| 魁罡相犯之年 | **大凶** | 主贫困早夭 |

**原则**：日贵**喜行贵人财旺运**，**大忌冲破害和空亡**。

## 一句话总结
- 日贵 = 丁亥、丁酉、癸巳、癸卯**四个"贵人坐日"的日柱**。
- 贵人不被冲合破害 + 官星有财来升 → **谦和仁德、贵人运强、逢凶化吉**。
- 喜带合为最佳（辰酉合、寅亥合、卯戌合、巳申合）→ 贵气牢固。
- 怕冲破合空、怕枭夺食冲、最忌遇魁罡。
- 性格决定贵人发挥——**豁达者贵人多，偏执者再多贵人亦为小人**。
