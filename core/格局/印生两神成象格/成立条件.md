# 印生两神成象格

## 成立条件（严格依《滴天髓》两气双清论）

### 条件 1：命局"两气双清"——只见两种相生的五行

- **古籍出处**：
  - 《滴天髓·源流》："**两气双清，其象最妙**。"
- **判据**：
  - 命局的天干地支五行只集中于**两种相生之气**（如纯水 + 纯木，纯木 + 纯火），几乎无其他五行杂入。
  - **五种具体形式**：
    - **水木相生格**：壬癸日主 + 甲乙木食伤；或甲乙日主 + 壬癸水印绶
    - **木火相生格**：甲乙日主 + 丙丁火食伤；或丙丁日主 + 甲乙木印绶
    - **火土相生格**：丙丁日主 + 戊己土食伤；或戊己日主 + 丙丁火印绶
    - **土金相生格**：戊己日主 + 庚辛金食伤；或庚辛日主 + 戊己土印绶
    - **金水相生格**：庚辛日主 + 壬癸水食伤；或壬癸日主 + 庚辛金印绶
- **不满足则破**：命局见第三、第四种五行之气 → 不成"两气双清"，转入普通格论。

### 条件 2：两气之间必为相生关系（非相克）

- **古籍出处**：
  - 《滴天髓》："**两神双清，贵在相生**——相克则为战、不为清。"
- **判据**：
  - 两种五行必须是相生关系（水生木、木生火、火生土、土生金、金生水）。
  - 如果两气为相克关系（水火、木土、火金、土水、金木），则不是"两神成象"，而是"两气相战"或"水火既济"等另论。
- **不满足则破**：两气相克 → 转入"水火既济""水火相战"等其他格论。

### 条件 3：原局无财星、无官杀（或财官力量极微）

- **古籍出处**：
  - 《滴天髓》："**两气成象，忌见财官破局**。"
- **判据**：
  - 原局天干无财星透出；若地支仅藏一位余气，气微可忽略。
  - 原局天干无官杀透出；若地支仅藏一位余气，气微可忽略。
- **不满足则破**：
  - 财透破印（两气中包含印者）→ 格破；
  - 官杀透来克日主 → 两气中的"我"被克 → 格破；
  - 转入普通身强见财官讨论。

### 条件 4：日主属两气之一，且不过弱

- **判据**：
  - 日主必须属两气中的其一（"生"方或"受生"方），不是局外第三方。
  - 日主于月令得气或他柱有比印通根（不至极弱）。
- **不满足则破**：日主若是第三方五行 → 格不成立；日主极弱 + 印在对方 → 另论从印类。

---

**四条同时满足**方为真印生两神成象格。此格之贵在"**双清**"——气势集中、不被杂气搅扰；最忌财官之混入——一见财官则格破，降入普通命局论。

## 喜忌总则

- **喜**：比劫、食伤、印绶（顺两气相生之势）
- **忌**：财、官（破坏双清之局）

## 与相关格局的区别

| 格局 | 区别 |
|---|---|
| **印生两神成象格**（本条）| 两气相生为清；五种五行组合对应五种成象格 |
| **两气成象**（总论）| 两气并立的总概念；含相生、相克两种情况 |
| **水火既济** | 两气相克（水 + 火）但有木通关——本格不含此 |
| **专旺格** | 一气独尊（曲直 / 炎上等）——本格是两气而非一气 |
