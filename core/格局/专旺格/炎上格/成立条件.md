# 炎上格

## 成立条件（严格依《渊海子平·炎上格》）

### 条件 1：日主为丙或丁火

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·炎上格》："**炎上格——丙丁日生人**。"
- **客观判据**：日主天干必须为丙火或丁火。
- **不满足则破**：非丙丁日 → 不入炎上格。

### 条件 2：地支会成火局（三合寅午戌 或 三会巳午未）

- **古籍出处**（本条核心）：
  - 《渊海子平·炎上格》："**地支寅午戌三合火局，或巳午未南方火气**。"
- **客观判据**（二者之一）：
  - ① 地支见 寅 + 午 + 戌 三合火局；或
  - ② 地支见 巳 + 午 + 未 三会南方火。
  - 若仅半合（寅午、午戌）或二字会气，格品降但勉强成立。
- **不满足则破**：地支不成火局 → 一气不聚。

### 条件 3：天干多透火，最好有甲乙木生之

- **古籍出处**：
  - 《渊海子平·炎上格》："年月时干宜见丙丁……**有甲乙木相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位丙或丁透出；
  - ② 若另见甲或乙透干生火，格品升（薪柴充足）。
- **不满足则破**：天干火气不足 → 一气未遍布。

### 条件 4：原局无壬癸水透干灭火

- **古籍出处**（本条核心）：
  - 《渊海子平·炎上格》："**年月时干无水相克**，主威武贵显。"
- **客观判据**：
  - ① 天干无壬水或癸水透出；
  - ② 地支本气无亥水、子水成势。
- **不满足则破**：水透克火 → 《渊海子平》"水相克"即明破格 → 转入"水火相战"或普通克身格讨论。

### 条件 5：土金搭配适度（土微泄可喜，土重泄衰；金微耗可，金重则忌）

- **古籍出处**：
  - 《渊海子平·炎上格》："若得土稍泄其气，更为秀丽。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - ① 土透干或地支本气仅一位 → 微泄秀气，可喜；
  - ② 土多透多根 → 泄气过度 → 降格为"火土夹带"或"火炎土燥"讨论；
  - ③ 金微则火有用武之地，金重则耗火无谓。
- **不满足则破**：土过重或金过重 → 一气变。

---

**前四条必须同时满足**方为真炎上格；条件 5 关乎格品高低——有木生则薪柴不断，微土泄则秀气外显。
