# 帝王命造

## 成立条件（古籍 + 客观判据）

### 条件 1：格局必须清纯，不可混杂

- **古籍出处**：
  - 《滴天髓·通神论》："**清纯者贵，混杂者贱**。"
  - 《子平真诠·论用神成败救应》："格局贵清不贵浊。"
  - 《三命通会》：帝王命造中多言"**格清气旺**"为至贵之标。
- **客观判据**：
  - 原局格局单一明确——或走正格（正官 / 七杀 / 财 / 印）、或走专旺 / 从强 / 从势、或走化气。
  - 无官杀混杂、财印混杂、食伤混杂、真假神混杂等情形；若有混杂，必有"去混留清"的合化或冲去机制。
- **不满足则破**：格局混杂 → 气散不凝 → 虽有贵气也难登极顶。

### 条件 2：五行流通或气势纯粹二者居其一

- **古籍出处**：
  - 《滴天髓·通神论》："**五气之流通，始生之妙；一气之独盛，专旺之奇**。"
  - 《子平真诠·论用神》："**流通不悖，成大贵之基**。"
- **客观判据**（两条路径任择其一）：
  - **路径甲：五行流通型**——财生官、官生印、印生身、身生食伤、食伤生财，首尾相连成环；或成财官印全之结构。
  - **路径乙：一气独盛型**——专旺、从强、化气、天元一气等偏枯到极致的气势。
- **不满足则破**：五行零散、既不流通也不纯粹 → 气势无骨架 → 无帝王之相。

### 条件 3：日主或格局核心"立得住"

- **古籍出处**：
  - 《滴天髓·体用》："**用神有力、日主能承，方为上命**。"
  - 《子平真诠·论用神成败救应》："**真用真忌，不摇不动**。"
- **客观判据**（根据格局类型分论）：
  - 正格：日主于月令得比劫 / 印星之根，或他柱有多处通根。
  - 从格 / 化气：顺势之方无任何一字反叛，且化神 / 从神纯净无破。
  - 专旺：独旺之行独大、无克无泄或有泄而不伤本。
  - 身杀两停：身杀势均而有辅助调节。
- **不满足则破**：日主或格局核心被大幅削弱、被冲克 → 承不住位 → 帝位无缘。

### 条件 4：大运顺行且与命局深度配合

- **古籍出处**（本条为"凡帝之基"）：
  - 《三命通会》："**命好不如运好**；命运相济，方登极贵。"
  - 《滴天髓·岁运》："**大运顺则发福，大运逆则虽贵不显**。"
- **客观判据**：
  - 大运连续 20–40 年行于命局之**喜用方向**，无中断。
  - 关键大运（青年立业、中年登顶）与命中用神之方一致。
  - 不见大运严重冲破命局之核心根（如月令之冲）。
- **不满足则破**：命虽好但运背逆 → 一生踟蹰、难成大业 → "**命运不齐，富贵无缘**"。

### 条件 5：原局无根本性的破格之伤

- **古籍出处**：
  - 《子平真诠·论用神成败救应》："成中有破，败中有成。"
- **客观判据**：
  - 无"真神被合去"、"真用被冲破"、"忌神独大压制用神"等致命破格。
  - 若有小破格，须原局有"救应"（合 / 冲 / 生 / 化）之机制。
- **不满足则破**：根本破格无救 → 格局塌陷 → 无望大位。

---

**五条同时满足**方为真帝王命造之结构。五条中**第 1（清）、第 2（流 / 纯）、第 4（运）**最为核心——古籍屡言"格清、气旺、运顺"三者齐备方为帝王之基。

## 补充条件（依民间命理实证与经典深读）

以下三条为对原五条的**细化与外延**——不是替代，而是对"帝王命造"结构的进一步雕刻。

### 补充条件 A：格局成格的"三关"（对条件 1"清纯"的细化）

- **来源**：知乎《聊聊格局成格》、《小白入门-成格局的基本条件》等民间命理整理。
- **三关**：
  1. **护卫**——立向十神（立格之十神）周围有**完善的保护 / 抑制机制**。即用神之侧有忌神的克制之物、有喜神的扶持之物、形成攻守互备。
  2. **真假**——护卫十神**根气、力量**的问题：**有根则有力为真；无根则无力为假**。
  3. **清纯**——无混杂（对应原条件 1）。
- **客观判据**：
  - 立格之十神在命局中有直接护卫（官星旁有印化杀、财旁有食生 / 官制比劫救财等）。
  - 护卫之神**地支有本气或中气根**，而非孤立的天干虚透。
  - 全局十神不混杂（官杀不混、财印不混、食伤不混）。
- **意义**：帝王命造"**质配**"要求的最低门槛；三关俱备方为中富贵以上，帝王级更须此为基础。

### 补充条件 B：七杀化权的核心机制（对条件 2、3 的具象化）

- **古籍出处**：
  - 《杀印相生格》（传统命理集成）："**杀印两停且日主有根，主文武双全、将相王侯之命**。"
  - 《子平真诠·论七杀》："**七杀有制化为权**"——七杀不怕多，只要有制（食制）或有化（印化）即可。
- **客观判据**（帝王命造中七杀处理的三层）：
  - ① **食神制杀**：命中食神透干通根且与七杀紧贴、力量两停 → 杀被制为权。
  - ② **印化七杀**：印星透干通根且位于杀与日主之间 → 杀印相生。
  - ③ **羊刃驾杀**：身旺之羊刃与七杀两停 → 武贵将帅型。
- **意义**：三路处理七杀的任一路径成立 → 格局可具帝王级的"承压 + 征伐"能量；无处理之七杀 → 纯克身，与帝王无缘。
- **与原条件关系**：条件 3"立得住"的**具体实现路径之一**；条件 2"气纯"的**进阶要件**。

### 补充条件 C：地支骨架（四正 / 四库齐全的加分项）

- **实证来源**：
  - 清代乾隆皇帝八字：**子午卯酉四正齐全**（"遍野桃花"之格）。
  - 明代朱元璋八字：**辰戌丑未四库齐全**且顺行。
- **判据**（两类任一成立即为地支帝王骨架）：
  - **四正齐全**：命中四地支分别为 **子、午、卯、酉**（北、南、东、西四极之气）。
  - **四库齐全**：命中四地支分别为 **辰、戌、丑、未**（四时之库、万物之藏）。
- **意义**：
  - 四正齐全 → 气机在四方极点之间转动，主气象恢宏、八方来朝之象。
  - 四库齐全 → 万物归藏、底盘厚重，主**开创守成兼备**。
- **非必要但为加分**：
  - 非帝王命造的必要条件（绝大多数帝王命造并不具备此结构）。
  - 若具备则格品直上顶级，是"**天地骨架俱备**"的特殊标记。

---

**综合**：原五条（清、纯 / 流、承、运、无破）+ 三条补充（三关、七杀化权、地支骨架）= 帝王命造的**结构完整识别体系**。

现有五条为**必要条件**；补充 A 是条件 1 的深化，补充 B 是条件 2、3 的具象路径，补充 C 是加分而非必要。
