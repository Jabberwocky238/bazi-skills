# 官禄宫

## 定义

**官禄宫**是七政四余命盘里的核心之位，代表一个人的**事业、社会地位、自我价值实现**。但它不仅仅局限于此——还**隐藏着关于责任、成长与人生课题**的信息。

古人讲"**尊莫尊乎日月，美莫美乎官福**"——把官禄宫、福德宫直接与日月（最尊贵的存在）相提并论，可见其分量。

**别称**：**中天之位**，象征人在社会中的终极追求：功名、利禄、事业权柄。

## 核心命题

### 一、官禄宫的多重维度

**古典层面**
- 对应**官职、俸禄、爵位、读书考官**
- 与田宅宫相对，承载"**光宗耀祖**"的家族使命

**现代层面**
- 不只是"能不能当领导 / 企业里的职位"
- 还包括在**整个行业中的影响力**：
  - CEO、教授、自由职业者的业界口碑
  - 是核心人物还是边缘人物
  - 是发号施令的还是执行的

**精神层面**
- 象征**生命成熟度**
- 是否有担当的勇气、独立思考的智慧
- 这些不是天生具备的，要经历事情、波折、人情冷暖后**慢慢锻炼出来**
- "一将功成万骨枯"——正是官禄宫最好的写照

### 二、人生时间表

- **30 岁以后**开始走 **15 年的官禄宫大运**
- 此时是**精力、资源、成熟度三者巅峰叠加**
- 这段运势对人生成就有决定性作用

## 判断维度：五层联动

看官禄宫不是单独看某一颗星是吉是凶，而是要五个层面综合：

| 层面 | 内容 |
|---|---|
| **1. 星体状态** | 官禄宫主星是否旺、受克、被卸 |
| **2. 星体互动** | 各星体之间的生克、照合 |
| **3. 特殊组合** | 格局、凶吉组合 |
| **4. 神煞** | 贵人、凶煞的加持 |
| **5. 流年变化** | 当下运势触发点 |

**另需配合**
- 奴仆宫的辅助（团队资源）
- 官禄宫与命身的关系（能不能驾驭）

## 核心判断要点

### 一、官禄宫主星强弱

| 配置 | 效果 |
|---|---|
| 主星是**日、月**或有日月落着 | 能量强，易得贵人提携 |
| 星体被克 | 能量减弱，按"克它的星"所主领域受影响 |

### 二、宫位间的影响示例

- **夫妻宫主星克官禄宫主星**
  - 亲密关系影响工作发展
  - 另一半想让你换轻松工作照顾家庭，你不同意
  - 易吵架、易冷战
- 星体间的影响 → 引导人世间的事

### 三、流年行运的触发

- 用神星进入官禄宫来生 → 事业加能量
- 流年太阳与官禄宫主星产生互动 → **事业起飞信号**

**典型场景**
- 岁某年：木星对官禄宫有利 + 入官禄宫
- 恰好单位内部竞聘
- → 凭过往积累 + 天时助力，**从普通职员晋升部门经理**

## 现实应用：成长路径把握

### 信任但不依赖

- 可借助**贵人力量、平台资源**
- 但最终**自己要努力**
- **没人能替你成长，也没人能替你收获**

## 官禄克命：能量驾驭不了

### 概念

- 官禄主星**克到命宫或命度**
- 意味着**官禄宫能量非常强**，但**品德与智慧驾驭不了**自己的社会成就
- 容易被**权力和名利反噬**

### 化解之道

- **时刻反省**，保持谦卑
- **不弄虚作假，不炫耀成就**
- 学古代贤臣：地位再高**永远弓着腰、客客气气、谨慎低调**
- 这样不易因傲慢树敌

## 现实表现

| 配置 | 典型画像 |
|---|---|
| 日/月入官禄 | 事业能量强，贵人易到 |
| 主星受克 | 事业推进有阻，按克星所主领域受制 |
| 流年用神入官禄 | 事业起飞的黄金窗口 |
| 官禄克命 | 名利大但难驾驭，易被反噬 |

## 误区

- "官禄宫 = 仅看当官" → 错，**所有事业、行业影响力都在这里**
- "主星好就一定能成事" → 错，要看**五层联动**（星、互动、组合、神煞、流年）
- "30 岁前看不到事业好坏" → 可以看苗头，但**大运走到官禄宫**才是黄金决定期

## 一句话总结

> **官禄宫不仅是地位与成就，更是责任、成长与选择——主星强弱是基础，宫位互动是轨道，流年行运是时机，而"驾驭得住"则是避免反噬的底线。**
