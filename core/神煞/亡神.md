# 亡神

## 定义
- **亡**：损失、失去、失神。
- **亡神**：三合局中"**冲中气**"的位置——代表**内失、内耗、暗损**的凶煞。
- 又称"**天罡煞**"或"**官符煞**"（与官司、是非、内部瓦解相关）。

## 起法（按年支或日支查）
《三命通会·论亡神》原文："**寅午戌亡在巳，巳酉丑亡在申，申子辰亡在亥，亥卯未亡在寅**。"

| 三合局（年支或日支）| 亡神位置 |
|---|---|
| 寅午戌 | **巳** |
| 巳酉丑 | **申** |
| 申子辰 | **亥** |
| 亥卯未 | **寅** |

**背后原理**：亡神是三合局中**紧邻中气的前一位（长生位）**，代表**气机未成而先露**——内部有漏。
- 寅午戌火局，中气午，亡神巳（火之长生）；
- 亡神 = "尚未成形就已经开始漏"的状态。

## 亡神 vs 劫煞 vs 灾煞

| 神煞 | 位置 | 气质 | 主方向 |
|---|---|---|---|
| **亡神**（本条） | 三合局长生位（长生前的冲气）| 内失、暗损 | **内部瓦解** |
| **劫煞** | 三合局长生前一位 | 外夺、被抢 | **外部侵害** |
| **灾煞** | 三合局中气对冲 | 明灾、突袭 | **重大意外** |

传统上"**亡神 + 劫煞 + 灾煞**"合称"**三煞**"——三合局中最凶的三股气。

## 核心作用：内部耗损的预警
- 亡神不是"**外面有人害你**"，是"**你自己的内耗 / 内乱**"：
  - 精神内耗、反复自我怀疑。
  - 团队内讧、合作散伙。
  - 家族内部矛盾、兄弟反目。
  - **自己打败自己**的典型。
- 古籍论："**亡神入命，主是非口舌，内部瓦解**。"

## 判断维度

### 1. 位置
- **年柱亡神**：家族、祖上有耗——可能家族内部传承断裂。
- **月柱亡神**：事业 / 同事 / 合伙人内部矛盾。
- **日柱亡神**：婚姻或自身精神内耗。
- **时柱亡神**：子女或晚年内部问题。

### 2. 亡神配十神
- **亡神 + 比劫**：兄弟合伙内讧、被同行或合伙人暗算。
- **亡神 + 官杀**：官非口舌、被人暗中构陷。
- **亡神 + 印星**：学业 / 文书上的内耗。
- **亡神 + 财**：暗地破财、被亲近者用钱的事拖垮。

### 3. 大运流年引动
- 大运 / 流年走亡神位 → 那段时期**内部裂痕集中爆发**。
- 原局已带亡神 + 流年再引 → 症状加倍。

## 常见误区

### 1. "带亡神 = 必破败"
- 亡神只是"内部的风险点"，不是必然破败。
- **身强 + 格清** → 能扛，亡神之险被化解。
- **身弱 + 亡神引动** → 容易真的出状况。

### 2. 把亡神当成"外部凶煞"
- 最大的误解。亡神的本质是"**自己的内部漏洞**"——不是"外人害你"。
- 识别对应的生活策略：**自省、内修、少与旧人起冲突**。

### 3. 忽视亡神的"警示"价值
- 带亡神的人若能**主动修自己的内部**（情绪管理、决策克制）→ 反而把凶煞之气转化为"**自我觉知的能力**"。
- 这就是命理里"**凶煞可化**"的实际意义。

## 现实表现

### 性格
- **内心矛盾重**：常常自己跟自己打架。
- **对身边亲近人敏感、易猜疑**。
- **情绪波动大**：表面平静内心翻腾。
- **一旦"崩"则崩得很彻底**：不是小事，常是"熬了很久终于受不了"。

### 典型情境
- **团队散伙**：老合伙人忽然闹僵。
- **家族内斗**：兄弟姐妹为利益反目。
- **婚姻内耗**：夫妻冷战、同屋不同心。
- **官司是非**：被人暗中举报、被家人牵累。
- **情绪崩溃**：表面稳妥的人突然抑郁、辞职、远走。

### 健康层面
- 长期精神紧绷 → **失眠、消化问题、慢性病**。
- 亡神 + 刑冲 → 需警惕**突发性健康事件**（尤其内脏类）。

## 化解与应对

### 结构层
- **命局有印星化煞**：印能消耗亡神之凶。
- **大运喜神来**：煞被压制。

### 人事层
- **减少与旧关系的内部纠缠**：懂得切割。
- **学会情绪管理**：练习冥想、倾诉、写作等"内部泄洪"方式。
- **警惕"亲近人的话"**：亡神引动时，最大的伤害常来自最亲的人。

### 防御层
- 流年走亡神 → 那一年**少签合同、少卷入家族争议、少做合伙决策**。

## 核心判断
- 亡神不是"外面的鬼"，是"**内部的魂不在**"。
- 化解亡神的核心：**把自己找回来**——精神不散、心不乱、决策不慌。

## 一句话总结
- 亡神 = **自己瓦解自己的倾向**。
- 这类命造的人：**最大的敌人从来不是外面，是自己**。
- 把内部稳住，亡神再凶也伤不到你；**精神散了，亡神不来也会出事**。
