# 化气

## 定义

- **化气**：天干五合在特定条件下不再是异性相克，而是**合而化之**，转化为一种新的五行之气；此五行即称**化神**。
- 化气是天干五合的最高级形态，属传统格局中的**化气格**（又称"化象"），因其类似从格，可视作从格体系中演变出来的极罕见特殊格局。
- 五合化气对照：
  - **甲己合化土**
  - **乙庚合化金**
  - **丙辛合化水**
  - **丁壬合化木**
  - **戊癸合化火**
- 化气格最大的特异之处：在化气格中**日元可以变**——平常我们讲日元不可变，而化气格里日主已随化神转换，如同变色龙，故此命难被大运真正伤动，一旦真化成象，即为帝王将相之命或其配偶之命。

## 核心原理

- 化气之机在于：日主所参与的五合**升级为五行转换**——
  - 日主与贴邻之异性天干合，且所化之神在月令当权 / 无克无争 / 日干无根。
  - 合化既成，日主不再以原本之五行论喜忌，而以**化神为太极点**。
- 传统口诀："化之成象，则谓之真化。" 真化则贵，假化则贱。
- 化气的本质是"让度"：日主放弃自身五行，顺从合化之气；让得越彻底，化气越纯，贵气越显。
## 真化 vs 假化

| 类别 | 特征 | 定性 |
| --- | --- | --- |
| **真化** | 五条齐备、化神当令、化干无根、无克无争 | **极品富贵**——帝王将相或其配偶 |
| **假化** | 部分条件不足（如月令失气、微有克神、日干微根） | 中等富贵或"有其形无其实"——孤儿异姓 |
| **不化** | 化气条件大面缺失 | 按普通格局异性相克论 |

## 化神得令与不得令的差异

- **化神得令**：月令即化神本气 → 化气"根正苗红"，格局最稳；大运流年走化神之乡则持续兑现富贵。
- **化神不得令但局中极旺**：须靠地支会局 / 三合支持 → 化气虽成但稍逊，最怕会局被冲破。
- **化神失令又失势**：即便天干合化之象完整，仍属假化，福泽折半。

## 与相关格局的对比

- **化气格 vs 从格**：两者皆属外格。从格是日主顺从满局之势（五行不变），化气格是日主通过合化**转换为新的五行**。化气格在"让度"的深度上超过从格——从格"顺"，化气"变"。
- **化气格 vs 专旺格**：专旺是"原本就强"，化气是"合而变强"。两者都忌破坏主气，但专旺不需合化这一步。
- **化气格 vs 普通正格**：正格以日主本气论喜忌，化气格日主已变，须以化神为太极点重新论格。混用两套逻辑是初学者最大错误。
- **化气格 vs 合而不化**：合而不化是"想化又化不成"的中间态——能量反复横跳，是最常见也最凶险的失败形态。

## 判断维度

### 1. 日元是否参与
- 日干不参与合化 → 直接不构成化气格。

### 2. 合化是否紧邻
- 日干与月干或日干与时干紧贴为合化，隔柱不论。

### 3. 化神月令与旺度
- 月令是否为化神本气？地支是否会局助旺？是否被冲克？

### 4. 化干虚实
- 日干在地支是否有根？有根则不化，五化（除乙庚外）尤严。

### 5. 争合妒合
- 是否出现二合一的争合结构？争合一出，化气立败。

### 6. 克神有无
- 原局是否出现克化神之五行？有无救应？

## 现实表现

- **真化气格的人**：先天自带某一方面的显著红利——或为帝王将相之格局、或为其配偶之命；日元"变色龙"属性使其极难被大运真正伤动。
- **化神得令者**：一生顺遂度极高，关键节点上总有贵人 / 平台 / 机遇自动涌现。
- **化气不成者**：一生"感觉应该有但始终抓不住"——看似条件齐备却总差一步，是本格最常见的折损态，也是最耗心力的一种命理折磨。
- **假化者**：中等富贵，但带"出身异常"的色彩——孤儿、异姓、早年离家、过继，皆属此型。

## 常见误区

### 1. 见五合即论化气
- 最大误区。没有日元参与、没有月令根基、有争合或化干有根 → 只是普通五合的异性相克，不构成化气。

### 2. 以为化气一定是好的
- 错。**化气成象**方为极贵；**化气不成**反为凶；**化气淤积**（化神过旺无泄）亦为凶。凶险之源：气机欲化而未化。

### 3. 把年月之合也算作化气
- 年合月干的五合与日主无关 → **不论化气格**。必须日干自身参与。

### 4. 乙庚化金与其他化气等同处理
- 乙庚化金对"化干无根"的要求较宽，其他四化几乎必须日干深弱无根。混用标准会导致判格错误。

### 5. 原局明显见化神克神仍强论化气
- 如丁壬化木却月令见申酉 → 根本化不动，绝不可强论。

### 6. 心理暗示硬套
- 假的真不了。不要因"自己喜欢"就硬认化气格——假从格假化气各有本来面目，心里要有标准。

## 破格与救格

### 破格的典型情形
- **化干有根**：日主不肯让度。
- **化神被克**：外干或地支克神紧贴化神。
- **争合妒合**：二合一结构。
- **月令克化神**：根本性的结构错位。
- **地支冲破化神之根**：化神会局被冲散。

### 救格的可能路径
- **化神被克，地支会局化掉克神**：如丁壬化木见戊土克化神，若地支寅卯辰三会木已将戊土化掉，则化象仍可保全。
- **化干微根而化神极旺**：局中化神当令又会局，可勉强成假化，但降一档论。
- **大运补足化神之根**：原局化神不够旺，若大运走化神帝旺之地，可在那一段时期内按"化气真成"兑现。

## 大运流年

| 大运 / 流年 | 效果 |
| --- | --- |
| 顺化神之气（化神临官、帝旺、库地） | **大喜** → 化气兑现、富贵昭显 |
| 泄化神之五行运（化神所生者） | **喜** → 耗泄化神，流通有序 |
| 克化神之五行运 | **大忌** → 化神被伤，化气崩解 |
| 冲化神之地支运 | **大忌** → 根破，贵气消散 |
| 引化干复根之运 | **大忌** → 日主本气回归，化象破 |
| 再来合化干之运（争合） | **忌** → 争合出现，日主分心 |

以丁壬化木为例：喜木火土（顺化神、泄化神、泄化神之气），忌金水（金克木、水引壬水返本 / 阻丁壬相合）。

## 一句话总结

- **化气成则极贵，化气不成反为凶**——古籍铁律。
- 判化气看五点：**日元参与、月令化神、无克化神、无争合、化干无根**。
- 真化之命日元可变，如同变色龙——这是化气格独有的最大福泽，也是其"帝王将相之格"的根本所在。

## 补充：化气的位置取象与效率曲线

以下内容保留自既有非冲突论述，与本派古籍标准并列参考。

### 化气的四类（按位置划分）

化气按"五合天干所处位置 + 根基所在位置"分为四类，各带不同意义：

| 类型 | 五合位置 | 根基位置 | 所主之事 |
| --- | --- | --- | --- |
| **外化气** | 年月天干 | 月支 | **外部机遇 / 事业助力**——他人、贵人、大平台带来的推力 |
| **自化气** | 月日天干 | 月支或日支 | **先天个性 / 性格优势**——自带的气质禀赋 |
| **内化气** | 日时天干 | 时支 | **内部田产 / 家境资源**——早年环境、父母供给、不动产 |
| **偏化气** | 天干在一处、根基在另一处（错位） | 偏离常位 | **部分扭曲的先天优势**——有天赋但带后遗症 |

> 注：古籍派认为"年月相合不论化气"，故"外化气"一类在古籍标准下不属真化气格，仅作位置取象的辅助视角参考。

#### 外化气（辅助视角）
- 核心取象：外部化来一股不属于自己但可为己所用的力量。
- 现实表现：事业上有贵人持续相助、进入大平台顺风顺水。
- 注意：不必强求"用"它，只要**不拒绝**，它便潜移默化地帮你。

#### 自化气
- 核心取象：化气加持在性格层面，日主获得一种"并非原属自己"的先天个性优势。
- 现实表现：性格上带有强烈辨识度，社交面具完整。
- **双面性警示**：自化气的优势会在另一方面反噬（事业极顺 / 对亲密关系极苛刻等）。

#### 内化气
- 核心取象：化气落于日时，转化为内部资源——早年生活、父母支持、家境田产。
- 现实表现：出身环境温厚、父母开明有能力。

#### 偏化气
- 核心取象：五合天干位置与根基位置错位。
- 现实表现：确有先天优势，但带着戾气，优势越大反噬面越显著。

### 化气效率曲线（辅助理解）

可将"化气率"视作横轴，"增益"视作纵轴：

- **0%**（不化反克）：增益约 0.5，按异性相克论。
- **0% ~ 40%**（化气不成）：增益降至 0 甚至负值——反为凶。
- **40% ~ 70%**（化气将成）：增益快速上升。
- **70% ~ 100%**（化气成气）：增益接近顶峰。
- **100% 之上**（化气淤积）：增益下滑——做功堵塞。

曲线为**突变式**——这正是《滴天髓》"化之真者为贵，化之不真者为祸"的数学写照。

### 不要拒绝化气的原则

即使自己不主动用化气，也**不要在原局配置刻意冲克化神**——典型拒绝结构如外部化气火土 vs 日时配置金水，内外交战，白瞎天赋。
