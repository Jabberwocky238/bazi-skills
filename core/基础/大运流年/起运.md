# 起运

## 定义
- **起运** = 一个人**正式进入大运轨道**的起点。
- 出生后并非立刻入运，而是按**阴阳性别 + 出生距节气远近**算出一个岁数，从那一年开始第一步大运。
- 起运之前这段"真空期"，命主基本跟着**月柱的气场**过日子（童年主要受月令主导）。

## 起运岁数的计算

### 基本规则——阳男阴女顺数、阴男阳女逆数

| 类别 | 顺逆 | 数法 |
| --- | --- | --- |
| **阳年生男 / 阴年生女** | 顺行 | 从出生日**顺数到下一个节气** |
| **阴年生男 / 阳年生女** | 逆行 | 从出生日**逆数到上一个节气** |

- 阳年：甲、丙、戊、庚、壬年。
- 阴年：乙、丁、己、辛、癸年。

### 换算口诀
- **三天折一岁，一天折四个月，一个时辰折十天**。
- 数出生日到节气之间的天数，用这个公式换算成**虚岁起运**。

### 举例
- 出生距下一个节气 **9 天** → 9 ÷ 3 = **3 岁起运**。
- 出生距上一个节气 **4 天 6 个时辰** → 4÷3 ≈ 1.33 岁 + 6 时辰×10 天 = 1 岁零数月，实际约 **1 岁半起运**。

## 起运月柱推算——月上起运法

### 第一步：确定出生月柱
- 运用**年上起月法**（五虎遁元诀）：**甲己之年丙作首，乙庚之岁戊为头，丙辛之岁寻庚上，丁壬壬寅顺水流，戊癸甲寅好追求**。
- 由年柱天干推出当年寅月（正月）天干，依次顺排至出生月支。

### 第二步：从月柱起大运
- **阳男阴女**：从**月柱的下一柱**开始顺排。
- **阴男阳女**：从**月柱的上一柱**开始逆排。
- 每一步大运管 **10 年**。

### 示例
- 某乙年男命，月柱为"戊寅"。
- 乙属阴，男命 → **阴男逆排**。
- 第一步大运：**丁丑**；第二步：**丙子**；第三步：**乙亥**……依次向上逆推。

## 大运干支排列规则

| 要点 | 内容 |
| --- | --- |
| **周期** | 10 年 1 步大运 |
| **顺序** | 六十甲子顺排或逆排 |
| **阴阳配合** | 干支组合永远是**阴干配阴支、阳干配阳支** |
| **数量** | 常推算 8 步大运 = 覆盖 80 年人生 |

## 早运 vs 晚运的影响

### 早运（起运岁数小，如 1—5 岁）
- 童年就开始**换赛道**，家庭、学业、环境易受大运牵动。
- 若首运为喜用 → 出生家庭条件改善、父母事业上扬。
- 若首运为忌神 → 童年坎坷、多病、家境波折。

### 晚运（起运岁数大，如 8—10 岁）
- 童年基本跟着**月柱气场**走，运程启动偏晚。
- 人生真正的大戏从十几岁之后展开。
- **起运越晚的人，人生的拐点越集中在青年—中年**。

### 起运早晚与命运的关系

| 类型 | 命主体感 |
| --- | --- |
| **早起运 + 首运好** | 少年得志、家世上扬 |
| **早起运 + 首运差** | 童年动荡、先苦后甜的底子 |
| **晚起运 + 青年好运** | 十几岁之后开始发力 |
| **晚起运 + 中年好运** | 大器晚成、厚积薄发 |

## 古籍出处
- **《子平真诠·论大运》**：立定月令，阳男阴女顺推，阴男阳女逆推，以管十年之命。
- **《三命通会》**：大运以月建为主，顺逆排之，观其与四柱生克刑冲，可知一生休咎。
- **《渊海子平》**：三日折一岁，一日折四月，一时折十日，以定起运岁数。
- 三部典籍观点一致：**月柱是大运之母，阴阳顺逆是大运之法**。

## 一句话总结
起运 = **出生时的时空印记**与**未来十年风向**的第一次对接。
- 算起运不是算"什么时候开始人生"，而是算**命主第一个可被推导的气运拐点**。
- 早运看家世，晚运看自立；**顺排接节气之气，逆排引节气之余**，一步一十年。
