# 天干相克

## 定义
- **天干相克** = 基于五行生克规律，一个天干对另一个天干的**制约关系**。
- 相克的本质 = 五行之间"**木克土、土克水、水克火、火克金、金克木**"的循环。
- 据阴阳属性，天干相克可分为**有情之克**与**无情之克**——这是八字实战中极为重要的一对概念。

## 二、有情之克：异性相克

### 1. 定义
- **阳克阴** 或 **阴克阳** = **异性相克** = **有情之克**。
- 被视为**柔质、善意、人情**的克制——**克性较小**。

### 2. 背后的逻辑
- 传统观念：男女互补、男阳女阴，两者相互制约、相互平衡 → 方能达到和谐。
- 现实中，**男性一般不对女性施加暴力**，若施暴会受社会广泛谴责。
- 女性对男性的攻击通常不会造成严重伤害（除非极度仇恨）。
- 男性试图控制女性，往往是为了**获得顺从、跟随、成为伴侣**。
- 女性试图控制男性，往往是为了**获得关爱与关心**。
- 对无关紧要的人，**人们不会投入太多关注、更不会去克制**。

### 3. 与五合的关联
- **有情之克的五组**恰好也是**五合的五组**：
  - **甲 → 己**（甲己合化土）。
  - **丙 → 辛**（丙辛合化水）。
  - **戊 → 癸**（戊癸合化火）。
  - **庚 → 乙**（乙庚合化金）。
  - **壬 → 丁**（丁壬合化木）。
- 五合本质即"**合中带克**"——合化关系源于异性相克的柔情基础。

### 4. 现实意义
- 有情之克 → 克制力度小、带有**柔情、牵制、感情色彩**。
- 常见现实画像：
  - 在职场中上下级存在克制关系，但因性别或性格互补，**实际相处融洽**。
  - 命主与配偶存在官/财相克关系，但感情深厚、彼此扶持——克中有合、合中有克。

## 三、无情之克：同性相克

### 1. 定义
- **阳克阳** 或 **阴克阴** = **同性相克** = **无情之克**。
- 被视为**强制攻击、猛烈、不留情面**的克制。

### 2. 背后的逻辑
- 同性之间**缺乏互补性**，相互克制易引发冲突与矛盾。
- 现实中：
  - 两男冲突 → 往往全力以赴，甚至使用武器、至死方休。
  - 两女争执 → 也可能激烈打斗，拉扯头发、抓对方的脸。
- 无情之克不仅克性强，且**留有深怨**，不易和解。

### 3. 典型组合
- **甲 → 戊**、**乙 → 己**（木克土）。
- **丙 → 庚**、**丁 → 辛**（火克金）。
- **戊 → 壬**、**己 → 癸**（土克水）。
- **庚 → 甲**、**辛 → 乙**（金克木）。
- **壬 → 丙**、**癸 → 丁**（水克火）。

### 4. 现实意义
- 无情之克 → 克制力度大、**伤害深、持久**。
- 常见现实画像：
  - 同性别、同性格上下级之间的严重对抗。
  - 事业竞争中的你死我活。
  - 家族内部同辈之间的矛盾与争斗。

| 感情色彩 | 有牵制、有人情 | 直接、冷酷 |

## 五、相克与相冲的辨析

### 1. 相克
- 基于**五行属性的对立**——木克土、土克水等。
- 力量较小，变化渐进。
- 戊己土参与相克（甲木克戊己土、戊己土克壬癸水）。

### 2. 相冲
- 基于**方位的对立**——东西冲、南北冲。
- 力量较大，变化突发。
- **戊己土不参与相冲**（戊己居中央）。

### 3. 两者关系
- **冲比克力量大**——冲中含克，克中未必含冲。
- 例：甲庚冲（既是方位冲又是金克木）= 冲克并存、力度最大。
- 例：甲戊克（只是木克土、无方位对冲）= 仅为克、力度次之。

## 六、天干相克的实战应用

### 1. 日干遭克
- 日干受克 → 易**身体受损、压力倍增**。
- 有情之克 → 压力以柔性方式呈现（如职场管理、长辈督促）。
- 无情之克 → 压力以刚性方式呈现（如严苛竞争、硬碰硬冲突）。

### 2. 天干十神相克的意义
- **官杀克比劫** → 权威约束自我。
- **食伤克官杀** → 主观表达消解规则。
- **财克印** → 物质欲望冲击精神依赖。
- **印克食伤** → 思想理性压制主观表达。
- **比劫克财** → 同辈分夺财富。

### 3. 有情与无情在十神中的体现
- **正官**（异性克日主）= 有情之克 → 柔性权威、规则约束。
- **七杀**（同性克日主）= 无情之克 → 刚性压力、强势挑战。
- **正财**（异性被日主克）= 有情之克 → 稳定的财富、妻子（男命）。
- **偏财**（同性被日主克）= 无情之克 → 流动的财富、情人（男命）。
- **正印**（异性生日主）vs **偏印**（同性生日主）—— 生的对照，非克的对照，但同样有"正/偏"的柔刚差异。

## 七、辨证要点

### 1. 克并非一定为凶
- **忌神被克** → 为吉（病得药）。
- **喜神被克** → 为凶（良辰被伤）。
- 不能见克就判凶。

### 2. 有情之克的"贪合忘克"
- 当有情之克的两干满足五合条件、却不能合化 → 形成**合而不化**：
  - 被克者力量减弱。
  - **主克者不再全力克制对方**——贪合忘克。
- 例：乙庚合而不化 → 庚金不再克乙木；但仍可克甲木。

### 3. 克的力度受多因素调节
- **相邻 > 远隔**：紧贴的克力度大。
- **通根 vs 不通根**：有根者克有力，无根者克力弱。
- **有合 vs 无合**：合可消减克。
- **有冲 vs 无冲**：冲中含克，力度倍增。

## 八、一句话总结

- 天干相克 = **五行生克在十干上的具体呈现**。
- 依阴阳分：**异性相克为有情（克中带合、柔性）**、**同性相克为无情（刚烈不容）**。
- **有情之克** = 五合的五组，力度小、带人情。
- **无情之克** = 阳阳、阴阴相克，力度大、难调和。
- 克比冲力量小、过程渐进；冲比克力量大、变化突发。
- 克的吉凶看**喜忌**——忌神被克为吉、喜神被克为凶。
- 有情之克在一定条件下可转为**合化**或**贪合忘克**——展现了阴阳对立统一的哲学本质。
