# 天元一气格

## 成立条件（古籍 + 客观判据）

### 条件 1：四柱天干必须完全相同（一字贯通）

- **古籍出处**（本条为本格前提）：
  - 《三命通会·论天元一气》："**年月日时四柱之干，同为一字者，谓之天元一气**。"
- **客观判据**：
  - 年干 = 月干 = 日干 = 时干，四者为同一天干（十天干中任一）。
- **不满足则破**：只要有一柱天干与其他不同，即不入本格（可能转入专旺、从格或两干不杂格等讨论）。

### 条件 2：地支须配合天干之气势，不可严重背离

- **古籍出处**：
  - 《三命通会·论天元一气》："**天元一气，须看地支之配**；地支合则大贵，地支反则大凶。"
  - 《滴天髓·通神论》："干支一气，气势贯通者，其命不凡。"
- **客观判据**：
  - 地支五行与天干之气**相生 / 相同 / 不冲克** → 气势贯通成格。
  - 典型配合形态（据古籍所载十日主）：壬寅、辛卯、庚辰、己巳、戊午、丁未、丙申、乙酉、甲戌、癸亥——此十组地支与天干分别构成特定气势关系。
- **不满足则破**：地支严重冲克天干（如四庚却配四寅——申寅冲被打乱）→ 气势被冲破 → 格局不成或凶。

### 条件 3：气势类型明确（相生 / 相克 / 全同 三种之一）

- **古籍出处**：
  - 《三命通会·论天元一气》："**干同一气，支有相生、相克、同类三象**。"
- **客观判据**（地支配合类型）：
  - ① **相生类**：如壬寅（水生木）、己巳（火生土）、丁未（火生土）—— 日主与支中主气成相生关系。
  - ② **相克类**：如庚辰（金克木 / 土生金之复合）、乙酉（金木相克）、甲戌（木土相克）、辛卯（金木相克）、丙申（火金相克）—— 日主与支中主气成相克关系。
  - ③ **全同类**：如癸亥（水水全同）、戊午（火土同流）—— 日主与支中主气完全同类（纯气到极致）。
- **不满足则破**：若地支五行混杂不能归入以上三类 → 气势涣散 → 不成天元一气之贵。

### 条件 4：原局无严重刑冲破害扰动气势

- **古籍出处**：
  - 《三命通会·论天元一气》："气势贯通者贵，刑冲破害者凶。"
- **客观判据**：
  - 四柱之间无明显刑冲（如子午冲、寅申冲、卯酉冲等）打破气势。
  - 若有轻微刑冲但被合化解或被顺势吸收，可视为不破。
- **不满足则破**：地支间严重冲克 → 一气之贯通被撕裂 → 由贵转凶。

---

**四条同时满足**方为真天元一气格。本格最核心的是条件 1（四干同气）与条件 2（地支配合）——此两条是格局的骨架，条件 3、4 决定格局的层次与吉凶倾向。
