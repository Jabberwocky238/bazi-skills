# 飞刃

## 定义
- **飞刃**：与**羊刃（阳刃）**正对冲的地支。
- 羊刃是"**自身之刃**"（日主帝旺之地）；飞刃是"**从对面飞来的刃**"——对冲羊刃的位置。
- 本质：**外来的、迎面而至的刚烈之气** → 主突发冲突、意外外伤、被动伤害。

## 起法（按阳干日主查飞刃）
《三命通会·论羊刃》相关论述："**羊刃对冲，谓之飞刃**。"——羊刃位置的正对冲即飞刃。

| 日干 | 羊刃位置 | 飞刃位置（羊刃对冲）|
|---|---|---|
| 甲 | 卯 | **酉** |
| 丙、戊 | 午 | **子** |
| 庚 | 酉 | **卯** |
| 壬 | 子 | **午** |

**注**：阴干（乙丁己辛癸）传统上不立羊刃，故也不立飞刃。阴刃之说有但作用不显。

## 飞刃 vs 羊刃

| 对比项 | 羊刃 | 飞刃 |
|---|---|---|
| 位置 | 日主帝旺之地（命中地支为羊刃）| 对冲羊刃的地支 |
| 本性 | 自身之刚 | 外来之刚 |
| 典型表现 | 脾气、主动冲突 | 意外、被动受伤 |
| 组合 | 羊刃 + 飞刃（刃冲刃）→ 最凶 |

**重要组合**：命中**原局既有羊刃又有飞刃** → 称"**刃冲刃**"，主**自相砍杀 / 内外双伤**，为羊刃格的极端破格。

## 核心作用：外来的血光
- 羊刃是"**自己手里的刀**"（主动出击型凶煞）。
- 飞刃是"**飞过来的刀**"（被动受击型凶煞）。
- 现实表现：
  - **车祸、被撞、被打**。
  - **突发意外事件**（非自己引起）。
  - **被上司 / 权威 / 对手突然攻击**。
  - **女命产厄、外伤**。

## 判断维度

### 1. 位置
- **年支飞刃**：早年遇外来伤害。
- **月支飞刃**：青年至中年事业中遇突袭。
- **日支飞刃**：本人易被外来伤害——尤其身体。
- **时支飞刃**：晚年或子女易遇外力伤害。

### 2. 飞刃配其他神煞
- **飞刃 + 羊刃（原局同现）**：刃冲刃 → 最凶。
- **飞刃 + 七杀**：暴力伤害、血光之灾。
- **飞刃 + 灾煞**：突发意外倍增。
- **飞刃 + 白虎**：血光更明显。

### 3. 大运流年引动
- 大运 / 流年走飞刃位 → **那段时间外部伤害事件集中**。
- 原局已带飞刃 + 流年再引动 → **重大意外年**。

## 常见误区

### 1. "飞刃就是另一个羊刃"
- 二者方向完全不同：羊刃主动，飞刃被动。
- 羊刃的凶在"自己冲动惹祸"；飞刃的凶在"**被动遭遇意外**"。
- 应对方式也不同：羊刃修自控力；飞刃修警觉性。

### 2. 没有羊刃就不看飞刃
- 严格说飞刃是"羊刃对冲位"——若命中无羊刃，飞刃的凶性会降低。
- 但大运流年走到飞刃位（即便原局无羊刃）仍可引动突发事件。

### 3. 以为阴干也有飞刃
- 传统羊刃只论阳干 → 飞刃自然只对阳干日主成立。
- 阴干日主（乙丁己辛癸）不论飞刃。

## 现实表现

### 典型情境
- **车祸、交通意外**（尤其作为无责方被撞）。
- **被他人攻击**：斗殴、被偷袭。
- **突发疾病**：急症、外伤住院。
- **女命产厄**：生产意外、剖腹产。
- **职业意外**：工伤、安全事故。

### 性格层面
- **对外部风险敏感**——但因感知弱有时反应不及。
- 与羊刃命人相反：飞刃命人**并不主动惹事**，但**易被惹上**。

## 化解与应对

### 结构层
- 命局有印星、贵人（尤其天乙、月德）→ 飞刃之凶减。
- 大运走喜神年 → 飞刃被压制。

### 人事层
- **提高警觉性**：不酒驾、不疲劳驾驶、远离是非场合。
- **注意身体防护**：高危活动加倍小心。
- **流年走飞刃**：**减少远行、避免争执场合、不参与斗殴 / 挑衅**。

### 时间层
- **羊刃 + 飞刃并引动的年份**：最凶之年——出门、做决策、人际关系都需格外谨慎。

## 核心判断
- 飞刃是"**外来刚气的标记**"——对应被动型血光。
- 与羊刃不同：羊刃是"**你的脾气**"，飞刃是"**别人砸过来的拳**"。
- 警觉 + 防护 → 飞刃也能化为虚惊。

## 一句话总结
- 飞刃 = **飞过来的刀**。
- 不是你想惹，是事情找上你。
- 走得稳、看得远、少进危险场合 → **飞刃再凶也擦你而过**。
