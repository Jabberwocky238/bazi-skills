# 做功（有效流通）

## 定义
- **做工**（又称**有效流通**）：八字中**具有导向性**的作用关系——当两个或多个字（元素、组合）相互作用时，能**指明某种元素的用途、规定其流转方向**，这个"作用"本身就叫做工。
- 严格来说，做工**不是某个字**，而是字与字之间的**那个作用（映射）**：a 与 b 之间存在一种作用 f，f 规定了 a 和 b 之间的元素流转与变换 → 这个 f 就是做工。
- 古籍对应：
  - 《子平真诠·论用神》："**用神者，八字中所用之神也**"——用神即八字中真正"起作用"的字与关系。
  - 《滴天髓·通神论》："**何处起根源，流到何方住**"——气的流转与归宿，正是做工的古典表达。

## 核心：把静态的字转化为动态的"取向"

- 原局 8 个字若**无做工**，只是一堆静态的符号。
- 做工是**河道**：天干地支的相互作用是源头之水，做工之河道**规定了水流的方向**，把静态的字转化为动态的人生取向。
- **河道决定方向，水情（细节）填充血肉**：
  - 有做工 → 有明确的人生方向。
  - 无做工 / 做工弱 → 人生方向模糊，容易随大流。

## 做工的四类对象

八字的 8 个字按研究对象划分，产生四类相互作用：

| 类型 | 内容 | 是否做工 |
|---|---|---|
| **天干 ↔ 天干** | 十神生克合化 | **是**（典型做工） |
| **地支 ↔ 地支** | 刑冲合害 | **是**（典型做工） |
| **地支 → 天干** | 十二长生、旺衰正作用 | **不是**（只立框架） |
| **天干 → 地支** | 旺衰逆作用 | **一般不是**（特殊情况下是） |

### 为何只有前两类是做工
- 做工的必要特征是"**导向性**"——能改变元素的走向、决定人生的大方向。
- 天干与天干的生克、地支与地支的刑冲合害，具有明确的导向作用（生食伤、制七杀、冲夫妻宫等）。
- 旺衰（地支 → 天干）只建立"某元素是强是弱、是否高级"的**框架**，不直接规定方向。

### 旺衰逆作用何时成为做工
- 当原局**天干 ↔ 天干、地支 ↔ 地支作用皆无或皆弱**，人生目标空缺时，**旺衰逆作用**（天干调动地支）可以**替代性**地承担导向作用。
- 例：原局作用稀少，大运来"壬午"（天干克地支），壬午的天地合克成为本运段的主要做工。

### 组合作为做工
- **三合 / 三会 / 化气**成局的整体，可作为**组合做工**参与原局流通。
- 典型：寅午戌三合火局以整体去生戊土 → 整体作为做工。
- 《子平真诠·论杂气》："三合、三会，以气之所聚论之"——古籍亦以"气聚为一"视整体参与流通。

## 判断维度

### 1. 是否具有导向性
- **有导向** → 做工成立（如壬水 + 甲木 = 食神生财，方向明确为"以想法换钱"）。
- **无导向**（仅是旺衰支撑） → 不算做工。

### 2. 做工的效率（权重）
- 两字之间距离越近 → 做工效率越高；隔柱越远 → 效率越低。
- **就近原则**：相邻柱 > 隔一柱 > 年时两端相望。
- 《滴天髓·通神论》："**贴身者力重，遥隔者力轻**"——紧贴者为真做工。

### 3. 做工的清浊
- **清做工**：链条单一、方向明确（如"食神生财"一气而下）。
- **浊做工**：链条多条交织，互相牵扯（如"食神生财"又逢"比劫夺财"）。
- 清做工成就专精，浊做工成就杂糅但灵活。

### 4. 原局做工 vs 流年大运做工
- **原局做工**：先天结构决定的人生底色方向。
- **流年大运做工**：
  - 若大运 / 流年**辅助原局用神的做工** → 不构成新做工，只是加强既有方向。
  - 若大运 / 流年**自成一对作用**（引出新的流转） → 是新的做工，通常代表**新方向、新机遇**。

## 做工的两类功能

### 功能 A：规定人生大方向
- 典型：**食神生财**的天干做工 → 人生大方向是"用想法换钱"。
- 典型：**财官印全**的连珠做工 → 人生大方向是"体制 / 学术 / 管理"。

### 功能 B：分化宫位含义
- 地支刑冲合害作为做工，分化出宫位的具体取向：
  - **月支 ↔ 日支** 刑冲 → 父母宫与夫妻宫相扰 → 父母介入婚姻、婆媳不和。
  - **日支 ↔ 时支** 刑冲 → 婚姻宫与子女宫相扰 → 夫妻因子女起争执。

## 常见误区

### 1. 把"做工"当成字本身
- 错。做工**不是某个字**，而是"字与字之间的作用关系"。
- 命盘里"有食神、有正财"不等于"有食神生财做工"——要看它们**是否相邻、是否有通路**。

### 2. 把做工等同于物理学的"做功"
- 两者只是**过程对过程**形式上相似，但本质不同：
  - 物理做功是**力沿路径的积分**。
  - 八字做工是**字与字之间的二元函数关系**。
- 《三命通会·论十干》语义上更近"**气之所用**"——气的流转与使用。
- **建议用语**：在学理讨论中优先使用"**有效流通**"一词，避免与物理学混淆。

### 3. 以为旺衰就是做工
- 错。旺衰只立"框架"（身强 / 身弱 / 高级 / 低级），**不规定方向**。
- 框架是地基，做工是河道——两者缺一不可，但**不能等同**。

### 4. 以为所有八字都有高效做工
- 错。相当一部分八字做工**权重极低 / 隐藏不显**：
  - 同八字之人人生天差地别，正是因为**没有高权重做工可以主导**，策略选择差异极大。
  - 这类八字的主人需要**主动发掘**自身被动流通、借助流年大运建立做工。

## 做工与"细节"的关系

**细节是做工的生命线**——结构好而细节差，依然成不了大格局。

### 细节为何重要
- 同样是"食伤生财"的做工，细节不同，结局天差地别：
  - **清纯有气**：财星无冲无合、食神通根有力 → 富贵稳定。
  - **被污染**：食神被偏印贴身夺、财星被比劫贴脸争 → 徒有其表、能量耗散。
- 《滴天髓·通神论》："**吉凶祸福，皆系乎五行之生克制化，而尤必察其微**"——微即细节。

### 判断细节的三个维度
1. **就近原则**：天干地支的作用严格按"就近优先"。隔柱作用需打折扣，其效力约按"距离平方反比"递减。
2. **是否被污染**：原局的主要元素是否被**合掉、冲散、耗尽、夺食**。
3. **是否被激发**：做工是否在**紧贴位**发生；是否有化气、合化、三合局实现。

### 结构好但细节差的典型表现
- 八字**整体架构看似清纯**（如比劫生食伤、食伤生财、财通印等），但：
  - 月令被**刑冲污染**，气被消耗。
  - 天干财**状态不好**（被污或无根）。
  - 日主所坐的**日支性质偏杂**（辰戌丑未混配、五行交战）。
- 结果：**无明显优势、无明显劣势**，先天高低被平均化 → 易成平庸之命。

### 细节缺失的后果
- **能量溢散**：做工方向明确，但每条路径都被"小刀割肉"，结果稀释。
- **污染蔓延**：原局一处病气可通过合化、刑冲蔓延到全局。
- **先天优势与劣势都不显**：无法形成"突出点"，注定平庸。

## 现实表现

### 做工高效者
- 人生方向明确，早年便能定位事业路径。
- **专精一道**：靠单一长板吃饭，容易做成名家、行业头部。
- 偏执、难转弯、不适合做多面手。

### 做工多样者（杂气格）
- 多条路径可选，**人生充满变化与选择**。
- 需要**因地制宜**：顺大运、看国运、随事业潮流调整。
- 成就可能晚发，但路径更自由。

### 做工稀少 / 隐藏者
- 前半生**方向模糊**，容易随大流（如时代风口强迫选专业）。
- 依赖**流年大运**建立做工。
- 若能找到内核（常在财星），可通过充实内核、抓住大运机遇翻身。

## 一句话总结
- 做工 = **字与字之间的"作用 / 河道"**。
- 有做工，人生有方向；做工清，人生走得远；做工细节无损，人生才真正兑现潜力。
- 只看字、不看作用 → 看不出做工；只看结构、不看细节 → 看不出做工的实际效率。
