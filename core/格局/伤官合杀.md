# 伤官合杀

## 定义
- **伤官**：我所生之异性者，代表**才华、锋芒、独立思考、不受约束**。
- **合**：天干五合 —— 阴阳相配的合化关系。
- **杀**：七杀。
- **伤官合杀**：命局中**伤官与七杀通过天干五合**相合 → 伤官不直接克官、也不被七杀压身 → 两股刚烈之气**互相牵制、彼此约束**。
- 与"**伤官见官**"对照：见官是冲突破坏，合杀是**迂回化解**。

## 核心：两股刚以合相解
- 伤官刚（锋利外放）、七杀刚（高压克制）—— 通常两者都是不好惹的力量。
- 但命局中：
  - 如果伤官与七杀**天干相合**（如甲 + 己、乙 + 庚、丙 + 辛、丁 + 壬、戊 + 癸）且五合发生在伤官与杀之间；
  - 两者就会**互相牵住对方** → 谁也不再单独发作。
- 结果：
  - 伤官不再去冲官。
  - 七杀不再去压身。
  - 两者在合里**各自停下**，日主反而得**清净**。

## 判断伤官合杀的关键
1. **命局同时见伤官 + 七杀**。
2. **两者之间存在五合关系**。
3. **合得住**：位置紧贴、力量相当、无其他字阻隔或争合。
4. **日主旺度合适**：
   - 身不过弱 → 才能借合解出生机。
   - 身过弱 → 合完了，杀还没消净，仍然伤日主。

## 与相关格局对比

| 格局 | 动作 | 用途 |
|---|---|---|
| **伤官合杀** | 伤合杀 | 迂回化解，身得清净 |
| **伤官见官** | 伤克官 | 破格，惹事 |
| **食神制杀** | 食克杀 | 主动解决压力 |
| **杀印相生** | 印化杀 | 接纳转化 |
| **羊刃驾杀** | 身抗杀 | 刚对刚 |
| **伤官佩印** | 印制伤 | 约束锋芒 |

**伤官合杀**的独特在于：**不是压、不是抗、不是化，是"合"**。两股力量**用互相牵绊的方式**变成安静。

## 成立条件（严格依《子平真诠》《滴天髓》《渊海子平》）

### 条件 1：日主必为阴干（乙、丁、己、辛、癸之一）

- **古籍出处**：
  - 《渊海子平·论合》："甲己合、乙庚合、丙辛合、丁壬合、戊癸合。"
  - 《子平真诠·论伤官》："**合杀者，取其合以去杀**。"
- **客观判据（结构推演）**：
  - 伤官 = 我生之异性；七杀 = 克我之同性。
  - 逐日主枚举伤官与七杀所构成的天干对：
    - 甲日：伤官丁 vs 七杀庚 — 丁庚**不成五合**；
    - 乙日：伤官丙 vs 七杀辛 — **丙辛合** ✓；
    - 丙日：伤官己 vs 七杀壬 — 己壬**不成五合**；
    - 丁日：伤官戊 vs 七杀癸 — **戊癸合** ✓；
    - 戊日：伤官辛 vs 七杀甲 — 辛甲**不成五合**；
    - 己日：伤官庚 vs 七杀乙 — **乙庚合** ✓；
    - 庚日：伤官癸 vs 七杀丙 — 癸丙**不成五合**；
    - 辛日：伤官壬 vs 七杀丁 — **丁壬合** ✓；
    - 壬日：伤官乙 vs 七杀戊 — 乙戊**不成五合**；
    - 癸日：伤官甲 vs 七杀己 — **甲己合** ✓。
  - → **仅乙、丁、己、辛、癸五阴日主**在结构上可形成伤官与七杀之五合。
- **不满足则破**：甲、丙、戊、庚、壬五阳日主 → 伤官与七杀结构上无五合关系，本格根本不适用。

### 条件 2：伤官与七杀均透于天干

- **古籍出处**：
  - 《渊海子平·论合》："天干五合，合而化者从化论；不化者，羁绊而已。"（五合只在天干）
- **客观判据**：
  - 伤官与七杀均透于年 / 月 / 日 / 时天干——五合是天干现象，地支不论。
- **不满足则破**：其一仅藏于地支 → 无合可成。

### 条件 3：合的位置紧贴（同柱或相邻柱）

- **古籍出处**：
  - 《滴天髓·通神论》："**合之力，以紧贴为真**。"
  - 《子平真诠》："隔柱而合，合而不力。"
- **客观判据**：
  - 伤官与七杀两天干之间：月 / 日、日 / 时相邻，或月 / 时（间隔一柱，需条件 4 无争合才作真合）。
- **不满足则破**：年 / 时两端遥合 → 合力微弱，不作真合论。

### 条件 4：无其他天干争合（非"争合""妒合"）

- **古籍出处**：
  - 《渊海子平·论合》："**一物可合，二物不可合**——两干争一，谓之争合。"
- **客观判据**：
  - 原局无其他天干与伤官或七杀构成同一五合关系（如己日论伤官合杀，乙庚合；若另有一庚在年干争合，即为争合）。
- **不满足则破**：争合 → 合力分散，两边都合不实，气机混乱。

### 条件 5：合化或不化，其化神（若化）对日主非忌

- **古籍出处**（本条最核心）：
  - 《滴天髓·合化》："**合为解，合为化。解而不化，两神相绊而各不为害；化而成象，另起一局**。"
  - 《渊海子平·论合》："合而化者从化论；不化者，羁绊而已。"
- **客观判据**（两路分处）：
  - ① **不化路径**（月令五行不支持合化）：伤官与七杀"相绊而各不为害"——日主得清净，本格典型吉象。
  - ② **化成路径**（月令五行支持合化，且合化出的五行对日主为喜用神）：化神助日主，加贵。
- **不满足则破**：月令支持合化但化出的五行对日主为忌神 → 虽合了杀，却引入新忌 → 格降或破。

---

**五条同时满足**方为真伤官合杀。本格与"伤官见官"三救应之别：佩印是"压"，生财是"泄"，合杀是"合"——五合为真合、无争合、化神不忌，三项俱备方为沈氏所谓"取其合以去杀；杀去而伤官独清"。

## 判断维度

### 1. 合的对象是什么样的"伤官"与"七杀"
- 伤官原本的强度（是否得月令、有几根）。
- 七杀原本的强度（是否得月令、有几根）。
- 两者力量接近 → 合得住；力量悬殊 → 合不牢。

### 2. 日主是否受益
- **身中或身略强**：日主借合得清净 → 正面。
- **身弱**：合了伤，却没消化完杀 → 杀仍克身 → 作用打折。
- **身极强**：合没合对日主没多大影响 → 本格效用不显。

### 3. 有无争合
- 旁边出现**能与伤或杀相合的其他字** → 形成"争合"。
- 争合发生 → 合力减弱 → 格局乱。

### 4. 合化后的气质
- **甲己合（伤官合正官 / 杀）化土**：化为财星 → 若财为用神 → 大吉。
- **乙庚合化金**：化为官杀本气 → 可能继续压身。
- **丙辛合化水**：化为印星或食伤 → 看对日主作用。
- **丁壬合化木**：化为印星或比劫 → 看结构。
- **戊癸合化火**：化为比劫或印星 → 看结构。
- 不是所有合化都是好事，要具体配合日主喜忌。

## 常见误区

### 1. "有伤官 + 有七杀 + 在一起 = 伤官合杀"
- 必须真的**五合成立**，且位置紧贴、力量匹配。
- 伤官与七杀只是"同时存在"不一定构成合杀。

### 2. 把伤官合杀等同"吉格"
- 是一种**救应手段**，不必然是吉格。
- 合杀之后的**化神是否对日主有利**才决定最终好坏。
- 合化成忌神 → 反倒由可控变失控。

### 3. 以为合了就永远不动
- 大运流年可以**引动合**：
  - 新来的字**破合**（冲掉合的一方）→ 原本被合住的伤或杀**重新活跃** → 可能突然爆发。
- 所以"合住了"不等于"永远安全"。

### 4. 与伤官见官混淆
- **伤官见官**：两者直接对冲 → 破坏。
- **伤官合杀**：两者被合关系固定 → 暂时平息。
- 同样是"伤 + 官杀"组合，处理方式和结果完全不同。

## 现实表现

### 性格
- **外表温和、内里有锋芒**。
- **遇事会绕弯子**，不硬碰。
- **有才华但不张扬**：伤官的锐被合住了。
- **对压力的应对方式是"柔"**：找破口、找合作、找台阶下。
- **善于用关系化解冲突**：合的本质就是"关系的力量"。

### 事业路径
- **需要"才华 + 柔性应对压力"**的岗位：
  - 外交、谈判、公关、商务合作。
  - 精神 / 心理顾问型职业。
  - 协调型管理岗。
- 不适合需要**正面硬刚**的岗位（那是羊刃驾杀或食神制杀的场子）。

### 典型画像
- 遇到强势对手不退让，也不硬顶 —— 而是**找一个两者都能接受的位置**。
- 在冲突中显示"**化解能力**"。

## 破格

### 合被冲破
- 大运流年出现与伤或杀相冲的字 → **冲开合**：
  - 伤官重新能克官。
  - 七杀重新能克身。
  - 长期被压住的问题**瞬间爆发**。
- 典型现象：原本平静的人在某一年突然**失控 / 发生剧变**。

### 合化成忌
- 合化出的五行若对日主为忌 → 看似"解决"了伤杀问题，实则**引入了新的忌神**。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 合神加持运 | 合更牢 → 格局稳 |
| 冲破合的大运 | **大忌** → 合散 → 伤杀重新活跃 |
| 印运 | 视结构 → 若化神为印则喜；否则中性 |
| 财运 | 视合化 → 合化成财时最妙 |
| 比劫运 | 视结构 → 帮身可解部分杀 |

## 与相关救杀格局的系统对比

| 救杀方式 | 动作 | 气质 | 日主需要 |
|---|---|---|---|
| **杀印相生** | 化 | 温和接纳 | 身中 + 有印 |
| **食神制杀** | 克 | 主动出击 | 身中强 + 食神有力 |
| **羊刃驾杀** | 抗 | 刚对刚 | 身极强 |
| **伤官合杀** | 合 | 迂回牵绊 | 身中 + 合要合得住 |
| **从杀格** | 顺 | 彻底臣服 | 身极弱 |

## 核心判断
- 伤官合杀的精髓是"**合而非争**"。
- 两股刚烈的力量通过合**互相变成对方的约束** —— 谁都不再单独伤人。
- 格是否成立的关键：
  - **合得住吗？**
  - **合化成什么？**
  - **大运流年会不会冲破这个合？**
- 比见官要聪明，比驾杀要温和，比顺杀要保留自我。

## 一句话总结
- 两股刚烈的力量**用一场合约彼此约束** —— 谁都不再伤人。
- 伤官合杀 = **不靠压、不靠化、不靠抗，靠"绑在一起"** 解开难题。
- 人生智慧层：学会把对立的力量**放进同一个框架**里互相牵制，反而都能安下来。
