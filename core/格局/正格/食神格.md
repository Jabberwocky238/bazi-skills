# 食神格

## 定义
- **食神**：日主所生之同性五行——阳日主生阳为食神；阴日主生阴为食神。
  - 例：甲日见丙（阳火）为食神；乙日见丁（阴火）为食神。
- **食神格**：**月令取食神为用神**的基础正格。
- 《子平真诠》十正格之一——**吐秀、生财、制杀**之用。

## 核心：吐秀之用
- 食神 = 我生之气，代表**才华、表达、福气、饮食、子女**。
- 食神格的核心气质："**温文尔雅、聪明秀发**"——《子平真诠》谓之"吐秀"。
- 不像伤官那样锋利，食神**温厚有度**。

## 食神 vs 伤官

| 对比项 | 食神 | 伤官 |
|---|---|---|
| 阴阳 | 与日主相同 | 与日主相反 |
| 气质 | 温厚、秀丽 | 锋锐、张扬 |
| 表达 | 娓娓道来 | 锋芒毕露 |
| 风险 | 较安稳 | 易见官致祸 |
| 典型 | 学者、研究、养生、烹饪 | 艺人、发明、破局、变革 |

## 成立条件（严格依《子平真诠·论食神》）

### 条件 1：月令地支本气为食神，或食神透干且月令有根

- **古籍出处**：
  - 《子平真诠·论食神》："**食神者，我生之同性**。月令食神为格。"
- **客观判据**：
  - ① 月令地支本气为食神；或
  - ② 月令地支藏食神且食神透于天干。
- **不满足则破**：月令食神不显 → 当以月令本气另论他格。

### 条件 2：食神透干且通根

- **古籍出处**：
  - 《子平真诠·论食神》："**食神必得生旺之地，方能吐秀**。"
- **客观判据**：
  - 食神透于天干，且在地支有本气或中气根至少一位。
- **不满足则破**：食神虚透无根 → 吐秀不成，格难立。

### 条件 3：日主有根，不极弱

- **古籍出处**：
  - 《子平真诠·论食神》："**食神生旺，胜似财官**——但须身不过弱方可**。"
- **客观判据**：
  - 日主于月令有比劫 / 印星余气，或他柱有一位以上比劫 / 印星通根。
- **不满足则破**：日主极弱 + 食神旺 → 身弱泄气过度 → 转入"身弱食伤多"破局或考虑从儿格（须日主完全无根）。

### 条件 4：原局无偏印（枭神）紧贴夺食，或有财护食

- **古籍出处**（本条最核心）：
  - 《子平真诠·论食神》："**食神最畏枭神**。枭印夺食，福去祸至。"
- **客观判据**：
  - ① 原局无偏印与食神紧贴；或
  - ② 有偏印但有财紧贴克偏印（**财护食**）救应。
- **不满足则破**：枭印夺食无救 → 破格转入"**枭神夺食**"病格讨论。

### 条件 5：食神清而不杂（不与伤官混杂）

- **古籍出处**：
  - 《子平真诠·论食神》："**食神清而不杂，为吐秀之象**。"
- **客观判据**：
  - 原局食神透干有根为主，伤官不透或仅藏支一位不显。
- **不满足则破**：食伤混杂 → 转入"食伤混杂"讨论，格品降。

### 条件 6（辅佐条件）：有财星接应（食神生财）

- **古籍出处**：
  - 《子平真诠·论食神》："**食神生财，美格也**。"
- **客观判据**：
  - 财星透干有根，位于食神之后柱（食→财顺流）。
- **非必要**：不满足不破格，但格品降——纯食神格属"秀而不实"（有才华无变现）。
- **最佳结构**：食神生财 → 才华变现，最兑现本格贵气。

---

**前五条必须同时满足**方为真食神格；条件 6 关乎格品。《子平真诠》"**食神生财，美格也**"——食神格的兑现路径在食→财的链条。

## 判断维度

### 1. 食神与七杀的关系
- 食神 + 七杀 → **食神制杀**，独立成格（见 `食神制杀.md`）。
- 食神纯泄 + 无杀 → **食神吐秀 / 食神生财**，走才华变现之路。

### 2. 大运流年
- **食伤运**：食神加力 → 才华绽放、作品期。
- **财运**：食生财 → 变现期 → 喜。
- **印运（尤其偏印）**：**大忌** → 枭印夺食。
- **比劫运**：帮身 → 身强则可，身极强则闷。
- **官杀运**：视有无食制 → 有食神制杀则转贵。

### 3. 食神在不同位置
- **月柱食神**：天生才华外显。
- **日支食神**：配偶好、自身有福。
- **时柱食神**：晚年享福、子女聪慧。

## 常见误区

### 1. "食神 = 必聪明"
- 食神是"**聪明秀发的底牌**"，需不被枭印夺、需日主有根方显。
- 身弱 + 食神旺 + 无印护 → 有才华而无体力兑现。
- 身过强 + 食神虚 → 有力无处使。

### 2. 把食神等同"饮食口福"
- 古籍"**食神主口福**"是一面，但核心是"**吐秀之气**"——才华与表达。
- 现代对应：创作、教学、研究、服务都是食神之秀的延伸。

### 3. 忽视枭神的潜在威胁
- 原局无枭不代表永远安全——大运流年走枭印地，原局食神被突然克住 → "**才华突然用不出**""**抑郁 / 闭塞**"的典型表现。

## 现实表现

### 性格
- **温和、秀气、有福相**。
- **善表达 + 不张扬**：娓娓道来的气质。
- **对美、对食、对生活品质敏感**。
- **有"**创作冲动**"但不狂躁**。
- **聪明但不炫耀**。

### 职业路径
- **学者、研究者、作家**：温厚型知识分子。
- **烹饪、美食、营养**：字面意义的食神。
- **教师、讲师、文化传承**。
- **医疗（尤其中医、养生）**：调养与福气相关。
- **艺术（非张扬型）**：古典艺术、工艺、文玩。
- **服务业 + 品质型**：以服务品质立身。

### 人生节奏
- **早期显露才华**：读书 / 写字 / 技艺早显。
- **中年稳步变现**：食神生财路径兑现。
- **晚年有福**：食神主福寿，老年往往享福。

## 核心判断
- 食神格的核心在"**吐秀 + 生财**"：吐秀是才华外显，生财是才华变现。
- 识别意义：这类人**走温和型创造 / 服务路径**最顺，硬做对抗型工作则压抑本色。

## 一句话总结
- 食神 = **温秀的才华**。
- 不抢、不躁、不锋利——靠**积累、表达、创造、服务**过好一生。
- 把才华变成可传递的东西（作品 / 服务 / 教诲）→ 食神格真正的福气。
