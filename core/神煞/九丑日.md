# 九丑日

## 定义
- **九丑**：九个特殊的"**阴阳不调、刑克自身**"的日柱。
- **九丑日**：按**日柱干支成格**——命主出生日柱为特定干支之一。
- 主**婚姻不利、自身刑克、气质异于常人**的神煞。
- 与"**日德日**""**六秀日**"同为"**按日柱成格**"类神煞，但性质相反：
  - 日德 → 厚德之人。
  - 六秀 → 秀丽之才。
  - 九丑 → **刑克不调之象**。

## 九丑日的干支
《三命通会·论九丑日》原文："**九丑日——阴阳干支，两凶相合**。"

古籍所列九丑日（十干支中带子午卯酉"四败"之地的组合）：

| 九丑日柱 | 结构特点 |
|---|---|
| **壬子** | 阳水坐阳水刃——刚烈过盛 |
| **壬午** | 阳水坐对冲——冲克不调 |
| **戊子** | 阳土坐财——土虚水冷 |
| **戊午** | 阳土坐刃——燥烈 |
| **己卯** | 阴土坐杀——受克 |
| **己酉** | 阴土坐食——泄气 |
| **乙卯** | 阴木坐刃——过刚 |
| **乙酉** | 阴木坐杀——受克 |
| **辛卯** | 阴金坐财——自克 |
| **辛酉** | 阴金坐刃——过烈 |

**注**：古籍列数略有分歧（有版本作十日），因"九"为约数或"阴阳极"之意；**本质是子午卯酉"四败之地"与阴阳干的特定组合**。

## 九丑日 vs 日德日 / 六秀日

| 对比项 | 日德日 | 六秀日 | 九丑日 |
|---|---|---|---|
| 性质 | 厚德 | 秀丽 | 刑克不调 |
| 典型 | 甲寅、戊辰等 | 丙午、丁未等 | 乙卯、辛酉等 |
| 用法 | 贵格 | 才格 | 病格（需辨） |

## 核心作用：自身的刑克倾向
- 九丑日主"**自坐不调**"：
  - 日柱干支五行关系**或冲、或刑、或克身**。
  - 生来带一种"**别扭、不顺、刚烈或脆弱**"的气质。
- 古籍论："**九丑日主，婚姻不利，身多刑克**。"
- 现代理解：**内在冲突感强、人际关系易紧张、婚姻容易波折**。

## 九丑日的三大影响

### 1. 婚姻方面
- 九丑日坐在夫妻宫 = **婚姻宫自带不调**：
  - 对伴侣要求高且易冲突。
  - 易有二婚、晚婚、离婚等迹象。
  - 典型："**缘分难久、情感多变**"。

### 2. 身体 / 气质方面
- **刚柔不调** → 脾气刚烈或情绪极化。
- **易有慢性病 / 心理问题**。
- **外貌或气质"与常人不同"**——古籍称"**丑**"非外貌之丑，是"**气场不和谐**"之意。

### 3. 人际关系方面
- **人际关系易紧张**——带着锋锐或阴郁的气场。
- **独行侠倾向**：不合群、易孤立。
- **自我矛盾**：内心冲突重。

## 判断维度

### 1. 是否带其他贵神化解
- **九丑 + 天乙贵人**：刑克之凶有化解。
- **九丑 + 月德 / 天德**：气场被调柔。
- **九丑 + 印星护身**：内部冲突减轻。

### 2. 日主强弱
- **身强**：九丑之刚气能扛 → 转化为**刚烈型强人**。
- **身弱**：九丑之气反伤自身 → 易有慢性病、情绪问题。

### 3. 大运流年
- 走喜神大运 → 九丑之凶被压制。
- 走冲击日支的大运（如乙卯日走酉运） → 九丑引动，多事年。

### 4. 配偶宫状态
- 日支为九丑之地 → 夫妻宫刑冲重 → 婚姻反复。
- 若有合化 / 贵人 → 婚姻虽波折但能成。

## 常见误区

### 1. "九丑 = 长得丑"
- 最大误解。"**丑**"非外貌，是**气场不调、阴阳不和**之意。
- 很多九丑日人外貌端正甚至出众，但**气场或命运有"**别扭**"感**。

### 2. "带九丑 = 婚姻必坏"
- 九丑只是"**婚姻不调的倾向**"，不是"**必然婚败**"。
- 配合命局其他因素（夫妻宫有贵人、大运吉等）→ 婚姻可过。
- 识别意义：这类命造**需对婚姻特别用心经营**。

### 3. 把九丑当成"必凶"
- 九丑命的人**走独特路径反而成就高**：
  - 艺术家、独立研究者、特殊技艺。
  - 不走传统路径反而发挥自身之"不同"。

## 现实表现

### 性格
- **刚烈或阴郁**：两极化倾向。
- **内心有矛盾感**：常自我对抗。
- **对平庸不耐受**：不愿走寻常路。
- **独立性极强**：不依赖他人。
- **气场独特**：别人容易记住他但不易亲近。

### 典型情境
- 一生婚姻波折（晚婚 / 离婚 / 不婚）。
- 人际关系中"**合得来的人极少，但极深**"。
- 身体或心理上有长期的慢性问题。
- 事业上走偏门 / 独特路径。

### 职业路径（适合）
- **艺术、创意、独立创作**：把独特气质化为作品。
- **研究型、学者型**：独处能力强。
- **非主流专业**：玄学、小众研究。
- **不喜集体的行业**：自由职业、个人工作室。

## 化解与应对

### 结构层
- 贵人（尤其天乙、月德、月德）→ 调和九丑之气。
- 大运走喜神 → 九丑被压。

### 人事层
- **婚姻宜晚**：年轻冲动时结婚易败，成熟后再择偶更稳。
- **选择能接受"不同"的伴侣**：对方理解并欣赏这份独特，而非试图"改造"。
- **把独特气质导向职业**：别硬走大众路径。

### 心态层
- **接受自己的"不同"**：不必刻意合群。
- **找到自己的节奏**：九丑命的人最大的敌人是"自我否定"。

## 核心判断
- 九丑日是"**自身不调的命定结构**"——婚姻、身体、人际多有挑战。
- 但也带来"**独特气质 + 非凡力量**"——用对方向就是贵。
- 识别意义：**让这类命造走独特路径，别硬套寻常模板**。

## 一句话总结
- 九丑 = **阴阳不调的刚烈 / 阴郁**。
- 不是命不好，是**走寻常路必憋，走独特路反而自在**。
- 接受自己的"不同" → **九丑之命也能活出灿烂**。
