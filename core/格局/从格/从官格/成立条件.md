# 从官格成立条件

### 条件 1：日主极弱，无根无柱
- 天干周围无比印，地支四位**本气、中气、余气皆无比印**。

### 条件 2：月令必须是正官
- 要么月令本气为正官；
- 要么地支月令**参与三合或三会官局**（按日干"克我异性"之五行核对）。

### 条件 3：正官数量与纯度
- 天干地支中**正官数量 ≥ 财星**；
- 且**正官数量 > 食伤**。
- **官不混杀**：全局无七杀透干——若有七杀混入，性质转变为从杀或官杀混杂下局。

### 条件 4：无食伤克官
- 天干无食神 / 伤官透出；地支本气、中气无食伤藏干。
- 食伤一见 → 伤官见官 → 官被克 → 破从。

### 条件 5：无印星化官
- 无印（化官生身）。
- 印一见 → 化官 + 日主有路可退 → 变成"官印相生"正格，非从官。

### 条件 6：无比劫帮身
- 无比劫（帮身抗官）。

### 条件 7（助力条件）：有财星透干生官
- 财生官 → 官有源 → 格局更稳。
- 这是"**财官**"复合路径。
