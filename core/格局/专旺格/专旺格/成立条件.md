# 专旺格

## 成立条件（严格依《渊海子平》《三命通会》《滴天髓》）

### 条件 1：日主与月令同气（比劫或印星得月令本气之助）

- **古籍出处**：
  - 《渊海子平·杂格》："**五行专旺之格，须日主与月令同气**——甲乙日见亥卯未或寅卯辰；丙丁日见寅午戌或巳午未；戊己日见辰戌丑未；庚辛日见巳酉丑或申酉戌；壬癸日见申子辰或亥子丑。"
- **客观判据**：
  - 日主五行与月令地支同属一气（木日 + 木月；火日 + 火月；土日 + 土月；金日 + 金月；水日 + 水月）。
- **不满足则破**：月令非日主之气 → 无一气成象之根，不入专旺。

### 条件 2：地支会该五行之局（三合或三会）

- **古籍出处**（本条核心）：
  - 《三命通会·五行专旺》："**一行得气，必以三合或三会成局为成象**。"
- **客观判据**（按五行分）：
  - 木专旺（曲直）：亥卯未三合木，或寅卯辰三会东方木；
  - 火专旺（炎上）：寅午戌三合火，或巳午未三会南方火；
  - 土专旺（稼穑）：辰戌丑未四库全见或三见；
  - 金专旺（从革）：巳酉丑三合金，或申酉戌三会西方金；
  - 水专旺（润下）：申子辰三合水，或亥子丑三会北方水。
- **不满足则破**：地支只散见同气不成局 → 一气不成象，作普通身旺论。

### 条件 3：天干多透比劫或印星以遍布一气

- **古籍出处**：
  - 《渊海子平·杂格》："**天元须透……使一气遍布**。"
- **客观判据**：
  - 除日主外，天干另有至少两位为同五行的比劫或生我之印星。
- **不满足则破**：天干一气不足 → 未能"遍布"，格难以成立。

### 条件 4：原局无克本气之字（无官杀透干，无财重克印 / 克比劫）

- **古籍出处**（本条核心）：
  - 《渊海子平·杂格》："**一气成象者，不宜见克**——克者破格之源。"
- **客观判据**（按五行分）：
  - 木专旺：天干无庚辛金透，地支申酉本气不成势；
  - 火专旺：天干无壬癸水透，地支亥子本气不成势；
  - 土专旺：天干无甲乙木透，地支寅卯本气不成势；
  - 金专旺：天干无丙丁火透，地支巳午本气不成势；
  - 水专旺：天干无戊己土透，地支辰戌丑未不成势。
- **不满足则破**：克神透干或本气成根 → 一气被破 → 格败。

### 条件 5：食伤之泄可有可无——微泄秀气可喜，重泄破气

- **古籍出处**：
  - 《滴天髓·从象》："**一气成象，微泄秀气方妙；若泄过而衰，非吉象也**。"
- **客观判据**：
  - 食伤仅一位透干或地支一位根为"微泄秀气"，可喜；食伤多透多根 → 泄过。
- **不满足则破**：重食伤泄过 → 一气衰 → 不成象。

---

**五条同时满足**方为真专旺格（分化为曲直 / 炎上 / 稼穑 / 从革 / 润下五种，各有专章）。专旺格是唯一"身极强"的外格，要求"一气遍布 + 无克 + 微泄"——与从财 / 从杀等"身极弱"的外格形成两极对照。
