# 水土对（总论）

## 性质：相克两气对
- **五行关系**：土克水（官杀 / 财关系）。
- **非"印生两神成象格"**——该格专论相生两气，水土属相克，不入双清贵格。
- 属"两气成象"的**相战 / 对立**一支，但有特殊贵格"壬骑龙背"。

## 两气成象通则（适用于本对）

### 成象要件
- 命局只见水、土两种五行。
- 水、土力量大致相等。
- 月令为水或土之一。

### 与相生对的根本区别
- **相生对**：两气顺生、气通为清。
- **水土对（本条）相克**：土克水——属"壬骑龙背"等特殊结构。

### 特殊情况：壬骑龙背（贵格）
- 壬水日主 + 坐辰土（水库）→ 水归库而不被克灭 → **深藏不露的贵格**。
- 条件：
  - 日柱必为壬辰。
  - 命局多壬 / 多辰为加分。
  - 无戊己燥土重压。
  - 无戌冲辰。
- 壬骑龙背是水土对中的特殊贵格——利用**水库**（辰）承水而非克水。

### 病象：土重克水
- 戊己燥土过多 + 水弱 → 水被土堵塞 → 水气不通。
- 现实：智慧 / 才华被体制 / 传统 / 规则压制，发挥不出。

### 取用要点
- **喜金通关**（土→金→水）→ 土生金、金生水，化克为顺生。
- **喜木疏土**（木克土救水）→ 但木过多也伤土根。
- **忌土过重**——尤其戊戌、未土等燥土。
- **忌辰戌冲**（冲开水库、水气散）。

### 古籍背书
- 《三命通会·论壬骑龙背》："**壬骑龙背者，壬辰日也。辰为龙，壬水之库**。"
- 《滴天髓》："**土重克水，得金则化**。"
- 《三命通会》："壬骑龙背，**所喜者多壬、多辰；所忌者，土重及辰戌冲**。"

## 本对下属格局
- **壬骑龙背**：壬水坐辰土的深藏贵格。详见 `壬骑龙背/`。

## 一句话总结
- 水土对 = **相克两气、水易被土克**。
- 壬骑龙背（水归辰库）→ 贵格，靠"**库**"承载而非对抗。
- 要解：**得金通关**（土→金→水）；或**水入墓库**自成深藏。
- 土重水弱 + 辰戌冲 = 最凶。
