# 正印格

## 成立条件

### 条件 1：月令地支本气为正印，或正印透干且月令有根
- **判据**：
  - ① 月令地支本气为正印；或
  - ② 月令地支藏正印且正印透于天干。
- **取用要点**：正印为生我之异性、是八字中**能量最为纯粹的十神**。
- **不满足则破**：月令正印不显 → 当以月令本气另论他格。

### 条件 2：正印透干且通根
- **判据**：正印透于天干，且在地支有本气或中气根至少一位。
- **取用要点**：**印要透干方能生身**——虚透无根则生身无力。
- **不满足则破**：印虚透无根 → 生身无力，格难立。

### 条件 3：原局无财星紧贴破印
- **判据**：
  - ① 天干无财星与正印紧贴（月 / 日、日 / 时邻位）直接相克；
  - ② 若有财，须有比劫隔之（比劫挡财护印）。
- **取用要点**：**印绶最忌财破**——财来破印则学业、资源、庇佑俱失。
- **不满足则破**：财贴印无救 → 印被克 → 转入"以财破印"讨论（需辨印是否为病）。

### 条件 4：身强身弱决定印的角色
- **判据**：
  - **理想取用态**：日主身弱 / 身中，印可为喜用生身；
  - **实务常见态**：正印成格者绝大多数身强（印在月令且透干 + 日元再得一根基本即身强）。
- **取用要点**：
  - 身弱用印化官杀生身 → **不怕食伤**（印可疏通 → 日主由弱转强）。
  - 身弱而官众印轻 → **最忌财运**（财生官杀、又破印 → 日主不胜官杀之克、行财运则有灾）。
  - 身强 + 印重官轻 → **喜财制印同时生扶官星**（财代表勤勉行动、压制印的懒惰心态）。
  - 身强 + 以食伤为喜用 → **最喜财来克印**（让食伤自由发挥 = "印旺不怕财克"）。
- **不满足则破**：身强 + 印重且无财破印或食伤泄 → 印闷气机，必须另取用神。

### 条件 5：官星透干为辅（官生印，格更贵）
- **判据**：正官透干且通根，位于印星之前（官→印生身的顺流）。
- **取用要点**：**印喜官生、官印相生贵不可言**——杀印相生的贵气尤胜官印相生。
- **非必要**：不满足不破格；满足则格品升——转入"**官印相生**"贵格讨论。

---

**前四条必须同时满足**方为真正印格；条件 5 为升格要件。正印格的最佳兑现路径是**官印相生**，形成"官→印→身"的贵格链条。
