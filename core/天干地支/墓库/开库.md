# 开库

## 前置概念：墓库是墓还是库？

决定墓库是"墓"还是"库"的并非墓库本身，而是**保存其中的事物、物资、五行的状态**。

### 四种判断

1. **某五行只有墓气、既不透干也不露支**
   - 此时墓库是**墓**，该五行为**入墓**。
   - 命局或岁运不能开库，它将始终被困于墓中——此时墓库反而对其**起保护作用**。

2. **某五行透干或露支，且命局中无冲克**
   - 此时墓库是**库**，天干或地支通根墓库，能作为该五行的"根据地"。

3. **某五行透干或露支，但命局中有冲克它的五行**，分两种情形：
   - **克得气息奄奄、快要死了** → 逃入墓库也活不了，墓库成为**安息长眠之墓**。
   - **克得程度不大、只受伤不危及生命** → 逃入墓库可躲避、蓄势待发，墓库成为**休养生息之库**。

4. **某五行既透干又露支（可能多处），或另得令而旺**
   - 这个五行**比较旺**，命局中若有对应墓库 → 会**自动入墓库**。
   - 此时的墓库可看作**储存之库**。

### 总结
- **较旺相**的五行入的墓库 → **库**。
- **较衰弱**的五行入的墓库 → **墓**。

### 变化原则
原局中透出的天干或露出的地支很弱时，若**大运流年得到补强** → 原局中的"墓"可能变成"库"；反之亦然。**学艺一定要注意"变"**。

## 开库的三大类情况

### 一、自动开库

针对库中的**墓气透出天干或露出地支**（或既透又露）的情况。

#### 情形 1：透干露支 + 无冲克
- 墓气对应的天干、地支没遇冲克 → **不会入墓库**。
- 墓库会**源源不断为透干、露支的干支提供力量**。
- 提供能量的多少，需根据**月令旺衰**（墓气在月令的旺相休囚死）判断。

#### 情形 2：透干露支 + 遇冲克 → 自动入库 → 自动出库
- 透干露支的干支遇冲克 → 自动进入墓库躲避。
- 大运流年出现帮助己方、克制敌方的五行 → 相当于赶走了敌人 → 透干露支的干支**又会自动出库**，回到原来位置。
- 此时墓库再次源源不断地提供力量。

**形象比喻**：城门对己方人员可以随时打开。提供能量的多少综合看月令旺衰（墓气的旺相休囚死）。

### 二、强行开库（冲刑破害）

通过**地支相冲、相刑、相破、相害**开库：
- **行冲最容易开库**。
- 破、害**力量足够时**也能开库。

例：
- **丑戌相刑**：可以互相开库。
- **两库力量不等**：力量强的开力量弱的。例如辰土得令 → 辰库力量强，可以去开其他库，但其他墓库基本打不开辰库 → 必须等流年大运来丑或戌，才能打开辰库。

### 三、开天库（天干冲合引动地支）

强行开库的一种特殊情况——**天干相冲引动地支墓库开库**。

#### 规则
当命局中有某墓库，且天干同时透出该库的**墓气**（需是**与墓气完全相同的阴天干**），若天干又存在能与墓气相冲的天干 → 天干相冲会**引动地支墓库**，起到开库作用。

#### 四组天干冲开天库

| 墓库 | 墓气透干 | 相冲天干 | 天干冲 → 引动开库 |
| --- | --- | --- | --- |
| **辰库** | 癸水 | 丁 | 丁癸冲 → 开辰库 |
| **未库** | 乙木 | 辛 | 乙辛冲 → 开未库 |
| **戌库** | 丁火 | 癸 | 丁癸冲 → 开戌库 |
| **丑库** | 辛金 | 乙 | 乙辛冲 → 开丑库 |

#### 关键注意
- **天干必须是与墓气完全相同的阴天干**（如辰库必须是癸水，不能是亥水）。
- 这相当于"墓气癸水的真身来到了天干"，完全可以代表墓库——**天干的癸水受冲就等同于地支的辰库受冲**。
- 另一角度：丁火以戌土为库，**丁癸天干冲 = 地支辰戌冲**的映射。

## 应用要点

1. **自动开库**：只要需要就可开库，像城门对己方人员随时打开。
2. **强行开库**：通过冲刑破害触发，力量对比决定谁开谁的库。
3. **开天库**：天干冲引动地支墓库，要求墓气必须透出且为同阴天干。

**开库是墓库运用的核心机制**——开库后，墓库中封存的能量才能真正参与命局作用。
