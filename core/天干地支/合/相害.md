# 地支相害（六害 / 相穿）

## 定义

- **相害**：地支之间的一种相互损害关系，又称**相穿、穿**。
- 意为**极不和谐、相互损害**——是一种深刻的仇恨、难以调和的矛盾、绝对对立、根深蒂固、具有破坏性的关系。
- **古籍定位**：其威力有时甚至超过冲——应世俗猛，比刑冲更加厉害。
- **与冲的区别**：
  - **冲**：**明争**、正面冲突、撕破脸、掀桌子。
  - **穿害**：**暗箭伤人**、你争我斗的社会现象、尔虞我诈、表面和气内心充满矛盾、利益之争从未停止。

## 六害组合

共六组：

- **子未相害**
- **丑午相害**
- **寅巳相害**
- **卯辰相害**
- **申亥相害**
- **酉戌相害**

## 相害与相合、相冲的关系

相害的形成原理 = **相合被相冲破坏**。每一对相害都可由此推出：

| 相害 | 形成逻辑（合被冲破） |
| --- | --- |
| 子未 | 午未合被子午冲破；子丑合被未丑冲破 → 故子未相害 |
| 丑午 | 午未合被丑未冲破；子丑合被子午冲破 → 故丑午相害 |
| 寅巳 | 寅亥合被巳亥冲破；巳申合被寅申冲破 → 故寅巳相害 |
| 卯辰 | 卯戌合被辰戌冲破；辰酉合被卯酉冲破 → 故卯辰相害 |
| 申亥 | 巳申合被巳亥冲破；寅亥合被寅申冲破 → 故申亥相害 |
| 酉戌 | 辰酉合被辰戌冲破；卯戌合被卯酉冲破 → 故酉戌相害 |

**核心理解**：相害是"合被破坏"的结果——**本可成之合被第三方冲散**，遂结成深仇。

## 六害的实质关系与象意

### 一、子未相害（世家之害）

- **子属水、未属燥土**，原本就是土克水，再加两旺相持必有一伤。
- 一般**子水受伤**：
  - 未中己土减力。
  - 未中丁火受伤程度也很大。
- 代表癸水、丁火之类的六亲与身体部位易受损。
- 古称"**世家之害**"——**对六亲骨肉最为不利**。
- 若为贵格 → 多妻妾；若为间隔 → 孤独无依。

### 二、丑午相害（官鬼相害）

- **午中丁火生丑中己土**，午中己土并入丑中己土。
- 午中丁火本气冲克丑中癸水；己土加力克制丑中癸水；午中丁火克制丑中辛金。
- **癸水、辛金**所代表的六亲与身体部位易受损；**癸水**所代表的事项容易受阻。
- 故称"**官鬼相害**"。
- 命主旺盛时 → **易怒、缺乏忍耐力**。
- 命主弱时 → **可能出现残疾**。

### 三、寅巳相害（两强相害 + 相刑）

- **寅巳这个组合非常复杂**：既有寅生巳、又有寅刑巳、还有寅巳相害。
- 寅巳原本相刑，可以直接按相刑讨论；相害主要涉及**六亲**的影响。
- **寅是强木、巳是强火**，两强相争、相互伤害——各自逞强。
- 受伤害主要是**巳火中的庚金和戊土**。
- 庚金代表的六亲要特别注意；**重金者疾病缠身**。

### 四、申亥相害（争嗔之害 / 对婚姻最凶）

- **申是庚金的临官之地，亥是壬水的临官之地**，两者发生关系便有"争先恐后"的意味。
- 但**亥是申金的衰病之地**，亥水也代表申金的进一步发展趋势。
- 故申亥相害体现为**亥水对申金的勾引、引导、预谋**，以及**申金主动配合**——古称"**争嗔之害**"。
- 凡相害若穿害了夫妻宫或夫妻星皆对婚姻不利。
- **申亥相害对婚姻的影响最严重**，可能导致严重的克害和灾祸。

### 五、卯辰相害（欺凌之害）

- **卯木砌砖于辰林旺地**。
- 辰中乙木虽虚弱，但在相害后获得利益。
- 辰中**戊土、癸水受损**——尤其癸水受损严重：
  - 癸水本就受戊土克制。
  - 乙木增强后对癸水消耗更严重。
- 戊土受损主要因辰土是戊土的死地，无法使其旺盛。
- 故称"**欺凌之害**"——**年轻欺压年长**。
- 可能导致命主**腰脚疾病和筋骨问题**。

### 六、酉戌相害（嫉妒之害）

- **戌中戊土本可生金**，戌中辛金与酉中辛金叠加 → 辛金力量倍增。
- 然而**戌中丁火为死火**，遇到旺盛的辛金不能甘辛，嫉妒别人加害酉金。
- 故称"**嫉妒之害**"。
- 若金得令或旺相 → 遇戌土不会产生太大妨害，可不加考虑。
- 若戌得令或旺相 → 遇酉金则会有不好事情发生。
- 可能导致命主**头面生疮、面部伤疤**，严重者甚至**聋哑**。

## 六害产生的结果

### 1. 破坏关系、加剧矛盾

- 穿坏了对方但治不了对方。
- 矛盾加深，产生仇恨、互相敌对甚至强烈对抗。
- 如子未、卯辰、酉戌等力量悬殊的情况，容易穿坏。

### 2. 穿倒（穿错位）

- 意即相反 → 穿倒正印当偏印看、穿倒七杀当正官看。
- 通常在寅巳、丑午、申亥相穿或隔穿时易发生。

### 3. 羊刃逢穿害

- **其人死得快**，恐有一刀之灾。
- 死刑犯人多是羊刃逢穿害。

## 六害在八字中的应用原则

### 力量规律

- **主克的一方对被克的一方产生穿害**（如子未相害实际是未对子产生穿害），但也需考虑双方旺衰——**旺的一方对衰的一方产生穿害**。

### 六害的相互影响

- 可以一对一相害，也可以一对二、一对三甚至更多的相害。
- 六害可能表现为**隔位遥远相害**——但遥远的相害虽有害意，基本没什么实质作用。
- **近害**（距离越近）：产生的力量就大，六害的性质就越强。
- **二害一、三害一**（多数害少数）：力量显得非常强大。

### 八字本命局有六害的命主可能出现的情况

- 与**长辈、上司**沟通时可能遇到困难、感到心结和代沟；不愿主动攀援上流社会。
- **配偶**在事业上无法提供足够帮助；配偶与长辈关系不佳、难得援助；长辈对命主事业持否定态度。
- **子女**与长辈关系疏离；长辈也不愿帮忙照顾小孩。
- 对配偶关注度不高、难以维持长久关系（聚少离多、同床异梦、婚姻变故或离婚等）。
- 夫妻之间心结较多——婚前可能交往融洽，婚后却相对无言。
- 与子女沟通存在困难。
- 工作**更换频繁**，工作态度不够专注，常敷衍了事，责任感较弱。
- 配偶对命主的人际关系持怀疑态度，甚至可能产生反感、不支持命主事业。

## 六害在各宫位的应事

### 一、穿害了年支

- 年柱代表**长辈和领导**的宫位。
- 多主对长辈亲人身体健康的影响。
- 工作中可能遇到领导或长辈亲人突然有事需要你出门的情况。
- 意味着可能**无法得到祖上福荫**。
- 感情婚姻方面：长辈亲人介入可能导致感情婚姻不顺。
- 财运方面：得不到长辈贵人帮助。
- **寅巳申亥相穿** → 主要表示**走动频繁、出门离家**（寅申巳亥为驿马）。

### 二、穿害了月支

- 月柱代表**事业、父母、兄弟姐妹、朋友**的宫位。
- 工作环境频繁变动。
- 父母在工作中可能遇到意外变动或调整，导致父母因工作分合不顺。
- 感情方面：可能受父母或朋友干预，导致感情生活或婚姻出现问题。
- 也表示因兄弟姐妹关系出现问题。

### 三、穿害了日支

- 日支代表**夫妻宫**。
- 感情婚姻方面可能遇到不少麻烦——**第三者介入**、配偶容易陷入桃花纠纷、夫妻关系出现问题。
- 工作方面得不到配偶援助。
- 日支也代表一个人的思想——这种人通常**想法丰富、容易给自己制造压力**、心理状态波动较大、情绪起伏明显。
  - 日支为**喜** → 心胸宽广。
  - 日支为**忌** → 容易感到孤独。
- 暗示命主可能在**没有经过充分思考的情况下突然改变工作、追求财富**的信息。

### 四、穿害了时支

- 时柱代表**门户**（住址） → 意味着生活节奏频繁变动——经常更换居住地点或频繁出差。
- 时柱代表**子女宫** → 子女可能不太孝顺：
  - 离家远去。
  - 父母年老时可能无法给予足够关心照顾。
  - 老年人在晚年感到孤独。
- 时柱穿害者晚年可能面临挑战和困境。

### 五、月日相害

- 尤其女性更容易感到**孤独和命运多舛**。
- 当月柱与夫妻宫因相害产生不利影响时，**必须合住夫妻宫从而冲开心性**，否则：
  - 婚前难找到合适伴侣。
  - 即使结婚，婚后关系难以和谐，离婚可能性非常高。

### 六、日时相害

- 代表命主可能在**老年面临各种健康问题**。

## 六害与凶神同现的加重

- 若六害无化解、命主又遇到**羊刃、劫煞、官符**等凶神 → 很可能遭遇**重大灾祸**。
- 命理中若出现**六害无救应** → 预示一生可能**孤独无依、亲人分离、经济拮据**。
- **女性**在命理中最怕遭遇六害。

## 判断相害的要点

### 1. 力量与距离

- **近穿 > 远穿**：相邻的穿害力量最大、远穿基本无实质作用。
- **多穿 > 单穿**：一对多形式的穿害力量倍增。

### 2. 化解

- 八字组合良好、格局佳、或六害得到化解 → 伤害可减轻甚至消失。
- 反之无化解 → 穿害性质非常强烈、表现为凶险。

### 3. 相克 + 相害

- 当**相克又带相害同时存在**时最为严重。

### 4. 命宫与宫位

- 剖析六害必须**结合命宫**观察六害位于哪个宫位，再深入分析。

## 常见误区

- **"见害必大祸"**：不能单独解读六害，需结合八字整体组合和格局判断。
- **"远穿一样应"**：隔位相害基本无实质作用。
- **"穿害就是冲的弱化版"**：穿害与冲性质完全不同——冲是明争，穿是暗箭。
- **忽视"相合被冲破"的底层逻辑**：理解相害必须理解"合被冲散"这个形成机制。

## 一句话总结

- 相害是**"合被冲破"的深层仇恨** → 表面和气、暗中拆台。
- 应验要看**距离、力量、喜忌、宫位、有无救应**。
- 穿害的核心提醒是：**在被穿害的那个领域（夫妻、家庭、事业、合作、健康），天生有暗线，不适合存侥幸心理，须早看清、早处理**。
