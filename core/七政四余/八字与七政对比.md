# 八字与七政四余

## 定位：两把不同的工具
- **八字**：看**大势、格局、天赋、思维** —— 框架性的说明书。
- **七政四余**：看**应期、时机、月份级别的转折** —— 细活的刻刀。
- **一把锤子砸不了所有钉子**：
  - 八字能敲下大部分钉子（方向、格局、天赋）。
  - 七政四余更擅长做细活（哪月换线、哪月买房、哪月出事）。
- **不是谁比谁强 —— 各司其职，合参更精准**。

## 两者的设计精度差

### 八字的时间颗粒度
- 最小时间单位是**时辰**（两小时）。
- 优势：排盘简单、看框架快。
- 局限：**难以分清上半年 / 下半年**；看流月仍有限，精度有限。
- 不是缺陷，是**设计精度**决定的。

### 七政四余的时间颗粒度
- 基于**真实天文数据**，行线与特定星度结合。
- 可精确到**月份、正负几天**。
- 能够看到：
  - **何时结婚 / 生子**
  - **运势突然转变的精确节点**
  - **避险 / 出手的时间窗口**

## 八字擅长什么

### 1. 格局与天赋
- 判断"这辈子方向是什么"。
- 识别你的**天生优势**与**bug**。
- 适合的路数、禁忌的领域。

### 2. 思维模式与秉性
- 印重 → 爱学习、重体面。
- 伤官 → 聪明、不服管、想自由。
- 比劫 → 讲义气、易卷入竞争。
- 上手快，两三个月就能用得像样。

### 3. 身弱身强 / 喜忌大方向
- 整套框架明确，批盘效率高。

## 七政四余擅长什么

### 1. 应期精度
- 具体**月份甚至日期级别**的重要事件定位。
- 行线走到特定星度时，能量状态瞬间改写。

### 2. 宫限逻辑
- 古老派一句话：**宫限是框架，限度是主体**。
- **宫**：舞台（给这段时间划定场景）。
- **限度**：主角（到哪颗星、顶哪颗星决定具体戏码）。
- 顶到木星 → 一种感应；顶到土星 → 另一种感应 —— 同一大限宫里应期和事件性质已被决定。

### 3. 看事件性质细分
- 八字看不清的"做什么行业的技术"，七政的火星特征可能指向：
  - 文化 / 电器 / 互联网 / 电子等。
- 细化方向的能力是八字弱项。

## 合参方法论

### 何时换线（关键节点判断）
- 八字大运**固定十年一换**。
- 七政的大限**节点不同**，可能与八字大运错开。
- 一个人生的重要转折，**先看哪一方先换线**，再结合另一方做确认。

### 先后顺序建议

#### 用八字定大势
- 先排八字：判断格局、身强身弱、喜忌、整体人生方向。
- 决策层面的"要不要、方向对不对"由八字回答。

#### 用七政定应期
- 再看七政：判断上半年还是下半年、哪个月换工作、哪几天避险。
- 时机层面的"什么时候、怎么做"由七政回答。

## 案例画像一：事业应期无法落地到月时
- 丁火生亥月，身弱财官两旺（食伤生财 → 财生官），典型"富屋贫人"模式。
- 虽地支有胃土、戌土之根、亥水藏甲木之印，仍属身弱用木火格局。
- 提出的问题：上半年结婚还是下半年？
- **八字局限**：能看到"这一年整体运势不错，但未戌相刑会很难受"，但上下半年切分不清。
- **七政看法**：
  - 行线仍在深宫闭月度，三方见不到火罗调猴。
  - 可以预判：前段时间（火星加持期）事业顺、花钱多；火星一过运势下滑。
  - 应期可精确到**哪几个月必出事**。
- 实际落地：上半年筹备婚礼/蜜月；下半年被辞退 → 完全符合七政判断。
- **启示**：八字难判上下半年、七政能精确到月份。

## 案例画像二：职业方向的细分识别
- 壬水生卯月，食伤局，伤官见官、难进体制；身弱用丑土（官）。
- 八字能断："不走体制、大概率做技术。"
- 但"做什么技术" —— 八字没给清楚。
- **七政补充**：行线受火星特征影响 → 文化 / 电器 / 互联网 / 电子。
- 实际反馈：电脑城做电脑生意、电子产品。
- **启示**：行业细分方向上，七政颗粒度更细。

## 案例画像三：关键节点的先后判断
- 八字大运 1996 / 2006 / 2016 ……一条线。
- 七政大限转换节点与八字不同 —— 可能在某些年份**先于**八字一步换气。
- 当事人实际感受到的"运势真正转变点" —— 往往是两门术数**共同勾动**的那一年。
- 对应实操：回看过去 → 哪个术数先判出转折？那就是那一次真正的分水岭。

## 使用建议

### 刚入门
- 先学八字打基础：格局、喜忌、思维模式、身强弱、大运流年 —— 足够你回答"我是什么样的人、这辈子的大方向"。

### 有具体决策
- 涉及**哪月行动、哪段时间避险、何时结婚 / 买房 / 换工作** —— 加上七政四余合参。

### 看"普通人怎么把日子过好"
- 八字告诉你**天赋与 bug**。
- 七政告诉你**什么时候出手、什么时候保守**。
- 真正重要的人生节点不过那么几个 —— 两门合参，把这几个节点**看清楚、做对选择**，就足以改变人生轨迹。

## 一句话总结
- **八字定大势、七政定应期**。
- 用八字回答"**要不要、方向对不对**"。
- 用七政回答"**什么时候、怎么做**"。
- 合参 = 既看清大局，又精确时机 —— 把有限的能量押在**对的那几个节点**上。
