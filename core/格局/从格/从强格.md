# 从强格

## 定义

- **从强格**：命局中**印星和比劫共同极旺**，且**印星力量大于比劫**（或印星为主、比劫为辅），全局**无食伤、财星、官杀等一党**，日主顺从这股"印比一片"的整体旺势。
- 若天干出现一位虚浮无根的一党 → 即为**假从强格**（甲从强格）。
- 属从格家族中与"从弱"（从财、从杀、从儿、从势）**相反方向**的一类——身极强的反从。

---

## 一、核心：印比一片 → 顺身

### 1.1 为什么要"顺"身？

- 正格思路：身旺则克泄耗——用官杀制、用食伤泄、用财耗。
- 从强思路：**印比旺到极致、盘中皆是印比**时，用克泄耗反而激怒旺神 → 不如**顺其旺势**。
- 从强格之"顺"：顺印比之流，让日主在印比加持下自然昂扬。

### 1.2 从强格的本质
- **印 + 比联合独大**：两股生身力量合流 → 日主如山岳般沉稳坚实。
- 顺从对象不是"某一股"而是"印比的整体旺势"。
- 性格上比专旺格**更稳重、更有谋略**——因为多了印星的沉淀，不像纯比劫那样冲动。

---

## 二、从强 vs 专旺格

| 对比项 | 从强格 | 专旺格（曲直、炎上、稼穑、从革、润下） |
|---|---|---|
| 主导 | **印星为主、比劫为辅** | **比劫为主、印星为辅** |
| 核心 | 印比联合独旺 | 日主五行一气独旺 |
| 气质 | 稳重、有谋略 | 冲动、直接 |
| 成格难度 | 严格依月令、印比数量关系 | 需月令与日主五行一致 |
| 古籍 | 《滴天髓·从强》 | 《渊海子平·杂格》 |
| 命名 | 以"强"为纲（日主被多层加持而强） | 以"五行"为名（曲直 = 木、炎上 = 火等） |

### 关键区分
- **月令是比劫**时，印星**必须大于比劫** → 才成从强；否则为专旺。
- **月令是印星**时，印星**数量等于或大于比劫**即可 → 成从强。

### 两者都是"身极强"的外格，差别在气势的构成：
- 专旺 = 一气成象、五行一致。
- 从强 = 印 + 比分担旺势，印星当主。

---

## 三、成立条件（严格依《滴天髓·从强》）

### 条件 1：全局皆为印星和比劫

- 天干地支中**没有食伤、财星、官杀**任何一党。
- 若天干出现一位**虚浮无根的一党** → 是假从强格。
- 地支绝不能出现非印比的本气根。

### 条件 2：月令为印星或比劫

- 月令是这股旺势的源头。
- 月令若为食伤 / 财 / 官杀 → 从强不成立。

### 条件 3：印星力量大于比劫（或至少不小于）

- **月令为比劫时**：**印星数量必须大于比劫数量**。
  - 否则印星力量弱于比劫 → 变成专旺格。
- **月令为印星时**：**印星数量等于或大于比劫数量**。
  - 月令印星加持下，印比数量相当时印星自然占优。

### 条件 4（若假从强）：天干可有一位虚浮一党

- 这一位虚浮一党必须：
  - 天干透出；
  - 地支无本气根；
  - 被旺势压制（如坐在克方之上、被干头克掉等）。
- 其余五行仍须全是印比。

### 条件 5：古籍出处

- 《滴天髓·从强》："**从强者，四柱印绶重重，比劫叠叠，日主又当令**。"
- 《滴天髓·从强》（任铁樵注）："**须行比劫印绶则吉，如行官杀运则触其怒，逆其旺神必凶；行财运则群比劫争之，九死一生；行食伤运，泄其太过，亦不吉**。"

## 四、取用（顺印比）

### 4.1 喜用神
- **印星、比劫**——顺势加强，为基础喜用。
- **官杀亦可为喜**（特殊情况）：
  - 印星极旺时，官杀来生印 → 官杀成了印星的源头 → 也可为喜。
  - 但官杀不能太旺 → 太旺会引发战意。
  - 此条针对**真从强**中"印星极重者"。

### 4.2 忌神
- **食伤**：泄印比旺气 → 破格。
- **财星**：克印 + 被比劫争夺 → 双重破格。
  - **财破印的后果**：事业受阻、破财、生病、甚至危及生命。

### 4.3 大运
| 大运 | 效果 |
|---|---|
| 印运、比劫运 | **大喜** → 事业顺利、贵人相助、身体健康 |
| 官杀运（配合印极旺时） | 喜（生印）；但过重则凶 |
| **食伤运** | **忌** → 泄旺气 |
| **财运** | **大忌** → 破印、群比争财 → 重大事件 |

### 4.4 假从强的特别注意
- 假从强天干有一位虚浮一党。
- **大运流年若加强这个一党** → 虚浮者被激活得根 → 引发破格。
- 对这位一党所属五行的运要**特别小心**。

## 五、现实表现

### 5.1 性格
- **为人稳重**：不轻举妄动、有定力。
- **身体强健**：印比多 = 身心俱旺。
- **个性刚毅**：认定的事不回头。
- **有谋略**：印多 = 思考深、不像专旺格那么冲动。
- **气场沉厚**：给人"靠得住"的感觉。

### 5.2 事业路径
- **学术、研究、文化**（印星主文）：学者、宗师、修行者。
- **继承、守成、平台型岗位**：家族企业继承人、平台高管。
- **宗教、命理、心灵导师**：印 + 比的组合天然有"玄"的气场。
- **专业权威**：靠学识与资历立身。

### 5.3 案例口吻（依古籍常识）
- **真从强**：四柱土金（戊己土日主，印 = 火、比 = 土 → 或印 = 土、比 = 金）纯净一气 → 为人深沉、有道行、画符观气皆通。
- **假从强（天干一位庚金食神坐午火被截）**：局中印比极旺 → 其人稳重刚毅，但走金水年（食伤被激活、财运到来）必有病灾、破财。

### 5.4 破格的六亲提示
- 从强格遇**食伤运** → 子女可能有灾（食伤代表子女，逆格局气势者易受冲击）。
- 从强格遇**财运** → 父亲、妻子可能出事（财代表父、妻）。
- 从强格遇**官杀运过重** → 事业地位震荡。

## 六、常见误区

### 1. 把从强当专旺
- 从强 = 印比并重、**印为主**。
- 专旺 = 日主五行一气、**比为主**。
- 月令是比劫时印不大于比 → 是专旺，不是从强。

### 2. "印比多就是从强"
- 还必须**全局无一党**（或仅一位虚浮一党）。
- 原局出现财 / 官 / 食伤任一位有根 → 破格，不是从强。

### 3. 以为从强格一定一生平顺
- 从强格的贵在"稳"，不在"顺"。
- 遇忌神运（财、食伤过重）仍可能破格。
- 假从强对忌神运更敏感。

### 4. 忘记印比之间的数量关系
- 月令为比劫时 → 印必须大于比（否则专旺）。
- 这是判定的关键一票。

### 5. 把"印多"误认为"笨"
- 印旺者外表憨厚、内心实则**极聪明**、有灵性。
- 观人勿以外表论，从强格常是大智若愚型。

## 七、古籍背书

- 《滴天髓·从强》："**从强者，四柱印绶重重，比劫叠叠，日主又当令——须行比劫印绶则吉**。"
- 《滴天髓·从强》（任铁樵注）："**如行官杀运则触其怒，逆其旺神必凶；行财运则群比劫争之，九死一生**。"
- "**格局选对用神自出**。"

---

## 八、一句话总结

- 从强格 = **印比一片，顺身而行**。
- 与专旺格的差别：从强**以印为主**，专旺**以比为主**；从强更稳重，专旺更直接。
- 喜印比、不忌官杀（印旺能化）、**大忌财星**（财破印时九死一生）。
- 一旦成格，**不能用普通五行断吉凶** —— 须顺格局气势而论。
