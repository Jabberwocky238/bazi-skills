# 食神生财

## 成立条件（严格依《子平真诠》《滴天髓》）

### 条件 1：食神透干且通根（或月令本气为食神）

- **古籍出处**：
  - 《子平真诠·论食神》："**食神必得生旺之地，方能吐秀生财**。"
- **客观判据**：
  - ① 食神透于月 / 日支 / 时干，且地支有本气或中气根至少一位；或
  - ② 月令地支本气为食神。
- **不满足则破**：食神虚浮无根 → 无生财之源。

### 条件 2：财星透干且通根

- **古籍出处**：
  - 《子平真诠·论食神》："食神生财，**财要有根**——无根则秀而不实。"
- **客观判据**：
  - 财星（正财 / 偏财皆可）透于天干，且地支有本气或中气根至少一位。
- **不满足则破**：财虚透无根 → 才华无处兑现，止于"吐秀"。

### 条件 3：食神与财星位置相连（食 → 财 无阻隔顺流）

- **古籍出处**：
  - 《滴天髓·通神论》："用神要紧贴，格局要通气。"
  - 《子平真诠》："生气相贴为妙，隔则气泄。"
- **客观判据**：
  - ① 食神与财星分居相邻两柱，或同柱之干与支；
  - ② 食 → 财之间无印星阻断（印克食则链断）、无比劫夺财。
- **不满足则破**：食神与财被印 / 比劫隔开 → 气不连贯 → 生财链断。

### 条件 4：日主身不过弱（能承载食神 + 财的双重泄耗）

- **古籍出处**（本条核心）：
  - 《子平真诠·论食神》："**身强食神旺，胜似财官**。"
  - 《子平真诠·论食神》："身强能任食神、能任财；身弱则食神反为耗。"
- **客观判据**：
  - 日主于月令有比劫 / 印星余气以上，或他柱至少两位比劫 / 印星通根（身中偏强至身旺）。
- **不满足则破**：身极弱 + 食神旺 + 财旺 → 转入"身弱食伤多"或"财多身弱"破局；若日主完全无根可考虑从儿格（见 `从儿格.md`）。

### 条件 5：原局无偏印（枭神）紧贴夺食

- **古籍出处**（本条铁律）：
  - 《子平真诠·论食神》："**食神最畏枭神**——枭印夺食，福去祸至。"
- **客观判据**：
  - ① 原局无偏印与食神紧贴相克；或
  - ② 有偏印但有财星紧贴克之（**财护食**救应）——此时财既护食又被食生，双重功用。
- **不满足则破**：枭神夺食无救 → 食神通道被堵 → 生财链源头断，转入"枭神夺食"破格讨论。

### 条件 6：原局无比劫紧贴夺财（对财端的保护）

- **古籍出处**：
  - 《子平真诠·论财》："**财怕比劫之夺**。"
- **客观判据**：
  - 天干无比劫与财星紧贴（月 / 日、日 / 时邻位）直接相克；
  - 若有比劫，须有官杀制比劫救财，或食伤通关化比劫。
- **不满足则破**：比劫贴财无救 → 挣到的钱被合伙人 / 家人 / 同行夺走。

---

**六条同时满足**方为真食神生财。本格之贵正在"**食旺 + 财旺 + 身能任 + 通路无阻**"四全——才华不仅能外显（吐秀），还能真正落到现实收益。
