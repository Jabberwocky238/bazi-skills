# 身强弱

## 本目录概要

**身强弱**是命理判断的**起点**——日元旺衰是八字分析的第一道关口。旺衰定则喜忌随之定，喜忌定则用神、格局、流年推演方有依托。

本目录将"身强 / 身弱 / 身旺 / 从格"这一整组概念做系统梳理，把传统命理中最常混用的"**身强**"与"**身旺**"严格区分开来，并给出完整的判断流程。

### 阅读顺序建议

1. 先读 **身强身旺辨析.md**——把最容易混淆的概念先拎清楚。
2. 再读 **身强.md** 与 **身弱.md**——分别掌握两极的特征与喜忌。
3. 最后读 **旺衰判断流程.md**——拿到八字能上手操作。

## 本目录的定位

- **概念层**：区分身强 / 身旺 / 身弱 / 从格。
- **判据层**：得令 / 得地 / 得势 / 综合加权 / 等级划分。
- **应用层**：喜忌、出路、误区、古籍印证。

学完本目录，即可对任何一个八字完成**旺衰定位**，进而进入喜用神取舍、格局判断、岁运推演的下一阶段。
