# 木火相煎

## 定义
- **木火相煎**：命局**火气过旺 + 木被火烧 + 无水润** → 木不是生火而是被焚 → **才华 / 生机被自己点燃的火吞噬**。
- 别名：**火多木焚**、**火炎木焚**。
- 属**五行失衡病象**之一。
- 是"**木火通明**"的**失败版** —— 同样木火相连，有水润则通明，无水润则相煎。

## 核心：母生孩子，反被孩子烧空
- 木生火为吉，但**火过旺反过来烧木**。
- 适度火 + 木 = 光明通透（通明）。
- 过度火 + 弱木 = 燃烧殆尽（相煎）。
- 关键不是"火多"，是**火过旺 + 木根虚 + 无水润**。
- 所谓"**煎**"：木被火不断煎熬 → 逐渐枯焦、最后成炭。

## 与相关结构对比

| 结构 | 木与火比例 | 水的救应 | 结果 |
|---|---|---|---|
| **木火通明** | 木火相当 + 木有根 | 有水或土泄 | 聪明外显（吉） |
| **木火相煎**（本条） | 火过旺 + 木不足 | 无水 | 焚毁（病） |
| **火多木焚** | 火极旺 + 木孤 | 无救 | 彻底毁木（重病） |
| **木多火塞** | 木过多 + 火弱 | — | 母太多压死孩子（相反病象） |

**木火通明 → 木火相煎 → 火多木焚** 是**程度递进**：
- 通明：木火**共荣**。
- 相煎：木被**煎熬**，尚有残存。
- 木焚：木已**成灰**。

## 成立条件
1. **火气过旺**：
   - 月令为火（巳午未）。
   - 地支合火（寅午戌 / 巳午未）。
   - 天干丙丁重透。
2. **木弱 / 根虚**：
   - 甲乙透干无根 或 根被冲合。
   - 日主若为木 → 虚浮。
3. **无水润木**：
   - 无壬癸透 / 无亥子根。
   - 或水被火蒸干、被土克。
4. **无湿土缓冲**：
   - 辰丑湿土无 → 火土俱燥。
5. **无金制火**：
   - 金无 / 金被火先熔。

## 判断维度

### 1. 甲木 vs 乙木

| 日主 | 木火相煎的表现 |
|---|---|
| **甲木** | 大树被烈火烧 → 表面挺立、内里已空 → 外强中干 |
| **乙木** | 花草被烈日烤 → 枯萎凋零 → 才华 / 情绪快速耗尽 |

### 2. 火的类型
- **丙火**过旺：明炽 → 木烧得快且外显 → 才华迸发式过劳。
- **丁火**过旺：精火 → 木细煎慢熬 → 才华内耗式枯竭。
- **丙丁并透**：双重煎烤 → 最严重。

### 3. 水的残存
- **水全无**：煎极 → 木根被烤枯 → 身心同焚。
- **水虚透无根**：象征性救应 → 症状时轻时重。
- **水藏未发**：尚有一线 → 大运见水即可激活。

### 4. 与"木火通明"的分界
- 木火通明 = **通 + 明** → 木能生火、火不焚木。
- 木火相煎 = **煎 + 焚** → 火过旺反焚木。
- 判定标准：**木有无根 + 水有无调候**。

## 常见误区

### 1. "木火旺 = 木火通明"
- 通明需**木火势均 + 有水润 / 有金制**。
- 少一个节制 → 立刻转相煎。
- "差一步"是极常见的误判。

### 2. 把"相煎"等同"有才华"
- 通明的才华是**可持续的光**。
- 相煎的才华是**一次性的灰烬**：爆发快、熄灭也快。
- 看似聪明，实则**自己烧自己**。

### 3. 单纯补木
- 补木 = 加柴继续烧。
- 必须**先灭火 / 先润局 / 先护根**。
- 顺序：**水克火 → 再护木根**。

### 4. 忽视"才华型过劳"风险
- 木火相煎的人常以才华立身。
- 但容易把才华当无限资源使用 → **创作力耗竭、精神崩溃、中年塌方**。
- 是**才华型人格**里最危险的画像。

## 现实表现

### 性格
- **表面热情、实则易耗**：迅速投入、迅速燃尽。
- **创作力旺盛但难持续**：高产爆发 + 长期枯竭交替。
- **易激动 + 易崩溃**：情绪管理能力被火耗空。
- **思维快但浅**：想法如烟火，来得快散得快。
- **自我怀疑重**：才华烧空之后对自己产生强烈否定。

### 事业路径
- **内容 / 创作 / 艺术型工作者**的典型困境：
  - 年轻时才华外显 → 成名快。
  - 中后期创作力枯竭 → 江郎才尽。
- **表达 / 表演类岗位**：舞台上光芒四射，台下体力精神透支。
- **咨询 / 顾问**：高智力输出 → 高精神消耗。
- 若走水运 → 是"**润泉**"期，创作力可回春。

### 身体
- 木主肝胆、筋、情志、神经。
- 火主心、血脉、眼、神明。
- 木火相煎的典型症状：
  - **慢性疲劳综合征**：燃尽状态的身体表达。
  - **肝火旺 + 心火旺**：两头冒火。
  - **失眠 / 噩梦**：神被火扰。
  - **偏头痛**：火上冲。
  - **眼疲劳 / 目赤**：火灼精明。
  - **情绪障碍**：抑郁 / 躁郁交替。

### 感情
- **感情如火**：炽烈开始，迅速消耗。
- **关系易燃易熄**：来得凶、散得快。
- **理想化对方**：初期投射过多，后期失望强烈。
- 走水运 → 感情可进入平稳期。

## 化解思路

### 结构层
- **补水为最优**：
  - 壬癸透 → 润木 + 灭火过旺 → 全面调候。
  - 大运水地 → 才华型回春窗口期。
- **补湿土次之**：
  - 辰丑 → 吸火 + 润底 → 间接救木。
- **避免补火 / 燥土**：
  - 火加剧。
  - 戌未 → 火上添燥，大忌。
- **金制火需谨慎**：
  - 金先被火熔，难以真正制火。
  - 除非金水并见，否则金单行易成"火多金熔"+"木火相煎"双重病象。

### 生活层
- **节奏调候**：
  - 不追求持续高产 → **创作节律要有周期**（工作期 + 休整期）。
  - 避免长期熬夜、通宵赶工。
  - 学会"**适可而止**"。
- **环境调候**：
  - 多接触水、阴凉、绿植。
  - 避免高热、高光、高强度刺激环境。
- **饮食调候**：
  - 润肝养心食物：枸杞、菊花、莲子、百合、桑葚。
  - 避免辛辣、咖啡、酒。
- **情志调候**：
  - 建立**情绪泄压机制**：静坐、冥想、慢运动、写作（非创作型）。
  - 主动**断开过度刺激**：减少短视频、社交媒体等快节奏刺激。

### 认知层
- 意识到"**才华是池塘里的鱼，不是池塘本身**"。
- 要养池（根基、休整、润泽），而不是一直捞鱼。
- 相煎格的人**活到 40 岁仍在创作**才算真赢 —— 持久比爆发更重要。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 水运（尤壬） | **大喜** → 润木灭火 → 才华回春 |
| 湿土运（辰丑） | **喜** → 吸火润底 |
| 金运（配水） | **喜** → 制火 + 助水源 |
| 金运（无水配合） | 忌 → 先熔金再焚木，双病象 |
| 木运 | 视结构 → 有水则通；无水则加料继续烧 |
| 燥土运（戌未） | **忌** → 燥上加燥 |
| **火运** | **大忌** → 焚上加焚 → 燃尽期 |

## 与相关格局的对照

| 格局 | 火在结构中的作恶对象 | 结果 |
|---|---|---|
| **木火通明**（吉） | 木火通顺 | 聪明外显 |
| **木火相煎**（本条） | 木被焚 | 才华耗尽 |
| **火多金熔** | 金被熔 | 骨气毁 |
| **火炎土燥** | 土被烤 | 生机失 |
| **火多蒸水** | 水被蒸 | 智气涸 |

**火过旺的系列病象**：
- 焚木：木火相煎（焚**生机 / 才华**）。
- 熔金：火多金熔（熔**骨气 / 决断**）。
- 燥土：火炎土燥（烤**根基 / 生养**）。
- 蒸水：火多蒸水（涸**智慧 / 流动**）。

## 核心判断
- 木火相煎不是"木火都旺"，是"**火过旺 + 木虚根 + 无水润**"。
- 判断重点：
  - **木有无根**（寅卯 / 长生）。
  - **水有无救应**（壬癸 + 亥子）。
  - **火有无节制**（金 / 土）。
- 若是**短期木火相煎**（大运流年引起）→ 熬过去 + 主动调整节律。
- 若是**原局结构性木火相煎** → 需**终身配合节律化生活方式**，并善用水运的每一次窗口期。

## 一句话总结
- 太阳太强的日子，森林会自己烧起来。
- 木火相煎 = **有才、有光、有热情，但不懂节制时会把自己烧成灰**。
- 救法不是再点一把火，是**给自己下一场雨、留一段空**。
