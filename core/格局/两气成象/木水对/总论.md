# 木水对（总论）

## 性质：相生两气对（水→木）
- **五行关系**：水生木（印→身 或 身→食伤的顺流）。
- **属"印生两神成象格"之一**——《滴天髓》五种相生组合中的"水木相生格"。
- 相生两气双清，若气纯无杂，属贵格。

## 两气成象通则（适用于本对）

### 成立条件（严格依《滴天髓》两气双清论）
1. **命局"两气双清"**——只见水木两种五行。
2. **水生木**——水为母、木为子，顺流无克。
3. **原局无财官**（土为财、金为官）——一见财官则破局。
4. **日主属水或木**，且不过弱。

### 具体形式（水木相生格）
- **壬癸水日主 + 甲乙木食伤**：身→食伤顺流（水木清华）。
- **甲乙木日主 + 壬癸水印绶**：印→身顺流（水生木润，若水过则木漂）。

### 气质
- **才华 + 生发** —— 创意、敏锐、有学问、善开拓。
- 典型特征：**清秀灵动** —— 脑子好使、反应快。

### 典型职业
- 学者、文化、教育、研究、出版、设计、策划、研发、创意型产业。

### 禀赋与缺失
- **禀赋**：才华、智慧、创造、生发。
- **缺失**：
  - 缺火 → 缺落地、缺行动力、想得多做得少。
  - 缺金 → 缺约束、缺决断、易散漫。

### 贵格 vs 病象
- **水木清华**（贵格）：水木双清、气机流畅 → 清秀型学者。
- **水冷木寒**（病象）：水木过旺无火调候 → 生机不显、抑郁倾向。
- **水多木漂**（病象）：水过旺而木失根 → 漂泊不定。

### 喜忌
- **喜**：水（加强本气）、木（加强本气）、**火**（通关泄秀，水→木→火成生生不息、**尤其调候**）。
- **忌**：
  - **金**（克木破局）——大忌，金透克木格毁。
  - **土**（克水破局）——大忌。
- **调候之需**：本对多偏寒（水旺），**微火调候必需**——无火则"水冷木寒"。

### 古籍背书
- 《滴天髓·源流》："**两气双清，其象最妙**。"
- 《穷通宝鉴》："壬癸见甲乙，**水木清华**。"
- 《滴天髓》："**水木相生，秀气内含**。"

- **水冷木寒**：水木过寒无火的病象。详见 `水冷木寒/`。
- **水多木漂**：水过旺木失根的病象。详见 `水多木漂/`。

## 一句话总结
- 木水对 = **水生木之相生双清**——才华、敏锐、有学问。
- 贵在"**水养木而木得秀**" → 水木清华；忌"**水过旺木失根**" → 水多木漂。
- 关键看**水木比例 + 火的调候**：水适度生木 + 微火调候 = 贵；水过旺或无火 = 病。
