# 出库

## 哪些地支可以入墓库

### 基本规则
在十二地支中，除了辰戌丑未四墓库土外，还有：
- **四桃花**：子、午、卯、酉。
- **四驿马**：寅、申、巳、亥。

**四桃花不能进入相应墓库**——因为它们与四墓库分别是**三合半合**的关系（相合）→ 一碰面就闭库。

**四驿马可以进入相应墓库**——它们是"经常在外奔波的人，很辛苦，需要经常回到基地休息"。

### 具体对应

| 墓库 | 可入库（驿马） | 不能入库（桃花，半合关系） |
| --- | --- | --- |
| **辰**（水库） | 亥 | 子（子辰半合） |
| **戌**（火库） | 巳 | 午（午戌半合） |
| **丑**（金库） | 申 | 酉（酉丑半合） |
| **未**（木库） | 寅 | 卯（卯未半合） |

### 另一种视角：辰为万物之库
有一种看法认为辰为万物之库——除了酉、戌外，巳午寅卯亥子申丑未等地支**都能入辰库**（因为辰戌相冲、辰酉相合，所以酉戌无法收入辰库）。

### 天干的情形
天干都是由地支透出——若存在墓库，他们都是从墓库中透出，所以**不存在不能回归墓库的情况** → 天干都可以回到墓库之中。

## 岁运带来的墓库
即使本命局没有墓库，每三年会有一个墓库流年——随岁运运转会出现入库、出库、开库、闭库的各种情况，参照前述规则应用即可。

## 出墓库的两大判断原则

### 一、分辨墓库中的五行是旺是衰

例：
- **月令为未月** → 辰库里的水基本干涸，这点水出不出来意义不大。
- **月令为亥子月** → 辰库里的水很旺，水出不出来对命局影响很大，需慎重考虑。

### 二、分辨墓库中的五行是喜用还是忌仇

**总体原则**：喜用神出墓库为吉，忌仇神出墓库为凶。

**详细判断**（需结合命局大环境）：

| 墓库内 | 墓库外大环境（四柱及岁运） | 结果 |
| --- | --- | --- |
| **喜用神** | 良好、生扶喜用神 | 喜用神溢出墓库 |
| **喜用神** | 不佳、克制喜用神 | 喜用神移藏于墓库 |
| **忌仇神** | 良好、克制忌仇神 | 忌仇神移出墓库 |
| **忌仇神** | 不佳、生扶忌仇神 | 忌仇神移藏于墓库（降低危害） |

## 入墓库的吉凶（五大判断点）

### 一、入墓还是入库

#### 入墓（大凶）
- **喜神入墓** → 失去力量，必定大凶。
- **忌神入墓** → 力量尽失，对日主是好事，但**对相应六亲不利**：
  - **印入墓** → 母亲身体不好或早逝。
  - **官星入墓** → 老公不成器，很难结婚、离异、多婚。
  - **财星入墓** → 父亲、妻子身体不好，很难结婚、离异、多婚。
  - **食伤入墓** → 子女不成器、夫妻少。
  - **比劫入墓** → 兄弟姐妹没什么成就。

**婚姻特别提醒**：
- **男命财星入墓、女命官星入墓** → 一般很难结婚。
- 很多时候岁运冲开墓库会结婚，但**岁运一过可能又离异**（非常常见）。

**事业提醒**：
- 命局中**官星入墓**（无论喜忌）→ 事业不会太好。
- 若无好大运配合 → 一辈子可能只是普通人。
- 反之，若命局中**无官星** → 反不会出现此问题，命局其他组合好则事业仍可发展。

#### 入库（尚有生机）
- **喜神入库** → 力量有残余 → **杯中有喜**。
- **忌神入库** → 力量有残余 → **喜中带忧**。

例：
- 自带**财库** → 即使某时段会亏损，不会亏得一无所有，总留翻本的本钱。
- 自带**官库** → 事业遇挫折但不会一败涂地，容易东山再起。

### 二、喜用神的轻重缓急不同

- **用神入墓库** → 难成格局，对自己极为不利。
- **喜神入墓库** → 可能只是对六亲不利或降低格局层次。
- **忌神中又分忌神和仇神** → 一般忌神入墓库比仇神入墓库更好。

### 三、旺弱入墓库结果不同

- **喜用神旺而入墓库** → 凶多灾众。
- **喜用神弱而入墓库** → 灾轻。

**原因**：五行旺而入墓库是**不正常的现象**，违背喜神强旺的特性。就像：
- 把一个想干事业的人关起来不让做事 → 可能比让他去死还难受。
- 把一个懒散得过且过的人关起来 → 他可能随遇而安。

古诀：**旺官旺印与旺财入墓生灾；官清印清财清入墓无妨**。

### 四、过旺入库反是好事

一个五行**因太旺而入库** → 一般是好事：
- 从病药角度：这五行太旺，可能是命局病根。
- 它能收敛一些能量入库 → **对整个命局平衡有帮助**。

**示例**：
- 某八字阳刃当令，金特别旺盛——金为命局病根。
- 命局中存在丑库 → 旺而入库，部分能量入库 → **对八字平衡有很大帮助**。
- 但若岁运冲开比劫库 → 可能就是**破财的时候**。

## 什么情况下喜欢出库

综合以上原则：

| 情形 | 是否喜欢出库 |
| --- | --- |
| 命局大环境生扶**喜用神**，喜用神藏于墓库 | **喜欢出库**——能为命主效力 |
| 命局大环境克制**忌仇神**，忌仇神藏于墓库 | **喜欢出库**——出来即被制服 |
| 命局大环境克制**喜用神**，喜用神藏于墓库 | **不喜出库**——出来即受伤 |
| 命局大环境生扶**忌仇神**，忌仇神藏于墓库 | **不喜出库**——出来助纣为虐 |
| **病神过旺**，有墓库封存 | **不喜出库**——封存更利命局平衡 |

## 核心提示

- **入墓的五行（无论喜忌）代表**：这个五行相关的**六亲可能有灾难、缘分浅、对应事业/婚姻/学业得不到发展**。
- **入库的五行代表**：这个五行相关的东西暂时无法得到，需**冲开库后**才会对命主产生帮助或危害。
- 具体冲开库后是喜是忌，要结合**冲开之后的整体作用**判断。

出墓库的关键在于：**先看大环境，再看旺衰，最后看喜忌**——三者配合，才能判断一个墓库的开合是福是祸。
