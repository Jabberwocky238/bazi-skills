## 天干争合

## 定义

- **争合**：天干五合之中，**两个相同的天干同时合向另一个天干**，形成"两字争一合"的局面。
- 本质是**异性相合**中的**多方抢占**：一个天干成为被争夺对象，另外两个同五行天干争做它的"合伴"。
- 象意通俗比喻："两个男人追同一个女人"、"两个女人爱上同一个男人"。
- 争合**必然削弱合的纯度**，亦影响合化是否成立。

## 形成条件

争合需同时满足：

1. 四柱天干中出现**两个相同**的天干（同字）。
2. 另有一个天干与它们构成**天干五合**关系（合的另一端）。
3. 三字**两两相邻或在可作用的距离**内，共同构成合的牵拉。

例：

- **两乙争合一庚**：年干乙、日干乙、月干庚，两乙共争庚之合。
- **两庚争合一乙**：两个庚金同时向一个乙木发动合力。
- **两丁争合一壬** / **两壬争合一丁**：丁壬合的争合形态。

## 典型争合组合对照表

以天干五合为基础，每一组五合都可衍生出**两种争合方向**（看哪一干被重复）：

| 基础五合 | 争合形态一 | 争合形态二 |
| --- | --- | --- |
| 甲己合化土 | 两甲争一己 | 两己争一甲 |
| 乙庚合化金 | 两乙争一庚 | 两庚争一乙 |
| 丙辛合化水 | 两丙争一辛 | 两辛争一丙 |
| 丁壬合化木 | 两丁争一壬 | 两壬争一丁 |
| 戊癸合化火 | 两戊争一癸 | 两癸争一戊 |

## 争合的五种影响

### 一、力量减弱，合而不牢

- 合的力量被两方分摊，原本纯粹的吸引变成**三角拉扯**。
- 合意存在但**关系不稳**，常呈拉锯与摇摆。
- 表现：事情**有苗头无结果**、关系**似合非合**。

### 二、夫妻宫争合 → 感情争夺

- **女命夫星被争合**：容易成为填房、遇二婚男，或**介入他人婚姻成为第三者**。
- **男命妻星被争合**：妻子可能另有情感牵连，或娶到已有婚姻经历的女子。
- 婚恋之路多见**同时被两人追求**、**感情三角**的现实戏码。
- 命主常被八字推向**"被争抢"的位置**——表面受宠，实则易陷入感情纠葛。

### 三、事业宫争合 → 合作拆散

- 日干两侧出现同字合日主或合同一合神，象征**两个合作方同时争主导权**。
- 结果是合作关系**难以持久**：开头热络、中期角力、末期拆伙。
- 尤其忌讳在公司股权、合伙项目中出现此格，常见**合伙拆股、分道扬镳**。

### 四、财星争合 → 财来财去

- 男命两正财、女命两偏财被争合，**财星虽多却难聚**。
- 象意：**进财快、出财更快**，财气在拉扯中流失。
- 若日主本身旺、驾驭得住两财，反而代表**能力强、事业旺**——但需配合全局判断。

### 五、官星争合 → 地位不稳

- 官星被争合 → 权位来源有竞争者，**升迁路上多变数**。
- 易有**小报告、被人争抢头衔、地位不牢**之象。
- 女命尤以夫星争合最敏感，对应感情与名分的双重不稳。

## 合化成立条件

- **争合一般不化**：三字争一的拉锯，本身就破坏了合化所需的"专一"条件。
- 合化成立本需合的两干紧邻、月令化神得气；争合之下**合的专注性被瓦解**，即使化神在地支也多半化不成。
- 例外：整体地支化神**极旺**、争合一端被其他字克制或冲开，剩下的"一对一"方有机会成化——此时另一端沦为**闲合**。

## 贪合忘克

争合格局中，**贪合忘克**现象尤为突出：

- **原理**：两干贪于合同一对象，容易忘记自己原本应当克制的对象。
- 例：两庚争一乙，庚本要克甲木，却因专注争乙而**放松对甲的克制**。
- 引申：
  - 原局中被克之字**暂时得到喘息**。
  - 日主若是被克之字，争合反成一种**保护机制**。
  - 但克力只是**被分散**，并未消失；岁运一到，克制仍会补回。

## 现实象意

- **感情层面**：同时拥有两个追求者、介入他人关系、或自己的伴侣被他人争夺。
- **事业层面**：合作方之间互相拉锯、老板身边有两个"近臣"抢位、项目因多头管理而内耗。
- **性格层面**：被争合者往往**风流、好强、喜欢被众人环绕**；争合双方则易**猜忌与较劲**。
- **健康层面**：长期处于争合格局者，精神消耗大、易焦虑，决策常见反复。

## 判断维度

| 维度 | 要点 |
| --- | --- |
| 被争者位置 | 日干 / 配偶宫 / 月干（父母宫 / 事业宫）所主事不同 |
| 争合性质 | 财星 / 官星 / 印星 / 比劫被争 → 分别对应财、权、学、人际 |
| 地支化神 | 化神得气 → 争而微成；化神不得气 → 争而不化 |
| 冲克情况 | 争合的某一端若被冲克除去 → 回归正常五合 |
| 岁运触发 | 大运流年再引同字 → 争合烈度加倍；引对立字 → 解合 |

## 误区

- **误区一**：见两字同合即判"此人必二婚 / 必破财"。实际需看**是否为日主的关键十神**，以及日主旺衰、地支组合。
- **误区二**：把争合当作"双倍合力"。恰恰相反，争合是**合力被稀释**。
- **误区三**：以为争合必化。真相是**争合绝大多数不化**，需严格按合化条件判定。
- **误区四**：只盯天干不看地支。地支若配合支持，能缓解争合的破坏性；反之会放大。

## 古籍背书

- **《渊海子平·论合》**：论十干合之序列，**强调"合需专一"**，凡一合之中出现多字拉扯，"合而不纯，纯而不化"。争合即此类典型。
- **《子平真诠·论用神》**：言用神清纯为贵，**"用神被合则减力，被争合则失用"**。用神若为被争合之字，命格必受损。
- **《三命通会·天干争合》**：明列争合之害——"合而不化，其力涣散；配偶被争，婚姻多变"。并指出**财官被争合者，一生多起落**。

## 一句话总结

**争合 = 一合被两家拉扯，力涣而不专；事有苗头难落定，情有主角常易主。**

## 附：争合 vs 妒合 的本质对比

| 对比维度 | 争合 | 妒合 |
| --- | --- | --- |
| 字面含义 | 两字**争夺**一合 | 两字**妒忌**一合 |
| 阴阳属性 | **异性相合**中的多方争夺（合本身成立） | 合是异性合，但参与争的两字**与合者同性同字** |
| 典型组合 | 两乙合一庚、两丙合一辛等 | 甲己乙、乙庚丙、丙辛丁、丁壬戊、戊癸己（末字为日主） |
| 情感象征 | 三角恋、被多人追求 | 同性嫉妒、小姑子/姐妹争哥嫂之爱 |
| 心理特质 | 风流、被动受宠、摇摆 | 猜疑、嫉妒、挑拨、难容他人亲密 |
| 合化结果 | 多半**合而不化** | 一般**亦不化**（且常被第三字冲克破坏） |
| 破坏方向 | 破坏**合的专一** | 破坏**合的稳定**（因妒而克、因克而散） |

争合重在"**抢**"，妒合重在"**坏**"。争合看**关系结构**，妒合看**心理动机**。两者虽然形式相近，但底层能量截然不同。
