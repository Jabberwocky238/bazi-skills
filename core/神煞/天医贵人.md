# 天医贵人

## 定义
- **天医**：天赐的医缘。
- **天医贵人**：主**医药、健康、治愈、救助**的贵人神煞。
- 命带天医：**与医疗、健康、治愈领域有缘**——或从事医药，或遇病易遇良医。

## 起法（按月支查地支）
《三命通会·论天医贵人》原文："**天医者，月建前一位也**。"

即按月支查，天医在月支前一位：

| 月支（出生月）| 天医位置 |
|---|---|
| 寅月（正月）| 丑 |
| 卯月（二月）| 寅 |
| 辰月（三月）| 卯 |
| 巳月（四月）| 辰 |
| 午月（五月）| 巳 |
| 未月（六月）| 午 |
| 申月（七月）| 未 |
| 酉月（八月）| 申 |
| 戌月（九月）| 酉 |
| 亥月（十月）| 戌 |
| 子月（十一月）| 亥 |
| 丑月（十二月）| 子 |

**背后原理**：天医在月建之前——"**先病后医**"的医缘象征。古人取"医者先行于月"之意。

## 核心作用：医药与治愈
- 天医贵人有**两面**的含义：
  - **主动面**：本人**适合从医 / 从事健康相关职业**。
  - **被动面**：**遇病时能遇良医**——健康层面有贵人护持。
- 古籍论："**天医入命，主医药之缘**。"

## 判断维度

### 1. 位置
- **年支天医**：家族或祖辈与医疗相关 / 少年多病但能治愈。
- **月支天医**：青年期与医疗领域结缘。
- **日支天医**：本人有医缘——要么从医、要么遇医贵人。
- **时支天医**：晚年与医疗紧密 / 子女从医。

### 2. 天医配其他神煞
- **天医 + 天乙贵人**：双贵，一生健康有保障。
- **天医 + 华盖**：精神层面的疗愈——心理治疗师、心灵导师。
- **天医 + 文昌**：医学学者、研究型医生。
- **天医 + 太极贵人**：**玄学 / 宗教层面的治愈**——僧道、宗教治愈师、气功师。

### 3. 天医与食伤的关系
- **食伤 + 天医**：典型的**治愈型才华**——通过作品 / 服务帮人康复。
- 食神主"福气、滋养" + 天医 → 健康顾问、养生专家。

### 4. 大运流年引动
- 走天医运 → 健康层面改善 / 遇良医 / 学医契机。
- 家中有人患病时流年天医 → 多有医疗转机。

## 常见误区

### 1. "带天医 = 一定从医"
- 天医是"**医缘**"，不保证**职业从医**。
- 可能是：
  - **从医**（最直接）。
  - **身体不好但常能遇良医**（被动性医缘）。
  - **家人中有医者**（间接医缘）。
  - **喜欢养生、懂医理**（业余医缘）。

### 2. 把天医当成"不生病的福气"
- 天医不保"**不生病**"，保"**生病能治好**"。
- 很多带天医的人其实**年轻时多病**，但每次都能遇到对的医生，化险为夷。

### 3. 忽视现代医疗的拓展
- 古代"医"专指**医药治病**，现代拓展到：
  - 心理治疗、精神疗愈。
  - 康复理疗、营养师。
  - 健康管理、健身教练。
  - 养生、中医、气功、疗愈系服务。
- 命带天医从事以上任一，都算"**医缘显现**"。

## 现实表现

### 性格
- **对他人的痛苦敏感**：天然想帮人。
- **有"疗愈型"气质**：别人愿意向他倾诉。
- **对身体 / 健康 / 生命议题**特别关注。
- **耐心、温和、有责任感**——医生需要的基本性格。

### 职业路径
- **传统医学**：中医、西医、牙医、外科、内科等。
- **辅助医疗**：护士、药师、康复师、营养师。
- **心理治疗**：心理咨询师、精神科医生、社工。
- **养生疗愈**：气功、针灸、按摩、中医推拿。
- **健康产业**：医药研发、医疗器械、保健品、健身房。
- **精神慰藉**：宗教、灵性导师、哀伤辅导。

### 典型画像
- 从小对"**治病救人**"有认同感。
- 生病时不慌——知道找谁、怎么治。
- 家人生病时常成为"**家庭健康管家**"。
- 中老年多涉足养生、中医、健康相关领域。

## 核心判断
- 天医是"**治愈能力的底牌**"。
- 命带天医的人**走健康 / 治愈相关的路线最合其命**——不一定是医生，但总与"**帮人变好**"有关。

## 一句话总结
- 天医 = **天赐的医缘**。
- 要么你救人，要么有人救你——一辈子跟"**治愈**"两字有关。
- 让这类命造从事健康 / 治愈行业，是**最顺其天性**的选择。
