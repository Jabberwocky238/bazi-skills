# 木火对（总论）

## 性质：相生两气对（木→火）
- **五行关系**：木生火（印→身 或 身→食伤的顺流）。
- **属"印生两神成象格"之一**——《滴天髓》五种相生组合中的"木火相生格"。
- 相生两气双清，若气纯无杂，属贵格。

## 两气成象通则（适用于本对）

### 成立条件（严格依《滴天髓》两气双清论）
1. **命局"两气双清"**——只见木火两种五行。
2. **木生火**——木为母、火为子，顺流无克。
3. **原局无财官**（金为财/官在特定日主、水为印）——一见财官则破局。
4. **日主属木或火**，且不过弱。

### 具体形式（木火相生格）
- **甲乙木日主 + 丙丁火食伤**：身→食伤顺流（木火通明）。
- **丙丁火日主 + 甲乙木印绶**：印→身顺流（木火两盛，带秀气）。

### 气质
- **生机 + 光明** —— 热情、有魅力、聪明外显、富有感染力。
- 典型特征：**光彩照人** —— 走到哪都显眼。

### 典型职业
- 教师、演说家、宗教、思想家、表演、文化名人、公众人物、传媒、影视、培训。

### 禀赋与缺失
- **禀赋**：热情、才华、表达、感染力、领袖气质。
- **缺失**：
  - 缺金 → 缺决断、做事易散漫、缺边界。
  - 缺水 → 智慧受限、过旺易焦躁、易上火。

### 贵格 vs 病象
- **木火通明**（贵格）：木生火明、气机流畅 → 聪明外显。
- **木火相煎**（病象）：木火相克、气机冲突（木被火过度燃尽）→ 燥烈自焚。
- **木多火塞**（病象）：木过多压火、反而使火气不通 → 思路堵塞。
- **寒木向阳**（调候格）：冬木见火暖局 → 生机得启；本应相生却因寒而显特殊。

### 喜忌
- **喜**：木（加强本气）、火（加强本气）、**土**（通关泄秀，木→火→土成生生不息）。
- **忌**：
  - **金**（克木破局）——大忌，金透克木则格毁。
  - **水**（克火破局，但微水润木不忌）——过旺则灭火。
- **调候之需**：本对偏燥（火旺），**微水润木不忌**（润而不克）；但水过则灭火。

### 古籍背书
- 《滴天髓·源流》："**两气双清，其象最妙**。"
- 《穷通宝鉴·甲木章》："**甲木用丁，木火通明**。"
- 《滴天髓》："**木火相生，聪明秀发之象**。"

## 一句话总结
- 木火对 = **木生火之相生双清**——热情、才华、光彩照人。
- 贵在"**木生火显**" → 木火通明；忌"**木过多塞火**""**水重灭火**"。
- 关键看**木火比例 + 调候平衡**：木适度 + 火得透 + 微水润 = 贵。
