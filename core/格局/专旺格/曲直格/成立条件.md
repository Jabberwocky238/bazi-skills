# 曲直格

## 成立条件（严格依《渊海子平·曲直仁寿格》）

### 条件 1：日主为甲或乙木

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·曲直仁寿格》："**曲直仁寿格——甲乙日生人**。"
- **客观判据**：日主天干必须为甲木或乙木。
- **不满足则破**：非甲乙日 → 不入曲直格。

### 条件 2：地支会成木局（三合亥卯未 或 三会寅卯辰）

- **古籍出处**（本条核心）：
  - 《渊海子平·曲直仁寿格》："**地支亥卯未三合木局，或寅卯辰东方木气**。"
- **客观判据**（二者之一）：
  - ① 地支见 亥 + 卯 + 未 三合木局；或
  - ② 地支见 寅 + 卯 + 辰 三会东方木。
  - 若仅半合（亥卯、卯未）或二字会气，格品降但仍可勉强成立。
- **不满足则破**：地支不成木局 → 一气不聚，转入普通身强木论。

### 条件 3：天干多透木，最好有壬癸水生之

- **古籍出处**：
  - 《渊海子平·曲直仁寿格》："年月时干宜见甲乙……**有壬癸水相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位甲或乙透出；
  - ② 若另见壬水或癸水透干生木，格品升。
- **不满足则破**：天干木气不足 → 一气未能遍布。

### 条件 4：原局无庚辛金透干克木

- **古籍出处**（本条核心）：
  - 《渊海子平·曲直仁寿格》："**年月时干无金相克**，主仁寿双全。"
- **客观判据**：
  - ① 天干无庚金或辛金透出；
  - ② 地支本气无申金、酉金成势（一位藏支微弱尚可，多则破）。
- **不满足则破**：金透克木 → 《渊海子平》"有金相克"即明破格 → 转入"斧斤伐木"或普通克身格讨论。

### 条件 5：火木搭配适度（无火偏闷、微火通明、重火耗气）

- **古籍出处**：
  - 《渊海子平·曲直仁寿格》："**若有丙丁微火吐秀，则为木火通明，更见贵气**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - ① 原局无火 → 纯曲直，偏闷但成格；
  - ② 微火（一位透干或藏支）→ "曲直 + 木火通明"复合贵格；
  - ③ 火多透多根 → 耗木过度 → 降格为"木火相煎"等讨论。
- **不满足则破**：重火大量耗气 → 格变。

---

**前四条必须同时满足**方为真曲直仁寿格；条件 5 关乎格品高低——有微火点缀者为"曲直 + 通明"复合贵格，无火则偏闷，重火则耗。
