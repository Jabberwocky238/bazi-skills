# 伤官合杀

## 定义
- **伤官**：我所生之异性者，代表**才华、锋芒、独立思考、不受约束**。
- **合**：天干五合 —— 阴阳相配的合化关系。
- **杀**：七杀。
- **伤官合杀**：命局中**伤官与七杀通过天干五合**相合 → 伤官不直接克官、也不被七杀压身 → 两股刚烈之气**互相牵制、彼此约束**。
- 与"**伤官见官**"对照：见官是冲突破坏，合杀是**迂回化解**。

## 核心：两股刚以合相解
- 伤官刚（锋利外放）、七杀刚（高压克制）—— 通常两者都是不好惹的力量。
- 但命局中：
  - 如果伤官与七杀**天干相合**（如甲 + 己、乙 + 庚、丙 + 辛、丁 + 壬、戊 + 癸）且五合发生在伤官与杀之间；
  - 两者就会**互相牵住对方** → 谁也不再单独发作。
- 结果：
  - 伤官不再去冲官。
  - 七杀不再去压身。
  - 两者在合里**各自停下**，日主反而得**清净**。

## 判断伤官合杀的关键
1. **命局同时见伤官 + 七杀**。
2. **两者之间存在五合关系**。
3. **合得住**：位置紧贴、力量相当、无其他字阻隔或争合。
4. **日主旺度合适**：
   - 身不过弱 → 才能借合解出生机。
   - 身过弱 → 合完了，杀还没消净，仍然伤日主。

## 与相关格局对比

| 格局 | 动作 | 用途 |
|---|---|---|
| **伤官合杀** | 伤合杀 | 迂回化解，身得清净 |
| **伤官见官** | 伤克官 | 破格，惹事 |
| **食神制杀** | 食克杀 | 主动解决压力 |
| **杀印相生** | 印化杀 | 接纳转化 |
| **羊刃驾杀** | 身抗杀 | 刚对刚 |
| **伤官佩印** | 印制伤 | 约束锋芒 |

**伤官合杀**的独特在于：**不是压、不是抗、不是化，是"合"**。两股力量**用互相牵绊的方式**变成安静。

## 成立条件
1. **伤官与七杀同时透干或均有根**。
2. **两者形成明确的五合**。
3. **合的位置紧贴**（月 / 日、日 / 时相邻）。
4. **无争合**：其他天干不要来争同一个合对象。
5. **不化或化得合适**：
   - 伤官合杀后**不化**：两者互相牵住，格清；
   - 化成其他五行（如甲己合化土）：视化神对日主是喜是忌再论 —— 若化神对日主有利，则更妙；若化神为忌，则合化后反为害。

## 判断维度

### 1. 合的对象是什么样的"伤官"与"七杀"
- 伤官原本的强度（是否得月令、有几根）。
- 七杀原本的强度（是否得月令、有几根）。
- 两者力量接近 → 合得住；力量悬殊 → 合不牢。

### 2. 日主是否受益
- **身中或身略强**：日主借合得清净 → 正面。
- **身弱**：合了伤，却没消化完杀 → 杀仍克身 → 作用打折。
- **身极强**：合没合对日主没多大影响 → 本格效用不显。

### 3. 有无争合
- 旁边出现**能与伤或杀相合的其他字** → 形成"争合"。
- 争合发生 → 合力减弱 → 格局乱。

### 4. 合化后的气质
- **甲己合（伤官合正官 / 杀）化土**：化为财星 → 若财为用神 → 大吉。
- **乙庚合化金**：化为官杀本气 → 可能继续压身。
- **丙辛合化水**：化为印星或食伤 → 看对日主作用。
- **丁壬合化木**：化为印星或比劫 → 看结构。
- **戊癸合化火**：化为比劫或印星 → 看结构。
- 不是所有合化都是好事，要具体配合日主喜忌。

## 常见误区

### 1. "有伤官 + 有七杀 + 在一起 = 伤官合杀"
- 必须真的**五合成立**，且位置紧贴、力量匹配。
- 伤官与七杀只是"同时存在"不一定构成合杀。

### 2. 把伤官合杀等同"吉格"
- 是一种**救应手段**，不必然是吉格。
- 合杀之后的**化神是否对日主有利**才决定最终好坏。
- 合化成忌神 → 反倒由可控变失控。

### 3. 以为合了就永远不动
- 大运流年可以**引动合**：
  - 新来的字**破合**（冲掉合的一方）→ 原本被合住的伤或杀**重新活跃** → 可能突然爆发。
- 所以"合住了"不等于"永远安全"。

### 4. 与伤官见官混淆
- **伤官见官**：两者直接对冲 → 破坏。
- **伤官合杀**：两者被合关系固定 → 暂时平息。
- 同样是"伤 + 官杀"组合，处理方式和结果完全不同。

## 现实表现

### 性格
- **外表温和、内里有锋芒**。
- **遇事会绕弯子**，不硬碰。
- **有才华但不张扬**：伤官的锐被合住了。
- **对压力的应对方式是"柔"**：找破口、找合作、找台阶下。
- **善于用关系化解冲突**：合的本质就是"关系的力量"。

### 事业路径
- **需要"才华 + 柔性应对压力"**的岗位：
  - 外交、谈判、公关、商务合作。
  - 精神 / 心理顾问型职业。
  - 协调型管理岗。
- 不适合需要**正面硬刚**的岗位（那是羊刃驾杀或食神制杀的场子）。

### 典型画像
- 遇到强势对手不退让，也不硬顶 —— 而是**找一个两者都能接受的位置**。
- 在冲突中显示"**化解能力**"。

## 破格

### 合被冲破
- 大运流年出现与伤或杀相冲的字 → **冲开合**：
  - 伤官重新能克官。
  - 七杀重新能克身。
  - 长期被压住的问题**瞬间爆发**。
- 典型现象：原本平静的人在某一年突然**失控 / 发生剧变**。

### 合化成忌
- 合化出的五行若对日主为忌 → 看似"解决"了伤杀问题，实则**引入了新的忌神**。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 合神加持运 | 合更牢 → 格局稳 |
| 冲破合的大运 | **大忌** → 合散 → 伤杀重新活跃 |
| 印运 | 视结构 → 若化神为印则喜；否则中性 |
| 财运 | 视合化 → 合化成财时最妙 |
| 比劫运 | 视结构 → 帮身可解部分杀 |

## 与相关救杀格局的系统对比

| 救杀方式 | 动作 | 气质 | 日主需要 |
|---|---|---|---|
| **杀印相生** | 化 | 温和接纳 | 身中 + 有印 |
| **食神制杀** | 克 | 主动出击 | 身中强 + 食神有力 |
| **羊刃驾杀** | 抗 | 刚对刚 | 身极强 |
| **伤官合杀** | 合 | 迂回牵绊 | 身中 + 合要合得住 |
| **从杀格** | 顺 | 彻底臣服 | 身极弱 |

## 核心判断
- 伤官合杀的精髓是"**合而非争**"。
- 两股刚烈的力量通过合**互相变成对方的约束** —— 谁都不再单独伤人。
- 格是否成立的关键：
  - **合得住吗？**
  - **合化成什么？**
  - **大运流年会不会冲破这个合？**
- 比见官要聪明，比驾杀要温和，比顺杀要保留自我。

## 一句话总结
- 两股刚烈的力量**用一场合约彼此约束** —— 谁都不再伤人。
- 伤官合杀 = **不靠压、不靠化、不靠抗，靠"绑在一起"** 解开难题。
- 人生智慧层：学会把对立的力量**放进同一个框架**里互相牵制，反而都能安下来。
