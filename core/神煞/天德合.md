# 天德合

## 定义
- **天德合**：与**天德贵人**构成**天干五合关系**的字——天德之气的"合煞神"。
- 本质：**承接天德之气、辅助天德贵人**的辅助型神煞。
- 与天德贵人**配对出现**，但独立成神煞——非天德贵人的别名或延伸。

## 天德贵人 vs 天德合

| 对比项 | 天德贵人 | 天德合 |
|---|---|---|
| 起法 | 按月支查固定天干 / 地支 | 与天德贵人天干五合之字 |
| 性质 | **天时加持的主神** | **合天德之气的辅神** |
| 作用 | 遇凶化吉、关键时刻的天时护持 | 承接天德之气、间接受福 |
| 出现概率 | 单命造必定有一字 | 需特定合字出现 |

**关系**：月令定天德 → 与天德相合之字 = 天德合。两者**同气相接**——天德合到位时，天德贵人之气被"合住、留住"。

## 核心作用：承接天德之气
- 天德贵人是"**天时加持的主神**"——直接化凶为吉。
- 天德合是"**把天德之气留住**"的辅助：
  - 与天德贵人紧贴合住 → 天德之气不散、不被冲破。
  - 即便命中无天德贵人，只要有天德合出现 → 仍受天德之气的间接加持。
- 古籍论："**天德合入命，凡事有救，小凶化无**。"

## 判断维度

### 1. 天德合单独出现 vs 与天德贵人并现
- **单独天德合**：承接天德之气，**间接受福**——遇事仍有化解之力。
- **天德合 + 天德贵人并现**：**双重护持**——主神 + 辅神俱备，凶煞近身皆可化。

### 2. 位置
- **年柱天德合**：家族 / 早年福气。
- **月柱天德合**：成长期与天德贵人相应。
- **日柱天德合**：本人福气承接——最贴身。
- **时柱天德合**：晚年 / 子女受福。

### 3. 配合其他贵神
- **天德合 + 月德合**：双合之贵——福气加倍。
- **天德合 + 天乙贵人 / 月德贵人**：多重化凶之力。

### 4. 是否被破坏
- 天德合所在天干被合化为异神 → 合力转移。
- 地支被冲 → 合力被冲开。

## 常见误区

### 1. "天德合 = 天德贵人"
- 最大误判。两者起法不同、性质有别。
- 天德贵人是"**主神**"；天德合是"**合神**"。
- 虽都属天德系统，但作用有别——命书若混为一谈 → 判断会失准。

### 2. 把天德合视作"比天德贵人低一级"
- 天德合的力量确实**不如天德贵人直接**。
- 但两者**并现时有叠加效应**——**天德合 + 天德贵人同柱或紧贴 → 福气牢固不散**。

### 3. 忽略"合"的本质
- 合 = 留、牵、接住。
- 天德合的核心是"**把天德之气合住**"——防止天德贵人之气被冲 / 被克 / 被散。
- 单有天德贵人而无天德合 → 天德之气可能被流年大运冲散；
- 有天德合 → 天德之气被牢牢锁住。

## 现实表现

### 性格
- **温和、有福相、不激进**。
- **关键时刻往往有"隐形"的化解力**——自己未必察觉，但周围人觉得"**他很顺**"。
- **遇事能退一步**：不硬刚。

### 典型情境
- **遇危险自然避开**——不是警觉度高，是"**刚好**"。
- **合作 / 关系中能"合住"**：与人合拍、易促成事。
- **凶事来时总有缓冲**：该出大事的时候出小事、小事化无。

### 与天德贵人并现的典型
- **一生很少碰到大灾难**。
- **身边常有"**意外的帮助**"**——事情即将坏掉时有人 / 有契机出现。
- **即便命中有凶煞 / 七杀 / 羊刃 → 也难显恶性**。

## 核心判断
- 天德合是"**天德之气的锁**"——把福气锁住。
- 单独出现 → 间接受福。
- 与天德贵人并现 → 天时加持牢不可破。
- 识别意义：这类命造**可能自己察觉不到福气之源**，但一生下来确实比一般人"**顺**"。

## 一句话总结
- 天德贵人 = 天赐的福。
- 天德合 = **把天赐的福锁住的那根线**。
- 有线在，福就散不掉——**这就是天德合的真正价值**。
