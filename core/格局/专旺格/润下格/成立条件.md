# 润下格

## 成立条件（严格依《渊海子平·润下格》）

### 条件 1：日主为壬或癸水

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·润下格》："**润下格——壬癸日生人**。"
- **客观判据**：日主天干必须为壬水或癸水。
- **不满足则破**：非壬癸日 → 不入润下格。

### 条件 2：地支会成水局（三合申子辰 或 三会亥子丑）

- **古籍出处**（本条核心）：
  - 《渊海子平·润下格》："**地支申子辰三合水局，或亥子丑北方水气**。"
- **客观判据**（二者之一）：
  - ① 地支见 申 + 子 + 辰 三合水局；或
  - ② 地支见 亥 + 子 + 丑 三会北方水。
  - 若仅半合（申子、子辰、亥子）或二字会气，格品降但勉强成立。
- **不满足则破**：地支不成水局 → 一气不聚。

### 条件 3：天干多透水，庚辛金生水更佳

- **古籍出处**：
  - 《渊海子平·润下格》："年月时干宜见壬癸，**有庚辛金相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位壬或癸透出；
  - ② 若另见庚或辛透干生水，格品升（源源不绝）。
- **不满足则破**：天干水气不足 → 一气未遍布。

### 条件 4：原局无戊己土透干克水

- **古籍出处**（本条核心）：
  - 《渊海子平·润下格》："**年月时干无土相克**，主智慧清贵。"
- **客观判据**：
  - ① 天干无戊土或己土透出；
  - ② 地支本气无辰戌丑未成势（一位藏支中气无碍）。
- **不满足则破**：土透克水 → 《渊海子平》"土相克"即明破格 → 转入"水火相战"反向（或身弱官重）讨论。

### 条件 5：原局无重火蒸水

- **古籍出处**：
  - 《穷通宝鉴·壬水章》："**水忌火烈蒸干**。"
- **客观判据**：
  - 天干丙丁不多透（一位无根尚可，微火调候反喜）；地支本气无巳、午成势。
- **不满足则破**：重火蒸水 → 水气涸竭 → 破格转入"水火相战"讨论。

### 条件 6（助力条件）：金源不断 + 微木泄秀 → "金白水清""水木清华"复合贵格

- **古籍出处**：
  - 《穷通宝鉴》："**庚金逢壬水，为金白水清**。"
  - 《穷通宝鉴》："壬癸见甲乙，**水木清华**。"
- **客观判据**：
  - ① 金透干生水 → 金白水清叠加润下；
  - ② 木一位透干或藏支泄秀 → 水木清华叠加润下。
- **非必要**：不满足不破格；满足则格品升（智慧之中添清秀）。若木多过度泄水则降格。

---

**前五条必须同时满足**方为真润下格；条件 6 关乎格品——纯润下成但偏幽深，得金生则源不竭、得木泄则秀外显。
