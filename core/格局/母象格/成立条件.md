# 母象格

## 成立条件（严格依《滴天髓》母象论）

### 条件 1：满盘皆比劫（比劫统治全局）

- **古籍出处**（本条核心）：
  - 《滴天髓》："**知慈母恤孤之道，始有瓜瓞无疆之庆**。"
  - 《滴天髓》（任铁樵注）："**满盘皆比劫**——比劫为母、食伤为子，母多子少需含饴弄孙。"
- **判据**：
  - ① 比肩 + 劫财在天干 + 地支本气共四位以上，或
  - ② 月令 + 地支三合 / 三会成比劫局；
  - ③ 命局整体由比劫之气统治。
- **不满足则破**：比劫不成势 → 非母象格，按普通身旺论。

### 条件 2：命局带食伤（食伤为"子"）

- **古籍出处**：
  - 《滴天髓》："**母象格者，比劫为母、食伤为子**——子不可无、无子则气闭。"
- **判据**：
  - 食神或伤官透于天干，或地支本气有食伤。
  - 食伤与比劫形成"**比劫生食伤**"的顺流。
- **不满足则破**：满盘比劫 + 无食伤 → 转入"**子象格**"（若有印）或"**比劫独旺**"破格讨论。

### 条件 3：原局无印绶（或印绶力微）

- **古籍出处**：
  - 《滴天髓》："母象格**忌印绶**——印来则食伤被克、母子之间不通。"
- **判据**：
  - 原局天干无印星透出；地支本气无印，仅藏支一位余气可忽略。
- **不满足则破**：有强印 → 转入"**子象格**"讨论（比劫为子、印为母，与母象相反）。

### 条件 4：原局无官杀（或官杀力微）

- **古籍出处**：
  - 《滴天髓》："母象格**忌官杀**——官杀克比劫（母），使母不恤孤子。"
- **判据**：
  - 原局天干无官杀透出；地支无官杀本气。
- **不满足则破**：官杀透且有根 → 克比劫之母 → 母子关系被破 → 格变。

### 条件 5（辅佐条件）：有财星接应食伤顺流

- **古籍出处**：
  - 《滴天髓》："**食伤为子、财星为孙**——母喜含饴弄孙。"
- **判据**：财星透干或有根，位于食伤之后（比劫→食伤→财 的三级顺流）。
- **非必要**：不满足不破格，但格局助力缺失。满足则为"瓜瓞绵绵"之象（家族昌盛、后辈兴旺）。

---

**前四条必须同时满足**方为真母象格；条件 5 为格品助力。

## 喜忌总则

- **喜**：**食伤**（子）、**财星**（孙）、**比劫**（母不失势）
- **忌**：
  - **官杀**：克比劫（伤母）
  - **印绶**：克食伤（灭子）

## 核心气机
- 母象格 = **母多子少，需多生多养**：
  - 比劫（母）满盘 → 需要食伤（子）来分担母气。
  - 食伤（子）带财（孙）→ 母喜含饴弄孙 → 家族兴旺。
- 古籍意象："**瓜瓞无疆**"——子孙繁衍、绵延不绝。

## 古籍原文

- 《滴天髓》："**知慈母恤孤之道，始有瓜瓞无疆之庆**。"
- 《滴天髓·慈母恤孤》："**母象格，满盘比劫而有食伤，喜食伤财星比劫，忌官杀印绶**。"

## 判据特征

- **典型画像**：比劫重的人本就不服管；再加食伤 = 有才华有出口 → 兴家立业。
- **大运核心**：食伤运、财运为最佳；印运、官杀运为忌。
