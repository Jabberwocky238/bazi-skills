# 建禄格

## 定义
- **建禄**：日主的**禄位**恰好落在月令（月支）上。
- **禄**：天干对应的临官之地 —— 十干各自有一个本气之地，那一地就是自己的"禄"。
  - 甲禄在寅、乙禄在卯、丙禄在巳、丁禄在午、戊禄在巳、己禄在午、庚禄在申、辛禄在酉、壬禄在亥、癸禄在子。
- **建禄格**：月支是日主的禄 → 身自带旺气、气象挺拔。
- 属八字**正格**之一，与阳刃格、官印格、食神格等并列。

## 核心：身正气足，不靠外援
- 禄 = **自己的地盘**，站稳脚跟的根基。
- 建禄 = **月令就是我的主场** → 身强、有底气、自尊心强。
- 不像财格、官格、印格需要"靠外在"成格：
  - **建禄格的根基来自日主自身**。
  - 本格成立意味着**人**稳了，接下来只需要处理"**用什么去发力**"。

## 成立条件
1. **月支为日主之禄**：基本条件。
2. **日主天干与月令气机一致**：如甲日生寅月（甲木得寅中甲木本气）。
3. **无强力破坏**：月令被冲、被合化成他物 → 建禄性质改变。
4. **原局配置有用神**：建禄格自身不是用神，需要找**官、财、食伤**作为发力出口。

## 判断维度

### 1. 用神选择
建禄格因身强，**身本身不需要扶**，重点在找**出口**：

| 结构 | 用神方向 | 特征 |
|---|---|---|
| 有**七杀**有制 | **七杀为用** | 有挑战 / 压力 → 带出事业心，典型贵格 |
| 有**正官**清透 | **正官为用** | 走仕途、管理岗、稳定事业 |
| 有**财星**通关 | **财为用** | 能赚钱、生活落地 |
| 有**食伤**秀气 | **食伤为用** | 才华型、表达型、技术型 |
| **四无依靠** | 印比重重 | 格局平平，做平凡人为主 |

### 2. 是否有官杀做事
- 建禄格最怕**无官杀**：身强无制 → 气无处去 → 容易倔、容易自负。
- 有官杀 + 有制 → **化杀为用**或**官为贵** → 格局上台阶。

### 3. 大运走向
- 建禄格身本强，**忌再走比劫 / 印运**（扶身无意义，反而壅塞）。
- **喜走财官食伤**：有出口才出成就。

## 常见误区

### 1. "建禄 = 命一定好"
- 身强 ≠ 命好。
- 如果原局**无官无财无食伤** → 人有精气神，却没地方发挥 → 空有一身力气。
- 现实可能表现：有能力但没赛道、有脾气但没事业、不服气又没办法。

### 2. 把建禄当成阳刃
- **阳刃**：月支为**阳干的羊刃位**（如甲日见卯、丙戊日见午、庚日见酉、壬日见子）→ 比"禄"**更旺一格**。
- **建禄**：月支为**本气禄位** → 是"站稳"。
- 阳刃更刚烈、建禄更端正。

### 3. 以为身强就不需要帮扶
- 即便身强，走逆运时**也要学会收敛**。
- 建禄格的人容易觉得自己"扛得住"，结果越逆运越硬刚 → 反而摔得重。

## 现实表现

### 性格
- **自立、自尊心强、独立性高**。
- **骨子里不依赖人**，不喜欢欠人情。
- **遇事扛得住**：抗压性强、遇事不慌。
- **不愿委曲求全**：关键时候宁折不弯。

### 与家人关系
- **兄弟姐妹关系复杂**：建禄多对应比劫 → 小时候家里资源分配敏感；长大后多独立自主。
- **父母资助有限**：身强主要靠自己打拼，不太能指望父母大额帮扶。
- 适合**早早分家、早早独立**的路径。

### 事业路径
- 适合**自负盈亏 / 独当一面**的岗位：
  - 创业、自由职业、技术专家、独立项目负责人。
- 在**需要靠自己拍板**的场景里最有优势。
- 在**需要卑躬屈膝 / 拼关系**的场景里吃亏。

### 钱财
- **不易一夜暴富**：靠稳扎稳打积累。
- **不易大破产**：身强守得住钱。
- 典型是**中产 / 小康偏稳**的路线。

## 破格情形
1. **月令被冲破**：如甲禄在寅，寅申冲 → 建禄受损。
2. **月令被合化成他物**：如丙禄在巳，巳申合化水 → 火禄变水 → 格局变质。
3. **身强无出口**：无官无财无食伤 → 空身强，格不成。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 财运 | **喜** → 给出口、给落地 |
| 官杀运 | **喜**（有制）→ 带来事业台阶 |
| 食伤运 | **喜** → 才华变现、表达立业 |
| 比劫运 | **忌** → 身更强、竞争更大、分财 |
| 印运 | **忌** → 旺上加旺，更倔、更闷 |

## 与相关格局的关系

| 格局 | 月令角色 | 日主性质 |
|---|---|---|
| **建禄格** | 日主之禄 | 身强，稳重挺拔 |
| **阳刃格** | 日主之刃 | 身极强，刚烈 |
| **正官 / 七杀格** | 月令克日主 | 身受制，有外部目标 |
| **正 / 偏印格** | 月令生日主 | 有资源 / 保护 |
| **正 / 偏财格** | 日主克月令 | 重物质、重现实 |
| **食神 / 伤官格** | 日主生月令 | 重表达、重输出 |

## 核心判断
- 建禄格的核心不是"禄"本身，而是**身站稳之后的用法**。
- 有出口 → 自成一格，独立一生。
- 没出口 → 空有精力没处使，最怕**大运再不给路**。

## 一句话总结
- 建禄 = 地基打得牢。
- 地基牢以后的人生，看你**盖什么楼、盖多大**。
- 没有蓝图（财官食伤），再牢的地基也只是光秃秃的地基。
