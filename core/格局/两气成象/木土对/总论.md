# 木土对（总论）

## 性质：相克两气对
- **五行关系**：木克土（财 / 官杀关系）。
- **非"印生两神成象格"**——该格专论相生两气，木土属相克，不入双清贵格。
- 属"两气成象"的**相战 / 对立**一支。

## 两气成象通则（适用于本对）

### 成象要件
- 命局只见木、土两种五行。
- 木、土力量大致相等。
- 月令为木或土之一。

### 与相生对的根本区别
- **相生对（如木火对 / 金土对）**：两气顺生、气通为清。
- **木土对（本条）相克**：木克土——属"木疏厚土"等特殊结构。

### 特殊情况：木疏厚土（贵格的条件型）
- 木克土本为"克"，但若土过厚（如稼穑格的厚土）而缺疏泄之物 → 木来疏土反为救应 → 贵格。
- 条件：
  - 土必须是"厚重之土"（稼穑格 / 四库全土）。
  - 木适度而有根、不过旺。
  - 典型：戊己日主 + 四库厚土 + 天干透甲 → 甲木疏厚土。

### 病象：木多克土 / 土重木折
- **木多克土**：木过旺克土 → 土失力、事业失根。
- **土重木折**：土过厚而木纤弱 → 木被压扁、无生发。

### 取用要点
- 关键判断：**木土力量对比 + 土的厚度**。
  - 厚土需木疏 → 木疏厚土为贵。
  - 轻土被木克 → 破格。
- **喜通关五行**：**火** = 木 → 火 → 土（木生火、火生土），化克为顺生。
- **喜金护土**：金克木救土，但金要适度不过旺（否则又克木之根）。

### 古籍背书
- 《滴天髓》："**土重得木则疏，**木疏厚土**之象**。"
- 《三命通会·稼穑格》："厚土得木疏，**方显其用**。"

## 本对下属格局
- **木疏厚土**：木疏厚重之土的特殊救格。详见 `木疏厚土/`。

## 一句话总结
- 木土对 = **相克两气、以克为用（或为病）**。
- 木疏厚土 → 贵格（厚土得木疏而显用）。
- 木多克薄土 → 病格（土失根）。
- 要解：**得火通关**（木→火→土）；或**土厚木适度**自成救应。
