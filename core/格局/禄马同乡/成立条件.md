# 禄马同乡

## 成立条件（严格依《渊海子平》《三命通会》）

### 条件 1：日主的禄位落于命局某一地支

- **古籍出处**：
  - 《渊海子平·论禄》："**干禄于支——甲禄寅、乙禄卯、丙戊禄巳、丁己禄午、庚禄申、辛禄酉、壬禄亥、癸禄子**。"
- **客观判据**（十干禄位对照表）：
  - 日干在年 / 月 / 日 / 时四地支任一位上，遇到本干之禄位地支。
- **不满足则破**：日主无禄于命局 → 不具"禄"之半 → 不入本格。

### 条件 2：驿马星出现于命局地支

- **古籍出处**：
  - 《三命通会·论驿马》："**寅午戌马在申，申子辰马在寅，巳酉丑马在亥，亥卯未马在巳**。"
- **客观判据**（三合局对照）：
  - 按年支（传统主法）或日支（现代兼用）查驿马：
    - 年 / 日支属寅午戌 → 驿马为申；
    - 年 / 日支属申子辰 → 驿马为寅；
    - 年 / 日支属巳酉丑 → 驿马为亥；
    - 年 / 日支属亥卯未 → 驿马为巳。
  - 命局年 / 月 / 日 / 时任一地支上须见该驿马字。
- **不满足则破**：命局无驿马 → 不具"马"之半 → 不入本格。

### 条件 3：禄与驿马落于同一地支（"同乡"的本义）

- **古籍出处**（本条最核心、为本格定义）：
  - 《三命通会·论驿马》："**驿马与禄并居一宫，为之禄马同乡**。"
  - 《渊海子平·禄马赋》："**禄马同乡，身居富贵之地**。"
- **客观判据**：
  - 条件 1 所得的禄支与条件 2 所得的驿马字**为同一地支**。
  - 典型结构例：
    - 日主甲木 + 年支申 → 驿马寅 + 甲禄寅 → 寅为禄马同乡；
    - 日主庚金 + 年支寅 → 驿马申 + 庚禄申 → 申为禄马同乡；
    - 日主壬水 + 年支巳 → 驿马亥 + 壬禄亥 → 亥为禄马同乡。
- **不满足则破**：禄与马分居不同地支 → 非"同乡"，只作独立的"带禄""带马"论，不入本格。

### 条件 4：禄马同乡之地支不被冲、不被合化

- **古籍出处**：
  - 《渊海子平·禄马赋》："**禄马逢冲，功名无托**。"
- **客观判据**：
  - ① 禄马所在地支与其他地支无六冲（如禄马在寅，全局无申来冲；禄马在亥，全局无巳来冲）；
  - ② 该地支不因三合 / 三会 / 六合而合化为他气。
- **不满足则破**：冲者禄马散；合化者根基转移——《渊海子平》"功名无托"即指此败局。

### 条件 5：日主不至极弱（能承载禄马之"动"）

- **古籍出处**：
  - 《三命通会·论驿马》："**马忌身衰**——身弱而马旺，奔波劳苦。"
- **客观判据**：
  - 日主于月令至少有比印余气，或年 / 月 / 时有一位比劫 / 印星通根。
- **不满足则破**：身极弱而禄马同乡 → 动而无力，《三命通会》"马忌身衰"——奔波而无成。

---

**五条同时满足**方为真禄马同乡。本格核心在一"同"字——禄（古籍定义甲禄寅等）与马（古籍定义寅午戌马申等）必须落入同一地支，方得"稳中有动、动中有根"的气机结构。
