# 木疏厚土

## 成立条件
1. **土极为厚实**：
   - 天干戊己透出且多。
   - 地支辰戌丑未见两个以上。
   - 整个局面以土为底色。
2. **木有力疏土**：
   - 甲乙透干。
   - 地支见寅卯 / 亥卯未合木局。
   - 木有根能真的"扎进土里"。
3. **木不过旺**：
   - 木若过旺 → 转为"**木多土崩**" → 破格。
   - 理想比例：**土 3，木 1-2**。
4. **有水润木**：
   - 水生木 → 木有生机 → 疏土更有力。
   - 无水则木"枯根"，疏土力弱。
5. **无重金克木**：
   - 金若过多 → 克断木 → 疏土失败 → 退回土重金埋或稼穑格状态。
