# 库（财库）

## 核心口诀
**坐下财库，喜行冲。**

## 常见误读
- 很多人以为：做生意能不能发财，看**财星旺不旺**就行。
- 这种判法**偏颇**。
- 真正能发大财的人，**不是财多的人**，而是八字里**有库**的人。
- 为什么？因为**库是装钱的器皿**——没有库，再多钱也留不住。

## 什么是财库
- **辰、戌、丑、未**这四个地支就是"**四库**"（土库）。
- 命局里"坐下财库" = **日支/主要支位形成财库**。
- 就像**开好了一个银行账户**——进来的钱有地方存、守得住。

## 只有库还不够：要"行冲"才真正开库
- 财库光**守着**还不行。
- **冲 = 开库**：
  - 库被冲动，钱才能真正流动起来、赚进口袋里。
- 库不动 → **死财死气**，变成一堆无用的资产。
- 因此——
  - 命带财库 + 大运流年来冲库 = **钱财大动、发家的关键节点**。

## 为什么发大财的人都有库

### 没有库的人
- 哪怕**财星再旺**，也容易"**钱来得快、去得快**"。
- 走财运时可能赚了不少，但——
  - 全赔在投资或人情上。
  - 或根本攒不住，年底算账一场空。

### 有库的人
- 哪怕财星不强，运走对了，**钱就能聚起来**。
- 这类人往往**稳扎稳打、不声不响地发家**。
- 现实典型：很多老板表面普通，实则**暗藏库根**，几十年生意越做越稳、资产越来越厚。

## "坐下财库，喜行冲"怎么用

### 库 = 自带的聚财格
- 你本身有"库根"——是先天的聚财结构。

### 但库不开 = 保险箱锁死
- 再多钱也用不出来、散不开、转不动。

### 冲库之年就是大节点
- 走到**能冲能动这个库**的流年大运——
  - 就是**财运开始动起来**的关键时刻。

### 敢不敢抓、会不会用
- 懂的人在这种节点会：
  - **转行、扩张、投资**。
  - 踩中节奏，**一鸣惊人**。
- 不懂的人会错过——等机会过去了再想动，就晚了。

## 一句话总结
**有库是聚财的基础，行冲是开库的钥匙。**
- 财星决定你能赚多少，库决定你**能留下多少**。
- 没有库：赚再多也是过路财神。
- 有库无冲：钱被锁死，不能发力。
- 有库 + 行冲：真正的发家节点。
