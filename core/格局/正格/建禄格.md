# 建禄格

## 定义
- **建禄**：日主的**禄位**恰好落在月令（月支）上。
- **禄**：天干对应的**临官之地** —— 十干各自有一个本气之地，那一地就是自己的"禄"。
  - 甲禄在寅、乙禄在卯、丙戊禄在巳、丁己禄在午、庚禄在申、辛禄在酉、壬禄在亥、癸禄在子。
- **建禄格**：月支是日主的禄 → 身自带旺气、气象挺拔。
- 属八字**正格**之一，与阳刃格、正官格、正印格等并列。

### 相关禄位概念
- **日支作禄 = 专禄**：禄位落在日支。
- **时支作禄 = 归禄**：禄位落在时支。
- **月支作禄 = 建禄**（本格）。

### 戊己土的特殊处理
- **戊禄在巳、己禄在午**——并不是辰戌丑未土；
- 戊己为日元时即使辰戌丑未土出现在月令上，也只能当作比肩或劫财来看。
- 简记：**除戊己土外，地支上出现比肩 = 日元的禄神；出现在月令上面 = 建禄格**。

## 核心：身正气足，不靠外援
- 禄 = **自己的地盘**，站稳脚跟的根基。
- 建禄 = **月令就是我的主场** → 身强、有底气、自尊心强。
- 不像财格、官格、印格需要"靠外在"成格：
  - **建禄格的根基来自日主自身**。
  - 本格成立意味着**人**稳了，接下来只需要处理"**用什么去发力**"。
- 月令作为四柱中**能量最大、最重要**的宫位——建禄能为日主提供**无穷无尽的能量源泉**。
- 但如果**制伏不当**则会**与日元争夺名利、财官等资源**——因此必须**制之、化之**。
- **推论**：只要是建禄格——由于禄神在月令的独特作用——这个八字的日元基本上**不会太弱**；若再出现印星或比劫 → 基本可判定日元为**强旺状态**。

## 成立条件

### 条件 1：月支本气为日主之禄
- **判据**（十干对照，月支本气须为以下其一）：
  - 甲日寅月、乙日卯月、丙戊日巳月、丁己日午月、庚日申月、辛日酉月、壬日亥月、癸日子月。
- **取用要点**：**月支为日元禄神即为建禄格**——除戊己土外地支出现比肩、在月令即为禄神。
- **不满足则破**：月支非本禄 → 不入建禄格、按月令本气另论（如甲日生辰月论偏财格）。

### 条件 2：月令不被冲、不被合化为他五行
- **判据**：
  - ① 月令地支与年 / 日 / 时支均无六冲（寅申、卯酉、辰戌、巳亥、子午、丑未）；
  - ② 月令不因三合、三会、六合而合化为异五行。
- **取用要点**：**建禄格最怕月令逢冲**——大运流年冲月令则格局力破、往往带来大变数（羊刃逢冲尤其严重）。
- **不满足则破**：冲者禄根动；合化者建禄性变、按合化所成之局另论。

### 条件 3：原局有官、财、食伤之一透干且通根，作为日主的泄 / 克出口
- **判据**：
  - ① 正官 / 七杀 / 正财 / 偏财 / 食神 / 伤官至少一类于年 / 月 / 时天干透出、且在地支本气或中气通根至少一位；
  - ② 优先级：正官（清纯不带杀）> 七杀（有食制或印化）> 财（带食伤通关）> 食伤（最宜走财）。
- **取用要点**：
  - **禄神在月令如能提供最大能量**——基本上日元不会太弱；若再见印星或比劫即身强。
  - **身强必须"制之、化之"**——通常用财官食伤来平衡调节。
  - 若能量"制伏不当"则会与日元争夺名利、财官资源。
- **不满足则破**：无官无财无食伤之透根者 → "日元既旺而无所泄克"属空身强、不入贵格。

### 条件 4：原局其他三柱不再层层叠加比劫印星致身过旺无泄
- **判据**：年 / 日 / 时三柱不全为比劫或印星本气；若月令建禄之外、其他三柱再见三位以上比印通根、而官 / 财 / 食伤之根微弱、则身过旺无出口。
- **取用要点**：**身过旺无泄**——转论"比劫重重"（破格）；或官杀反被合尽 → 可能转入"专旺格"讨论。
- **不满足则破**：身过旺无泄 → 降格或另论。

---

**四条同时满足**方为真建禄格。条件 1、2 为结构前提、条件 3 为贵格要件、条件 4 为降格防线。

## 建禄格的四大取用规则

### 规则一：建禄格用官——天干透出官星为贵、须财印相随、不可孤官无辅

#### 核心逻辑
- 官星透出时，其力量与食伤对比哪一者更强 → **优先取用神**的原则：
  - 若**官星作为主导力量** → 选官星为用神；
  - 若**食伤更有作用** → 食伤优先。
- 建禄格中若官星还透出天干 → 表明此八字格局**更为上乘**。

#### "财印相随、不可孤官无辅"的三种情形
**并非指财星和印星必须同时出现**，其实包含三种情形：
1. **只有财星相助**。
2. **只有印星相护**。
3. **财星与印星两者兼备**。

这三种情形的共同目的：**保护官星使其免受原命局及岁运中食伤的冲克**。

- 财星作用：在官星和食伤之间起**通关**作用，同时还可以**生官星**。
- 印星作用：**克制食伤**，从而保护官星。

#### 通用原理
- 身强 + 官星通根 + 印护 / 财生 → 古代多对应贵官、贵妇命造。
- 岁运喜财官之地；忌伤官运伤害用神官星。

#### 财官印三奇
- **天干同透财官印者称为三奇**——但必须官星**隔开**财星与印星、使**财印两不相伤**方成贵格。
- **三奇贵人格的天干顺序**：
  - 甲戊庚（天上三奇）。
  - 壬癸辛（地上三奇）。
  - 乙丙丁（人中三奇）。
  - **必须按特定顺序排列**。
- 地支也有重要限制——**不能出现刑冲或空亡**。
- 建禄格中若天干同时透出财官印、且财和印被官分隔 → 财就不会去克印——**也是一种三奇说法**；即使地支出现刑冲，整个命局依旧贵气四溢。
- **核心意义**：财官印在传统命理学里向来被视作一个人的"三宝"——各自代表不同的人生要素与运势走向、对命局影响深远。

### 规则二：建禄格用七杀——必须制伏

#### 正官 vs 七杀的关键区别
- **正官**：一般**不适合采用强行克制**的方式——若正官力量极为强大，优先考虑借助**印星来化解宣泄**正官的力量——因为正官代表相对温和的约束力量（社会秩序、道德规范）；以善意方式规劝和引导，适度时能培养**荣誉感与责任心**；对正官采取克制手段 → 是对规则的对抗、贵气会随之流失。
- **七杀**：**可以被克制**——七杀被视为凶神、不仅会对日元进行管束、还会对日元造成伤害、进而给整个命局带来不稳定因素；所以一旦在原命局中出现七杀，**无论命主是身强还是身弱、最先要考虑的就是如何将七杀制伏**。

#### 最理想的状态
- **身强时**：原命局中有**食伤来克制七杀**——最佳（其中**食神制杀**为最优）；印星对七杀也起到抑制作用。
- **身弱时**：原命局中最好有**印星化解七杀**——**七杀生印、印再生日元**、形成良性循环、增强命主力量。
- 当原命局条件并非如愿时：
  - **身强仅有印星** → 依靠印星化解七杀。
  - **身弱而唯一存在的制衡七杀的只有食伤** → 选用食伤克制七杀（虽进一步打破平衡，但受限于原命局、这也是不得已而为之；毕竟首要任务是削弱七杀对命主的威胁）。

#### 通用原理
- 食神合制七杀的元神（如合克偏财）→ 即便天干无印无食，也可变相制化。
- 岁运若再来财生杀而命中食神已制 → 无需担忧，格局稳。

### 规则三：建禄格官杀混杂时——必取清方为贵
- 一个命局**官杀混杂就没贵气**了——需通过**相合、相克**的方法**去掉正官或七杀中的一个** → 这样的命局也能显贵。
- **不论何者被去、存官制杀运喜食伤**。

#### 去留的选择
- 食神既可克制七杀也可合制正官 → 具体看**距离远近**：谁近则谁先受克 / 合。
- **原局自带去留配置者贵气最纯**；靠大运流年帮忙去留 → 效果打折。
- 伤官见官的"病"若有合绊伤官者（如乙庚合绊伤官乙木）→ 也可保住正官贵气。

### 规则四：建禄格用财——须带食伤
- 建禄格**单独用财**是这个格局所忌讳的——月令为比肩、力量非常大、会**直接克财**。
- 若命局不带食伤则**无法用财** → 有了食伤则可化解比肩克财之弊 → 形成"**比肩生食伤、食伤生财**"的流通之势。
- 具体还需结合财星与食伤的强弱判断：
  - **财星与食伤均强旺** → 喜**印星克制食伤**来平衡、但仍不喜比劫（比劫会进一步生旺食伤）。
  - **财星与食伤较弱** → 辅助财星和食伤、**且不喜印星克制食伤与比劫分夺财星**。
- 此外，若岁运再来官杀——因原命局食伤回克则无妨碍、但也并非福运。

### 规则五：建禄格用食伤——最宜走财运

#### 核心口诀
> **建禄格用食伤，无财官而用食伤，泄气太过，意为秀气，财运最宜，杀无不济，行印非极，透冠不美；若命中食伤太重，则财运固利而印亦不济矣。**

- 若建禄格且采用**食伤为用**——**走财运最为适宜**。
- 若再走食伤运即为**食伤泄秀** → 才华横溢的运势。
- 即便遇到七杀也并非忌讳——然而走印运就不吉利、天干透出官星也不是好的情况。
- 但若命中伤官食神力量太重、那么**走财运固然有利、而且印运也不再忌讳了**。

#### 木火通明的特殊结构
- 甲木建禄（月令寅）+ 丙火透干 → "**木火通明**"之象。
- 若再得子水为调候（非作用神）→ 形成"**食伤配印**"的精妙平衡。
- 逢南方火运 → 食伤泄秀、才华井喷。

## 判断维度

### 1. 用神选择
建禄格因身强，**身本身不需要扶**，重点在找**出口**：

| 结构 | 用神方向 | 特征 |
|---|---|---|
| 有**七杀**有制 | **七杀为用** | 有挑战 / 压力 → 带出事业心，典型贵格 |
| 有**正官**清透 | **正官为用** | 走仕途、管理岗、稳定事业 |
| 有**财星**通关 | **财为用** | 能赚钱、生活落地 |
| 有**食伤**秀气 | **食伤为用** | 才华型、表达型、技术型 |
| **四无依靠** | 印比重重 | 格局平平，做平凡人为主 |

### 2. 是否有官杀做事
- 建禄格最怕**无官杀**：身强无制 → 气无处去 → 容易倔、容易自负。
- 有官杀 + 有制 → **化杀为用**或**官为贵** → 格局上台阶。

### 3. 大运走向
- 建禄格身本强，**忌再走比劫 / 印运**（扶身无意义，反而壅塞）。
- **喜走财官食伤**：有出口才出成就。

## 取用神选择的核心思维
在分析八字时选用神优先从八字本身已有的字中筛选——倘若八字内部已经存在能够作为用神的字，那就**无需再寄望于大运或流年**里会出现所谓更好的字。

**用神的关键地位**：命局分析中选出来的用神极为关键、且极易受到伤害——这种伤害不拘泥于原命局自身、大运与流年所带来的影响同样不可小觑。一旦用神遭受伤害、整个命局的格局都会大打折扣、原本可能具备的良好运势与特征都会受到负面影响。如同精密运转的钟表、其中一个关键零件受损、整个钟表的运行都会出现偏差。

**所以用神在命局分析中有着举足轻重的地位——其是否完好直接关乎命局格局的优劣**。

## 共同特征（古代高官的启示）
古代高官的建禄格八字都存在一个共同的显著特点——**日元与官星出现了相合**的情况。

- 无论最终是否成功合化、只要日元将官星合住、就相当于把官星拉拢过来、**也可以理解为自己所用**。
- **对男性**：正官代表自己的上司——日元合官 → 通常意味着命主与上司关系融洽、上司会在工作发展等方面提供强有力的支持与协助、有助于事业顺遂前行。
- **对女性**：正官代表丈夫——日元合住正官 → 通常暗示丈夫对她关爱有加、在生活与事业等诸多方面给予极大的帮助、夫妻关系和睦且丈夫助力良多。

## 常见误区

### 1. "建禄 = 命一定好"
- 身强 ≠ 命好。
- 如果原局**无官无财无食伤** → 人有精气神，却没地方发挥 → 空有一身力气。
- 现实可能表现：有能力但没赛道、有脾气但没事业、不服气又没办法。

### 2. 把建禄当成阳刃
- **阳刃**：月支为**阳干的羊刃位**（如甲日见卯、丙戊日见午、庚日见酉、壬日见子）→ 比"禄"**更旺一格**。
- **建禄**：月支为**本气禄位** → 是"站稳"。
- 阳刃更刚烈、建禄更端正。

### 3. 以为身强就不需要帮扶
- 即便身强，走逆运时**也要学会收敛**。
- 建禄格的人容易觉得自己"扛得住"，结果越逆运越硬刚 → 反而摔得重。

### 4. 单用财而无食伤通关
- 建禄格用财最怕**比肩直接夺财**。
- 必须有食伤作为"**缓冲垫**"，让比肩之气先泄到食伤、再由食伤生财。

## 现实表现

### 性格
- **自立、自尊心强、独立性高**。
- **骨子里不依赖人**，不喜欢欠人情。
- **遇事扛得住**：抗压性强、遇事不慌。
- **不愿委曲求全**：关键时候宁折不弯。

### 与家人关系
- **兄弟姐妹关系复杂**：建禄多对应比劫 → 小时候家里资源分配敏感；长大后多独立自主。
- **父母资助有限**：身强主要靠自己打拼，不太能指望父母大额帮扶。
- 适合**早早分家、早早独立**的路径。

### 事业路径
- 适合**自负盈亏 / 独当一面**的岗位：
  - 创业、自由职业、技术专家、独立项目负责人。
- 在**需要靠自己拍板**的场景里最有优势。
- 在**需要卑躬屈膝 / 拼关系**的场景里吃亏。

### 钱财
- **不易一夜暴富**：靠稳扎稳打积累。
- **不易大破产**：身强守得住钱。
- 典型是**中产 / 小康偏稳**的路线。

## 破格情形
1. **月令被冲破**：如甲禄在寅，寅申冲 → 建禄受损。
2. **月令被合化成他物**：如丙禄在巳，巳申合化水 → 火禄变水 → 格局变质。
3. **身强无出口**：无官无财无食伤 → 空身强，格不成。
4. **用神无保护**：原局官为用神但无财印保护 → 伤官岁运一到就破格。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 财运 | **喜** → 给出口、给落地 |
| 官杀运 | **喜**（有制）→ 带来事业台阶 |
| 食伤运 | **喜** → 才华变现、表达立业 |
| 比劫运 | **忌** → 身更强、竞争更大、分财 |
| 印运 | **忌** → 旺上加旺，更倔、更闷 |

## 与相关格局的关系

| 格局 | 月令角色 | 日主性质 |
|---|---|---|
| **建禄格** | 日主之禄 | 身强，稳重挺拔 |
| **阳刃格（羊刃格）** | 日主之刃 | 身极强，刚烈 |
| **正官 / 七杀格** | 月令克日主 | 身受制，有外部目标 |
| **正 / 偏印格** | 月令生日主 | 有资源 / 保护 |
| **正 / 偏财格** | 日主克月令 | 重物质、重现实 |
| **食神 / 伤官格** | 日主生月令 | 重表达、重输出 |

## 核心判断总结
- 建禄格的四大取用规则不是死记硬套——命理格局千变万化、**能抓住以点带面的窍门**才算真的入门。
- **用神的保护情况**在分析八字时至关重要——不能只了解一个命局如何取用神、更要了解这个用神在本命局里是否缺乏其他五行的保护。
- **取用顺序**：官 → 杀 → 财 → 食伤——优先用官（财印相随）、次用杀（必制伏）、再用财（必带食伤）、最后用食伤（最宜走财）。
- **官杀混杂**必**取清方贵**——去一留一、存官制杀运喜食伤。
- 建禄格的核心不是"禄"本身，而是**身站稳之后的用法**。
  - 有出口 → 自成一格，独立一生。
  - 没出口 → 空有精力没处使，最怕**大运再不给路**。

## 一句话总结
- 建禄 = 地基打得牢。
- 地基牢以后的人生，看你**盖什么楼、盖多大**。
- 没有蓝图（财官食伤），再牢的地基也只是光秃秃的地基。
- 建禄格的本质是"**身站稳之后的用法**"——有出口则自成一格、独立一生；没出口则空有精力无处使、最怕大运再不给路。
