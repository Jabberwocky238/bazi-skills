# 阳刃格

## 定义
- **阳刃**：阳干日主在十二长生中"**帝旺**"之地对应的地支。
  - 甲木刃在卯、丙戊刃在午、庚金刃在酉、壬水刃在子。
  - 阴干无刃（或称"阴刃"但作用弱，古籍多不单立为格）。
- **阳刃格**：**月令为日主之刃位**的基础正格（亦称"羊刃格"）。
- 《子平真诠》十正格之一——与建禄格对照：
  - 建禄：月令为禄（临官之地，身自旺）。
  - 阳刃：月令为刃（帝旺之地，身过旺）。
- 阳刃格是**身极旺**的正格。

## 核心：刚烈过旺
- 阳刃 = 阳干之气"**旺极转厉**"之象。
- **禄**是站稳，**刃**是过盛——过盛则刚烈，刚烈则需制。
- 古籍论："**阳刃如刀，须有鞘**"——必得官杀相制方为贵格。

## 阳刃 vs 建禄

| 对比项 | 阳刃格 | 建禄格 |
|---|---|---|
| 月令性质 | 帝旺（过旺）| 临官（正旺）|
| 日主状态 | 身极旺、刚烈 | 身旺、稳重 |
| 取用必要 | **必需官杀制** | 官财食伤均可取用 |
| 气质 | 刚猛、易冲动 | 稳重、不动声色 |

## 五阳干的刃位
《子平真诠·论阳刃》原文："**甲禄在寅，刃在卯；丙戊禄在巳，刃在午；庚禄在申，刃在酉；壬禄在亥，刃在子**。"

| 日干 | 禄位 | 刃位 |
|---|---|---|
| 甲 | 寅 | **卯** |
| 丙、戊 | 巳 | **午** |
| 庚 | 申 | **酉** |
| 壬 | 亥 | **子** |

**注**：阴干（乙丁己辛癸）古籍多不立阳刃格——虽有"阴刃"之说（乙刃在辰、丁己刃在未、辛刃在戌、癸刃在丑），但作用显著弱于阳刃，实操中不作为基础正格。

## 成立条件（严格依《子平真诠·论阳刃》）

### 条件 1：日主必为阳干（甲、丙、戊、庚、壬）

- **古籍出处**（本条前提）：
  - 《子平真诠·论阳刃》："**阳刃者，阳干之刃也**。"
- **客观判据**：日主天干必须为甲、丙、戊、庚、壬五阳干之一。
- **不满足则破**：阴干日主 → 不入本格（阴刃另论，非基础正格）。

### 条件 2：月令为日主之刃位

- **古籍出处**（本条核心）：
  - 《子平真诠·论阳刃》："**月令逢刃，谓之阳刃格**。"
- **客观判据**：月令地支为条件 1 所述对应的刃位（甲卯、丙戊午、庚酉、壬子）。
- **不满足则破**：月令非刃 → 不入本格。

### 条件 3：必有官杀制刃（阳刃格的必要条件）

- **古籍出处**（本条最核心、本格铁律）：
  - 《子平真诠·论阳刃》："**阳刃用官则喜露，用杀则不忌——刃以杀制为贵**。"
  - 《子平真诠·论阳刃》："**阳刃无官杀，凶顽之辈**。"
- **客观判据**：
  - ① 正官透干通根；或
  - ② 七杀透干通根。
  - 有其一制刃即可。
- **不满足则破**：阳刃无官杀 → 沈氏明判"**凶顽之辈**"——身极旺而无制，气无出口，易走极端。

### 条件 4：制刃之官杀有根有力，能与刃相抗

- **古籍出处**：
  - 《子平真诠·论阳刃》："**刃杀相济，声威赫奕**——刃强杀强则贵。"
- **客观判据**：
  - 制刃之官或杀须在地支本气或中气有根，力量与阳刃大致对等或略强。
- **不满足则破**：官杀虚透无根 → 制不住刃 → 阳刃格变"阳刃独旺"的破局。

### 条件 5：原局无伤官克官（若用官制刃时）

- **古籍出处**：
  - 《子平真诠·论阳刃》："**阳刃用官，忌伤官**——伤官克官，制刃之物反被克。"
- **客观判据**：若取正官制刃，原局无伤官紧贴克官；若有伤官须有印制之。
- **不满足则破**：伤官克官 → 制刃链断 → 转入"阳刃无制"或"伤官见官"讨论。

### 条件 6（格局升级）：身杀两停 → 升为"阳刃驾杀"

- **古籍出处**：
  - 《子平真诠·论阳刃》："**羊刃无杀，不足为贵；七杀无刃，不足为威**。"
- **客观判据**：若用杀制刃且刃杀力量势均 → 转入"**阳刃驾杀**"贵格讨论（见 `羊刃驾杀.md`）。

---

**前五条必须同时满足**方为真阳刃格；条件 6 为升级路径。阳刃格的本质是"**身极旺必须用官杀制之**"——无制之阳刃，沈氏判为"凶顽"。

## 判断维度

### 1. 制刃之物
- **用正官制**：气质偏文——管理、体制、规范型领导。
- **用七杀制**：气质偏武——军警、强硬型企业家。
- **刃杀两停**：**阳刃驾杀**的贵格（最高境界）。

### 2. 大运流年
- **官杀运**：制刃加强 → 升职、权柄到手。
- **食伤运**：制住官杀 → 破格（除非有印救）。
- **财运**：身旺能任财 → 财运亨通。
- **比劫 / 印运**：身更旺 → 需要更强制刃，反而易失衡。
- **刃运（如本命刃字再临）**：刃气加倍 → 需格外警觉冲动事件。

### 3. 阳刃的其他位置
- **月刃**：标准阳刃格。
- **日支阳刃**（坐刃）：自坐刚烈——脾气大、婚姻多波折。
- **时支阳刃**：晚年或子女刚烈。

## 常见误区

### 1. "带阳刃 = 一定凶"
- 阳刃只有"**无制 + 身过旺**"时才真凶。
- 有官杀制 + 身能任 → 阳刃反而**贵气过人**（将帅 / 强人）。

### 2. 把阳刃等同"羊刃煞"
- 阳刃格（月令刃位 → 正格）与羊刃神煞有重叠但不完全相同。
- 格局层面强调"月令关系"，神煞层面强调"刃气标记"——实操中两者常并论。

### 3. 忽视阳刃格的"必须要制"
- 阳刃格与其他正格的最大区别：**必需官杀制刃**，否则纯破格。
- 不像其他格局有多条出路，阳刃格的出路**只有一条**：有官杀 + 官杀能扛住刃。

## 现实表现

### 性格
- **刚烈、果敢、有主见**。
- **身体健、精力旺**。
- **不怕事、遇强更强**。
- **脾气直、不善掩饰**。
- **行动力强、有领袖气**。

### 职业路径（需配官杀）
- **军警、武职**：刃 + 杀的典型路径。
- **司法、执法、监察**：刃 + 官的体制路径。
- **竞技体育、高压决策岗**。
- **企业家、强势管理**：身旺能担事业。
- **外科医生、急救**：刀气 + 决断。

### 无制阳刃的典型
- 脾气暴躁、不懂约束。
- 易与人冲突、官非是非多。
- 冲动决策、冒险失控。
- 现代版："**愤青型**"或"**易怒型**"人格。

## 核心判断
- 阳刃格的关键是"**制**"字——有官杀制则贵，无制则凶。
- 识别意义：这类命造**天生要走"刚性路径"**——有对抗、有压力、有制约的环境反而最合其性。
- 硬放到柔和岗位 → 刃气无处发作，反而生出事端。

## 一句话总结
- 阳刃 = **身极旺的双刃剑**。
- 有官杀这把鞘 → 是利器 / 是将军 / 是英雄。
- 无鞘则伤己伤人 —— 阳刃格必须给自己找"**对手**"，否则对手就是自己。
