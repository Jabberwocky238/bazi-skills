# 金水对（总论）

## 性质：相生两气对（金→水）
- **五行关系**：金生水（印→身 或 身→食伤的顺流）。
- **属"印生两神成象格"之一**——《滴天髓》五种相生组合中的"金水相生格"。
- 相生两气双清，若气纯无杂，属贵格。

## 两气成象通则（适用于本对）

### 成立条件（严格依《滴天髓》两气双清论）
1. **命局"两气双清"**——只见金水两种五行，其他三种基本不现。
2. **金水相生**——金生水，顺流无克。
3. **原局无财官**（木为财、火为官）——一见财官则破清破局。
4. **日主属金或水**，且不过弱。

### 具体形式（金水相生格）
- **庚辛金日主 + 壬癸水食伤**：身→食伤顺流（秀气外放）。
- **壬癸水日主 + 庚辛金印绶**：印→身顺流（学识内化）。

### 气质
- **清冷 + 智慧** —— 聪明、清秀、思辨力强、洞察深。
- 典型特征：**过寒而清**——需注意火调候。

### 典型职业
- 研究、分析、法律、金融、科技、高端咨询、战略谋士、学者。

### 禀赋与缺失
- **禀赋**：理性、清透、思辨、学问。
- **缺失**：
  - 缺火 → 易抑郁、缺乏行动力（"志而无成"）。
  - 缺土 → 根基浮薄、易漂泊。

### 喜忌
- **喜**：金（加强本气）、水（加强本气）、木（通关泄秀，金→水→木成生生不息）。
- **忌**：
  - **火**（克金破局）——大忌，破双清。
  - **土**（克水破局）——大忌。
  - 火土同现 → 格毁。
- **调候之需**：本对过寒，**微火调候不忌**（暖局而不克金水之气势）——此为特殊规则。

### 古籍背书
- 《滴天髓·源流》："**两气双清，其象最妙**。"
- 《穷通宝鉴·庚金章》："**庚金逢壬水，为金白水清**。"
- 《滴天髓》："**金水相生，清而不杂者贵**。"

## 本对下属格局
- **金寒水冷**：金水过寒、无火调候的病象。详见 `金寒水冷/`。
- **金白水清**：金水双清、秀气外显的贵格。详见 `金白水清/`。

## 一句话总结
- 金水对 = **金生水之相生双清**——聪明、清冷、有学问。
- 贵在"清" → 忌火土之克、喜木之泄、微火暖局不忌。
- 金白水清为贵、金寒水冷为病——同为金水对，清浊之分在于**火的调候**。
