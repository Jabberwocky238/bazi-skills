# 正官格

## 定义
- **正官**：与日主阴阳相反之克——阳日主见阴克我之五行为正官，阴日主见阳克我之五行为正官。
  - 例：甲日见辛金（阴金）为正官；乙日见庚金（阳金）为正官。
- **正官格**：**月令取正官为用神**的基础正格。
- 《子平真诠》十正格之首——古籍视为**最端方、最正统**的贵格。

## 核心：柔克之贵
- 正官 = 规则、秩序、身份、权威。
- 相对七杀（刚克、压迫性）：正官是**柔克**——克而不伤，约束而不压迫。
- **身能任官** → 正官可为用神 → 走**体制内、正规系统、声望身份**之路。

## 正官 vs 七杀

| 对比项 | 正官 | 七杀 |
|---|---|---|
| 阴阳 | 与日主相反 | 与日主相同 |
| 克性 | 柔克、讲规矩 | 刚克、压迫性 |
| 典型路径 | 体制、文职、管理 | 军警、强势、危难 |
| 气质 | 端方、循规 | 刚烈、破局 |

## 成立条件（严格依《子平真诠·论正官》）

### 条件 1：月令地支本气为正官，或正官透干且月令有根

- **古籍出处**：
  - 《子平真诠·论正官》："**取用之法，月令为先**。"
  - 《子平真诠·论正官》："官者，阳见阴克、阴见阳克。如甲日见辛之类是也。"
- **客观判据**：
  - ① 月令地支本气为正官；或
  - ② 月令地支中气 / 余气为正官，且正官透于天干。
- **不满足则破**：月令正官不显 → 当以月令本气另论他格。

### 条件 2：正官清纯，不混七杀

- **古籍出处**（本条最核心）：
  - 《子平真诠·论正官》："**正官最忌杂以七杀；官杀混杂，格局乱矣**。"
- **客观判据**：
  - 天干无七杀透出；若地支藏七杀一位，须有食神制之或合去（合杀留官）。
- **不满足则破**：官杀混杂 → 转入"官杀混杂"病格讨论。

### 条件 3：原局无伤官紧贴克正官

- **古籍出处**（铁律）：
  - 《子平真诠·论伤官》："**伤官见官，为祸百端**。"
  - 《子平真诠·论正官》："**正官最忌伤官**。"
- **客观判据**：
  - 原局天干无伤官与正官紧贴；若有伤官，须有印制（此时成"伤官佩印 + 官"复合结构）。
- **不满足则破**：伤官贴官无救 → 破格转入"伤官见官"讨论。

### 条件 4：日主身中偏强，能承载官气

- **古籍出处**：
  - 《子平真诠·论正官》："**正官取用，喜身强能任**。"
- **客观判据**：
  - 日主于月令为比劫 / 印星本气，或他柱两位以上比劫 / 印星通根（身中偏强）。
- **不满足则破**：日主极弱 → 不任官 → 官反为克身之病，须印化之。

### 条件 5：官星有根，不虚透

- **古籍出处**：
  - 《子平真诠·论正官》："**官星宜透干有根，方为真官**。"
- **客观判据**：
  - 正官透于天干且在地支有本气或中气根至少一位。
- **不满足则破**：官虚透无根 → 无真官之气，格不成。

### 条件 6（辅佐条件）：有财生官或印护官

- **古籍出处**：
  - 《子平真诠·论正官》："**凡正官为用者，喜印护官，喜财生官**。"
- **客观判据**：
  - 财星透干通根，位于官星之前（财→官顺流），或
  - 印星透干通根，位于官星之后（官→印化身）。
- **非必要**：不满足不破格，但格品降——无财无印的正官格属"纯正官"，贵气略薄。
- **最佳结构**：财、官、印三者俱全、链条顺流 → 转入"**财官印全**"贵格讨论。

---

**前五条必须同时满足**方为真正官格；条件 6 关乎格品。《子平真诠》明言："**正官喜印护官、喜财生官**"——这是正官格的最佳兑现路径。

## 判断维度

### 1. 身强身弱之别
- **身强用官**：直接取官为用，官印或财官配合。
- **身弱见官**：须印化官（转入"官印相生"），或比劫帮身（则官太弱时需财生官以使官有力再化）。

### 2. 官的来源
- **月令正官**：最正宗——"月令正官格"为本格的经典结构。
- **透干正官**：官透年 / 时干但月令非官 → 次级，按组合论。

### 3. 大运流年
- **印运**：印化官 → 喜。
- **财运**：财生官 → 喜（身强前提下）；若财破印则降格。
- **比劫运**：视身强身弱 — 身弱喜、身强忌（分官）。
- **食伤运**：伤官克官 → 忌（无印救时）。
- **官运**：加官进爵——升职升学。

## 常见误区

### 1. "有正官 = 一定有官位"
- 正官是"**规则与身份的底牌**"，不保证实际官位。
- 现代含义：体制内、大公司、规范行业的**正式身份 + 逐级上升**。
- 需配大运、用神、身承载综合看。

### 2. 混淆正官与七杀
- 阳日见阴克 → 正官；阳日见阳克 → 七杀。
- 例：甲日见辛（阴金）为正官，甲日见庚（阳金）为七杀。
- 错认克我之物属性 → 用法全盘皆错。

### 3. 忽视"伤官见官"的致命性
- 正官格最大破格因素之一。
- 命中正官清透而又见伤官贴克 → 必有是非、职场冲突、权威反目。

## 现实表现

### 性格
- **端方、循规、有分寸**。
- **尊重权威、讲究程序**。
- **责任感强、诚信为本**。
- **不轻易越界**。

### 职业路径
- **体制内**：公务员、事业单位、国企正职。
- **规范行业**：银行、教育、医疗（管理方向）、法律、会计等持证专业。
- **大型机构**：跨国企业、政府平台、央企。
- **需要"身份 / 资质 / 级别"的岗位**。

### 人生节奏
- 稳步上升型，不走捷径。
- **中年稳步进阶**，不会突然暴富。
- 晚年享受累积的身份与地位。

## 核心判断
- 正官格的核心是"**身能任官**"——官须有人撑得起才显贵气。
- 识别意义：这类命造**走正规路径最顺**，硬放到高风险江湖赛道反而埋没。

## 一句话总结
- 正官 = **规则托起的那个人**。
- 有规则保护、有身份认可、有秩序支撑——这是正官格一生走的路。
- 别硬打破规则，**规则就是这类人的朋友**。
