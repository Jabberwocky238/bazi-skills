# 流月流日

## 定义
- **流月** = 当前**月份的干支**——一年 12 个流月，每月主一个气场。
- **流日** = 当前**日期的干支**——每天一对干支，直接影响当日吉凶。
- 两者是**大运与流年的细化**——大运定十年、流年定一年、流月定一月、流日定一天。

## 流月的推算

### 基本原理：年上起月法（五虎遁元诀）
- 天干 10 个、月份 12 个，月柱天干**随年柱天干而变**。
- **地支固定**：寅月（正月）、卯月（二月）、辰月（三月）……依十二月建顺排。
- **天干变化**：由年柱天干合化五行推算。

### 五虎遁元口诀

| 年干 | 正月（寅月）天干 |
| --- | --- |
| **甲 / 己** 年 | 丙寅起首 |
| **乙 / 庚** 年 | 戊寅起首 |
| **丙 / 辛** 年 | 庚寅起首 |
| **丁 / 壬** 年 | 壬寅起首 |
| **戊 / 癸** 年 | 甲寅起首 |

### 推算步骤
1. 确定**年干**。
2. 查表或背诵口诀，得到**寅月**（正月）的天干。
3. 从寅月起，沿六十甲子顺排——**二月卯、三月辰、四月巳**……依次推出。

### 推算逻辑原理
- **第一步**：年干五合化五行（甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火）。
- **第二步**：找**生**该五行的五行（火生土、土生金、金生水、水生木、木生火）。
- **第三步**：寅为阳，取**阳干**。
- **第四步**：从寅月起顺排。

### 举例
- 2026 年为**丙午**年，丙辛合水，**金生水** → 寅月天干为阳金 **庚**。
- 2026 年正月为**庚寅**、二月**辛卯**、三月**壬辰**、四月**癸巳**、五月**甲午**……依次顺推。

## 流日的推算

### 流日 = 每日的干支
- 规则比流月繁琐得多，因为**月份有 28/29/30/31 天之别**，无法用简单口诀覆盖。
- 古法有口诀可推，但**工程量大**。
- 现代直接查**万年历**或排盘软件。

### 日柱的实际用法
- 大多数命理师**不专门背流日**——直接看万年历。
- 但对**择日、应事**的精细推算极为重要。

## 流月对流年的细化

### 一年的气机不是平均分布的
- 流年干支笼罩全年，但**12 个月中，每个月强弱不同**。
- **某些月份**：流月与流年相生相合 → **该月气顺**。
- **某些月份**：流月与流年相冲相克 → **该月应事集中**。

### 流月与流年干支的互动

| 关系 | 体感 | 典型应事 |
| --- | --- | --- |
| **流月助流年用神** | 气顺 | 好事发生、贵人显现 |
| **流月助流年忌神** | 气乱 | 小人、破财、情绪波动 |
| **流月冲流年** | 震动 | 突发事件、大变动 |
| **流月合流年** | 纠缠 | 拉扯、合作、情感纠葛 |

### 例子
- 2026 丙午年，原局以水为用。
- **寅月（庚寅）**：金水渐生 → **偏顺**。
- **午月（甲午）**：火势最旺、水被煎熬 → **需防燥动、心病、冲突**。
- **亥子月**：水气回暖 → **可谋大事、可调整**。

## 大运 + 流年 + 流月 + 流日 四层叠加

### 叠加的层级结构

| 层级 | 覆盖范围 | 作用 |
| --- | --- | --- |
| **大运** | 10 年 | 人生赛道、风向 |
| **流年** | 1 年 | 当年的主题 |
| **流月** | 1 月 | 当月的具体气机 |
| **流日** | 1 天 | 具体某天的吉凶 |

### 四层组合的判断逻辑
1. **大运+流年** 定"**这一年的基调**"——顺或逆、升或降。
2. **流月** 定"**哪几个月发生什么**"——把一年的气机切片。
3. **流日** 定"**哪一天动手**"——择日落实。

### 叠加共振 = 大事应期
- 当大运、流年、流月**三者共振**——例如三者共同冲原局某字 → **重大事件必发生在这几个月**。
- 若再叠加**流日共振** → 精确到"**某月某日**"。
- 这就是**应期推演**的核心方法。

### 案例推演
- 某命局**日支子水**为用神。
- 大运走**午火**，冲子水 → 十年婚姻不稳的大趋势。
- 流年**午年** → 当年婚姻最危险。
- 流月**午月** → 当年五六月最危险。
- 流日**午日** → 某月某日事件可能落地。
- 四层共振 → **大概率当年午月午日应婚姻变故**。

## 现实应用——大事择日

### 传统择日的核心思想
- 重大事件——**开业、搬家、结婚、签约、手术**——选择**流月流日与本命局和气**的日子。
- 忌用流日与**本命日柱冲克**的日子。
- 忌用流月流日与**当年流年冲克**的日子。

### 择日的简化原则

| 用途 | 选择原则 |
| --- | --- |
| **喜事（婚、开业、搬家）** | 流日生扶日主或用神；避开日冲 |
| **远行、签约** | 流日五行与当事人用神同频；避开驿马冲击日 |
| **手术、看病** | 流日对病位五行有利；避开与病脏器所对五行相冲 |
| **见重要客户、谈判** | 流日对"口舌"五行（金水）有利；避开伤官见官日 |

### 择日的误区
- **迷信"黄道吉日"**——不结合本命局，吉日也可能是忌日。
- **只看流日忽略流月** → 流月基调不对，流日再好也白搭。
- **只择开始日不看整体** → 婚姻要看整个流年，搬家要看整个大运。

## 一句话总结
流月、流日是大运流年的"**放大镜**"。
- 大运是十年长镜头、流年是一年中镜头、**流月是一月特写、流日是一天定格**。
- 四层共振 → 大事必发；一层不顺 → 小事可避。
- **真正的应期判断，从不只是看一个字，而是看四层气机的共鸣点。**
