# 润下清华（润下格贵格）

> **定位**：润下格的**贵格升级态**——润下（水专旺）+ 木泄秀 / 金生水 → 成就"**智慧兼秀气**"的顶级水气贵格。
> 本文是 `润下格/成立条件.md` 中"**条件 6（助力条件）**"的完整专论。

## 定义
- **润下清华**：润下格（水专旺）+ **木泄秀气**（水木清华） 或 **金生水源**（金白水清）→ 成就"**智慧清贵**""**秀气外显**"的水气贵格。
- "**清华**"二字：
  - **清** = 水之清澈（源自润下本性）；
  - **华** = 木泄水秀而成花（水生木木生华）。
- 古籍《穷通宝鉴》："**壬癸见甲乙，水木清华**"——润下之水得木而显秀。

## 与普通润下格的区别

| 对比项 | 纯润下格 | 润下清华（本条）|
|---|---|---|
| 水气 | 满盘水气（三合或三会水局）| 满盘水气 + **木 / 金点缀** |
| 秀气 | 缺——偏幽深内敛 | 有——水生木成秀，金生水成源 |
| 气质 | 深沉智慧但偏孤寂 | 智慧之上带清秀外放 |
| 贵气 | 贵格（纯专旺）| **贵格之贵**（专旺 + 秀气双全）|
| 代表行业 | 深度研究 / 幽深学问 | **学者 + 表达 + 才华**（水木清华） / **精英思辨 + 源不断**（金白水清）|
| 气象 | 深潭之水 | **清溪活水**（水中生木成花，或金泉润水）|

## 两种清华路径

### 路径一：**水木清华**（以木泄水为秀）
- **结构**：润下（水专旺）+ 一至两位木（甲乙天干 或 寅卯地支）。
- **古籍**：《穷通宝鉴》"**壬癸见甲乙，水木清华**"。
- **气机**：水生木 → 水气由内而外开花 → 智慧转为**才华与创造**。
- **气质**：**清秀、灵动、有创意**——智慧不再沉于深处，而是**长出美丽的枝叶**。
- **典型职业**：
  - 学者 + 作家 / 学者 + 艺术家 → 学问能外化为作品。
  - 创意型研究、设计、写作、策划。
  - 文化传播（学者 + 媒体）。

### 路径二：**金白水清**（以金生水为源）
- **结构**：润下（水专旺）+ 一至两位金（庚辛天干 或 申酉地支）。
- **古籍**：《穷通宝鉴》"**庚金逢壬水，为金白水清**"。
- **气机**：金生水 → 水源源源不断 → 智慧**清澈通透且持久**。
- **气质**：**思辨力强、理性、有洞察**——金为决断、水为智慧，二者合为"精英思辨"。
- **典型职业**：
  - 分析师、战略家、高端咨询。
  - 法律、金融、科研——需要**清晰判断 + 源源思考**的领域。
  - 哲学家、理论工作者。

### 两路同时具备——双清贵格
- 若原局同时带金与木 → **金白水清 + 水木清华双叠加** → 顶级复合贵格。
- 气机链条：**金 → 水 → 木** → 三行顺生，智慧之源不竭、秀气之花长开。
- 此为润下格的**最高形态**。

## 成立条件（严格依《穷通宝鉴》《渊海子平》）

### 条件 1：先满足润下格所有成立条件
- 日主为壬或癸水。
- 地支会成水局（申子辰 三合 或 亥子丑 三会）。
- 天干多透水。
- 原局无戊己土透干克水。
- 原局无重火蒸水。
- 详见 [`成立条件.md`](./成立条件.md) 前五条。

### 条件 2：有木泄秀（水木清华路径）或 有金生水（金白水清路径）

- **古籍出处**：
  - 《穷通宝鉴》："**壬癸见甲乙，水木清华**。"
  - 《穷通宝鉴》："**庚金逢壬水，为金白水清**。"
- **客观判据**（两条任一或同时成立）：
  - ① **木泄路径**：原局有甲乙木透干，或地支寅卯木（一至两位），且位置紧贴水气以形成顺流。
  - ② **金源路径**：原局有庚辛金透干，或地支申酉金（一至两位），且位置紧贴水气以生之。
- **不满足则破**：
  - 无木无金 → 只成纯润下格，偏幽深而不秀。
  - 木过多（三位以上成木局）→ 水泄过度 → 转入"水多木漂"讨论。
  - 金过多（三位以上成金局）→ 喧宾夺主 → 格变金水两气成象（金水对）。

### 条件 3：木 / 金 不过旺，维持"润下为主"的基调

- **古籍出处**：
  - 《滴天髓·从象》："一气成象，**微泄秀气方妙**。"——关键在"微"。
- **客观判据**：
  - 木：一至两位透干或藏支为"微"；三位以上为"过"。
  - 金：一至两位透干或藏支为"微"；三位以上为"过"。
  - 月令仍为水（申子 / 亥子），若月令变金或木则格变。
- **不满足则破**：木或金过旺 → 本末倒置 → 不再是润下清华，降为两气成象对（如金水对、木水对）。

### 条件 4：无戊己土透干克水（与润下格本体条件一致）

- 润下清华仍以"无土克水"为**铁律**。
- 若有土透克水 → 格破，连润下都不成，遑论清华。

---

**四条同时满足**方为真润下清华。核心在**"微泄秀气"** —— 水为主角、木 / 金为配角；配角点缀成秀，不可喧宾夺主。

## 喜忌总则（相对纯润下格的变化）

| 类别 | 纯润下格 | 润下清华 |
|---|---|---|
| **水运** | 大喜 | 喜（维持本气）|
| **金运** | 喜（生水）| **大喜**（金白水清兑现）|
| **木运** | 视情况 | **大喜**（水木清华兑现）|
| **土运** | 大忌 | 大忌（不变）|
| **火运** | 忌（蒸水）| 忌 |

## 现实表现

### 气质（三种层次）
- **纯润下**：深沉 + 幽静 + 智慧内敛（学者型但不外放）。
- **水木清华**：智慧 + 创造 + 清秀外显（学者型 + 作家 / 艺术家）。
- **金白水清**：智慧 + 理性 + 源源思辨（战略家型、思想家）。
- **三清复合**（金白水清 + 水木清华）：**智慧、思辨、创造三合** → 顶级贵格。

### 典型人生节奏
- **早年**：显才华、静水深流、人称"**聪明**"。
- **中年**：学问 / 作品 / 思想开始外显——从沉默寡言转为**有影响力的思考者**。
- **晚年**：作品 / 学说 / 流派成形，桃李满天下、思想传世。

### 人物画像
- 学者型大家（钱钟书型——水木清华）。
- 战略思想家（谋士型——金白水清）。
- 诗人 / 画家 / 深度创作者（水木清华）。
- 数学家 / 哲学家 / 理论家（金白水清）。

## 古籍背书

- **《穷通宝鉴》**：
  - "**壬癸见甲乙，水木清华**。"
  - "**庚金逢壬水，为金白水清**。"
- **《渊海子平·润下格》**：
  - "年月时干宜见壬癸，**有庚辛金相生更妙**。"（暗合金白水清）
- **《滴天髓·从象》**：
  - "**一气成象，微泄秀气方妙**。"——专旺格的通则。
- **《滴天髓·源流》**：
  - "**生生不息，流通有情**。"——清华 = 生生不息的具体体现。

## 与相关格局的关系

| 格局 | 关系 |
|---|---|
| **润下格**（本格的基础）| 润下清华是润下格的**升级态** |
| **金水对 / 金白水清**（两气成象）| `../../../两气成象/金水对/金白水清/`——名近但结构不同：**两气成象的金白水清**是金水两气双清，**润下清华的金白水清**是水专旺 + 金点缀 |
| **木水对 / 水木清华**（两气成象）| `../../../两气成象/木水对/水木清华/`——同理，两气成象的水木清华是水木两气，**润下清华的水木清华**是水专旺 + 木点缀 |
| **从革格**（金专旺）| 反面——金独旺而非水独旺 |
| **水多木漂**（病格）| 水过旺克木——润下清华若木过少反被水漂即成此病 |
| **稼穑毓秀**（稼穑格贵格）| 平行——土专旺 + 金秀；润下清华为水专旺 + 木 / 金秀 |

## 润下清华 vs 两气成象的"清华"（易混淆）

| 对比项 | 润下清华（本条，专旺格贵）| 金白水清 / 水木清华（两气成象）|
|---|---|---|
| **所在分类** | 专旺格（水一气独旺）| 两气成象（金水对 / 木水对）|
| **水气** | 四柱满盘水 | 水为两气之一（非满盘）|
| **配气** | 点缀（一至两位金 / 木）| 与水力量相当的两气之一 |
| **气象** | 深水生秀木、金泉涌水源 | 两气双清对立 |
| **贵气来源** | 水专旺 + 点缀吐秀 | 两气相生双清 |
| **详见** | 本文 | `../../../两气成象/金水对/金白水清/` 等 |

## 一句话总结
- 润下清华 = **深潭生秀木**——深水养出美丽的东西（文 / 才华 / 思想）。
- 纯润下是"深"——智慧沉潜；润下清华是"**深而秀**"——智慧开花。
- 关键在"**主次有序**"——水是主、木 / 金是秀；水专旺生秀则贵，木 / 金过反客则降。
- 三清复合（金 + 木 + 水）→ 水气顶级贵格的极致形态。
