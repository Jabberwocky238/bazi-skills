# 伤官见官

## 构成条件（病象 / 破格，严格依《子平真诠》《滴天髓》）

> **注**：本条为"破格 / 病象"，非"成格"。故用"构成条件"而非"成立条件"——当以下条件**全部满足**时，方构成真正的伤官见官；任一条不满足即不构成或有救应。

### 条件 1：伤官与正官同时存在，至少一方透干

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官见官，为祸百端**。"
- **客观判据**（三种之一）：
  - ① 伤官透干 + 正官地支本气；
  - ② 正官透干 + 伤官地支本气；
  - ③ 两者俱透干。
- **不构成**：两者皆仅藏支一位余气 → 气微不显，不发病。

### 条件 2：两者力量均有显现（非一强一无、非一方伤尽）

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官伤尽，方为贵；伤官不尽，祸端百出**。"
- **客观判据**：
  - 伤官与正官均有天干显示或地支本气根至少一位，力量大致对等或伤官略强于官。
- **不构成**：伤官极旺而正官被冲去 / 合去 / 制尽 → 沈氏"伤尽反贵"——此时不构成见官，反为"伤官伤尽"之格。

### 条件 3：两者柱位相邻，克制可直接发生

- **古籍出处**：
  - 《滴天髓·通神论》："**贴身者力重，遥隔者力轻**。"
- **客观判据**：
  - 伤官与正官分居相邻柱（月 / 日、日 / 时）或同一柱的干与支。
- **不构成**：伤官在年柱、正官在时柱中间遥隔 → 只构成"远伤"，作用轻微，不成真见官。

### 条件 4：原局无救应（无印制伤 / 无财通关 / 无合化解）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论伤官》："**若夫伤官见官，须辨其有救无救。有救则吉，无救则凶**。"
  - 《滴天髓·伤官》："伤官见官，祸患百出；**佩印则化，生财则泄，合杀则和**。"
  - 《渊海子平·伤官不可见官诀》："**伤官不可例言凶，有制还他衣禄丰**。"
- **客观判据**（三条救应任一成立即不构成真见官）：
  - ① **佩印救**：有印星透干通根且紧贴伤官（转入"伤官佩印"）；
  - ② **生财救**：有财星透干通根且位于伤官与正官之间（伤→财→官，转入"伤官生财"进阶）；
  - ③ **合杀 / 合官救**：伤官与正官或七杀构成天干五合且合得住（转入"伤官合杀"）。
- **构成真见官**：三条救应全无 → 沈氏"无救则凶"，真破格。

---

**四条同时满足**方为真伤官见官。其中条件 4"无救"为最关键的破格判据——《子平真诠》明文"有救则吉"，故一见伤官 + 官即断凶的简单化论法，实与沈氏原旨相违。
