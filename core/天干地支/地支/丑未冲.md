# 丑未冲

## 定义

- **丑未冲**：地支六冲之一。
- **丑**：阴土，水之墓库，藏**己土（本气）、癸水（中气）、辛金（余气）**。
- **未**：阴土，木之墓库，藏**己土（本气）、乙木（中气）、丁火（余气）**。
- 本质：**两土墓库相冲**，本气同为己土，越冲越旺；藏干中**癸水对丁火、辛金对乙木**互相损伤，中气、余气尽散。
- 力量定位：六冲中属**墓库之冲**，仅次于辰戌冲，主**开库引动、变动渐次显现**。

## 核心原理

### 墓库相冲的特殊性
- 两支本气皆为**己土**（湿土 vs 燥土），非真克，实为**混杂合并**——本气土不减反增。
- 真正相冲在藏干之间：
  - **中气冲**：丑癸水 ↔ 未丁火（水火相战）。
  - **余气冲**：丑辛金 ↔ 未乙木（金木相战）。
- 藏干中气、余气**同时损耗**，中气受损尤重，未土中余气伤最深。

### 开库见藏干
- 丑库被开 → **癸水、辛金**之气外泄。
- 未库被开 → **乙木、丁火**之气外泄。
- 原本入库（藏而不显）的财、官、印、食之气被**引动而出**，命局此前稳定之结构被搅动。

## 冲的方向与力量

| 维度 | 表现 |
| --- | --- |
| 本气 | 己土 ↔ 己土：同气合并，土性更旺 |
| 中气 | 癸水 ↔ 丁火：水火相战，水制火 |
| 余气 | 辛金 ↔ 乙木：金木相战，金克木 |
| 综合受伤 | **未土余气乙木丁火**损伤更重 |

关键判断：若原局**火土旺**，则未土占优，丑中水气外泄被蒸散；若原局**水木旺**，则丑土得势，未中木火被截制。

## 四柱宫位应事

### 年月相冲
- 外部环境之变：**祖业田宅**动摇、父辈家业难以传承。
- 为吉时：祖上余荫转为动产变现；为凶时：少年家境起伏、求学受阻。

### 年日相冲
- 祖根与自身**不相应**，多主远离故土发展。
- 男命婆媳（母妻）之间**田产财物**之争较显。

### 月日相冲
- 青中年期**家庭与工作环境同步变动**；
- 办公地点或职务多次更迭，常伴随**购置或出售不动产**。

### 月时相冲
- 工作大环境与晚年落脚之所错位；
- 子女教育、居所安置多反复。

### 日时相冲
- 自身与子女、配偶与子女之间的田宅、财物纠纷；
- 晚年健康易生脾胃、皮肤慢性病。

## 现实象意

- **事业**：工作环境、办公地点、职务多变动；丑未辰戌为"**脚星**"，主**住宅搬迁、购置房产**之信息。
- **财富**：丑中藏辛金、未中藏乙木——若为财星则冲动之年财库开启，或进或出，常为大进大出。
- **家庭**：女命逢此冲，夫星易有外遇或心思外移；男命家族田产易生纠纷。
- **性格**：内向不善言辞；常有**宗教、玄学、神秘文化**之倾向（丑属道教、未属佛教之传统取象）。
- **吉用**：相冲为喜用引动时，可得田宅、晋升、转型机遇。

## 健康对应

| 部位 | 原理 | 常见症 |
| --- | --- | --- |
| 脾胃 | 丑未皆属土，**土主脾胃** | 腹胀、消化不良、慢性胃炎 |
| 皮肤 | 土主肌肤 | 湿疹、皮肤敏感 |
| 痰湿 | 丑中癸水被冲外泄 | 痰喘、湿滞 |
| 肾泌尿 | 土旺克水（忌神时） | 肾功能、泌尿系统不适 |
| 呼吸 | 辛金受损 | 肺部、呼吸系统慢性问题 |

判断要点：原局**土旺为忌**时，冲年最易引动上述慢性病灶；土弱为喜时，冲反引动阳气，常无大碍。

## 化解思路

1. **合去一方**：
   - 取**子水合丑**（子丑合土），使丑被绊住，冲势缓解。
   - 取**午火合未**（午未合火/土），使未被绊住。
   - 取**三合局**：巳酉丑合金或亥卯未合木，拉走一支。
2. **通关调和**：
   - 以**金**泄土生水（丑未→金→水），缓土之壅塞。
   - 以**木**疏土（天干透甲乙），引土气向财官流通。
3. **土旺用法**：若命局从土格，丑未相冲反加强土气，为喜；需岁运不破土根。
4. **身体层**：脾胃慢性问题提早调养；湿土之冲忌寒湿，注意保暖与饮食节制。
5. **宫位避让**：冲年冲月不宜做重大不动产交易与大额资金调动。

## 古籍出处

- 《**滴天髓**》："辰戌丑未，四库之神，冲刑之地，各有其用。"
- 《**三命通会·论辰戌丑未**》：以丑未为"**杂气墓库**"，论冲开则藏干出，显则为用，显非其用则为灾。
- 《**渊海子平**》："丑未相刑冲，主田宅动摇、亲情分散。"

## 一句话总结

丑未冲是**两座湿燥不同的土仓对撞**——本气土越冲越旺，中气水火、余气金木却在撞击中**同归于尽**；其吉凶不在冲本身，而在开库所露之物是喜是忌、能否被原局与岁运**接得住**。
