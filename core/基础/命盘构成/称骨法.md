# 袁天罡称骨法

## 起源

袁天罡，益州成都人，**隋末唐初的玄学家、天文学家**，擅长相术。据《新唐书》《旧唐书》记载，他曾给年幼的武则天看相，说武则天可以成为天下之主。

袁天罡称骨算法是根据个人八字（出生的年、月、日、时辰）分别查出对应的骨重，将四个骨重相加得到"八字骨重"，然后查找对应的批注诗。

## 算法步骤

1. 查出生年的骨重（按干支纪年）。
2. 查出生月的骨重（按**农历月份**，非地支月）。
3. 查出生日的骨重（按**农历日期**）。
4. 查出生时辰的骨重（按地支时辰）。
5. 将四者**相加** = **八字骨重**。
6. 根据骨重查对应的男命 / 女命批注诗。

## 计算示例

以甲子年、亥月（农历十月）、丙辰日初七、戊子时的八字为例：
- 甲子年 = **1.2 两**
- 亥月（农历十月）= **0.8 两**
- 初七日 = **0.8 两**
- 子时 = **1.6 两**
- 合计 = 1.2 + 0.8 + 0.8 + 1.6 = **4.4 两**

查对应批注诗："**此命推来福不同，为人能干异凡庸，中年还有逍遥福，不比前时运来通**。"

## 骨重范围与吉凶

骨重范围从 **2.1 两** 到 **7.2 两**。一般规律：
- **八字重**（骨重大）→ 比较不容易被撼动，不容易被邪物侵扰。
- **骨轻不等于差**：也有比较好的批示，但从一般意义上讲，字重者批示普遍比字轻者好。
- **"好"的相对性**：一般意义上的好，在有些人眼中未必是好。

**注意**：**八字重未必绝对好**，须结合个人具体情况理解。

## 批注诗的使用方式

- 批注诗分**男命**和**女命**两种。
- 总骨重精确到 0.1 两，对应一条诗。
- 计算出骨重后直接查对应的男命或女命批注诗，**自行理解**即可。

## 与八字体系的关系

### 相同点
都基于干支纪年—依据人的出生时间推命。

### 根本差异

| 维度 | 称骨法 | 子平八字 |
| --- | --- | --- |
| **方法** | 四数相加 → 查一条诗 | 四柱干支五行 → 综合分析 |
| **粒度** | 人群被分成有限组（骨重数） | 每个八字独特组合 |
| **断事能力** | 一首诗做**大致描述** | 可分析旺衰、格局、喜忌、六亲、事业等多层次 |
| **准确性** | 颗粒度粗 | 颗粒度细 |
| **适用** | 仅作参考 | 主判断系统 |

### 核心结论

> **袁天罡称骨法只是对人生做了一个大致的、简略的描述——只能作为参考。**

### 正确使用方式

1. **不替代八字**：称骨法不能替代四柱八字的系统分析。
2. **作为辅助**：可与八字结论交叉印证。
3. **不过分执着**：得出的诗句只是大致倾向，**不代表命运的全部**。
4. **理解局限**：同一骨重的人太多，没有独特性，不能当作个人命运的决定性依据。

## 一句话总结

称骨法 = **简化版命运描述工具**——操作简单、诗句形象，但颗粒度粗、不够精准；作为了解辅助尚可，不能作为主判断依据。
