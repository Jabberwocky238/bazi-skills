# 正财格

## 定义
- **正财**：日主所克之异性五行——阳日主克阴为正财；阴日主克阳为正财。
  - 例：甲日见己（阴土）为正财；乙日见戊（阳土）为正财。
- **正财格**：**月令取正财为用神**的基础正格。
- 《子平真诠》十正格之一——**稳健型财富结构**。

## 核心：正途之财
- 正财 = 稳定的、合法的、靠劳动取得的财。
- 对应现代：薪水、合同收入、正规经营所得、妻财（男命）。
- 与偏财（机会财、灵活财）对照：
  - **正财**：稳、长、合规。
  - **偏财**：活、快、灵动。

## 正财 vs 偏财

| 对比项 | 正财 | 偏财 |
|---|---|---|
| 阴阳 | 与日主相反 | 与日主相同 |
| 来源 | 劳动、工资、夫妻（男命）| 机会、投资、众人之财 |
| 稳定性 | 高 | 波动 |
| 典型 | 白领、公务员、工程师 | 商人、投资者、中介 |

## 成立条件（严格依《子平真诠·论正财》《论财》）

### 条件 1：月令地支本气为正财，或正财透干且月令有根

- **古籍出处**：
  - 《子平真诠·论财》："**财者，我克之物也**。正财，阳见阴财、阴见阳财。"
  - 《子平真诠·论财》："**取用之法，月令为先**。"
- **客观判据**：
  - ① 月令地支本气为正财；或
  - ② 月令地支藏正财且正财透于天干。
- **不满足则破**：月令正财不显 → 当以月令本气另论他格。

### 条件 2：正财透干且通根，不虚浮

- **古籍出处**（本条核心）：
  - 《子平真诠·论财》："**财要有根；无根则不成格**。"
- **客观判据**：
  - 正财透于天干，且在地支有本气或中气根至少一位。
- **不满足则破**：财虚透无根 → 沈氏"**无根则不成格**"——日主克之无物，格不成。

### 条件 3：日主强能任财

- **古籍出处**（本条核心）：
  - 《子平真诠·论财》："**财格喜身旺财旺**。"
  - 《子平真诠·论财》："**身旺能任财，财多身弱则贫苦**。"
- **客观判据**：
  - 日主于月令为比劫 / 印星本气，或他柱至少两位比劫 / 印星通根（身中偏强）。
- **不满足则破**：身弱财重 → 转入"财多身弱"病格讨论（挣钱辛苦、守不住钱）。

### 条件 4：原局无比劫紧贴夺财

- **古籍出处**：
  - 《子平真诠·论财》："**财怕比劫之夺**。"
- **客观判据**：
  - 天干无比劫与财星紧贴（月 / 日、日 / 时邻位）直接相克。
  - 若有比劫存在，须有官杀制之（官杀克比劫救财）。
- **不满足则破**：比劫贴财无救 → 财被夺 → 挣到钱留不住，或被合伙人 / 家人耗。

### 条件 5（辅佐条件）：财生官或食伤生财，链条顺流

- **古籍出处**：
  - 《子平真诠·论财》："**财生官，富而且贵**。"
  - 《子平真诠·论财》："**食伤生财，财源不竭**。"
- **客观判据**：
  - ① **财生官**：官星透干有根，位于财后（财→官顺流）；或
  - ② **食伤生财**：食伤透干有根，位于财前（食→财顺流）。
- **非必要**：不满足不破格，但格品降——纯财格属"自食其力"，配官印食伤则升。

---

**前四条必须同时满足**方为真正财格；条件 5 关乎格品。《子平真诠》"**财生官，富而且贵**"——正财格的贵路在于"**财生官**"，走入官禄之位方为最高兑现。

## 判断维度

### 1. 身财两停
- 身与财力量接近 → 格最稳。
- 身过强财过弱 → 财被劫，求财辛苦。
- 身过弱财过强 → 财多身弱，忙碌无所得。

### 2. 大运流年
- **财运**：财旺，收入增加 → 喜（身能任）。
- **官杀运**：财生官 → 既富又贵，但身弱则反被克。
- **食伤运**：生财 → 喜。
- **比劫运**：夺财 → 忌（无官杀救时）。
- **印运**：印重则财被克 → 视身强身弱。

## 常见误区

### 1. "月令正财 = 一定富有"
- 正财格只是"**稳健求财的底盘**"。
- 若身弱不任财 → 反而财多身弱。
- 需配身强 + 财有根 + 无比劫夺 → 方为真富。

### 2. 把正财等同"妻财"（男命）
- 传统论"男命财为妻"——正财格男命确有**婚姻稳定 + 妻助事业**之象。
- 但现代语境：**正财的核心是"正途之财"**，未必指妻。
- 独立女性亦有正财格，不能简单套"妻财"。

### 3. 忽视比劫的破坏
- 正财格最大破格因素：比劫紧贴夺财。
- 生活现实：合伙人分利、家人借钱、朋友蹭便宜 → 都是比劫夺财的现代表现。

## 现实表现

### 性格
- **务实、稳重、重信用**。
- **不爱投机、不贪快钱**。
- **做事有计划、有节奏**。
- **对数字、对账目敏感**。

### 职业路径
- **财务、会计、审计**：与正财本性最合。
- **工程、技术、制造**：靠技艺与劳动换取稳定收入。
- **公职、规范企业**：薪资制下的稳定路径。
- **传统行业 / 家业传承**：稳健经营，不冒进。

### 人生节奏
- **早年薪资积累**，中年有房有车。
- **中年稳步增长**，不暴富但不贫。
- **晚年有储蓄、有保障**。
- 典型小康 / 中产阶层命造。

## 核心判断
- 正财格的核心在"**身能任财 + 财有根 + 无比劫夺**"。
- 配财生官 → 既富又贵。
- 识别意义：**走稳扎稳打的正途，不要贪快钱**。

## 一句话总结
- 正财 = **稳定的、靠自己的、合规的钱**。
- 别贪快、别冒险、别指望横财——**正财的福气是"时间喂大的"**。
- 守得住 + 攒得下 + 生得出官（事业）= 正财格一生的完整路径。
