# 魁罡格

## 成立条件（古籍出处 + 客观判据）

### 条件 1：日柱为庚辰、庚戌、壬辰、戊戌之一

- **古籍出处**：
  - 《三命通会·论魁罡》："**魁者斗魁，戌也；罡者天罡，辰也。庚戌、庚辰、壬辰、戊戌四日，惟此四日乃魁罡**。"
  - 《渊海子平·论魁罡》同。
- **客观判据**：
  - 日柱干支严格为 庚辰 / 庚戌 / 壬辰 / 戊戌 之一。
- **不满足则破**：日柱非此四者，根本不入魁罡格讨论。

### 条件 2：日主身旺

- **古籍出处**：
  - 《三命通会·论魁罡》："**身强遇之转荣华，身弱逢之贫到骨**。"
  - 《三命通会·魁罡歌》："魁罡聚会，发福非常。"
- **客观判据**：
  - 月令为日主之比劫 / 印星本气；或
  - 年 / 月 / 时三柱中，至少两位比劫 / 印星通根。
- **不满足则破**：身弱则刚气无所载，反噬自身，多伤灾贫困。

### 条件 3：天干不透破格之五行

此条为本格最具操作性的判据，按日柱分别论：

| 日柱 | 忌透五行 | 所属十神 |
|---|---|---|
| **庚辰、庚戌** | 丙、丁火 | 官杀（火克金） |
| **壬辰** | 丙、丁火 | 财星（水财为火） |
| **戊戌** | 壬、癸水 | 财星（土财为水） |

- **古籍出处**：
  - 《三命通会·论魁罡》："**运见财官，禄科尽丧**。"
  - 《渊海子平·论魁罡》："魁罡最怕财官。"
- **简记**：
  - 两个金的日元（庚辰、庚戌）**不喜见火**。
  - 水的日元（壬辰）**不喜见火**。
  - 土的日元（戊戌）**不喜见水**。
- **客观判据**：
  - 原局天干无上述忌见之五行透出；若仅藏支且力微，格品下降但未必破格。
- **不满足则破**：忌五行透干紧贴日主 → 魁罡刚气被犯 → 禄科尽丧。

### 条件 4：日支不被冲破

- **古籍出处**：
  - 《三命通会·魁罡歌》："**若逢冲破祸灾牙**。"
- **客观判据**：
  - 日支为辰，则年 / 月 / 时支中无戌；
  - 日支为戌，则年 / 月 / 时支中无辰。
  - **辰戌冲最忌**，刑害相对轻。
- **不满足则破**：辰戌冲破日支 → 根基动摇 → 重大灾祸集中爆发。**日柱魁罡被冲为最凶**；时柱、月柱魁罡被冲次之；**形相对没那么可怕**。

### 条件 5（加分）：魁罡叠见

- **古籍出处**：
  - 《三命通会·论魁罡》："**魁罡聚会，发福非常**。"
- **客观判据**：
  - 除日柱外，月 / 年 / 时柱再有庚辰 / 庚戌 / 壬辰 / 戊戌任一组合。
- **效应**：
  - 叠见 + 前四条具足 → 格品大增，富贵非常。
  - 叠见 + 任一条破 → 破得更猛，灾祸亦倍增。

---

**前四条须同时满足**方为真魁罡格。身旺无破之魁罡为将帅之贵；一旦犯财官冲刑——即便日柱为魁罡，格品亦由贵转凶。
