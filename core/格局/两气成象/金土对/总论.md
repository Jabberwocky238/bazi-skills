# 金土对（总论）

## 性质：相生两气对（土→金）
- **五行关系**：土生金（印→身 或 身→食伤的顺流）。
- **属"印生两神成象格"之一**——《滴天髓》五种相生组合中的"土金相生格"。
- 相生两气双清，若气纯无杂，属贵格。

## 两气成象通则（适用于本对）

### 成立条件（严格依《滴天髓》两气双清论）
1. **命局"两气双清"**——只见土金两种五行。
2. **土生金**——土为母、金为子，顺流无克。
3. **原局无财官**（水为财、木为官）——一见财官则破局。
4. **日主属土或金**，且不过弱。

### 具体形式（土金相生格）
- **戊己土日主 + 庚辛金食伤**：身→食伤顺流（土金毓秀）。
- **庚辛金日主 + 戊己土印绶**：印→身顺流（土重金埋若土过多则变病）。

### 气质
- **沉稳 + 刚毅** —— 厚德、讲信义、有决断、善治事。
- 典型特征：**沉稳持重** —— 不张扬但有分量。

### 典型职业
- 司法、金融、精密制造、工程、管理、军警、实业、守成型家业。

### 禀赋与缺失
- **禀赋**：信义、厚德、决断、持久。
- **缺失**：
  - 缺火 → 缺热情、缺生发、易刻板。
  - 缺木 → 缺生机、缺灵活、易僵化。

### 贵格 vs 病象
- **土金毓秀**（贵格）：土生金流畅、气纯无杂 → 稳重型贵气。
- **土重金埋**（病象）：土过旺反压金 → 金被埋没、发挥不出 → 需木疏土或金多以自立。

### 喜忌
- **喜**：土（加强本气）、金（加强本气）、**水**（通关泄秀，土→金→水成生生不息）。
- **忌**：
  - **木**（克土破局）——大忌，木透克土格毁。
  - **火**（克金破局）——大忌（但火若弱则可调候暖土）。
  - 火土同现 → 土燥难生金，降格。

### 古籍背书
- 《滴天髓·源流》："**两气双清，其象最妙**。"
- 《三命通会》："稼穑得金吐秀，**名稼穑毓秀**。"
- 《滴天髓》："**土金相生，清秀之象**。"

## 本对下属格局
- **土金毓秀**：土生金流畅的贵格。详见 `土金毓秀/`。
- **土重金埋**：土过旺反埋金的病象。详见 `土重金埋/`。

## 一句话总结
- 金土对 = **土生金之相生双清**——沉稳、刚毅、厚德。
- 贵在"**土生金流畅**" → 土金毓秀；忌"**土过重埋金**" → 土重金埋。
- 关键看**土金比例**：土适度生金 = 贵；土过重压金 = 病。
