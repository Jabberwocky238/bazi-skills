# 曲直格

## 定义
- **曲直**：《尚书·洪范》"木曰曲直" —— 木的本性是曲而能直、直中带曲，顺势生长。
- **曲直格**：**木日主 + 全局木气一片** → 形成**木的专旺格局**。
- 属**五行专旺格**之一（与炎上格火、稼穑格土、从革格金、润下格水并列）。
- 典型人格：**生机勃勃、向上生长、有生命力与仁德**。

## 核心：一气向上、生而不止
- 木 = **生发、成长、向上、仁爱**。
- 曲直格 = 整个命局都是木 → 这个人的人生底色就是"**永不停止的生长**"。
- 木的"曲直"双性：
  - **曲**：柔韧、能绕、能适应。
  - **直**：向上、刚正、不屈。
- 两者合一 → **既柔韧又坚定**的成长型人格。
## 判断维度

### 1. 木的纯度
- **地支三合 + 天干多木**：格最纯。
- **地支方局（寅卯辰）+ 天干多木**：次之。
- **散见木气但不成局**：曲直难成，按普通身强论。

### 2. 水的参与
- **有水生木**：木更茂 → 格更稳。
- **水过多**：木漂 → 破格。
- **无水**：木略干 → 格偏燥。

### 3. 火的参与（关键）
- **微火**：泄秀 → 曲直 + 木火通明 → 贵格组合。
- **无火**：纯木一气 → 贵但偏闷。
- **火过旺**：木气外泄过度 → 格局变质。

### 4. 大运流年
- **木水运**：顺气加强。
- **火运（适度）**：泄秀开花。
- **金运**：破格大忌。
- **土运（适度）**：不碍；土过重则忌。

## 常见误区

### 1. "木日主 + 身强 = 曲直格"
- 身强 ≠ 曲直。
- 曲直要求"**一气独尊 + 全局木气**"。
- 仅身强只是普通身强格。

### 2. 以为曲直 = 必温和
- 曲直的"**直**"是刚正的一面。
- 真正的曲直格人**该硬的时候比谁都硬**。
- 表面随和不等于无锋。

### 3. 忽视"火"的点缀作用
- 纯曲直 = 闷。
- 加一点火 = 曲直 + 通明 → 格局升。
- **有火点缀的曲直才是完整的曲直格**。

### 4. 把曲直等同"植物型人格"
- 木不只是植物象。
- 它代表**一切生长 / 扩张 / 向上**的力量 → 事业、学识、生命力、仁爱、教育、传承。

## 现实表现

### 性格
- **生机勃勃、积极向上**。
- **心怀仁爱、喜欢助人**（木主仁）。
- **有长期主义精神**：不急于求成。
- **柔韧性强**：能弯能直。
- **对成长敏感**：自我成长、帮别人成长。
- **有理想主义色彩**。

### 事业路径
- **教育 / 培育**：教师、导师、学校领导者。
- **医疗 / 养生**：尤其中医、营养、康复。
- **文化 / 出版**：作家、编辑、文化传承者。
- **林业 / 农业 / 园艺**：字面意义的木。
- **建筑 / 木制品 / 家具**：与木材相关的实业。
- **NGO / 公益**：仁爱精神的体现。
- **创业 / 产品**：从零到一的"**生长型事业**"。
- 共性：**需要"长期耐心 + 向上生长 + 帮扶他人"**的领域。

### 成长节奏
- **前期积累慢**：像树苗一样需要时间。
- **中期绽放**：40-50 岁达到事业高峰。
- **晚年树大根深**：影响力持续扩散。

### 人际
- **朋友多**：木多为比劫 → 天然的群体性。
- **兄弟姐妹 / 同辈关系紧密**。
- **喜欢结交志同道合者**。

### 身体
- 木对应**肝、胆、筋骨、眼目**。
- 曲直格：
  - 肝气充足 → 情绪通畅。
  - 筋骨好 → 喜欢运动、伸展类。
  - 视力通常不错。
  - 破格时：肝气不舒、脾气急躁、筋骨问题。

## 破格

### 破格情形
- **金重克木**：木被砍倒 → 曲直变伐木 → 格破。
- **火过旺**：木气耗尽 → 变"木火相煎"。
- **水过多**：木漂无根。
- **土重断水塞根**：源被断。

### 真假曲直
- **真曲直**：木气纯净 → 一生顺长。
- **假曲直**：夹杂异气却被误判为曲直 → 用神方向错 → 人生反复。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 木运 | **大喜** → 一气加强 |
| 水运 | **喜** → 生木助源 |
| 火运（适度） | **喜** → 泄秀开花 |
| 火运（过度） | **忌** → 耗木过度 |
| 金运 | **大忌** → 克木破格 |
| 土运（过度） | **忌** → 断水塞根 |

## 与其他专旺格对比

| 格名 | 一气 | 核心品质 |
|---|---|---|
| **曲直格**（本条） | 木专旺 | 生长、仁爱、向上 |
| **炎上格** | 火专旺 | 光明、热情、显化 |
| **稼穑格** | 土专旺 | 承载、厚德、稳重 |
| **从革格** | 金专旺 | 变革、裁断、肃杀 |
| **润下格** | 水专旺 | 流动、智慧、深远 |

## 核心判断
- 曲直格的精髓在**一气向上 + 火来点亮**：
  - 一气成象 → 成格。
  - 无火点缀 → 虽成但偏闷。
  - 有火通明 → **曲直 + 木火通明**的复合贵格。
- 识别意义：让木旺命造的人**顺势生长、不要强求其他方向**。

## 一句话总结
- 曲直 = **生长的力量** —— 能柔能直、向光而行。
- 这类人**一辈子都在生长**：自己成长、也帮别人成长。
- 别砍它、别烧它、别塞它的根 —— 给它阳光和水，它会长成一片森林。
