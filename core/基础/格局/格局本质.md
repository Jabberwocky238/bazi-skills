# 格局本质

## 定义
- **格局** = 一个人对事物的理解程度，包括其视野、境界、胸怀等。
- 八字的格局同样是一种**对八字命盘特征的总结**——为每个命盘赋予独特的标签。
- 格局的本质机理：
  - **月令**在命盘中起**提纲挈领**的作用。
  - **日干**则是命盘的核心。
  - **月令所藏的藏干与日干之间，必定存在某种对应的十神关系**。
  - 月令藏干的强弱、透与不透 → 对日主产生不同影响。
  - 影响的程度不同 → 命主在社会地位、财富、吉凶、寿命等方面的层次就会有所差异。
  - 这种差异的影响，就构成了不同类型的命局 → 即**格局**。

## 格局判定的核心机理

- **月令在命盘中起提纲挈领的作用，而日干则是命盘的核心**。
- **月令所包含的藏干与日干之间必定存在某种对应的十神关系**——月令藏干的强弱、透与不透都会对日主产生不同的影响。
- 这种影响与命主的命运好坏密切相关：影响程度不同，命主在社会地位、财富、吉凶、寿命等方面的层次就会有所差异——这种差异的影响，就构成了不同类型的命局，也就是**格局**。

格局派的**核心逻辑**：
1. 以月令藏干透出天干者为格神。
2. 按格神十神属性划入成格名目。
3. 依"成格 → 破格 → 救应"三步法取用神。
4. 格局清纯、链条完整 → 成格成贵；气杂链断 → 破格为凶。

## 格局的分类

八字的格局大致可以分为以下几类：

### 1. 正八格
- 正印格、偏印格、正官格、偏官格（七杀格）、正财格、偏财格、食神格、伤官格。

### 2. 建禄格 与 阳刃格
- 月支为日元禄神 → 建禄格。
- 月支为日元阳刃（帝旺之位）→ 阳刃格。

### 3. 专旺格
- 从革格（金旺）、曲直格（木旺）、润下格（水旺）、炎上格（火旺）、稼穑格（土旺）。

### 4. 从格
- 从财格、从杀格、从儿格（从食伤）、假从格等。

### 5. 富贵格局
- 杀印相生、伤官配印、食神制杀、杀印相生、官印相生 等。

### 6. 其他格局
- 天干顺生格、井栏叉格、日贵格、六阴朝阳格 等。

## 正八格的成格条件

### 条件一：月支本气透出 → 以透干定格
- 月支藏干的**本气**在天干上透出 → 以此透出干与日主的关系定格。
- 案例：月支为巳火，其中本气为丙火。若丙火在天干（时干）透出 → 以丙火与日主的关系定格（若日主为壬水，丙为偏财 → **偏财格**）。

### 条件二：月支本气未透、其他藏干透出 → 以该透干定格
- 月支本气未透，但中气或余气之藏干在天干透出 → 以此透出者定格。
- **重要规则**：月支藏干中的**比劫透出时，正八格不将其视为格**（比劫不入正八格）。
- 案例：月令寅木，本气甲木、中气丙火、余气戊土。
  - 天干只有丙火透出（透出月令中气）→ 此八字可视为**正印格**（若日主为戊土）。
  - 若只有戊土（本气土）透出、同五行己土 → **不能视为建禄或比肩格**。

### 条件三：月支藏干并透 → 取有利者定格
- 月支藏干有多个同时透出 → 根据**有利者**（气势最旺者）定格。
- 案例：月令辰土，本气戊土、中气乙木、余气癸水。
  - 月干透戊土（本气）；年干、时干透甲木（中气）。
  - 因本气戊土比中气乙木更具力量 → 定为**七杀格**（若日主为壬水）。

### 条件四：月支藏干全不透 → 月支强旺有力可定格
- 月支藏干全部未在天干透出，但月支本身强旺有力 → 亦可定格。
- 案例：月令酉金，本气辛金未透，但**有申金贴身帮扶 + 月干己土强有力生扶** → 酉金虽不透但强有力 → 可定为七杀格。

## 建禄格的成格条件

### 核心定义
- **月支为日元禄神**（即**临官位**）。

### 各日干的禄位
| 日干 | 禄位 |
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙、戊 | 巳 |
| 丁、己 | 午 |

### 关键
- 只要月支为日元禄神 → **不需要藏干本气在天干透出**即可判为建禄格。

## 传统格局 vs 做功流通：两种视角

### 传统格局派视角

- **着眼点**：月令为主体，分析月令以外各元素对月令的作用。
- **做法**：归纳出本八字的实际作用效果、主要路径、人生方向，在格局表中选出最符合的一格。
- **常见格目**：杀印相生、官印相生、伤官配印、食神制杀、财官双美等。
- **优势**：
  - **标签化**：方便描述一个人的人生整体，类似 MBTI。
  - **量化**：提供贵格层次的参考坐标。
  - **抓主要矛盾**：集中分析月令主导，减少干扰。
  - **经验积累可检验**：古籍与案例支撑丰厚。

### 做功流通派视角（本流派补充）

- **着眼点**：不是把命盘归类为某一"格"，而是看元素之间**互相做功的配料关系**。
- **比喻**：整个八字是一杯鸡尾酒，各种做功（杀印相生、财坏印、比劫克财、羊刃驾杀等）是各自的配料。
  - 某配料占比 60% → 主导做功，但**不等于整杯酒就叫这个名字**。
  - 某配料占比 10% → 存在但非主流。
- **关键原则**：
  - **做工 ≠ 格局**：配料不等于整杯酒。
  - **元素强势时会"乱捅一气"**：例如比劫强旺，不一定是"帮你分官杀"，也可能是"分了你的一切"。
  - **不存在 100% 纯粹的单一做功**：大部分命盘是多种配料的春秋战国式混战。

## 两派的关键分歧

### 分歧 1：格局是否等于整个八字
- 传统派：成某格 → 此人即此格 → 按格论命。
- 做功派：某种做功 0.6 占比 → 主导但不能代表全部 → 需综合各配料实际流通。

### 分歧 2：命名法
- 传统派：以最主要的配料（如官印相生）命名整杯酒 → 简洁但可能遮蔽其他配料的作用。
- 做功派：命名仅代表一种做功方式，不代表整盘质量。

### 分歧 3：贵格的来源
- 传统派：有某种好配料（如官印相生）即贵。
- 做功派：
  - **配料的纯度**：官印相生是否纯净、无干扰。
  - **配料的比例**：是否在合理区间（约 0.4 ~ 0.8）。
  - **与其他做功的配合**：是否有顺向流动的食伤、财、印配合。
  - **做功链条是否闭环**：做功是否真正产生实际效果。

## 判断格局是否真成立的关键

（本流派视角，作为对传统格局派的补充判据）

### 1. 位置连贯
- 元素之间必须在命盘中**位置相邻或链条贯通**，而非只是同时存在。
- 例：七杀在年柱、印在时柱、中间隔开 → 不成杀印相生。

### 2. 气的纯度
- 同一做功内部不得被异气污染。
- 例：官印相生中混入财星贴克印 → 印被破 → 链条断。

### 3. 做功的实际落脚
- 做功必须产生具体、可观察的效果（事业、关系、身份）。
- 没有实际落脚的"格"只是字面格。

### 4. 能量的匹配
- 日主是否承得住此做功？
- 外部配料是否压过本身，导致"成也格局、败也格局"？

## 不同视角：锐评格局派的滥用

（本节独立呈现一派观点，**不否定传统格局派**的价值，仅对其滥用形式提出批评）

### 被批评的对象

此处所批并非真正的传统格局派（有古籍支撑、经验丰厚的真师傅），而是**带引号的"格局派"假大师**——

### 典型公式化话术

一种极其固定的"公式流"：
- "XX 日主生于 XX 月，是 XX 季节，XX 当令，成 XX 格。"
- "作用效果是 XXX。"
- "应该用 XXX 为喜用。"
- 有时再套一句身强 / 身弱、五行缺什么补什么。

### 此类话术的问题

#### 1. 只套公式不看具体
- 仅凭月令 + 日主推格，不看其他六个字的流通与干扰。
- 例：月令是七杀即断"七杀格 / 食神制杀格"，不看原局是否有真实制杀做工。

#### 2. 无限放大七杀之凶
- 只要原局见七杀 → 一律渲染为"七杀攻身、大祸临头"。
- 借此制造焦虑、诱导客户买"化解"服务。
- 实际上七杀做功方向有多种（杀印相生、食神制杀、身强任杀、羊刃驾杀等），不可一概而论凶。

#### 3. 生搬"贵格"套语
- 见官印相生即断"一生大贵"；见杀印相生即断"权位高升"。
- **忽略两个关键问题**：
  - 配料占比与纯净度。
  - 日主是否承得住这个贵格。
- 家境不足、内核不强者，官印相生反成**被家庭与外力强塞而窒息的"牢笼"**——并非贵。

#### 4. 说怪话、拿捏情绪
- 典型话术："哦原来你是大富大贵之命呢""你一定是武将之风""你妻子很贵气呢"。
- 借助语气词拿捏客户情绪，掩饰内容空洞。

#### 5. 二极管思维
- 成格 vs 破格：非黑即白，无中间态。
- 喜用 vs 忌神：不分庚金、辛金、申金、酉金、戌土之差别，全归一类。
- **实际上任何格局都有中间态、局部态、场景态**——"成格 / 破格"不能描述人的完整情况。

### 格局派的"正当价值"

需要明确：**真正的传统格局派老师是有价值的**——

- 经过长期积累，分析一盘时一定会把所有相互作用都考虑进去。
- 最终归纳出的格局名是**高度概括的标签**，而非全部。
- 分析完格局后**继续分析特殊性**——具体细节、实际做功、路径差异。
- 一条原理推出一条事件，而非空泛套话。

**判断真假的标准**：看师傅是否在格局标签之后**追溯具体原理与细节**。只套公式无细节者即"假大师"。

## 八字的实际运行原理（本流派观点）

### 八字研究的四类相互作用

八字本质是 8 个字的静态特性、动态特性与相互作用：

1. **天干 ↔ 天干**：基于十神关系的作用（食神制杀、官杀制比、伤官配印等）。
2. **地支 ↔ 地支**：基于刑冲合害的作用（寅申冲、丑寅暗合、辰子半合等）。
3. **地支 → 天干**：基于十二长生的旺衰作用（地支决定天干的旺衰状态）。
4. **天干 → 地支**：旺衰逆作用（天干对地支的反作用，常被忽略）。

**重要提醒**：天干不会直接克地支，地支也不会直接克天干。常规的"克"只发生在天干之间。

### 各流派的本质

- 各流派不过是同一棵大树上的不同枝杈：
  - 盲派着重研究静态特性与动态特性。
  - 旺衰派着重研究后两类相互作用。
  - 格局派着重研究天干与天干的十神关系。
- **殊途同归**：真师傅无论从哪一派入手，最终推出的四大基本特征一致。
- **高低贵贱无分**：流派本身不分正邪，只有**真师傅 vs 假大师**之分。

## 如何鉴别"假大师"（附录）

实用清单——

1. **只套公式、不追溯细节**——只有"身强 / 身弱、缺五行、XX 格成格"，没有具体做功原理。
2. **无限放大七杀之凶**——拿七杀吓人，制造焦虑。
3. **说怪话拿捏情绪**——"苟富贵勿相忘""武将之风""贵气漫漫"等语气词。
4. **二极管思维**——只谈成格 / 破格，不谈中间态。
5. **宿命论调**——"命里有什么是什么""改不了"。
6. **抄袭热点**——九紫离火运、赤马红羊之类随热点一顿分析，结论是"大家都发财"。
7. **不能一原理一事件**——无法从一条具体原理推出一条具体事件。

**判断公式**：真师傅讲 **普遍性 + 特殊性**；假大师只讲**普遍套话**。

## 命理的效力边界

本流派对命理作用范围的估计：

| 影响因素 | 占比 |
| --- | --- |
| 普世性命理（八字、紫微斗数、七政四余、星盘） | 约 50% |
| 普遍联系（父母、祖上、密友的命运递归影响） | 约 20% |
| 地理风水（地理位置为主，小摆件其次） | 约 10% |
| 特殊性 / 祖业 / 个体选择 | 约 20% |

**核心观点**：八字 = 给你发的一副牌。同一副牌能打出不同分数——**人是八字的主人，不是八字的奴隶**。

## 一句话总结

- **传统格局派**是古典子平法的主流，以月令为本、成格破格分贵贱——有其深厚古籍依据，不可全盘否定。
- **做功流通派**视格局为"配料名字"而非整杯酒——提醒我们不要把标签当成全部。
- 两派在真师傅手里**殊途同归**；在假大师手里**都会沦为套话**。
- 判一盘好坏，不看它叫什么名字，**看它的做功链条是否真正流动**。
