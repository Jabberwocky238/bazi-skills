# 羊刃驾杀

## 成立条件（严格依《子平真诠·论阳刃》）

### 条件 1：日主为阳干（甲、丙、戊、庚、壬五者之一）

- **古籍出处**（本条为本格前提）：
  - 《子平真诠·论阳刃》："**阳刃者，阳干之刃也**。"
- **客观判据**：
  - 日主天干必须为甲、丙、戊、庚、壬五阳干之一。
- **不满足则破**：阴干日主（乙、丁、己、辛、癸）**结构上不具羊刃**——沈氏明示"阳刃"之名专归阳干。本格根本不适用。

### 条件 2：月支、日支或时支为日主之刃位

- **古籍出处**：
  - 《子平真诠·论阳刃》："**甲禄在寅，刃在卯；丙戊禄在巳，刃在午；庚禄在申，刃在酉；壬禄在亥，刃在子**。"
- **客观判据**（五阳干刃位对照）：
  - 甲日见卯、丙日见午、戊日见午、庚日见酉、壬日见子——落于月支、日支或时支。
- **不满足则破**：阳干日主但刃位不见于命局地支 → 不入阳刃格，本格不适用。

### 条件 3：七杀透干且通根，与羊刃势均（刃杀两停）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论阳刃》："**羊刃无杀，不足为贵；七杀无刃，不足为威**。"
  - 《子平真诠·论阳刃》："**阳刃用官则喜露，用杀则不忌——刃杀相济，声威赫奕**。"
- **客观判据**：
  - ① 七杀透于年 / 月 / 时干，地支有本气或中气根至少一位；
  - ② 羊刃与七杀力量势均（两停）——可综合透干数、通根位数、月令气势判断。
- **不满足则破**：
  - 七杀虚透无根 → 刃无敌可驾，转入"阳刃格"单论；
  - 七杀过重压过羊刃 → 身不胜杀 → 破格转入身弱杀重讨论；
  - 羊刃过重压过七杀 → 杀被压灭 → 羊刃独旺，空有蛮力。

### 条件 4：原局无重印化杀

- **古籍出处**：
  - 《子平真诠·论阳刃》："阳刃驾杀，忌印多夺杀之势。"
- **客观判据**：
  - 天干无两位以上印星透出；地支本气无两位以上印星。
- **不满足则破**：印多化杀 → 本格转入"杀印相生"讨论，非驾杀。

### 条件 5：原局无重食伤制杀

- **古籍出处**：
  - 《子平真诠·论阳刃》："刃杀相济，忌食伤多制。"
- **客观判据**：
  - 天干无两位以上食神 / 伤官透出；地支本气无两位以上食伤。
- **不满足则破**：食伤过重制杀 → 杀倒 → 本格转入"食神制杀"或"伤官见官"讨论。

---

**五条同时满足**方为真羊刃驾杀格。本格之要义在"**两停**"——沈孝瞻"刃杀相济，声威赫奕"八字明示：刃与杀须平衡互制，任一方过强过弱即破格。此格专属阳干（条件 1 不可破），且极依赖大运流年维持两停。

## 补充：羊刃驾杀 vs 身杀两停 的关系

### 结论：包含关系，非同一概念
- **羊刃驾杀 ⊂ 身杀两停**（前者是后者的**特例 / 子集**）。
- 所有羊刃驾杀都是身杀两停；身杀两停**不一定**是羊刃驾杀。

### 本格（羊刃驾杀）相对身杀两停的**特殊限定**
| 维度 | 羊刃驾杀（本格，更严）| 身杀两停（上位概念，更宽）|
|---|---|---|
| **日主** | **必须阳干**（甲丙戊庚壬）| 阳阴皆可 |
| **身强来源** | **限定"羊刃"**（日主帝旺之地）| 比肩、劫财、禄、刃、印等任意 |
| **月令要求** | **必须月令刃位** | 不限月令，身旺即可 |
| **核心气机** | "以刃驭杀"——特定神煞层面 | "势均力敌"——整体能量层面 |
| **印星要求** | 命局**必须带印** | 非必需 |

### 严格起法（羊刃专论）
按民间命理详解，羊刃驾杀的严格起法：
- 日主五阳干取**阴干刃合杀**。
- 例：甲木生卯月——庚为甲之七杀透干而旺，取卯为甲刃，**卯藏阴干乙合庚杀**——称"羊刃驾杀"。
- 此严格说法将本格限定于特定合杀结构，比单纯"月令刃 + 七杀"更精细。

### 识别优先级
- 命局中若符合**羊刃驾杀四条硬要件**（阳干 + 月令刃 + 七杀有根 + 带印）→ 按本格（羊刃驾杀）论。
- 若仅"身旺与七杀两停"但身强来源非羊刃 → 按 **`身杀两停/`** 格讨论（见该格文件）。

### 古籍参照（两格对比）
- **羊刃驾杀**：《子平真诠·论阳刃》"**阳刃用官则喜露，用杀则不忌——刃以杀制为贵**"；"**羊刃无杀，不足为贵；七杀无刃，不足为威**"。
- **身杀两停**：沈孝瞻"**身杀两停，方为上贵**"——层次上被部分古籍列为"七杀格中尤为上上等富贵"，涵盖范围更广。
