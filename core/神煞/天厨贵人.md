# 天厨贵人

## 定义
- **天厨**：上天之厨、天赐饮食——象征**衣食禄、饮食之福、食禄充足**的贵人神煞。
- 本质：**饮食无忧、食禄有福、不愁吃穿**的命理标记。
- 与**文昌贵人**起法相近但含义不同：
  - **文昌**：主智慧、学业、考试。
  - **天厨**：主**饮食、食禄、衣食**。
- 对应古人最看重的"**食禄**"——吃得好、有俸禄、有口福。

## 起法（按日干查"食神之禄位"）
《三命通会·论天厨贵人》原文："**天厨者，日干食神之禄也**。"

| 日干 | 食神 | 食神之禄 = 天厨贵人 |
|---|---|---|
| 甲 | 丙 | 丙禄在 **巳** |
| 乙 | 丁 | 丁禄在 **午** |
| 丙 | 戊 | 戊禄在 **巳** |
| 丁 | 己 | 己禄在 **午** |
| 戊 | 庚 | 庚禄在 **申** |
| 己 | 辛 | 辛禄在 **酉** |
| 庚 | 壬 | 壬禄在 **亥** |
| 辛 | 癸 | 癸禄在 **子** |
| 壬 | 甲 | 甲禄在 **寅** |
| 癸 | 乙 | 乙禄在 **卯** |

**起法逻辑**：食神 = 福气之源（古代"**食神为福神**"）；食神的禄位 = **福气最旺处** → 天厨贵人。

## 天厨贵人 vs 文昌贵人

| 对比项 | 天厨贵人 | 文昌贵人 |
|---|---|---|
| 起法 | 严格按"日干食神之禄" | 按传统口诀（丙戊→申，丁己→酉）|
| 丙丁差异 | 丙→巳、丁→午（各归食神禄）| 丙→申、丁→酉（口诀对应）|
| 主方向 | **饮食、食禄、衣食福** | 智慧、学业、考试 |
| 现代意义 | 不愁吃穿、有口福 / 禄 | 读书、写作、考试力 |

**注意**：两者在部分日干（甲乙戊己庚辛壬癸）上起法结果相同，但**丙日与丁日**明显不同——这正是两者被混淆又需要分开的关键。

## 核心作用：衣食之福
- 天厨主"**吃得好、有禄、不缺**"：
  - **三餐不愁**——古代最大福气的底层。
  - **有食禄、有俸禄、有薪水**。
  - **家中饭桌丰盛、食物品质高**。
  - 现代对应：**饮食品质好、收入稳定、不缺花销**。
- 古籍论："**天厨入命，主一生衣禄丰盈**。"

## 天厨贵人的三层含义

### 1. 物质层：衣食无忧
- 家境不差、吃穿不愁。
- 收入稳定、现金流足。
- 家中常有美食、懂得享受。

### 2. 事业层：食禄 = 薪水禄位
- 古代"食禄"指官员俸禄——**领薪水的命**。
- 现代："**体面且稳定的收入**"——白领、公务员、专业岗。
- 与"**禄神**"呼应但角度不同（禄神主立身之根；天厨主食禄之福）。

### 3. 职业层：与食相关的行业有缘
- **厨师、餐饮业主、美食博主**。
- **食品研发、营养师、农业**。
- 天生对食物敏感、懂美食。

## 判断维度

### 1. 位置
- **年支天厨**：家境丰足、少年时衣食之福好。
- **月支天厨**：中年事业带食禄——典型的"**领薪型**"。
- **日支天厨**：本人一生不愁吃穿 / 配偶与饮食相关。
- **时支天厨**：晚年有福、子女不缺钱。

### 2. 配合其他贵神
- **天厨 + 天乙贵人**：有食禄 + 遇贵化险——双重福气。
- **天厨 + 金舆**：物质享受全面——典型富贵命。
- **天厨 + 文昌**：才 + 福兼得（学问之人带食禄）。
- **天厨 + 食神格 / 伤官格**：命主本就靠"吃"或"食相关行业"立身。

### 3. 与食神本身的关系
- 命中食神有力 + 带天厨 → 食神之福完全兑现。
- 食神被枭印夺（枭神夺食）+ 天厨被冲 → 食禄之福被夺 → 衣食不济。

### 4. 大运流年引动
- 走天厨地 → 食禄加身 → 升职加薪、收入增加、搬家到饮食丰足之地。
- 原局已带天厨 + 流年再引 → **食禄大兴之年**。

## 常见误区

### 1. "天厨 = 一定发胖"
- 天厨主"**食禄丰足**"，不必然导致发胖。
- 现代饮食多样化——带天厨的人可能**懂得选择优质食物**而非暴饮暴食。
- 典型画像：**品味型美食家**，而非饕餮食客。

### 2. 把天厨等同"一定富贵"
- 天厨是"**衣食不愁**"的底牌，不保"**大富大贵**"。
- 许多带天厨的人一生中等富裕——**不缺钱但不豪奢**。
- 大富需配合财星、官杀、格局等综合判断。

### 3. 混淆天厨与文昌
- 文昌主"**脑子**"，天厨主"**嘴**"。
- 同一地支既是文昌又是天厨（部分日干）→ 需从性质不同角度解读。
- 甲日巳既是文昌又是天厨——脑子灵 + 有口福。
- 但丙日巳是天厨（不是文昌）、丙日申是文昌（不是天厨）。

## 现实表现

### 性格
- **对食物敏感、有品味**。
- **懂享受生活**：不委屈自己的胃。
- **慷慨、不吝啬**：乐于请客吃饭。
- **家中氛围温暖**：饭桌常是家庭重心。

### 典型情境
- 朋友聚会时往往是**"带路的那个"**——知道哪家餐厅好吃。
- 职业选择上对"**好福利、有食堂、包餐**"的岗位天然亲近。
- 家中冰箱 / 厨房常丰富，不让自己或家人受饿。
- 中老年后可能涉足**餐饮业 / 美食博主 / 养生饮食**。

### 职业路径
- **餐饮业**：厨师、餐厅经营、美食博主。
- **食品工业**：食品研发、营养师、食品贸易。
- **农业、畜牧、渔业**：食物源头。
- **白领 / 公务员 / 专业持证岗**：典型的"**领薪食禄**"命。
- **饮食文化**：茶道、酒文化、烹饪教学。

## 核心判断
- 天厨贵人是"**衣食之福的底牌**"——不保大富但保不愁温饱。
- 与食神有力并现 → 食福完整兑现。
- 识别意义：这类命造**一生对"吃"与"领薪水"都有缘**——食物是福气入口，薪水是生活保障。

## 一句话总结
- 天厨 = **老天管饭的那颗星**。
- 不管混得多差，**饭总有的吃、禄总能领到**。
- 这是平民福气里最朴实也最重要的那一份 —— 不饿肚子就是天赐的大福。
