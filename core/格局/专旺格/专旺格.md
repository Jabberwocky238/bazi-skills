# 专旺格

## 定义
- **专**：专注、唯一。
- **旺**：极旺、一气独大。
- **专旺格**：命局**单一五行旺到极致**，其他五行**完全被这一气压制或消化** → 日主与这股独强的气**完全同频** → **顺势而不逆反**。
- 又称"**一行得气格**""**从旺格**"的一类 —— 属**外格**，与从财、从杀、从儿、从势并列。

## 一气为主：五类典型

| 格名 | 一气 | 条件 |
|---|---|---|
| **曲直格** | 木 | 木日主，全局木气一片 |
| **炎上格** | 火 | 火日主，全局火气一片 |
| **稼穑格** | 土 | 土日主，全局土气一片 |
| **从革格** | 金 | 金日主，全局金气一片 |
| **润下格** | 水 | 水日主，全局水气一片 |

这五个格合称为"**五行专旺格**"或"**一行得气格**"，是专旺格的标准类型。

## 核心：一色天下，逆则破格
- 正格思路：身强抗克、有制有化 → 平衡。
- 专旺格思路：身强到**一气独尊** → **不能再克、不能再化**，只能**顺气独行**。
- 比喻：
  - 正格 = 团队作战，各有角色。
  - 专旺 = 独角戏，一个人把所有戏份演完。

## 成立条件（严格依《渊海子平》《三命通会》《滴天髓》）

### 条件 1：日主与月令同气（比劫或印星得月令本气之助）

- **古籍出处**：
  - 《渊海子平·杂格》："**五行专旺之格，须日主与月令同气**——甲乙日见亥卯未或寅卯辰；丙丁日见寅午戌或巳午未；戊己日见辰戌丑未；庚辛日见巳酉丑或申酉戌；壬癸日见申子辰或亥子丑。"
- **客观判据**：
  - 日主五行与月令地支同属一气（木日 + 木月；火日 + 火月；土日 + 土月；金日 + 金月；水日 + 水月）。
- **不满足则破**：月令非日主之气 → 无一气成象之根，不入专旺。

### 条件 2：地支会该五行之局（三合或三会）

- **古籍出处**（本条核心）：
  - 《三命通会·五行专旺》："**一行得气，必以三合或三会成局为成象**。"
- **客观判据**（按五行分）：
  - 木专旺（曲直）：亥卯未三合木，或寅卯辰三会东方木；
  - 火专旺（炎上）：寅午戌三合火，或巳午未三会南方火；
  - 土专旺（稼穑）：辰戌丑未四库全见或三见；
  - 金专旺（从革）：巳酉丑三合金，或申酉戌三会西方金；
  - 水专旺（润下）：申子辰三合水，或亥子丑三会北方水。
- **不满足则破**：地支只散见同气不成局 → 一气不成象，作普通身旺论。

### 条件 3：天干多透比劫或印星以遍布一气

- **古籍出处**：
  - 《渊海子平·杂格》："**天元须透……使一气遍布**。"
- **客观判据**：
  - 除日主外，天干另有至少两位为同五行的比劫或生我之印星。
- **不满足则破**：天干一气不足 → 未能"遍布"，格难以成立。

### 条件 4：原局无克本气之字（无官杀透干，无财重克印 / 克比劫）

- **古籍出处**（本条核心）：
  - 《渊海子平·杂格》："**一气成象者，不宜见克**——克者破格之源。"
- **客观判据**（按五行分）：
  - 木专旺：天干无庚辛金透，地支申酉本气不成势；
  - 火专旺：天干无壬癸水透，地支亥子本气不成势；
  - 土专旺：天干无甲乙木透，地支寅卯本气不成势；
  - 金专旺：天干无丙丁火透，地支巳午本气不成势；
  - 水专旺：天干无戊己土透，地支辰戌丑未不成势。
- **不满足则破**：克神透干或本气成根 → 一气被破 → 格败。

### 条件 5：食伤之泄可有可无——微泄秀气可喜，重泄破气

- **古籍出处**：
  - 《滴天髓·从象》："**一气成象，微泄秀气方妙；若泄过而衰，非吉象也**。"
- **客观判据**：
  - 食伤仅一位透干或地支一位根为"微泄秀气"，可喜；食伤多透多根 → 泄过。
- **不满足则破**：重食伤泄过 → 一气衰 → 不成象。

---

**五条同时满足**方为真专旺格（分化为曲直 / 炎上 / 稼穑 / 从革 / 润下五种，各有专章）。专旺格是唯一"身极强"的外格，要求"一气遍布 + 无克 + 微泄"——与从财 / 从杀等"身极弱"的外格形成两极对照。

## 专旺 vs 从旺

| 对比项 | 专旺格 | 从旺格 |
|---|---|---|
| 定义 | 一行五行独大 | 比劫印星主导 |
| 侧重 | 某一五行（如木） | 自身气势 |
| 成格条件 | 更严格，要"一色天下" | 略宽松 |

实际使用中，**专旺格** 和 **从旺格**常混用 —— 本文可视为其合并理解。

## 判断维度

### 1. 气是否"纯"
- **全局一色**：格最纯 → 成就最高。
- **略有杂气**：格破成废 → 转为普通正格论。
- 判断标准：**有没有哪怕一丝"抵抗"**。

### 2. 印比的比例
- **比劫为主**：偏"独立自信"型，个人色彩强。
- **印星为主**：偏"学识资源"型，气场厚重。
- **印比并重**：典型的专旺成格。

### 3. 大运流年的关键
- 走**印比地**：一气加强 → 格更稳 → 大发。
- 走**食伤地**：轻度泄气 → 本格勉强不破，若食伤轻可喜；若食伤重则破。
- 走**财官地**：**大忌** → 直接打破一气 → 格瓦解 → 可能大起大落。

### 4. 日主位置与气场
- 日主透干 + 有强根 → 格完整。
- 日主坐下为自身五行 → 典型专旺的"坐下有根"。

## 常见误区

### 1. "身旺 = 专旺格"
- 身旺 ≠ 专旺。
- 正格身旺只是"强"，仍在正常范围。
- 专旺格是**极端情况**，要求一气独尊，几乎没有其他五行来牵制。

### 2. 把专旺格等同"必贵"
- 专旺是**贵格之一**，但：
  - **极度依赖大运顺气**。
  - 大运一背 → 比任何格都崩得快。
- 专旺格的人**人生曲线非常陡峭**：顺时狂飙，逆时骤降。

### 3. 忽视"无对手"的代价
- 专旺格的人**缺少外部制约**：
  - 内心里"**对自己太认同**"，常常**自负到听不进反对**。
  - **缺少挫折教育** → 走入顺境时过于张扬，走入逆境时难以调整。

### 4. 以为专旺"没弱点"
- 专旺的弱点是**不能被冲破**。
- 大运流年**任何对一气的冲击**都是致命的。

## 现实表现

### 性格
- **自信爆棚**：自己就是一面军队，不需要别人。
- **不妥协**：你说东我往东，我就往东；你说西我往西，我还是往东。
- **独立性极强**：不愿跟人合作，一个人也能把事做成。
- **气场强大**：外人一眼能感到"这个人不一般"。
- **我行我素**：对世俗规则缺乏敬意。

### 事业路径
- **单干型**：独立创业者、艺术家、自由职业。
- **意见领袖 / 精神领袖**型：自带磁场，有追随者。
- **极致专业型**：一辈子深耕一个领域，不旁顾。
- 共性：**不需要团队、不靠平台、自己就是品牌**。

### 人际
- **朋友稀少但极铁**。
- **婚姻独特**：要么找完全互补的配偶，要么不婚。
- **家族关系疏离**：容易独立生活、远离原生家庭。

## 破格

### 真假专旺
- **真专旺**：一气极纯 → 顺其气则成就非凡。
- **假专旺**：看似一气，实则夹杂微弱异气 → 最糟糕的格 → **既不是专旺，又不是正格** → 人生混乱。
- 识别要点：命局里有没有**哪怕一丝异气的字**。

### 大运破坏
- **财运**：打破一气 → 因财败事、意外变故。
- **官杀运**：一气被克 → 压力超载、突然失控。
- **冲破日主之根的大运 / 流年**：格瞬间瓦解。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 印比运 | **大喜** → 一气加强，大发 |
| 食伤运（适度） | 视结构 → 轻泄可喜；重泄则破 |
| 财运 | **大忌** → 一气破 → 重大变故 |
| 官杀运 | **大忌** → 一气被克 → 格局瓦解 |

## 与相关外格的关系

| 格局 | 顺从对象 | 状态 |
|---|---|---|
| **专旺格 / 从旺格** | 身与印比 | 身极强，顺自身气 |
| **从财格** | 财 | 身极弱 |
| **从杀格** | 七杀 | 身极弱 |
| **从儿格** | 食伤 | 身极弱 |
| **从势格** | 财官食伤综合 | 身极弱 |

专旺格是唯一**身极强**的外格，其他外格都是**身极弱**。两极对照：**强到一个人 = 外格；弱到没有人 = 外格**。

## 核心判断
- 专旺格的核心不是"强"，是**纯**。
- 一丝杂气 → 格破。
- 纯到独尊 → 借一色之气成就独一无二的人生。
- 识别要极度谨慎 —— 比一般格局更难判，错判代价极高。

## 一句话总结
- 专旺 = **一个人走完所有人的路**。
- 不合群、不妥协、不平衡，凭一股纯气走到底。
- 成格时惊艳，破格时惊险 —— 这是一种极致的人生方式。
