# 十神与六亲

## 一、六亲的定义

八字中的"**六亲**"指命主周围最核心的血缘 / 伴侣关系：

- **自己**（日元）
- **父母**
- **子女**
- **配偶**（妻或夫）
- **兄弟姐妹**

十神之所以能对应六亲，是因为这些关系与命主的**生克方向**天然吻合：生我者对应父母类、克我 / 我克者对应配偶与子女类、同我者对应兄弟姐妹。

## 二、男命六亲对应

### 直接定义的六亲（最重要）

| 六亲 | 十神 | 原理 |
| --- | --- | --- |
| **母亲** | 正印 | 异阴阳生我——全心全意的滋养 |
| **父亲** | 偏财 | 我克同阴阳——父为母之夫，推导所得 |
| **妻子** | 正财 | 我克异阴阳——有情之克，正配 |
| **儿子** | 七杀 | 同阴阳克我——父子相克同性 |
| **女儿** | 正官 | 异阴阳克我——父女相克异性 |
| **兄弟** | 比肩 | 同五行同阴阳 |
| **姐妹** | 劫财 | 同五行异阴阳 |

### 推导出的六亲（隔辈）

| 六亲 | 十神 | 推导路径 |
| --- | --- | --- |
| 祖父 | 偏印 | 父亲（偏财）的父亲（偏财的偏财）= 日元的偏印 |
| 祖母 | 伤官 | 祖父（偏印）的妻子（偏印所克异性）|
| 孙子 | 食神 | 儿子（七杀）的儿子（七杀的七杀）|
| 孙女 | 伤官 | 儿子（七杀）的女儿 |
| 情人 | 偏财 | 偏财 = 正财之"偏"，家外的女性 |
| 继母 | 偏印 | 非本生之母，枭神代之 |

## 三、女命六亲对应

| 六亲 | 十神 | 原理 |

| **父亲** | 正财 | 我克异阴阳——推导所得（女儿克父为异性）|
| **丈夫** | 正官 | 异阴阳克我——有情之克，正配 |

| **女儿** | 食神 | 我生同阴阳——母女同性 |
| **姐妹** | 比肩 | 同五行同阴阳 |
| **兄弟** | 劫财 | 同五行异阴阳 |

### 推导出的六亲（隔辈）

| 六亲 | 十神 | 推导路径 |
| --- | --- | --- |
| 祖父 | 食神 | 父亲（正财）的父亲 |

## 四、男命 / 女命六亲差异对比

| 六亲 | 男命 | 女命 |
| --- | --- | --- |
| 母 | 正印 | 偏印 |
| 父 | 偏财 | 正财 |
| 配偶 | 正财（妻）| 正官（夫）|
| 婚外异性 | 偏财（情人）| 七杀（情人）|
| 儿子 | 七杀 | 伤官 |
| 女儿 | 正官 | 食神 |
| 兄弟 | 比肩 | 劫财 |
| 姐妹 | 劫财 | 比肩 |

**规律**：

1. 男命以"**克我者**"为子女，女命以"**我生者**"为子女——与生育的生理方向呼应。
2. 男命以"**我克者**"为妻，女命以"**克我者**"为夫——与传统婚姻权力结构呼应。
3. 母亲、父亲的正偏在男女命间**刚好互换**（男命正印母、偏财父；女命偏印母、正财父）——因为"母"与命主同性、"父"与命主异性。
4. **兄弟姐妹的比肩 / 劫财也互换**——比肩是同阴阳的兄弟 / 姐妹（男命是兄弟、女命是姐妹）。

## 五、正偏差异：同一六亲的两副面孔

### 正印（母） vs 偏印（继母）
- **正印**：全心全意、发自内心的关爱、保护与付出——亲生之母。
- **偏印**：也有关心爱护，但不一定出自本心——更多是"爱屋及乌"式的继母、养母、或感情疏远的生母。

### 正财（妻） vs 偏财（情人）
- **正财**：交往的异性偏顾家、稳定、持家——家里的妻子。
- **偏财**：交往的异性风流、浪漫、多情——家外的情人、或流动的财源。

### 正官（夫） vs 七杀（情人 / 男友）
- **正官**：规矩、端正、愿意承担责任的丈夫。
- **七杀**：有压力、有冲击力、可能不婚或婚外的男性。

### 食神（女） vs 伤官（子）
- **食神**：温和、乖巧、文艺的女儿。
- **伤官**：聪慧、张扬、易与父母顶撞的儿子。

## 六、六亲的宫位对应

命盘四柱各有其宫位意义，**十神 + 宫位双重判断**才能精确看六亲：

| 柱 | 宫位代表 | 主要六亲 |
| --- | --- | --- |
| **年柱** | 祖上宫 | 祖父母、家族、1–16 岁少年运 |
| **月柱** | 父母宫 / 提纲 | 父母、兄弟姐妹、16–32 岁青年运、事业 |
| **日柱** | 自己 + 配偶宫 | 日干 = 自己；日支 = 配偶、32–48 岁中年运 |
| **时柱** | 子女宫 | 子女、晚年、48 岁后老年运 |

### 宫位 + 十神配合口诀

- **月柱见正印 / 偏印**：父母有文化、家境重教育。
- **日支见正财**（男）/ **正官**（女）：配偶条件好、婚姻稳定。
- **日支见七杀**（女）/ **劫财**（男）：配偶性格强、婚姻有冲击。
- **时柱见食神 / 伤官**：子女聪慧、有才艺。
- **时柱见七杀无制**：子女与自己关系紧张、晚年孤寒。
- **年柱十神**：看祖上与早年家境、与祖辈缘分。

### 宫位冲克对六亲的影响
- **月柱被冲**：父母缘薄、少年动荡。
- **日支被冲**：婚姻不顺、配偶缘变动。
- **时柱被冲**：子女缘薄、晚景动荡。

## 七、六亲看盘的三个步骤

1. **先定十神**：男女命分别定出对应六亲的十神是哪一个。
2. **再定宫位**：看该十神是否落在对应宫位（如妻子 = 正财，最好落日支或与日支相合）。
3. **最后看强弱喜忌**：
   - 该十神为喜用且有根有护 → 该六亲助力、关系好、缘分深。
   - 该十神为忌仇或被冲克 → 该六亲为负担、关系差、缘分浅。

## 八、一句话总结

十神对应六亲不是死记，而是从"**我克、克我、我生、生我、同我**"的五行方向 + 阴阳同异自然推出——**男女命因生育方向不同而部分对应互换**，再叠加**宫位**的双重定位，才能精确判断某一六亲与命主的**缘分、关系、助力或拖累**。
