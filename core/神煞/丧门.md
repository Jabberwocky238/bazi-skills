# 丧门

## 定义
- **丧**：丧失、死亡、哀伤。
- **门**：门户、进出口。
- **丧门**：主**丧葬、哀事、孝服、长辈 / 亲人去世**相关的凶煞。
- 与"**吊客**"配对出现——丧门在前，吊客在后；两者分主不同层次的**哀事**。

## 起法（按年支查四柱地支）
《三命通会·论丧门》原文："**丧门在岁前二辰**"——即**年支前两位**。

具体排法：

| 年支 | 丧门位置 |
|---|---|
| 子 | 寅 |
| 丑 | 卯 |
| 寅 | 辰 |
| 卯 | 巳 |
| 辰 | 午 |
| 巳 | 未 |
| 午 | 申 |
| 未 | 酉 |
| 申 | 戌 |
| 酉 | 亥 |
| 戌 | 子 |
| 亥 | 丑 |

**背后原理**：丧门 = 年支前两位（**岁前二辰**）。
- 子年丧门在寅——子前两位为寅；
- 古人以十二神煞围绕岁支排列，丧门位居岁支之前，代表"**迎面而来的哀事**"。

## 丧门 vs 吊客

| 对比项 | 丧门 | 吊客 |
|---|---|---|
| 位置 | 岁前二辰 | 岁后二辰 |
| 性质 | 前迎 → 直接哀事（自家） | 后随 → 伴随哀事（他家 / 吊唁） |
| 主事 | 家中丧事、长辈去世 | 外部丧葬、吊唁之事 |
| 现代 | 亲人离世、重病、家庭变故 | 参与他人丧事、接触死亡事件 |

丧门 + 吊客同现 → **主丧事密集**，是命理中最明显的丧事神煞标记。

## 核心作用：哀事的预警
- 丧门不是"**必然死人**"，是**与丧葬气场靠近**：
  - 家中长辈病重、去世。
  - 亲友突发重病或离世。
  - 自己接触死亡相关事件（医院工作、殡葬业、重大疾病照顾）。
- 古籍论："**丧门入命，主孝服之事**。"

## 判断维度

### 1. 位置
- **年支丧门**：祖辈或父母易有丧事（尤其少年时）。
- **月支丧门**：成长期至中年遇家中丧事。
- **日支丧门**：自身与丧事气场近 / 配偶健康事。
- **时支丧门**：晚年或子女面对的丧事。

### 2. 丧门配其他神煞
- **丧门 + 吊客**：丧事频密 → 多次面对死亡。
- **丧门 + 白虎**：**血光 + 丧事** → 意外死亡相关。
- **丧门 + 天医**：从事医疗 / 殡葬 / 临终关怀 → 职业性接触。
- **丧门 + 华盖**：与宗教 / 超度 / 精神哀悼相关。

### 3. 大运流年引动
- 大运 / 流年走丧门位 → 家中或亲友有丧事的概率增加。
- 原局已带丧门 + 流年再引 → **家中丧事集中爆发年**。

## 常见误区

### 1. "带丧门 = 一定死人"
- 最大的误判。丧门是"**与丧事气场接近**"，不是"**必然有人死**"。
- 很多人带丧门一生平安——关键看**命局承载力**。
- 从事医疗、殡葬、护理、心理咨询等接触死亡的职业 → 丧门以**职业身份**体现，而非家庭悲剧。

### 2. 把丧门当成"最凶煞"
- 丧门是神煞中的**情境性标记**，不是最凶。
- 真正的凶煞是"**灾煞、七杀无制、羊刃冲**"等 → 主**本人的灾险**。
- 丧门主**周围的哀事**——间接影响自己。

### 3. 忽视"心理层面"
- 丧门命的人即便没经历真正的丧事，也可能长期有**对死亡的敏感、哀伤倾向**。
- 这种敏感若能导向艺术、哲学、心理等领域 → 深刻的创作力。

## 现实表现

### 性格
- **对死亡、哀伤有敏锐感知**。
- **情感深沉、不易外露**。
- **容易被"沉重的事情"吸引**：古建筑、墓葬、历史、哲学。
- **关怀型气质**：愿意陪伴他人渡过难关。

### 典型情境
- 少年早失亲人（尤其父母或祖辈）。
- 一生多次参加丧事——家族或朋友圈中有"**总是帮忙料理后事**"的角色。
- 从事医疗、殡葬、临终关怀、心理咨询等职业。
- 对死亡的态度比常人平和——**看多了、习惯了**。

### 职业路径
- **医疗 / 护理 / 殡葬**：职业性接触死亡。
- **心理 / 哀伤辅导**：帮人面对失去。
- **历史 / 考古 / 宗教**：与已逝世界相关。
- **艺术 / 文学**：深度哀伤作为创作源泉。

## 化解与应对

### 结构层
- 命局有印星、贵人 → 丧门之凶减弱。
- 大运走喜神年 → 家中平安概率高。

### 人事层
- **保持与家人的密切联系**：丧门引动时，陪伴长辈尤其重要。
- **做好丧葬相关的准备**：不是迷信，是现实。
- **注意家人健康**：定期体检，早发现早处理。

### 心态层
- 接受丧门命的人**与死亡近一步的现实**。
- 把这份"近距离"转化为**更珍惜活着的每一天 + 更善于陪伴他人**。

## 核心判断
- 丧门是"**靠近哀事的气场**"，不是"**必然丧事**"。
- 命局能承载 + 从事相关职业 → 这份气场成为**帮人渡难关的能力**。
- 命局弱 + 流年引动 → 真可能遇到家中重大变故。

## 一句话总结
- 丧门 = **与哀事走得近的倾向**。
- 带丧门的人并非倒霉，是"**看过更多生死**"。
- 能把这份经历转化为同理心与智慧 → 反而是独特的生命礼物。
