# 真从格

## 定义

- **真从格**：日主**极弱无根**，命局**一党**（财、官杀、食伤其一）气势**绝对独旺**，天干地支**绝无印比一丝一毫救应** → 日主彻底放弃本位，顺从旺势而行。
- 古人称之为"**气命从财、气命从杀、气命从儿**"——日主是"俘虏"、不得不顺从命局的旺势。
- 真从之"真"体现在两点：
  - **日主无根到底**：地支四位本气、中气、余气皆无比劫 / 印星。
  - **原局清净**：天干地支都没有一点印比来搅局。

## 核心：顺势借力

- 打个比方：你是一个特别弱小的国家、手里没兵没粮；隔壁住着超级大国兵强马壮。
  - 硬刚 → 被拍死。
  - 举白旗投降 → 乖乖顺从、还能活下来、甚至获得资源。
- 当日主弱到极点时：
  - **强行用印（找妈妈帮忙）或比劫（找兄弟帮忙）** → 不但帮不上忙、反而激怒旺势、"**先打倒用神、再把余怒发泄到日主身上**"。
  - **唯一的活路就是顺从这个强大的气势**——这就是从格的本质。
- 从财格 = 跟了当老板的；从杀格 = 跟了当官的、跑前跑后被提拔；从儿格 = 跟了艺术家、跟着学手艺。

## 与其他相关格局的对比

### 真从 vs 专旺格
- **专旺格**：命局气势与日主一致、日主极强（如从革格日主庚辛金、满局金旺）——日主是"霸王"。
- **真从格**：正好相反——日主极弱是"俘虏"、不得不顺从异党旺势。

### 真从 vs 假从

| 维度 | 真从格 | 假从格 |
|---|---|---|
| 天干印比 | 无 | 有一位虚浮无根者 |
| 地支印比根 | 绝无 | 绝无（共同底线） |
| 旺势 | 纯而不杂 | 旺但略杂 |
| 取用 | 顺旺势 | 顺旺势（与真从一致） |
| 运势起伏 | 顺运极贵、逆运暴跌 | 动荡比真从更剧 |

**关键铁律**：只要地支出现印比本气 / 中气 / 余气根 → 既不论真从，也不论假从 → 归入身弱破格讨论。

## 成立条件（严格依古籍）

### 条件 1：日主无根（天地人元绝无一毫生扶）

- **古籍出处**：
  - 《滴天髓·从象》（任铁樵注）："**日主孤立无气，天地人元绝无一毫生扶之意**。"
  - 《子平真诠·论杂格》："日主无根，不得不从其旺神。"
- **客观判据**：
  - 天干日主周围无任何比劫或印星透出；
  - 地支四位的**本气、中气、余气**皆无一丝比劫或印星藏干。
- **不满足则破**：但凡地支有一位印比之根（哪怕冲库之余气） → 不可论真从。

### 条件 2：旺神成气势满局

- **古籍出处**：
  - 《滴天髓·从象》："**从得真者只论从，从神又有吉和凶。**"
- **客观判据**：
  - 命局中财、官杀、食伤之某一党压倒性独旺；
  - 月令为该党之本气，或地支月令参与三合三会成该党之局；
  - 天干地支中该党数量占大半。
- **不满足则破**：若旺神势散、无一方独大 → 须按从势格另论。

### 条件 3：无印星救日主

- **古籍出处**：
  - 《滴天髓·从象》："**若有一毫印绶，即不为从**。"
- **客观判据**：
  - 天干无印透出；
  - 地支本气、中气皆无印藏干（严格论：余气亦无）。
- **不满足则破**：印透或通根 → 日主有救 → 不可论从。

### 条件 4：无比劫帮日主

- **古籍出处**：
  - 《滴天髓·从象》："**比劫一见，即不为从**。"
- **客观判据**：
  - 天干无比劫透出；
  - 地支本气、中气皆无比劫藏干。
- **不满足则破**：比劫帮身 → 日主抗拒旺势 → 从格不立。

### 条件 5：克泄耗之党内部不互相厮杀

- **古籍出处**：
  - 《子平真诠·论杂格》："从者，必其势一气贯通，乃为真从。"
- **客观判据**：
  - 从财则无比劫夺财；
  - 从杀则无食伤制杀（若有须弱且有财通关）；
  - 从儿则无印克食伤、无官杀混入。
- **不满足则破**：一党内部有克制 → 旺势被打断、气不归一 → 从格不纯。

## 判断维度

### 1. 根的判据（以壬水日主为例）

| 条件 | 判据 | 结果 |
|---|---|---|
| 地支有本气比劫 | 见亥、子 | **不从** |
| 地支有本气印星 | 见申、酉 | **不从** |
| 地支有中气比劫 | 见辰中癸、丑中癸 | 严格论不从 |
| 地支有中气印星 | 见戌中辛、巳中庚 | 可论从（力微时） |
| 所从之气透干且通根 | 所从之神清晰 | **可论真从** |

### 2. 三合三会、合化、刑冲对根的影响
- **三合三会是否"洗掉"他气**：不洗。哪怕寅卯辰三会把丑土"压"住，丑中癸水仍是根 → 不从。
- **合化是否消除他气**：不消除。戊子合、丑戌合等只是遮蔽、不消根。
- **刑冲是否可以拔根**：冲库根（辰戌冲、丑未冲）确实能拔根 → 库被冲破 + 藏干无用时可论无根。

### 3. 顺从方的细分

| 从格类型 | 月令要求 | 数量要求 |
|---|---|---|
| **从财** | 月令财星或参与三合三会财局 | 财星 ≥ 食伤、财星 > 官杀；最好有食伤生财 / 财生官杀 |
| **从杀** | 月令官杀或参与三合三会官杀局 | 官杀 ≥ 财星、官杀 > 食伤；可有微弱食伤，最好有财通关 |
| **从官** | 月令正官或参与三合三会官局 | 全局无七杀混入、无食伤（伤官见官立破） |
| **从儿** | 月令食伤或参与三合三会食伤局 | 食伤 ≥ 比劫、食伤 > 财星；无印无官杀 |
| **从势** | 财官食伤同时强旺、无一方独大 | 详见 [从势格.md](从势格.md) |

## 喜忌总则

### 喜神
- **顺从对象本身**（财 / 官杀 / 食伤）。
- **能生顺从对象的五行**（从财喜食伤、从杀喜财）。
- **顺流下去的五行**（从财之后喜官杀泄财；食伤生财的"吾儿又遇儿"顺流链）。

### 忌神
- **印星**：生扶日主 → 激怒旺神 → 招灾。
- **比劫**：帮身抗旺 → 破从。

### 大运判断

| 大运类型 | 对真从格的影响 |
|---|---|
| 走顺从对象及其生助方 | **大喜** → 贵格兑现、事业高峰 |
| 走食伤（从儿 / 从财 / 从势适用） | 喜、但过重可能泄旺 |
| 走官杀（从财 / 从杀 / 从势适用） | 喜、但过重压身 |
| **走印运** | **大忌** → 破从 → 母亲有灾、学业受阻 |
| **走比劫运** | **大忌** → 夺财 / 抗杀 / 帮身 → 破财、兄弟疏远 |

## 现实表现

### 共性
- 日主弱到极致 → 不再与旺势对抗 → 反而与旺势气场同步。
- 福报比普通身弱命造大得多。
- 大运顺时，成就远超常规命造；大运逆时，跌落也极剧烈。
- 从格之人像"**顺毛驴**"——顺着喜神走则飞黄腾达，逆着走则立马撂蹶子。

### 各从格的典型气质

#### 从财格
- **多情重义、异性缘爆棚**：对感情真挚、对朋友守信。
- **爱面子爱漂亮**：穿衣讲究、在意他人看法。
- **勤俭理财能力强**：对钱敏感、脑中有本账。
- **企划能力强、奋发向上、交际能力佳**。
- **嫉恶如仇、爱憎分明**。
- 事业路径：自身即为财星化身、财资追随；适合做生意、投资、文化娱乐、金融等一切以财为中心的领域。

#### 从杀格
- **大佬气场**：不怒自威、一坐下四周气氛就不同。
- 有的人天生长一张"大佬的脸"——穿龙袍便是皇帝、穿便衣也见不凡。
- **行事果决、杀伐决断**：说干就干、不拖泥带水。
- **领导才能出众、言出必行、责任感重**。
- **管理能力强**：会用人也会驾驭人。
- **好胜心强、不畏强权、具坚韧骨气**。
- 事业路径：军警、政法、高层管理、影视圈的权力线、靠权力吃饭、生杀大权在手。
- 因官杀为喜、贵人云集——身边围绕的领导不但不害他、反而主动提携。

#### 从官格
- **端正守规、适应体制、不张扬、温顺上行**。

#### 从儿格
- **聪明秀气**：脑子转得快、反应灵敏。
- **创造力强**：不喜旧路、总能想出别人想不到的点子。
- **文章卓越、善计谋略**：表达能力强、做策划写文章都是一把好手。
- **恭敬慈善、适应力强**：到什么山唱什么歌。
- **对音乐艺术有偏好**、文艺天赋出众。
- **社交能力佳、终生有靠、深得下属拥戴**。
- 事业路径：**伤官伤尽反可坐官为领导** → 非只能搞艺术，入任何行业皆可成佼佼者（艺术家、科学家、作家、管理者）。

#### 从势格
- **恭敬有礼、处事圆满、心地善良**。
- 待人接物客气有礼、处世圆滑、多赛道并行。

## 破格与救格

### 破格情形
- 地支出现任意印比之本气 / 中气 / 余气根 → 真从立破。
- 旺神中途被另一五行克制（如从财见比劫夺财、从杀见食伤制杀）→ 气不归一。
- 旺神内部混杂、无一方独大 → 转从势格。

### 救格
- 天干虚浮印比被财克、被官杀制、被食伤泄尽 → 不再破从、可转假从格讨论。
- 刑冲虽拔根但若旺神更旺、反成真从之助。

## 大运流年影响

### 走顺势运
- 喜神得力 → 事业、财运、名声节节高。
- "**格局足够大时，常规断法要让路**"——真从格在顺势运中常突破常规判断。

### 走印运 / 比劫运
- 最凶的是**天干地支一气的忌神运**（如甲寅、乙卯之于从财格）。
- 轻则破大财、重则有性命之忧。
- 若再逢同类忌神流年、"**岁运并临**" → 生离死别之险。

### 走非一气忌神运
- 若忌神运的干支本身有泄化（如忌神甲木坐午火泄、或甲木被其他喜神克化），力量不大 → 只会变成普通人、比普通中产以上过得好不成问题。
- 这就是从格的福报：**哪怕走差运也比一般人过得好**。

## 常见误区

### 1. "身弱一党旺 = 从格"
- 最大误判。真从格极罕见。
- 判定前先问：**日主还有没有一丝根**？有 → 身弱破局；绝无 → 才可能真从。

### 2. 真从 = 必大富大贵
- 真从是贵格不假、但**极度依赖大运**。
- 顺运一飞冲天；逆运（尤其一气忌神运）暴起暴落是常态。

### 3. 忽视"从"的心理代价
- 从财：精神依附于钱、钱失则自我崩。
- 从杀：内心长期高压、外刚内焦虑。
- 从儿：才华横溢但不走寻常路、常被世俗不理解。
- 从势：内心分裂、不知自己是谁。

### 4. 把"从"当作"软弱"
- 从不是怯懦、是**承认力量悬殊后借势**。
- 真从格的人往往**胆识过人、能屈能伸**——知进退方能借天地。

### 5. 死抠宫位断事
- 从格的人**长令宫位断事常失灵**——伤官在年本主败祖业，但真从财格家境仍可不错；此乃"**格局足够大时常规断法要让路**"。

## 一句话总结

- 真从格 = **日主无根 + 旺神满局 + 绝无印比**。
- 顺之者贵、逆之者凶。
- 看从格八字的核心：**找到他顺从的那个五行、然后顺着它走**。
