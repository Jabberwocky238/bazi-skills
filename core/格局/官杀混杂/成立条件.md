# 官杀混杂判断维度

### 1. 官杀混杂的两种形态
- **显混杂**：天干同时透出正官 + 七杀 → 最典型。
- **隐混杂**：天干只透其一，地支藏干另一个 → 轻度混杂。
- **流年引混杂**：原局清，流年来官 / 杀 → 临时混杂。

### 2. 混杂的严重度
- **两者力量相当**：最难处理 → 两头拉扯。
- **一强一弱**：主从清 → 可以通过"留主去从"化解。

### 3. 去留的方向
传统命理的**救应手段**：
- **去杀留官**：留下柔性规则，去掉刚性压力 → 格局清。
- **去官留杀**：留下刚性挑战，去掉柔性约束 → 格局清。
- 具体怎么"去" → 靠**合、冲、制** 等方法：
  - 食神合杀 / 伤官合官。
  - 比劫分杀 / 印化杀。
  - 食伤制杀 / 财生杀。

### 4. 日主承载力
- **身强**：能扛两套规则 → 可以"**官杀并用**"成为富贵双收的人。
- **身弱**：官杀混杂 = **双重压身** → 破格 → 事业反复、身心两耗。
