# 自坐绝

## 定义
- **绝**：十二长生的第十阶段，象征**断绝、归于虚无** → 能量**彻底归零**的状态。
- **自坐绝**：**日主天干坐在绝位地支上** → 日主在日支**无根无源**。
- 各天干绝位：
  - **甲木** 绝在 **申**。
  - **乙木** 绝在 **酉**。
  - **丙戊** 绝在 **亥**。
  - **丁己** 绝在 **子**。
  - **庚金** 绝在 **寅**。
  - **辛金** 绝在 **卯**。
  - **壬水** 绝在 **巳**。
  - **癸水** 绝在 **午**。

## 典型日柱
- **甲申、乙酉、丙亥、戊亥、丁子、己子、庚寅、辛卯、壬巳、癸午**。
- 注意：绝位是**完全克日主的五行**或**完全泄日主的五行**。

## 核心：无根无援、但恰恰孕育新生
- **绝**的字面意思让人恐惧，但命理上：
  - 绝 = **旧生命的结束**。
  - 紧接的是**胎** = **新生命的孕育**。
  - 所以绝是"**旧终而新始**"的临界点。
- 自坐绝的人：
  - **底子极弱、无根可依**。
  - **但心思极其空灵 / 敏锐**：因为没有"自我执着"的底盘。
  - **极度依赖外缘**：贵人、平台、环境的影响力决定命运。
  - 带**"无中生有"的可能性**：没有任何包袱，反而能创造全新。

## 与相关弱位对比

| 阶段 | 能量 | 气质 |
|---|---|---|
| **衰** | 刚回落 | 老成 |
| **病** | 虚弱 | 多思 |
| **死** | 最低点 | 归零 |
| **墓** | 归藏 | 深藏 |
| **绝**（本条） | 断灭 | **无根无依、白纸状态** |
| **胎** | 孕育 | 潜伏等发 |

**绝是十二长生里最"极端"的弱位** —— 但也最具"可塑性"。

## 判断维度

### 1. 绝位的五行性质
- 绝位**通常是克日主五行 + 所泄五行之源**：
  - 甲绝在申：申是庚金本气（甲之七杀）。
  - 乙绝在酉：酉是辛金本气（乙之七杀）。
  - 丙绝在亥：亥是壬水本气（丙之七杀）。
  - 庚绝在寅：寅藏甲丙（庚之财官）。
  - 壬绝在巳：巳藏丙戊（壬之财杀）。
- 绝位地支**藏的几乎都是"克我 / 耗我"的力量**。

### 2. 原局有无救
- **有印生身**：绝中得救 → **转危为机**。
- **有比劫帮身**：有同伴 → 不至于彻底孤立。
- **无印无比**：**纯粹身极弱 + 无根** → 接近"**从格**"的条件。

### 3. 是否成从格
- 绝位无根 + 全局一方势力压倒 + 无印比抗衡 = **从格**的前提。
- 若满足 → 自坐绝反而是**从财 / 从杀 / 从势**的贵格。
- 若不满足 → 为普通破局。

### 4. 大运流年
- **印 / 比劫运**：给根 → 天地翻转 → 重大转折。
- **顺势方向的运**（若从格）→ 贵格兑现。
- **再克身的大运**：绝上加绝 → 极难捱。

## 常见误区

### 1. "自坐绝 = 早夭 / 必凶"
- **错得离谱**。
- 绝是能量阶段，不是生死判断。
- 许多**历史上的名人、大成就者都自坐绝** → 因为绝的"**白纸状态**"允许全新创造。

### 2. "绝 = 无救"
- 绝反而**最容易形成从格**：
  - 无根 + 势单 → 彻底顺势 → **极致贵格**。
- 绝也可能**靠大运彻底翻盘**：
  - 大运给印比 → 枯木逢春。

### 3. 把绝等同"绝望"
- 绝是"**死的终点 + 生的起点**"。
- 很多自坐绝的人**有一次"脱胎换骨"的人生转折** —— 往往是改命的关键时刻。

### 4. 用强命的方法处理绝命
- 自坐绝 ≠ 努力不够。
- 强行硬拼 → 消耗殆尽。
- 必须**借力 / 顺势 / 等待时机**。

## 现实表现

### 性格
- **敏感到极致**：对外界的感知细如发丝。
- **空灵、不执着**：不像身强者那样"主见强"。
- **极易受环境影响**：跟对人、跟对地方就起飞。
- **有时显得"没主意"**：其实是底子的无根使然。
- **内心有巨大的"未被命名"的空间**：可以被任何好的力量填充。
- **容易在关键时刻"顿悟"**：因为没有固执的框架。

### 事业
- **极度依赖平台 / 机构 / 贵人**：
  - 进对团队 → 迅速发展。
  - 独自打拼 → 极其艰难。
- **适合"**嫁接型**"岗位**：
  - 嫁接资源、嫁接才华、嫁接关系。
  - 顾问、中介、公关、商务开发。
- **艺术 / 修行 / 宗教**：绝的"无我"气质 → 适合这些领域。
- **翻译 / 教育 / 传播**：把一个世界的东西带到另一个世界。

### 感情
- **极易动心**：因为内心是"空的"。
- **容易被伴侣塑造**：伴侣的气质会深深影响自己。
- **婚姻选择极为关键**：选对 = 改命、选错 = 被耗。
- **女命自坐绝**：容易遇到"**主导型**"伴侣 → 需慎选。
- **男命自坐绝**：妻缘的影响对整个人生走向极大。

### 身体
- **先天底子偏弱**：易疲、恢复慢。
- **对环境敏感**：气候 / 饮食 / 作息的变化立刻体现。
- **需要长期养护**：一辈子养生是必修课。
- **长寿仍有可能**：但**养得不好 → 多病**。

### 心理
- **对"存在"的思考深刻**：因为本身就站在"绝"的临界点。
- **可能有**"空虚感" / 存在焦虑"**。
- **需要找到有意义的精神寄托**：信仰、艺术、修行、爱好。

## 自坐绝的**成就路径**

### 1. 借力而生
- 天然需要**贵人 / 平台 / 伴侣**。
- "**攀附强者**" 不是贬义，是自坐绝的生存智慧。

### 2. 选对环境
- 城市、行业、圈子、伴侣 —— 每一个选择都会深刻塑造。
- **主动选择"好的环境"**是自坐绝最大的功课。

### 3. 利用"空灵"优势
- 没有"**自我执着**" → 容易学新东西、接受新观念。
- 走**跨界 / 翻译 / 桥梁**型路径。

### 4. 从格路径
- 如果原局符合从格条件 → **顺势** = 贵。
- 不要硬**装强**。

### 5. 精神修炼
- 绝的"**无**"天然适合修行、哲学、艺术。
- 在**精神 / 宗教 / 文化**层面往往有惊人成就。

## 核心判断
- 自坐绝是"**无根 + 白纸 + 潜力无限**"的特殊底色。
- 识别要点：
  - 有无救应（印 / 比劫）。
  - 是否符合从格条件。
  - 大运是否给转机。
- 成就路径：**借力、顺势、选环境、修精神**。
- 切忌：**硬拼 / 独闯 / 不顾身体**。

## 一句话总结
- 自坐绝 = **你是一张白纸，命运由你遇到的人和事写就**。
- 别怕"**无根**" —— 没有包袱的人反而走得最远。
- 但前提是：**选对环境、遇对人、顺对势** —— 这是自坐绝最大的智慧。

## 古典标准：绝的阶段特征与类象

### 绝的本质

**绝就好比人的形体灭绝后化为泥土，也被叫做"受气"**——就像人还没受孕时，母亲的肚子里空空如也——意思是**万物前气已绝、后气还没来**，就跟万物还没出现似的；也**比喻人气息断绝、没有生机**。

### 绝的记忆口诀（以日主为基准）

- **甲日见申、乙日见酉、丙日见亥、丁日见子**
- **戊日见亥、己日见子、庚日见寅、辛日见卯**
- **壬日见巳、癸日见午**

### 绝是十二长生表格的四大核心

十二长生记忆法的**四个核心对应**中，绝占一席：**绝这一列地支，其藏干本气与天干是冲克关系**。这是快速记忆十二长生的关键法则之一。

### 绝在各宫位的类象

| 宫位 | 类象 |
| --- | --- |
| **年柱** | 得不到祖业，能**在他乡白手起家**；中年后财运好，晚年平安顺遂 |
| **月柱** | 个性急躁激进；不擅长交际，会避开大众而孤立；心情起伏不定；一生多变动——可能主要因为个人的主观意识导致对现实不满，最终孤立无援、朋友不多 |
| **日柱** | 感情方面**夫妻容易分离**，总是喜新厌旧、花哨风流、冲动急躁；事业上**在他乡经营必定成功**，但要注意不要轻易相信别人 |
| **时柱** | 容易对子女完全放任不管，最后得不到子女奉养；所以要关注子女的性格发展，一定要加强教育 |

### 自坐绝的典型日柱

**甲申日、乙酉日、庚寅日、辛卯日**——日柱为绝的四组组合。

### 自坐绝的命主特征

1. 日干坐绝的人**喜欢群居生活、耐不住寂寞**；做事缺乏长远规划和思考；爱好广泛，凡事只有三分钟热度；**不太适合固定踏实的工作**——大多在**艺术、自由职业、娱乐**等领域有不错的发展。
2. **才华不错，思考和理解力也很敏锐**；行为处事比较**以自我为中心**，要多注意别人的举动；性格善良、个性急躁，做事缺乏坚持和执着；容易发怒、**有始无终**；个性胆怯、轻率、多嘴，容易被人误解。
3. **女命**个性开朗，有时候因为语言行为上过于开放所以很容易受到诱惑，经常会因为喜欢而喜欢，所以**婚姻缘分容易发生变化**——特别要注意感情上容易受骗导致情路坎坷，**最好晚婚或者嫁给年纪大的男人**，可能会比较吉利。

### 重要原则：弱也可能克不动

如果一个天干处于**死、绝之地**，那就意味着它的力量**非常弱**——**就算在五行规则上他可以克别人，但由于他太弱了，可能也克不动**。要联系**天干地支之间的作用关系、盖头截脚、负载**综合考虑。

### 古典标准铁律

> 十二长生在八字命理中**只起到辅助作用**，不能作为主要依据；以上类象**准确率并不是很高，只能作为参考**——"绝"是能量断绝、但紧接着就是"胎"，代表着**新生命的孕育开始**。

## 现代解读

以下内容为**现代综合性延伸**：
- "从格路径"（从财、从杀、从势）属于格局篇的专门内容——经典十二长生章节并未直接展开此等论述。
- "白纸状态、可塑性"、"无我气质适合修行"等哲学化延伸属现代表达。
- 具体的职业建议（顾问、中介、公关、翻译、教育）属现代职业归纳。
