# 丑未戌三刑（恃势之刑）

## 定义

- **丑未戌三刑**，又称**恃势之刑**。
- 三刑关系：**未刑丑、丑刑戌、戌刑未**（三土互刑）。
- 地支相刑中**能量与破坏力最大**的一组。

## 核心原理

### 为何称"恃势之刑"

- 丑、未、戌内部是土的杂气。
- 当这些土杂气相互作用时，会加剧混乱，形成**物极土混杂**的复杂局面。
- 涉及各方势力的较量和制衡 → 故称"恃势"。

### 藏干相刑机制

| 刑支 | 本土位置 | 藏干作用 |
| --- | --- | --- |
| 未土 | 南方火旺土象之地 | 藏丁己乙，丁生己 → 土旺；未以旺土克丑中癸水；以丁火克丑中辛金 |
| 丑土 | 北方亥子丑水旺之地 | 藏癸辛己，利用旺水克戌中丁火 |
| 戌土 | 西方申酉戌金旺之地 | 藏戊辛丁，利用辛金克未中乙木 |

### 刑的最终结果

- 从作用力角度：土旺克水、癸水克丁火、丁火克辛金、辛金克乙木——**暗战连锁**。
- 最终结局：
  - 癸水、辛金、乙木**严重受损**。
  - 所有中气、余气被刑罚消除。
  - **只留下越来越旺盛的本气土**。

## 性格与人生含义

- 命带丑未戌三刑者：
  - **逆境中**：容易被他人掌控。
  - **顺境中**：能够掌控他人。
- 可引申为：**因仗势欺人、权势泛滥、为所欲为而受惩罚**。

### 三刑出贵与三刑出凶

- **三刑出贵**：此人公正无私、令人尊敬；相貌端正、眉毛浓密、脸庞宽阔、直言不讳、待人真诚。
- **三刑出凶**：性格乖戾、喜幸灾乐祸、行为古怪、是非不分、喜嘲笑别人的不幸，最后落得孤苦伶仃。

## 成立条件与判断

- 丑未戌是否为吉凶，**完全取决于土在命主八字中是喜神还是忌神**。
- 为喜神 → 吉祥而有力。
- 为忌神 → 意味着灾祸。

### 三刑俱全为忌时

- **夫妻关系紧张**。
- **身体虚弱多病**。
- **女性**：与夫家关系不合、生活孤独清冷。
- **健康**：多患**脾胃疾病、皮肤疾病**（土旺所致）。
- **六亲**：藏干五行混乱交战，土越旺其他藏干越弱甚至消散——过旺或过弱的五行对应六亲皆有不同程度影响和损害。

## 宫位应事

- 所刑宫位决定应哪一位六亲：
  - **月柱被刑** → 兄弟姐妹有灾。
  - **日柱被刑** → 配偶有灾。
  - **年柱被刑** → 长辈（常为母亲）有灾。
  - **时柱被刑** → 子女有灾。
- 刑到日支**夫妻宫** → 感情不利；谈恋爱易分手，已婚者夫妻出现重大变故甚至离婚。

## 应期

### 原局已有三刑 + 土为忌

- **每逢丑、未、戌年份或大运** → 三刑被触发。
- 特别是**戊己土透出天干的月份** → 极大可能破财、分手、健康出问题。

### 原局仅有一两支

- 岁运补齐缺口时触发。
- **原局自带三支或两支者**引发频率相对较高，但由于天干未透，灾情相对较小，命主逐渐适应。

### 最危险情境

- 原局**只自带三刑中的一个地支**。
- 大运和流年**各带来一个地支**凑齐。
- 同时**天干透出戊己土**。
- **→ 往往给命主带来极大灾难**，因命主对此毫无准备。

## 案例

一个男性八字，日元极强且无法从强，喜财官、忌火土：
- 年支未土、日支戌土。
- **甲午大运、辛丑流年** → 丑补齐，构成丑未戌三刑。
- **刑到日支（夫妻宫）**：感情不利；恋爱易分手，已婚者夫妻变故甚至离婚。
- **刑到年支（母亲宫）**：年支未土同时是庚金正印——代表命主母亲。星宫同位情况下，基本可判定母亲会出现健康问题。
- 土旺 → 脾胃疾病、皮肤疾病；土旺克水 → 肾脏或泌尿系统问题。
- 应期月份：戊己土透出天干的月份。

## 现实象意

- **健康**：脾胃、皮肤、肾脏、泌尿系统。
- **财务**：破财、投资失败、家业耗散。
- **婚姻**：夫妻不睦、女命与夫家不合、感情破裂。
- **母亲健康**：星宫同位时尤其明显。
- **人际权力**：因恃势逞强而受挫，或被权势所困。

## 常见误区

- **"必有大灾"**：若命主喜土，反能因刑得助；忌土才是凶兆。
- **"必须三支齐全才应"**：岁运补齐即可应验。
- 害怕三刑 → 实际三刑俱全自带者反而**一生起起落落已成习惯**，真正可怕的是**突然被岁运凑齐**的人。

## 一句话总结

- 丑未戌三刑是三土互刑，**土越刑越旺，越旺越伤藏干**。
- 应验不应验看喜忌；灾难大小看是否有心理准备。
- "君子行出贵"——既然带了三刑，不妨**以平常心对待，苦中作乐**。
