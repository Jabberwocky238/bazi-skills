# 母慈灭子格

## 成立条件（严格依《滴天髓·母慈灭子》）

### 条件 1：满盘皆印，印绶过旺到克身的程度

- **古籍出处**（本条最核心）：
  - 《滴天髓》："**母慈灭子关头异**。"
  - 《滴天髓》任铁樵注："**满盘皆印，原局无财，或财微浮无根**——印旺而反灭其子。"
- **判据**：
  - ① 印星（正印 + 偏印）在天干 + 地支本气共三位以上；或
  - ② 月令 + 地支三合 / 三会成印局，印气统治全局；
  - ③ 日主虽看似"被生"，但印过旺反生成克（"土多金埋""水多木漂""木多火塞""火多土焦""金多水浊"）。
- **不满足则破**：印一般旺 + 命局有其他平衡之物 → 只是身弱可用印，非"母慈灭子"。

### 条件 2：原局无财星（或财星浮弱无根）

- **古籍出处**：
  - 《滴天髓》（任铁樵注）："**无财为臣以护君主**——无财则印无制、反克日主。"
- **判据**：
  - 原局天干无财星透出；或财透但无根极弱。
  - 地支无财本气；若仅藏支一位余气，力微可忽略。
- **不满足则破**：原局有有根之财 → 财能制印，母慈不灭子 → 转入"以财破印"（救格）讨论。

### 条件 3：日主被印过旺所压，呈"母旺灭子"之象

- **古籍出处**：
  - 《滴天髓》："**五行过生反主灭**——金赖土生，土多金埋；水赖金生，金多水浊；等等。"
- **判据**（按日主五行查"反生为克"对应）：
  - 金日（庚辛）+ 土过多 → **土多金埋**
  - 水日（壬癸）+ 金过多 → **金多水浊**
  - 木日（甲乙）+ 水过多 → **水多木漂**
  - 火日（丙丁）+ 木过多 → **木多火塞**
  - 土日（戊己）+ 火过多 → **火多土焦**
- **不满足则破**：印虽多但日主仍能受生、不被压灭 → 非母慈灭子。

### 条件 4：命局无比劫救应（或比劫仅为浮弱）

- **古籍出处**：
  - 《滴天髓》："**母慈灭子，唯比劫可救**——比劫帮身、同时泄印。"
- **判据**：
  - 原局天干无比劫透出且有根；或比劫仅一位且无根力微。
- **不满足则破**：原局有强比劫 → 比劫帮身泄印 → 母慈不灭子 → 转入"印重身弱比劫救应"讨论。

---

**四条同时满足**方为真母慈灭子格。此格为"**从旺格中最底层者**"——印过旺压身、无财无比劫救应。救格唯一路径：**大运流年走比劫**。

## 喜忌总则

- **喜**：**比劫**（唯一救应）——帮身 + 泄印
- **忌**：
  - **印绶运**：母旺益加灭子，母性益盛而子气愈枯
  - **食伤运**：群枭夺食，食伤被强印完全克灭
  - **财星运**：贪财坏印，逆母之性（反而激怒强印）
  - **官杀运**：官生印，母性益发淫威
- **意义**：此格只有一条救应之路，其他任何五行运都不吉。

## 古籍原文

- 《滴天髓》："**母慈灭子关头异**。"
- 《滴天髓·反生之义》："**金赖土生，土多金埋；水赖金生，金多水浊；木赖水生，水多木漂；火赖木生，木多火塞；土赖火生，火多土焦**。"——此即"母慈灭子"的五行机制。

## 判据特征

- **格局层级**：为从旺格中**最底层者**——严格意义上的"被母性压灭的子"。
- **人生画面**：青少年常多病、体弱、不顺；逢比劫运方能扭转；一生依赖比劫的"朋友、兄弟、伴侣"的扶持。
