# 从革格

## 成立条件（严格依《渊海子平·从革格》）

### 条件 1：日主为庚或辛金

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·从革格》："**从革格——庚辛日生人**。"
- **客观判据**：日主天干必须为庚金或辛金。
- **不满足则破**：非庚辛日 → 不入从革格。

### 条件 2：地支会成金局（三合巳酉丑 或 三会申酉戌）

- **古籍出处**（本条核心）：
  - 《渊海子平·从革格》："**地支巳酉丑三合金局，或申酉戌西方金气**。"
- **客观判据**（二者之一）：
  - ① 地支见 巳 + 酉 + 丑 三合金局；或
  - ② 地支见 申 + 酉 + 戌 三会西方金。
  - 若仅半合（巳酉、酉丑、申酉）或二字会气，格品降但勉强成立。
- **不满足则破**：地支不成金局 → 一气不聚。

### 条件 3：天干多透金，戊己土生金更佳

- **古籍出处**：
  - 《渊海子平·从革格》："年月时干宜见庚辛，**有戊己土相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位庚或辛透出；
  - ② 若另见戊或己透干生金，格品升（土生金助势）。
- **不满足则破**：天干金气不足 → 一气未遍布。

### 条件 4：原局无丙丁火透干熔金

- **古籍出处**（本条核心）：
  - 《渊海子平·从革格》："**年月时干无火相克**，主威武显达。"
- **客观判据**：
  - ① 天干无丙火或丁火透出；
  - ② 地支本气无巳、午成势。
- **不满足则破**：火透熔金 → 《渊海子平》"火相克"即明破格 → 转入"火旺金衰"或"火多金熔"讨论。

### 条件 5：原局无甲乙木过重对抗（木亦忌）

- **古籍出处**：
  - 《渊海子平》："从革者，忌财破其气——木为金之财。"
- **客观判据**：
  - 天干不多透甲乙（一位无根尚可）；地支本气寅卯不成势。
- **不满足则破**：木多 → 金需对抗 → 一气被牵动，转入普通身强见财格。

### 条件 6（助力条件）：水秀气 → "金白水清"复合贵格

- **古籍出处**：
  - 《穷通宝鉴·庚金章》："**庚金逢壬水，为金白水清**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 壬水或癸水在命局中一位透干或藏支——此时从革格升级为"从革 + 金白水清"复合贵格。
- **非必要**：不满足不破格；满足则格品升（锋锐之外添清秀）。若水多过度泄金则降格。

---

**前五条必须同时满足**方为真从革格；条件 6 关乎格品——纯从革成但偏冷刚，得水秀气则为"清贵"。
