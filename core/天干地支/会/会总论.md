## 三会 vs 三合 vs 六合 的本质区别

### 一、组成原理不同

| 合类 | 组成原则 | 典型代表 |
| --- | --- | --- |
| **三会** | 同一季节的三支 | 寅卯辰（春）、巳午未（夏）、申酉戌（秋）、亥子丑（冬） |
| **三合** | 长生 + 帝旺 + 墓库（精气神俱全） | 亥卯未木、寅午戌火、巳酉丑金、申子辰水 |
| **六合** | 一阴一阳两支相合 | 子丑、寅亥、卯戌、辰酉、巳申、午未 |

### 二、力量与稳定性

| 合类 | 力量 | 稳定性 | 类比 |
| --- | --- | --- | --- |
| **三会** | **最大** | **最不稳定**，易被冲破（尤怕相冲） | 老乡会、联谊会、临时拼凑的团伙 |
| **三合** | 很大 | **最稳定**，其他合作用难以破之 | 结拜兄弟、桃园结义、铁三角 |
| **六合** | 中等 | 中等，遇三合让位 | 一阴一阳的紧贴结合 |

### 三、合化条件的差别

| 条件 | 三会 | 三合 | 六合 |
| --- | --- | --- | --- |
| 三支 / 两支齐聚 | 三支必全（**无半三会**） | 三支必全（缺则半合 / 拱合） | 两支必全 |
| 紧贴 | **不要求紧贴** | **要求紧贴**（正三合或暗三合） | **要求紧贴** |
| 顺序 | 不要求 | 不要求（中神居中最佳） | 无 |
| 透干 | **须透化神五行** | **须透化神五行** | **须透化五行** |
| 中神要求 | 无中神概念 | 中神（帝旺）最好占月令或当旺 | 无中神概念 |
| 无严重冲克 | 须 | 须 | 须 |

### 四、半局机制的差别

- **三合有半合**（生地半合、墓地半合）**和拱合**（缺中神）。
- **六合没有半合**，两支即合。
- **三会没有半三会**，缺一字则必等岁运补齐方能成局。

### 五、化解冲刑害破的能力

| 合类 | 化解能力 |
| --- | --- |
| **三合** | **最佳**。一旦形成，**贪合忘冲、忘刑、忘害、忘破**，其他合作用让位于三合 |
| **六合** | 能化解六冲（紧贴合冲时"贪合忘冲"），但**遇三合让位**，化解力度弱于三合 |
| **三会** | 力大但自身**易被冲破**，化解效果不如三合稳定 |

### 六、应事性质的差别

- **三会** → 代表**人生重大变化、季节性转折**。
  - 本命局已有 → 通常不剧烈（已与全局平衡）。
  - **岁运补齐缺字 → 重大应期**：好事更好、坏事更糟。
- **三合** → 代表**稳定的合作、团队、贵气流转**。
  - 合成喜用 → 财多利厚 / 仕途扬鞭 / 文贵学成。
  - 合成忌神 → 祸患反重。
- **六合** → 代表**情感结合、合伙关系、化冲解绊**。
  - 各宫位见六合，分别对应祖辈、兄弟、婚姻、子女象意。

### 七、批命实务的应用差别

- **看三会** → 推算**人生重大转折应期**（岁运何时补字）。
- **看三合** → 分析**合作稳定性、格局成立、化解冲刑害破**的能力。
- **看六合** → 辨识**情感、合作、宫位事件**的象意与化冲机制。

## 合的总力量排序

**三会 > 三合 > 生地半三合 > 六合 > 墓地半三合 > 暗合**

- 力量排序须结合**全局**动态判断，非绝对。
- 力量大不等于稳定，稳定与否才是**实际发挥作用**的关键。

## 核心判断原则

1. **三会力大而散** → 适合判重大转折，不适合判长期稳定。
2. **三合稳定而贵** → 适合判合作成局、格局成立、化解冲刑害破。
3. **六合紧密而绵** → 适合判情感姻缘、人际合作、化冲机制。
4. **遇冲刑害破**，化解能力：**三合 > 六合 > 三会**。
5. **合的吉凶**始终取决于**合出的是日元的喜用还是忌神**。

## 古训警示

- **新手怕冲，老手怕合**——合的作用关系比冲更为复杂精细。
- **力大未必吉，稳定方为根**——这是三会、三合、六合取舍之要。

## 一句话总结

- **三会**：同季三支聚为一方之气，**力最大、最不稳**，是人生重大转折的应期标志。
- **三合**：精气神俱全的**稳定贵气**之合，合作、格局、化解冲刑害破的王牌。
- **六合**：一阴一阳的紧贴合化，**情感、姻缘、化冲**之基础。
