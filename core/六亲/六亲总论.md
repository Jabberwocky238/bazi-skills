# 六亲总论

## 什么是六亲

八字里的"六亲"不是民间口耳相传的笼统概念，而是以**日主为中心**向外延伸出的六条关系线：

- **父母**：生身之源（父 + 母）
- **兄弟姐妹**：同气之辈
- **配偶**：结发之亲
- **子女**：延续之脉

有些流派把"父母"合为一亲，加入"祖辈"或"朋友"凑成六；本系按最主流版本——**父、母、兄弟姐妹、配偶、子女**——分述，另以专篇补祖辈与情人。

## 核心原则

六亲**不是由"一个符号"对应"一个人"**，而是由**十神 + 宫位 + 气机**三者共同呈现。
单独记"印=母、财=父"这种口诀是入门，不是全部。

## 六亲的十神对应（男女有别）

| 关系 | 男命 | 女命 |
| --- | --- | --- |
| 父亲 | 偏财（主）/ 财星 | 偏财（主）/ 财星 |
| 母亲 | 正印（主）/ 印星 | 正印（主）/ 印星 |
| 兄弟 | 比肩（同性）/ 劫财（异性） | 劫财（同性）/ 比肩（异性） |
| 姐妹 | 劫财（同性）/ 比肩（异性） | 比肩（同性）/ 劫财（异性） |
| 配偶 | **正财**（妻）/ 偏财（偏妻、情人） | **正官**（夫）/ 七杀（偏夫、情人） |
| 儿子 | **七杀** | **伤官** |
| 女儿 | 正官 | 食神 |

**关键差别**：

- 男命以"**我克者**"（财）为妻、父，以"**克我者**"（官杀）为子女。
- 女命以"**克我者**"（官杀）为夫、情人，以"**我生者**"（食伤）为子女。
- 印星、比劫的父母 / 兄弟姐妹对应男女相同。

## 六亲的宫位对应

四柱宫位是六亲的"**位置**"：

| 宫位 | 主要承载 |
| --- | --- |
| 年柱 | 祖辈、父母（远景）、家族根基 |
| 月柱 | 父母（近景）、兄弟姐妹、青年社会关系 |
| 日柱 | 自身（日干）+ 配偶（日支 = 夫妻宫） |
| 时柱 | 子女、晚年、事业归宿 |

**宫位与十神的配合原则**：

- **星宫同位**（如男命正财在日支）→ 六亲象清晰、关系稳定。
- **星宫分离**（如妻星不在日支）→ 需综合判断，关系易有位置错置。
- **宫位逢冲刑破害** → 对应六亲位置不稳。
- **宫位逢合** → 关系紧密，或有暗中牵连。

## 喜忌判断原则

判断某位亲人的"好坏"，不是看十神强弱表面数字，而看：

1. **该十神是否为喜用**
   - 妻星（正财）为喜用 + 不被克破 → 妻贤、婚美。
   - 妻星为忌 + 被劫冲 → 妻多病、婚易变。
2. **该宫位是否安稳**
   - 夫妻宫不逢冲 + 配偶星入位 → 婚姻安稳。
   - 夫妻宫逢冲、被合走 → 婚姻波动。
3. **该十神气机是否流通**
   - 孤立、被截脚、被盖头 → 六亲关系僵。
   - 有生有泄、有救应 → 六亲关系活。

## 六亲紧张 / 和谐的典型结构

### 和谐结构

- **财官印食俱全，生化流通** → 父母、配偶、子女皆有位置。
- **日主中和，各星各守其位** → 六亲各安其分。
- **喜用神落在六亲宫** → 该位亲人成为命主助力。

### 紧张结构

- **六亲星全无或极弱** → 对应关系淡漠、缘分浅。
- **六亲星过旺无制** → 对应关系反客为主，压迫命主。
- **六亲星被冲克合化** → 该位亲人多灾、早别或关系破裂。
- **六亲宫与命主五行相战** → 内在对立，难以和顺。
- **官杀混杂、比劫争财、印枭夺食** → 对应关系层次混乱。

## 论命次第

看六亲须按下述顺序，不可颠倒：

1. **先定喜忌**：日主旺衰 → 喜用忌神。
2. **再看十神**：六亲星在局中是喜是忌、是旺是衰。
3. **再看宫位**：六亲宫是否受刑冲合破。
4. **后看岁运**：大运流年引动六亲星 / 宫时的应事。
5. **最后整合**：三象一致才下结论，三象分离须存多解。

## 一句话总结

**六亲不是八字里的几个固定符号，而是日主气机向外延伸时，与家庭结构发生对应的若干作用关系——读懂十神、宫位、喜忌三层，才读得懂一个人的家。**
