# 杀印相生

## 成立条件（严格依《子平真诠》《滴天髓》）

### 条件 1：七杀透干且通根，或月令本气为七杀

- **古籍出处**：
  - 《子平真诠·论七杀》："七杀宜透不宜藏，透则其性显。"
  - 《子平真诠·论七杀》："**杀不离印，印不离杀，杀印相生，功名显达**。"
- **客观判据**：
  - ① 七杀透于年 / 月 / 时干，且地支有本气根至少一位；或
  - ② 月令地支本气为七杀。
- **不满足则破**：七杀仅藏支一位余气而不透干 → 杀气不显，无可"化"之对象，格不成立。

### 条件 2：印星透干且通根

- **古籍出处**：
  - 《子平真诠·论七杀》："杀用印化，**印须有根**，方能制杀。"
  - 《滴天髓·官杀》："化杀须得印生旺。"
- **客观判据**：
  - 印星（正印 / 偏印皆可）透于天干，且在地支本气或中气通根至少一位。
- **不满足则破**：印虚透无根 → 沈氏"虚化无力"——印自身微弱，化不动七杀，链接不成。

### 条件 3：七杀 → 印 → 日主三者位置连贯（紧贴通关）

- **古籍出处**（本条核心）：
  - 《子平真诠·论用神》："用神须紧贴日主，方为真用。"
  - 《滴天髓·通神论》："**贴身者力重，遥隔者力轻**。"
- **客观判据**：
  - 七杀与印至少相邻一柱，且印与日主紧贴；典型结构：
    - 七杀在月干、印在日支（贴日主），或
    - 七杀在年干、印在月干、日主居后，顺流相生。
- **不满足则破**：七杀在年柱、印在时柱中间隔开 → 七杀先克日主，再到印已过日主，链条反向。

### 条件 4：原局无财星紧贴克印

- **古籍出处**：
  - 《滴天髓》："财能破印，杀印格最畏。"
  - 《子平真诠·论七杀》："杀印之局，最忌财星。"
- **客观判据**：
  - 天干无财星与印星紧贴相克；若有财，须比劫隔之，或财被合去 / 被冲去。
- **不满足则破**：财贴印无救 → 印被克 → 七杀脱化 → 变成纯"七杀克身"的破局。

### 条件 5：日主不至极弱（印有所载）

- **古籍出处**：
  - 《滴天髓·官杀》："**杀印相生，身旺为贵**。"
  - 《滴天髓·体用》："衰者宜帮不宜损。"
- **客观判据**：
  - 日主于月令为比印余气以上，或他柱至少有一位比劫 / 印星通根。
- **不满足则破**：日主极弱 → 沈氏"印反为累"——印太重反压日主，变成"印重身弱"而非相生。
- **附**：若日主完全无根且七杀极旺而无他物相助 → 须考虑是否转入"从杀格"讨论（反面极端）。

---

**五条同时满足**方为真杀印相生。本格与"食神制杀"之别：食制为"克杀"（主动），印化为"化杀"（接纳）——《滴天髓》明示："**制杀太过，反为不利；化杀得宜，方为妙用**"。
