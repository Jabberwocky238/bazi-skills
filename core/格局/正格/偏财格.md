# 偏财格

## 定义
- **偏财**：日主所克之同性五行——阳日主克阳为偏财；阴日主克阴为偏财。
  - 例：甲日见戊（阳土）为偏财；乙日见己（阴土）为偏财。
- **偏财格**：**月令取偏财为用神**的基础正格。
- 《子平真诠》十正格之一——**机会型财富结构**。

## 核心：众人之财
- 古籍《子平真诠·论偏财》明言："**偏财者，非我正妻之财，乃众人之财**。"
- 偏财 = 大家都可争的财、流动的财、机会的财。
- 对应现代：**投资、商贸、中介、副业、意外之财**。

## 偏财 vs 正财

| 对比项 | 偏财 | 正财 |
|---|---|---|
| 阴阳 | 与日主相同 | 与日主相反 |
| 来源 | 机会、众人、灵活 | 劳动、稳定、合规 |
| 性格 | 慷慨、豁达、重情义 | 节俭、稳重、守成 |
| 典型 | 商人、投资者、中介 | 白领、公职、技工 |
| 气质 | 豪气、外放 | 细致、内敛 |

## 成立条件（严格依《子平真诠·论偏财》）

### 条件 1：月令地支本气为偏财，或偏财透干且月令有根

- **古籍出处**：
  - 《子平真诠·论偏财》："**月令偏财，取用为格**。"
- **客观判据**：
  - ① 月令地支本气为偏财；或
  - ② 月令地支藏偏财且偏财透于天干。
- **不满足则破**：月令偏财不显 → 当以月令本气另论他格。

### 条件 2：偏财透干且通根

- **古籍出处**：
  - 《子平真诠·论偏财》："**偏财喜透，透则显达**。"
- **客观判据**：
  - 偏财透于天干，且在地支有本气或中气根至少一位。
- **不满足则破**：偏财虚透无根 → 财气浮而不实，钱来钱去。

### 条件 3：日主身强能任财

- **古籍出处**（本条核心）：
  - 《子平真诠·论偏财》："**偏财喜身旺**。"
  - 《子平真诠·论财》："身旺能任财，财多身弱则贫苦。"
- **客观判据**：
  - 日主于月令为比劫 / 印星本气，或他柱至少两位比劫 / 印星通根。
- **不满足则破**：身弱偏财多 → 财多身弱破局，易破财、被骗。

### 条件 4：原局无比劫紧贴夺财（对偏财尤严）

- **古籍出处**（本条核心）：
  - 《子平真诠·论偏财》："**偏财最忌比劫**。"
  - 《子平真诠·论财》："财怕比劫之夺。"
- **客观判据**：
  - 天干无比劫与偏财紧贴相克；若有比劫，须有食伤化比劫（食伤通关），或官杀制比劫救财。
- **不满足则破**：比劫贴财无救 → 钱来钱散、合伙翻船、兄弟争财。
- **注**：偏财为"众人之财"，比劫夺之尤烈——偏财格比正财格更怕比劫。

### 条件 5（辅佐条件）：食伤生财 / 财生官为贵

- **古籍出处**：
  - 《子平真诠·论偏财》："**偏财生官，富而且贵**。"
  - 《子平真诠·论财》："食伤生财，财源不竭。"
- **客观判据**：
  - ① 食伤透干有根，位于偏财前（食→财）；或
  - ② 官星透干有根，位于偏财后（财→官）。
- **非必要**：不满足不破格，但格品降。

---

**前四条必须同时满足**方为真偏财格；条件 5 关乎格品。偏财格的现代典型：**商人、投资者、中介、副业经营者**——钱不是死工资，是**流动中获取的**。

## 判断维度

### 1. 偏财与大运
- **走财运**：赚钱年——偏财得财快而多。
- **走比劫运**：**大忌**——典型的"**破财年**"。
- **走食伤运**：生财源 → 事业扩展期。
- **走官杀运**：财生官 → 可能转入管理 / 地位提升。

### 2. 偏财在不同位置
- **年柱偏财**：祖辈或家族有财、早年有外财缘。
- **月柱偏财**：事业中年的财富主轴。
- **日支偏财**：**配偶有财 / 财运入妻宫**（男命尤显）。
- **时柱偏财**：晚年财 / 子女有财。

### 3. 身财配比
- 身旺财旺 → 真富。
- 身弱财旺 → 财多身弱之苦（见 `财多身弱.md`）。
- 身旺财弱 → 有能力但无大财。

## 常见误区

### 1. "偏财 = 横财"
- 部分正确。偏财确可表现为**意外之财、大额投资收益**。
- 但偏财本质是"**众人流动之财**"——不限于横财。
- 经商、投资、中介等**正当但非工资**的收入都是偏财。

### 2. 把偏财等同"小三财"（男命）
- 传统论"**偏财为妾**"——有时指婚外关系的财。
- 现代语境：更多指**非妻财的收入 / 副业财**。
- 不能简单套"男命偏财 = 出轨"。

### 3. 以为偏财命必大富
- 偏财格确是富命的一种底牌，但需**身能任 + 无比劫夺**。
- 身弱偏财多 → 反而一生求财辛苦。

## 现实表现

### 性格
- **慷慨、豪气**：钱财不计较。
- **重情义、讲义气**：朋友之间大方。
- **机会敏感**：对赚钱门道嗅觉灵。
- **不喜被管**：偏好自己做主。
- **外向**：社交能力强。

### 职业路径
- **商业、贸易、流通**：进货、销售、中介。
- **投资、金融**：炒股、基金、房产。
- **个人品牌、自由职业**：不领固定工资但能挣钱。
- **多元收入**：主业 + 副业 + 投资组合。
- **创业者**：偏财格的典型职业。

### 人生节奏
- **早年多动**：到处找机会。
- **中年暴富机会多**：但需把握时机、守得住钱。
- **晚年视守财能力**：能守则富，守不住则起伏大。

## 核心判断
- 偏财格的核心在"**身能任 + 财有根 + 无比劫**"——三者俱备方真富。
- 配食伤生财 + 财生官 → 富贵双全。
- 识别意义：这类人**走灵活赚钱的路最顺**，硬放在死工资岗位会憋住。

## 一句话总结
- 偏财 = **动起来才有的钱**。
- 别指望坐拿薪水，**要主动找机会、敢投入、能社交**。
- 但前提是**身要强、圈子要清**——否则财来了也留不住。
