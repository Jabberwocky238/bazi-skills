# 正财格

## 成立条件

### 条件 1：月令地支本气为正财，或正财透干且月令有根

- **判据**：
  - ① 月令地支本气为正财；或
  - ② 月令地支藏正财且正财透于天干。
- **取用要点**：正财成格以月令透干或地支本气为核心判定标准、为代表稳定财富、养命之源的吉星。
- **不满足则破**：月令正财不显 → 当以月令本气另论他格。

### 条件 2：正财藏透有讲究，透财须护卫

- **判据**：
  - **正财喜藏于地支**、尤其日支坐下能量强易掌控；
  - 天干多透两三个以上 → 无论身强身弱均易遭比劫夺财；
  - **透财须以官杀贴身护卫**（官杀克比劫才能守财）。
- **取用要点**：透财无护卫则财星亦被争夺、难以守财；比劫夺财为正财格的核心矛盾。
- **不满足则破**：财虚透无根或透财无护卫 → 格难立。

### 条件 3：日主强能任财

- **判据**：日主于月令为比劫 / 印星本气，或他柱至少两位比劫 / 印星通根（身中偏强）。
- **取用要点**：**正财格最喜日主身强**——身强才能担财、承受财富压力；**身弱财旺者难命、一生被财压身**；男命易妻夺夫权。
- **不满足则破**：身弱财重 → 转入"财多身弱"病格讨论（挣钱辛苦、守不住钱）。

### 条件 4：原局无比劫紧贴夺财

- **判据**：
  - 天干无比劫与财星紧贴（月 / 日、日 / 时邻位）直接相克；
  - 若有比劫存在，须有官杀制之（官杀克比劫救财）。
- **取用要点**：**身强遇比劫夺财**时核心喜用为食伤通关（化比劫生财）或官杀制比劫护财——食伤官杀互补伤害、形成"身强泄食伤财官"的良性格局。
- **不满足则破**：比劫贴财无救 → 财被夺 → 挣到钱留不住、或被合伙人 / 家人耗。

### 条件 5：身弱正财格的取用逻辑

- **判据**：身弱而财旺，核心取用为**通关**、**互用助身**：
  - **财克印严重时**：最喜官杀运（官杀通关护印、缓解"财破印致贪财失德"的矛盾）；
  - **财生官杀且有食伤破局时**：最喜印星（印克食伤、借官杀生印生身）；
  - **财生官杀攻身时**：食伤或印比均可为用、但优先印比（印比直接助身更契合身弱需求）。
- **取用要点**：特别要区分辰戌丑未中的**燥土（未、戌）与湿土（辰、丑）**——身弱忌财时、燥土助财为忌、湿土帮身制财为喜。
- **不满足则破**：身弱财杀两旺又无印化杀 → "财党杀"——日主难当。

### 条件 6（辅佐条件）：财生官或食伤生财，链条顺流

- **判据**：
  - ① **财生官**：官星透干有根，位于财后（财→官顺流）；或
  - ② **食伤生财**：食伤透干有根，位于财前（食→财顺流）。
- **取用要点**：**食伤生财**者多靠技艺、经商、口财立身——最忌官杀运（会引发食伤至官杀博弈、格局失衡），这也是纯商人专注经营则顺、强行从政则阻的核心原因。
- **非必要**：不满足不破格，但格品降。

---

**前五条必须同时满足**方为真正财格；条件 6 关乎格品。正财格的核心是**身材平衡、护卫流通**：身强防比劫夺财、身弱助身担财、需注意财星藏透讲究。
