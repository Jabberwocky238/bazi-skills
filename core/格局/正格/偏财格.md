# 偏财格

## 定义
- **偏财**：日主所克之同性五行——阳日主克阳为偏财；阴日主克阴为偏财。
  - 例：甲日见戊（阳土）为偏财；乙日见己（阴土）为偏财。
- **偏财格**：**月令取偏财为用神**的基础正格。
- 正格十格之一——**机会型财富结构**。

## 核心：众人之财
- **偏财 = 非我正妻之财、乃众人之财**——大家都可争的财、流动的财、机会的财、属于"浮财、无主之财"的范畴。
- **相对正财的保守务实**：偏财无"小富即安"的执念、更倾向"富贵险中求"。
- 对应现代：**投资、商贸、中介、副业、意外之财**。

## 偏财 vs 正财

| 对比项 | 偏财 | 正财 |
|---|---|---|
| 阴阳 | 与日主相同 | 与日主相反 |
| 来源 | 机会、众人、灵活 | 劳动、稳定、合规 |
| 性格 | 慷慨、豁达、重情义 | 节俭、稳重、守成 |
| 典型 | 商人、投资者、中介 | 白领、公职、技工 |
| 气质 | 豪气、外放 | 细致、内敛 |

## 成立条件

### 条件 1：月令地支本气为偏财，或偏财透干且月令有根

- **判据**：
  - ① 月令地支本气为偏财；或
  - ② 月令地支藏偏财且偏财透于天干。
- **取用要点**：偏财成格以月令透干或地支本气为核心判定标准、核心指向**意外横财、商业大财、众人之财**。
- **不满足则破**：月令偏财不显 → 当以月令本气另论他格。

### 条件 2：偏财藏透无严苛禁忌

- **判据**：偏财既可以藏于地支能量稳固、也不忌天干透出。
- **取用要点**：**偏财与正财喜藏忌透不同**——天干透偏财者交际能力强、手腕灵活、自带大老板气场；但透干时最怕比劫夺财、需有官杀贴身护卫。
- **不满足则破**：偏财虚透无根 → 财气浮而不实。

### 条件 3：日主身强要求宽松（偏财独特）

- **判据**：
  - **偏财格无需日主非常身强**——即便身弱也可能凭借运势获得意外之财；
  - 但**身极弱又无印比帮身者**难以承载横财、获利后出现健康受损、破财回流等问题（即"担不住财"）。
- **取用要点**：**偏财对身强的要求比正财宽松**——身弱逢偏财旺运未必全凶、具体需结合格局搭配。
- **不满足则破**：日主过弱无印比傍身 → 难担财。

### 条件 4：原局无比劫紧贴夺财

- **判据**：天干无比劫与偏财紧贴相克；若有比劫、须有食伤化比劫（食伤通关）、或官杀制比劫救财。
- **取用要点**：**偏财透干或旺相时最宜有官杀贴身护卫**——官杀可克制比劫、避免财星被夺、形成"财生官、官护财"的良性循环、这是偏财格守财致富的关键。
- **不满足则破**：比劫贴财无救 → 钱来钱散、合伙翻船。

### 条件 5：时上偏财格（偏财独有贵格）

- **判据**：时柱透偏财且有官杀护卫。
- **取用要点**：**时上偏财格**为偏财格独有贵格——**有官杀护卫则晚年多能富贵双全**；若时上偏财无护卫、逢比劫运则易晚年破财。
- **非必要**：非必备、若具备则格品升。

### 条件 6（辅佐条件）：食伤生财 / 财生官杀为贵

- **判据**：
  - ① 食伤透干有根、位于偏财前（食→财）；或
  - ② 官杀透干有根、位于偏财后（财→官）。
- **取用要点**：**食伤生财**组合多靠投机经营、口才智谋、创意策划立身求财——核心优势在精准把握机遇获利；此类格局即使行官杀运、会引发食伤至官杀博弈、轻则事业波折、重则因财犯官非。
- **非必要**：不满足不破格、但格品降。

---

**前四条必须同时满足**方为真偏财格；条件 5、6 关乎格品。偏财格取用核心：**身材平衡、护卫流通**——与正财格三大差异为：**不忌偏财透干**、**对身强要求宽松**、**独有"时上偏财格"**。

## 判断维度

### 1. 偏财与大运
- **走财运**：赚钱年——偏财得财快而多。
- **走比劫运**：**大忌**——典型的"**破财年**"。
- **走食伤运**：生财源 → 事业扩展期。
- **走官杀运**：财生官 → 可能转入管理 / 地位提升。

### 2. 偏财在不同位置
- **年柱偏财**：祖辈或家族有财、早年有外财缘。
- **月柱偏财**：事业中年的财富主轴。
- **日支偏财**：**配偶有财 / 财运入妻宫**（男命尤显）。
- **时柱偏财**：晚年财 / 子女有财。

### 3. 身财配比
- 身旺财旺 → 真富。
- 身弱财旺 → 财多身弱之苦（见 `财多身弱.md`）。
- 身旺财弱 → 有能力但无大财。

## 常见误区

### 1. "偏财 = 横财"
- 部分正确。偏财确可表现为**意外之财、大额投资收益**。
- 但偏财本质是"**众人流动之财**"——不限于横财。
- 经商、投资、中介等**正当但非工资**的收入都是偏财。

### 2. 把偏财等同"小三财"（男命）
- 传统论"**偏财为妾**"——有时指婚外关系的财。
- 现代语境：更多指**非妻财的收入 / 副业财**。
- 不能简单套"男命偏财 = 出轨"。

### 3. 以为偏财命必大富
- 偏财格确是富命的一种底牌，但需**身能任 + 无比劫夺**。
- 身弱偏财多 → 反而一生求财辛苦。

## 现实表现

### 性格
- **慷慨、豪气**：钱财不计较。
- **重情义、讲义气**：朋友之间大方。
- **机会敏感**：对赚钱门道嗅觉灵。
- **不喜被管**：偏好自己做主。
- **外向**：社交能力强。

### 职业路径
- **商业、贸易、流通**：进货、销售、中介。
- **投资、金融**：炒股、基金、房产。
- **个人品牌、自由职业**：不领固定工资但能挣钱。
- **多元收入**：主业 + 副业 + 投资组合。
- **创业者**：偏财格的典型职业。

### 人生节奏
- **早年多动**：到处找机会。
- **中年暴富机会多**：但需把握时机、守得住钱。
- **晚年视守财能力**：能守则富，守不住则起伏大。

## 核心判断
- 偏财格的核心在"**身能任 + 财有根 + 无比劫**"——三者俱备方真富。
- 配食伤生财 + 财生官 → 富贵双全。
- 识别意义：这类人**走灵活赚钱的路最顺**，硬放在死工资岗位会憋住。

## 一句话总结
- 偏财 = **动起来才有的钱**。
- 别指望坐拿薪水，**要主动找机会、敢投入、能社交**。
- 但前提是**身要强、圈子要清**——否则财来了也留不住。

## 古典取用规则补充

### 核心性格特质

- **慷慨热情、重情重义**，颇具侠义心肠，乐于助人且出手阔绰。
- **聪明机灵、乐观开朗**，商业嗅觉敏锐，善于捕捉机遇。
- 与正财格保守务实不同，偏财格无"小富即安"的执念，更倾向"富贵险中求"。
- 容易流于虚浮——挥霍成性、缺乏节制。
- 作为偏星若运用得当，往往具备正星难以企及的魄力与开创性——行事敢于剑走偏锋，输得起也赢得起。
- **感情特征**：偏财亦代表异性缘——成格者往往在感情方面意外收获较多，艳遇概率更高。

### 命理关键要点

- **藏透无颜苛忌**：偏财与正财"喜藏忌透"不同——既可以藏于地支能量稳固，也不忌天干透出；天干透偏财者交际能力强、手腕灵活，自带大佬气场。
- **最怕比劫夺财**：偏财为众人之财，最忌八字中比肩、劫财过旺——若无官杀护卫，财星易被争夺，轻则破财耗财、重则祸患百出。
- **身强要求宽松**：偏财格无需日主非常身强，即便身弱之人也可能凭借运势获得意外之财；但身极弱又无印比帮身者，难以承载横财——往往在获利后出现健康受损、破财回流等问题（即"担不住财"）。
- **护财依赖官杀**：偏财透干或旺相时，最宜有官杀贴身护卫——官杀可克制比劫，避免财星被夺，形成"财生官、官护财"的良性循环，这是偏财格守财致富的关键。

### 取用要诀（三大核心差异）

与正财格不同：偏财不忌透干、对身强要求宽松、独有"时上偏财格"三大特点。

**身强偏财格**：
- 与比劫夺财时，核心喜用为**食伤通关**或**官杀制比劫**——与正财格取用逻辑一致。
- 若命局为食伤生财，多靠投机经营、口才智谋、创意策划立身求财。
- 忌行官杀运——官杀出现会引发食伤克官杀的博弈。

**身弱偏财格**：
- 用神选择核心逻辑为"通关互用、助身强根"——与正财格基本一致，但因偏财对身强要求宽松，身弱之人逢偏财旺运未必全凶。
- 财克印严重时最喜官杀运——通关护印。
- 财生官杀且有食伤破局时最喜印运——印星可克制食伤。
- 财生官杀攻身时食伤或印比均可为用，但优先印比。

### 特殊取用要点

- **格成偏财喜食伤运、财运**：进一步强化食伤生财获利格局。
- **最忌比劫运**：若有官杀护卫则可无惧比劫夺财。
- **偏财格日主过弱者仍需印比帮身**：即便获得意外之财，若无印比增强日主力量也难以守住财富。
- **时上偏财格为偏财独有格局**：时柱透偏财且有官杀护卫者，晚年多能富贵双全；若时上偏财无护卫，逢比劫运则亦晚年破财。

### 案例一：时上偏财喜官杀护卫（男命）

- **八字**：庚寅 乙酉 甲子 戊辰
- **分析**：甲木生于酉月施令，酉金七杀当权。年干透庚金七杀，时柱戊辰为偏财，形成"时上偏财格"。日主甲木地支无根、天干乙木被庚金合克酉金截脚柱里微弱——判定为日主偏弱。
- **喜印比**：木水帮身。
- **核心亮点**：时上偏财格加官杀护卫——虽日主偏弱，但格局架构优良。时上偏财格为偏财独有贵格，若有官杀护卫，即便一生印比大运绵长，走比劫运时也不会损耗富贵。
- **关键作用**：官杀克制比劫，同时形成"财生官→官生印→印生身"的流通链条，完美解决"身弱担财"与"比劫夺财"两大矛盾。
- **运势印证**：男命顺行大运戌亥子丑寅卯辰，戌运之后连续走亥子二步印运（水助力日主强根），后续再走寅卯两步比劫运（木此时寅有原局官杀护卫、比劫无法夺财、反而能增强日主担财之力）——命主一生印比大运长达四十年，运势契合格局需求，属于富贵非凡的好命。
- **印证**：时上偏财格需官杀护卫的核心规则——偏财格即便日主偏弱，只要运势配合、护卫得当，依然能成富贵格局。

### 案例二：身弱不忌偏财透、偏财亦贵（男命）

- **八字**：丙辰 癸巳 壬戌 壬寅
- **分析**：壬水生于巳月临于"绝地"，月令巳火偏财当权。命局中财星火、沙星土两旺，天干透出壬水、癸水两重比劫帮身，无法从弱——判定为普通身弱格。
- **月令巳火偏财透干**明确成偏财格。核心优势：
  - **七杀不透而偏财透出**——原局七杀藏于地支，虽有财星生助但有食神制约、攻身力量有限。
  - **天干透两重比劫**——既缓解身弱之弊，又无过量夺财之患。
- **最喜**：一生印比大运绵长——官杀藏于地支可暗中克制比劫、护偏财，待日主稍旺便能承载财星，实现富贵双全。
- **运势印证**：男命顺行大运午未申酉戌亥子——申有两步印运金紧随其后，助力日主强根；后续亥子两步比劫运进一步增强日主力量，运势与格局需求高度契合。
- **印证**：偏财格与正财格不同——身弱之人未必忌偏财透干，只要有比劫帮身、官杀护卫，依然能成贵格。

### 案例三：日主过弱需印比帮身（男命）

- **八字**：乙卯 乙酉 丁丑 辛丑
- **分析**：丁火生于酉月，月令施令酉金偏财当权。时干透辛金偏财，明确成偏财格。日主丁火地支无根，天干虽有两个乙木偏印且同根地支卯木，但乙木被酉金冲克处于绝地；同时日主无根不受生——无法有效生扶日主。再加上命局中土金旺相持续消耗日主——判定为日主过弱。
- **喜印比**：木火帮身。
- **核心矛盾**：日主过弱无法担财——虽有偏财成格，但日主元气不足难以承载财星能量。命局中三个乙卯木偏印因受损严重、生身之力微弱，无法解决核心问题。
- **运势印证**：青少年时期走甲申、癸未两步忌神大运——甲申运为印运与正财运，财星克印不利学业；癸未运为七杀运与食伤运，土七杀攻身、食伤泄身，进一步消耗日主力量。因此命主岁之前无学历无正式工作，长期依靠父母救济生活；岁之后命主步入寅卯两步比劫运（木），日主力量得到显著增强，才逐渐具备担财能力、开始稳定工作积累财富。
- **印证**：偏财格日主过弱仍需印比帮身的规则——即便偏财为"意外之财"，日主过弱也难以承受，必须借助印比增强元气才能成格。

### 核心要点总结

偏财格取用的核心：**身财平衡、护卫流通**。

与正财格相比的核心差异：
- **不忌偏财透干**。
- **对身强要求宽松**。
- **独有时上偏财格**。

三点把握偏财格"浮财、横财"的核心属性，精准判断身财力量与护卫情况，才能将偏财机遇之财的能量转化为运势助力。
