# 日德格

## 定义
- **日德**：日柱之干支配合成"天地德和之气"——天干与日支同气相生、德行深厚之象。
- **日德格**：日柱为 **甲寅、丙辰、戊辰、庚辰、壬戌** 五者之一，合称"**五日德**"。
- 属**特殊格局**，以日柱论，不由月令十神定格。

## 五日德日柱
| 日柱 | 天干 | 地支 | 配法 |
|---|---|---|---|
| **甲寅** | 阳木 | 寅（木） | 天地同气为木 |
| **丙辰** | 阳火 | 辰（湿土） | 火生湿土 |
| **戊辰** | 阳土 | 辰（湿土） | 天地同气为土 |
| **庚辰** | 阳金 | 辰（湿土） | 土生金 |
| **壬戌** | 阳水 | 戌（燥土） | 土藏水库 |

**记忆口诀**：先记辰三位——丙辰、戊辰、庚辰（阳土配阳干），再配壬戌，最后配甲寅（天地同气为木）。

## 核心原理
- **"德"之本义**：天干与日支关系**同气或相生**，无相克、无刑冲之气 → 命主内在调和，性情宽厚。
- 五日德共性：
  - **阳干 + 四库或自气**：底气厚实、含蓄不张扬。
  - **天地之气不相战**：无内在冲突之苗头。
- 传统评断：
  - **心善稳厚**、慈祥端庄。
  - **魁梧端正**、体貌堂堂。
  - **怜贫敬老**之心：有恻隐之意，对弱者有怜悯。
  - **逢凶化吉、遇难能解**：命中自带"福缓"——遇险能解围。
  - 多有**素食倾向**、对动物与弱者存怜悯。
- 总评：**大吉大善之命**，不仅富贵荣显且**寿命长**、心态乐观。
## 与其他类似格局的对比

| 格局 | 结构 | 与日德的对比 |
|---|---|---|
| **魁罡格** | 四魁罡日柱 | 魁罡主**刚威**，日德主**柔厚** |
| **日贵格** | 四日贵日柱 | 日贵主**贵气**，日德主**福厚** |
| **天德贵人** | 神煞层面 | 天德为神煞；日德为格局结构 |
| **月德贵人** | 神煞层面 | 月德见于月支；日德见于日柱全体 |
| **财官印全** | 正格结构 | 财官印以十神论贵；日德以干支配置论德 |

**特别注意**：
- 日德与天德 / 月德贵人**不是一回事**——天 / 月德是神煞，日德是格局。
- **庚辰日**同时是魁罡日，兼具两种性格——"又刚强又心软、厚道而经常吃亏"。

## 判断维度

### 1. 日德的"纯度"
- 四柱中几位日德干支：位数越多，德气越厚。
- 仅日柱日德 + 他柱混杂破坏：只具日德性格，格未必成立。

### 2. 是否有相冲刑破
- 无冲无刑：德气全备。
- 有冲有刑：德气散失。

### 3. 是否犯魁罡
- 无魁罡或魁罡不冲克：格稳。
- 有魁罡相冲克：破格，吉转凶。

### 4. 日主强弱
- 身强日德：德有依托，福寿兼备。
- 身弱日德：德气难显，易被外力侵扰。

## 现实表现

### 性格
- **心善慈祥**：对人对事存善意，不好斗。
- **稳重厚道**：不急不躁，做事有分寸。
- **恻隐怜贫**：对弱者、动物、长者存敬心。
- **英俊端正**：男命多魁梧、女命多端庄。
- **颇得人缘**：自带让人愿意亲近的气场。
- **素食倾向**：不少日德命主自然选择清淡饮食。
- **吃亏是福**：厚道易被利用，但总能化险为夷。

### 事业
- **教育 / 文化 / 公益**：与"德行"直接相关的岗位。
- **医疗 / 养老 / 社会工作**：怜贫敬老之心的延展。
- **管理中层**：稳重厚道，适合做缓冲与协调。
- **传统行业**：不急功近利，适合长线行业。
- **宗教 / 精神领域**：日德命主常自然向此靠近。
- **非拓荒型**：不同于魁罡之武贵，日德走的是"守成与化解"的路径。

### 婚姻
- **男命日德**：为人和善、颇得人缘；婚姻大多平和稳定。
- **女命日德**：温婉厚道、贤良持家。
- **破格日德**：即使破格也不失日德之性格——只是现实层面不如正格风光。

### 身体
- **寿命长**：传统评断日德格"福必丰厚、寿命绵长"。
- **少大病**：德气调和 → 气血平顺。
- **忌大运冲刑年**：破格运中仍有灾祸，但多能化解。

## 常见误区

### 1. 以为日德 = 天德 / 月德贵人
- **日德是格局**，天德月德是神煞。
- 神煞层面力薄，格局层面结构性强。

### 2. 以为五日德人人皆为日德格
- **必须叠见**才为真日德格。
- 单独一位日德日柱 → 只具日德性格，非真格。
- 日德格极稀有，实务中多为"日德性格"的弱化版。

### 3. 忽视魁罡相冲之特殊破法
- 日德最特殊的破格即为**犯魁罡**——此为本格独有判据，不可漏看。

### 4. 以为日德必富贵
- 日德主"德"与"福寿"，**不必然主富贵**。
- 富贵需再看格局高低、财官配合。

### 5. 把庚辰当纯日德看
- **庚辰日兼具魁罡与日德**——判断时需两套标准并用，更为谨慎。

## 破格与救格

### 破格情形
1. **日支被冲**（辰被戌冲、戌被辰冲、寅被申冲）。
2. **空亡加身**。
3. **七杀紧贴克日**。
4. **犯魁罡相冲克**（日德最特殊的破法）。
5. **刑破集中**。

### 破格后的表现
- 沦为普通之命，但仍保有日德之性格（慈心、素食、英俊得人缘）。
- 富贵荣显无从谈起，但**为人品性仍厚**。
- **案例印证**：丙辰日遇壬戌月 → 天干壬丙相克、地支辰戌相冲，即为典型破格 → 沦为普通人。

### 救格思路
- **冲被合解**：有他支合住冲支，冲得以缓。
- **比劫 / 印运**助身：可稳定日德气。
- **食伤泄秀**：将日德之厚气转化为才德之显。
- **避免犯魁罡**的关键流年。

## 大运流年影响

| 运 | 喜忌 | 效应 |
|---|---|---|
| 比劫 / 印星 | **喜** | 助身扶德 |
| 食伤 | **喜** | 德厚转为才华之显 |
| 财运 | 中性偏喜 | 日德不特别忌财 |
| 官杀运 | **忌** | 七杀犯身，破日德之和 |
| **冲日支之年** | **大忌** | 德气散，关键事件年 |
| **魁罡相冲之年** | **大忌** | 犯本格特殊破法 |

**原则**：日德喜"平和顺遂"之运，最忌"冲破相犯"之运。

## 一句话总结
- 日德 = 甲寅、丙辰、戊辰、庚辰、壬戌**五个带德行之气的日柱**。
- 叠见无破 → **心善厚道、福寿绵长、逢凶化吉**。
- 单日柱日德 → 具日德性格，未必真格。
- 最怕魁罡相冲克——这是日德格**独有的破法**。
- 庚辰日既是日德又是魁罡——**又刚强又心软、厚道经常吃亏**。
