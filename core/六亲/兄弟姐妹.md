# 兄弟姐妹

> 参见：`十神/比劫.md`、`十神/比肩.md`、`十神/劫财.md`、`六亲/六亲总论.md`

## 一、核心对应

- **十神**：兄弟姐妹 = **比肩（同性）+ 劫财（异性）**
  - 男命：比肩 = 兄弟；劫财 = 姐妹
  - 女命：比肩 = 姐妹；劫财 = 兄弟
- **宫位**：月柱（主位，兄弟姐妹宫）+ 年柱（长辈所出的年长手足）+ 日时（同辈 / 朋友辈）
- **作用方式**：与我同气的力量 → 外则现为手足朋友，内则现为自我中的另一股劲

## 二、宫位判断

### 月柱（兄弟姐妹宫 · 主位）

- **月干比肩** → 性情独立、社交能力强、兄弟姐妹中较突出。
- **月干比肩 + 其他柱也有比劫** → 兄弟姐妹较多。
- **月干比肩衰弱 / 逢空亡** → 兄弟姐妹有过激或得不到手足帮助。
- **月支比肩（建禄格）** → 自身强旺、不需印比帮身，喜见财官。
- **月干劫财** → 仗义、朋友多、但为兄弟姐妹吃亏。
- **月干劫财为忌 + 月支有正财** → 一生财富来去不稳、与配偶有冲突。
- **身强 + 月干劫财透出** → 兄弟争财、感情淡漠失和。

### 年柱（长辈所出 / 兄姐）

- **年干比肩** → 个性强，但多非长子长女。
- **年柱干支皆比肩** → 男命易晚婚，女命亦然；与亲生父母缘分浅。
- **年干劫财** → 出生家境不佳、不易得遗产。
- **年柱干支皆劫财（尤五阳干日元）** → 不利父亲、父亲易早逝。

### 日时（同辈 / 子女辈）

- **日支比肩** → 脾气犟、与配偶如朋友、与兄弟姐妹圆、但不利妻 / 不利父。
- **时干比肩** → 男命子息不多、女命婚姻多不顺、晚年事业不易。
- **时支比肩 / 归禄** → 个性独立、晚年或可自安。
- **时柱劫财重** → 子女粗鲁顽劣 / 多败家之子。

## 三、兄弟姐妹数量 / 关系的判断

### 数量

| 结构 | 象意 |
| --- | --- |
| 比劫多、散布四柱 | 兄弟姐妹多 |
| 比劫少 / 被官杀克尽 | 兄弟姐妹少，或独生 |
| 月干比劫 + 其他柱再见比劫 | 兄弟姐妹众多（八零年代后需结合国策背景） |
| 比劫逢空亡 / 夭折 | 兄弟姐妹中有夭折 / 早逝 |
| 比劫遇七杀透干克 | 手足夭折风险提升 |

### 和谐 vs 紧张

| 关系结构 | 象意 |
| --- | --- |
| 比劫为喜 + 有食伤泄 + 无官杀重克 | 手足和睦、相互扶持 |
| 月干比肩为喜用 | 男得姐妹帮、女得兄弟助 |
| 月干劫财为忌 + 身强 + 争财 | 兄弟争产、感情淡、合作易败 |
| 月支比肩被阳刃反覆 | 手足外柔内刚、表面和内里争 |
| 年支比肩 + 月干比肩 | 不论男女皆易晚婚、手足自立门户难相融 |
| 比劫与日主争财 | 兄弟争妻 / 争财，合伙慎重 |

## 四、兄弟姐妹对命主的影响

### 比肩（同性手足）

- 独立型：教命主"靠自己"。
- 竞争型：教命主"不服输"。
- 为忌则：争财夺妻 / 夺官。

### 劫财（异性手足）

- 豪爽型：给命主人际扩张力。
- 破财型：为忌时挥霍、合伙失败。
- 劫财 + 阳刃 → 最凶，易在财 / 女方面重挫。

### 结构性意义

- **比劫 = 同气** → 兄弟姐妹的命运常与命主同向波动。
- 家庭内比劫象的一致性 → 手足常共享家族运势曲线。
- 命主若把比劫之气用在**事业 / 竞争**上，对手足关系反有疏解。

## 五、大运流年涉及比劫的应事

| 岁运组合 | 常见应事 |
| --- | --- |
| 大运走比劫 + 身弱 | 得手足 / 朋友助力，合作起势 |
| 大运走比劫 + 身强 | 兄弟争财、合伙破败、朋友反目 |
| 流年劫财透干 + 财星被克 | 破财、妻病、父忧 |
| 流年比劫合日支 | 兄弟姐妹介入自己的婚姻 |
| 比劫流年冲夫妻宫 | 桃花外扰、婚姻不稳 |
| 羊刃流年 + 官杀弱 | 冲动决策、手足遭意外 |
| 流年官杀制比劫得当 | 手足有所约束，关系回稳 |

## 六、古籍补充（作为真实标准）

- **年干劫财 + 年支劫财 → 不利父亲，父亲容易早逝**——"劫财克财（父）"的直接表现。
- **月干比肩 + 其他柱有比劫 → 兄弟姐妹多**；**衰弱逢空亡 → 手足过激 / 难得帮**。
- **月支比肩称建禄格 → 天干透官者贵、透财者富**——此为兄弟宫最常见的"**好命结构**"。
- **月干劫财为忌 → 男命与妻不睦、常吃亏于朋友兄弟；十五岁之前家境清苦**。
- **月支劫财 + 身强官杀克尽 → 出生家境贫寒**。
- **比劫如阳刃成片去克财 → 对父亲 / 妻子极不利，幼年易失恃失怙**。
- **比劫为喜 + 月干透出** → 男得姐妹助、女得兄弟助；若空亡则助力薄且手足易早逝。
- **时柱劫财重 + 身强为忌** → 晚年辛苦劳累、妻子子女易早逝。

## 七、最常见的误判

- 把"比劫多"直接等于"兄弟多、朋友多" → 未分喜忌。
- 把"比劫旺"看作"好" → 比劫旺更可能克财克父。
- 忽略男女差异：女命比肩不是"姐妹"那么简单，还常牵动**官星（夫星）** 被争夺的隐患。

## 八、一句话总结

**兄弟姐妹 = 比肩劫财为星、月柱为宫——比劫为喜则手足相扶，比劫为忌则争财夺爱；同气之人同命运，读懂比劫就读懂了"与你一路走来的同龄人"。**
