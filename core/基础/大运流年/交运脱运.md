# 交运脱运

## 定义
- **交运** = 进入新大运的**准确时间节点**。
- **脱运** = 离开旧大运、旧运气场正式退场的节点。
- 两者通常**不是同一个点**——脱运先于交运（或重叠），中间隔着一小段"**真空过渡带**"。

## 交运与脱运的时间计算

### 交运的精准时点
- 大运每 10 年一换，但"这一天几点几分"并非模糊概念。
- **起运岁数 = 出生到节气的日数换算**：三日折一岁、一日折四月、一时折十日。
- 按此规则往后每 10 年推一次，即可得到每一步大运的**准确交运日期、甚至时辰**。

### 示例
- 某命 **5 岁 3 个月零 10 天**起运，出生日期为农历 X 月 Y 日。
- 第一步大运**交运点**：出生日期 + 5 年 3 月 10 日。
- 第二步大运交运点：第一次交运 + 10 年（同月同日）。
- 依此类推。

## 交运与脱运的关系

| 阶段 | 状态 | 体感 |
| --- | --- | --- |
| **脱运前 1—2 年** | 旧运已衰、命主已无法再享受旧运之利 | 事业、感情等领域**后劲不足** |
| **脱运点** | 旧运正式退场 | 明显的"断点"感——某件事告一段落 |
| **真空过渡带** | 旧运退、新运未稳 | **气机最乱**，判断力弱，易做错决定 |
| **交运点** | 新运正式登场 | 风向切换——新人、新事、新机会出现 |
| **交运后 1—2 年** | 新运进气中 | 体感逐步清晰，新格局开始成型 |

## 交脱运节点的应事密度

### 为什么交脱运节点应事特别密
- 两股气机**交接博弈**，原局多个字同时被触动。
- 天干引动 → **事件的定性**出现。
- 地支引动 → **事件的定量**落地。
- 因此**交脱运点前后 2—3 年**，是人生大事的**高发期**。

### 常见应事类型

| 领域 | 交脱运期的典型事件 |
| --- | --- |
| **事业** | 辞职、创业、升迁、转行、退休 |
| **感情** | 结婚、离婚、分手、复合、重要桃花 |
| **家庭** | 父母离世、子女出生、成家分家 |
| **健康** | 突发疾病、长期慢性病确诊、大手术 |
| **财务** | 大笔进出、置业、破产、中奖 |
| **居所** | 搬家、移民、换城市、购房 |

### 交脱运应事的"叠加效应"
- 若**流年又与交运干支相冲相合** → 应事**更明显、更突发**。
- 若**流月再加一层触发** → 可精确到**哪个月发生什么事**。
- 这就是"**大运定十年、流年定当年、流月定当月**"的应期推演逻辑。

## 如何观察交运应期

### 第一步：定位交运的精确时点
- 根据起运公式算出**每一步交运的日期**。
- 注意**农历 / 公历**的对照——起运是节气制，严格用节气划分。

### 第二步：分析新旧两步大运的组合关系
- 旧运是什么气？新运是什么气？
- 两者是**相生、相克、相合、相冲**？
- 举例：
  - 旧运**甲寅**、新运**乙卯** → 同属木、相生顺接 → **过渡平顺**，气机延续。
  - 旧运**甲寅**、新运**庚申** → 寅申相冲 → **过渡剧烈**，一段时期极乱。

### 第三步：叠加流年、流月、流日
- 用交运时点对照当年**流年干支**——看是助运还是破运。
- 再对照当月**流月干支**——精确到"哪几个月变动最大"。
- 对重大决策者，甚至可查**流日**——择日避凶。

### 第四步：核对原局的应期字
- 命盘里**天干透出、地支藏干**的那些字，是"信号灯"。
- 交运的干支**冲、合、刑、害**了哪个原局字，就看那个字所主的**宫位 / 十神 / 事象**。
- 如冲日支 → **婚姻、身体**；冲月柱 → **事业、父母**。

## 交脱运期的现实操作建议

### 脱运期（旧运末 1—2 年）
- **不要对旧模式再抱幻想**——旧运的红利已经过去。
- **把旧阶段的事情收口**：结项目、结清账务、处理旧关系。
- **不要在此时启动新的大赌局**——旧气衰、新气未到，力不从心。

### 真空过渡带
- **静**字当头：少做重大决策、少签长期合同。
- **多观察、少行动**：感受新气机的方向。
- **把身体养好**：交运期最大的资产是**健康与清醒的头脑**。

### 交运期（新运前 1—2 年）
- **顺着新气机的方向做小尝试**——先用小成本试水。
- **调整圈子**：主动接近与新运同频的人。
- **重大项目等气机稳定后再启动**（通常是新运第 3 年起）。

## 一句话总结
交运与脱运是大运的**出入口**。
- 脱运是"**旧戏谢幕**"，交运是"**新戏开锣**"，中间是**换场**的几分钟。
- 应事最密集、决定最危险、机会与风险并存。
- **看懂交脱运的节奏，就能在人生最关键的拐点上不慌不乱。**
