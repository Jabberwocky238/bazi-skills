# 自坐死

## 定义
- **死**：十二长生的第八阶段，象征**能量到最低点、"死亡"归零**。
- **自坐死**：**日主天干坐在死位地支上** → 日主能量处于**十二长生中最弱**的阶段。
- 各天干死位：
  - **甲木** 死在 **午**。
  - **乙木** 死在 **亥**。
  - **丙戊** 死在 **酉**。
  - **丁己** 死在 **寅**。
  - **庚金** 死在 **子**。
  - **辛金** 死在 **巳**。
  - **壬水** 死在 **卯**。
  - **癸水** 死在 **申**。

## 典型日柱
- **甲午**：甲木死在午（木被火烧）。
- **乙亥**：乙木死在亥（花草怕大水）。
- **丙酉 / 戊酉**：丙丁死在酉（火被金泄 / 克）。
- **丁寅 / 己寅**：丁己死在寅（火 / 土被木克）。
- **庚子**：庚金死在子（金沉水底）。
- **辛巳**：辛金死在巳（精金怕烈火）。
- **壬卯**：壬水死在卯（水被木泄）。
- **癸申**：癸水死在申（并非所有派系都这么算）。

## 核心：能量归零、最弱之位、但未必短命
- 死 = 能量到**最低点**。
- 自坐死 = 日主坐在自己**最虚的地方**：
  - **底气不足 / 根基薄**。
  - **需要外援才能立**。
  - **内心常有"空"的感觉**。
- **但"死"≠ 字面意义的死亡** —— 很多成就非凡的人都自坐死位。

## 与相关结构对比

| 结构 | 能量状态 |
|---|---|
| **自坐死**（本条） | 最低点、归零 |
| **自坐绝** | 断绝、无根 → 与死类似但更彻底 |
| **自坐墓** | 归藏 → 内收 |
| **自坐胎** | 潜伏 → 等待复苏 |
| **自坐衰** | 开始下滑但仍有气 |

**死绝墓胎**都属能量弱的阶段，但意味不同：
- 死：**归零**。
- 绝：**断灭**。
- 墓：**收藏**。
- 胎：**孕育新生**。

## 判断维度

### 1. 死位的五行性质
- 死位地支对日主是"**十神的哪一种**"：
  - 甲午：午为甲之伤官 → 才华外泄、心思多。
  - 乙亥：亥为乙之印（藏壬）→ 有印暗生 → "死中有生"。
  - 丙酉：酉为丙之财 → 财在死位 → 求财辛苦但**财临死位易守**。
  - 庚子：子为庚之伤官 → 聪明外显但伤身。
  - 辛巳：巳为辛之官（藏丙）→ 官在死位 → 事业艰辛但成就高可贵。
  - 壬卯：卯为壬之伤官 → 表达锋利但内耗。
- 死位的性质深刻影响命造质感。

### 2. 有无外援
- **有印生身**：死中得救 → 身不至于崩。
- **有比劫帮身**：死中有靠 → 同伴与自助。
- **无印无比**：纯粹身弱于死地 → 依赖环境 / 大运。

### 3. 大运流年
- **印运 / 比劫运**：救死 → 发力期。
- **财官运**：死加耗 → 若无救则凶。
- **冲死位的大运 / 流年**：**关键节点** → 转折点。

## 常见误区

### 1. "自坐死 = 早亡 / 短命"
- **错**。
- 十二长生是能量周期，不是生死判断。
- 很多自坐死的人健康长寿、成就深远。
- 比如典型命造：
  - 壬卯日 → 水在木中死 → 但木是食伤 → 可能是才华型成就。
  - 辛巳日 → 金在火中死 → 但官杀成格则贵。

### 2. "死 = 命一定差"
- 自坐死 = 先天底牌偏弱，但：
  - **有印 / 比劫救**时：反而走"**逆袭式**"人生。
  - **大运补扶**：完全可能大成。
- 也有很多"**死中得贵**"的贵格命造。

### 3. 把"死"当性格
- 自坐死不等于"**死气沉沉**"。
- 只是**底色能量偏低** → 表现为内敛 / 敏感 / 多思。

### 4. 忽视"死"带来的觉察力
- 能量低的人往往**对生命 / 关系 / 存在有更深的思考**。
- 自坐死的人常常是**哲学家型、思想者型**。

## 现实表现

### 性格
- **敏感、多思、内心丰富**。
- **对世界有深刻感知**：看问题比常人更深。
- **不张扬**：外表低调、不爱出风头。
- **容易自我怀疑**：底色的"虚"直接反映为自信不足。
- **对"生命 / 无常"有天然的敏感**。
- **精神层面需求高**：不易满足于物质。

### 事业
- **适合"**沉淀 + 深度**"的行业**：
  - 哲学、心理、宗教、文化。
  - 医疗、咨询、精神辅导。
  - 艺术、文学、深度研究。
- **不适合"快节奏 + 高对抗"**的岗位：因为底气不够扛。
- **靠专业深度而非爆发力**立身。
- 很多**作家、哲学家、心理学家、艺术家**都是自坐死。

### 感情
- **对情感的感知极敏锐**。
- **容易受伤但也容易动情**。
- **需要精神层面匹配的伴侣**。
- **关系中常常"付出比要求多"**。
- **一旦认定就深情至极**。

### 身体
- **先天底子偏弱**：易疲、易生小病。
- **恢复期长**：一场病可能影响较久。
- **长寿可能性仍在**：但需要**终身养生**。
- **情绪与身体联动紧密**：情绪问题容易投射为身体问题。

### 心理
- **存在感较弱**：常感觉"别人看不到我"。
- **内心对"存在意义"有深度追问**。
- **可能有轻度抑郁倾向**。
- 需要**建立内在的意义感 + 外在的支持系统**。

| 日柱 | 十神组合 | 含义 |
|---|---|---|
| **甲午** | 甲坐伤官（午中丁）+ 死 | 才华 + 自耗 → 创作型，但易透支 |
| **乙亥** | 乙坐印（亥中壬）+ 死 | 有救 + 有源 → 学者 / 思想者 |
| **丙酉** | 丙坐财（酉中辛）+ 死 | 为财奔波、财临死位易守 |
| **戊酉** | 戊坐食（酉中辛）+ 死 | 才华内敛、精细型 |
| **庚子** | 庚坐伤官（子中癸）+ 死 | 聪明 + 表达欲 → 易口舌 |
| **辛巳** | 辛坐官（巳中丙）+ 死 | 官在死位 + 磨砺 → **金火铸印**类成才 |
| **壬卯** | 壬坐伤官（卯中乙）+ 死 | 才华锋利 → 表达型成就但要小心伤官见官 |

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 印运 | **大喜** → 救死 → 转折期 |
| 比劫运 | **喜** → 帮身 → 得贵人 / 同伴 |
| 官杀运（有印化） | **喜** → 杀印相生 |
| 财运 | **看结构** → 身极弱则大忌；有印则可任财 |
| 食伤运 | **看结构** → 身弱则耗更虚；身略强则成才 |
| **冲死位的年份** | **大忌**（原局无救）/ **大转折**（原局有救） |

## 自坐死命造的**成就路径**

### 1. 借力而行
- 天然需要**导师、伴侣、机构、平台**的支持。
- 不要硬撑 → 学会借力是第一课。

### 2. 深度立身
- 放弃"**与身强者拼速度**"的想法。
- 用"**深度 + 沉淀**"做出自己的差异化。

### 3. 精神层面深耕
- 很多自坐死的人**天然适合修行、学术、艺术**。
- 在精神领域**最能兑现价值**。

### 4. 身体是底座
- 一辈子养生 → 不是"可有可无"，是"**本钱**"。
- 作息、饮食、心情管理**比身强者重要十倍**。

## 核心判断
- 自坐死不是"命苦"，是"**底气低的感知型人格**"。
- 识别要点：
  - 原局有无印 / 比劫救应。
  - 死位是哪个十神。
  - 大运是否补充身力。
- 成就路径：**借力 + 深度 + 长寿养生**。
- 误判代价：当成身强去拼 → 耗身早衰。

## 一句话总结
- 自坐死 = **底气天然偏低**。
- 但"低"不等于"差" —— 许多能感知最深、思考最远、创造最美的人都在这条路上。
- **别和强者比速度，比你自己一生能看多深、能做多久** —— 这是自坐死的真正贵。
