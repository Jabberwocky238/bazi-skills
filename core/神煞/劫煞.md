# 劫煞

## 定义
- **劫**：抢夺、劫掠、被夺。
- **劫煞**：三合局中"**长生前一位**"——代表**外部侵害、被抢、被夺**的凶煞。
- 与"亡神"配对：亡神主内失，劫煞主外夺。

## 起法（按年支或日支查）
《三命通会·论劫煞》原文："**寅午戌劫在亥，巳酉丑劫在寅，申子辰劫在巳，亥卯未劫在申**。"

| 三合局（年支或日支）| 劫煞位置 |
|---|---|
| 寅午戌 | **亥** |
| 巳酉丑 | **寅** |
| 申子辰 | **巳** |
| 亥卯未 | **申** |

**背后原理**：劫煞 = 三合局**长生前一位（绝位）**——气机未生而先死、未成而先散。
- 寅午戌火局，长生在寅，寅前一位为亥（绝位）→ 亥为劫煞；
- 气绝之地 = "未开始就被劫走"的状态。

## 劫煞 vs 亡神 vs 灾煞

| 三煞 | 位置 | 动作 | 主方向 |
|---|---|---|---|
| **亡神** | 长生本位 | 内失 | 自己瓦解 |
| **劫煞**（本条）| 长生前一位（绝位）| **外夺** | **被外人劫** |
| **灾煞** | 中气对冲 | 明灾 | 突发意外 |

三煞**同出一局**，往往在命中同时出现，合称"**三煞并见**"——此时凶性倍增。

## 核心作用：被外夺的预警
- 劫煞不是"**你去抢别人**"，而是"**别人来抢你**"：
  - **金钱被骗、被盗、被耗**。
  - **生意合作被对方占便宜**。
  - **感情被第三者介入**。
  - **外部突发的损失**：意外、官司、强制执行。
- 古籍论："**劫煞入命，主外部侵害，财物流失**。"

## 判断维度

### 1. 位置
- **年柱劫煞**：祖业、家产易被外夺。
- **月柱劫煞**：事业、合作、生意上易遇小人。
- **日柱劫煞**：本人易遇意外、被亲近者占便宜。
- **时柱劫煞**：子女或晚年财产受损。

### 2. 劫煞配十神
- **劫煞 + 比劫**：同行、合伙人抢你饭碗、夺你资源。
- **劫煞 + 财星**：钱财被外人夺——借钱不还、投资被骗、被盗。
- **劫煞 + 官杀**：被官方处罚、诉讼、强制执行。
- **劫煞 + 食伤**：作品 / 创意被抄袭 / 剽窃。

### 3. 大运流年引动
- 走到劫煞地（如火局人走亥运）→ **外部侵害事件集中爆发**。
- 原局已带劫煞 + 大运再引动 → **财产 / 感情 / 事业上的重大损失年**。

## 常见误区

### 1. "带劫煞 = 一定被骗"
- 劫煞只是一个"**外部侵害的风险点**"。
- **身强 + 警觉性高** → 能识破并防住。
- **身弱 + 天真易信人** → 真容易吃亏。

### 2. 把劫煞等同于"破财煞"
- 劫煞的"劫"不只限于钱——感情、时间、机会、声誉都可能被"劫"。
- 现代语境：**投资诈骗、情感诈骗、信息诈骗、知识产权被盗**都算劫煞。

### 3. 忽视"自己防"的重要性
- 劫煞来自外部，但**能不能被劫走，取决于自己**。
- 命中有劫煞而做事谨慎的人，未必真遭受大损失。
- 对劫煞的正确姿态：**提高警觉，不贪不急**。

## 现实表现

### 性格
- **容易遇到贪便宜的人、虚伪的朋友**。
- 带劫煞的人**自己未必贪**，但**招来贪心者**。
- **对"捡便宜"的机会要特别警惕**——那往往是劫煞在引诱。

### 典型情境
- **合伙被坑**：出钱出力，最后被踢出局。
- **借钱不还**：把钱借给看似可靠的朋友，音讯全无。
- **投资被骗**：高回报陷阱。
- **感情第三者**：婚姻里被外人介入。
- **突发意外**：车祸、失窃、抢劫。

## 化解与应对

### 结构层
- 命局有印星（化煞） → 吸纳凶性。
- 比劫强 → 帮身抗劫。

### 人事层
- **不贪"天上掉馅饼"的机会**。
- **借钱 / 合伙 / 投资必须有合同、有见证**。
- **对新认识但"感情太快升温"的关系保持警觉**——尤其劫煞流年。

### 时间层
- **流年走劫煞**：那一年**少碰合伙、少做大额出借、少做高风险投资**。

## 与劫财的区别
两个"劫"字，很多人混淆：
- **劫财（十神）**：与日主同五行、异阴阳 → 命局内部的气性。
- **劫煞（神煞）**：三合局绝位 → 外部的风险标记。
- 比喻：**劫财是自己性格里爱争的一面；劫煞是别人朝你伸出的手**。

## 核心判断
- 劫煞是"**提示你留神**"的信号，不是"**注定倒霉**"的判决。
- 应对核心：**提防贪念（自己的）+ 提防贪心者（别人的）**。
- 很多人一生带劫煞却安然无恙——靠的就是**不贪、不急、不信偏门**。

## 一句话总结
- 劫煞 = **外人朝你伸手的倾向**。
- 带劫煞的人一生要学的是：**分得清谁是真的关心你，谁只是想捞你一把**。
- 警觉就是最好的护身符。
