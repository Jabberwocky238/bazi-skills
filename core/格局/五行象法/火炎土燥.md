# 火炎土燥

## 定义
- **火炎**：火五行过旺、炎上不已。
- **土燥**：土被火烤，失去湿润与承载之性。
- **火炎土燥**：命局**火气极盛 + 土受火烤 + 无水润局** → 土成**焦干板结**、万物不生。
- 属**五行失衡病象**之一。
- 是"**火土夹带**"的**失败版** —— 同是火土相连，有水润则稳厚，无水润则燥烈。

## 核心：火烤土焦，寸草不生
- 适度火 + 土 = **温暖肥田** → 孕育生机。
- 过度火 + 无水 = **烤干焦裂** → 毁灭生机。
- 关键不是"火生土"，是**火生土过度 + 无水调候**。
- 所谓"**炎**"：火已不是温度，而是烈焰。
- 所谓"**燥**"：土已不是沃壤，而是焦砂。

## 与相关结构对比

| 结构 | 火与土比例 | 水的角色 | 结果 |
|---|---|---|---|
| **火土夹带** | 火适中生土 | 有水润 | 稳厚吉象 |
| **火炎土燥**（本条） | 火过旺烤土 | 无水 | 燥烈病象 |
| **土重火熄** | 土过厚压火 | 无所谓 | 闷堵病象 |
| **火多土焦** | 火极旺 + 土弱 | 无水 | 土被毁的极端 |

## 成立条件
1. **火气极旺**：
   - 月令为火（巳午未）。
   - 地支寅午戌或巳午未三合火局。
   - 天干丙丁重透。
2. **土被火烤**：
   - 戊己土透干，但处火乡。
   - 燥土（戌未）成势 → 火土同党、燥上加燥。
3. **无水救应**：
   - 无壬癸透干。
   - 亥子水被冲被合被克。
4. **无湿土调和**：
   - 辰丑（湿土）无 / 被破。

## 判断维度

### 1. 日主类型

| 日主 | 火炎土燥的表现 |
|---|---|
| **戊土** | 阳土遇火上加火 → 干裂如龟甲 → 固执、躁动、口干舌燥 |
| **己土** | 田园土被烈日烤 → 裂纹细密 → 敏感、焦虑、皮肤问题 |
| **丙火日主** | 火本旺 + 土再泄反而不成 → 燥上加燥 → 暴躁外显 |
| **丁火日主** | 烛火 + 燥土 → 火被土泄 + 土反助燥 → 内耗型焦躁 |
| **其他日主** | 对应十神之燥：食伤燥、印燥、财燥 |

### 2. 燥湿土的比例
- **纯燥土**（戌未）成群：火炎土燥的典型底色。
- **湿土**（辰丑）出现：一丝救应，症状减轻。
- **燥湿混杂**：看哪方势大，燥多则病，湿多则缓。

### 3. 水的残存
- **水全无**：燥极 → 身心全面干裂。
- **水虚透**：象征性存在 → 症状时轻时重。
- **水藏而未出**：内里尚有一点润 → 不算绝燥。

### 4. 金的通关
- **金透**：土生金 + 金生水（若有水） → 间接救应。
- **金被火克**：金尚未发力已被熔 → 救应失败。

## 常见误区

### 1. "火土旺 = 火土夹带"
- **差一滴水**：
  - 有水 → 火土夹带（吉）。
  - 无水 → 火炎土燥（病）。
- 水是分水岭。

### 2. 把"燥"等同"热情"
- 火炎土燥不是**有热情**，是**热到失控 + 干到无味**。
- 看似积极，实则是焦躁的外露。

### 3. 单纯补水
- 少量补水 = 杯水车薪 → 瞬间蒸发。
- 必须**先泄火 + 再补水**。
- 顺序：**金泄土、水灭火、湿润结构**。

### 4. 忽视对生殖 / 子嗣的暗示
- 火炎土燥的传统意象："**田焦难种、物不能生**"。
- 现实中常对应：**生育困难、子嗣不顺、孕产障碍**（尤其女命土日主）。
- 非绝对，但需警觉并配合调候。

## 现实表现

### 性格
- **急躁、易怒**：火气无出口。
- **固执、难劝**：土燥板结，思维僵化。
- **缺乏耐心**：凡事想快 → 欲速不达。
- **情绪化表达**：易说气话、冲动行事。
- **睡眠差**：火盛神不宁。

### 事业路径
- **冲劲足但持续性差**：上半场冲、下半场熄。
- **人际冲突多**：火气易外泄 → 得罪人、失团队。
- **决策偏激**：燥烈状态下做的判断易极端。
- **身体拖累事业**：健康问题影响工作稳定度。
- 若走水运 / 金运 → 状态全面回稳。

### 身体
- 火主心、小肠、血脉、眼。
- 土主脾、胃、肌肉、口。
- 火炎土燥的典型症状：
  - **口干舌燥**：常年觉得嗓子干、水不解渴。
  - **便秘 / 痔疮**：肠道燥热。
  - **皮肤干裂 / 湿疹 / 痤疮**：燥热外发。
  - **目赤 / 失眠 / 心悸**：火扰心神。
  - **高血压 / 头痛**：火炎上冲。
  - **女命：月经失调、生育障碍**：胞宫燥热，难以孕育。

### 感情
- **感情炽烈但不长久**：燃得快、灭得快。
- **易冲突**：两人一点就炸。
- **难生育或育儿紧张**：同结构性暗示。
- 若见水运调候 → 感情状态可显著缓和。

## 化解思路

### 结构层
- **补水为最优**：
  - 壬水 → 直接压火 → 见效快。
  - 癸水 → 细雨润物 → 持久调候。
  - 大运走水地 → 转折期。
- **补金为辅**：
  - 金泄土 → 减燥。
  - 金生水（若水有根）→ 间接润。
- **湿土代燥土**：
  - 走辰丑运 → 湿土掺和 → 燥度降低。
- **避免补火 / 燥土**：
  - 火加剧。
  - 戌未运 → 燥上加燥，大忌。

### 生活层
- **环境润局**：
  - 多接触水：海边、湖边、游泳池。
  - 室内放加湿器、水景。
  - 避免干旱、高温、暴晒环境。
- **饮食润局**：
  - 多喝水、多吃润燥食物（雪梨、银耳、百合、蜂蜜）。
  - 避免辛辣、煎炸、烧烤、酒类。
- **作息润局**：
  - 早睡 → 养阴。
  - 避免熬夜 → 熬夜加重燥火。
- **情绪润局**：
  - 慢节奏爱好：泡茶、插花、听水声、写毛笔字。
  - 远离争吵型人际与高压环境。
- **生育调候**：
  - 女命火炎土燥者备孕前建议**长期润燥调理**（中医 / 生活方式双管齐下）。
  - 选择水运年份备孕。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 水运（尤壬） | **大喜** → 压火润土 → 全面转折 |
| 金运 | **喜** → 泄土助水源 |
| 湿土运（辰丑） | **喜** → 掺湿减燥 |
| 燥土运（戌未） | **大忌** → 燥上加燥 |
| 火运 | **大忌** → 炎上不已 → 健康 / 情绪低谷 |
| 木运 | **忌** → 生火加燥 |

## 与相关格局的对照

| 格局 | 关键 | 结果 |
|---|---|---|
| **火土夹带** | 火生土 + 有水润 | 稳厚吉象 |
| **火炎土燥**（本条） | 火烤土 + 无水 | 燥烈病象 |
| **火炎木焚 / 木火相煎** | 火烤木 | 焚毁（木版本） |
| **火多金熔** | 火烤金 | 熔毁（金版本） |
| **土重火熄** | 土太厚压火 | 闷（相反病象） |

**火在不同五行结构中作恶的系列**：
- 遇木：**木火相煎**（焚）。
- 遇金：**火多金熔**（熔）。
- 遇土：**火炎土燥**（燥）。
- 遇水：**火多蒸水**（涸）。

## 核心判断
- 火炎土燥不是"火土多"，是"**火土多 + 无水**"。
- 判断重点：
  - 看**是否有水（壬癸 + 亥子）**。
  - 看**土是燥（戌未）还是湿（辰丑）**。
- 若是**短期火炎土燥**（大运流年引起）→ 熬过去即可。
- 若是**原局结构性火炎土燥** → 需**长期润燥生活方式**配合，大运见水为真正转折。

## 一句话总结
- 热情过了头，把田烤成了焦。
- 火炎土燥 = **有能量、有冲劲，但干到什么都种不出来**。
- 救法不是再加一把火，是**加一滴水、慢一点、润一点**。
