# 天干相冲

## 定义
- **天干相冲** = 十天干中处于**方位对立**关系的两干之间的对冲。
- 相冲 = 阳干冲阳干、阴干冲阴干；**戊己土居中央，不参与相冲**。
- 相冲**不仅含相克之意，且主动方向相对**——力量比纯粹的相克更大。

## 一、天干四冲

天干相冲共四组，又称**天干四冲**：

| 方位对立 | 相冲组合 | 阴阳 |
| --- | --- | --- |
| 东西方 | **甲 ↔ 庚** | 阳冲阳 |
| 东西方 | **乙 ↔ 辛** | 阴冲阴 |
| 南北方 | **丙 ↔ 壬** | 阳冲阳 |
| 南北方 | **丁 ↔ 癸** | 阴冲阴 |

> **戊己居中央，不参与四冲。**

## 二、相冲的方位由来

以河图定位五行方位，再配入十天干：

| 方位 | 天干 |
| --- | --- |
| 东 | 甲乙木 |
| 南 | 丙丁火 |
| 中央 | 戊己土 |
| 西 | 庚辛金 |
| 北 | 壬癸水 |

- **东西相冲**：甲庚冲（阳木 vs 阳金）、乙辛冲（阴木 vs 阴金）——金木对立。
- **南北相冲**：丙壬冲（阳火 vs 阳水）、丁癸冲（阴火 vs 阴水）——水火对立。
- **中央不冲**：戊己土各居其位，不与其他方位对冲。

## 三、相冲 vs 相克

### 1. 性质不同
- **冲**：主要表现为**方位上的对抗**——空间上的相互作用。
- **克**：主要表现为**五行属性上的制约**——属性上的相互作用。
- 例：甲木克戊土是五行之间的制约；戊土属中央，不参与相冲。

### 2. 力量不同
- 一般而言，**冲的力量 > 克的力量**。
- 例：丙壬之间的冲克 > 丙火对庚金的克制（前者是冲，后者是克）。

### 3. 表现方式不同
- **冲**：更多表现为**突发性的、剧烈的变化**。
- **克**：更多表现为**渐进的、缓慢的变化**。

### 4. 化解方式不同
- **冲**：可通过**和化**（五合等关系）等方式化解。
- **克**：主要通过**五行生克的平衡调节**实现。

## 四、天干相冲的典型应用

### 1. 主六亲关系
- 天干**主外**，是公开、直接表现出来的——代表人世的表象与社会关系。
- **天干四冲多主六亲的关系**（父母、兄弟、配偶、子女之间的关系）。
- 具体落点要结合四柱宫位判断：
  - 年干与月干相冲 → 祖辈与父辈关系冲突。
  - 年干与日干相冲 → 命主与祖父辈关系紧张。
  - 月干与日干相冲 → 命主与父母辈关系不睦。
  - 日干与时干相冲 → 命主与子女关系不睦。

### 2. 主身体方面的伤害
- 天干相冲多主**对应部位的伤病**：
  - **甲庚冲** → 肝胆与肺部、骨骼、手部。
  - **乙辛冲** → 肝胆与呼吸道、咽喉。
  - **丙壬冲** → 心脏、眼目与肾脏、泌尿。
  - **丁癸冲** → 心血管与肾、泌尿。
- 具体应验需结合大运流年的引动。

### 3. 主事业与社会关系的冲突
- 天干代表显露在外的人际与事业关系。
- 四冲易引发：
  - 工作中的矛盾、竞争、对抗。
  - 合作破裂、合伙散伙。
  - 公开场合的摩擦。

## 五、冲的吉凶辩证

- **冲**不必然为凶，同样要看**喜忌**：
  - **忌神被冲** → 吉。
  - **喜神被冲** → 凶。
- 例：
  - 原局火为忌、大运流年壬水冲丙 → 忌神被冲 → 反为顺遂之机。
  - 原局水为用、大运流年戊土冲壬（此为克不是冲，仅为对照）→ 用神受伤 → 为凶。
- **冲未必散，合未必合** —— 需结合整体命局判断。

## 六、天干相冲的两组形象对比

### 东西冲：甲庚冲、乙辛冲
- **象意**：斧斤砍伐森林、刀剑斩藤蔓。
- **易发事端**：突发性伤害、意外、手术、肝胆问题。
- **庚金日主遇流年甲木**：财来冲身 → 常伴钱财纠纷或家中长辈健康波动。

### 南北冲：丙壬冲、丁癸冲
- **象意**：江河映日、阴水侵烛。
- **易发事端**：情绪起伏剧烈、婚姻波折、水火相关事故。
- **丙火日主遇流年壬水**：官杀来冲身 → 常伴工作压力、与权威者冲突。

## 七、天干相冲在大运流年中的应用

### 1. 原局无冲、大运来冲
- 原局平稳，大运引入相冲 → 整个大运期间对应柱位**动荡剧烈**。
- 例：月柱甲木、大运庚金 → 父母宫 / 早年事业遭遇重大变动。

### 2. 原局已冲、大运再冲（冲起冲）
- 原局矛盾已存在，大运再冲 → **矛盾被放大、集中爆发**。

### 3. 原局有合、大运来冲（冲开）
- 原局中合住的天干，被大运冲散 → **合的功能失效**，原本被锁住的力量释放出来。

### 4. 原局有冲、大运来合（合解）
- 大运或流年的天干合住冲中的某一端 → **冲势减弱或暂时解除**。

## 八、一句话总结

- 天干相冲 = **方位对立的阳冲阳、阴冲阴**——四组：甲庚、乙辛、丙壬、丁癸。
- **戊己不冲**，因居中央。
- 冲比克更**剧烈、更突发、破坏力更大**；冲多主六亲关系与身体伤害。
- **冲不必凶、合不必吉** —— 喜忌决定吉凶。
- 解读冲，必须结合**喜忌、宫位、大运流年引动**综合判断。
