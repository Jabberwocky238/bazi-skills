# 身杀两停格

## 成立条件（古籍 + 客观判据）

### 条件 1：日主有根有扶，自成一方之势

- **古籍出处**：
  - 《子平真诠·论七杀》："身杀两停，贵格也。"沈孝瞻明示此格以"身旺足以敌杀"为第一前提。
  - 《滴天髓·官杀》："**杀旺身强，两停之贵**。"
- **客观判据**：
  - 日主于月令得比劫 / 印星本气或中气之根，或
  - 他柱至少有两位比劫 / 印星通根相扶（含禄、刃、长生、余气库）。
- **不满足则破**：日主虚浮无根、比印亦弱 → 身不能敌杀 → 转入"身弱杀重"讨论，须以印化或从杀论。

### 条件 2：七杀透干且通根，成势显威

- **古籍出处**：
  - 《子平真诠·论七杀》："七杀宜透不宜藏，透则其性显。"
  - 《滴天髓·官杀》："七杀有制化方为权，无制化则为鬼。"
- **客观判据**：
  - ① 七杀透于年 / 月 / 时干，地支本气或中气有根至少一位；或
  - ② 月令地支本气为七杀，且天干有呼应（透出或余气相助）。
- **不满足则破**：七杀虚透无根 / 仅藏支一位余气 → 杀势不成，无"停"之对象，格不成立。

### 条件 3：身与杀的旺度相当（两停之要）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论七杀》："**身杀两停，方为上贵**；一方独旺，反为破格。"
  - 《滴天髓·体用》："强弱相当，势均力敌，方有制衡之妙。"
- **客观判据**：
  - 可容忍一方**稍旺**（如日主比杀稍旺一线，或杀比日主稍旺一线），但**差距不可悬殊**。
  - 评估方式：综合月令气势、通根位数、透干数、生扶关系打分，两者分值接近即为"停"。
- **不满足则破**：
  - 身过强压过杀 → 杀被压制、威势消解 → 转入"身旺无依"或"羊刃独旺"讨论；
  - 杀过强压过身 → 日主无力承受 → 转入"身弱杀重"或"从杀格"讨论。

### 条件 4：命中有食神 / 印星 / 羊刃其中之一作为辅助

- **古籍出处**：
  - 《子平真诠·论七杀》："**杀旺之局，喜食神以制之；身旺之局，喜印星以化之**。"
  - 《子平真诠·论阳刃》："刃杀相济，声威赫奕。"
- **客观判据**：
  - 杀稍旺于身者 → 命中见食神（可制杀）或印星（可化杀）为辅助，格局更上一层。
  - 身稍旺于杀者 → 命中见印运相助或财星轻透生杀，维持两停。
  - 身刃成势而杀亦强者 → 即为羊刃驾杀（本格之特例）。
- **不满足则破**：原局完全无任何通关 / 调节之物，两停硬碰 → 虽成格但格局层次偏低，遇破格之运即崩。

### 条件 5：原局无严重冲克、混杂动摇根基

- **古籍出处**：
  - 《滴天髓·官杀》："官杀混杂则气浊，贵而不纯。"
  - 《子平真诠·论七杀》："杀局忌见官星，混则格乱。"
- **客观判据**：
  - 天干无正官与七杀并透成混；地支七杀之根无被冲、被合化之情。
  - 日主之根亦无被严重冲刑（如月支刃位被冲破）。
- **不满足则破**：杀根动摇 → 杀气不稳；身根动摇 → 身势不立；皆使"两停"无法维持。

---

**五条同时满足**方为真身杀两停格。本格的核心在第 3 条"两停"——沈孝瞻"身杀两停，方为上贵"八字明示：力量平衡是本格立命之本。
