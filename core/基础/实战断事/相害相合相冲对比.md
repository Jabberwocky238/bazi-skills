# 相害、相合、相冲对比

地支的合、冲、害，是批八字的三大核心作用关系。三者并非孤立存在，而是一环扣一环——**合被冲破则生害**。弄清三者的定位、相互转化与判断优先级，是看命从"看字面"走向"看动态"的分水岭。

---

## 一、三者的基本定义

- **相合**：地支之间五行相互依附、结合或合化，包括子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合。本质是**结合、依附、合化**，作用偏向正面，但合住的是喜用则吉、合住的是忌神则凶。
- **相冲**：地支间隔六位、五行相克、同性阴阳相斥所形成的正面冲撞，共子午、丑未、寅申、卯酉、辰戌、巳亥六组。本质是**动、破、散、变**，属中性力量，主离别、搬迁、变动、破败、疾病。
- **相害**（又名**相穿**）：地支之间因合被冲破而产生的**隐性损害**。共子未、丑午、寅巳、卯辰、申亥、酉戌六组。表面平和，内里暗损，属暗箭伤人。

一句对比：**合是结合，冲是明撞，害是暗伤**。

---

## 二、本质关系（核心章节）

### 2.1 相害的本质——"相合被破坏"的产物

六害不是凭空产生的，它是由**六合被六冲打断**之后，在两支之间留下的"暗账"。每一组害，背后都有**两层合被冲破**的逻辑。以子未害为例：

- 午未本可相合，却被**子午冲**撞散，子水去破了午未合。
- 子丑本可相合，却被**未丑冲**撞散，未土去破了子丑合。
- 子与未因此结下仇怨，形成"子未害"。

这就是为什么相害又叫**穿**——它不是明面上的冲，而是被合、被冲交错作用之后残留下来的一道暗伤口。

### 2.2 六合被冲破 → 形成六害的对照逻辑

| 被破坏的六合 | 破合之冲（A） | 破合之冲（B） | 最终形成的相害 |
|---|---|---|---|
| 子丑合 / 午未合 | 子午冲破午未合 | 丑未冲破子丑合 | **子未害** |
| 子丑合 / 午未合 | 丑未冲破午未合 | 子午冲破子丑合 | **丑午害** |
| 寅亥合 / 巳申合 | 巳亥冲破寅亥合 | 寅申冲破巳申合 | **寅巳害** |
| 寅亥合 / 巳申合 | 寅申冲破寅亥合 | 巳亥冲破巳申合 | **申亥害** |
| 卯戌合 / 辰酉合 | 辰戌冲破卯戌合 | 卯酉冲破辰酉合 | **卯辰害** |
| 卯戌合 / 辰酉合 | 卯酉冲破卯戌合 | 辰戌冲破辰酉合 | **酉戌害** |

可以看出：**每一组害，都是"合 + 冲"共同作用的残局**。没有合意，就没有害；没有冲破，害也无从成立。

### 2.3 相害 vs 相冲

| 维度 | 相冲 | 相害 |
|---|---|---|
| 动静 | 明争、动得大 | 暗耗、动得小 |
| 显隐 | 显于外，动静响 | 藏于内，表面平和 |
| 伤害深度 | 破而不必深 | 深而不易察 |
| 应事典型 | 搬家、调动、车祸、争吵、疾病发作 | 暗流、心结、慢性病、六亲隔阂 |
| 作用方式 | 正面对撞 | 暗箭伤人、尔虞我诈 |

俗语可参：**冲是明枪，害是暗箭**。冲能被看见，害常常只在结果上才浮现。

### 2.4 相害 vs 相合

| 维度 | 相合 | 相害 |
|---|---|---|
| 本质 | 结合、依附、加强 | 破坏、暗损、消耗 |
| 气机 | 聚 | 散中带损 |
| 可见度 | 明 | 隐 |
| 与吉凶关系 | 合喜则吉，合忌则凶 | 常为凶，但可被三合、六合化解 |
| 产生逻辑 | 阴阳相吸、方位相配 | 合被冲破之后的残余作用 |

合是"**加**"，害是"**减得不动声色**"——常常是合被冲破后的遗留。

## 三、六合、六冲、六害的对应表

| 六合（结合之气） | 被何冲破（拆散之力） | 形成的六害（残余之伤） |
|---|---|---|
| 子丑合 | 午冲子 / 未冲丑 | **丑午害 / 子未害** |
| 寅亥合 | 申冲寅 / 巳冲亥 | **申亥害 / 寅巳害** |
| 卯戌合 | 酉冲卯 / 辰冲戌 | **酉戌害 / 卯辰害** |
| 辰酉合 | 戌冲辰 / 卯冲酉 | **卯辰害 / 酉戌害** |
| 巳申合 | 亥冲巳 / 寅冲申 | **寅巳害 / 申亥害** |
| 午未合 | 子冲午 / 丑冲未 | **子未害 / 丑午害** |

横读：每一组合都能被两路冲力打破；竖读：每一组害，都背后对应两个合被破。

**六害的命名象意（可作记忆补充）**：
- **子未害**——世家之害，骨肉不睦
- **丑午害**——官鬼之害，性暴无忍
- **寅巳害**——恃势之害，两强相争
- **申亥害**——争进之害，婚姻最忌
- **卯辰害**——欺凌之害，少欺长
- **酉戌害**——嫉妒之害，面伤头疮

## 四、力量等级与优先级

按地支作用力量强弱排序如下（由强到弱）：

**三会 > 三合 > 生地半三合 > 六合 > 墓地半三合 > 拱合 > 六冲 > 相刑 > 暗合 ≥ 六害 > 六破 > 相绝**

由此可得出几条极重要的判定法则：

1. **三合、三会一旦成立，最稳固**。其他地支再来冲、刑、害、破，基本无效——**三合、三会可化解冲害刑破**。
2. **六合力量稍小**，一旦遇到三合、三会，要**让位**；六合合住之后，往往出现**贪合忘冲、贪合忘克、贪合忘刑、贪合忘害、贪合忘破**。
3. **六冲大于相刑**，相刑再大于六害、六破、相绝。
4. **六害力量虽小于冲，但应事的深度有时反而超过冲**——尤其是相克又带相害同时存在时，伤害最深。
5. 局中出现**贪合忘冲**的情况，相冲被化解；但若合被岁运再冲开，**原本的害就会暴露出来**，此时断事要回头看"害"。

## 五、四柱宫位中的表现

四柱宫位各有所主：**年柱长辈祖根，月柱父母事业与兄弟，日柱自身与配偶，时柱子女与晚景**。合、冲、害在不同宫位，呈现出不同的"家庭剧本"。

| 宫位关系 | 合（结合） | 冲（动荡） | 害（暗伤） |
|---|---|---|---|
| **年月** | 祖荫顺，家世和 | 祖业破败，背井离乡 | 长辈心结，得不到祖上福荫 |
| **月日** | 父母与配偶相协，事业顺 | 青年离乡创业，婚姻不易 | 配偶与父母暗中不合，工作环境多变 |
| **日时** | 晚年得子女之助 | 子女远离，中晚年破财 | 晚年健康多忧，子女沟通难 |
| **年时** | 一生首尾呼应 | 祖孙不相认，远赴他乡 | 一生首尾有暗伤，子孙难回故土 |

**日支最关键**：日支主夫妻宫与本人身心。日支被冲主感情动荡、离婚、健康动摇；日支被合主姻缘至、配偶黏；日支被害则主夫妻同床异梦、心结不断。

## 六、现实应事的差异

同样一组地支作用，合、冲、害所应之事大相径庭：

- **合** → 结合、合作、姻缘、合伙、合同、结盟、黏着。合住的若是喜用则大吉，若是忌神则凶而不自知。
- **冲** → 动得大。主要应事：搬家、调动工作、出国、远行、车祸、夫妻争吵、官非败诉、求事不成、疾病加重。冲一动就是大动。
- **害** → 动得小，伤得深。主要应事：心结、代沟、暗流、慢性病、六亲隔阂、配偶冷淡、下属背刺、同床异梦、事业上的暗耗。
  - **寅巳申亥之害**——驿马相关，主走动频繁、出门离家。
  - **子未、卯辰、酉戌之害**——力量悬殊，容易穿坏，主六亲骨肉伤。
  - **寅巳、丑午、申亥之害**——常见"**穿倒**"：正印当偏印看，七杀当正官看。
  - **羊刃逢穿害**——其人死得快，恐有凶灾。

一句话记忆：**合主结合，冲主动散，害主暗耗**。

## 七、三者并见时的判断逻辑

八字中三者同时出现非常常见，正确的断命顺序是：

**第一步：先看合**

- 有没有三合、三会？若有，直接锁定，**其他冲害刑破一律让位**。
- 有没有六合？六合是否紧贴？天干有没有引化？
- 若合住的是喜用神：吉中带绊；若合住的是忌神：反而省心。

**第二步：再看冲**

- 局中原有的合，是否被冲开？——若被冲开，合失效，原本压住的忌神、原本藏着的害，都会重新活跃。
- 冲在哪一柱？对应什么宫位、什么六亲、什么人体部位？
- 冲的是喜用还是忌神？喜用被冲则凶，忌神被冲则吉。

**第三步：最后看害**

- 合被冲破后，留下的那道暗伤就是害。
- 害紧贴的力量最大，隔位的害**虽有其义，基本无实**。
- 局中若本已有害，遇岁运再引动，则害变明伤——此时应事最猛。

**记忆口诀**：**一看合否合住，二看冲否冲开，三看害否暗作**。三步走完，吉凶应事才算看全。

## 八、常见误区

1. **把害当成小问题忽视**——害虽力量小于冲，但在"相克又相害"同在时，伤害比单冲更深。
2. **认为合一定吉、冲一定凶**——合忌则凶，冲忌则吉，合冲本身是中性作用，吉凶看合/冲的对象。
3. **只看字面，不看动态**——八字最忌"静态断"。三合一旦被破，六合一旦被冲，局面全变。必须动态看合被冲破的整个过程。
4. **忽视隔位的力量变化**——相害越紧贴越凶，隔柱、遥位的害可以基本忽略；但合、冲紧贴才最有力。
5. **见相害就断婚破**——害要结合喜忌、宫位、格局、岁运综合看；格局佳、化解多时，害的凶性会大大减轻甚至消失。
6. **忽视"贪合忘冲、贪合忘害"**——一旦局中合成立且力量大，冲与害都要让位，此时反而无事。

## 十、一句话总结

- **相合 = 合作**；**相冲 = 明撞**；**相害 = 暗伤**。
- **相害是相合与相冲共同作用的产物**——没有合就不成害，没有冲也不成害。**这是三者之间最深的关系**。
- 断命必须**先合、再冲、后害**三者综合看，不可偏废。
- **三合、三会一成，冲害刑破皆可化**；**合被冲开，害方活起**——看命看的是动态，不是字面。
