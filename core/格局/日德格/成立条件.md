# 日德格

## 成立条件（古籍出处 + 客观判据）

### 条件 1：日柱为五日德之一

- **古籍出处**：
  - 《三命通会·论日德》（万民英）："**日德者，甲寅、丙辰、戊辰、庚辰、壬戌是也**。"
  - 《渊海子平·论日德》同。
- **客观判据**：
  - 日柱干支严格为 甲寅 / 丙辰 / 戊辰 / 庚辰 / 壬戌 之一。
- **不满足则破**：日柱非此五者，根本不入本格。

### 条件 2：四柱中日德叠见（严格成格要求）

- **古籍出处**：
  - 《三命通会·论日德》："**日德之格，贵叠见而嫌独逢**。须四柱中有两位以上方为真日德格。"
- **客观判据**：
  - 除日柱外，年 / 月 / 时柱至少再出现一位日德干支（五组合之一）。
- **现实考量**：符合此条者**极为稀少**——故实务中亦讨论"**日柱日德**"的弱化版（只具日德性格，格未必成立）。

### 条件 3：日柱不逢冲空亡

- **古籍出处**：
  - 《三命通会·论日德》："**忌刑冲空亡**。"
- **客观判据**：
  - 日支不被年 / 月 / 时支冲：
    - 丙辰 / 戊辰 / 庚辰日 → 忌戌在他柱。
    - 壬戌日 → 忌辰在他柱。
    - 甲寅日 → 忌申在他柱。
  - 日支不落空亡。
- **不满足则破**：日支被冲 → 德气散 → 格破。

### 条件 4：原局无刑破、无财官阳刃相犯

- **古籍出处**：
  - 《三命通会·论日德》："**忌刑冲空亡、财官阳刃**。"
- **客观判据**：
  - 原局无刑（如丑戌未三刑、寅巳申三刑等）加于日德。
  - 天干未透**紧贴日主的七杀**或**财星制日德**。
  - 无阳刃直接犯日支。
- **不满足则破**：七杀 / 偏官 / 刑冲集中加身 → 德气被破 → 沦为普通之命。

### 条件 5：最忌与**魁罡相冲克**（本格特有破格）

- **古籍出处**：
  - 《三命通会·论日德》："**日德最忌魁罡冲克**——各日德最怕见相应之魁罡**。"
- **客观判据**（各日德忌见之魁罡）：

| 日德日柱 | 最忌魁罡 | 破格理由 |
|---|---|---|
| **甲寅** | 庚辰 | 庚克甲、辰与寅不合 |
| **丙辰** | 壬辰 | 壬克丙、辰辰自刑 |
| **戊辰** | （自身已近魁罡之结构，注意壬戌冲） | — |
| **庚辰** | 庚戌、甲寅 | 辰戌冲、寅木犯之 |
| **壬戌** | 戊戌 | 戊土克壬、戌戌自刑 |

- **特殊说明**：**庚辰日**兼具日德与魁罡双重身份——既为日德之柔厚、又为魁罡之刚烈。其破格规则从魁罡格严论。
- **不满足则破**：日德与魁罡相冲克 → 大运流年遇到亦有大灾。

---

**前四条须同时满足**方为真日德格；第五条为本格之特殊破法，须额外留意。
