# 母子关系（成立条件）

> 母子关系是两气成象的"拟人化"变体，包含 **母慈灭子格**、**母象格**、**子象格** 三种子结构。各子结构的严格成立条件如下。

---

## 一、母慈灭子格（病极）

### 条件 1：满盘皆印，印绶过旺到克身的程度
- **古籍出处**：《滴天髓》"**母慈灭子关头异**"；任铁樵注"**满盘皆印，原局无财，或财微浮无根**——印旺而反灭其子"。
- **判据**：
  - ① 印星（正印 + 偏印）在天干 + 地支本气共三位以上；或
  - ② 月令 + 地支三合 / 三会成印局，印气统治全局；
  - ③ 日主虽看似"被生"，但印过旺反生成克。
- **不满足则破**：印一般旺 + 有其他平衡之物 → 只是身弱可用印，非母慈灭子。

### 条件 2：原局无财星（或财星浮弱无根）
- **古籍出处**：《滴天髓》（任铁樵注）"**无财为臣以护君主**——无财则印无制、反克日主"。
- **判据**：原局天干无财透出；或财透无根极弱；地支无财本气。
- **不满足则破**：原局有有根之财 → 财能制印 → 转入"以财破印"救格讨论。

### 条件 3：日主被印过旺所压，呈"五行过生反主灭"
- **古籍出处**：《滴天髓》"**五行过生反主灭**"。
- **五种具体表现**：

| 日主 | 印过多 | 现象 |
|---|---|---|
| 庚辛金 | 戊己土过多 | **土多金埋** |
| 壬癸水 | 庚辛金过多 | **金多水浊** |
| 甲乙木 | 壬癸水过多 | **水多木漂** |
| 丙丁火 | 甲乙木过多 | **木多火塞** |
| 戊己土 | 丙丁火过多 | **火多土焦** |

### 条件 4：命局无比劫救应（或比劫仅为浮弱）
- **古籍出处**：《滴天髓》"**母慈灭子，唯比劫可救**——比劫帮身、同时泄印"。
- **判据**：原局天干无比劫透出且有根；或比劫仅一位且无根力微。
- **不满足则破**：原局有强比劫 → 比劫帮身泄印 → 母慈不灭子 → 转入"印重身弱比劫救应"讨论。

**四条同时满足**方为真母慈灭子格。此格为"**从旺格中最底层者**"——印过旺压身、无财无比劫救应。救格唯一路径：**大运流年走比劫**。

## 二、母象格（贵）

### 条件 1：满盘皆比劫（比劫统治全局）
- **古籍出处**：《滴天髓》"**知慈母恤孤之道，始有瓜瓞无疆之庆**"。
- **判据**：比肩 + 劫财在天干 + 地支本气共四位以上；或月令 + 地支三合 / 三会成比劫局。

### 条件 2：命局带食伤（食伤为"子"）
- **古籍出处**：《滴天髓》"**母象格者，比劫为母、食伤为子**——子不可无、无子则气闭"。
- **判据**：食神或伤官透于天干，或地支本气有食伤；食伤与比劫形成顺流。
- **不满足则破**：无食伤 + 有印 → 转入"子象格"；无食伤无印 → "比劫独旺"破格。

### 条件 3：原局无印绶（或印绶力微）
- **古籍出处**：《滴天髓》"母象格**忌印绶**——印来则食伤被克、母子之间不通"。

### 条件 4：原局无官杀（或官杀力微）
- **古籍出处**：《滴天髓》"母象格**忌官杀**——官杀克比劫（母），使母不恤孤子"。

### 条件 5（辅佐条件）：有财星接应食伤顺流（比劫→食伤→财）
- **古籍出处**：《滴天髓》"**食伤为子、财星为孙**——母喜含饴弄孙"。
- **非必要**：不满足不破格，但格局助力缺失。满足则为"**瓜瓞绵绵**"之象。

**前四条必须同时满足**方为真母象格；条件 5 为格品助力。

---

## 三、子象格（贵）

### 条件 1：满盘皆比劫（比劫统治全局）
- **古籍出处**：《滴天髓》"**子象格者，满盘皆比劫为子，四柱有印绶为母**——子众母孤"。

### 条件 2：命局带印绶（印为"母"）
- **古籍出处**：《滴天髓》"**比劫为子、印绶为母**——子众之格，母孤则贵"。

### 条件 3：子象格与母象格的关键区别——看食伤还是印
- **古籍出处**：《滴天髓》"**同为满盘皆比劫，有食伤者为母象格，有印绶者为子象格**"。

### 条件 4：原局无强财星夺印（否则"恋妻不顾母"）
- **古籍出处**：《滴天髓》"**子象格忌财星**——财克印为'恋妻不顾母'、大忌"。
- **判据**：原局天干无财透出；地支财本气力微。
- **不满足则破**：财透克印 → 母失护 → 格破。

### 条件 5：官杀须辨"有情无情"
- **古籍出处**：《滴天髓》"**官杀两头蛇，有印则生，无印则克**"。
- **有情**（有印接应、官→印→身）→ 喜官杀（贵格升）。
- **无情**（无印化官、官直接克身）→ 忌官杀（格破）。

### 条件 6：食伤须辨"原局有无财"
- **原局无财** → 食伤为喜（纯泄比劫秀气）。
- **原局有财** → 食伤为忌（食伤生财 → 财克印 → 母被害）。

**前四条必须同时满足**方为真子象格；条件 5、6 关乎大运与流年的喜忌分判。

---

## 三者判定对照

| 对比项 | 母慈灭子 | 母象格 | 子象格 |
|---|---|---|---|
| 母身份 | 印（过旺）| 比劫 | 印 |
| 子身份 | 日主（被灭）| 食伤 | 比劫 |
| 核心结构 | 满盘印 + 无财无比劫 | 满盘比劫 + 有食伤 | 满盘比劫 + 有印 |
| 核心病 / 贵 | 病：母过灭子 | 贵：母恤孤子 | 贵：子保母孤 |

## 母子十神对应关系

| 关系位 | 十神 | 气向 |
|---|---|---|
| **母** | 印绶（生我）/ 比劫（同我，在食伤视角为母）| 生 / 同 |
| **身** | 日主 | 中点 |
| **子** | 食伤（我生）/ 比劫（同我，在印视角为子）| 泄 / 同 |

**因视角不同，比劫可为"母"（生食伤时）也可为"子"（被印生时）**——这是理解本类格局的关键。
