# 闭库

闭库分为四大类：**自动闭库、地支相合相会闭库、天干相合闭库、再次相冲闭库**。

## 一、自动闭库

### 情形 1：墓气遭冲克 → 自动入库 → 自动闭库
- 墓气透出天干或露出地支时遭遇各种冲克 → 自动进入墓库相应五行。
- 进入墓库后便会**自动闭库**。
- 此类情况下的墓库，可通过：
  - 大运流年中其他五行帮助制服克己方的五行后 → 自动开库、出库。
  - 地支相冲、相刑、相破、相害 → 强行开库。

**示例**：辰库墓气癸水在天干透出壬水，壬水被贴身戊土克制 → 壬水进入墓库躲避 → 辰库自动闭库。当大运或流年天干出现甲木 → 甲木克制戊土，相当于赶跑或压制戊土 → 辰库自动开库，让壬水从墓库出来回到天干。也可能大运或流年地支出现戌土 → 辰戌冲开库。这种情况相当于"**敌人打到基地，大家必须齐心协力医治，对外可以迸发出强大力量**"。

### 情形 2：五行过旺 → 自动入库 → 自动闭库
- 某种五行在命局中特别旺盛，若命局中有对应墓库 → 此时墓库视为**储存之库**。
- 该五行进入库后**自动闭库**。
- 要打开此库，必须通过**地支相冲、相刑、相破、相害**。

### 核心理解
自动闭库是为了**保护进入墓库内的五行**：
- 就像玩游戏时，我方外围据点被敌人攻破，战斗人员撤回基地后必关闭基地大门。
- 也像粮食大丰收收入仓库后，必紧闭仓库大门防止盗窃。

自动闭库是**符合常理的现象**，无需借助外力，可与自动开库结合应用。

## 二、地支相合或相会闭库

本命局或大运流年地支中，**四大墓库遇到六合、三合、半三合、三会时会闭库**。

### 情形 1：自动开库遇相合相会 → 闭库
针对墓气透干露支、无冲克（或有其他五行帮助制服克己方的五行），墓库自动开库的情况。当这种自动开库的墓库遇到相合相会 → **闭库**，同时将透干、露支的力量**收回大部分**，不再为墓库外的干支提供能量。

一般来说墓库不可能完全闭合，至于收回多少力量，初学阶段可以**先理解为把所有力量都收回来**，也可理解为"在外驻守的战斗人员都返回基地"。

**示例**：戌库墓气丁火在天干透出丙火，地支中有巳火，命局中丙火巳火不受冲克 → 此时戌库为丙火、巳火提供能量。但命局中出现卯木 → **卯戌合** → 戌库力量收回大部分，不再为丙火、巳火提供能量。

### 情形 2：强行开库后遇相合相会 → 闭库
命局中通过地支冲刑破害已经开库，参与冲刑破害的四大墓库，在大运或流年中遇到相合相会又可关闭。

**示例**：命局中同时存在戌库和辰库，两库相冲 → 两库可能都开库。大运或流年来**酉金** → 辰酉合 → 戌库和辰库**全部闭库**。大运或流年来**卯木** → 同样情况（卯戌合）。

## 三、天干相合闭库

与"开天库"类似，是相合闭库的特殊情况——**天干相合引动地支墓库闭库**。

### 规则
命局中有某墓库、天干透出该库的墓气（需是**与墓气完全相同的阴天干**）、透干的墓气不存在冲克，若天干又存在能与墓气相合的天干 → 天干相合会**引动地支墓库**起到闭库作用。

### 四组天干合闭天库

| 墓库 | 墓气透干 | 相合天干 | 天干合 → 引动闭库 |
| --- | --- | --- | --- |
| **辰库** | 癸水 | 戊 | 戊癸合 → 闭辰库 |
| **未库** | 乙木 | 庚 | 乙庚合 → 闭未库 |
| **戌库** | 丁火 | 壬 | 丁壬合 → 闭戌库 |
| **丑库** | 辛金 | 丙 | 丙辛合 → 闭丑库 |

### 关键注意
- **天干必须是与墓气完全相同的阴天干**（如辰库须癸水，不能壬水）。
- 这相当于墓气真身到了天干，天干的癸水被合 = 地支辰库被合。

## 四、再次相冲闭库

### 基本规则
- 本命局中通过地支冲（主要）、刑、破、害可开库。
- 若**大运、流年再次出现与墓库相冲或相刑的地支** → 闭库。
- 如同开关：按一下打开，再按一下关闭。
- 破、害力量足够时同样可实现此开合。

### 示例
命局中同时存在戌库和辰库，两库相冲 → 两库同时开库。
- 大运或流年再来**戌土** → 辰库可能闭库。
- 大运或流年再来**辰土** → 戌库可能闭库。

## 冲量的三种结果：开库 / 闭库 / 毁库

以**未库**为例：

### 1. 适量冲刑 → 开库
本命局中有一个未库且无刑冲，岁运中出现一个冲或刑 → 适量刑冲 → 未库开库。

### 2. 适量刑冲 + 再次相刑 → 闭库
本命局有未库 + 戌库（未戌相刑），属于适量刑冲 → 未、戌都可能开库。
- 若戌力量较大 → 只有未开库。
- 岁运再逢戌 → 未库被二次相刑 → **未库闭库**。

### 3. 过量刑冲 → 毁库
本命局有一未库 + 两戌库（两个戌土力量 > 一个未土）→ 只有未开库。
- 岁运再逢戌 → 未库被**三个戌土相刑** → 三刑冲一 → 不是再次开库，而是**墓库直接被刑坏** → **毁库**。
- 若本命局直接出现三刑冲一 → 也属毁库。
- 力量强大的二刑冲一有时也可能毁库。

**毁库** = 相当于用大铁锤砸毁墓库的门和里面的财物。

## 开闭库的总体原则

| 状态 | 触发方式 |
| --- | --- |
| **自动开库** | 墓气透干露支 + 无冲克 |
| **自动闭库** | 墓气透干露支 + 遇冲克 / 五行过旺入库 |
| **强行开库** | 地支冲、刑、破、害 |
| **相合相会闭库** | 六合、三合、半三合、三会 |
| **开天库** | 墓气透干 + 阴天干相冲 |
| **闭天库** | 墓气透干 + 阴天干相合 |
| **再次相冲闭库** | 已开库状态下再次冲刑 |
| **毁库** | 过量刑冲（三刑冲一等） |

开闭库的运用需**结合月令旺衰与整体命局格局**综合判断。
