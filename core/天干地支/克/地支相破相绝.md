# 地支相破、相绝

## 定位

- **相破、相绝**：地支作用关系中**力量较弱、应用频率较低**的两种关系。
- 在八字实战中应用较少，但必须了解其基本原理以完成整个地支作用体系。
- 本文附录"**地支作用力量大小排序**"——这是用刑冲合害破绝六者的综合定位。

## 一、地支相破

### 定义

- **相破**：互相破坏、斗争、克制。
- 表现为**破坏、破产、捣乱**以及**内部矛盾冲突**等。
- 指的是在四地旺（子卯午酉）、四长生（寅巳申亥）和四墓库（辰未戌丑）之间发生的**五行之气的相互冲突与克制状态**。
- 这种相破关系是**两两之间循环往复、彼此相互破坏**的。

### 六组相破

- **子酉相破**
- **卯午相破**
- **寅亥相破**
- **巳申相破**
- **辰丑相破**
- **未戌相破**

### 四地旺之破（子酉、卯午）

- 这种破**原本是相生关系**（酉金生子水、卯木生午火） → 协调合作天经地义。
- 然而**双方都处于帝旺状态**，旺而不生。
- 原因：双方都认为自己强大（帝旺），但又得不到好处，**不愿意履行自己的职责**——就像应该提供服务的职能部门不为你服务。
- 具体表现：
  - **酉金不生子水**。
  - **子水不生卯木**。
  - **卯木不生午火**。

#### 四地旺相破的象意

**子酉相破**：
- 象征**小灾**，通常表示经常遇到**经济困境**。
- **子破酉**：做事情缺乏主见，常常自行其事。
- **酉破子**：所托非人，容易自欺欺人。

**卯午相破**：
- 代表**门户破败**，多见纷争或**情感方面麻烦**。
- **卯破午**：犹豫不决，行事缺乏原则，行为往往无规无矩。
- **午破卯**：真假难辨，容易轻信他人，结果容易功败垂成。

### 四长生之破（寅亥、巳申）

- **寅亥相破**：生中带破——亥中壬水克寅中丙火；寅中甲木化解亥中壬水。
- **巳申相破**：合中带破——巳中丙火、戊土克申中庚金、壬水；申中壬水克巳中丙火。
- 这两种破是**一种较为隐蔽的小矛盾**，通常不易被察觉，只有在双方力量悬殊较大时才显露。

#### 四长生相破的象意

**寅亥相破**：
- 代表**矛盾中蕴含合作、失败中孕育成功**。
- 暗示性格上的优柔寡断，容易受外界影响、容易被他人说服。
- **寅破亥**：过于自负，企图剥夺他人成就和荣誉，通过阿谀奉承获取利益。
- **亥破寅**：狡诈不诚实，违背内心真实想法行事，最终面临天罚。

**巳申相破**：
- 表达矛盾与合作共存，但也带有一定克制——预示在经历失败后能东山再起，但成功中也存在瑕疵。
- **巳破申**：需要通过委曲求全获取帮助，结果却受到侮辱。
- **申破巳**：表面上假装顺从，实则暗中捣乱，心怀不轨。

### 四墓库之破（辰丑、未戌）

- **辰丑相破**：辰中乙木破丑中己土；丑中辛金破辰中乙木。
- **未戌相破**：未中丁火破戌中辛金；戌中辛金破未中乙木。
- 这种破坏是**同行内部的矛盾**，通常**较为隐蔽，不易被察觉**。

#### 四墓库相破的象意

**辰丑相破**：
- 象征**墙壁和坟墓的破败**，亲戚之间难以和谐相处，经常发生争吵和矛盾。
- **丑破辰**：意味着**疑虑和误解**。
- **辰破丑**：象征**孤独和沉默**。

**未戌相破**：
- 代表**人物的伤害和破损**，破损中带着刑罚，尤其不利，也暗示着**人们有城府**。
- **未破戌**：无端的竞争和指责、相互辱骂和责备、破坏并失去。
- **戌破未**：做事没有依据，真假难辨，**名誉和声望被毁**。

### 相破总结

| 类型 | 力量 | 特点 |
| --- | --- | --- |
| 四长生（寅亥、巳申） | 弱 | 破坏力不强，更多是和合的关系，通常**不以破来论命** |
| 四帝旺（子酉、卯午） | 重 | 各气已成形、各自强势不让、自以为是；此时的破**较为严重**，具有破产冲破含义 |
| 四墓库（辰丑、未戌） | 隐蔽 | 各气已在退期；此时的破更像**同行内部矛盾**，相对隐蔽，使用情况较少 |

## 二、地支相绝

### 定义

- **相绝**：十二地支中存在的**四个相绝组合**。
- 相绝涵盖多种含义：**相克、断绝、感情互不相关**等。
- 可以看作一种**特殊的克**——相绝的两个字都容易受伤，只是程度有所不同。
- 相绝代表**彻底的断绝关系**——恩断义绝、不再往来，甚至意味着**牢狱之灾、疾病和灾祸**，同时表示事情告一段落、事情不成、恋爱不成、家庭分离、亲情分散。

### 四组相绝

- **寅酉相绝**
- **卯申相绝**
- **子巳相绝**
- **午亥相绝**

### 一、寅酉相绝（寅酉金绝）

- 寅属木、酉属金 → **金克木**。
- 它们两个是**互相绝**的，双方都受一定程度损伤，但**寅木受到的伤害更为严重**。
- **寅木象意**：四肢骨骼、身体部位、肝脏、胆囊。
  - 寅木受损 → 可能导致**车祸、身体损伤甚至生命危险**，以及肝脏、胆囊方面的健康问题。
- **酉金象意**：法律、牢狱之灾。
- **寅在六亲中象征离散和绝交** → 犯寅酉绝的人往往**老死不相往来**、情感上断绝、毫无情谊。
- 典型现象：夫妻离婚后、或夫妻在同屋檐下却形同陌路、互不理睬。

#### 案例

男性生辰八字带寅酉相绝：
- 有丑半合年住辛酉 → 代表第一任妻子：**寅酉相绝最终导致婚姻破裂**。
- 寅作为印星 → 暗示婆婆介入了夫妻关系；命主对第一任妻子也存在不满。
- 第二段感情日时丁壬相合、时柱人寅 → 表示第二任妻子天干地支都相合，暗示第二任妻子对命主管得严。

### 二、卯申相绝（卯申木绝）

- 申属金、卯属木 → **金克木**。
- 因此卯申绝主要是**申绝卯**。
- 这种绝称为**卯申木绝**，**卯木受损伤**。
- **卯木象意**：神经系统、肝胆、眼睛。
  - 卯绝 → 可能导致**脑神经问题、脑中风、脑血栓、神经痛**，以及由此引发的**眼病、耳病、聋哑、痴呆**等伤残问题。
- **申金象意**：道路上行驶的车辆 → **车祸、刀剑之伤**。
- 卯申相绝可能引发**路途中的伤害**。

#### 案例

女性八字，**卯木代表她的丈夫**：
- 八字中戊癸相合、卯戌相合——分别是天干五合和地支六合——象征夫妻和谐美满。
- 然而**卯申之间存在相绝关系**。
- 不幸的是，她的丈夫**在一场车祸中去世**。

### 三、子巳相绝（子巳火绝）

- 子中隐藏癸水、巳中隐藏戊土 → **戊癸合是天干五合之一**。
- 从表面看，子水克制巳火，但实际上它们的**作用力很小**——因此尽管它们相绝，作用微乎其微，有时甚至可以忽略不计。
- 这种绝称为**子巳火绝** → 子水和巳火都会受伤。
- **子水象意**：肾脏、泌尿生殖系统、子宫、腿脚、肠道、牙齿、阑尾。
- 子巳绝意味着**男女之间的感情破裂，容易产生口舌是非，导致分离**。

### 四、午亥相绝（午亥水绝）

- 它们既相克又相合：午属火、亥属水 → 午中藏丁火，亥中藏壬水，**丁壬相合**。
- 因此这种绝被称为**合绝**——彼此交战但同时又与对方关系挺好，类似于两个人恋爱时融洽，吵架时又针锋相对。
- 故**午亥之间的绝野是好坏参半**。
- 这种绝称为**午亥水绝**。
- 在午亥相绝过程中，**阴水阳火反倒而行，无情无益**。
- 古籍云："八字中午亥绝最无情。"
- **亥水象意**：膀胱、生殖系统、血液。
  - 亥水被午火中的己土污染 → 容易有**输卵管结石、血管堵塞**。

## 三、相破、相绝与五行相克的力量排列

### 五行相克在地支层面的意义

- 地支层面的"克"通常就是**五行相克**：
  - **水克火**（子克午、亥克巳）。
  - **火克金**（巳克酉、午克申）。
  - **木克土**（寅克辰戌丑未、卯克辰戌丑未）。
  - **金克木**（申克寅、酉克卯）。
  - **土克水**（辰戌丑未克子、亥）。
- 但地支实际作用时，还需结合**刑、冲、合、害、破、绝**综合判断。
- 单纯的五行相克**力量最弱**——地支实际发生作用的是上述成组关系。

### 地支作用力量大小总排序（按优先级）

按作用力量大小进行排序，优先级别如下：

**三会** > **三合** > **生的半三合** > **六合** > **墓库的半三合** > **拱合** > **六冲** > **相刑** > **暗合** ≥ **六害** > **六破** > **相觉（相绝）**

### 分层理解

| 层级 | 类型 | 说明 |
| --- | --- | --- |
| 第一层（合力最强） | 三会、三合 | 五行成局，能量最聚 |
| 第二层 | 生的半三合、六合 | 紧密而稳定的相合 |
| 第三层 | 墓库半三合、拱合 | 次级合力 |
| 第四层（对抗力） | 六冲 | 最直接的对抗 |
| 第五层 | 相刑 | 气机不协、间接牵制 |
| 第六层 | 暗合 ≥ 六害 | 隐性作用、相互损耗 |
| 第七层（最弱） | 六破、相绝 | 影响力有限，多作辅判 |

### 力量大小的应用原则

- **理论上力量大的作用可以化解力量小的作用**。
- 但八字即人生，所有事都不能以绝对思维考虑。
- 根据八字整体组合，**一定程度的缓解肯定是可以做到的**。

### 应用要点

- **相破和相绝力量比起刑害要小很多**。
- 而且**相绝组合中的卯申、子巳、午亥都是"暗合"** → 它们也属于暗合的一种表现。
- 实战中：
  - 相破常与刑冲合害同现 → 看整体合力，不单独论破。
  - 相绝看**彻底断绝的含义** → 感情、合作、亲情上的"绝决"信号。

## 常见误区

- **"相破相绝不用理会"**：虽然力量小，但在**感情断绝、疾病、意外**等特殊场景中仍有指示意义（如寅酉绝主车祸、卯申绝主脑神经）。
- **"相绝就一定凶"**：子巳相绝作用微乎其微，有时可忽略；需结合整体判断。
- **"破绝与冲无异"**：三者力度天差地别——**冲 >> 害 > 破 > 绝**。
- **不分类型看待相破**：四长生相破弱、四帝旺相破重、四墓库相破隐——须分别对待。

## 一句话总结

- 地支相破、相绝是**地支作用力量中最弱的一层**，但各有其独特指示意义——相破主"内部矛盾、暗中对抗"，相绝主"彻底断绝、恩断义绝"。
- 地支作用的力量顺序是**合 > 冲 > 刑 > 害 > 破 > 绝**——以此顺序理解地支互动，便可抓住主次。
- **大作用可以化解小作用**，但不能以绝对思维看命——**要以八字整体组合综合判断**。
