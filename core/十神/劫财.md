# 劫财

## 定义
- 与日主五行相同、阴阳不同（甲见乙、丙见丁…）。
- "劫"有争夺、分取；"财"指钱财资源成果。
- 与比肩同属比劫一类，但阴阳不同导致作用方式不同：
  - 比肩偏向"并列同类并立"。
  - 劫财偏向"切入同类风度"——更容易出现"风财、夺财、插手资源"。

## 真相：根本不是单纯"夺财"
劫财对财运的影响是**同类之气对财与资源分配权的介入**，关键看身强身弱、财星状态、位置、以及有没有官杀食伤来制化。

## 判断四步骤

### 1. 看日主强弱
- **日主弱见劫财**：先表现为帮身，拖得住自己。
  - 如"财多身弱"结构，适量劫财反而能让日主站稳、担财。
- **日主旺劫财多**：同类已不缺，再来则从帮身转为风夺。
  - 若命局又见财星透出，财易成比劫争夺的对象。

### 2. 看财星状态
- 财星透干：易被劫财盯上、争夺。
- 财星有根：有来源基础，不易一碰就没。
- 财星无根又外露：最怕比劫风夺。
- 财弱又被劫财贴身而克，再无官杀制比劫、无食伤泄比劫——钱财留不住，合作易散。

### 3. 看劫财落位
- **年柱**：早年家境、家族资源分散、成长环境竞争感。
- **月柱**：力量最直接（月令贴近现实），同事同行竞争。
- **日支**：直接影响婚姻；男命财兼看妻子，劫财切入此位，关系易有外部介入、利益搅动。
- **时柱**：后期财务、项目分成、子女花费、晚年资源调度。

### 4. 看有无官杀食伤制化
- 官杀制比劫：劫财虽在也不敢乱争。
- 食伤泄比劫：劫财之气转化为行动力、执行力。
- 最怕结构：**身旺 + 比劫 + 正财漏 + 官杀无力**——财易被劫财争夺。

## 现实表现

### 三类常见负面模式
1. **财路被封**：辛苦做起的事中途有人插手，合作易分力不均。
2. **钱财来去快**：不一定是自己挥霍，也可能是人情往来、朋友合作、兄弟拖累。
3. **男命感情竞争格局**：财先看妻子，劫财切财易致感情不稳、外力介入、婚后财务与感情纠缠。

### 劫财的"双刃"性质
- 竞争：同类竞争、人情开支、合作分立。
- 流动消耗：重义气，为朋友兄弟花钱，应酬较多。
- 行动力胆量：敢拼敢抢，商业销售创业中可成助力。

## 能带来正面财运的三种情形
1. 命主适合创业、销售、市场竞争型行业。
2. 命局财星根深，足以承受比劫风夺。
3. 行运配合得宜（大运流年让财星流通）。

## 核心判断
劫财真正代表的不是"破财星"，而是**同类之气对财与资源分配权的介入**。
看劫财抓一条：先看身强身弱，再看财星状态，再看位置，最后看有没有官杀食伤。
离开这几步，断"破财、克妻、朋友害你"都不严谨。

## 补充

### 为什么用"劫"这个字

- 在古人的社会里，同宗、同辈、同门、同行，**资源往往放在同一口锅里**——你多一点，我就少一点。
- 所以"劫"字背后藏的，是一种很原始的焦虑：**"我不想吃亏，我不能被挤掉"**。
- 劫财不是真的有外人来抢你钱，而是**把你丢进一张"同类的关系网"里**——让你一辈子都绕不开一个课题：**"我和别人之间到底怎么划分界限"**。

### 劫财重者的三大底层机制

1. **极强的同类敏感**：别人和自己一样时，内心立刻亮起三个灯——可以一起、可以比一比、也可能被抢。
2. **身份感容易受外界影响**：很难只按自己的内心生活，总在意别人做到了哪一步、别人怎么看你——面子、输赢、位置，对你来说都不是小事。
3. **边界感先天不清晰**：习惯把自己和别人绑在一起考虑——钱、时间、精力、人情都容易因"我们"这个概念被摊出去。

### 劫财的本质：同质能量外溢

- 劫财本质是**同质能量外溢**：
  - 你身上那股"做事、争取、主张自我"的力量，不只用在自己身上。
  - 还会被一群和你差不多的人激发、放大、互相碰撞。
- 一个人的安全感不再来自"我是谁"，而来自——
  - 我在这群人里排第几。
  - 我有没有被落下。
  - 我是不是被当成有用的人。
- 很多劫财重的人嘴上说"讲义气、够仗义"，深处藏的是**怕被抛下、怕被轻看**的恐惧。
- 这份恐惧不面对，劫财只会表现为：**冲动、破耗、疲惫**。

### 劫财让钱不再"独占"

- 命理上说"劫财夺财"过于简化——更准确的说法是：
  - **劫财让钱不太容易"只在你一个人手上安安静静待着"**。
- 这类人赚钱往往：
  - 靠**圈子、人脉、合作**。
  - 钱来时周围站着一圈人。
  - 钱花出去理由也多：朋友的事、兄弟的项目、家里谁谁谁有困难、一起做生意的要周转。
- 结果不是没有收入，而是**独占感很弱**——
  - 说好听叫**共赢**。
  - 说难听叫"到你手里都成了**流动**，而不是**存量**"。

### 为什么劫财者"边界容易模糊"

边界本质有三个支柱：
1. **自我价值感**：我觉得自己值得什么。
2. **规则感**：我愿意和别人怎么分。
3. **承受力**：我能承受别人的不满意、关系的紧张吗？

劫财重的人，**前两样不一定差**，往往是第三样绷不住——
- 不是不知道该拒绝、该划分、该谈清楚。
- **而是太怕一说"不"，关系就变了**。
- 太害怕被贴上"小气、不够意思、不够兄弟"的标签。

> 与其说劫财劫的是"钱"，不如说它**在持续考验你——敢不敢承受关系里的那一点点紧张和不讨好**。

### 劫财的正面用途：不是所有人都破财

- 劫财有它非常好用的地方。
- 现代社会很多位置，本来就需要你**和别人竞争、协调、博弈**：
  - 销售、渠道、代理、经商、开店、带团队、跑市场。
- 一个完全没有劫财那股"不服输、不怕抢、敢出头"的劲儿的人，反而很难做出成绩。
- **看盘的高手**：一见劫财不会摇头，而是看——
  - 它落在什么位置。
  - 搭配了什么十神。
  - 整体气势是往上走还是往下塌。
- 站对位置：劫财是能带起一批人做事、在复杂场合里拿捏关键的那股锐气。
- 站错位置：四处救火、到处借力、永远替别人兜底。

### 劫财的代价

- 代价往往不在某一笔钱的得失。
- 而在**长期习惯性地**：
  - 把自己的资源优先给同类用。
  - 把自己的时间优先用在别人需要你的地方。
  - 把真正该留给自己的经历、学习、积累**放在了后面**。
- 很多劫财重的人容易在**中年以后突然醒悟**：
  - 朋友是有的、感情也不是没有、事也没少做。
  - 但好像**真正属于自己的东西不多**。

### 劫财不只是"资源分夺"，还是"资源重新分配的力量"

- 劫财出现意味着**竞争与共享同时存在**——可以靠合作变强，也可能因争夺而失衡。
- **正向面**：资源共赢、朋友带财、团队合作、市场扩张。
- **负向面**：内耗竞争、被分裂、情绪焦虑、方向混乱。

### 劫财旺的人为什么"钱多却守不住"

- 劫财旺的人**行动快、野心强、敢冒险**。
- 赚钱靠**局感与人脉**，不靠制度或积累。
- 能第一时间嗅到机会，也会第一时间动用资金——因为"进来快、出去也快"。
- 赚的钱不属于**稳态财富**，而是**流动性收益**。
- 能量越大、风险越高；动得越快，稳定越难。
- 所以**"破财"只是表象**——真正的问题是**他们不擅长"守"**，习惯用**风险换空间**。

> 命中劫财旺的人，不该追求"存款"，而应追求"**资金循环**"。

### 劫财的正向人格

- 参与感、竞争性、控制欲——
  - 用在**创业、管理、谈判**上 = 能量的正向释放。
  - 用在**情绪宣泄、关系博弈**上 = 钱和人都散的剧本。
- 盲派有句话：**"劫财见财星，反为吉"**——
  - 只有当**资源足够大、结构足够稳**时，劫财的动能才能被吸收进循环，而不是掀翻系统。
  - 能量要有出口，动机要有结构。

### 劫财做大的原则："资源动，不代表人动"
- 把劫财用在**布局而非抢占**，用在**盘活而非对抗**。
- 通过**联盟、合作、股份、平台**来赚钱——看似把蛋糕分了，其实是在做更大的盘。
- **会分的人越分越多，不会分的人越抢越少**。

### 适合的行业类型
- 适合：创业、经商、销售、管理、谈判、投资、社交型岗位。
- 不适合：求稳守城、依附他人资源的岗位。
- 劫财旺而官弱：烦体制、不服管。
- 劫财旺而财弱：急功近利、难聚财。

### 劫财真正动的是什么

- 劫财真正动的不是你的"钱"，而是**你和钱之间的关系**——
  - 你是**一个人拿钱**，还是**一群人一起分钱**？
- 劫财做了三件事：
  1. **把直线的一对一关系变成多人博弈**——从"你和财富"变成"你和一群跟你差不多的人一起碰同一块财富"，要分、要争、要谈判。
  2. **改变了财富的流转方式**——从水龙头直接入手，变成"风扇"（合伙、团队、家族、轮番借用）。钱在流，只是通过了更多人的手。
  3. **放大了"人"的变量**——财运不再只是能力问题，而是**人际关系问题**：你跟谁站队、和谁共享、怎么分账、敢不敢说"不"。

### "破财"的真正含义

- 传统视角"劫财夺财"——从个人看，多一人分就是少自己一份（直觉）。
- 但把镜头拉远：**有些人的钱，恰恰是靠劫财才能做大的**。
  - 一个人干最多是个手艺人、自由职业者。
  - 一群同类聚在一起 → 团队、公司、圈子、社群。
- 你拿到的不是一份**干净的独享**，而是**掺杂着人情、合作、博弈的那部分利润**。
- 这就是劫财与财富之间更真实的关系：
  - **既是稀释，也是放大**。
  - 稀释的是"独享的比例"。
  - 放大的是"盘子本身"。

### 三个关键自问

1. 你的路径本身**适不适合单打独斗**？如果不适合却硬孤军深入，劫财就会以各种形式提醒你（有人出现来插手、来分城、来抢客户）——其实是在**纠正你的路径**。
2. 你有没有**边界感和分配能力**？——不懂边界就会：逞义气、不好意思拒绝、把合伙当感情、把分账当作"不好意思开口"。学会立规则、谈条件，劫财就会变成"朋友圈 = 资源池、同行 = 合作方、团队 = 杠杆"。
3. 你到底**想要什么样的财富体验**？
   - 要**绝对控制感**——钱全在自己账上。
   - 要**规模和速度**——哪怕分掉大部分，只要盘子够大，剩下那一块也够。
   - 执着于第一种 → 劫财会让你痛苦。
   - 转向第二种 → 劫财会成为你**拉帮结派、放大利润**的通道。

### 劫财与婚姻

- 劫财一出现，婚姻就不再只是两个人的事——而是**两套同类能量在同一个屋檐下争位置、争资源**。
- 劫财把婚姻推向三个方向：
  1. **同类对抗而非互补合作**：两个都像"老板"、都觉得自己有理。争执常常不是为了事情本身，而是为了"你凭什么不听我的"。婚姻里最伤人的不是吵架，**而是"你每一次都要赢"**。
  2. **资源变成战场而非工具**：钱、时间、对外关系、谁牺牲多——这些最容易出事。更隐蔽的是**注意力资源**：谁更被重视，谁更像附属品。表面吵琐事，实际在争**安全感的来源**。
  3. **把"第三方变量"带进婚姻**：这里的第三方不一定是异性，更多是**同类圈层**——兄弟朋友、原生家庭、合作伙伴、江湖义气。一方活在"组建家庭"里，一方活在"圈子"里——你说他不顾家，他说你不懂人情世故。

### 婚姻中的三条出路

1. **婚姻靠"愿意守规则"维系，而不是靠"谁更讲理"**——劫财旺的人讲理往往锋利，但锋利不是稳定。稳定来自**关键处愿意退一步，把共同体放在面子前面**。
2. **真正的忠诚不是"不出轨"，而是把家庭放在"外部关系"前面**——劫财容易重外圈、重兄弟义气、重关系网；若优先级不摆正，伴侣很难相信自己是第一位。
3. **劫财不是注定离婚的符号，而是婚姻必须升级"管理能力"的信号**——越是同类能量强，越需要学会**谈判、分配、边界**，把话说在前面，而不是靠情绪补救。

### 劫财最怕的不是吵架，而是**吵完仍然没有规则**
- 仍然靠情绪处理、靠冷战处理、靠"谁先低头"处理。
- 会进入循环：争执越频繁、信任越薄、感情越像战场。

### 现实调整建议

1. **学会把钱和感情分开记账**：帮一次忙可以，但心里要知道这是**支出、代价**，而不是某种"将来必然要回报的投资"。不要一边偷偷记账，一边又装义气。
2. **在关系中说出自己的底线**：承认自己的边界，别人才会慢慢理解你的局限。
3. **对圈子保持清醒**：不是所有兄弟都值得你亮底牌；真正靠谱的人，是**愿意和你一起承担风险、也愿意听你说"不"的人**——不是只在你掏钱时最热情的脸。

### 核心提示

- 劫财的本质：**让你在关系里学会站稳脚跟**。
- 命里只是提供一个**看清模式的视角**——提醒你：
  - 你容易在哪些地方把自己送得太快、太多。
  - 你的优势恰恰也是"在圈子里做事、在合作里放大自己的力量"。
- **当你看见结构、承认它、调整它**——劫财就不再只是来抢你东西的星，而是**逼着你学会在关系里站稳脚跟、在义气与底线之间找到那条既能共赢又不牺牲自己的线**。
- 钱从冰冷的数字变成"围绕数字的人"。**所谓"破财"，其实是你被人和关系逼着学会了筛选**；所谓"共赢"，其实是**你找到了一群能一起赚钱也讲规则的同类**。

## 补充：女生择偶为何要避开劫财男

### 劫财男的本质
劫财男的底色是"**分你利、夺你权、抢你爱**"的那股劲。他**表面仗义、骨子里自我**，身边哥们多、圈子大，嘴里挂着兄弟情谊，但一到分利益、分钱、分资源时谁都不让——哪怕是最亲近的人。

### 四种典型现实表现
1. **钱财不稳、爱合伙、钱花得快**：舍得给外人花大钱，不舍得为你花大钱；对外跟哥们多，对家里反而计较。
2. **感情里爱较劲**：明明说爱你，家务、钱、孩子教育都要比谁对谁错谁更厉害；吵架嘴不输气场不服。
3. **婚姻不稳、容易有第三者**：天性爱闯荡，应酬多、暧昧不断；容易出现吃回头草、出轨、再婚、复合等反复。
4. **压力大时转移责任、躲事**：擅长"和你一起扛"，但扛到最后把锅丢给你、丢给朋友，自己全身而退。

### 为什么不适合做老公
天生资源分配感极强——**在职场朋友圈是一把好刀，进到婚姻里这种气场就变成"自家人都要防着点"**。什么都要分一半，没有界限感、没有安全感。

| 维度 | 你希望的 | 他真实的样子 |
| --- | --- | --- |
| 钱财 | 指望他养家 | 只想着圈子与合伙，攒自己的小金库 |
| 感情 | 你想求稳 | 他总和你比，把感情当竞技场 |
| 生活 | 你想安稳 | 他喜欢折腾，应酬天天有 |
| 危机 | 你希望他顶事 | 他从不正面扛 |

### 核心警示
- 劫财男不是"不行"——而是**注定你会累**。
- 女生择偶能避则避；若命里就遇上了，**认清现实、别幻想他会为你改变**。

## 补充：劫财不是破财，是人生考验

### 最浅层的误读
- "听到劫财 = 要破财、钱守不住"——没错，但**太浅**。
- 劫财真正的含义并不是单纯"钱被拿走"，而是命局出现了一股**跟你抢的力量**：
  - 可能是兄弟姐妹、朋友、同事。
  - 也可能是市场上的竞争者。
  - 甚至是你自己心里的**那股冲动**。
- 换句话说，劫财就是你命里的**一场资源争夺**。

### 劫财重者的两个显著特征
1. **钱来得快、去得也快**：
   - 不怕没机会，但机会一来身边总有人来分一杯羹。
   - 或者自己不安分，钱刚到手就投出去——**很难攒下来**。
2. **人际关系复杂、带争强气**：
   - 跟他做朋友，容易兄弟闹掰。
   - 跟他合作，容易因利益翻脸。
   - 在家庭里，兄弟姐妹之间也可能因为财产分配闹得不愉快。

### 劫财的正面：胆识与拼劲
- 做生意没点劫财，往往**缺乏那股狠劲**——机会来了也不敢下手。
- 劫财重的人常想："**别人能做，我凭什么不敢**"——正是这股劲能闯出局面。

### 劫财的负面现实画像
- 钱留不住、容易惹是非。
- 身弱的人碰上劫财 = **雪上加霜**：别说发财，连自己都可能被拖累。
- 常见困境：
  - 想赚钱，最后钱被兄弟朋友套走。
  - 合伙创业到最后闹掰，钱没了人也反目。
  - 婚姻里因钱分不清导致感情破裂。
  - 自己心气太重，喜欢冒险投机。

### 劫财最忌"心浮气躁"
- 命里带劫财的人如果不懂收心——日子过得很折腾，钱守不住，人也累。
- 但如果把这股劲用在**对的地方**（市场竞争、拼业绩、开拓新领域）——反而能成事。

### 劫财的真正面貌
- 劫财看似破财，其实是**人生的一道关卡**。
- 考验的是：**你能不能稳得住心，能不能把这股劲用在对的地方**。
- 用得好：突破的利器。
- 用不好：跌倒的根源。
- **劫财不是坏，它只是不允许你小富即安**——逼着你要么折腾到破产，要么折腾到成功。

## 补充：劫财七杀在恋爱中的性格表现

### 核心原理
- 七杀劫财在事业上是**一个好作用**——能打、能干、能直接做事。
- 但用在**恋爱中**却呈现出**鲜明的冰冷人机感**。
- 原因：
  - **比劫** = 官杀所拥有之物，也是去拥有财星之物。
  - **过度用比劫** = 只和客观物质世界打交道，忽视情感需求。

### "人机感"的三大症状
1. **过分关注物质世界而忽视情感需求**。
2. **过分在意事物本身而抛弃主观能动**。
3. **过分要求自我价值的实现**，不愿意纳入别人的世界去思考。

### 官制比劫的极端表现
- 全盘都在"**官制比劫**"做工 → **必然出人机感**。
- 全盘只用"**比劫克财**"做工 → **必然出舔狗**。
- 无论哪一种，事业上都可能不错，但**婚恋层面一定出现隔阂**。

### 典型聊天场景（反面案例）
- 对方发照片逗小猫 → 你回"小猫很可爱"，把天聊死。
- 对方问"你今天过得怎么样" → 你答"还行"，然后没有下文。
- 对方聊美食 → 你追问"好吃吗"、"多少钱"，效率至上。
- 关键词：
  - **过于直接**。
  - **过于讲究效率**。
  - **表达像"表逻辑"，而不是"表情感"**。

### 反面案例的深层成因
- 驱动宫位（时干 → 月干）以"**七杀制劫财**"为主：
  - 七杀 = 用物质世界去压别人。
  - 劫财 = 用直接手段去执行。
  - 合在一起 = **用法相洞天去砸别人** = 极度严肃、冰冷、直接。
- 这种取向应用在事业上 → 大将冲杀之气；但应用在恋爱上 → **情感结冰**。

### 驱动能量强弱的差异
- **驱动能量强 + 七杀劫财驱动** → 冷硬、有压迫感、处处主导。
- **驱动能量弱 + 七杀劫财驱动** → 不强势但"聪明"、事业稳定、内心却缺乏柔情储备。
- 后者尤其容易出现在女命身上：事业不错、收入稳定，但**找不到对象**。

### 月柱四正（子午卯酉）的放大效应
- 月柱地支若为子午卯酉其中之一 → **易把四正之气向内聚**：
  - 例如卯木伤官合穿戌土 → 表面视野好、实际内部"卯辰相争"。
  - 表现：**对任何交往对象都保持竞争心态**。
  - 深层：**将恋爱对象视为陌生人、视为有图谋的交易对象**。

### 矫正方向
- **关注点转换**：从"事物本身"转到"对方这个人"。
- **话题核心**：目的不是完成一次对话、而是**加深了解**。
- **原则**：不卑不亢、不讲效率。
- **核心意识**：恋爱是"**把自己脆弱的一面给对方**"，而不是"完成一次高效沟通任务"。

### 一句话总结
- **官制比劫用在事业** = 雷厉风行。
- **官制比劫用在恋爱** = 冰冷人机。
- 八字里看到"七杀 + 比劫 + 官星"三者齐全时，必须先**在恋爱中主动抑制这套做工**，否则一定会把亲密关系聊成竞争关系。

## 意象体系

### 人物层面

- **男命劫财 = 姐妹**；**女命劫财 = 兄弟**（同五行 + 不同阴阳 → 性别相反）。
- 泛指**异性兄弟姐妹、异性朋友、竞争对手、合伙人**——尤其带有"**争夺、分利、介入资源分配权**"的特征。
- **劫财旺的命主异性缘好**（不论男女）：劫财为喜用时异性缘尤其佳——劫财代表异性兄弟姐妹或朋友。
- **劫财克正财** → 男命劫财旺对**父亲与妻子不利**；遇到劫财大运流年严重者会打骂妻子、甚至分开；并且容易**丢工作**——正财的财富往往通过工作获得，正财被克可能失业。

### 精神范畴

- **热情坦诚、应变力强、善于交际、胆大刚强、见义勇为、积极竞争、轻财重义**（正面）。
- **冲动鲁莽、具攻击性、豪赌投机取巧、冒险犯难、克妻损财对父亲不利、背信弃义**（负面）。
- 劫财与比肩最大的差异：**比肩较旺的人初识给人本分不懂变通、交际能力一般**，**劫财较旺者在交际方面具备与生俱来的应变能力**——尤其月干透劫财时，初次见面就能迅速变得熟络，热情坦诚、聊到别人内心深处。

### 事业方面

- **适合从事消耗性质、新闻性质或错综复杂多变的行业**：拆除业、运动员、武人军人、券商股民、资本运营商、旅游娱乐、风月中人、赌徒、骗子、窃贼等——**有其锐利的一面，也有其危险的一面**。

### 劫财 + 比肩过多的共有现象（劫财更严重版本）

1. **劫财为忌、劫财特别旺 + 财星弱**：**十赌九输**；劫财为喜的人情况好一些。比劫为忌者甚至可能倾家荡产——每个人都应远离赌场。
2. **劫财为忌 + 严重劫财克财组合**：容易成为**到处拉人搞传销、去投资的骗子**。
3. **劫财比比肩更会克妻损财、对父亲不利**——从名字就能看出。
4. **劫财为忌者借钱不想还，别人借他的钱也收不回来** → 容易成为背信弃义之人。

## 与其他十神组合

### 劫财 + 七杀（羊刃驾杀）

- **羊刃驾杀**：天干有七杀、地支有阳刃——意志极其坚强、竞争意识特别强。
- **非常适合武职工作**：参军、政法类事业；也可引申为**外科手术医生**——七杀是病痛、羊刃是刀，用刀解决病痛就是外科手术医生。

### 劫财 + 正官（官制比劫）

- 劫财过多是"**本位主义**"的人、为人相当冷酷无情，虽然外表和蔼可亲好像能服从别人意见，其实内心并非如此 → **不宜从事公职**。
- 四柱中有正官、正官能克制劫财 → **化险为夷，把损失变为利益**——一向被认为嚣张的态度会转为"**强稳的领导力**"，促成事业发达。

### 劫财 + 财星（劫财克财）

- **劫财好比土匪**：不光是"争"还要"夺"，不仅夺还能抢，不只是抢还能杀。什么都抢——命主想要的东西劫财都想去抢、往往趁火打劫（钱财、感情、事业）。
- **为喜用**：可以帮助命主得财。
- **为忌**：反而致使命主破财遭灾。
- **劫财专门克正财** → 男命劫财旺对父亲和妻子不利。

### 劫财 + 食伤

- **劫财生食伤**：比劫的冲击力通过食伤被转化为**表达、创意、技能输出**——食伤又生财，形成"比劫 → 食伤 → 财"的良性流通。
- **劫财通过食伤泄**：攻击性 / 竞争欲被转换为创造力与才艺 → **赚同行的钱而非抢同行**。

### 劫财 + 比肩（比劫成群）

- 劫财与比肩同透或成势 → **一群同类**围绕日主，既可能抱团也可能分化；**关键看是否有官杀制、有食伤泄、有财星引**。

## 各宫位象意

### 年柱

**年干劫财**：

- **全局财星弱 + 劫财旺**：**出生贫寒、幼时困苦**——年柱代表祖辈父母宫位，劫财劫走祖辈资源 → 家境不富裕。
  - 凡日主得令且旺、年柱干支皆劫财，或年柱 / 月柱天干皆透劫财者 → 出身家境不佳。
  - 若**命局印星有力 + 日主健旺 + 年干劫财坐下财星 + 月柱干支不见官杀星** → 命主出生之后家运从贫困中复苏、逐渐迈向荣昌。
- **大多不是长子长女，也不会得到遗产**——经验总结。
- **年干 + 年支皆劫财者**：**不利父亲，父亲容易早逝**——劫财克财（命局中财代表父亲）+ 年柱代表父辈宫位 → 双重不利。
- **年干劫财**：**双亲与邻里关系好、善于交际**——尤其母亲不是家庭主妇型；年柱是父母宫，父母宫透劫财代表双亲善于交际（劫财擅应变）。

**年支劫财**：

- **无祖产可继承、需白手起家**——特别是**甲、丙、戊、庚、壬五个阳干日元**尤其应验（这几个日元在地支上的劫财即"**阳刃 / 羊刃**"）。
- **兄弟姐妹的个性外柔内刚**：其中一个可能过房给他人——劫财透天干代表命主性格圆滑，藏地支代表内在固执刚强。
- **年支可隐身祖国、故乡**：年支劫财 → 容易**移民外国、在他乡成家立业**——没办法在故乡或祖国干出一番事业。

### 月柱

**月干劫财**：

- **为人慷慨仗义但会结交酒肉朋友**。
- **为了照顾朋友及兄弟姐妹常常吃亏**。
- **具有挥霍的习性**：若月支存在正财则一生财富来来去去、难以实现富裕。
- **男命**：对政治抱有兴趣、与妻子不和睦。
- **自小志向高远、做事孤注一掷、三十岁之前难以有所发展**。
- **月干劫财为忌**：年少时家境贫寒。
- **月透劫财为喜用**：男性会得到姐妹的帮助、女性会得到兄弟的协助；若劫财空亡则帮助力度不足，且恐怕手足会早逝。

**月支劫财**：

- **性格固执、自尊心强烈、喜欢打抱不平、爱打听他人隐私**。

- **月支劫财为喜用 + 遭官杀严重克制**：出生家境贫寒。
- **月支为阳刃 = 阳刃格**：格局详见 [`格局/羊刃格.md`](../格局/羊刃格.md)。

### 日支（配偶宫）

**劫财在日支**：

- **性格**：**脾气犟、认死理、装强不回头**；若四柱其他柱比劫旺盛者更是如此；**性情刚直**——若四柱无官杀克制 → 暴躁且有攻击性，极易引发纠纷。

### 时柱

时柱代表人最终的归宿、结果之地；时干与时支大体相同：
