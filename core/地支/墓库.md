# 墓库

## 定义
- 八字中**辰、戌、丑、未**四个地支，被称为**四库（墓库）**。
- 是气走到一个阶段后的"收尾与储存"状态。
- 不是消失，而是被**收起来、压下去**，等合适的时候再被打开。

## 核心：收尾 + 储存
墓库有两层意思：
1. **收尾**：这股力量不再高调表现，容易沉下去、慢下来。
2. **储存**：力量并没有消失，它在暗处积累、沉淀，等机缘一到又会被翻出来。

## 四库对应
| 库 | 收藏的主气 |
| --- | --- |
| 丑 | 金库 |
| 未 | 木库 |
| 辰 | 水库 |
| 戌 | 火库 |

## 核心警示
真正可怕的往往不是"有墓"，而是**有东西一直被压在墓里不肯动**——久而久之反而成为负担。

## 墓库重者的气质特征
- 事情很少干干脆脆翻篇，喜欢拖着、关联旧账、心事又来。
- 感情上：有缘分，但容易**迟迟放不下**——表面断了，心里还留着角落。
- 工作上：有人很会收尾、理烂摊子；也有人任何事都拖到最后一刻。
- 情绪上：不属于来得快去得快的类型。很多感受不在当下爆发，而是**夹在心里过很久才翻出来**。

## 判断维度

### 1. 谁入墓，味道完全不同

#### 日主入墓
- **有印生扶** → 像暂时退回房间、关起门蓄势。沉稳内敛，不等于没前途。
- **被财官消耗厉害、生气接不上** → 像掉进封闭的坑，一来事只会蒙头扛，既不求助也不调整，最后把自己压垮。

#### 用神入墓
- 命主身上最能帮他的力量不在明面。
- 例：财星为用但藏丑库 → 现实里财路不一定亮眼，反而在冷门行业、边缘位置、不显山漏水处慢慢积累。账面紧巴，手上有底。
- 例：官星为用但浮在戌库 → 不一定是仕途不行，更多是人在规则系统里**走得小心翼翼，靠熬和稳定慢慢上升**。
- 这类格局，一旦大运冲开墓库 → 有时反而是**憋了多年突然爆发**。

#### 忌神入墓
- 忌神被压，表面看是"稳"。
- 但一旦大运流年把库冲开，压着的不满不甘不服会**集中爆发**。
- 现实表现：突然辞职、突然翻脸、突然做出意料之外的决定。
- 很多人事变故表面是某一年冲开，其实是**墓库里积了太久的气找到了出口**。

### 2. 墓库的位置决定在生活的哪一块收尾
- **日支入墓**：婚姻关系里容易有拖着不绝的情况——不是恩断义绝，而是藕断丝连。
- **时支入墓**：和晚年、子女、长期计划有关——很多事要到后半程才慢慢打开。

### 3. 大运走入墓库
- 不等于"这十年完了"。
- 多半意味着：
  - 某段状态要收尾。
  - 某种表现要收起来。
  - 工作节奏放缓、人际圈收缩、身体开始提醒。
- **冲开墓库的年份**，多是旧账被翻出来——之前压着没处理的事要面对，很少是凭空掉下来的祸事。

## 现实表现
- 墓库重的人：
  - 好处 = **韧劲、能熬事、不轻易被风吹走**，记性深。
  - 麻烦 = **一旦走偏容易想不开，把不值当的事也压在心里，越久越难解**。

## 一句话总结
命里的墓库不是在吓人，而是在提醒一个规律：
- 很多事情不是失去，是**收起来了**。
- 很多能力不是没有，是**藏起来了**。
- 很多情绪不是消失，只是**被压下去了**。

墓库既让人能扛事、守势、不轻易放弃，也让人容易拖延、不肯了结。
- 命里墓库重，现实中就更需要学会**真实的收尾**。
- 敢于承认：**这件事到此为止，这段关系就停在这里，这个阶段的自己已经走完了**。
- 在序与藏之间，少塞一些垃圾，多留一点空间给真正值得积累的东西。
