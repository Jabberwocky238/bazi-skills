# 金火对（总论）

## 性质：相克两气对
- **五行关系**：火克金（官杀 / 伤官关系）。
- **非"印生两神成象格"**——该格专论相生两气，金火属相克，不入双清贵格。
- 属"两气成象"的**相战 / 对立**一支。

## 两气成象通则（适用于本对）

### 成象要件
- 命局只见金、火两种五行（其他三种基本不见）。
- 金、火力量大致相等。
- 月令为金或火之一。

### 与相生对的根本区别
- **相生对（如金水对）**：两气顺生、气通为清。
- **金火对（本条）相克**：火克金、气机对立——属"金火铸印""火多金熔""火旺金衰"等病象或特殊结构。

### 特殊情况：金火铸印（贵格）
- 火克金本为病，但若火用来"**铸印**"（炼金成器）→ 化克为用 → 贵格。
- 条件：金稍强而火适度克之，金被炼成精钢而非被熔毁。
- 典型：庚金坐午 / 巳之类——火炼庚金成大器。

### 病象：火多金熔 / 火旺金衰
- 火过旺而金弱 → 金被熔毁 → 身弱无力、事业多败。
- 现实：过度压力之下崩溃、健康受损、工作中断。

### 取用要点
- **火炼金得当** → 贵格（金火铸印）。
- **火过旺 / 金过弱** → 病格（火多金熔 / 火旺金衰）。
- **喜通关五行**：**土** = 火 → 土 → 金（火生土、土生金），可化战为顺。
  - 金火对 + 土通关 → "火土金"三气相生，格局升级。

### 古籍背书
- 《滴天髓》："**火旺金衰，得土则救**。"
- 《穷通宝鉴》："庚辛金逢丁火，**锻炼而成器**——金火铸印之象。"
- 《滴天髓·象论》："两气成象，见土金火皆平衡者，贵格也。"

## 本对下属格局
- **火多金熔**：火过旺克金的病象。详见 `火多金熔/`。
- **火旺金衰**：火旺而金弱的身弱格。详见 `火旺金衰/`。
- **金火铸印**：火适度炼金成器的贵格。详见 `金火铸印/`。

## 一句话总结
- 金火对 = **相克两气、以克为用（或为病）**。
- 火炼金成器 → 贵格（金火铸印）。
- 火过旺毁金 → 病格（火多金熔 / 火旺金衰）。
- 要解：**得土通关**（火→土→金），可化战为顺流贵格。
