# 偏印格

## 定义
- **偏印**：生我之同性五行——阳日主受阳生为偏印；阴日主受阴生为偏印。
  - 例：甲日见壬（阳水）为偏印；乙日见癸（阴水）为偏印。
- **偏印格**：**月令取偏印为用神**的基础正格。
- 《子平真诠》十正格之一——与**正印格**对照的"偏"一面。
- 又称"**枭神**"或"**枭印**"——古籍中对偏印评价复杂，既有贵气之用，也有"夺食"之凶。

## 核心：偏门之贵，独特之路
- 正印 = 正统学问、主流资源。
- 偏印 = **偏门专业、独特技艺、非主流路径**。
- **气质更锐、更深、更孤**：
  - 对一般人看来奇怪的东西特别感兴趣。
  - 走的是"**少有人走的路**"。
- 也是"**枭神夺食**"病格的前提条件——偏印过旺克食神即成病。

## 偏印 vs 正印

| 对比项 | 偏印 | 正印 |
|---|---|---|
| 阴阳 | 与日主相同 | 与日主相反 |
| 气质 | 偏、锐、深 | 正、和、厚 |
| 典型学问 | 玄学、心理、医卜、技艺 | 学术、教育、文史 |
| 风险 | 易克食神（枭神夺食）| 较少病象 |
| 母缘 | 后母、远亲、非主流长辈 | 慈母、正缘长辈 |

## 成立条件（严格依《子平真诠·论偏印》）

### 条件 1：月令地支本气为偏印，或偏印透干且月令有根

- **古籍出处**：
  - 《子平真诠·论偏印》："**偏印即枭神**，月令为格。"
- **客观判据**：
  - ① 月令地支本气为偏印；或
  - ② 月令地支藏偏印且偏印透于天干。
- **不满足则破**：月令偏印不显 → 当以月令本气另论他格。

### 条件 2：偏印透干且通根

- **古籍出处**：
  - 《子平真诠·论偏印》："**偏印须透干有根，方能为用**。"
- **客观判据**：
  - 偏印透于天干，且在地支有本气或中气根至少一位。
- **不满足则破**：偏印虚浮 → 格难立。

### 条件 3：原局无食神紧贴被偏印克（或有财护食）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论偏印》："**偏印与食神相克，最忌夺食**。"
  - 《子平真诠·论食神》："食神最畏枭神——枭印夺食，福去祸至。"
- **客观判据**：
  - ① 命局无食神 → 偏印无夺食之事，格清；或
  - ② 有食神但偏印与食神不紧贴 → 无直接克制；或
  - ③ 有食神 + 紧贴偏印 + **有财护食**（财克偏印救食）。
- **不满足则破**：偏印贴食无救 → 转入"**枭神夺食**"病格讨论。

### 条件 4：日主不过强

- **古籍出处**：
  - 《子平真诠·论偏印》："**偏印身弱亦可用**。"
  - 《子平真诠·论印绶》："印绶喜身弱，忌身强。"
- **客观判据**：
  - 日主于月令非比劫本气，或他柱比劫印星位数不超过食伤财官之和。
- **不满足则破**：身强 + 偏印重 → 印闷气机 → 当取食伤泄身或财破印，非纯偏印格。

### 条件 5：偏印不宜过重（不成枭神过旺）

- **古籍出处**：
  - 《子平真诠·论偏印》："**偏印宜少不宜多**——多则夺食为凶。"
- **客观判据**：
  - 偏印在天干 + 地支本气合计不超过两位。
- **不满足则破**：偏印过重 → 枭神之气太盛 → 即便无食神也闷气机 → 降格或转入"印重身弱"讨论。

---

**五条同时满足**方为真偏印格。偏印格之所以复杂：**偏印本身既可为用（生身、带偏门才华），也可为病（夺食、闭塞生机）**——关键在与食神、财、日主的关系配合。

## 判断维度

### 1. 偏印的位置
- **月柱偏印**：最正宗——"月令偏印格"。
- **日支偏印**：自坐偏印 → 本人气质偏门。
- **时柱偏印**：晚年或子女走偏门路径。

### 2. 偏印与食神
- **偏印 + 食神不紧贴**：各司其职，偏印生身、食神泄秀，双美。
- **偏印 + 食神紧贴 + 无财护**：枭神夺食 → 破格大忌。
- **偏印 + 食神紧贴 + 有财护**：救应到位，格保。

### 3. 大运流年
- **印运（偏印加重）**：身弱喜，身强忌。
- **食伤运**：视原局——若命局本不重食伤，走食伤可泄秀；若原局已枭印贴食，走食伤则事件爆发。
- **财运**：克偏印，救食神（若食神为用）。
- **官杀运**：生偏印 → 强化偏印之力，视吉凶视结构。

## 常见误区

### 1. "带偏印 = 必枭神夺食"
- 只有"**偏印紧贴食神 + 无救**"才叫夺食。
- 单纯偏印格不带食神时，并无夺食之事。
- 命理上偏印不是凶煞，是"**偏门之印**"。

### 2. 把偏印等同"后妈 / 继母"
- 传统论"偏印为后母"——取其"**非亲生的滋养**"之意。
- 现代语境：偏印的"滋养"偏向**非主流、非传统、非血缘的支持**——导师、精神信仰、偏门专业、网络社群等。

### 3. 以为偏印一定不好
- 偏印格带来**独特的气质与才能**：
  - 对玄学、心理、医学、技艺、直觉等领域有天赋。
  - 在"**冷门但深奥**"的领域容易出众。
- 许多命理大家、心理学家、宗教人士命中带偏印。

## 现实表现

### 性格
- **深沉、内敛、思想独立**。
- **对偏门学问敏感**：玄学、心理、神秘主义、小众专业。
- **直觉强**：常有非逻辑的判断准确。
- **与众不同的气质**：不合群但不孤僻。
- **母缘较复杂**：与母亲关系疏远 / 后母带大 / 或与长辈关系非典型。

### 职业路径
- **玄学、命理、宗教**：传统意义的偏印职业。
- **心理学、精神分析**：现代版偏印。
- **医学（尤其中医、针灸、心理）**：医卜同源。
- **技艺型专业**：手工艺、古典乐器、特殊工艺。
- **研究者 + 偏门专业**：古文字、小众语言、濒危技艺。
- **创作者 / 艺术家（非主流）**：走自己的路。

## 核心判断
- 偏印格的复杂在"**有食神则防夺食，无食神则清气独显**"。
- 识别关键：
  - 偏印旁边有没有紧贴的食神？
  - 有食神则有救吗（财护食）？
- 格清 + 走偏门路径 → 偏印命造的独特贵气。

## 一句话总结
- 偏印 = **偏门之印**——非主流的滋养。
- 这类命造**不适合走大众路径**，硬走反而被憋。
- **深挖一门冷门技艺 + 守住独特气质** → 是偏印格真正的福气。
