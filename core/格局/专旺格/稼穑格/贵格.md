# 稼穑毓秀（稼穑格贵格）

> **定位**：稼穑格的**最贵形态**——稼穑（土专旺）+ 金秀气吐露的复合贵格。
> 本文作为稼穑格的"**贵格进阶**"专论，详细论述稼穑毓秀的成立条件、气质与古籍背书。

## 定义
- **稼穑毓秀**：稼穑格（土专旺）+ **金点缀泄秀** → 土生金，秀气外显 → 成就**稳重中见灵秀**的顶级贵格。
- 古籍谓"**稼穑得金吐秀，名稼穑毓秀**"——是稼穑格从"**纯厚**"升为"**秀贵**"的关键跃迁。
- "**毓秀**"二字：**毓 = 养育 / 生发；秀 = 秀气 / 灵秀**——指稼穑厚土能养育出秀气之象。

## 与普通稼穑格的区别

| 对比项 | 纯稼穑格 | 稼穑毓秀（本条） |
|---|---|---|
| 土气 | 四柱满盘土气 | 满盘土气 + 金点缀 |
| 秀气 | 缺——偏纯厚闷重 | 有——土生金成秀 |
| 气质 | 厚重稳实但偏沉闷 | 稳重之上带灵秀 |
| 贵气 | 贵格（纯专旺）| **贵格之贵**（专旺 + 秀气双全）|
| 代表行业 | 实业 / 土地 / 传统行业 | **学者 / 专业人士 / 文化 / 艺术**（土气厚 + 金秀立身）|
| 气象 | 厚重的山 | **山中玉**（厚土中长出美玉）|

**关键点**：稼穑毓秀**不是另一种格局**，而是稼穑格的**升级态**——多了"金"这一秀气之神。

## 成立条件（严格依《三命通会》）

### 条件 1：先满足稼穑格所有成立条件

- 日主为戊或己土。
- 月令为土（辰、戌、丑、未之一）。
- 地支土气遍布（辰戌丑未全见，或四柱土多）。
- 原局无甲乙木透干克土。
- 原局无重水冲土。
- 详见 [`成立条件.md`](./成立条件.md) 前五条。

### 条件 2：原局有金（庚辛 或 申酉）点缀

- **古籍出处**：
  - 《三命通会》："**稼穑得金吐秀，名稼穑毓秀**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 金（庚辛天干 或 申酉地支）在命局中**至少一位**——透干最佳，藏支亦可。
  - 金**不宜过多**——一至两位为"点缀"，三位以上则过度泄土，降格为"土薄金旺"。
- **不满足则破**：
  - 无金 → 只成纯稼穑格，不成毓秀。
  - 金过多 → 土被金泄过度 → 格降（非毓秀，转入"土薄金旺"讨论）。

### 条件 3：金须得土生而不被木克

- **古籍出处**：
  - 《三命通会》："毓秀之金，须土生而无木克。"
- **客观判据**：
  - 金的位置靠近土——即土与金紧贴相生（天干戊己旁有庚辛；或地支丑 / 辰旁有申 / 酉）。
  - 原局无甲乙木透干与金相战（木克不了土反被金克，但若金弱反被木克间接通路）。
- **不满足则破**：金被木克、金远离土 → 毓秀之气不足。

### 条件 4：金不过旺（维持"稼穑为主、金为辅"的基调）

- **古籍出处**：
  - 《三命通会》："**金宜点缀，不宜喧宾夺主**。"
- **客观判据**：
  - 金的数量 ≤ 土的一半（如土四位、金一至二位）。
  - 月令仍为土（若月令为金，则格变"土金相生格"或"金专旺格"，非稼穑）。
- **不满足则破**：金过旺 → 本末倒置，主角由土变金 → 不再是稼穑毓秀。

---

**四条同时满足**方为真稼穑毓秀。核心在**"主次有序"**——稼穑（土）为主、金为辅。金是"秀气的出口"，不是"取代土"的主角。

## 喜忌总则（相对纯稼穑格的变化）

| 类别 | 纯稼穑格 | 稼穑毓秀 |
|---|---|---|
| **土运** | 大喜 | 喜（但过多转为纯稼穑）|
| **金运** | 喜（适度泄秀）| **大喜**（强化毓秀）|
| **水运** | 忌（冲土破格）| 喜（适度 —— 金生水成**土金水**三行顺流更贵）|
| **木运** | 大忌（克土）| 大忌 |
| **火运** | 喜（温土）| 视金之强弱——金弱则火太旺反克金 |

## 现实表现

### 气质
- **稳重中见灵秀**——既有稼穑的厚德承载，又有金气的清秀决断。
- **说话不多但说必中肯**——金为义，言辞有分寸。
- **有实力也有品味**——不像纯稼穑那样朴实，带着**文化与审美感**。

### 典型职业
- **学者、研究者、专业人士**——靠"**稳厚底子 + 精微判断**"立身。
- **司法、审计、精密工程**——土金相生所重的"裁断与精密"。
- **文化传承 + 实业经营**——一手做文化、一手做实业。
- **艺术 + 收藏**——尤其玉器、金石、陶瓷、古董相关行业。
- **传统医学（中医、针灸）**——土为脾胃、金为肺骨，医道两兼。

### 与纯稼穑的职业对比
- **纯稼穑**：地产、建筑、农业、实业、仓储、餐饮。
- **稼穑毓秀**：加上**需要"**慢工细活 + 精准判断**"的高端专业**——如精密工程、古籍研究、中医名家、书画收藏等。

## 与相关格局的关系

| 格局 | 关系 |
|---|---|
| **稼穑格**（本格的基础）| 稼穑毓秀是**稼穑格的升级态** |
| **土金两气对 / 土金毓秀** | `../../../两气成象/金土对/土金毓秀/`——虽名相近但结构不同；**两气成象的土金毓秀**是土 + 金两气（非土独旺），**稼穑毓秀**是土专旺 + 金点缀 |
| **从革格**（金专旺）| 反面——金独旺而土弱 |
| **土重金埋**（病格）| 土过旺而金被埋——稼穑毓秀的反面病象 |

## 稼穑毓秀 vs 土金毓秀（容易混淆）

| 对比项 | 稼穑毓秀 | 土金毓秀（两气成象）|
|---|---|---|
| **所在分类** | 专旺格（土一气独旺）| 两气成象（金土对）|
| **土气** | 四柱满盘土 | 土为两气之一（非满盘）|
| **金气** | 点缀（一至两位）| 两气之一（与土力量相当）|
| **气象** | 厚土生秀玉 | 土金两气双清 |
| **贵气来源** | 土专旺 + 金吐秀 | 土金相生双清 |
| **详见** | 本文 | `../../../两气成象/金土对/土金毓秀/` |

## 一句话总结
- 稼穑毓秀 = **厚土中长出美玉**——稳重的山，养出秀气的金。
- 不是把土换成金，而是**稼穑之厚 + 金之秀气** 两全。
- 关键在"**主次有序**"——土是主、金是秀；土厚生秀则贵，金过反客则降。
