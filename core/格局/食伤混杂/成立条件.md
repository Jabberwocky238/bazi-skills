# 食伤混杂判断维度

### 1. 食伤的比重
- **食神为主，伤官为辅**：偏温润，但有锋芒穿插。
- **伤官为主，食神为辅**：偏锋利，但有柔软底色。
- **两者势均**：最典型的混杂 → 风格最难统一。

### 2. 日主与食伤的关系
- **身强 + 食伤混杂**：混杂的双输出都能驾驭 → 灵活多面。
- **身弱 + 食伤混杂**：两边都泄身 → 双耗 → 更不妙。

### 3. 救应条件
- **印制伤官 / 印克食神之一**：
  - 留下一个、去掉另一个 → 格清。
  - 但要小心"枭神夺食"破格。
- **财化食伤**：食伤生财 → 把两种输出都落到"财"上 → 通关化解。
- **合化去留**：
  - 食神合官（甲己合）等组合 → 食神离场 → 留伤官。
  - 反之亦然。

### 4. 大运流年
- 大运补齐能去掉混杂一方的五行 → 格清化。
- 大运反而加剧混杂 → 问题放大。
