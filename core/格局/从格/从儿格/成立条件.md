# 从儿格成立条件（真从儿格）

### 条件 1：日主极弱
- 天干周围无比印，地支四位**本气、中气、余气皆无比印**。

### 条件 2：月令必须是食伤
- 要么月令本气为食神或伤官；
- 要么地支月令**参与三合或三会食伤局**（如甲日见巳午未会火为食伤局；庚日见亥子丑会水为食伤局——按日干"我生"之五行核对）。

### 条件 3：无印星和官杀来捣乱
- **无印**：印克食伤——印来就破。
- **无官杀**：食伤克官杀——官杀来就打架破格。
- 若有一点点但很弱 → 是**假从儿格**。

### 条件 4：食伤力量要最大
- 天干地支中**食伤数量 ≥ 比劫**；
- 且**食伤数量 > 财星**。
- **可以建一点点财星**——这样才华才能变现（食伤生财、才华换钱），但**财星不能太多**——多了就变成从势格了。
