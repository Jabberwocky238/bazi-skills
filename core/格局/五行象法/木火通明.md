# 木火通明

## 定义
- **木火通明**：命局以**木为体、火为用**，木生火、火透达 → 整个结构**光明温暖、气机通透**的一种**象意格局**。
- 标准画面：**甲 / 乙木日主 + 丙 / 丁火透出 + 地支有根**。
- 属于"**象法格**"而非正格 —— 不是单一十神成格，而是**整体五行意象**的描述。

## 核心：木被火照亮，气机外显
- 木 = 生发之气，潜藏的生命力。
- 火 = 显化之气，向外的光与热。
- 木没有火 → **阴郁、闷气、才华藏而不发**。
- 木有火透 → **聪明外显、灵感涌现、才情如光**。
- 所以叫"**通明**"：整个人像被点亮了一样，气清、神清、思路清。

## 象意来源
- 传统：**甲木丙火或乙木丁火** 组成的结构被视为"森林或草木在阳光下生长"。
- 生机 + 光 + 温度 → **生长环境最理想**的五行组合。
- 《滴天髓》与《渊海子平》等古籍中对木火相生之象多有褒扬。
- 象意派命理中，木火通明与"**金水相涵**""**火土夹带**""**水木清华**"等并列为**意象型吉格**。

## 成立条件（严格依《穷通宝鉴》《滴天髓》）

### 条件 1：日主为甲或乙木

- **古籍出处**：
  - 《穷通宝鉴·甲木总论》："**甲木参天，脱胎要火**。"
  - 《穷通宝鉴·乙木总论》："乙木纤柔，丙丁相助。"
- **客观判据**：日主天干必须为甲木或乙木。
- **不满足则破**：非甲乙日 → 不入木火通明。

### 条件 2：火透干（丙或丁）且通根（巳 / 午本气或寅 / 戌中火根）

- **古籍出处**（本条最核心，直接命名本格）：
  - 《穷通宝鉴·甲木》："**甲木用丁，木火通明**。"
  - 《穷通宝鉴·甲木寅月》："甲木生寅月，余寒尚存，**必得丙火解冻，丁火透为秀才**。"
- **客观判据**：
  - 丙或丁透于月 / 日 / 时干，且地支有巳或午本气根（或寅中丙火、戌中丁火中气）至少一位。
- **不满足则破**：火虚透无根 → 不能"明"；无火透而仅藏支 → 不能"通"（光不外显）。

### 条件 3：木有根（寅、卯本气根，或亥、子长生生扶之地）

- **古籍出处**：
  - 《穷通宝鉴·甲木》："**甲木须得寅卯之根，方能生火**。"
- **客观判据**：
  - 地支有寅或卯本气根至少一位，或亥、子（甲木长生于亥、乙木长生于午，衍生生扶之意）通根。
- **不满足则破**：木无根 → 无源可生火 → 通明虚浮。

### 条件 4：原局无庚辛金透干克木

- **古籍出处**：
  - 《穷通宝鉴·甲木》："**甲木逢庚，斧斤伐木**。"
- **客观判据**：
  - 天干无庚金或辛金透出；地支本气无申、酉成势（一位无根的庚辛尚可）。
- **不满足则破**：金透克木 → 木不能生火 → 转入"斧斤伐木"讨论。

### 条件 5：原局无壬癸水透干灭火

- **古籍出处**：
  - 《穷通宝鉴》："**丙丁火见水旺，光熄无明**。"
- **客观判据**：
  - 天干壬癸水不多透（一位无根可起调候作用；多透或有根则灭火）。
- **不满足则破**：水多灭火 → 通明失 → 转入"水火相战"讨论。

---

**五条同时满足**方为真木火通明。《穷通宝鉴》"**甲木用丁，木火通明**"是本格名称与核心结构的直接来源——木日 + 火透 + 木根 + 无金克 + 无水灭，五者俱备方显"通"与"明"。

## 判断维度

### 1. 甲木配丙火 vs 乙木配丁火

| 组合 | 意象 | 气质 |
|---|---|---|
| **甲 + 丙** | 森林被太阳照耀 | 大气、宽厚、格局外放 |
| **乙 + 丁** | 花草被烛火温煦 | 细腻、秀美、情感丰沛 |

两者都是通明，但**规模与气质不同**。

### 2. 火的位置
- **火透月干**：最有力 → 一出生就光明。
- **火透时干**：后运发力 → 中后期通明兑现。
- **火藏地支**：内明 → 气在内，不显于外。

### 3. 水的角色
- **适量水**：润木，木更有生机 → 水木火三方流通，格更高。
- **水过多**：灭火 → 通明不通。
- **无水全旱**：火炎木焦 → 转为"木火相煎"。

### 4. 土金的位置
- **有土泄火**：火不过烈，气机温和。
- **无土** + 火大 → 燥烈，易暴躁。
- **金过多克木** → 根断，通明瓦解。

## 常见误区

### 1. "木火都有 = 木火通明"
- 必须**通**（能量流通）+ **明**（外显）。
- 木旺火弱 = 不明。
- 火旺木弱 = 不通（木会被火烧光，不是生火，是被焚）。

### 2. 把"通明"等同于"一辈子顺"
- 通明是**五行象意**层的优势，现实结果仍需大运流年、格局用神配合。
- 也可能出现"**看起来光鲜 / 有才华 / 有名气，实则收入 / 家庭生活一般**"的情况。

### 3. 误以为只有甲丙 / 乙丁才算
- 主流是这两组组合，但**局中整体木火气流通、光照充足**也可借用这一象。
- 仍以木日主 + 火为用为典型。

### 4. 忽视"亮过头"的风险
- 过度通明 → 精神外耗、易上火、易泄精气。
- 需有**水润 / 土载**缓冲。

## 现实表现

### 性格
- **聪明、反应快、思路清晰**。
- **有光彩、有感染力**，容易被人记住。
- **乐观、积极、正能量**。
- **口才 / 文采突出**（火代表表达，木火通明自带表达力）。

### 事业路径
- **文化艺术**：作家、设计师、艺术家、策展人。
- **教育传播**：教师、讲师、主持人、媒体人。
- **创作输出**：编剧、内容创作者、自媒体。
- **思想工作**：策略师、咨询师、研究员。
- 共性：**需要"表达 + 灵感 + 感染力"**的岗位。

### 身体
- 火气旺 → 注意眼睛、心脏、血压、睡眠。
- 木气盛 → 肝胆功能与情绪管理。
- 通明过度 + 无水润 → 易焦虑、易失眠、易头痛。

### 感情
- 有**温度感**，容易被人吸引。
- 若火过旺 → 感情炽烈但易冲动 / 易烧空。
- 若有水调候 → 感情细水长流。

## 破格与救格

### 破格
- **重金克木**：木被砍 → 火源断 → 通明灭。
- **重水灭火**：光被扑灭 → 通明转为阴郁。
- **火过旺无制**：转为"**木火相煎**" → 烧毁自身。

### 救格
- **有水润木**：水木火三步流通，格更高。
- **有土泄火**：承接火气 → 温暖而不燥。
- **大运补齐缺失**：如原局火虚，走火运时通明才真正兑现。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 火运 | **喜** → 才华迸发、名气打开 |
| 木运 | **喜** → 生机涌动、能量充足 |
| 水运（适度） | 视结构 → 润木则喜；灭火则忌 |
| 金运 | **忌** → 克木 → 灵感枯竭、健康受影响 |
| 土运（适度） | 视结构 → 泄火可喜；克水则忌 |

## 与相关格局 / 象意的关系

| 象意格 | 关键组合 | 特点 |
|---|---|---|
| **木火通明** | 木 + 火 | 聪明外显、才华明亮 |
| **水木清华** | 水 + 木 | 清秀内敛、学术深沉 |
| **火土夹带** | 火 + 土 | 稳厚实在 |
| **金白水清** | 金 + 水 | 清冷透彻、思辨力强 |
| **木火相煎**（破象） | 火过旺焚木 | 燥烈自焚 |

## 核心判断
- 木火通明的价值不在"木"也不在"火"，而在**通 + 明**两个字。
- **通**：木能顺畅生火，不被金砍断。
- **明**：火能透达外显，不被水扑灭。
- 通明兼备 → 才华 + 光彩 + 影响力俱备。

## 一句话总结
- 内里有生机（木），外在有光芒（火），中间有通道（水润 / 无金克）。
- 人像一棵**在阳光下生长的树** —— 这就是木火通明。
