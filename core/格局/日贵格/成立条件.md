# 日贵格

## 成立条件（古籍出处 + 客观判据）

### 条件 1：日柱为四日贵之一

- **古籍出处**：
  - 《三命通会·论日贵》（万民英）："**日贵者，丁亥、丁酉、癸巳、癸卯四日是也**。贵人乘日干之位，故名日贵。"
  - 《渊海子平·论日贵》同。
- **客观判据**：
  - 日柱干支严格为 丁亥 / 丁酉 / 癸巳 / 癸卯 之一。
- **不满足则破**：非此四日，根本不入本格。

### 条件 2：最好有官星

- **古籍出处**：
  - 《三命通会·论日贵》："**喜官星，官星代表贵气有所依托**。"
- **客观判据**：
  - 原局天干或地支见官星或七杀（官杀皆可作贵气之承载）。
- **效应**：
  - 有官：贵气有依托 → 富贵可期。
  - 无官：仅具"贵人气场"，贵而不显。
- **不严格破格**：无官未必破，但格品下降。

### 条件 3：官星不被冲

- **古籍出处**：
  - 《三命通会·论日贵》："官星不忘需有财来升官，且官星忌冲。"
- **客观判据**：
  - 原局官杀之所在支不被冲。
- **不满足**：官星被冲 → 贵气折损。

### 条件 4：官星不孤 —— 有财来升官

- **古籍出处**：
  - 《三命通会·论日贵》："官星不忘需有财来升官。"
- **客观判据**：
  - 天干有财星通根，能生官 → 官贵可发挥。
- **效应**：财生官 → 官贵显化，贵气真正落地。

### 条件 5：贵人（日支）不被冲破合去

- **古籍出处**（本条最核心）：
  - 《三命通会·论日贵》："**日贵最喜待合，最忌冲破害空亡**。"
- **客观判据**：
  - 日支不被年 / 月 / 时支冲：
    - 丁亥日忌巳；丁酉日忌卯；癸巳日忌亥；癸卯日忌酉。
  - **喜带合**：
    - 丁酉日见辰（辰酉合）→ 贵气牢固。
    - 丁亥日见寅（寅亥合）→ 贵气牢固。
    - 癸卯日见戌（卯戌合）→ 贵气牢固。
    - 癸巳日见申（巳申合）→ 贵气牢固。
  - **或无冲**亦可。
- **不满足则破**：贵人被冲 → 贵气尽失；被合去（贵人与他支合化他物）→ 贵人不为己用。

### 条件 6：贵人 / 财官不在空亡位

- **古籍出处**：
  - 《三命通会·论日贵》："**贵人财官不能在空亡位**。"
- **客观判据**：
  - 日支贵人不落空亡；原局财官所在支不落空亡。
- **不满足则破**：空亡加身 → 贵气不实。

### 条件 7（加重）：不犯魁罡、不遇大运流年冲破合害

- **古籍出处**：
  - 《三命通会·论日贵》："若大运流年犯忌，再遇魁罡，主贫困早夭。"
- **客观判据**：
  - 原局不带魁罡相犯；
  - 大运流年不冲、不合、不害日支贵人。
- **例外**：若柱中构成**更大格局**（如杀印相生、食神制杀等），则让位于大格局论贵。

---

**前六条尽量满足**方为真日贵格。核心判据是**贵人日支不被冲破合去**；最能加分者为"**喜带合**"——贵气牢固。
