# 白虎

## 定义
- **白虎**：四象之一（东青龙、西白虎、南朱雀、北玄武），白虎对应**西方、金、秋、肃杀**。
- 在神煞系统中，"白虎"引申为**血光、伤灾、凶险**之煞。
- 主**突发性外伤、手术、严重意外**——与"灾煞"性质相近，但更偏**血光层面**。

## 起法（按年支查地支）
《三命通会·论白虎》原文："**子年申、丑年酉、寅年戌、卯年亥、辰年子、巳年丑、午年寅、未年卯、申年辰、酉年巳、戌年午、亥年未**。"

即按年支查，白虎在年支前八位（相当于年支的对冲前两位）：

| 年支 | 白虎位置 |
|---|---|
| 子 | 申 |
| 丑 | 酉 |
| 寅 | 戌 |
| 卯 | 亥 |
| 辰 | 子 |
| 巳 | 丑 |
| 午 | 寅 |
| 未 | 卯 |
| 申 | 辰 |
| 酉 | 巳 |
| 戌 | 午 |
| 亥 | 未 |

## 白虎 vs 其他凶煞

| 神煞 | 主方向 |
|---|---|
| **白虎**（本条）| **血光、外伤、手术** |
| 灾煞 | 突发性重大意外 |
| 劫煞 | 外夺 / 被抢 |
| 亡神 | 内部瓦解 |
| 血刃 | 与白虎相近，主血光 |

白虎与血刃都是"血光煞"，但：
- 白虎按**年支**起——广泛性血光风险；
- 血刃按**日干 / 日支**起——个人性血光风险。

## 核心作用：血光与外伤的预警
- 白虎的本质是"**带血的意外**"：
  - 车祸、摔伤、运动伤。
  - 手术、重病住院。
  - 女命：可能涉及**产厄 / 妇科手术**。
  - 男命：可能涉及**斗殴 / 职业伤害**。
- 古籍论："**白虎主血光之灾**。"

## 判断维度

### 1. 位置
- **年支白虎**：早年或祖上有血光事。
- **月支白虎**：青年至中年易遇外伤 / 手术。
- **日支白虎**：本人或配偶有血光风险。
- **时支白虎**：晚年或子女有血光风险。

### 2. 白虎配其他凶煞
- **白虎 + 灾煞**：**血光 + 突发** → 重大事故高发。
- **白虎 + 羊刃**：刚猛 + 血光 → 车祸、斗殴、手术概率高。
- **白虎 + 七杀无制**：易有刀兵之灾。
- **白虎 + 桃花 / 红艳煞**：可能涉及**因感情引发的血光**——冲突、情变、堕胎等。

### 3. 大运流年引动
- 大运 / 流年走到白虎位 → 血光事件概率明显上升。
- 原局已带白虎 + 流年再引动 → **重大血光年**，需加倍注意安全。

## 常见误区

### 1. "带白虎 = 一定出血光事故"
- 白虎是"**血光倾向**"，不是"**必然事件**"。
- 现代生活防护（安全出行、定期体检、远离危险场合）→ 大幅降低白虎之凶。
- 许多带白虎的人一生只做过一次小手术（如拔牙）→ 已算"**白虎应验**"。

### 2. 把白虎视作"最凶煞"
- 白虎主"**血光**"这一层，不主"**生死**"。
- 真正决定生死的是日主强弱、大运、命局整体承载力。
- **身强 + 印星护身 + 大运喜** → 白虎也只是"破点皮"。

### 3. 忽视女命白虎的特殊意义
- 女命白虎传统上**更关注产厄**：
  - 怀孕、生产时的血光风险。
  - 现代：**孕期体检、选择正规医院**即可化解大部分。
- 流年走白虎 + 本命年 + 怀孕 → 需格外重视产检。

## 现实表现

### 性格
- **对危险敏锐**：本能知道哪里有风险。
- **经历过血光者更谨慎**：一次大意外后往往成为"**最小心的人**"。
- **对医院 / 医生 / 身体**有超常关注。

### 典型情境
- 一次严重的外伤经历（骨折、手术、车祸）。
- 因突发疾病需要手术（阑尾、胆囊、骨科等）。
- 女命涉及妇科手术或难产。
- 运动、极限活动中易受伤。

### 职业风险
- **高危职业要额外注意**：运输、施工、高空作业、武职。
- **运动员、极限运动爱好者**：伤病概率比普通人高。
- **外科医生、护士**：职业层面接触血光——以**工作**抵消命理之白虎。

## 化解与应对

### 结构层
- 命局有印星、贵人（尤其天乙、月德）→ 白虎之凶减。
- 大运走喜神 → 白虎被压制。

### 人事层
- **注意安全**：不酒驾、不疲劳驾驶、不参加高危活动。
- **定期体检**：早发现早治疗，避免小病酿大病。
- **女命**：孕期认真产检，不冒险。

### 时间层
- **流年走白虎**：那一年**少远行、少极限运动、不做高风险手术（非必要）**。

## 核心判断
- 白虎是"**血光的气场**"——现代生活能大幅降低应验率。
- 命局承载力 + 生活警觉性 → 白虎可以**只是个符号**，而不是悲剧。

## 一句话总结
- 白虎 = **带血的意外倾向**。
- 活得警觉、活得稳妥，**白虎命的人一样可以一生安全**。
- 真正要害不在"命中有白虎"，在"**有没有防意识 + 命局能不能兜底**"。
