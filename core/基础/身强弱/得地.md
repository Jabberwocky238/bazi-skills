# 得地

## 定义
- **得地**：日主（或任一天干）在八字**地支藏干**中有与自身相同的五行出现，即所谓**通根**。
- 得地的核心不是天干上多几个比劫，而是地支深处有没有属于自己的"根系"。
- 未通根称为**虚浮**或**无根**，犹如无根之木、无源之水，即使天干印比再多也难言身强。

## 核心原理
- 天干为用、地支为体。天干的力量是否稳固，取决于地支是否为其提供了**扎根之所**。
- 通根分三级：**本气根（强根）、中气根（中强根）、余气根（弱根）**，分别对应地支藏干中的本气、中气、余气位。
- 日元本气通根一旦缺失，再多的印比也属于"虚不受补"，甚至引发**水多木漂、金多水泛**等反噬。
- 此即古人所说"干多不如根重"——地下一根胜过天上三比。

## 判断条件

### 三类通根
1. **本气通根**：天干五行与地支本气五行完全相同。力量最强。
   - 甲乙遇寅卯；丙丁遇巳午；戊己遇辰戌丑未；庚辛遇申酉；壬癸遇亥子。
2. **生扶通根**：天干得到藏干生我之气的支持。
   - 甲乙遇亥子（水生木）；丙丁遇寅卯（木生火）；戊己遇巳午（火生土）；庚辛遇辰戌丑未（土生金）；壬癸遇申酉（金生水）。
3. **同季余气通根**：天干与地支属同一季节的土，以余气形式存在。
   - 甲乙遇辰（春湿）；丙丁遇未（夏火）；庚辛遇戌（秋燥）；壬癸遇丑（冬冻）。

### 日主通根对照表
| 日主 | 有根之地支 |
|---|---|
| 甲乙木 | 寅、卯、辰、亥、未 |
| 丙丁火 | 巳、午、未、寅、戌 |
| 戊己土 | 辰、戌、丑、未、巳、午、寅、申 |
| 庚辛金 | 申、酉、戌、巳、丑 |
| 壬癸水 | 亥、子、丑、申、辰 |

### 根的强弱六级排序
| 等级 | 组合特征 | 力量 |
|---|---|---|
| 第一强 | 本气通根 + 正根 + 天干藏干**同阴阳**（如庚坐申）| 最强 |
| 第二强 | 本气通根 + 正根 + 阴阳不同（如丙坐午、癸坐亥）| 次强 |
| 第三强 | 本气通根 + **旁根** + 同阴阳（如辛坐酉但非日支）| 中上 |
| 第四强 | 本气通根 + 旁根 + 阴阳不同 | 中 |
| 第五强 | 中气通根（如癸坐丑——丑中藏癸为中气）| 中强根 |
| 第六强 | 余气通根 | 弱根 |

### 正根与旁根
- **正根**：天干坐下地支即为其通根地支。力量大，命主随身自带。
- **旁根**：天干坐下不通根，而在其他柱的地支中通根。力量小，隔柱作用有距离衰减。

### 判断先后顺序
- 判断天干强弱时，**第一位看是否通根**，第二位才看天干上是否有生扶比劫。
- 此顺序不可颠倒——无根者，天干再多印比也虚浮。

## 活木与死木（木日干专论）
- **活木**：**同时有根又有水**——仍属植物，根系能吸水保持生命。
- **死木**：**无根或无水**——至少缺一条件，已转化为木材。

### 特性对比
| 情境 | 活木 | 死木 |
|---|---|---|
| 遇火 | 开花泄秀（生食伤秀气） | 只会燃烧（泄身太过）|
| 遇金 | 害怕金铁伤根 | 不怕金（反需金雕琢成材）|
| 遇水 | 喜湿润环境 | 不喜水（木材遇水易腐）|
| 空亡 | 生长不良 | 难以成为有用之材 |
| 根的阴阳互用 | 可以阴阳互用（甲可用卯根、乙可用寅根）| 不可阴阳互用 |

## 与得令、得势的关系
- **互相独立**：得令只审月支一柱；得地审**全盘地支藏干**，日支尤重；得势审全局五行势力对比。
- **互相补充**：失令但得地（尤其本气正根），日元就不会太弱，仍可接受印星生扶。
- **加权权重**：得地中，**日支为正根最贵**（夫妻一体、紧贴日元），其余为旁根，距离越远力量越小。
- 无根之干遇强印只会虚浮漂泛；**得地是印星得以生身的前提条件**。

## 误区
- **以天干比劫数量论强弱**：四柱天干全是比劫，地支无根，仍属弱命，不可言身强。
- **印星多即可代替根**：印再重，无根者难以化泄吸收，可能造成反噬。
- **把中气、余气根当本气根**：丑中藏癸为中气（中强），辰中藏乙为余气（弱根），力量不可等同。
- **忽视正旁根区别**：同是寅中甲木，日支为正根、年支为旁根，力量差数倍。
- **活木死木不分**：只看天干甲乙就以为一定"喜水"，忽略有无根决定的木性差异。
- **把空亡之根当有效根**：坐下地支若空亡，虽藏干存在，实际支撑力大打折扣。

## 现实应用
- **看人底气**：命主是否根深蒂固，决定他面对印星（贵人、资源）时是"受补"还是"漂泛"。
- **取用神**：身弱无根者，**首取补根之运**（即地支比劫或印库运），而非在天干上追求印比。
- **断财富根基**：财星通根者财富根基稳固；财星无根者，即使发财也难留。
- **断学业婚姻**：日支为正根者，内心稳定、配偶得力；日支被冲去根者，须防根基动摇。
- **大运流年**：岁运补根之年常为命主能量回升、事业转机之时；岁运冲根之年须防根基动摇。
- **死木取用差异**：死木命如甲木无水，遇火反能成材（食伤泄秀有价值），与活木之命走法不同。

## 古籍背书
- 《子平真诠·论用神变化》：**"干得一根，胜过三透。"** 地支一根之力，胜过天干三位透比——直接确立通根的第一优先级。
- 《三命通会·论天干坐支吉凶》详论天干坐下通根与否对命造强弱的决定性影响。
- 《滴天髓·天干》：**"得根者干旺，失根者干虚。"** 并以"五阳从气不从势"辨析干支配合。
- 《渊海子平》论藏干分野：**"本气为主，中气次之，余气为辅。"** 即三级根强弱次序的源头。

## 一句话总结
- **得地看地支，地支有根天干实；本气正根最为贵，无根虚浮再多比劫也枉然。**
