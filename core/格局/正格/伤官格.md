# 伤官格

## 定义
- **伤官**：日主所生之异性五行——阳日主生阴为伤官；阴日主生阳为伤官。
  - 例：甲日见丁（阴火）为伤官；乙日见丙（阳火）为伤官。
- **伤官格**：**月令取伤官为用神**的基础正格。
- 《子平真诠》十正格之一——**锋锐、破局、变革**之气。
- 古籍评价复杂："**伤官本非吉神，然有用之处**"——用之得法为贵，用之失法为祸。

## 核心：锋锐之用
- 伤官 = 我生之异性，代表**锋芒、创造、变革、独立、不受约束**。
- 相对食神（温厚）：伤官**锐利得多**——能破旧立新，也能惹祸上身。
- 伤官格的三个用法（见古籍核心）：
  - **伤官佩印**（身弱用印制伤）
  - **伤官生财**（身强用财泄伤）
  - **伤官合杀**（伤杀五合化解）

## 伤官格的基本结构
本文专论**伤官格的基础定义**——具体用法详见对应变格文件。

## 成立条件（严格依《子平真诠·论伤官》）

### 条件 1：月令地支本气为伤官，或伤官透干且月令有根

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官者，我生之异性**。月令伤官为格。"
- **客观判据**：
  - ① 月令地支本气为伤官；或
  - ② 月令地支藏伤官且伤官透于天干。
- **不满足则破**：月令伤官不显 → 当以月令本气另论他格。

### 条件 2：伤官透干且通根

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官须透干有根，方成其用**。"
- **客观判据**：
  - 伤官透于天干，且在地支有本气或中气根至少一位。
- **不满足则破**：伤官虚透无根 → 锋芒虚浮，格难立。

### 条件 3：原局无正官（伤官见官为祸）

- **古籍出处**（本条铁律）：
  - 《子平真诠·论伤官》："**伤官见官，为祸百端**。"
  - 《子平真诠·论伤官》："**伤官伤尽，方为贵；伤官不尽，祸端百出**。"
- **客观判据**：
  - 原局天干无正官透出；若地支藏正官，须力微（一位余气）。
- **不满足则破**：伤官贴官 → 破格转入"**伤官见官**"病格讨论；除非有印制伤（转"伤官佩印"）或财通关（转"伤官生财生官"）。

### 条件 4：日主与伤官配比（身不过弱亦不过强）

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官用印，身弱有力；伤官用财，身旺为宜**。"
- **客观判据**：
  - ① 身弱用印 → 转入"**伤官佩印**"（身弱而印有力）；
  - ② 身旺用财 → 转入"**伤官生财**"（身旺而财有根）；
  - ③ 身中偏强无财无印 → 纯伤官格（格品略薄）。
- **不满足则破**：身极弱 + 伤重且无印 → 身弱伤重之耗；身极强 + 伤重且无财 → 气无出路。

### 条件 5：伤官清而不杂（不与食神混杂）

- **古籍出处**：
  - 《子平真诠·论伤官》："**伤官清纯为贵，混杂则失**。"
- **客观判据**：
  - 原局伤官透干有根为主，食神不透或仅藏支一位不显。
- **不满足则破**：食伤混杂 → 转入"食伤混杂"讨论，格品降。

---

**前四条必须同时满足**方为真伤官格（条件 4 决定取用方向）；条件 5 关乎清纯度。

## 伤官格的三大用法（均独立成格）

| 用法 | 适用 | 核心古籍 |
|---|---|---|
| **伤官佩印** | 身弱 + 印制伤 | 《子平真诠》"伤官用印，身弱有力" |
| **伤官生财** | 身旺 + 财泄伤 | 《子平真诠》"伤官用财，身旺为宜" |
| **伤官合杀** | 阴日主 + 五合化杀 | 《子平真诠》"合杀者，取其合以去杀" |

详见各自专章。

## 判断维度

### 1. 身的状态
- **身弱**：用印（佩印）。
- **身旺**：用财（生财）。
- **身中**：看原局结构取用。

### 2. 伤官的类型
- **真伤官**：月令为伤 → 典型伤官格。
- **假伤官**：伤官虚透而月令非伤 → 次级。

### 3. 大运流年
- **印运**（身弱伤官者）：喜——佩印兑现。
- **财运**（身旺伤官者）：喜——生财兑现。
- **官杀运**：忌（伤官见官），除非有救应（印制、财通、合化）。
- **比劫运**：视原局身强身弱。

## 常见误区

### 1. "伤官 = 凶"
- 伤官本身无吉凶——**用之得法为贵**。
- 古籍明言："伤官本非吉神，**然有用之处**。"
- 佩印、生财、合杀三路，皆能使伤官为贵。

### 2. "带伤官 = 必惹事"
- 只有"**伤官见官 + 无救应**"才真正惹事。
- 命中有伤官但无正官 → 可能一辈子平静。
- 现代语境：伤官主创造力与变革力，**用对地方是顶级才华**。

### 3. 忽视伤官的刚性
- 伤官比食神**更强势**：
  - 不喜被管、不爱循规。
  - 对权威天然反抗。
  - 识别后应顺其气——**放到自由度高的岗位**。

## 现实表现

### 性格
- **聪明、锋利、有独立见解**。
- **表达欲强、不擅遮掩**。
- **对权威本能抗拒**。
- **创造力强、破旧立新**。
- **情绪波动明显**：开心则极开心，不爽则极不爽。

### 职业路径
- **创造型专业**：作家、艺术家、设计师、发明家。
- **媒体、自媒体、内容创作**：靠锋芒与独特立身。
- **销售、市场、谈判**：伤官的表达力转为业绩。
- **破局型岗位**：创业者、变革者、产品经理。
- **技艺型专家**：外科医生、工匠、技术专家。
- **娱乐、表演**：伤官的外放变为舞台感。

### 人生节奏
- **早期即显锋芒**：学生时代常是"**聪明但有个性**"的那个。
- **中期事业爆发或受挫**：取决于有无救应（印 / 财 / 合）。
- **晚年若有作品留世**：伤官之气完整兑现。

## 核心判断
- 伤官格的核心在"**用法**"而非"**本身**"——三种救应中必有其一方能为贵。
- 最大破格：伤官见官 + 无救应。
- 识别意义：这类命造要**找到用伤官的出路**（佩印 / 生财 / 合杀），否则锋芒会反噬。

## 一句话总结
- 伤官 = **才华与锋芒的集合**。
- 会用的人 → 作品惊人、成就斐然。
- 不会用的人 → 口舌是非、职场冲突、身心俱疲。
- 关键不在"命中有没有伤官"，在"**伤官有没有出路**"。
