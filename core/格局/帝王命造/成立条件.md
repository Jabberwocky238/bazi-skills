# 帝王命造

## 成立条件（古籍 + 客观判据）

### 条件 1：格局必须清纯，不可混杂

- **古籍出处**：
  - 《滴天髓·通神论》："**清纯者贵，混杂者贱**。"
  - 《子平真诠·论用神成败救应》："格局贵清不贵浊。"
  - 《三命通会》：帝王命造中多言"**格清气旺**"为至贵之标。
- **客观判据**：
  - 原局格局单一明确——或走正格（正官 / 七杀 / 财 / 印）、或走专旺 / 从强 / 从势、或走化气。
  - 无官杀混杂、财印混杂、食伤混杂、真假神混杂等情形；若有混杂，必有"去混留清"的合化或冲去机制。
- **不满足则破**：格局混杂 → 气散不凝 → 虽有贵气也难登极顶。

### 条件 2：五行流通或气势纯粹二者居其一

- **古籍出处**：
  - 《滴天髓·通神论》："**五气之流通，始生之妙；一气之独盛，专旺之奇**。"
  - 《子平真诠·论用神》："**流通不悖，成大贵之基**。"
- **客观判据**（两条路径任择其一）：
  - **路径甲：五行流通型**——财生官、官生印、印生身、身生食伤、食伤生财，首尾相连成环；或成财官印全之结构。
  - **路径乙：一气独盛型**——专旺、从强、化气、天元一气等偏枯到极致的气势。
- **不满足则破**：五行零散、既不流通也不纯粹 → 气势无骨架 → 无帝王之相。

### 条件 3：日主或格局核心"立得住"

- **古籍出处**：
  - 《滴天髓·体用》："**用神有力、日主能承，方为上命**。"
  - 《子平真诠·论用神成败救应》："**真用真忌，不摇不动**。"
- **客观判据**（根据格局类型分论）：
  - 正格：日主于月令得比劫 / 印星之根，或他柱有多处通根。
  - 从格 / 化气：顺势之方无任何一字反叛，且化神 / 从神纯净无破。
  - 专旺：独旺之行独大、无克无泄或有泄而不伤本。
  - 身杀两停：身杀势均而有辅助调节。
- **不满足则破**：日主或格局核心被大幅削弱、被冲克 → 承不住位 → 帝位无缘。

### 条件 4：大运顺行且与命局深度配合

- **古籍出处**（本条为"凡帝之基"）：
  - 《三命通会》："**命好不如运好**；命运相济，方登极贵。"
  - 《滴天髓·岁运》："**大运顺则发福，大运逆则虽贵不显**。"
- **客观判据**：
  - 大运连续 20–40 年行于命局之**喜用方向**，无中断。
  - 关键大运（青年立业、中年登顶）与命中用神之方一致。
  - 不见大运严重冲破命局之核心根（如月令之冲）。
- **不满足则破**：命虽好但运背逆 → 一生踟蹰、难成大业 → "**命运不齐，富贵无缘**"。

### 条件 5：原局无根本性的破格之伤

- **古籍出处**：
  - 《子平真诠·论用神成败救应》："成中有破，败中有成。"
- **客观判据**：
  - 无"真神被合去"、"真用被冲破"、"忌神独大压制用神"等致命破格。
  - 若有小破格，须原局有"救应"（合 / 冲 / 生 / 化）之机制。
- **不满足则破**：根本破格无救 → 格局塌陷 → 无望大位。

---

**五条同时满足**方为真帝王命造之结构。五条中**第 1（清）、第 2（流 / 纯）、第 4（运）**最为核心——古籍屡言"格清、气旺、运顺"三者齐备方为帝王之基。
