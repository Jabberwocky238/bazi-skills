# 从财格成立条件

### 条件 1：日主极弱，无根无柱
- 天干周围无比劫或印星透出。
- 地支四位本气、中气、余气中**皆无比劫 / 印星**。

### 条件 2：月令必须是财星
- 要么月令本气即为财星（正财或偏财）；
- 要么地支月令**参与三合或三会财局**。

### 条件 3：财星数量上的优势
- 天干地支中**财星数量 ≥ 食伤**；
- 且**财星数量 > 官杀**。

### 条件 4：整个命局形成财星的气势
- 最好有**食伤生财**或**财生官杀**的顺流链条。
- 但**财星必须是老大**——食伤、官杀都服务于财。

### 条件 5：不见印比
- **无印星**（无论天干地支）。
- **无比劫克破**财星。
- 印比任一见根 → 立即破格，归入身弱财多讨论。
