# 截脚

## 定义
- **截脚**：同柱之内**地支克天干**的组合。
- 地支如脚下之根基，反过来向上砍天干——天干站不住、立不稳，气无法下达、也无法长久。
- 古籍又称"支克干"或"下克上"，是衡量某一柱天干是否真能用事的关键条目。

## 核心原理：地支克天干的方向

地支克天干，按五行相克方向可归为五条主线：

- **金克木**：地支申、酉 压 天干甲、乙。
- **木克土**：地支寅、卯 压 天干戊、己。
- **土克水**：地支辰、戌、丑、未 压 天干壬、癸。
- **水克火**：地支亥、子 压 天干丙、丁。
- **火克金**：地支巳、午 压 天干庚、辛。

截脚的根本特征：**天干被自己的脚踢翻**——从外界看天干还在透出，实则根已断。

## 截脚的轻重判断

### 一、看地支本气是否直对天干
- **纯气直克**：如乙酉、丙子、己卯——地支只有一种藏干，且恰好克天干 → 克力最重，天干几乎无根。
- **本气直克但藏他干**：如甲申、庚午、癸未——虽有其他藏干分摊，本气依然直砍天干。
- **余气或中气偶有救应**：如戊寅中戊土自坐余气、辛巳中藏庚可帮身——截脚虽在，但同柱地支藏干**暗中有救**，截力减。

### 二、看地支是否得旺
- 地支参与三合、三会、方局 → 克力倍增。
- 地支被冲、被刑、被合化走 → 克力虚设，天干反而脱险。

### 三、看天干是否有他处通根
- 天干在他柱另有根（禄、刃、库、余气根）→ 截脚不能彻底断根，只是本柱根缺。
- 天干完全无根 → 截脚即真截，天干成浮气。

### 四、看全局是否救应
- 通关五行出现 → 化克为生。
- 比劫帮身 → 承受截力。
- 印星透出 → 从更高层补气。

## 截脚的应事

截脚落在哪一柱，该柱天干代表的外显气就被根基截断。

### 年柱截脚
- 祖业、远景层面外显的气被家底反噬。
- 家族表面光鲜但底子薄，或早年受家族底层问题牵连。

### 月柱截脚
- 月干常代表**父母、兄弟、事业平台**——被月支克 → **父母操劳、兄弟难靠、事业平台不稳**。
- 若月干为用神被月支截脚 → 格局发力点被釜底抽薪。

### 日柱截脚（日主截脚）
- **自身气受限**——日干是本人外显人格，被日支（夫妻宫）反克 → 本人在"自己的家里"反被压。
- 常见象意：对内在伴侣关系中容易委屈、在自己擅长的领域反遭掣肘。
- 若日主截脚且无他处通根 → 身弱难承事。

### 时柱截脚
- 时干常主子女之外显、晚年事业、事情结果层的气。
- 被时支反克 → 子女不听话、晚景奔波、做事到结果层反被自己埋下的因反噬。

### 用神截脚
- 喜用之天干被地支反克 → 命局最需要的那股气悬空无根 → 求而不得、有能力无平台、有资源无落地。

## 现实象意

- **一个字：累**——气出不去、根又断。
- **想扎根却被挖空**：刚想在某处立足，脚下的土被抽掉。
- **临门一脚卡壳**：谈好的合作黄、到手的机会被截、升职换人、婚姻临门变故。
- **恩情难续**：出过力换不来真实回报，甚至翻脸（带"无恩"味）。
- **上浮下空**：外看还在发力，实际内里已虚——像一根**没根的旗杆**，风一大就倒。

## 化解思路

- **取通关五行**：如庚午、辛巳（火克金截脚）取土通关，火生土生金。
- **他柱通根**：关键是让被截的天干在别处找到根，不单靠本柱。
- **用印生扶**：印星生天干，从上游补气。
- **合去截脚地支**：地支被合化转向 → 截脚消失。
- **冲动截脚地支**：冲开地支让克力涣散（需全局配合，慎用）。
- **人事应对**：不把赌注全押在同一根柱上，分散落脚点；不幻想一次成功，接受反复波折是常态。

## 常见误区

### 1. 认为截脚必然导致天干废掉
- 错。天干若在他柱有根，截脚只是本柱根缺，天干仍能用事。

### 2. 把截脚等同于盖头
- 盖头是天干克地支（上克下，影响地支）。
- 截脚是地支克天干（下克上，影响天干）。
- 两者方向、应事、轻重机制均不同。

### 3. 不看地支成局
- 地支若被三合三会合化走，截脚可能名存实亡，甚至反转为生（如亥子丑三会水，癸水坐丑土，丑土不仅不克癸，反而加大水力）。

### 4. 只看本柱不看全局
- 全局通关、合化、刑冲都可能化掉截脚。
- 判断须结合**旺衰、寒暖燥湿、阴阳平衡**综合看。

### 5. 忽略天干自身的旺相休囚
- 同是截脚：得令旺相之干与失令休囚之干，受克程度差异极大。

## 与盖头 / 覆载的对比

| 关系 | 天干 vs 地支 | 能量效果 | 象意 |
|---|---|---|---|
| **盖头** | 天干克地支 | 地支被压制 | 外显压内在 |
| **截脚** | 地支克天干 | 天干无根基 | 根基反克上层 |
| **覆载** | 同气或相生 | 天地一体 | 能量最稳 |

- 盖头是**从上方压**，压的是根；截脚是**从下方砍**，砍的是气。
- 盖头让地支**不能提供支撑**；截脚让天干**悬空飘摇**。
- 一般实战中，**截脚比盖头更伤柱力**——因为立足感是一切发力的前提。

## 古籍出处

- 古典命理以"支克干"论截脚，视为**天干有形无气、根基反噬**的关键征象。
- 看柱如看人：天干为身、地支为脚；截脚即**脚反踢身**，身未倒先失重。
- 判断一柱是否真能用事，必先看其有无截脚；截脚未解，透干只是虚透。

## 一句话总结

截脚 = 地支克同柱天干，外显的气被自身根基反噬；天干看似在透，实则根断；吉凶看被截的是喜神还是忌神，轻重看地支克力与天干他处是否有根，所有判断须结合全局合化、刑冲、通关做最终修正。
