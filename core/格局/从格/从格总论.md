# 从格总论

## 一、什么是"从"

- **从**：顺从、投降、放弃对抗。
- **从格**：日主**极弱**，命局中一股（或多股）五行之势**极旺**，日主无力自立 → **放弃日主本位，顺从旺势而行**。
- 古人称之为"**气命从财、气命从杀、气命从儿**"——把自己的气命交给旺神。
- 从格专指**从弱**（日主极弱）；与"专旺格"（日主极强、气势与日主一致）相对。

### 核心比喻
- 一个兵少粮绝的小国，紧挨着兵强马壮的超级大国。
  - **硬刚** → 被一巴掌拍死。
  - **举白旗投降** → 反而能存活，甚至当附属国过得不错。
- 从格就是**日主向旺神投降**，以求生存并借势。

### 反面教训：硬要用印比的后果
- 从格日主若强行找印星（妈妈）或比劫（兄弟）来帮身：
  - 帮手会被旺神**直接灭掉**。
  - 灭完帮手后，旺神的余怒再发泄到日主身上。
- 命理上叫"**先打倒用神，再将余怒发泄到日主身上**"。
- 现实表现：走印运母亲有灾；走比劫运破财且兄弟姐妹疏远。
- 所以**唯一的活路是顺从**。

## 二、从格的两大类

### 1. 从弱（狭义"从格"，日主极弱顺克我 / 泄我之方）

| 子格 | 顺从对象 | 气质 |
|---|---|---|
| **从财格** | 财星 | 多情重义、异性缘佳、企划能力强 |
| **从杀格** | 七杀 | 果决威严、领导才能、不怒自威 |
| **从官格** | 正官 | 端正温顺、循规蹈矩、入体制 |
| **从儿格** | 食伤 | 聪明秀气、创造力强、才华横溢 |
| **从势格** | 财、官、食伤混杂旺势 | 恭敬有礼、处事圆满、心地善良 |

### 2. 从强（广义"反从"，日主极旺顺印比之方）

| 子格 | 特征 |
|---|---|
| **从强格** | 印星为主、比劫为辅（或印星略强），无克泄耗之一党 |
| **专旺格** | 比劫为主、印星为辅，日主五行与命局气势一致（曲直、炎上、稼穑、从革、润下） |

另有**两气成象格**：命局只见两种五行且相生有情，势均力敌——介于专旺与从格之间的特殊格局。

## 三、真从 vs 假从的核心分界

| 维度 | 真从格 | 假从格 |
|---|---|---|
| 地支印比 | **完全无**本气、中气、余气根 | 绝对无根，但天干可有一位印或比 |
| 天干印比 | 无 | 可有一位**虚浮无根**的印或比，被旺势压制 |
| 气势纯度 | 一党独旺，纯而不杂 | 一党旺但略杂 |
| 取用方式 | **与真从一致**，仍顺旺势 | 与真从一致，喜忌相同 |
| 运势起伏 | 顺运极贵；逆运暴跌 | 更易被逆运波及，动荡更剧 |

### 一句话区分
- **真从**：日主"孑然一身"，想靠都没得靠，只能顺势。
- **假从**：日主有"一个虚浮的拐杖"，拐杖自己都站不稳，拿旺势毫无办法——**取用同真从**。
- **只要地支出现印比本气根 → 不论从**，直接归入身弱破局讨论。

## 四、从格取用总原则

### 喜用神
- **顺从对象本身**——从财喜财、从杀喜官杀、从儿喜食伤、从势喜一党中最旺者。
- **能生顺从对象的五行**——从财喜食伤生财；从杀喜财生官杀；从儿喜比劫生食伤。
- **顺流下去的五行**——从财之后喜官杀泄财；从儿之后喜财泄食伤。

### 忌神（从弱各格通用）
- **印星**：生扶日主 → 日主想站起来 → 激怒旺神 → 招灾。
- **比劫**：帮身抗旺 → 破从 → 旺神反击更烈。

### 口诀
- 从格之人如**一头顺毛驴**——顺着毛摸则步步顺，逆着毛摸立马撂蹶子。
- **顺之者吉，逆之者凶**。

## 五、破格：见印比的一丝救应即破从

### 铁律（古籍）
- "**若有一毫印绶，即不为从**。"
- "**比劫一见，即不为从**。"

### 具体判据
| 项目 | 判据 | 结果 |
|---|---|---|
| 地支本气为印或比 | 有 | 不从（必破） |
| 地支中气为印或比 | 有 | 基本不从 |
| 地支余气为印或比 | 有 | 严格论不从；宽论可勉强论假从 |
| 天干一位印或比，地支无根 | 有，被旺神压制 | **假从**（虚浮不论根） |
| 天干一位印或比，地支有根 | 有 | 不从（已成身弱格局） |

### 破格后的后果
- 若误判为从格而顺旺神用神 → 等于把该补身的人推向绝境 → **越走越衰**。
- 故识别从格的第一原则：**宁可错判为普通身弱破格而补身，也不要误判为从而顺旺神**。

## 六、古籍背书

- 《滴天髓·从象》："**从得真者只论从，从神又有吉和凶。**"
- 《滴天髓·从象》（任铁樵注）："**日主孤立无气，天地人元绝无一毫生扶之意。**"
- 《滴天髓》："**若有一毫印绶，即不为从。**"
- 《滴天髓·从儿》："**从儿不管身强弱，只要吾儿又遇儿。**"
- 《滴天髓·从旺》："**从旺者，四柱皆比劫，无官杀制，有印生之。**"
- 《滴天髓·从强》："**从强者，四柱印绶重重，比劫叠叠，日主又当令。**"
- 《子平真诠·论杂格》："日主无根，不得不从其旺神。"

## 八、一句话总结

- 从格的本质：**弱到极致反而是另一种强** —— 把自己彻底交给旺势，借旺势之力安身。
- 识别从格第一件事：**看地支有没有印比的一丝根**。有一丝 → 不论从；彻底没有 → 才可能真从。
- 用法：**顺之者吉，逆之者凶**。
