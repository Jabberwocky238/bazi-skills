# 伤官佩印

## 成立条件（严格依《子平真诠》《滴天髓》《渊海子平》）

### 条件 1：伤官透干且通根，或月令本气为伤官

- **古籍出处**：
  - 《子平真诠·论伤官》："伤官用印，必伤官有力而印有根。"
- **客观判据**：
  - ① 伤官透于月 / 日支 / 时干，且地支有本气或中气通根至少一位；或
  - ② 月令地支本气为伤官。
- **不满足则破**：伤官虚浮无根 → 无"伤"可"佩"，不立本格。

### 条件 2：印星透干且通根，其力量不弱于伤官

- **古籍出处**（本条核心）：
  - 《子平真诠·论伤官》："**伤官佩印者，身弱有印制伤官而用印**。"
  - 《滴天髓·伤官》："伤官见官为祸百端，**佩印则化**。"
- **客观判据**：
  - ① 印星（正印为佳，偏印次之）透于天干，且地支有本气根至少一位；
  - ② 印的力量在命局中不弱于伤官（印 ≥ 伤）——可通过通根数量与透干位置综合判断。
- **不满足则破**：
  - 印虚 → 制伤无力 → 伤官仍泄身且可能克官 → 格不成；
  - 印远强于伤官（印过重）→ 转入"枭神夺食"讨论（另一破格）。

### 条件 3：日主身弱（此格专为身弱用印）

- **古籍出处**（本条核心）：
  - 《子平真诠·论伤官》："**伤官用印，身弱有力；伤官用财，身旺为宜**。"
- **客观判据**：
  - ① 日主于月令非比劫 / 印星本气；
  - ② 全局比劫 + 印星总位数少于食伤 + 财 + 官杀之总位数（身偏弱至身弱）。
- **不满足则破**：身旺而强用印 → 沈氏明言此为"伤官用财"（伤官生财）的路径——强用印反闷塞生机，降格。

### 条件 4：原局无财星紧贴克印

- **古籍出处**：
  - 《渊海子平·伤官诗诀》："**伤官用印宜去财，用财宜去印**。"
- **客观判据**：
  - ① 天干无财星与印星紧贴相克（月 / 日、日 / 时邻位）；
  - ② 若有财，须处于印位之对侧（财远离印），或被合去 / 被比劫挡住。
- **不满足则破**：财破印 → 印失守 → 伤官脱缰 → 回到"伤官无制"态；遇官即破格为"伤官见官"。

### 条件 5：原局无正官透干（即便有印化仍尽量忌见）

- **古籍出处**（本条最严）：
  - 《子平真诠·论伤官》："**伤官佩印，须官星一点不见方妙**。"
- **客观判据**：
  - ① 原局天干无正官透出；
  - ② 若仅地支藏正官于余气一位，气微不显，尚可论佩印（但格品降一等）。
- **不满足则破**：正官透干 → 即便有印化，沈氏严格取法"一点不见方妙"——格品必降，按"伤官佩印 + 带官"复合结构讨论。

---

**五条同时满足**方为真伤官佩印。本格与"伤官生财"之别（沈氏原文二分）：
- 身弱用印 → 伤官佩印（本格）；
- 身旺用财 → 伤官生财。
用印与用财**不可兼取**（《渊海子平》："伤官用印宜去财，用财宜去印"）。
