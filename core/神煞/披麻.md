# 披麻

## 定义
- **披麻**：披麻戴孝，古代丧礼中孝服的象征。
- **披麻煞**：主**孝服、丧葬、家中长辈去世**的凶煞。
- 与**丧门、吊客**合称"**丧事三煞**"：
  - **丧门**：家中丧事（自家）。
  - **吊客**：吊唁他人（他家）。
  - **披麻**：**穿孝服、亲自参与丧礼**——最贴身的哀事。

## 起法（按年支查地支）
《三命通会·论披麻》原文："**披麻者，岁后三辰也**。"

即按年支**逆推三位**：

| 年支 | 披麻位置 |
|---|---|
| 子 | 酉 |
| 丑 | 戌 |
| 寅 | 亥 |
| 卯 | 子 |
| 辰 | 丑 |
| 巳 | 寅 |
| 午 | 卯 |
| 未 | 辰 |
| 申 | 巳 |
| 酉 | 午 |
| 戌 | 未 |
| 亥 | 申 |

**背后原理**：披麻在年支后三辰——"**岁神之后的哀事跟随位**"。古人以十二神煞围绕太岁排列，披麻位居岁后第三，代表"**紧跟岁神的孝服之象**"。

## 披麻 vs 丧门 vs 吊客

| 神煞 | 位置 | 含义 |
|---|---|---|
| **丧门** | 岁前二辰 | 迎面而来的家中丧事 |
| **吊客** | 岁后二辰 | 紧随而来的吊唁事 |
| **披麻**（本条）| 岁后三辰 | **亲自穿孝、披麻戴孝** |

三煞同现 → **丧事全流程**（家丧 + 吊唁 + 披麻戴孝）都会碰到。

## 核心作用：披孝之事的预警
- 披麻不是"**必然有人死**"，但**一生中与丧礼发生的接触明显多于平均**：
  - **亲族长辈去世**，需亲自服孝。
  - **参加的丧礼场次多**。
  - **从事与丧礼相关的工作**（殡仪、礼仪师）。
- 古籍论："**披麻入命，主孝服之事**。"

## 判断维度

### 1. 位置
- **年支披麻**：祖辈去世有孝服之事（早年）。
- **月支披麻**：青年至中年需为长辈服孝。
- **日支披麻**：本人或配偶家有孝服事。
- **时支披麻**：晚年或子女服孝。

### 2. 披麻配其他神煞
- **披麻 + 丧门**：家中丧事 + 披麻戴孝，事件完整。
- **披麻 + 吊客**：丧事密集年，多次参与。
- **披麻 + 丧门 + 吊客（三煞齐）**：一生多次经历丧葬。
- **披麻 + 华盖**：与宗教超度、哀伤抚慰相关工作。

### 3. 大运流年引动
- 大运 / 流年走披麻位 → 亲人丧事的概率增加，或自己参与丧礼的机会多。
- 原局已带披麻 + 流年再引 → **某年家中 / 亲族多人故去**。

## 常见误区

### 1. "带披麻 = 家中一定多死人"
- 披麻是"**披麻戴孝的倾向**"——不等于"**大量死亡**"。
- 现代对应：
  - 参加过较多的丧礼（比普通人多）。
  - 长辈去世时需自己承担丧事主持。
  - 从事殡葬 / 礼仪相关工作。

### 2. 把披麻视作"最凶煞"
- 披麻 ≠ 血光凶煞；披麻主**哀事情境**，非**本人生死**。
- 真正的凶煞是七杀无制、羊刃冲、灾煞等。
- 披麻只是"**哀事接触较多**"的标记。

### 3. 忽视"职业化解"
- 从事殡仪、礼仪、宗教、临终关怀、遗体化妆等职业 → **以职业方式兑现披麻之气** → 自己家反而无事。
- 这是命理"**职业化解神煞**"的典型应用。

## 现实表现

### 典型情境
- 早年某长辈去世，从小穿过孝服。
- 一生参加的丧礼比同龄人多。
- 家族中常被推为"**主持丧事**"的角色。
- 对生死议题的态度比常人平和。

### 职业路径
- **殡葬业**：殡仪馆、陵园、丧礼策划。
- **宗教事务**：超度、法事、丧礼主持。
- **临终关怀**：医院临终护理、临终陪伴。
- **古文化研究**：研究祭祀、丧葬礼仪。

### 性格
- **对死亡话题不回避**：反而有深入思考。
- **对生命的珍惜感强**——"**经历过丧事的人更懂活着**"。
- **同理心强**：能陪伴他人度过哀伤。

## 化解与应对

### 结构层
- 命局有印星、贵人 → 披麻之凶减。
- 从事殡仪 / 宗教 / 临终关怀等职业 → 消化披麻之气。

### 人事层
- **保持与长辈的密切联系**：披麻引动时，陪伴尤其重要。
- **做好家族丧葬的预案**：不是迷信，是现实准备。
- **接受披麻命造的"哀事接触多"**：把参与丧礼视为**对生命的致敬**，而非恐惧。

## 核心判断
- 披麻是"**披孝之缘**"——一生与丧礼的接触比平均多。
- 命局整体承载力 + 职业选择合理 → 披麻转化为**同理心与对生命的理解**。

## 一句话总结
- 披麻 = **穿过孝服的人**。
- 不是不祥，是**早懂生死的人**。
- 能把这份早懂转化为对活着的珍惜 → 反而是命带的一份礼物。
