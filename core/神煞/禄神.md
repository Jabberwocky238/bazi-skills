# 禄神

## 定义
- **禄**：十干的**临官之地**——天干在十二长生中"临官"位对应的地支。
- **禄神**：命局地支中出现日干之禄位 → 称"**带禄**"。
- 本质：**日干在地支的本气之家**——根基所在，身之所立。

## 起法（十干禄位对照）
《渊海子平·论禄》原文："**甲禄寅，乙禄卯，丙戊禄巳，丁己禄午，庚禄申，辛禄酉，壬禄亥，癸禄子**。"

| 日干 | 禄位 |
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙、戊 | 巳 |
| 丁、己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

**背后原理**：禄 = 比肩的本气之地。
- 甲木禄寅：寅中藏甲木本气；
- 庚金禄申：申中藏庚金本气；
- 禄是"**自己的地盘**"——天然站得稳。

## 禄的四种位置及含义

### 1. 月支禄 = 建禄格
- 月令是禄位 → **身自带旺气**，格局直接成"建禄"。
- 详见 `core/格局/建禄格.md`。

### 2. 日支禄 = 日禄归时 / 坐禄
- 日主坐禄 → 自坐根气，性格独立自主。
- 典型："**贵气由己出**"——不靠别人。

### 3. 时支禄 = 时禄（归禄）
- 晚年稳定、子女或后代得力。
- 古籍论"**时禄喜归家**"——晚运之福。

### 4. 年支禄 = 祖禄
- 祖上有根、家学传承、少年气场足。

## 核心作用：给日主立根
- 没有禄的命 → 日主**悬着**，全靠其他十神支撑。
- 带禄的命 → 日主**有自己的地盘**：
  - 身自立，不畏对抗。
  - 做事有底气、有韧性。
  - **关键时刻扛得住压力**。

## 禄与其他神煞的区别

| 神煞 | 本质 | 气质 |
|---|---|---|
| **禄**（本条） | 日干本气之地 | 稳、立、自主 |
| **羊刃** | 日干刃位（禄前一位，阳干才有）| 刚、猛、过旺 |
| **驿马** | 三合局冲中气 | 动、迁、奔波 |
| **将星** | 三合局中气 | 领、控、主导 |

**禄与羊刃的关系**：
- 禄 = "临官"（站稳）；
- 羊刃 = "帝旺"（过盛）；
- 阳干有禄也有刃（过禄一位即刃），阴干只有禄无刃（阴刃之说不常用）。

## 判断维度

### 1. 禄是否被冲破
- 禄位被冲（如甲禄寅被申冲）→ **根动**。
- 大运流年冲禄 → 那段时期人生动荡，尤其事业、健康。

### 2. 禄是否被合化
- 禄位被合化为他气（如丙禄巳与申合化水）→ **根移**。
- 此时禄之"立身"性质改变，需按合化之神另论。

### 3. 禄的数量
- 一禄 → 标准、稳重。
- 双禄 / 多禄 → 身过旺、需官杀制或食伤泄。
- 太多禄 + 无制 → 转入"比劫重重"或"建禄无出口"的破局讨论。

### 4. 禄与其他十神的关系
- **禄 + 官杀清透**：身强能担官，典型贵格。
- **禄 + 财星有根**：能挣能花、有实业之贵。
- **禄 + 食伤吐秀**：才华外显。
- **禄无用神**：空有身强无出口 → 普通命局。

## 常见误区

### 1. "带禄 = 一定好命"
- 禄只是"**身有根**"，不等于"**命必贵**"。
- 身强无出口 → 空禄。
- 身强有官财食伤接应 → 禄才能兑现价值。

### 2. 把禄等同于"财禄"
- 现代白话"禄"常指薪水财禄，但命理上**禄 ≠ 钱**。
- 禄是"**立身之地**"，财禄是结果之一。
- 有禄的人不一定有钱，但**有钱扎根的能力**。

### 3. 忽视"禄逢冲"的破坏性
- 大运流年冲禄 → 事业、健康、家宅常见重大变动。
- 这是带禄命最关键的防守年份。

## 现实表现

### 性格
- **独立、自立、有主见**。
- **不易被外界左右**。
- **做事有底气**：遇事不慌、遇压不退。
- **内心有稳定感**。

### 职业路径
- 适合**自负盈亏 / 独当一面**的岗位：
  - 自由职业、创业者、技术专家、独立项目负责人。
- 在**需要扛事、决策、独自拍板**的场景最有优势。

### 人生节奏
- **自立型人生**：很早就不依赖父母。
- **靠自己打天下**：学历、人脉、资产都靠自己积累。
- **晚年扎根**：有禄的人老年往往很稳。

## 核心判断
- 禄的价值不在"字面贵气"，在"**日主有根**"。
- 识别时的关键问题：
  - 禄有没有被冲？有冲则不稳。
  - 禄之外有没有官、财、食伤做出口？无出口则空身强。

## 一句话总结
- 禄 = **日主的家**。
- 家稳不稳，直接决定这个人敢不敢、能不能、扛不扛得住。
- **没家的人四处飘，有家的人哪里都不怕。**
