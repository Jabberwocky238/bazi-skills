# 三奇贵人格

## 定义

- **三奇贵人格**：命局天干顺次排列出**乙丙丁、甲戊庚、壬癸辛**三组中任意一组，即构成此罕见贵格。
- 三组三奇分属**天、地、人**三才：
  - **天上三奇：乙丙丁**——为三奇之最尊，唯一无争议者（奇门遁甲中的乙奇、丙奇、丁奇即此）。
  - **地上三奇：甲戊庚**——传统甲戊庚牛羊之天乙贵人所本，主地格。
  - **人间三奇：壬癸辛**——主人才之奇。
- 三奇之名取"天地人三才之精华在天干聚合"之意，一旦成格，命主常有**奇遇、一夜暴富、骤然腾达**之象。

## 核心原理

- 三奇格的贵气机关在**天干顺序排列的三才聚合**：
  - 三奇代表天、地、人三种精华之气于命盘中聚齐，形成罕见的"三才贯通"结构。
  - 顺次排列意味着三气不相妨碍、次第流通，故贵。
  - 排列一旦颠倒 / 隔断 / 地支冲破 → 贵气受损甚至消散。
- 三奇贵人格与其他格局（如正格、从格、化气格）不冲突——它是**叠加在原有格局上的额外贵气加持**；成格则锦上添花，不成格则退为普通命局。

## 成立条件（古籍 + 客观判据）

### 条件 1：天干三奇齐聚

- **古籍出处**：
  - 《三命通会·论三奇贵人》："天上三奇乙丙丁，地下三奇甲戊庚，人中三奇壬癸辛。"
  - 《渊海子平·论三奇》："命中三奇聚于天干，贵不可言。"
- **客观判据**：
  - 命局天干四位中，必须**同时含有**某一组三奇的全部三个字（乙丙丁、或甲戊庚、或壬癸辛）。
  - 三组只要出现任一组完整即可；不可混搭（如"甲丙庚"非三奇）。
- **不满足则破**：三奇不全 → 无格可言；两字者仅为"半奇"，不论三奇贵人。

### 条件 2：三奇顺次排列（次序不得颠倒）

- **古籍出处**：
  - 《三命通会·论三奇贵人》："三奇须顺排，颠倒即失其贵。"
- **客观判据**：
  - 三奇必须按**特定次序**排列在天干位上，可顺排也可逆排，**但次序不能颠倒**：
    - **乙丙丁**：可排作乙丙丁或丁丙乙；不能排为乙丁丙、丙乙丁、丙丁乙、丁乙丙。
    - **甲戊庚**：可排作甲戊庚或庚戊甲；不能排为甲庚戊、戊甲庚、戊庚甲、庚甲戊。
    - **壬癸辛**：可排作壬癸辛或辛癸壬；不能排为壬辛癸、癸壬辛、癸辛壬、辛壬癸。
  - 简言之：**中间字固定不动**，首尾两字可对调。
- **不满足则破**：中间字被换位 → 三奇气脉紊乱，退为普通命格。

### 条件 3：三奇不得隔断

- **古籍出处**：
  - 《三命通会·论三奇贵人》："三奇隔断，贵气减半。"
- **客观判据**：
  - 三奇须占**连续的三个天干位**（年月日、月日时，或年月日时中任意三连位）。
  - 三奇之间不得夹杂他干（如"甲丙戊庚"中甲戊庚被丙隔断，属隔断之象）。
- **不满足则破**：
  - 完全隔断 → 贵气大损，但**仍比完全没有三奇的人要强**，属"有残贵"。
  - 古籍对隔断不是"一刀切破格"，而是"贵气受损"，须结合其他因素综合判断。

### 条件 4：地支不见刑冲破害三奇之根

- **古籍出处**：
  - 《渊海子平·论三奇》："三奇清纯，地支不宜刑冲。"
- **客观判据**：
  - 三奇所通之地支本气 / 根气不宜被刑冲破害。
  - 若三奇完整但地支互相冲刑严重 → 贵气受损。
- **不满足则破**：地支严重冲破 → 三奇徒有其表，贵气大打折扣。

### 条件 5：三奇清纯不杂

- **古籍出处**：
  - 《三命通会·论三奇贵人》："三奇须清，忌杂。"
- **客观判据**：
  - 三奇之外的第四个天干不宜强克三奇（如甲戊庚格中第四位若为强火克庚，则三奇之庚受伤）。
  - 原局无严重的合化夺走三奇之一（如乙庚合化金，把乙丙丁之乙合走）。
- **不满足则破**：三奇之一被合 / 被克 → 清纯受损。

---

**条件 1、2 是成格硬杠杆，条件 3、4、5 决定贵气的完整度。** 三奇格的"全"与"残"之间差别极大，但即便是残三奇也比完全无三奇者强。

## 与相关格局的对比

- **三奇贵人格 vs 天乙贵人**：
  - 天乙贵人（甲戊庚见牛羊口诀）是**点状的神煞**，根据日干对应地支定之。
  - 三奇贵人格是**线状的天干结构**——三个特定天干聚合顺排。
  - 两者都主贵，但三奇格是命局层面的大贵格，天乙贵人是单点的吉神。
- **三奇贵人格 vs 三庚格**：
  - 三庚格是"天干三同字"（三个庚），主财禄。
  - 三奇贵人格是"天干三异字顺排"（三个不同字按次序），主奇遇。
  - 三庚重"气势贯通"，三奇重"次第流通"。
- **三奇贵人格 vs 化气格**：
  - 化气格是日元合化转换，改变命局太极点。
  - 三奇贵人格不改变日元，只是在原有格局之上加一层"贵气加持"。
  - 两格可叠加：若既成化气格又含三奇，则为双贵。
- **三奇贵人格 vs 正格 / 从格**：
  - 三奇格是"加持"，原格局（正格 / 从格）仍按其自身逻辑取用；三奇只负责"贵气的级别"。
  - 原局为从格、正格配以三奇 → 富贵上再添奇遇；原局若本身破败，三奇也难独挽狂澜。

## 判断维度

### 1. 三奇是否齐全
- 第一步：乙丙丁 / 甲戊庚 / 壬癸辛 中是否有一组三字齐聚于天干。

### 2. 次序是否正确
- 中间字必须在中间位。中间字错位即破。

### 3. 是否连续或隔断
- 连续为贵，隔断为残贵。

### 4. 地支是否刑冲
- 地支不见严重刑冲 → 贵气稳固。

### 5. 三奇是否清纯
- 无合去、无强克 → 三奇清纯可用。

### 6. 叠加原格局判断
- 三奇是加持，不是主格；具体富贵程度仍要看原格局（正格 / 从格 / 化气等）。

## 现实表现

### 成格之三奇贵人
- **人生奇遇不断**：关键节点常有意外机遇，改命式事件频发。
- **一夜暴富 / 骤然腾达**：大运得时，财运事业突然爆发，和缓慢积累的发展模式形成鲜明对比。
- **贵人如影随形**：逢难有救，几乎每一关都有贵人伸手。
- **气质特殊**：自带某种难以言喻的清贵之气，容易获得他人敬重。

### 成格但隔断 / 残缺
- **仍胜于常人**：比没有三奇的命格要强，关键机遇仍会出现，但频率和规模打折。
- **需靠大运引动**：原局残缺可靠大运补足，走到补齐之运方兑现。

### 不成格（自以为有三奇而实无）
- 与普通命局无异。
- 最大风险是**心理暗示**——明明格坏了、顺序被打断了，仍自称三奇贵人格，导致对人生的预期偏差。

## 常见误区

### 1. 三字齐全即认为成格
- 错。必须**顺次连续排列**。次序颠倒或隔断都要降级。

### 2. 以为三奇一定代表巨富大贵
- 三奇是"贵气加持"，不是"富贵本身"。仍要看原格局成不成、日主担不担得起。

### 3. 把"人间三奇"和"天上三奇"争论谁更高
- 古籍有不同说法，**不必拘泥**——乙丙丁、甲戊庚、壬癸辛 都是三奇贵人，知道这三组即可。

### 4. 心理暗示硬套
- 最典型误区：命局明明顺序被打断了，仍自称三奇贵人格。**假的真不了**——格坏了就是格坏了，应降一档判断。

### 5. 忽略地支冲破
- 天干三奇齐全，但地支严重刑冲三奇之根 → 表面贵格，实则残贵。

### 6. 把三奇与天乙贵人混为一谈
- 两者都是"贵人"，但定义和判法完全不同，不可混用口诀。

## 破格与救格

### 破格的典型情形
- 三奇不全（缺一字）。
- 中间字错位（如甲戊庚排成甲庚戊）。
- 天干隔断（三奇之间夹杂他干）。
- 地支刑冲三奇之根。
- 三奇之一被合去或被强克。

### 救格的可能路径
- **隔断型三奇**：大运补足中间位或调动三奇之气，可在该运段内兑现贵气。
- **合去型三奇**：若合之者本身为日主喜用，可变通为"以合为用"，贵气不减。
- **地支冲破型**：大运填实冲破之处 → 三奇之根复稳，贵气回归。
- **三奇 + 正格同成**：三奇作为加持，原格局稳固则整体贵气自然兑现。

## 大运流年

| 大运 / 流年 | 效果 |
| --- | --- |
| 三奇所通之干得根运 | **大喜** → 三奇兑现，奇遇频发 |
| 三奇天乙 / 月德 / 日德相会之年 | **喜** → 贵人明显 |
| 冲三奇之干的大运流年 | **大忌** → 贵气受损 |
| 合去三奇之一的大运 | **忌** → 三奇残缺暂时成立 |
| 补足原局隔断之运 | **大喜** → 三奇由残贵转为全贵 |
| 地支冲破三奇之根 | **忌** → 根基摇动 |

## 一句话总结

- **三奇贵人，乙丙丁、甲戊庚、壬癸辛——三才聚顶，顺排见贵。**
- 成格条件三句话：**齐全、顺序、不隔断；地支不冲三奇之根**。
- 三奇是加持不是主格，但一旦真成格：**一生必有奇遇，往往骤然腾达、一夜发迹**——这便是三奇贵人独有的命运印记。
