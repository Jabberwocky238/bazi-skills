# 太岁

## 定义
- **太岁** = 当年的**年干支**——每年由一位**值年太岁神**主管。
- 全称**太岁星君**，源自上古的"岁星纪年"传统。
- 在民间信仰中被视为**一年的主宰神**，掌管当年的吉凶祸福。
- 命理层面，太岁 = **流年天干地支本身**。

### 古籍依据
- **《渊海子平》**："太岁乃一年之天子，故不可犯，犯之则凶。"
- **《三命通会》**："夫太岁者，乃一岁之主宰、诸神之领袖。"
- 以六十甲子干支为运转周期 → **每 60 年一循环**，每年一位值年神。

## 犯太岁的五种情况

### 核心原理
- 犯太岁**不是得罪神明**，而是**本命年柱地支 / 八字地支**与**当年流年地支**产生**不合的作用关系**。
- 包括：**值、冲、刑、破、害**五种。

### 五种犯太岁的对照表

| 类型 | 机制 | 核心影响 | 严重程度 |
| --- | --- | --- | --- |
| **值太岁** | 本命生肖 = 流年生肖（伏吟） | 重复、内耗、情绪起伏 | 最严重 |
| **冲太岁** | 本命生肖与流年生肖相冲 | 变动剧烈、节奏打乱 | 次严重 |
| **刑太岁** | 本命生肖与流年生肖相刑 | 官司、是非、刑伤 | 中 |
| **害太岁** | 本命生肖与流年生肖相害 | 暗中影响、小人作祟 | 较轻 |
| **破太岁** | 本命生肖与流年生肖相破 | 稳定事被破坏、突发 | 最轻 |

### 1. 值太岁（本命年）
- **机制**：生肖与太岁相同——俗称"伏吟"。
- **古诀**：**"太岁当头坐，无喜恐有祸"**。
- 本质：**你想跟当年的"老大"平起平坐** → 气场相撞。
- 典型影响：
  - **事业**：决策失误、升迁被压。
  - **健康**：老毛病复发、免疫力下降。
  - **感情**：情绪冲动、易争吵。

### 2. 冲太岁
- **机制**：**地支六冲**——子午、丑未、寅申、卯酉、辰戌、巳亥。
- 核心体感：**生活节奏被突然打乱**。
- 典型影响：
  - 换工作、搬家、换城市。
  - 长期合作突然终止。
  - 感情易突生巨变。
  - 健康防意外。

### 3. 刑太岁
- **机制**：**地支三刑与自刑**——寅巳申、丑戌未、子卯、辰辰、午午、酉酉、亥亥。
- 核心体感：**招惹官非、是非口舌**。
- 典型影响：
  - 因言行招致投诉、官司、惩罚。
  - 亲友被连累。
  - 身体易有**血光、刀伤、手术**。

### 4. 害太岁
- **机制**：**地支六害**——子未、丑午、寅巳、卯辰、申亥、酉戌。
- 核心体感：**暗中被害，问题慢慢积累**。
- 典型影响：
  - 背后被说闲话、合作方背叛。
  - 被误解、被孤立。
  - 慢性病、焦虑抑郁。

### 5. 破太岁
- **机制**：**地支六破**——子酉、卯午、辰丑、未寅、亥巳、申巳。
- 核心体感：**稳定事突然被打破**。
- 典型影响：
  - 合作对方突然退出、计划临时取消。
  - 感情冷战、家人小矛盾。
  - 突发小病、小伤。

## 本命年 vs 非本命年的犯太岁

### 本命年 ≠ 唯一的犯太岁
- 大众认知里，**只有本命年才犯太岁**。
- 实际上：**冲、刑、害、破太岁都不是本命年**，但同样会被影响。
- 所以一个人**每 12 年里，至少有 4 次左右会"犯太岁"**——只是程度不同。

### 不看生肖，还要看八字地支
- 犯太岁不只看**年柱地支**（生肖）。
- 还要看**月、日、时支**是否与流年太岁形成**值、冲、刑、害、破**。
- 举例：
  - 生肖不是马，但月支是子 → 2026 丙午年 → **月支被冲** → 影响事业、父母。
  - 生肖不是马，但日支是丑 → 丑午相害 → **婚姻、健康暗耗**。

### 宫位对应的影响

| 被冲的地支位置 | 影响领域 |
| --- | --- |
| **年支** | 祖业、父母、整体运势 |
| **月支** | 事业、父母、同辈 |
| **日支** | 婚姻、配偶、身体 |
| **时支** | 子女、晚年、下属 |

## 太岁对命局的作用关系

### 不是所有犯太岁都是坏事
- **喜用之神忌被冲**：冲掉用神 → **凶**。
- **忌神喜被冲**：冲掉忌神 → **反而吉**。
- **命理俗语**："**不病不为奇**"——八字与太岁、流年、大运的组合如果有**冲动**，往往是成大事之人。

### 古籍与实战经验
- 历史上许多**大富豪、三公九卿**的八字与当年太岁都有激烈作用关系。
- 相反，**平淡无奇的八字 + 平静的流年大运** → 一生默默无闻。
- **《六爻梅花》**：动摇才有作用。
- 所以**犯太岁未必凶，不犯太岁未必吉**——关键看**本命局的喜忌**。

### 判断流程
1. 确定本命局的**用神与忌神**。
2. 看流年太岁**冲、合、刑、害**了原局**哪个字**。
3. 看这个字是**用神还是忌神**：
   - 冲用神 → **凶**。
   - 冲忌神 → **吉**。
   - 合用神 → **多为羁绊、有得有失**。
   - 合忌神 → **忌神被化解，多为吉**。

## 化解太岁的传统方法

### 三个层面的化解思路

| 层面 | 做法 |
| --- | --- |
| **地道（调整外部环境）** | 调整家居、办公室布局；**岁破方**（太岁正对面方位）少动土、少居住 |
| **天道（借助外部力量）** | 民俗层面可求正规太岁符、拜太岁；结合命局喜忌做具体调整 |
| **人道（最根本）** | 管住自己的**眼、耳、口、手、腿、心**——少冲动、少是非、少妄动 |

### 人道层面的具体做法
- **管住眼**：不看不该看的东西。
- **管住耳**：不听是非八卦。
- **管住嘴**：不说三道四、不与人口角。
- **管住手**：不与人动手冲突。
- **管住腿**：不凑热闹、不去险地。
- **管住心**：知"**好奇害死猫**"，时刻记住**今年大领导（太岁）在看着自己**。

### 化解的核心心态
- **敬畏**：对太岁保持尊重——这是传统文化里最根本的化解。
- **谨慎**：在犯太岁年份里，凡事多做准备、少冒险。
- **借势**：若犯太岁是冲走忌神 → **可借此势头谋大事**。

## 一句话总结
太岁是**一年运势的总管家**，犯太岁是**八字地支与太岁不合**的五种情况。
- **不犯太岁未必吉，犯太岁未必凶**——关键看本命喜忌。
- **最有效的化解不是烧符拜神**，而是管好自己的眼耳口手腿心。
- **太岁面前，敬畏是最稳的姿态，谨慎是最智慧的行动。**
