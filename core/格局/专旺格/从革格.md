# 从革格

## 定义
- **从革**：《尚书·洪范》"金曰从革" —— 金的本性是**"从" + "革"**，即顺从与变革。
- **从革格**：**金日主 + 全局金气一片** → 形成**金的专旺格局**。
- 属**五行专旺格**之一，与曲直格（木）、炎上格（火）、稼穑格（土）、润下格（水）并列。
- 典型人格：**锋利、决断、清冷、能破能立**。

## 核心：百炼成钢，一色成锋
- 金 = **锋锐、裁断、秩序、肃杀**。
- 从革格 = 全局金气 → **整个人都是金的气质**：
  - 杀伐果决。
  - 清冷理性。
  - 不妥协、不含糊。
- "从"的一面：顺从大势、识时务。
- "革"的一面：革故鼎新、破旧立新。
- **一颗锋利之心，既能服从纪律，也能改写规则**。

## 成立条件（严格依《渊海子平·从革格》）

### 条件 1：日主为庚或辛金

- **古籍出处**（本条为本格前提）：
  - 《渊海子平·从革格》："**从革格——庚辛日生人**。"
- **客观判据**：日主天干必须为庚金或辛金。
- **不满足则破**：非庚辛日 → 不入从革格。

### 条件 2：地支会成金局（三合巳酉丑 或 三会申酉戌）

- **古籍出处**（本条核心）：
  - 《渊海子平·从革格》："**地支巳酉丑三合金局，或申酉戌西方金气**。"
- **客观判据**（二者之一）：
  - ① 地支见 巳 + 酉 + 丑 三合金局；或
  - ② 地支见 申 + 酉 + 戌 三会西方金。
  - 若仅半合（巳酉、酉丑、申酉）或二字会气，格品降但勉强成立。
- **不满足则破**：地支不成金局 → 一气不聚。

### 条件 3：天干多透金，戊己土生金更佳

- **古籍出处**：
  - 《渊海子平·从革格》："年月时干宜见庚辛，**有戊己土相生更妙**。"
- **客观判据**：
  - ① 日主外，天干另有至少一位庚或辛透出；
  - ② 若另见戊或己透干生金，格品升（土生金助势）。
- **不满足则破**：天干金气不足 → 一气未遍布。

### 条件 4：原局无丙丁火透干熔金

- **古籍出处**（本条核心）：
  - 《渊海子平·从革格》："**年月时干无火相克**，主威武显达。"
- **客观判据**：
  - ① 天干无丙火或丁火透出；
  - ② 地支本气无巳、午成势。
- **不满足则破**：火透熔金 → 《渊海子平》"火相克"即明破格 → 转入"火旺金衰"或"火多金熔"讨论。

### 条件 5：原局无甲乙木过重对抗（木亦忌）

- **古籍出处**：
  - 《渊海子平》："从革者，忌财破其气——木为金之财。"
- **客观判据**：
  - 天干不多透甲乙（一位无根尚可）；地支本气寅卯不成势。
- **不满足则破**：木多 → 金需对抗 → 一气被牵动，转入普通身强见财格。

### 条件 6（助力条件）：水秀气 → "金白水清"复合贵格

- **古籍出处**：
  - 《穷通宝鉴·庚金章》："**庚金逢壬水，为金白水清**。"
  - 《滴天髓·从象》："一气成象，微泄秀气方妙。"
- **客观判据**：
  - 壬水或癸水在命局中一位透干或藏支——此时从革格升级为"从革 + 金白水清"复合贵格。
- **非必要**：不满足不破格；满足则格品升（锋锐之外添清秀）。若水多过度泄金则降格。

---

**前五条必须同时满足**方为真从革格；条件 6 关乎格品——纯从革成但偏冷刚，得水秀气则为"清贵"。

## 判断维度

### 1. 金的纯度
- **四柱全金**：格最纯。
- **三柱金 + 土生金 / 水秀金**：亦成格。
- **夹杂多种五行**：格次之。

### 2. 水的参与
- **有水秀气**：金白水清 → 思辨清晰、表达流畅、格局高。
- **无水**：纯从革 → 偏冷、偏刚，较难通人情。

### 3. 土的位置
- **适度土**：生金助势 → 稳固本格。
- **土过多**：土重金埋 → 反而埋了金 → 破格。

### 4. 大运流年
- **金水土运**：顺气加强。
- **木火运**：破格。
  - 火熔金 → 从革变废。
  - 木克金 → 金忙于应战，失去一色之势。

## 常见误区

### 1. "金日主 + 身强 = 从革格"
- 身强 ≠ 从革。
- 从革要求"**一气独尊 + 全局一色**"。
- 仅身强只是普通身强格。

### 2. 以为从革 = 一定成大器
- 从革确实是贵格，但极依赖大运：
  - 顺势时可大成。
  - 遇大木大火 → 瞬间瓦解。

### 3. 忽视"清"的重要性
- 纯从革 → 偏冷、偏刚。
- 有水秀气的从革 → 清雅 + 锋利 → 格局远高。
- 很多传统命理名人的从革格，都是"**带水秀**"。

### 4. 把"从革"等同"杀伐"
- 从革不止是"杀伐果断"，还有"**革新变法**"的意义。
- 从革格的人**改革精神强**：不怕推翻旧秩序，敢于重建。

## 现实表现

### 性格
- **锋利、决断、直接**。
- **理性 > 情感**：遇事先算 ROI，情感其次。
- **清冷、不善套近乎**。
- **正义感强**：见不得不公、见不得糊涂账。
- **不妥协**：该断就断、该革就革。
- **"冷幽默"式的机智**。

### 事业路径
- **法律、司法**：法官、律师、检察官、监察机构。
- **军警、武装**：军官、特警、情报系统。
- **金融、审计**：金融分析、审计、证券业。
- **医疗**：外科医生（用刀的医生）、急救。
- **机械 / 工程**：金属加工、精密制造、汽车工业。
- **改革派 / 革新者**：推动制度变革的人。
- 共性：**需要"切割 / 判断 / 重建"**的岗位。

### 成长节奏
- **早期锋芒显露**：青少年就显出"**与众不同的锐利**"。
- **中期决断成就**：30-40 岁关键决策树立事业。
- **晚期威名在外**：成为**领域内的定盘之人**。

### 人际
- **朋友不多但都可信**。
- **对下属严格**：但有功必赏、有过必罚。
- **对亲人冷硬外表下有柔情**。
- **婚姻需要"懂他的刚"**的伴侣，否则容易冷战。

### 身体
- 金对应**肺、大肠、皮肤、鼻**。
- 从革格：金气旺 → 这些功能通常**强壮**，但过度"锐"也可能导致**燥**。
- 容易有皮肤干燥、呼吸系统紧绷、长期精神高度集中带来的慢性疲劳。

## 破格

### 破格情形
- **木重克金**：从革被迫应战 → 失一色之美 → 格破。
- **火重熔金**：金被毁 → 从革废 → 大凶。
- **水过度泄金**：金气外泄过多 → 空有清秀无本体 → 格变弱。
- **夹杂严重异气**：一气不纯 → 从革假。

### 真假从革
- **真从革**：一气纯净、无木无火（极少）→ 顺势大贵。
- **假从革**：略有异气、自认从革 → 实则按普通格局论处 → 识别错误会酿大祸。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 金运 | **大喜** → 一气加强、本格兑现 |
| 土运 | **喜** → 生金助势 |
| 水运 | **喜** → 秀气外显、金白水清 |
| 木运 | **忌** → 克金破格 |
| 火运 | **大忌** → 熔金彻底 → 大起大落 |

## 与其他专旺格的对照

| 格名 | 一气 | 关键 |
|---|---|---|
| **曲直格** | 木专旺 | 生发之气 |
| **炎上格** | 火专旺 | 光明之气 |
| **稼穑格** | 土专旺 | 承载之气 |
| **从革格**（本条） | 金专旺 | 变革之气 |
| **润下格** | 水专旺 | 流动之气 |

从革格在五行专旺里**最"独立"** —— 金的本性就是裁断与改革，从革格的人**天然有破旧立新的倾向**。

## 核心判断
- 从革格的核心在"**一气独尊 + 气要清**"：
  - 一气独尊 → 不被异气牵动。
  - 气清 → 有水秀气更佳。
- 识别要点：**有没有木 / 火来动摇一气**？
- 成格时：**百炼成钢，能破能立**。
- 破格时：**空有锋芒，被时代碾碎**。

## 一句话总结
- 从革 = **顺从 + 革新**：既能服从大势，也能改写大势。
- 一把锋利到底的剑 —— **用在正处是除害，用在偏处是伤人**。
- 从革格的人：一辈子都在**剑锋上行走**，但正因如此才留下印记。
