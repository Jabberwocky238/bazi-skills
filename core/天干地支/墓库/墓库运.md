# 墓库运

## 定义

辰戌丑未被称为四墓库。分析大运时，如果**日主**在大运中遇到自己的墓库地支，就称为**墓库运**。

**需特别注意**：这里所说的墓库是针对**日主**而言，而不是针对财、官、印、食等其他十神。

## 各日主对应的墓库运

| 日主 | 墓库运 |
| --- | --- |
| **甲乙日** | 未运 |
| **丙丁日** | 戌运 |
| **戊己日** | 戌运 |
| **庚辛日** | 丑运 |
| **壬癸日** | 辰运 |

### 争议观点
- 有观点认为**辰土也是戊己土的墓库**（因辰是万物之库，可将戊己土收纳其中）。
- 另一种说法是通过**十二长生表**查找每个天干对应的墓地：
  - 甲木墓在未、**乙木墓在戌**。
  - 因此乙木日主行**未运为墓库运，行戌运也是墓库运**。

这两种说法都存在争议，建议入门后再深入探索。

## 墓库运的吉凶：分年龄阶段

古籍说法：
- 《三命通会·卷五·论墓运中秘诀》："**幼年不宜逢墓库，老年至此却丰隆**。"
- 《金锁铁秘籍》："**少年墓库，糊里糊涂；中年墓库，颠倒反复；老年墓库，发财发福**。"

墓库一般意味着**将一个五行收纳起来**，或代表将现实中对应这个五行的事物收纳起来。日主走墓库运 → 代表这段时间自己**各方面仿佛被禁锢住了**，各种事物都被收纳起来 → 整个人有**被压抑的感觉**，思维思路都受到限制。

### 三个年龄阶段的墓库意象

#### 1. 少年时期走墓库运
- 本应血气方刚、学习奋斗的年纪。
- 进入墓库运 → **缺乏活力，虚度光阴**。

#### 2. 青中年时期走墓库运
- 本应积极开拓事业的时期。
- 进入墓库运 → **多次起伏、成败不定、付出多收获少**。

#### 3. 老年时期走墓库运
- 对老年人来说**大多较好**。
- 墓库虽封闭，但也能承载资源，**适合养老**。
- 类似"庄稼成熟后秋收冬藏"，属于正常现象 → "**老年墓库发财发福**"。

### 总结规律
- 对日主而言，**少年、青年、中年**需要积极奋斗的时期——**入库或入墓都不好**，都属反常现象（与"旺官旺印与旺财入墓生灾"一致）。
- **老年时期**进入墓库 → 属正常现象，故有"老年墓库幸福"的说法。

## 墓库运的严重情形
如果**日主受到严重克制而进入墓库** → 如同人遭遇灾难住院、坐牢，或死后进入坟墓 → 能看出一些明显吉凶。

## 重要提醒

墓库运的这些异象**只能作为参考**，不能作为喜忌吉凶的定论。

这与命书中所说的"**老怕帝旺、少怕衰、中年最怕死绝胎**"一样——仅针对日主而言，也**只能作为参考**，不能作为格局喜忌成败、人生吉凶祸福的定论。

## 墓库取格的特殊规则

取格以月令为基础，当**墓库为月令**时，取格的最大原则是：
> **以墓库月令中透出的天干来确定格局**。

**原因**：墓库里的藏干之气较为复杂，只有**透出来的天干才是清晰而有用的**。

## 综合应用要点

1. **针对性**：墓库运专门针对**日主**，与其他十神墓库（财库、官库等）的概念不同。
2. **阶段性**：年龄段决定墓库运的基本吉凶方向。
3. **参考性**：一切"墓库运吉凶"只是参考，具体吉凶需结合格局、喜忌、整体命局判断。
4. **配合岁运**：墓库运中若遇流年冲刑 → 可能开库或毁库，应事剧烈程度不同。
5. **取格规则**：墓库为月令时，以透出天干定格局。

## 一句话总结

墓库运 = 日主被收纳的十年——少年怕误时、中年怕颠倒、老年却正好收成。**年龄决定吉凶方向，结构决定吉凶程度**。
