# 羊刃驾杀

## 定义
- **羊刃**：阳干日主的**刃地**（如甲见卯、丙戊见午、庚见酉、壬见子），是日主极旺的标志。
- **驾**：驾驭、乘坐。
- **杀**：七杀。
- **羊刃驾杀**：日主有羊刃（极旺）+ 命局见七杀 → **用羊刃之刚驾驭七杀之势** → 身杀两停，刚猛并行。
- 传统命理中的**猛格** —— 贵而凶，多出**将帅、豪强、商界强人**。

## 核心：以刚驭刚，刀入刀鞘
- 羊刃 = **日主刚烈到极点**。
- 七杀 = **外部压力 / 挑战 / 竞争到极点**。
- **羊刃驾杀** = 用日主的刚猛**恰好抵住**七杀的猛烈：
  - 身杀两停 → 不被克倒、不被克穿。
  - 互相制衡 → 刚而有度、狠而有节。
- 这不是"压制"或"化解"，是**两股极刚之力互相成全**。

## 与相关格局对比

| 格局 | 关键动作 | 日主状态 | 气质 |
|---|---|---|---|
| **羊刃驾杀** | 身杀两停 | 身极强 | 刚猛对冲、身杀互制 |
| **杀印相生** | 印化杀 | 身中 | 抗压后升华 |
| **食神制杀** | 食克杀 | 身中强 | 主动解决 |
| **伤官合杀** | 伤官去合 | 身中强 | 迂回化解 |
| **从杀格** | 顺杀 | 身极弱 | 彻底顺势 |

**羊刃驾杀**是唯一**身极强**对**杀极强**的组合方式 —— 对抗是主旋律，不是化解。

## 成立条件（严格依《子平真诠·论阳刃》）

### 条件 1：日主为阳干（甲、丙、戊、庚、壬五者之一）

- **古籍出处**（本条为本格前提）：
  - 《子平真诠·论阳刃》："**阳刃者，阳干之刃也**。"
- **客观判据**：
  - 日主天干必须为甲、丙、戊、庚、壬五阳干之一。
- **不满足则破**：阴干日主（乙、丁、己、辛、癸）**结构上不具羊刃**——沈氏明示"阳刃"之名专归阳干。本格根本不适用。

### 条件 2：月支、日支或时支为日主之刃位

- **古籍出处**：
  - 《子平真诠·论阳刃》："**甲禄在寅，刃在卯；丙戊禄在巳，刃在午；庚禄在申，刃在酉；壬禄在亥，刃在子**。"
- **客观判据**（五阳干刃位对照）：
  - 甲日见卯、丙日见午、戊日见午、庚日见酉、壬日见子——落于月支、日支或时支。
- **不满足则破**：阳干日主但刃位不见于命局地支 → 不入阳刃格，本格不适用。

### 条件 3：七杀透干且通根，与羊刃势均（刃杀两停）

- **古籍出处**（本条最核心）：
  - 《子平真诠·论阳刃》："**羊刃无杀，不足为贵；七杀无刃，不足为威**。"
  - 《子平真诠·论阳刃》："**阳刃用官则喜露，用杀则不忌——刃杀相济，声威赫奕**。"
- **客观判据**：
  - ① 七杀透于年 / 月 / 时干，地支有本气或中气根至少一位；
  - ② 羊刃与七杀力量势均（两停）——可综合透干数、通根位数、月令气势判断。
- **不满足则破**：
  - 七杀虚透无根 → 刃无敌可驾，转入"阳刃格"单论；
  - 七杀过重压过羊刃 → 身不胜杀 → 破格转入身弱杀重讨论；
  - 羊刃过重压过七杀 → 杀被压灭 → 羊刃独旺，空有蛮力。

### 条件 4：原局无重印化杀

- **古籍出处**：
  - 《子平真诠·论阳刃》："阳刃驾杀，忌印多夺杀之势。"
- **客观判据**：
  - 天干无两位以上印星透出；地支本气无两位以上印星。
- **不满足则破**：印多化杀 → 本格转入"杀印相生"讨论，非驾杀。

### 条件 5：原局无重食伤制杀

- **古籍出处**：
  - 《子平真诠·论阳刃》："刃杀相济，忌食伤多制。"
- **客观判据**：
  - 天干无两位以上食神 / 伤官透出；地支本气无两位以上食伤。
- **不满足则破**：食伤过重制杀 → 杀倒 → 本格转入"食神制杀"或"伤官见官"讨论。

---

**五条同时满足**方为真羊刃驾杀格。本格之要义在"**两停**"——沈孝瞻"刃杀相济，声威赫奕"八字明示：刃与杀须平衡互制，任一方过强过弱即破格。此格专属阳干（条件 1 不可破），且极依赖大运流年维持两停。

## 判断维度

### 1. 身杀两停的精度
- 力量悬殊 → 失衡：
  - **身过强、杀过弱** → 羊刃无敌可驾 → 变成"**羊刃独旺**"的烦躁命局。
  - **杀过强、身过弱** → 身被杀克 → 凶险。
- 两者相差不大 → **真正的驾杀成立**。

### 2. 大运流年的关键
- 羊刃驾杀格**最怕失衡**：
  - 走印比运 → 身过强，杀被压死 → 羊刃爆发 → 暴躁 / 惹祸。
  - 走食伤运 → 杀被克 → 身更孤旺 → 失去对手，无处发力。
  - 走财运 → 财生杀 → 看能否保持平衡：若身杀略偏身，财运可使之重新平衡；若身本不足，则变危险。

### 3. 羊刃与七杀的位置
- **羊刃在日支 + 七杀在月干**：最典型 → 日主的刚直接顶住月令的杀。
- **羊刃在时支**：晚运发力 → 中后期兑现。
- **两者相邻**：互制有力。
- **两者相距远**：制衡效果减弱。

### 4. 日主的五行
- **阳干**（甲丙戊庚壬）才讲羊刃。
- 不同阳干的羊刃驾杀性质略异：
  - **甲见卯 / 庚见酉**：剑拔弩张，偏军警武职。
  - **丙戊见午**：火土刚烈，偏商界豪强 / 行动派领导。
  - **壬见子**：水之极旺，偏智谋型强人。

## 常见误区

### 1. "有羊刃 + 有七杀 = 羊刃驾杀"
- 必须**力量势均**。
- 一边倒的 → 不是驾杀，是失衡。
- 必须**无重印食伤介入**。

### 2. 把羊刃驾杀等同于"一定大贵"
- 是贵格之一，但极不稳定。
- **大运流年一旦打破两停** → 格局瓦解 → 可能由大贵转大凶。
- 成贵的羊刃驾杀命造也常见**人生多风波**。

### 3. 忽视内在代价
- 羊刃驾杀的人：
  - **常年紧绷、不能放松**。
  - **控制欲与警觉心极强**。
  - **对冲突敏感、对妥协抵触**。
  - 是典型的**"开弓没有回头箭"**型人生。

### 4. 以为"杀是凶"不能带
- 带杀是问题 → 带杀能驾才是本事。
- 本格的精髓正在于**不惧杀、不逃杀、不化杀、直接硬碰**。

## 现实表现

### 性格
- **刚猛、果敢、有锋芒**。
- **不怕事、遇强更强**。
- **执行力极强、决断力极强**。
- **对权威不卑不亢**：既不怕上司，也不欺下属。
- **自带气场**：外人一眼能感到"不好惹"。

### 事业路径
- **军警 / 武职 / 特种行业**：典型羊刃驾杀的老地盘。
- **商界强人 / 创业者**：在强竞争环境里生存发展。
- **高压决策岗**：投行、律师、谈判专家。
- **体育 / 竞技**：爆发力和专注力的结合。
- 共性：**需要"对抗 + 意志力"**的赛道。

### 成就特征
- **不走常规路径**：硬干、硬闯、硬挣来的。
- **成就与凶险并存**：贵格但经常伴随**官非、伤灾、事故**。
- **英雄气 + 悲情色彩**：很多有羊刃驾杀的人一生精彩但不平静。

### 人际与感情
- **朋友少而铁**：认人认事都极严。
- **婚姻多波折**：对伴侣要求高、容不得太多妥协。
- **亲密关系中的主动权强**：但也容易让对方感到"压迫感"。

## 破格与救格

### 破格
- **身杀失衡**：任何一方过强过弱 → 格破。
- **印过多**：印化杀，变成杀印相生（换格）。
- **食伤过多**：食伤克杀 → 杀倒 → 羊刃无敌可驾 → 空有蛮力。
- **比劫再帮身**：身更旺 → 刃压杀 → 失衡。

### 救格
- **无多印、多食伤**：保持身杀两停。
- **大运再走杀地**（身略强时）：平衡重新建立。
- **大运走财地**（身不过弱时）：财生杀 → 维持两停。

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 杀运 | 视原局状态 → 身偏弱则补杀，身偏强则失衡 |
| 财运 | 视原局 → 略偏身时财生杀恢复平衡；身本不足则凶 |
| 食伤运 | **忌** → 制杀 → 格破 |
| **比劫 / 羊刃运** | **大忌** → 身过强 → 格破 → 易有冲动事件（官非、伤灾、牢狱） |
| **印运** | **忌** → 转杀印相生，性格从刚猛变温吞 |

## 核心判断
- 羊刃驾杀的**核心不是"刚"**，是**两股刚的互相成全**。
- 失去对方的刚 → 就是失去了自己的平衡点。
- 这种格局最怕的不是"对手太强"，而是**"没有对手了"**。
- 很多羊刃驾杀的人**成就巅峰时反而走下坡** —— 因为把对手打倒了，自己也没了发力点。

## 一句话总结
- 刚对刚不一定是冲突，可以是**相互成就的格局**。
- 羊刃驾杀 = **让刀和鞘一样刚硬**，互相磨、互相立。
- 没了对手，这把刀也就失去了意义 —— 这是它最深的宿命。
