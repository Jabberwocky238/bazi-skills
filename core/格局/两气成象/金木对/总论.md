# 金木对（总论）

## 性质：相克两气对
- **五行关系**：金克木（七杀 / 偏财关系）。
- **非"印生两神成象格"**——该格专论相生两气（如金→水），金木属相克，不入双清贵格。
- 属"两气成象"的**相战 / 对立**一支，不是相生双清的贵格。

## 两气成象通则（适用于本对）

### 成象要件
- 命局只见金、木两种五行（其他三种基本不见）。
- 金、木力量大致相等。
- 月令为金或木之一。

### 与相生对的根本区别
- **相生对（金水、金土、木水、木火、火土）**：两气顺生，气通为清，属贵格。
- **金木对（本条）相克**：金克木，气机对立，属"两气相战"或"斧斤伐木"病象——非贵格。

### 取用要点
- **忌加剧相克**：大运再走金旺或木旺之地，冲突加剧。
- **喜通关五行**：**水** = 金 → 水 → 木，通关和解，化战为顺。
  - 金木相战若得水通关，反而化为"金白水清 + 水木清华"的复合贵格。
- **喜温和调候**：火可制金救木；土可生金泄木——视日主喜忌而定。

### 古籍背书
- 《滴天髓·象论》："**两气合而成象，象不可破也**。"
- 《滴天髓》："**两气相战，须得通关之神以解**。"
- 金木相战的经典论述：**斧斤伐木**——金过旺克木，木被伤。

## 本对下属格局
- **斧斤伐木**：金旺克木的典型病象。详见 `斧斤伐木/`。

## 识别路径（成立条件简要）
- 日主为金或木。
- 命局金木两旺而少水火土。
- 金木力量对比接近，无第三通关五行 → 属相战 / 病象。
- 有水通关 → 升为复合贵格（不再是纯金木对）。

## 一句话总结
- 金木对 = **相克两气对立**——非双清贵格、而是冲战之象。
- 要解：**得水通关**（金→水→木，化战为顺）。
- 无水则凶，有水则贵。
