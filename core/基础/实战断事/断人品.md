# 断人品：看地支，不要看天干

## 核心原则
- **天干是外在的表象，地支是内在的本相**。
- **眼见不一定为实**——因为每个人的内心都很难揣测。
- 天干 = 一个人**摆在外面的样子**。
- 地支 = 他**骨子里的性子**。
- 大多数人只盯天干看，就像通过衣服判断一个人值不值得交往，容易被表象骗得团团转。

## 为什么天干只是表象

### 天干是"社交场合的自我介绍"
- 露在盘面上，是**别人第一眼看到的标签**。
- 可以被迎合、被包装、被修饰。

### 典型例子
- 天干透财：看着**大方、能说会道**。
- 天干透官：看着**正派、讲原则**。
- **但这些只是外壳**。
- 就像看到一个人穿西装打领带，**不代表他私下不邋遢**。

## 为什么地支才是真相

### 地支藏的是"根性"
- 那些**不轻易示人**的一面。
- 无法伪装的本能反应。

### 天干地支常见反差
- 天干透印，看似温和厚道——**地支比劫重**，内里其实争强好胜。
- 天干透财，看似热情大方——**地支伤官旺**，心里其实有一套自己的算盘。

### 只凭天干断人性的后果
= **凭朋友圈照片判断人品**。
- 容易被化妆照、滤镜、人设骗。

## 眼见不一定为实：天干地支的反差

### 反差的本质
- **天干可以迎合别人**。
- **地支是无法伪装的本能反应**。
- 有些人婚前婚后判若两人——**不是他变了**，而是你之前没看到他地支那一面。

## 如何看清一个人

### 不要停在表层
- 不要只看"他看起来是什么人"。
- 要追到"他**骨子里**是什么人"。

### 关键关系里必须看地支
- **做朋友**。
- **谈合作**。
- **选伴侣**。

### 看配偶、合伙人时尤其要紧
- 一定要"**掀翻底牌**"。
- 不然天干的笑脸可能藏着地支的刀锋。

## 一句话核心
**天干是化妆，地支是素颜。**
- 看人不能只看化妆照。
- 得看**卸妆之后**长什么样。

## 补充：道德不正的结构识别

### 核心命题
- 有些八字在天干看似"温和厚道"，地支却藏着尖锐的**财坏印、枭神夺食、伤官见官**。
- 这种表里反差是识别"心术不正"类人品的关键。

### 结构 1：天干印 + 地支财伤反差
- 天干透**偏印 + 比肩**，看起来老实、本分。
- 地支却是**偏财 + 伤官**，藏着"自利、爱搞事"的心态。
- 本质：
  - 印被天干伪装出来 → 别人眼中"稳重可靠"。
  - 地支伤官 + 偏财 → **骨子里是以自我为中心、愿意算计别人的本质**。

### 结构 2：偏印喜忌不明
- 偏印既是**为用**（生比肩帮身）又是**为忌**（克食神）。
- 命主自己说不清偏印的好坏 → 表现为**双标、看人下菜**。
- 对上装乖、对下算计；对外装温和、对内发泄。

### 结构 3：财坏印持续循环
- 财长期坏印、印又夺食。
- 三者互相损耗 → **自我认知扭曲**。
- 表现：
  - 对别人道德标准严苛。
  - 对自己却放松、放任。
  - **严于律人、宽于欲己**。

### 结构 4：水旺印重 + 食神受克
- 原局水汽太重、印为用但忌中忌。
- 食神被土重埋或水冲干 → **思想与表达的通道双双受损**。
- 表现：
  - 容易抑郁的同时也容易扭曲。
  - 对自己反复自我否定，对别人又无共情。

### 结构 5：年时伏吟 + 兄弟姐妹结构相似
- 年柱与时柱天地相同（伏吟）。
- 兄弟姐妹八字也可能呈现类似结构。
- 暗示：**家庭环境整体有深层问题**。
- 这不是要贴标签，而是要理解**道德底色是家庭共业的结果**。

### 识别的正确方式
1. **不要只看天干**：天干可以是高情商的表演。
2. **看地支 + 做工方向**：地支才能揭示本能与动机。
3. **看内外的反差**：反差越大、越要小心。
4. **看喜忌的清晰度**：某个十神无法明确喜忌 → 常常是人品模糊的信号。

### 谨慎用法
- 这些结构**不等于恶人**。
- 表示某种**人格倾向**，具体表现还看：
  - 家庭教育。
  - 受过的情感与挫折。
  - 是否有自省能力。
- 命理学只说倾向，不替天行道。

## 补充：人品失衡的典型底色

### 核心命题
- 某些八字的**结构失衡到"不清纯"**的程度 → 人品上易出现**反规则、无原则、轴而不自省**的底色。
- 不是所有不清纯的八字都是"坏人"，但**多个失衡叠加**容易显现人品问题。

### 底色 1：财坏印不彻底、又无辅助做工
- 财印碰撞激烈但没有清晰方向。
- 表现：
  - 除旧力心有，但**不能立新**。
  - 批判别人 / 旧势力有力，自己反而**跨越规则**做事。
  - 典型的"无君无臣、无长无业、无道无德、无规无矩"。

### 底色 2：偏印重 + 只靠自己的靶子
- 偏印给的是**思维的庇护**（自己认定的世界观）。
- 地支再被财破、印又夺食 → **没有箭只有靶子**。
- 表现：
  - 完全依赖自己的既有认知去评判世界。
  - 听不进别人的话。
  - 别人说的就是"屁话"。
  - **严于律人、宽于欲己 + 双标**。

### 底色 3：印性被彻底废掉 + 只剩财星求异
- 印被反攻至虚透 → 不能行使印的养护功能。
- 只剩财星的"求新求异、追求拥有"的特性。
- 表现：
  - **只讲利益、不讲规矩**。
  - 没有底线意识。
  - 在极端情境下容易做出"价值观之外"的事。

### 底色 4：做工太杂、三条路径同时做
- 原局财官印相互交织。
- 没有主路径 → 每条路径都在做一半。
- 表现：
  - 说话做事**反复、无定见**。
  - 自我认知混乱。
  - 一遇挑战就改口。

### 底色 5：全局皆忌神、无药可治
- 原局无贵气、无药、又逢忌神大运。
- 身心长期处于"病态"状态。
- 表现：
  - 对世界充满敌意。
  - 无力改变自己 → 把痛苦转嫁他人。

### 与"案例验证"的关系
- 这些结构的识别**要结合求测者的实际反馈**。
- 不要看到一个八字就直接下结论"你人品不好"。
- 真正的判断流程：
  1. 识别结构。
  2. 提出倾向假设。
  3. 让命主自己确认 / 否认具体事件。
  4. 最后得出"人品底色"的判断。

### 防坑建议（对求测者）
- 若命理师一看八字就说"你道德有问题" → 大概率在套公式 / 恐吓。
- 真正负责任的判断会先讨论**结构与做工**、再讨论"可能的性格倾向"、最后才谈到"人品"。

### 防坑建议（对合作 / 择偶者）
- 若对方八字呈现**多个失衡结构叠加** + 现实反馈也印证 → **保持距离、慎重投入**。
- 若只是单一结构 + 对方在现实中表现正常 → **不必过分担忧**。

### 一句话总结
- 人品的命理底色 = **原局清纯度 + 各十神是否正常发挥 + 自我认知是否打通**。
- 失衡越多 → 风险越大。
- 但命理只给"倾向概率"，最终的人品仍由**具体选择与环境**塑造。
