# 七政四余

## 定义
- 七政四余是中国古代的一门**星命学**，以**真实天象**为基础推演人生。
- 不是占卜游戏，而是建立在**天文观测**之上的符号系统。
- 与八字同属传统命理，但**测算路径完全不同**：
  - 八字 = **干支气象学**（五行气机的组合与生克）。
  - 七政四余 = **星命天文学**（星体落宫、行线与星度）。

## 组成

### 七政（七颗实星）
| 星 | 含义象征 |
|---|---|
| 太阳 | 权威、父亲、领导力 |
| 太阴 | 情感、母亲、内在感知 |
| 水星 | 智慧、学习、沟通 |
| 金星 | 情感、审美、财富、关系 |
| 火星 | 行动、冲动、技术、热力 |
| 木星 | 扩张、贵人、福气 |
| 土星 | 约束、考验、慢变与稳固 |

### 四余（四颗虚星 / 推算点）
| 余 | 含义象征 |
|---|---|
| 紫气 | 高妙、清贵、理想化 |
| 月孛 | 暗伤、隐藏的情绪与宿业 |
| 罗睺 | 迷惑、牵引、业力 |
| 计都 | 了结、剥离、释放 |

### 盘面结构
- **十二宫**：命、财帛、兄弟、田宅、男女、奴仆、夫妻、疾厄、迁移、官禄、福德、相貌。
  - 给星曜提供"**落在哪里发生**"的坐标。
- **二十八宿 + 星度**：决定星的精细位置与作用力强弱。
- **行线（大限 / 小限）**：星随时间行至不同宫度，触发人生节点。

## 核心定位：星命与气象的分工

| 层次 | 八字 | 七政四余 |
|---|---|---|
| 基础 | 干支五行 | 实际天象位置 |
| 颗粒度 | 时辰级 | 月份 / 日级 |
| 擅长 | 格局 / 天赋 / 大势 | 应期 / 事件性质 / 月级节点 |
| 路径 | 取象推演 | 取实测推演 |

- 两者**不是替代关系**，是**互补关系**。
- 八字回答"**方向、格局、我是谁**"；七政回答"**什么时候、发生什么**"。

## 判断维度

### 1. 宫限 vs 限度
- **宫限**：你当前走到了哪一个宫位 → 决定这段时间**舞台的场景**。
- **限度**：具体落到哪一颗星、顶到哪一度 → 决定**主角与戏码**。
- 同一个宫限，顶木星与顶土星，事件性质完全不同。

### 2. 星的旺衰与落宫
- 星落对位 + 入庙 / 入垣 → 力量加持。
- 星落失位 / 受克 → 力量削弱、问题显化。
- 不能只看"有没有这颗星"，要看**落在哪 + 受什么星的影响**。

### 3. 行线应期
- 七政最擅长的是**时间点精度**：
  - 某年某月行线触到某颗星 → 对应事件性质清晰。
  - 行线远离 → 能量衰退，机会窗口关闭。
- 因此适合回答：何时出手、何时避险、哪几月变动。

## 常见误区

### 1. 把七政当占星
- 七政是**命盘体系**，不是流行意义的"十二星座性格"。
- 落宫、星度、四余、节气都是传统变量，现代西洋占星完全不涉及。

### 2. 过度迷信"某星吉、某星凶"
- 没有绝对吉凶，只有**落宫 + 组合 + 当事人八字结构**综合后的**作用方向**。
- 木星入恶宫不一定好，土星入善宫不一定坏。

### 3. 想用七政代替八字做大势判断
- 大框架还是八字的强项：天赋、性格、格局层次。
- 七政拿来看大方向会**浪费它的颗粒度优势**，也容易误判。

### 4. 把七政当玄学技巧炫技
- 排盘靠的是真实天文数据，不是凭感觉。
- 没有天文基础、不理解宫度与行线 → 学再多术语也看不出应期。

## 现实表现

### 适用场景
- **重要节点的精确判断**：结婚、买房、创业、换城市的月份窗口。
- **事件性质细分**：八字难区分"做什么技术 / 什么行业"，七政通过星的象可以更具体。
- **避险窗口**：身体、财务、关系的高危月份提前知晓。

### 学习路径
1. 先把**八字基础**打好：格局、喜忌、身强身弱。
2. 再入门**七政四余排盘**：学会画盘、识别七政四余落宫与星度。
3. 理解**宫限 / 限度 / 行线**三层逻辑。
4. 通过**合参**积累感觉：同一件事，两门术数分别指向什么。

## 与八字合参的要点
- **八字定大势**：判断方向、格局、喜忌。
- **七政定应期**：判断时机、月份、事件性质细分。
- 先八字、后七政；先方向、后时机。
- 两门术数都在**某个年份同时发出信号** → 那就是**真正的分水岭节点**。

## 一句话总结
- 七政四余是**以实际天象为基础**的古代星命学。
- 它不取代八字，它是八字的**时间刻度放大镜**。
- 用八字定大势、用七政定应期 —— 把有限的能量押在**对的那几个节点**上，这是它真正的价值。
