# 学堂

## 定义
- **学堂**：**学问的殿堂**——主读书、学习、知识积累的贵人神煞。
- **起法基础**：以**日干本气（正五行）之长生位**查命中地支。
- 学堂本质是"**学问开始发芽的地方**"——长生之地 = 新生、初启、吸纳。

## 起法（按日干查"正五行长生位"）
《三命通会·论学堂》原文："**学堂者，日干长生之地也**。"

| 日干 | 五行 | 长生位 = 学堂 |
|---|---|---|
| 甲 | 阳木 | **亥** |
| 乙 | 阴木 | **午** |
| 丙、戊 | 阳火 / 阳土 | **寅** |
| 丁、己 | 阴火 / 阴土 | **酉** |
| 庚 | 阳金 | **巳** |
| 辛 | 阴金 | **子** |
| 壬 | 阳水 | **申** |
| 癸 | 阴水 | **卯** |

## 学堂 vs 正学堂 vs 词馆

| 神煞 | 起法 | 核心 |
|---|---|---|
| **学堂**（本条）| 日干正五行长生 | 学问萌芽（基础版）|
| **正学堂** | 日柱**纳音五行**长生 | 学问根基（精细版）|
| **词馆** | 日干正五行临官（禄位）| 才学显达 |
| **正词馆** | 日柱**纳音五行**临官 | 才学精微 |

**两级区分**：
- "**学堂 / 词馆**"按天干本气五行（正五行）起——粗略、基础判断。
- "**正学堂 / 正词馆**"按日柱纳音五行起——精细、进阶判断。
- 带正学堂 / 正词馆者**远贵于**仅带学堂 / 词馆。

## 核心作用
- 学堂主**学问能力、求学之缘**：
  - 读书接受快、记忆力好。
  - 对知识有天然热情。
  - 学业路上多贵人（老师、同学）。
- 对应现代：学业顺利 + 持续学习力 + 终身学习型人格。

## 判断维度

### 1. 位置
- **年支学堂**：家学渊源、祖辈重教。
- **月支学堂**：青年求学期得势——学业高峰。
- **日支学堂**：本人自带学问气场。
- **时支学堂**：晚年持续学习 / 子女好学。

### 2. 配合其他贵神
- **学堂 + 天乙贵人**：学业有师、有导师提携。
- **学堂 + 文昌贵人**：学 + 智兼备。
- **学堂 + 正学堂**：学问根基扎实——学者型典范。
- **学堂 + 词馆**：学 + 才双全。

### 3. 是否被破坏
- 学堂所在地支被冲 → 学业之路有阻。
- 被合化为异神 → 学问转向、可能不走正途。

## 常见误区

### 1. "有学堂 = 必成学霸"
- 学堂是"**学问缘分的底牌**"——需努力配合才显现。
- 古籍云："**学堂有气，方能成学**"——地支本气为学堂最佳，虚浮无根则效果弱。

### 2. 学堂等同于文昌贵人
- 学堂按**长生位**起；文昌按**食神之禄**起。
- 学堂偏"**积累与吸纳**"；文昌偏"**智慧与临场**"。
- 两者并见为"**学 + 智**"双全。

### 3. 忽视"正学堂"的精细度
- 学堂是基础判断，**正学堂**（按纳音）才是更精确的贵气标记。
- 仅有学堂无正学堂 → 学问有缘但根基一般。
- 学堂 + 正学堂并见 → 真正的学问型人才。

## 现实表现

### 性格
- **求知欲强**：对知识、书本、学习有天然热情。
- **持续学习能力**：老了还在学新东西。
- **遇事爱研究**：喜欢深挖问题而非浅尝辄止。

### 职业路径
- **学术、研究**：学者、研究员、博士。
- **教育**：教师、教授、培训师。
- **文化**：编辑、作家、翻译。
- **持续进修型**：技术专家、资深顾问。

## 核心判断
- 学堂是"**求学贵人的基础底牌**"。
- 与"**正学堂**""**文昌**""**词馆**"等配合 → 学问路径完整。
- 识别意义：让这类命造**投入学问** / **知识型工作** → 顺其天性。

## 一句话总结
- 学堂 = **知识萌芽的地方**。
- 不保成大家，但保**一辈子有学问可吸收**。
- 有学堂的人老了还在读书写字——这就是学堂的真正福气。
