# 排盘：年柱、月柱、时柱推算

## 为什么要学排盘

虽然现在大多使用软件排盘，但学习和理解年柱、月柱、时柱的推算方法仍有必要——这是**理解干支纪年规律、节气换柱、早晚子时**等关键命理问题的基础。

---

## 一、年柱的推算

### 规律总览

- **农历纪年法**：十进制。
- **干支纪年法**：天干 10 个，地支 12 个。
- 天干数量与十进制相同 → **天干规律容易**。
- 地支 12 进制 → 规律稍复杂。

### 年份尾数与天干对照

每隔十年，**年份尾数相同者其天干必然相同**：

| 年份尾数 | 天干 |
| --- | --- |
| 0 | 庚 |
| 1 | 辛 |
| 2 | 壬 |
| 3 | 癸 |
| 4 | 甲 |
| 5 | 乙 |
| 6 | 丙 |
| 7 | 丁 |
| 8 | 戊 |
| 9 | 己 |

### 掐指快速推算（以辛 = 1 为起点）

记住**辛对应 1** → 从辛开始数推其他天干对应尾数。

例：**2024 年**年份尾数为 4 → 从辛开始按顺序背诵天干（辛、壬、癸、甲、乙……），每背一个勾一个手指，勾到第 4 个时 → 停——对应**甲** → 2024 年天干为甲。

### 年柱地支推算

#### 方法一：除法公式

1. **(年份 - 3) ÷ 12**，所得**余数** = 地支序号。
2. 地支按子=1、丑=2、寅=3、……、亥=12 对照。
3. 若余数为 0 → 视为 12，对应**亥**。

**示例**：2024 年 → (2024 − 3) ÷ 12 = 168 余 5 → 第 5 位地支为**辰** → 2024 年地支为辰 → 整个年柱为**甲辰**。

**优点**：适用于任何年份，尤其对古老或遥远的年份很有用。

#### 方法二：以参考年份快速推算

**取最近一个"尾数为 0、地支为子"的基准年** → **2020 年 = 庚子年**。

#### 规律：年份往后退 10 年 → 地支提前 2 位
- 2020 庚子 → 2010 庚**寅**（子丑寅，寅在子后两位）。
- 2000 庚**辰**。
- 1990 庚**午**。
- 1980 庚**申**。
- 1970 庚**戌**。

#### 规律：年份往前进 10 年 → 地支后退 2 位
- 2020 庚子 → 2030 庚**戌**（戌亥子，戌在子前两位）。
- 2040 庚**申**。

#### 10 年内地支掐指推算（以 2020 年为起点）

从子（2020）开始，按顺序一边默背地支一边勾手指——**勾手指时要默背到第二个地支才勾第一个数**（因第一个地支子的年份尾数是 0）。

年份尾数为几就勾几个手指，被数到的相应地支即该年地支。

**示例**：2026 年 → 勾 6 个手指：子→丑→寅→卯→辰→巳→午 → 共 7 个地支，但第一个子不勾，只勾 6 → **2026 年地支为午**。

## 二、月柱的推算：年上起月法

### 规律

- **纪月法**：无论农历、公历、地支表示的月份，均为**十二进制**。
- 地支 12 个 → 月柱的**地支**在每年相同月份都是相同的（即十二月建）。
- 十二月建：寅月、卯月、辰月、……，大致对应农历正月、二月、三月……但**不完全一致**——起始日期并非初一，而是以**节令**为起始。

### 重点：找出月柱天干变化规律

月柱天干由**年柱天干**确定 → 称**年上起月法**。

### 五虎遁元口诀

> **甲己之年丙作首，乙庚之岁戊为头；丙辛之岁寻庚上，丁壬壬寅顺水流；若问戊癸何处走，甲寅之上好追求。**

### 五虎遁元对应表

| 年干 | 正月（寅月）天干 | 月柱起点 |
| --- | --- | --- |
| **甲、己** | 丙 | 丙寅 |
| **乙、庚** | 戊 | 戊寅 |
| **丙、辛** | 庚 | 庚寅 |
| **丁、壬** | 壬 | 壬寅 |
| **戊、癸** | 甲 | 甲寅 |

### 年上起月法的原理推演（四步法）

#### 第一步：确定年柱天干五合化出的五行

依据**天干五合**：
- 甲己合化**土**。
- 乙庚合化**金**。
- 丙辛合化**水**。
- 丁壬合化**木**。
- 戊癸合化**火**。

#### 第二步：找"生"合化五行的五行

- 生土 → **火**。
- 生金 → **土**。
- 生水 → **金**。
- 生木 → **水**。
- 生火 → **木**。

#### 第三步：确定当年寅月天干的五行

第二步确定的五行即当年寅月天干的五行。**寅为阳**，干支组合阴阳相配 → 天干必为阳。

#### 第四步：逐月推算

根据寅月干支，按**天干地支依次循环组合**推算出目标月柱天干。

### 示例

**甲子年、亥月**：
1. 年干甲 → 甲己合化**土**。
2. 生土的五行 → **火**。
3. 当年寅月天干五行为**阳火** → **丙** → 寅月干支为**丙寅**。
4. 亥月从寅月向后排 → 寅=1，亥=10（寅、卯、辰、巳、午、未、申、酉、戌、亥）。亥是寅后第 10 位 → 天干从丙向后数 10 位：丙、丁、戊、己、庚、辛、壬、癸、甲、**乙** → 第 10 位为**乙** → 亥月干支为**乙亥**。

## 三、时柱的推算：日上起时法

### 规律

- **纪时法**：一天十二时辰，12 进制。
- 时柱地支在每天相同时辰固定 → 重点找**天干变化规律**。
- 时柱天干由**日柱天干**确定 → 称**日上起时法**。

### 五鼠遁元口诀

> **甲己还生甲，乙庚丙作初；丙辛从戊起，丁壬庚子居；戊癸起何处，壬子是真途。**

即：
- **甲己日** → 从**甲子时**起
- **乙庚日** → 从**丙子时**起
- **丙辛日** → 从**戊子时**起
- **丁壬日** → 从**庚子时**起
- **戊癸日** → 从**壬子时**起

然后依次推：甲子 → 乙丑 → 丙寅 → 丁卯 → 戊辰……

### 日上起时法的原理推演（四步法）

#### 第一步：确定日柱天干五合化出的五行（同年上起月法第一步）

#### 第二步：找"克"合化五行的五行
- 克土 → **木**。
- 克金 → **火**。
- 克水 → **土**。
- 克木 → **金**。
- 克火 → **水**。

#### 第三步：确定当日子时天干的五行（同为阳干）

#### 第四步：逐时推算

### 示例

**丙辛日、午时**：
1. 日干丙 → 丙辛合化**水**。
2. 克水 → **土**。
3. 子为阳 → 天干为阳土 → **戊** → 子时干支为**戊子**。
4. 午在十二地支中排第 7（子、丑、寅、卯、辰、巳、午）→ 天干从戊往后数 7 位：戊、己、庚、辛、壬、癸、甲 → **甲** → 午时干支为**甲午**。

## 四、早子时 vs 晚子时

### 两种观点

#### 观点一：不分早晚子时
- 进入 23 点后就按照**第二天**计算。
- 部分软件遵循此法 → 过 23 点后日柱和时柱都按第二天排列。

#### 观点二：分早晚子时（推荐）
- **早子时**：23:00 – 24:00（接近零点）。
- **晚子时**：00:00 – 1:00（过了零点）。
- 23:00 – 零点之间：**日柱仍按当天，时柱按第二天**。
- 过了零点：日柱和时柱**都按第二天**。

### 倾向观点二的理由

事物转变都有一个**过渡阶段**，不可能一蹴而就——子时跨越了两天，正是因为存在这个过渡期。

### 但不绝对：灵活处理

- 任何事都不是绝对——23 点到零点都属子时，都处于过渡期。
- 大多数八字在此时段仍属当天，但**也有部分八字在过渡期内进展较快**，应按第二天计算才能与经历吻合。
- **遇到 23 点到零点之间的八字要特别注意**，可尝试按两种观点分别排盘后验证以往事件。

### 早晚子时的时柱推算示例

出生时间为**晚上 23 点**：
- 日柱仍是**丙辰**（保持当天）。
- 时柱推算用**第二天日柱丁巳**：
  - 丁壬合化**木** → 克木为**金** → 子为阳 → 子时天干为**庚**。
  - 时柱 = **庚子**。

### 正常时段的时柱推算
00:00 – 23:00 之间 → 日期用当天，时辰也用当天 → 按日上起时法正常推算。

## 五、月上起日法（简述）

日期的天干地支**没有固定规则**——有些月份 30 天，有些 31 天，有些 28 或 29 天，**规则十分复杂**，无法用固定规则推导。

网上有许多口诀可参考，但**现代基本依赖软件排盘**，此处不展开。

## 六、交节换柱与节气概念

### 节令与节气

- **节气**共 24 个，分两类：
  - **节令**（单数）：立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪、小寒。
  - **中气**（双数）：雨水、春分、谷雨、小满、夏至、大暑、处暑、秋分、霜降、小雪、冬至、大寒。

### 十二月建的起讫

**以节令为起始**，上一个节令结束即下一个地支月份结束：
- **寅月**起始 → **立春**；结束 → **惊蛰**（即卯月起始）。
- **雨水** = 寅月中气（月中）。
- 其余月份类推。

### 交节换柱要点

- 出生日若处于**节令前后** → 年柱、月柱可能需要**跨月/跨年**。
- 年柱以**立春**为界换年（并非农历正月初一）。
- 月柱以**各月节令**为界换月（并非初一）。

## 七、真太阳时

> 提醒：古法干支纪时是以**当地真太阳时**为基准的——现代使用区时（北京时间）时，应根据出生地经度做校正，以避免时辰定错。

真太阳时校正原则：
- **东经 120°** 为北京时间基准。
- 出生地经度每偏东 1° → 真太阳时比北京时间**快 4 分钟**。
- 出生地经度每偏西 1° → 真太阳时比北京时间**慢 4 分钟**。

对于出生时间接近时辰分界（每两小时一次）的命主，真太阳时校正**非常重要**。

## 一句话总结

排盘三步：**年柱查表或公式 → 月柱年上起月 → 时柱日上起时**。节令定换柱、真太阳时定时辰、早晚子时特别处理——**正确的排盘是八字分析的基石**。
