# 乙木

## 定义
- 天干第二位，**属阴木**。
- 五行中木主生发（这是共性），阴阳之分决定的是**形态与发力方式**。
- "乙"字本身有曲曲之意，抓的是一种**结构特征**，不是性格标签。

## 核心：曲折而生，依附而展
- 甲木 = 直冲生长，像主干。
- 乙木 = 曲折顺势，像枝条、藤蔓、草木。
- 乙木不是强硬突破，而是**顺势附着、柔中带刃**。
- 它的力量不在正面冲撞，而在**借势与渗透**。

## 十神推导（以乙木为日主）
推导依据是"阴阳同义为偏、异性为正"的规则，不靠感觉。
- 水生木 → 印星。
- 木生火 → 食伤。
- 金克木 → 官杀。
- 木克土 → 财星。
- 同类木 → 比劫。
- 阴阳关系细分：
  - 遇阴金为七杀，遇阳金为正官。
  - 遇阴水为偏印，遇阳水为正印。

## 判断维度
- 乙木要"有根、有水、有适度的邂逅与约束"，柔韧才立得住。
- 失去这些条件，所谓的柔韧只剩下**无力**。
- 与印、食伤、官杀、比劫的配置决定了表达方式，不是日主单独能说明的。

## 常见误区
1. **直接等同于"柔软好说话、容易受欺负"**
   - 这一步跳得太快。
   - 乙木的"柔"是发力方式的**曲折与顺势**，不等于软弱。
   - 藤蔓能蚕食、草根能破土。
2. **拿日主当人格全部来源**
   - "乙木女多情、乙木男顾家"一类说法的问题在于把日主五行当成全部人格来源。
   - 食伤、官杀、财印的配置，比劫是否争夺，都会改变表达方式。

## 现实表现
- 在多数结构中，乙木体现的是**一股顺势而活的生机**。
- 不擅长硬碰硬，擅长**找缝隙介入环境**。
- 如果印旺而食伤通关，表达力与学习能力都可能不错。

## 一句话总结
乙木只是日主的一种五行形态，它解释的是**气的特质**，不是人生的终局。
- 在五行里的位置、逻辑、结构中的常见落点，已经够用。
- 再往外延伸到具体成败、婚姻好坏，那是整盘八字共同作用的结果。
- **把一个天干当成全部的答案，是不负责任的。**

## 补充：乙木藤萝系甲的高格局原理

### 古籍溯源

- 《滴天髓·乙木》："**藤萝系甲，可春可秋**。"
- 《渊海子平·乙木歌》："乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。**虚湿之地，骑马亦忧**；藤萝系甲，可春可秋。"

**藤萝系甲**：乙木如同藤萝，攀附甲木（大树）而上，借其根基 → 乙木得以腾越升发。

### 结构条件

#### 条件 1：乙木必须有强根
- 地支见辰土、未土（乙木本气或余气根）。
- **辰土为佳**：辰土自带水土，与乙木相伴相生，为乙木最强根。
- 乙木无根 → 藤萝无所附，结构不成。

#### 条件 2：甲木必须位置合适
- **传统最佳结构**：年柱透甲木 + 日主乙木 → 甲木在年柱代表家境、祖荫深厚。
- **次佳结构**：月柱透甲木 + 日主乙木 → 甲木在月柱代表师长、人际关系、领导之助。
- 时柱透甲木 → 代表后辈、晚年依靠。

#### 条件 3：攀附方向必须单一
- **一乙可附多甲会降格**：乙木如藤蔓，方向必须确定。
- 若原局两甲并立 → 乙木犹豫不定，方向摇摆 → 格局降。
- 最佳：乙木明确攀附一甲，其他甲或被绝（如甲坐申金）、或被泄（如甲坐午火）、或死而可用。

#### 条件 4：食伤星（火）得用
- 乙木攀甲之后，需泄秀。
- **午火食神**最佳：午火调候，使甲木乙木生长暖和。
- 注意：午火若被水克太过 → 泄秀不足，需有适度制约使午火既为喜又不泛滥。

#### 条件 5：正官（庚 / 申金）制甲有度
- **申金为正官**：可绝甲木、使甲木成为乙木的"死木根器"。
- 甲木被申金绝 → 乙木专心攀附甲辰（活木 + 土根） → 方向确定。
- 过度克甲则无根可附 → 破格。

### 高格局的运作原理

#### 文运与学业
藤萝系甲常带高文运，原因：
1. **印星（壬水）滋润**：水生木 → 学习吸收能力强。
2. **食神（午火）泄秀**：才华外显、文采显露。
3. **甲木（贵人 / 手段）**：有强大的手段与平台。
4. **正官（申金）约束**：使能量不散漫，考试稳定。

#### 各文运星配合
- **印星 + 食神 + 正官**四颗文运星齐备 → 学历保底双一流。
- 若有一颗成病而无药 → 降格。

#### 水木土的平衡
- **辰土**：水库兼木之强根 → 调节寒暖、稳定结构。
- **水气**：滋润木，但不能泛滥到淹木。
- **午火**：温暖木气，但不能过燥。
- 三者均衡 → 藤萝系甲的清秀之气最完整。

### 高格局的兑现大运

- **火木大运**：最佳，如丙午、甲午——天干透火食神 + 地支火根 → 藤萝攀甲带文采。
- **壬水、癸水大运**：调润，适度为喜，过旺为忌。
- **庚申、辛酉大运**：制甲有度则贵，过度则破。
- **戊己土大运**：视具体——阳土（戊）制水 OK；阴土（己）污木不佳。

### 破格反面

1. **甲木孤立无根**：乙木无所攀附。
2. **两甲并立**：乙木方向不定。
3. **印星泛滥成灾**：水多木漂。
4. **食神被克重**：才华无法外显。
5. **比劫乱局**：兄弟姐妹争夺资源。

### 与其他乙木贵格的比较

| 格局 | 核心 | 气质 |
| --- | --- | --- |
| **藤萝系甲** | 附甲而上，借势成才 | 清秀、文雅、学术 |
| **乙木怀丁抱丙** | 阴木得火暖 → 外显才华 | 热烈、艺术、表达 |
| **乙木刲羊解牛** | 柔木制土（丑未） → 小刀小劲 | 精细、技术、巧思 |
| **乙木跨凤乘猴** | 得申酉官杀 → 有约束之贵 | 稳重、体制、仕途 |

### 现实表现

- **典型职业**：学者、作家、教师、文化工作者、研究员、文秘。
- **性格特征**：
  - 善于借助他人资源（甲木为贵人）。
  - 不硬碰硬，迂回曲折。
  - 对文字、思想、表达敏感。
- **成长路径**：早年得贵人提携 → 中年靠文运 / 学业立身 → 晚年桃李满园或著作等身。

### 一句话补充

- 藤萝系甲 = **找对大树再攀 + 自己的根要扎稳**。
- 关键判据：**一乙一甲 + 辰未作根 + 火来调候 + 金有度制**。

## 古典标准：乙木论命思路

### 日主五行本性
- 乙木属阴木，号"花草之木"、"藤萝之木"。
- 性情温柔、善变通、懂依附；不似甲木刚直冲撞，而以曲折借势取胜。
- 古法之形象：田里庄稼、墙角藤蔓，不能直立成梁，却能处处取活。
- 《滴天髓》总纲：**"乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。虚湿之地，骑马亦忧。藤萝系甲，可春可秋。"**

### 日主逢不同月令的状态

| 月令 | 气势 | 核心矛盾 | 取用要点 |
| --- | --- | --- | --- |
| 寅月 | 得令发生 | 初春寒气未退 | 透甲木形成藤萝系甲，兼需丙火 |
| 卯月 | 旺而当令 | 气盛喜泄 | 用火泄秀，金来可不惧 |
| 辰月 | 余气通根 | 带水气湿润 | 得根稳固，配火则贵 |
| 巳午未月 | 泄身过甚 | 火旺无水则枯 | 急需癸水调候，无水则一生无成 |
| 申月 | 七杀当令 | 金克伤身 | 借丙丁火制金，即"跨凤乘猴" |
| 酉月 | 官杀克重 | 克身更紧 | 火来救应，水破火则难 |
| 戌月 | 金库燥土 | 气衰身弱 | 以火暖局，忌土助金 |
| 亥子丑月 | 水旺寒凝 | 寒湿泡根 | 必须丙火调候，无火则废 |

### 喜用神优先级

1. **调候优先**：
   - 夏月必用癸水，无水难成大器。
   - 冬月必用丙火，无火则"一辈子废了"。
2. **扶抑次之**：
   - 身弱见甲可依，身强则以火泄秀。
3. **通关再次**：
   - 金克乙木时，以火制金（食伤制杀），不走金生水生木的印星路线，此与甲木不同。
4. **病药殿后**：
   - 夏无水、冬无火皆为重病；病而有药则格局成，病而无药则格局败。

### 与其他九个天干的关系

- **甲（阳木）**：劫财，藤萝系甲，借大树遮风避雨，四季皆固。
- **乙（同类）**：比肩，同为阴木，互相借力亦互相争资源。
- **丙（阳火）**：伤官，泄秀外放，"怀丙"则才华显。
- **丁（阴火）**：食神，细腻外显，"抱丁"则温雅。
- **戊（阳土）**：正财，乙不能克戊，戊厚反欺乙，正财旺反层次低。
- **己（阴土）**：偏财，乙可克己（但力不如甲）。
- **庚（阳金）**：正官，乙庚化金为合，阴木乐于附阳金。
- **辛（阴金）**：七杀，克身紧贴，喜丙丁火来制。
- **壬（阳水）**：正印，滋生有情，然冬月过旺则寒浸。
- **癸（阴水）**：偏印，润泽之功，夏月调候第一用神。

### 不同宫位的象意

- **年柱乙木**：祖上或家风温婉、善经营人脉。
- **月柱乙木透甲**：藤萝系甲最佳位，得师长、贵人、平台提携。
- **日支通根（卯、辰、未）**：配偶温和体贴、内秀聪明。
- **时柱乙木**：子女灵巧多才、晚景依附后辈。
- 乙木受伤之位即应身体部位（颈、胆、手腕、脚腕、经脉、头发）。

### 典型贵格组合

1. **藤萝系甲格**：日主乙木，年月或时透甲、地支有寅辰未为根——可春可秋，借势成贵。
2. **怀丁抱丙格**：乙木生秋（申酉月），天干透丙丁不被水破——才华制杀，文武兼备。
3. **刲羊解牛格**：乙木自坐丑未有根，能克柔土以取财——精巧、技术之贵。
4. **跨凤乘猴格**：乙生申酉之月而丙丁透干——体制之内有约束之贵。
5. **木火通明（阴木版）**：乙木生春，透丙火泄秀，文秀显达。

### 典型破格组合

1. **夏月无水**：巳午未月火炎土燥，四柱无水——一辈子庸碌无成。
2. **冬月无火**：亥子丑月天寒地冻，地支无巳午、天干无丙丁——废命。
3. **戊辰重财克身**：正财戊土、辰戌并立而乙木无根——身弱财多为累，层次低。
4. **丙被癸破**：秋生乙木本可怀丙抱丁，却癸水透干制丙——借火制杀失败，事倍功半。
5. **两甲并立乙无主**：藤萝无所系，方向摇摆。

### 古籍出处

- 《滴天髓·天干论·乙木》："乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。虚湿之地，骑马亦忧。藤萝系甲，可春可秋。"
- 《渊海子平·乙木歌》：详述"刲羊解牛、怀丁抱丙"之取用。
- 《穷通宝鉴·三春乙木、三夏乙木、三秋乙木、三冬乙木》：分月令定喜忌。

### 一句话总结

**乙木柔而不弱，贵在"会借势"——藤萝得甲则四季皆安，怀丙抱丁则不畏强金，然夏无水、冬无火则一生无望。**
