# 偏印格

## 成立条件

### 条件 1：月令地支本气为偏印，或偏印透干且月令有根

- **判据**：
  - ① 月令地支本气为偏印；或
  - ② 月令地支藏偏印且偏印透于天干。
- **取用要点**：偏印又称"**枭神**"——生我之同性五行、能量不如正印纯粹、但带神秘力量。
- **不满足则破**：月令偏印不显 → 当以月令本气另论他格。

### 条件 2：偏印透干且通根

- **判据**：偏印透于天干、且在地支有本气或中气根至少一位。
- **取用要点**：印透干方能生身——虚透无根则生身无力。
- **不满足则破**：偏印虚浮 → 格难立。

### 条件 3：原局无食神紧贴被偏印克（或有财护食）

- **判据**：
  - ① 命局无食神 → 偏印无夺食之事、格清；或
  - ② 有食神但偏印与食神不紧贴 → 无直接克制；或
  - ③ 有食神 + 紧贴偏印 + **有财护食**（财克偏印救食）。
- **取用要点**：**食神最怕枭神夺食**——若食神为喜用被偏印紧贴克夺、则轻则心情抑郁、重则失业多病灾。唯一化解方向是**用财制印以护食神**。
- **不满足则破**：偏印贴食无救 → 转入"**枭神夺食**"病格讨论。

### 条件 4：日主与印的力量关系决定取用方向

- **判据**：
  - **身弱用印化杀生身**：偏印可化杀成"**杀枭相生**"（层次一般低于正印化杀的"杀印相生"）；
  - **身强而印重**：印闷气机 → 必须以财制印或食伤泄身为用，非纯偏印格。
- **取用要点**：
  - 身弱 + 官众印轻：以印化官杀生身、**最忌财运**（财克印、财生杀 → 日主更弱）；
  - 身强 + 印重官轻：**喜财制印 + 生官**（财代表勤勉行动力、压制印的懒惰）；
  - 身强 + 以食伤为喜用：**最喜财来克印**（让食伤自由发挥）。
- **不满足则破**：身强 + 偏印重 → 印闷气机、须另取用。

### 条件 5：偏印不宜过重（不成枭神过旺）

- **判据**：偏印在天干 + 地支本气合计不超过两位。
- **取用要点**：**偏印宜少不宜多**——多则夺食为凶；即便无食神也会闷气机、造成多学少成、精神封闭、行动力瘫痪。
- **不满足则破**：偏印过重 → 降格或转入"印重身弱"讨论。

---

**五条同时满足**方为真偏印格。偏印格之所以复杂：**偏印本身既可为用（生身、带偏门才华）、也可为病（夺食、闭塞生机）**——关键在与食神、财、日主的关系配合。
