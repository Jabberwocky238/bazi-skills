# 三庚格

## 定义

- **三庚格**：八字天干出现**三个庚金一字排开**的特殊结构，源自盲派口诀"**三庚是财郎，万里至田庄**"。
- 属命局天干同现一气的罕见形态，由盲派口诀总结入子平命理后，归为特殊格局之一。
- 三庚为好口诀，是盲派"三字同干"系列中唯一明确主吉者；其余如三甲、三乙、三丙、三丁、三戊、三己、三壬、三癸多主孤克、疾厄、破耗，唯三庚独主财禄。

## 核心原理

- 三庚格的吉凶机关不在"三庚"本身，而在**庚金对日主的作用**：
  - 若庚金为日主之**财星或官星**（喜用神）→ 三庚为财郎，主大富贵。
  - 若庚金为日主之**比肩劫财**（忌神）→ 三庚为劫夺，格降为普通小康。
- 关键口诀原理：**三庚必须是用神且为日主的官星或财星，方论大富**；若为忌神则只是"不穷而已"，不为贵格。
- 本格的"富"之所以浓烈，在于一气聚顶：三庚集中在天干一线排开，气势贯通，做功指向极清晰——**全仗庚金这一路发力**；若这一路恰为日主所用，则力如千钧。

## 成立条件（古籍 + 客观判据）

### 条件 1：天干三位庚金一字排开

- **古籍出处**：
  - 《三命通会·论三庚》："天干三庚连排，号为三庚之格。"
  - 盲派口诀："三庚是财郎，万里至田庄。"
- **客观判据**：
  - 天干四位中，**至少有三位为庚金**；格局最纯者三庚连续相邻（年月日干、月日时干皆可）。
  - 隔一位但仍有三庚者亦算成格，但贵气略逊。
- **不满足则破**：仅两位庚金 → 不构成三庚格，按普通比劫论。

### 条件 2：庚金为日主之喜用（财星或官星）

- **古籍出处**：
  - 《三命通会·论三庚》："三庚得用，富贵之征；三庚失用，徒劳而已。"
- **客观判据**：
  - 判断日主与庚金的十神关系：
    - **丙丁火日主** → 庚金为财（丙以庚为偏财，丁以庚为正财）→ **三庚为财郎**，主富。
    - **甲乙木日主** → 庚金为官杀（甲以庚为七杀，乙以庚为正官）→ 三庚为官贵，主贵或主文武双全。
    - **戊己土日主** → 庚金为食伤（泄秀）→ 非财非官，不论大富大贵。
    - **庚辛金日主** → 庚金为比肩劫财（忌神）→ **破格**，只论小康，不论大富。
    - **壬癸水日主** → 庚金为印 → 主学业 / 文书 / 母荫，不主大富。
- **不满足则破**：庚金非财非官 → 不构成"三庚是财郎"之吉象，退为普通格局。

### 条件 3：三庚不被强火所克（庚金须有作用力）

- **古籍出处**：
  - 《三命通会·论三庚》："庚逢火制，三庚减威。"
- **客观判据**：
  - 原局地支不得见强旺午火或丙丁火透干贴克三庚（日主为丙丁火之财格除外，此时丙丁克庚为"我克财"，反为日主得财）。
  - 若三庚被强火制伏且此火非日主本气 → 庚金失用，格破。
- **不满足则破**：三庚受强克无救 → 财官之象受损。

### 条件 4（专论甲乙木日主）：甲木须有根以任官杀

- **古籍出处**：
  - 《子平真诠·论官杀》："身轻杀重，为杀所克；身旺杀强，方为富贵。"
- **客观判据**：
  - 甲乙木日主遇三庚，三庚为三重官杀，对日主压力巨大。
  - 日主必须**地支通根**（寅、卯、辰、未、亥等）或有印星化杀，方能任此三重官杀。
  - 若甲乙木日主无根又无印 → **从杀格**论，另行处置；若半根半无 → 破格。
- **不满足则破**：甲木无根 → 不能任杀 → 三庚反为伤身之物。

---

**条件 1、2 是成格核心，条件 3、4 是保底。** 三庚格的判断逻辑极简：**先看有没有三庚，再看庚是不是喜用**——这两步决定一切。

## 与相关格局的对比

- **三庚格 vs 比劫重重**：
  - 比劫重重指日主自身五行占三位以上，庚辛金日主的三庚即属此类，主破财、争夺。
  - 三庚格专指**庚金作为财或官**对于日主的吉象；与日主同为金时反退回比劫重重论。
- **三庚格 vs 食神制杀**：
  - 食神制杀是甲乙木日主以食伤制七杀庚金的格局，须**一杀一制**、杀不过旺。
  - 三庚格中甲乙木日主面临的是"三杀齐发"，需印化或身旺任之，非食伤所能独制。
- **三庚格 vs 从杀格**：
  - 甲乙木日主无根 + 三庚当令 → 从杀格。
  - 甲乙木日主有根 + 三庚 → 三庚格（官杀旺但身能任）。
- **三庚格 vs 财多身弱**：
  - 丙丁火日主无根 + 三庚 → 财多身弱（财重身轻）。
  - 丙丁火日主有根 + 三庚 → 三庚格（财郎之象）。

## 判断维度

### 1. 庚金与日主的十神关系
- 第一步也是最关键一步：**庚金是不是日主的财或官**？
- 只有丙丁火、甲乙木日主遇三庚，才有资格讨论"大富大贵"。

### 2. 日主强弱
- 日主任得住三庚之财或官，格局才稳。
- 日主过弱 → 财官反为累 → 退一档。

### 3. 三庚位置
- 三庚连续相邻 > 三庚隔位。
- 位置越紧贴，一气贯通力越强。

### 4. 三庚是否受克
- 强火克三庚（非日主本气之火）→ 格损。
- 若是日主自身之火克庚（丙丁火日主克财）→ 为"我克财"之正象，反吉。

### 5. 大运流年
- 走财星得地 / 庚金根地 → 三庚兑现。
- 走强火克庚（非日主所需）→ 格破。

## 现实表现

### 当三庚为喜用时
- **丙丁火日主**：财路宽广，白手起家而能聚财；中年以后多有横发之象。
- **甲乙木日主**（身能任杀）：**文武双全、有杀伐决断**；多见于武官、企业家、执法者、手握重权者。
- 人格气质：刚猛中带决断，行动力强，对目标极为专注。

### 当三庚为忌神时（庚辛金日主）
- **普通小康**之命，不富不贵但也不至于穷。
- 多从事劳力或同业竞争激烈的行业，一生辛劳。
- 人际上易遇竞争、分夺、合伙散伙。

## 常见误区

### 1. 见三庚即论大富
- 错。必须先看庚金是不是日主之喜用。庚辛日主见三庚是忌神叠加，反非吉事。

### 2. 以为三庚必须连续
- 理想是连续相邻，但隔一位仍有三庚者也算成格，只是贵气略减。
- 格局判断时不应因"隔一位"就全盘否定。

### 3. 忽视日主承载能力
- 甲乙木遇三庚必须有根或有印，否则"三杀齐克"反伤身，连普通格局都不如。

### 4. 把盲派口诀当作包治百病
- 盲派口诀"三庚是财郎"重结论不重推导；子平命理须补足"庚金与日主关系、日主强弱、克神有无"这几步分析。

### 5. 自我暗示硬套
- 不要因自己命中有三庚就先入为主认定自己是大富之命。**假的真不了**——口诀是概率总结，具体命局要具体分析。

## 破格与救格

### 破格的典型情形
- 庚金为比劫（庚辛日主）→ 天然破格。
- 甲乙木日主无根无印 → 不能任三杀。
- 丙丁火日主极弱无根 → 财多身弱，反受其累。
- 三庚被强火贴克且非日主所需 → 庚金失用。

### 救格的可能路径
- **身弱遇三庚**：大运走日主根地或印运 → 该运限内格局兑现。
- **三庚为官杀重**：原局有印化杀 → 转入**杀印相生**论，亦为贵格。
- **三庚为财重身弱**：原局有比劫帮身 → 转入身弱用比劫格，可得小富。

## 大运流年

| 大运 / 流年 | 庚金为喜用 | 庚金为忌神 |
| --- | --- | --- |
| 庚金根地（申酉戌丑） | **大喜** → 财官兑现 | **忌** → 劫财加剧 |
| 财星得地（日主财之根） | **喜** → 财库开启 | 中性 |
| 印运 | 身能任财官，助兑现 | **喜** → 印化比劫 |
| 强火克庚（非日主本气） | **大忌** → 三庚受伤 | **喜** → 克去忌神 |
| 食伤运 | 泄身助财或泄身生财，看结构 | 泄忌 → 中喜 |
| 冲破庚金根地之支 | **忌** → 财官根摇 | 中性 |

## 一句话总结

- **三庚是财郎，万里至田庄**——但前提是**庚金必须是你的财或官**。
- 若庚金是你的比劫 → 三庚不过是三兄弟分饭，普通小康而已。
- 判三庚格先问一句：**"庚金于我是什么十神？"**——这一问决定大富与平淡的分野。
