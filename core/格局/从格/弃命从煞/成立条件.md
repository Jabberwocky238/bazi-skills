# 从杀格成立条件

### 条件 1：日主极弱，无根无柱
- 跟从财格一样：天干周围无比印，地支四位**本气、中气、余气皆无比印**。

### 条件 2：月令必须是官杀
- 要么月令本气即为正官或七杀；
- 要么地支月令**参与三合或三会官杀局**。

### 条件 3：官杀数量上的优势
- 天干地支里**官杀数量 ≥ 财星**；
- 且**官杀数量 > 食伤**。
- **可以有食伤，但必须很弱**，而且最好有财星来通关（食伤生财、财生官杀），这样就不会食伤克官杀了。

### 条件 4：不能有印比来捣乱
- **无印**：印会化官杀 → 破从。
- **无比劫**：比劫会抗官杀 → 破从。

### 古籍出处
- 《滴天髓·从象》（任铁樵注）："**日主孤立无气**，财官强甚，乃日主顺从财官之势也。"
- 《滴天髓·从象》："**若有一毫印绶，即不为从；比劫一见，即不为从**。"
