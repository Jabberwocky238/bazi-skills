# 自坐衰

## 定义
- **衰**：十二长生的第六阶段，象征**壮年过后、老年开始** —— 能量从顶峰**开始回落**。
- **自坐衰**：**日主天干坐在衰位地支上** → 日主处于**从旺转衰的过渡期**。
- 各天干衰位：
  - **甲木** 衰在 **辰**。
  - **乙木** 衰在 **丑**。
  - **丙戊** 衰在 **未**。
  - **丁己** 衰在 **辰**。
  - **庚金** 衰在 **戌**。
  - **辛金** 衰在 **未**。
  - **壬水** 衰在 **丑**。
  - **癸水** 衰在 **戌**。

衰位多为**辰戌丑未**四个杂气土库。

## 典型日柱
- **甲辰、乙丑、丙未、戊未、丁辰、己辰、庚戌、辛未、壬丑、癸戌**。

## 核心：退意、守成、老成持重
- 衰 = **能量开始下降** → 不再年轻气盛。
- 自坐衰的人：
  - **外表看似平和，内里已经"老成"**。
  - **不爱冲动、偏守势**。
  - **智慧高于热情**：**知道什么能做、什么不能做**。
  - 带**"早熟"气质**：年纪不大但心已老。

## 与相关结构对比

| 阶段 | 能量 | 气质 |
|---|---|---|
| **帝旺** | 顶峰 | 刚烈 |
| **衰**（本条） | 开始回落 | 老成、守势 |
| **病** | 虚弱 | 多思、内耗 |
| **死** | 最低 | 归零 |
| **墓** | 归藏 | 深藏不露 |

**衰 → 病 → 死 → 墓** 是"**下降四阶段**"，衰是其开始。

## 判断维度

### 1. 衰位多为杂气土
- 衰位基本都在**辰戌丑未**：
  - 是土库、也是杂气。
  - 土库承载 + 杂气混合 → 衰的气**内敛但不简单**。
- 看**藏干与日主的关系**决定衰的具体质感：
  - 藏干是喜用 → 衰中有贵（如甲辰，辰中癸水为印，有生扶）。
  - 藏干是忌神 → 衰加耗。

### 2. 身的整体强弱
- **身强 + 自坐衰**：气势下降但根基还在 → 老成稳重。
- **身弱 + 自坐衰**：本已弱 + 再衰 → 易被动、易受外界影响。

### 3. 大运流年
- **长生 / 禄 / 帝旺运**：回补气势 → 重回上升。
- **病 / 死 / 墓运**：继续下滑。
- **冲衰位的大运**（辰戌冲、丑未冲）：**关键节点**。

## 常见误区

### 1. "自坐衰 = 倒霉"
- **错**。
- 衰只是能量回落，不等于倒霉。
- 自坐衰的人**往往比同龄人更成熟 / 更清醒** → 反而少走弯路。

### 2. 把"衰"当"衰败"
- 衰是"**转折点**"，不是终点。
- 衰之后是病、死、墓 —— 但也可能因大运而回升。
- 命理的"衰"是**周期意义**，不是不可逆的"衰落"。

### 3. 以为自坐衰 = 事业不顺
- 衰的人**不善冲锋**，但：
  - **善于守成、善于规划**。
  - 在需要**稳定 + 策略**的岗位反而吃香。

### 4. 忽视衰的"先知感"
- 自坐衰的人**对"风险"的感知特别敏锐**：
  - 别人看到机会 → 他们先看到风险。
  - 别人冲上去 → 他们退一步。
- 这种"先知感"**是成熟的表现**，不是消极。

## 现实表现

### 性格
- **老成持重、不浮躁**。
- **喜静不喜动**：内向型倾向。
- **思考多于行动**。
- **对风险敏感**：防御意识强。
- **不爱社交**：朋友少但都质量高。
- **带**"先知**" 气质**：看得到事情的终局。

### 事业
- **适合"守成型"**岗位：
  - 管理 / 运营 / 维护型工作。
  - 风险控制、合规、审计。
  - 传统行业、老字号。
  - 政策研究、学术研究。
- **不适合"冲锋型"**岗位：创业前线、激烈销售、快速扩张。
- **典型画像**：**稳健型管理者、深度研究者、守护型专业人士**。

### 成长节奏
- **年轻时就"老成"**：少年老成的典型。
- **中年稳定**：不暴富但不亏。
- **晚年反而可能爆发**：长期积累 + 大运回升（禄 / 帝旺运到来）。

### 感情
- **不热情但深情**：不会满腔热血去追，但一旦认定就守。
- **偏向成熟型伴侣**。
- **婚姻稳定**：不会轻易动摇。
- **关系中带"风险意识"**：对方做什么会影响关系，他 / 她能预判。

### 身体
- **看似平稳实则需要养**：
  - 气血开始回落的阶段。
  - 容易慢性疲劳、亚健康。
  - 不会突然生大病，但**日积月累易出问题**。
- **一辈子注意养生** → 可以长寿。

### 心理
- **带"中年气质"**：哪怕是年轻人也显得成熟。
- **对"无常"有深刻认知**。
- **不易被煽动 / 不跟风**。
- **容易"过度谨慎"** → 错失某些机会。

## 自坐衰 + 十神的叠加

| 日柱 | 组合 | 含义 |
|---|---|---|
| **甲辰** | 甲坐财库（辰中戊）+ 衰 | 财在衰位 → 守财型 |
| **乙丑** | 乙坐财库（丑中己）+ 衰 | 同上，更内敛 |
| **丙未 / 戊未** | 火土坐伤食库 + 衰 | 才华沉淀期 |
| **丁辰 / 己辰** | 火土坐伤食库 + 衰 | 早熟型创作者 |
| **庚戌** | 庚坐印库（戌中戊）+ 衰 + 魁罡 | 稳中带刚、深藏智慧 |
| **辛未** | 辛坐印库（未中己）+ 衰 | 秀中带深、守成型 |
| **壬丑** | 壬坐杀库（丑中己）+ 衰 | 智慧中带约束感 |
| **癸戌** | 癸坐官库（戌中戊）+ 衰 | 有规矩 + 有深度 |

## 大运流年影响

| 大运 | 效果 |
|---|---|
| 长生 / 禄 / 帝旺运 | **喜** → 回补气势 → 晚发 |
| 印 / 比劫运 | **喜** → 扶身 → 守得住 |
| 食伤运（适度） | 视身强弱 → 身不弱则可泄秀 |
| 财官运（有通关） | 视结构 → 身能任则为喜 |
| 冲衰位的大运 | 关键节点 → 看冲开的是喜是忌 |
| **衰 / 病 / 死运再叠** | **忌** → 双重下降 → 低谷 |

## 核心判断
- 自坐衰 = **身在下降期但不到最弱**。
- 识别意义：
  - **承认自己的"守势"本能**是智慧不是懦弱。
  - **不强行模仿外向 / 冲锋的风格** → 那不是你的。
- 成就路径：**守稳 + 策略 + 积累**。
- 破局情形：**在快节奏竞争中被逼冲 → 反而一败涂地**。

## 一句话总结
- 自坐衰 = **你天然带着一股"已见过风浪"**的气。
- 年纪不大但看得远、动作不多但步步为营。
- 这类人**不拼速度拼耐力** —— 笑到最后的常常是这种底色。
