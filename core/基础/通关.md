# 通关

## 定义
- **通关**：在五行相克的两股力量之间，**插入一个桥梁五行**，把"直接冲克"**转化为"顺生流通"**。
- 不是十神格局，而是**命理结构调整的基本方法论**。
- 适用于所有五行相克的命局，是**救命局、化病象**的核心手段之一。

## 核心：把"克"变成"生"
### 五行相克 → 通关 → 五行相生
| 冲克关系 | 通关五行 | 结果链 |
|---|---|---|
| **金克木** | 水 | 金 → 水 → 木 |
| **木克土** | 火 | 木 → 火 → 土 |
| **土克水** | 金 | 土 → 金 → 水 |
| **水克火** | 木 | 水 → 木 → 火 |
| **火克金** | 土 | 火 → 土 → 金 |

通关五行 = **被克者的"印"**（生被克者之物）：
- 水是木的印 → 金克木，靠水通关。
- 火是土的印 → 木克土，靠火通关。
- 以此类推。

## 为什么需要通关
- **相克 = 直接冲突**：能量碰撞、两败俱伤、动荡。
- **相生 = 流通连接**：能量传递、层层滋养、稳定。
- 通关的本质是"**把对立关系改为合作关系**"。
- 命局有冲克而无通关 → **病象**（如水火相战、金木交战）。
- 命局有冲克且有通关 → **贵格**（如水火既济、斧斤伐木的救应）。

## 通关的四种层次

### 1. 命局通关（先天）
- 命中**本来就有**通关的五行。
- 最稳、最持久。
- 例：原局水火两旺 + 有甲乙木 → **水火既济**自然成立。

### 2. 大运通关（后天补）
- 原局缺通关五行 → 走到对应大运时**临时通关**。
- 有效期 10 年。
- 例：原局金木交战无水 → 走水大运的十年 → 金白水清变通、木气得养。

### 3. 流年通关（短暂）
- 流年出现通关五行 → 当年问题缓解。
- 有效期 1 年。
- 例：水火相战的原局 → 流年甲子（甲木通关）→ 那一年比平常顺。

### 4. 人为通关（后天干预）
- 命局本无 + 大运流年也不给 → **主动制造通关**：
  - **环境调候**：住所、工作、城市选择相应五行的方位。
  - **人事配合**：找与通关五行对应的伴侣 / 合伙人 / 导师。
  - **行业选择**：选择通关五行属性的行业。
  - **物品 / 颜色**：风水象意上的辅助。
- 效果有限但**可持续**。

## 通关五行的识别要点

### 1. 两股冲克力量**势均**
- 一强一弱 → 不叫冲克，是"**偏克**" → 通关不适用。
- 势均才是真的"战" → 才需要"通关"来和解。

### 2. 通关五行**有力**
- 通关者必须：
  - 透干 / 有根。
  - 不被克制。
  - 位置在两股力量之间（或能覆盖两者）。
- 虚透的通关不起作用 = 形同虚设。

### 3. 通关**有位置**
- 通关五行在**两冲克的中间位置** → 效率最高。
- 两端 / 对角 → 力量传递不畅。

### 4. 通关五行**不被破**
- 若通关五行**被其他五行克**（例如水通关被土重克）→ 通关失败。
- 通关本身也需要"保护"。

## 常见误区

### 1. "有这个五行就能通关"
- 必须**真正起桥梁作用**。
- 数量不等于作用。

### 2. 把通关等同"用神"
- 通关五行常常是用神，但不一定。
- 用神是对**日主**而言。
- 通关是对**两股冲克力量**而言。
- 有时通关五行对日主不利，但对冲突双方有利 → 此时要取舍。

### 3. 忽视通关的脆弱性
- 大运流年可以**打掉通关**：
  - 通关被冲 → 通关失效。
  - 通关被合 → 通关转移。
- 原本通关成立的命局 → 一个冲合就破。

### 4. 以为"通关越多越好"
- 通关五行过多 → 反而让整个命局"**软"**：
  - 原有的刚性与张力被抹平。
  - 成就空间也变小。
- 适度的冲克 + 适度的通关 → 最理想。

## 通关的典型应用

### 1. 水火相战 → 木通关
- 原局水旺 + 火旺 + 无木 → **水火相战**（病象）。
- 加入甲乙木 → **水生木、木生火** → **水火既济**（贵格）。
- 木是桥梁 → 水的智慧借木变为火的行动力。

### 2. 金木交战 → 水通关
- 原局金旺 + 木旺 + 无水 → **金木交战**（病象）。
- 加入壬癸水 → **金生水、水生木** → 和平共处。
- 水是桥梁 → 金的锋利借水变为木的生机。

### 3. 木土相克 → 火通关
- 原局木旺 + 土旺 + 无火 → **木克土**（病象）。
- 加入丙丁火 → **木生火、火生土** → 通畅。
- 火是桥梁 → 木的生发借火变为土的承载。

### 4. 土水相争 → 金通关
- 原局土旺 + 水旺 + 无金 → 土水争衡。
- 加入庚辛金 → **土生金、金生水** → 和解。
- 金是桥梁 → 土的稳固借金变为水的流动。

### 5. 火金对立 → 土通关
- 原局火旺 + 金弱 → **火旺金衰**（病象）。
- 加入戊己土 → **火生土、土生金** → 保全金。
- 土是桥梁 → 火的热力借土变为金的凝聚。

## 通关与"五行流通"的关系
- **通关** = 局部的两股力量调和（一条小桥）。
- **五行流通** = 五行全部串连（金 → 水 → 木 → 火 → 土 → 金）（大循环）。
- 通关是**流通的最低形态**。
- 通关做得好 → 有可能升级为全局五行流通 → 顶级贵格。

## 生活层面的通关智慧

### 情绪层
- 感觉被某个情绪压住 → 找"**通关的情绪**"：
  - 焦虑（火）→ 土的稳（规律作息、物理落地）。
  - 愤怒（木）→ 火的转化（运动、创作、表达）。
  - 悲伤（金）→ 水的流动（倾诉、写作、时间）。

### 人际层
- 两个朋友 / 同事冲突 → 当中间人 → **做桥梁** = 通关之人。
- 合作失败 → 找第三方 → 通过引入新力量调和原冲突。

### 事业层
- 两个对立目标（稳定 vs 创新）→ 找中间方案：
  - 在稳定里留创新空间（土里种木）。
  - 在创新里保稳定核心（木带土根）。

### 心理层
- 两极思维 → 找"**既是也是**"的第三种可能。
- 通关 = **超越"非此即彼"的智慧**。

## 核心判断
- 通关是命理最**优雅**的结构：
  - 把敌对 → 变成合作。
  - 把毁坏 → 变成创造。
- 会看通关的人，能从"**看起来很糟**"的命局里**读出贵气**。
- 真正的命理高手 = 看病象**能判断出通关在哪里**、**有没有**。

## 一句话总结
- 五行相克不可怕，**怕的是没有桥**。
- 通关 = **在对立的两者之间找到第三个**。
- 命理层的通关是五行，人生层的通关是**智慧、包容、转化的能力**。
- 懂通关 —— 把所有看似的"冲突"都变成"机会"。
