# 七政四余入门

## 概述

七政四余作为古代皇室专属的测算体系，对出生时间要求极高——**十分钟的误差都会影响星体与宫位**，导致断事差约一年。精准根源在于它用**真实天体运行**锚定事件预测。

核心框架：
- **七政**（七颗实体星）+ **四余**（四颗虚星）
- 加上**星宿、干支、宫位**
- 整体结构是"天-地-人"层层嵌套——**天垂象、地成气、人应势**

## 一、盘面三大结构：天盘、地盘、人盘

### 天盘——天上的星

**七政（七颗实体星）**
- 日、月、木、火、土、金、水
- 对应最外圈标注的字

**四余（四颗虚星）**
- 由太阳、地球、月亮运行产生的焦点，不是真实存在的星
- 是其对应实星的"余奴"
- 五行属性与主星相同

| 余星 | 主星 | 五行 |
|---|---|---|
| **紫气** | 木星 | 木 |
| **罗睺** | 火星 | 火 |
| **计都** | 土星 | 土 |
| **月孛** | 水星 | 水 |

**特别提示**：金星没有余奴星（因金星本就有肃杀/特别明亮之意）。

**二十八宿**
- 从**角宿**开始逆时针首尾相连成圆环：角亢氐房心尾箕斗牛女虚危室壁奎娄胃昴毕觜参井鬼柳星张翼轸
- 对应天干从"甲"到"癸"
- 星宿分五行，盘面用不同颜色区分

### 地盘——十二地支

- 最中间一圈小字：子丑寅卯辰巳午未申酉戌亥
- **永远静止**（"地承接天盘能量，划定空间范围"）
- 可与后天八卦结合使用
- 五行属性盘面直接标注

### 人盘——十二宫

**人立于天地之间**，人出生一刻秉承天地之气立命。从命宫顺排十二宫：

命宫 → 财帛 → 兄弟 → 田宅 → 男女 → 奴仆 → 夫妻 → 疾厄 → 迁移 → 官禄 → 福德 → 相貌

- 主宰人在社会上的际遇
- 地盘十二宫与人盘十二宫**联动**

### 最外圈：神煞

- 东方术数独有的语言（除五行生克之外）
- 画像、细判吉凶非常好用
- **初学者可先不关注**

## 二、动位大限：时间的流转

- **大限**按人事宫顺排（顺时针）
- 每宫有固定年限，决定长期运势趋势

**典型宫位年限**

| 宫位 | 年限 | 作用 |
|---|---|---|
| 命宫 | 15 年 | 出生到青年期整体运势 |
| 相貌宫 | 10 年 | 外在形象、人际交往；青年期情绪波动大 |
| 官禄宫 | 15 年 | **中年黄金事业期**，看事业运重点 |

**流年**
- 最外圈标注的年份
- 看运势需结合：**大运宫位 + 二十八宿度数 + 流年星曜**
- 大运走官禄宫 + 宿度通畅 + 流年星体给力 → 事业突破
- 遇忌刑或宿度不顺 + 宫内环境差 → 提前防备

## 三、七政核心星体详解

### 日（太阳）——君王之威

- 主**核心运势、健康、领导力**
- 主"贵气"——狐假虎威，有它罩着就得好处
- **运行节奏**：一天约 1 度，一月走一宫，一年绕一圈（**小周天**）
- 太阳在官禄宫 → 事业易突破
- 选日子（婚丧嫁娶、盖房）最好能对到它的方位——**锦上添花**

### 月（太阴）——母仪之温

- 主**情绪、感情、母亲**
- 水之精华，阴之精华
- **运行节奏**：一天约 13 度，两天半走一宫，一月绕一圈
- 情感波动变化快
- 月亮离土星近的阶段 → 情绪容易低落
- 更像"妈妈"，也代表内心情绪

### 木星（岁星）——机遇与成长

- 主**机遇、事业成长、文化、学问**
- 曲木之象、生发之象
- **运行节奏**：一年走一宫，12 年绕一圈（地支生肖对应木星轨迹）
- 木主仁，跟文化学问近
- 木星顺行且有利 → 考试易有好结果
- 逆行 → 相关领域易出麻烦

### 火星（荧惑）——意外与挑战

- 主**挑战、意外、行动力**
- 急躁，又名"荧火星"（古代"荧惑守心"常为灾祸征兆）
- **运行节奏**：约 2 年绕一圈
- 火星在财帛宫来生命 → 突然横财
- 火星被克/被泄 → 突然花大钱
- 核心关键词：**意外、快速**

### 土星（镇星）——沉稳与转折

- 主**长期规划、重大转折、沉稳守旧**
- 走得最慢
- **运行节奏**：约 30 年绕一圈
- 对应人生 30 岁转行、60 岁一甲子等**重大转折节点**
- 伤害是**持续性、慢性**的

### 金星（太白）——收敛与吸引

- 主**艺术、品味、情爱、合作、金融**
- 有吸引凝聚之象
- **运行节奏**：一月走一宫，一年绕一圈（与太阳时间接近）
- 主收敛
- 怕走太快 + 倒走 + 来克 → 金星"杀气"外显，易出大麻烦

### 水星（辰星）——沟通与流动

- 主**沟通、表达、逻辑思维、传播、交通、流动**
- 水主智
- **运行节奏**：一天 1.5 度，不到一月走一宫，不到一年绕一圈
- **逆行**（水逆）：地球视角下的视觉现象，不是真实逆行
- 水星为所用又被克 → 反应慢、流动相关出问题（如交通）

## 四、四余星的含义

| 星 | 属性 | 典型含义 |
|---|---|---|
| **紫气** | 木 | 贵人运；也有说法主孤独 |
| **月孛** | 水 | 突发变化 |
| **罗睺** | 火 | 阻碍、转折 |
| **计都** | 土 | 阻碍、转折 |

**罗睺、计都**是人生坎坷的重要参考点——很多人生的坎都跟它们有关。

## 五、入门四步法

### 第一步：找到命和身

- 盘面右上角/中心标注命度位置
- **命宫落红线处** = 命度
- 命度主星是解读的关键
- 示例：命度在申水宿 → **命度主是水星**
- 身度主同理
- **命度主、身度主的五行与强弱 → 决定整体格局**

### 第二步：判断四时喜忌

- 星体能量大小需结合**季节/月令**判断
- 类似八字的"旺相休囚死"
- 七政四余里总结为**"四时论"**——不同季节星体喜忌倾向明确

### 第三步：星-宫-宿度联动

- 熟悉后结合**神煞**
- 通过天象 → 人世十二宫的象意 → 断事

### 第四步：运用大运与流年

- 通过命主星、命度主看喜忌
- 通过动位大限看长期趋势
- 通过流年星曜看当年吉凶

## 误区

- "星盘神秘看不懂" → 只要抓住"天-地-人三盘结构 + 星曜基本属性"，读盘是**抽丝剥茧**
- "七政四余 = 西方占星" → 底层是中国传统天文，用的是二十八宿、十二地支、十天干
- "一步登天" → 不可能，**先打基础、再深化**，按步骤即可上手

## 一句话总结

> **七政四余 = 七颗实星 + 四颗虚星 + 二十八宿 + 十二地支 + 十二宫 + 神煞——天垂象、地成气、人应势，层层嵌套决定一个人的命盘。掌握三盘结构与星曜属性，新手小白也能快速入门。**
