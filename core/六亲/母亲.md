# 母亲

> 参见：`十神/印星.md`、`十神/正印.md`、`十神/偏印.md`、`六亲/六亲总论.md`

## 一、核心对应

- **十神**：母 = **正印**（主）/ 印星（广义）
- **宫位**：年柱（先天家庭 / 祖辈母系）、月柱（主要成长期的母亲）
- **作用方式**：母亲 = 日主**生命来源中的养育、保护之气**

## 二、为什么"印星 = 母"

- 印的本质 = **生我之物**，让日主得以存在、成长。
- 人初生时最直接提供养分与庇护的，往往是母亲。
- 故印星自然带母象。

但**印星代表的是"养育关系"而非"血缘关系本身"**：

- 若成长主要由祖辈抚养 / 父亲承担照料 → 印星的母象会转移或淡化。
- **正印**：纯粹的养育之爱、保护、学习、传承。
- **偏印**：不情不愿的生，母性较薄，易为继母 / 养母 / 祖辈代养之象。

## 三、宫位判断

### 年柱（先天根基）

- **年柱稳固 + 印星透干为喜** → 母亲出身好、家庭教养佳、早年受宠。
- **年柱受冲破** → 成长环境波动，母亲位置不稳。
- **年干正印为喜，多出生于书香世家**——母亲有文化、有社会地位。
- **年支正印** → 母亲宫同位，母德出众，兄弟姐妹多容貌端正。

### 月柱（主要养育期）

- **月干正印为喜、未遭财破** → 学业有成、受母亲教导多、聪慧仁慈。
- **月支正印为喜** → 智慧型人才、善于布局。
- **月支正印为忌** → 多受父母宠溺，依赖心强。
- **月支正印 + 月干正官** → 官印相生，母亲贤能、家教良好。
- **月干伤官克月支印** → 与母亲意见不合、常有矛盾。

### 日支（内在影响）

- **日支正印** → 日主心性仁厚、夫妻相处平淡、配偶母性强。
- **日支偏印** → 易想多、结婚后操劳、对母亲既依赖又有距离感。

## 四、母亲健在 / 缘分 / 身体的判断

| 命局结构 | 象意 |
| --- | --- |
| 正印为喜 + 有官杀来生 + 无财星破 | 母亲健康、与母缘深、助力大 |
| 正印为忌 + 旺而无制 | 母亲溺爱 / 控制，命主易依赖 |
| 印星虚弱无根 + 财星多旺 | 财破印 → 母亲操劳、身体易弱 |
| 印星遇冲（年月） | 母亲健康 / 居所有变动 |
| 偏印为主 + 正印不见 | 早年非亲母养大，或与母聚少离多 |
| 枭神夺食（月支偏印克食神） | 母亲对命主造成隐性压力，易致命主忧郁 |
| 印星入墓 | 母亲晚年隐居、身体渐弱 |

## 五、母亲对命主的影响

### 印星的多重面向

除养育外，还含：**保护、包容、学习、传承、名誉**。

- 母亲表现为**保护者** → 印星贴合。
- 母亲性格强势、以管束影响子女 → 命局有时反在**官杀**里见母象。
- 母亲承担了部分父职（单亲家庭 / 父亲缺位） → 印与财同时带母象。

### 成熟的三步判断

1. 看命局中**哪股力量在滋养日主**。
2. 再看这种滋养**来自哪里**（哪一柱、哪一宫）。
3. 最后结合**宫位气势**确认这股力量是否就是"母亲"。

| 岁运组合 | 常见应事 |

| 枭神夺食流年 | 易抑郁、食欲不振、与母有口角 |

## 七、古籍补充（作为真实标准）

- **正印在年干为喜用，多出生于书香世家、富贵之家**——印为文昌学术之星、符寿之星。
- **年支正印 → 兄弟姐妹大多容貌端正，理智口才佳**——母德通过子女品行显化。
- **月干正印为喜用，未遭财星克破 → 学业有成，文采出众，聪慧仁慈**。
- **月支正印为忌 → 多受父母宠溺，易养成不思进取、华而不实、依赖心强的个性**。
- **偏印为凶神，亦称枭神；月支偏印 + 月干食神者，一生疾病缠身，常遭厄难**——枭神夺食之凶象，尤与母亲位置有关。
- **日支正印 → 男命配偶母性较强，贤良有助力**——母象与妻象在此重叠。
- **偏印在年干为祭 → 幼年较为孤单，缺少双亲呵护，通常并非亲生父母养大**（留守、祖辈代养）。

## 八、现实中的多种模式

- 母亲如一层保护壳，始终在日主周围（印星为喜、强而有根）。
- 母亲只在早年出现，之后影响减弱（印星在年柱、月柱不见）。
- 母亲的力量被其他家庭成员替代（印星不透、祖辈或父亲承担印象）。
- 母亲强势如父（印星克泄组合 / 印与官杀同位）。

## 十、一句话总结

**母亲 = 正印为主、年月为位、养育之气为魂——印旺为喜则得母之助，印弱被破则母身易忧；母象不在印时，须回头追问"是谁在滋养日主"。**
