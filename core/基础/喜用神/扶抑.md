# 扶抑取用法

## 定义
- **扶抑取用法**（又称**扶抑法**、**服役取用法**）：以**日干**为核心、以日干为太极点，根据**日干旺衰**进行**扶弱抑强**，从而找出用神的取用方法。
- 是徐乐吾先生提出的五种取用神方法（扶抑、病药、调候、专旺、通关）之一。
- 在五种取用法中，扶抑法**应用范围最广**、**适用命局最多**。

## 两个核心概念：扶 与 抑

### 扶
- 含义：**生扶、帮扶**。
- 用时机：日干**较弱**。
- 具体方式：用**印星生扶**或**比劫帮扶**，使日干趋向中和。
- 统称这些生扶帮扶的力量为"**扶**"。

### 抑
- 含义：**克制、约束**。
- 用时机：日干**较强**。
- 具体方式：用**官杀克制、食伤泄秀、财星耗身**，使日干趋向中和。
- 统称这些克制泄耗的方法为"**抑**"。

## 扶抑取用的五大情况

扶抑法可以高度概括为以下五种情况：

### 情况一：日干弱 + 官杀旺而多

**优先级**：**印枭 > 比劫 > 不取食伤**

#### 原理
- 官杀旺是造成日干弱的主要原因。
- **首选印枭**：
  - 印枭可以削弱官杀之力（泄官杀）。
  - 印枭可以生扶日干（补身）。
  - **化杀生身、化敌为友**——一举两得。
- **次选比劫**：
  - 比劫可分担官杀的克制。
  - 但效果**不如旺印通关**。
- **不选食伤**：
  - 食伤制官杀会造成**克泄交加**的局面。
  - 虽去了官杀的病，又添了食伤的病。
- **无印无比劫**时：日干受四面八方克泄耗 → 可能属于**从弱**，需另论。

#### 案例
- 癸水日元生于未月（失令），全局官杀气势极强。
- 同党与异党各占一半，但同党不占月令 → 日元偏弱，土为忌神、火为仇神。
- 必须以**金（印）为用神**，同时水可铸身、调候。
- 不适合用木（食伤）制官杀 → 易导致克泄交加。

### 情况二：日干弱 + 财星旺而多

**优先级**：**比劫 > 印枭**

#### 原理
- 财星过旺是造成日干虚弱的内在因素。
- **首选比劫**：
  - 比劫帮扶日干。
  - 比劫能**直接克制财星** → 削弱病根。
  - **一举两得**。
- **次选印枭**：
  - 印枭不能直接克财，但可生身 → 日干由弱转旺 → 能承受财星。
  - 但印星会被旺财所克（财克印）→ 效果次于比劫。

#### 案例
- 甲木日元生于丑月（失令），全局财星旺盛，印星不显。
- 日元得地、得助，但因失令仍偏弱。
- 选**木（比劫）为用神**帮扶日元。
- **水为喜神**（生用神）。
- 土为忌神，火为仇神。
- 此外，金（印星）若出现反而克日元使日元和比劫的力量减弱——间接助忌，因此**金也为仇神**。

### 情况三：日干弱 + 食伤旺而多

**优先级**：**印枭 > 比劫**

#### 原理
- 食伤过旺泄身是造成身弱的致命因素。
- **首选印枭**：
  - 印枭生身 → 日干由弱转强。
  - 印枭**克制食伤** → 食伤无法泄身。
  - **一举两得**——印星是克制食伤的有力法宝。
- **次选比劫**：
  - 比劫帮身可减轻食伤泄身的力量。
- **无印无比劫**时：属于**用神不显**，需改用其他取用法。

#### 案例
- 丙火日元生于午月（得令），但除月令外整个命局都是克泄耗日元的五行。
- 土（食伤）泄日元之气过重。
- 虽得月令帮扶，总体仍偏弱。
- 全局**无印**，只能选**比劫（火）为用**。
- 岁运遇印销时，木可为喜神。
- 土为忌神，金（财）帮忌克喜 → 为仇神。
- 水（官杀）既克主又耗忌但都起不了作用 → **为闲神**。

### 情况四：日干旺 + 印枭旺而多

**优先级**：**财星 > 食伤 / 官杀**

#### 原理
- 印枭过旺是日干过强的主要原因。
- **首选财星**：
  - 财星**克制生身的印枭** → 切断助日干之源。
  - 财星又**消耗日干** → 日干由强转中。
  - **一举两得**。
- **注意**：若印多仍身弱（例外情况），仍应选用比劫而非财。
- **次选食伤**：
  - 食伤泄身、食伤生财、食伤削弱印星。
  - 但要防**枭神夺食**的不利组合。
- **再选官杀**：
  - 官杀克制日干 → 削弱日干。
  - **重大例外**：若命局形成**官杀生印、印生身的组合** → **官杀绝不可取**！
    - 因为官杀反而成了助日干强旺的推手。

#### 案例
- 甲木日元出生在寅月、又逢寅年（比肩当令助身）。
- 天干三个壬水生身 → 日元强旺无疑。
- **全局无财**（辰中霸财不透，力量薄弱）。
- **全局无食伤**可用。
- 只能用克：以日支申金（七杀）为用神。
- 但注意：**天干不宜用官杀**——因天干全是印销，官杀出现会形成**官杀生印、印生身**，反而不利日元。
- 地支的官杀可用。
- 遇财财的岁运时土可作喜神。
- 火在此局为闲神（无食伤则天干不宜再出食神，否则枭神夺食）。

### 情况五：日干旺 + 比劫旺而多

**优先级**：**官杀 > 食伤 > 财星**；但注意 **"比劫旺有力时喜泄不喜克"**

#### 原理
- 比劫过多过旺是日干强盛的内在根源。
- **首选官杀**：
  - 官杀克制比劫和日干 → 保护财星。
  - **重大例外**：若命局形成**官杀生印、印生比劫**的组合 → 官杀**不可取**（反助忌神）。
- **次选食伤**：
  - 食伤削弱比劫和日干的力量。
  - 食伤生财 → 消耗日干。
- **最后选财星**：
  - 财星消耗日干、财生官杀再制比劫。

#### 关键原则：比劫旺且有力时，**最好取食伤而非官杀**
- 原因："**比劫旺者喜泄**"。
- 用官杀来克旺盛的比劫，容易**激怒比劫** → 导致"**比杀相战**"或"**杀比相战**"，局势失控。

#### 案例
- 癸水日元生于子月，地支亥子丑三会水局，全局水势旺盛，日元极其强旺。
- 虽有丑土七杀，但已被亥子合会化为水局，七杀力量远弱于伤官和食神。
- 比劫旺且有力 → **喜泄不喜克** → 选**伤官（木）为用神**。
- 火为喜神（木生火转化能量并消耗日主）。
- 土为忌神。

## 综合扶抑的重要提醒

### 提醒一：不能生硬套用
- 五种情况**不能机械套用**，必须根据八字命局的具体组合来定。
- 特别要注意关键组合：
  - **日元强 + 官杀生印 + 印生身**：此时官杀不可作用神。
  - **日元弱 + 官杀强**：一般用印化杀生身，**不宜用食伤制官杀**（克泄交加）。

### 提醒二：天干与地支用神可能不同
- 某些命局的组合，天干和地支所能选取的用神**不一定相同**。
- 案例：天干全是印销时，**天干不宜用官杀**（会形成官杀生印生身）；但**地支的官杀可用**。

### 提醒三：扶抑只是一种方法
- 有些八字喜用神的确定并非那么容易。
- 扶抑只是五种取用法之一。
- 当扶抑法难以确定时，需**结合调候、通关、病药、专旺**综合考量。

## 扶抑法的局限

### 不适用的命局
1. **中和偏难判的八字**：日元强弱判断本身就困难，扶抑套用容易出错。
2. **从格**：日元极弱到需顺从，扶抑法取的用神会完全相反（从弱应用财官食伤，而非印比）。
3. **专旺格**：全局一气，扶抑法会让局势失衡。
4. **调候为首的命局**：亥子丑、巳午未月至寒至暖的八字，需先调候再论扶抑。

### 案例：扶抑与调候冲突
- 己土生于丑月（得令）、月日伏吟增强、年支午火助旺 → 日元非常强旺。
- **扶抑角度**：以金水为喜用（食伤泄秀生财）。
- **调候角度**：丑月需火暖土 → 但若用火，日元反而更强旺。
- **冲突处理**：**以扶抑为主**，调候服从 → 最终以金水为用。

## 扶抑 vs 调候 的优先级

### 总原则
- 大多数情况下，扶抑、调候、阴阳三者**统一**（需求方向相同）→ 可相互配合使用。
- 当三者冲突时：**一般以扶抑（服役）取用为主**。

### 原因
- 扶抑法针对**整体力量失衡**的根本问题。
- 调候针对**季节性的偏燥偏寒**问题。
- 当日元过强时，用调候的"暖"或"润"反而会加剧失衡。

## 常见误区

### 1. 身弱就补印比、身强就抑官杀食伤财
- 错。必须**看病根在哪**。
  - 身弱因官杀旺 → 用印（不是比劫）。
  - 身弱因财旺 → 用比劫（不是印）。
  - 身弱因食伤旺 → 用印（不是比劫）。

### 2. 日干弱 + 官杀旺时用食伤制杀
- 错。容易造成**克泄交加**——杀去但食伤又变成新病。
- 应**用印化杀生身**。

### 3. 日干旺 + 比劫旺时用官杀克之
- 通常错。比劫旺有力时**喜泄**——应用食伤泄秀。
- 硬用官杀会激怒比劫，导致"比杀相战"。

### 4. 忽视"官杀生印、印生身"的禁忌
- 错。日干旺且印多时，**官杀不可取**——官杀反而会助印、印助日干。
- 此时只能用财星克印。

### 5. 天干地支用同一用神
- 不一定。某些命局天干用神与地支用神完全不同——需分别判断。

### 6. 扶抑是万能方法
- 错。扶抑法**不适用于从格、专旺格、至寒至暖命局**。

## 扶抑法的实战流程

### 第一步：判日元旺衰
- 综合得令、得地、得生、得助判定。
- 注意 **身强 ≠ 身旺**，无根者即使同党多也只是虚强。

### 第二步：锁定病根
- 日干弱时：是被官杀克、财耗、食伤泄？
- 日干强时：是印多、比劫多？
- **不同病根对应不同用神优先级**。

### 第三步：套用五种情况定用神
- 按上文五大情况的优先级选用。
- 注意关键例外：官杀生印生身、比劫旺喜泄等。

### 第四步：检验调候、阴阳是否冲突
- 三者统一 → 直接取用。
- 三者冲突 → 以扶抑为主，调候服从。

### 第五步：区分天干用神与地支用神
- 若组合有差异，分别确认天干用神与地支用神。

## 一句话总结
- **扶抑 = 扶弱抑强**：日干弱则扶（印、比），日干强则抑（官杀、食伤、财）。
- **对症下药**：病根决定用神方向——官杀旺用印、财旺用比劫、食伤旺用印、印旺用财、比劫旺用食伤。
- 扶抑法应用最广但非万能——从格、专旺、至寒至暖之局需改用他法。
- 与调候冲突时 **以扶抑为主**。
